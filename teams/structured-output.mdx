---
title: 構造化出力
---

チームは、個々のエージェントと同様に、Pydanticモデルを使用して構造化データを生成できます。この機能は、複数のエージェントが協力して構造化された結果を生成する、調整されたデータ抽出、分析、レポート生成に最適です。

## 例

構造化された`StockReport`を生成する株式リサーチチームを作成しましょう。

```python stock_team.py
from typing import List
from pydantic import BaseModel, Field
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.yfinance import YFinanceTools

class StockAnalysis(BaseModel):
    symbol: str
    company_name: str
    analysis: str

class CompanyAnalysis(BaseModel):
    company_name: str
    analysis: str

class StockReport(BaseModel):
    symbol: str = Field(..., description="株式ティッカーシンボル")
    company_name: str = Field(..., description="会社の正式名称")
    current_price: str = Field(..., description="現在の株価")
    analysis: str = Field(..., description="複数の視点を組み合わせた包括的な分析")
    recommendation: str = Field(..., description="投資推奨：買い、ホールド、または売り")

# 特殊なエージェントを作成
stock_searcher = Agent(
    name="Stock Searcher",
    model=OpenAIChat("gpt-4o"),
    response_model=StockAnalysis,
    role="現在の株価情報と価格データを検索します。",
    tools=[
        YFinanceTools(
            stock_price=True,
            analyst_recommendations=True,
        )
    ],
)

company_info_agent = Agent(
    name="Company Info Searcher",
    model=OpenAIChat("gpt-4o"),
    role="会社のファンダメンタルズと最近のニュースを調査します。",
    response_model=CompanyAnalysis,
    tools=[
        YFinanceTools(
            stock_price=False,
            company_info=True,
            company_news=True,
        )
    ],
)

# 構造化された出力を持つチームを作成
stock_research_team = Team(
    name="Stock Research Team",
    mode="coordinate",
    model=OpenAIChat("gpt-4o"),
    members=[stock_searcher, company_info_agent],
    response_model=StockReport,
    markdown=True,
    show_members_responses=True,
)

stock_research_team.print_response("NVDAの包括的な株価レポートを教えてください")
```

チームはメンバー間で連携し、構造化された`StockReport`オブジェクトを生成します。

```python
StockReport(
│   symbol='NVDA',
│   company_name='NVIDIA Corporation',
│   current_price='$875.42',
│   analysis='NVIDIAは、H100および今後のH200 GPUに対する強い需要により、AIチップ市場で引き続き優位に立っています。同社は、エンタープライズAIの採用とクラウドプロバイダーの拡大により、データセンター収益で例外的な成長を示しています。主要なテクノロジー企業との最近の提携は市場での地位を強化していますが、AMDやIntelからの競争が激化しています。',
│   recommendation='Buy'
)
```

## パーサーモデルの使用

プライマリモデルからの出力の解析と構造化に、追加のモデルを使用できます。このアプローチは、プライマリモデルが推論タスクに最適化されている場合に特に効果的です。そのようなモデルは、詳細な構造化応答を一貫して生成しない可能性があるためです。

```python
team = Team(
    name="Stock Research Team",
    mode="coordinate",
    model=Claude(id="claude-sonnet-4-20250514"),
    members=[stock_searcher, company_info_agent],
    response_model=StockReport,
    parser_model=OpenAIChat(id="gpt-4o"),
)
```

カスタムの`parser_model_prompt`をパーサーモデルに提供することもできます。

## ストリーミング構造化出力

チームはストリーミングと構造化出力をサポートしており、`content`イベントは完全な構造化結果を単一のイベントとして含みます。

```python streaming_team.py
from typing import List
from pydantic import BaseModel, Field
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.yfinance import YFinanceTools

class MarketAnalysis(BaseModel):
    sector: str = Field(..., description="分析対象の市場セクター")
    key_trends: List[str] = Field(..., description="セクターに影響を与える主要なトレンド")
    top_performers: List[str] = Field(..., description="セクターで最もパフォーマンスの高い株式")
    market_outlook: str = Field(..., description="全体的な市場見通しと予測")
    risk_factors: List[str] = Field(..., description="考慮すべき主要なリスク")

# リサーチエージェントを作成
trend_analyst = Agent(
    name="Trend Analyst",
    model=OpenAIChat("gpt-4o"),
    role="市場トレンドとセクターパフォーマンスを分析します。",
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True)]
)

risk_assessor = Agent(
    name="Risk Assessor",
    model=OpenAIChat("gpt-4o"),
    role="市場のリスクと機会を特定し評価します。",
    tools=[YFinanceTools(company_news=True, company_info=True)]
)

# ストリーミングチームを作成
market_research_team = Team(
    name="Market Research Team",
    mode="coordinate",
    model=OpenAIChat("gpt-4o"),
    members=[trend_analyst, risk_assessor],
    response_model=MarketAnalysis,
    markdown=True,
    show_members_responses=True,
)

# チームの応答をストリーミング
market_research_team.print_response(
    "2024年第1四半期のテクノロジーセクターを分析してください",
    stream=True,
    stream_intermediate_steps=True
)
```

<Note>
チームと構造化出力でストリーミングする場合、個々のチームメンバーからの途中経過が表示されますが、最終的な構造化結果は、段階的にストリーミングされるのではなく、単一の完全なチャンクとして配信されます。
</Note>
