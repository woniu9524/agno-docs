Teamは、タスクを達成するために協力するエージェント（または他のサブチーム）の集まりです。Teamはタスクを解決するために「調整」、「協調」、または「ルーティング」することができます。

`Team`は、`Agent`または`Team`のインスタンスである`members`のリストを持ちます。

```python
from agno.team import Team
from agno.agent import Agent

team = Team(members=[
    Agent(name="Agent 1", role="あなたは英語で質問に答えます"),
    Agent(name="Agent 2", role="あなたは中国語で質問に答えます"),
    Team(name="Team 1", role="あなたはフランス語で質問に答えます"),
])
```

Teamは、Teamの`mode`に応じてタスクをメンバーに転送します。

<Note>

Teamリーダーによる識別を容易にするために、Teamメンバーの`name`と`role`フィールドを指定することが推奨されます。

</Note>

## モード

### ルートモード

[**ルートモード**](/teams/route)では、Teamリーダーはリクエストの内容に基づいて、最も適切なTeamメンバーにユーザーのリクエストをルーティングします。メンバーの応答は直接ユーザーに返され、Teamリーダーは応答を解釈/変換しません。

<Note>

`async`実行では、Teamリーダーによって複数のメンバーに同時に転送された場合、これらのメンバーは並行して実行されます。

</Note>


### コーディネートモード

[**コーディネートモード**](/teams/coordinate)では、TeamリーダーはTeamメンバーにタスクを委任し、それらの出力を統合された応答にまとめます。Teamリーダーは、リクエストに応じて、複数のメンバーに同時に送信したり、順番に送信したりすることができます。

<Note>

`async`実行では、Teamリーダーによって複数のメンバーに同時に転送された場合、これらのメンバーは並行して実行されます。

</Note>

### コラボレーションモード

[**コラボレーションモード**](/teams/collaborate)では、すべてのTeamメンバーに同じタスクが与えられ、Teamリーダーはそれらの出力を統合された応答にまとめます。

<Note>

`async`実行では、すべてのメンバーは並行して実行されます。

</Note>

## Teamのメモリと履歴

Teamは以前のやり取りのメモリを維持し、文脈認識を可能にします。

```python
from agno.team import Team

team_with_memory = Team(
    name="メモリを持つチーム",
    members=[agent1, agent2],
    add_history_to_messages=True,
    num_history_runs=5,
)

# Teamは以前のやり取りを記憶します
team_with_memory.print_response("量子コンピューティングにおける主要な課題は何ですか？")
team_with_memory.print_response("言及された2番目の課題について詳しく説明してください")
```

Teamはユーザーのメモリを管理することもできます。

```python
from agno.team import Team
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory

# 持続的なストレージを持つメモリインスタンスを作成します
memory_db = SqliteMemoryDb(table_name="memory", db_file="memory.db")
memory = Memory(db=memory_db)

team_with_memory = Team(
    name="メモリを持つチーム",
    members=[agent1, agent2],
    memory=memory,
    enable_agentic_memory=True,
)

team_with_memory.print_response("こんにちは！私の名前はJohn Doeです。")
team_with_memory.print_response("私の名前は何ですか？")
```

## Teamの知識

Teamはナレッジベースを使用して情報を保存および取得できます。

```python
from pathlib import Path

from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.openai import OpenAIChat
from agno.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.vectordb.lancedb import LanceDb, SearchType

# パスを設定します
cwd = Path(__file__).parent
tmp_dir = cwd.joinpath("tmp")
tmp_dir.mkdir(parents=True, exist_ok=True)

# ナレッジベースを初期化します
agno_docs_knowledge = UrlKnowledge(
    urls=["https://docs.agno.com/llms-full.txt"],
    vector_db=LanceDb(
        uri=str(tmp_dir.joinpath("lancedb")),
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        embedder=OpenAIEmbedder(id="text-embedding-3-small"),
    ),
)

web_agent = Agent(
    name="Web検索エージェント",
    role="ウェブ検索リクエストを処理します",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    instructions=["常にソースを含めてください"],
)

team_with_knowledge = Team(
    name="知識を持つチーム",
    members=[web_agent],
    model=OpenAIChat(id="gpt-4o"),
    knowledge=agno_docs_knowledge,
    show_members_responses=True,
    markdown=True,
)

if __name__ == "__main__":
    # ナレッジベースがロードされたらFalseに設定します
    load_knowledge = True
    if load_knowledge:
        agno_docs_knowledge.load()

    team_with_knowledge.print_response("Agnoフレームワークについて教えてください", stream=True)
```

Teamはユーザーのメモリを管理することもできます。

```python
from agno.team import Team
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory

# 持続的なストレージを持つメモリインスタンスを作成します
memory_db = SqliteMemoryDb(table_name="memory", db_file="memory.db")
memory = Memory(db=memory_db)

team_with_memory = Team(
    name="メモリを持つチーム",
    members=[agent1, agent2],
    memory=memory,
    enable_user_memories=True,
)

team_with_memory.print_response("こんにちは！私の名前はJohn Doeです。")
team_with_memory.print_response("私の名前は何ですか？")
```

## セッション要約

セッション要約を有効にするには、`Team`で`enable_session_summaries=True`を設定します。

```python
from agno.team import Team
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory

team_with_session_summaries = Team(
    name="メモリを持つチーム",
    members=[agent1, agent2],
    enable_session_summaries=True,
)

team_with_session_summaries.print_response("こんにちは！私の名前はJohn Doeで、ニューヨーク市に住んでいます。")

session_summary = team_with_session_summaries.get_session_summary()
print("セッション要約: ", session_summary.summary)
```

## 例

### 多言語チーム

異なる言語で質問に答えるために異なるモデルを使用する簡単な例を見てみましょう。このチームは3つの専門エージェントで構成され、Teamリーダーはユーザーの質問を適切な言語エージェントにルーティングします。

```python multilanguage_team.py
from agno.agent import Agent
from agno.models.deepseek import DeepSeek
from agno.models.mistral.mistral import MistralChat
from agno.models.openai import OpenAIChat
from agno.team.team import Team

english_agent = Agent(
    name="英語エージェント",
    role="あなたは英語でのみ回答します",
    model=OpenAIChat(id="gpt-4o"),
)
chinese_agent = Agent(
    name="中国語エージェント",
    role="あなたは中国語でのみ回答します",
    model=DeepSeek(id="deepseek-chat"),
)
french_agent = Agent(
    name="フランス語エージェント",
    role="あなたはフランス語でのみ回答できます",
    model=MistralChat(id="mistral-large-latest"),
)

multi_language_team = Team(
    name="多言語チーム",
    mode="route",
    model=OpenAIChat("gpt-4o"),
    members=[english_agent, chinese_agent, french_agent],
    show_tool_calls=True,
    markdown=True,
    description="あなたは質問を適切な言語エージェントにルーティングする言語ルーターです。",
    instructions=[
        "ユーザーの質問の言語を特定し、適切な言語エージェントにルーティングしてください。",
        "Teamメンバーにエージェントがいない言語で質問された場合は、英語で次のように応答してください:",
        "'私は次の言語でのみ回答できます: 英語、中国語、フランス語。これらの言語のいずれかで質問してください。'",
        "エージェントにルーティングする前に、必ずユーザーの入力言語を確認してください。",
        "イタリア語のようなサポートされていない言語の場合は、上記のメッセージを英語で応答してください。",
    ],
    show_members_responses=True,
)


if __name__ == "__main__":
    # サポートされているすべての言語で「元気ですか？」と尋ねます
    multi_language_team.print_response("Comment allez-vous?", stream=True)  # フランス語
    multi_language_team.print_response("How are you?", stream=True)  # 英語
    multi_language_team.print_response("你好吗？", stream=True)  # 中国語
    multi_language_team.print_response("Come stai?", stream=True)  # イタリア語
```

### コンテンツチーム

2つの専門エージェントを使用してブログ記事を作成する別の例を見てみましょう。Teamリーダーはエージェントを調整してブログ記事を作成します。

```python content_team.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools

# 個別の専門エージェントを作成します
researcher = Agent(
    name="リサーチャー",
    role="情報を見つける専門家",
    tools=[DuckDuckGoTools()],
    model=OpenAIChat("gpt-4o"),
)

writer = Agent(
    name="ライター",
    role="明確で魅力的なコンテンツを書く専門家",
    model=OpenAIChat("gpt-4o"),
)

# これらのエージェントでTeamを作成します
content_team = Team(
    name="コンテンツチーム",
    mode="coordinate",
    members=[researcher, writer],
    instructions="あなたは高品質なコンテンツを作成するために協力するリサーチャーとライターのチームです。",
    model=OpenAIChat("gpt-4o"),
    markdown=True,
)

# タスクでTeamを実行します
content_team.print_response("量子コンピューティングに関する短い記事を作成してください")
```

### リサーチチーム

複数の専門エージェントを組み合わせたリサーチチームの例を次に示します。

<Steps>
  <Step title="HackerNews Teamを作成">
    `hackernews_team.py`ファイルを作成します

    ```python hackernews_team.py
    from typing import List

    from agno.agent import Agent
    from agno.models.openai import OpenAIChat
    from agno.team import Team
    from agno.tools.duckduckgo import DuckDuckGoTools
    from agno.tools.hackernews import HackerNewsTools
    from agno.tools.newspaper4k import Newspaper4kTools
    from pydantic import BaseModel

    class Article(BaseModel):
        title: str
        summary: str
        reference_links: List[str]


    hn_researcher = Agent(
        name="HackerNewsリサーチャー",
        model=OpenAIChat("gpt-4o"),
        role="hackernewsからトップストーリーを取得します。",
        tools=[HackerNewsTools()],
    )

    web_searcher = Agent(
        name="Webサーチャー",
        model=OpenAIChat("gpt-4o"),
        role="トピックに関する情報をウェブで検索します",
        tools=[DuckDuckGoTools()],
        add_datetime_to_instructions=True,
    )

    article_reader = Agent(
        name="記事リーダー",
        role="URLから記事を読みます。",
        tools=[Newspaper4kTools()],
    )

    hackernews_team = Team(
        name="HackerNewsチーム",
        mode="coordinate",
        model=OpenAIChat("gpt-4o"),
        members=[hn_researcher, web_searcher, article_reader],
        instructions=[
            "まず、ユーザーが尋ねていることについてhackernewsで検索してください。",
            "次に、記事リーダーにストーリーのリンクを読んで、詳細情報を取得するように依頼してください。",
            "重要: 記事リーダーに読むためのリンクを提供する必要があります。",
            "次に、Webサーチャーに各ストーリーを検索して詳細情報を取得するように依頼してください。",
            "最後に、思慮深く魅力的な要約を提供してください。",
        ],
        response_model=Article,
        show_tool_calls=True,
        markdown=True,
        debug_mode=True,
        show_members_responses=True,
    )

    # Teamを実行します
    report = hackernews_team.run(
        "hackernewsのトップストーリーは何ですか？"
    ).content

    print(f"タイトル: {report.title}")
    print(f"要約: {report.summary}")
    print(f"参照リンク: {report.reference_links}")
    ```

  </Step>
  <Step title="Teamを実行">
    ライブラリをインストールします

    ```shell
    pip install openai duckduckgo-search newspaper4k lxml_html_clean agno
    ```

    Teamを実行します

    ```shell
    python hackernews_team.py
    ```

  </Step>
</Steps>
