---
title: 固定サイズチャンキング
---

固定サイズチャンキングは、指定されたサイズの小さなチャンクにドキュメントを分割する方法で、チャンク間のオプションのオーバーラップも可能です。これは、大きなドキュメントをより小さく管理しやすいピースで処理したい場合に役立ちます。

## 使用方法

```python
from agno.agent import Agent
from agno.document.chunking.fixed import FixedSizeChunking
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=PgVector(table_name="recipes_fixed_size_chunking", db_url=db_url),
    chunking_strategy=FixedSizeChunking(),
)
knowledge_base.load(recreate=False)  # 初回実行後にコメントアウトしてください

agent = Agent(
    knowledge_base=knowledge_base,
    search_knowledge=True,
)

agent.print_response("タイカレーの作り方は？", markdown=True)
```
