---
title: Document Chunking
---

Document chunking は、段落やセクションのようなドキュメント構造に基づいて、ドキュメントをより小さなチャンクに分割する方法です。固定された文字数で分割するのではなく、自然なドキュメント境界を分析します。これは、セマンティックな意味とコンテキストを維持しながら、大きなドキュメントを処理したい場合に役立ちます。

## Usage

```python
from agno.agent import Agent
from agno.document.chunking.document import DocumentChunking
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=PgVector(table_name="recipes_document_chunking", db_url=db_url),
    chunking_strategy=DocumentChunking(),
)
knowledge_base.load(recreate=False)  # 初回実行後にコメントアウトしてください

agent = Agent(
    knowledge_base=knowledge_base,
    search_knowledge=True,
)

agent.print_response("How to make Thai curry?", markdown=True)

```
