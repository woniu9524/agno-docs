---
title: 语义分块
---

语义分块是一种通过使用嵌入分析文本片段之间的语义相似性来将文档分割成更小块的方法。它使用 chonkie 库来识别语义含义发生显著变化的自然断点，这基于可配置的相似度阈值。通过确保语义相关的内​​容保留在同一块中，同时在有意义的主题转换处进行分割，这比固定大小的分块能更好地保留上下文和含义。

```python
from agno.agent import Agent
from agno.document.chunking.semantic import SemanticChunking
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=PgVector(table_name="recipes_semantic_chunking", db_url=db_url),
    chunking_strategy=SemanticChunking(),
)
knowledge_base.load(recreate=False)  # 首次运行时注释掉此行

agent = Agent(
    knowledge_base=knowledge_base,
    search_knowledge=True,
)

agent.print_response("How to make Thai curry?", markdown=True)

```
