**評価**は、エージェントとチームの単体テストです。パフォーマンスを測定および改善するために、賢く使用してください。Agnoは、エージェントを評価するための3つの次元を提供します。
- **精度:** エージェントの応答はどの程度完全/正しい/正確か（LLMとしての評価）
- **パフォーマンス:** エージェントはどのくらいの速さで応答し、メモリ使用量はどのくらいか？
- **信頼性:** エージェントは期待されるツール呼び出しを行いますか？

## 精度

精度評価では、入力/出力ペアを使用して、エージェントとチームのパフォーマンスをゴールドスタンダードの回答と比較して測定します。より大きなモデルを使用して、エージェントの応答をスコアリングします（LLMとしての評価）。

#### 例

この例では、`AccuracyEval`がエージェントを入力で実行し、その後、より大きなモデル（`o4-mini`）を使用して、提供されたガイドラインに従ってエージェントの応答をスコアリングします。

```python calculate_accuracy.py
from typing import Optional
from agno.agent import Agent
from agno.eval.accuracy import AccuracyEval, AccuracyResult
from agno.models.openai import OpenAIChat
from agno.tools.calculator import CalculatorTools

evaluation = AccuracyEval(
    model=OpenAIChat(id="o4-mini"),
    agent=Agent(model=OpenAIChat(id="gpt-4o"), tools=[CalculatorTools(enable_all=True)]),
    input="What is 10*5 then to the power of 2? do it step by step",
    expected_output="2500",
    additional_guidelines="Agent output should include the steps and the final answer.",
)

result: Optional[AccuracyResult] = evaluation.run(print_results=True)
assert result is not None and result.avg_score >= 8
```

既存の出力に対して`AccuracyEval`を実行することもできます（エージェントを実行せずに）。

```python accuracy_eval_with_output.py
from typing import Optional

from agno.agent import Agent
from agno.eval.accuracy import AccuracyEval, AccuracyResult
from agno.models.openai import OpenAIChat
from agno.tools.calculator import CalculatorTools

evaluation = AccuracyEval(
    model=OpenAIChat(id="o4-mini"),
    input="What is 10*5 then to the power of 2? do it step by step",
    expected_output="2500",
    num_iterations=1,
)
result_with_given_answer: Optional[AccuracyResult] = evaluation.run_with_output(
    output="2500", print_results=True
)
assert result_with_given_answer is not None and result_with_given_answer.avg_score >= 8
```

## パフォーマンス

パフォーマンス評価は、エージェントまたはチームのレイテンシとメモリ使用量を測定します。

<Note>
レイテンシは主にモデルAPIの応答時間によって決まりますが、パフォーマンスは常に念頭に置き、特定のコンポーネントの有無にかかわらず、エージェントまたはチームのパフォーマンスを追跡する必要があります。例えば、ストレージ、メモリ、新しいプロンプト、または新しいモデルがある場合とない場合の平均レイテンシを知ることは有益です。
</Note>

#### 例

```python storage_performance.py
"""Run `pip install openai agno` to install dependencies."""

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.eval.perf import PerfEval

def simple_response():
    agent = Agent(model=OpenAIChat(id='gpt-4o-mini'), system_message='Be concise, reply with one sentence.', add_history_to_messages=True)
    response_1 = agent.run('What is the capital of France?')
    print(response_1.content)
    response_2 = agent.run('How many people live there?')
    print(response_2.content)
    return response_2.content


simple_response_perf = PerfEval(func=simple_response, num_iterations=1, warmup_runs=0)

if __name__ == "__main__":
    simple_response_perf.run(print_results=True)
```

## 信頼性

エージェントまたはチームを信頼できるものにするものは何でしょうか？

- 期待されるツール呼び出しを行いますか？
- エラーを適切に処理しますか？
- モデルAPIのレート制限を尊重しますか？

#### 例

最初のチェックは、エージェントが期待されるツール呼び出しを行うことを確認することです。例を次に示します。

```python reliability.py
from typing import Optional

from agno.agent import Agent
from agno.eval.reliability import ReliabilityEval, ReliabilityResult
from agno.tools.calculator import CalculatorTools
from agno.models.openai import OpenAIChat
from agno.run.response import RunResponse


def multiply_and_exponentiate():

    agent=Agent(
        model=OpenAIChat(id="gpt-4o-mini"),
        tools=[CalculatorTools(add=True, multiply=True, exponentiate=True)],
    )
    response: RunResponse = agent.run("What is 10*5 then to the power of 2? do it step by step")
    evaluation = ReliabilityEval(
        agent_response=response,
        expected_tool_calls=["multiply", "exponentiate"],
    )
    result: Optional[ReliabilityResult] = evaluation.run(print_results=True)
    result.assert_passed()


if __name__ == "__main__":
    multiply_and_exponentiate()
```

信頼性評価は現在`beta`版です。
</Note>
