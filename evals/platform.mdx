## 評価の追跡

CLIで評価を実行するだけでなく、Agnoプラットフォームでも追跡できます。これは、結果を追跡し、チームと共有するのに役立ちます。
次の手順で実行してください。

<Steps>

<Step title="認証">
  CLIまたはAPIキーを使用して認証できます。

  **CLIを使用する場合:**
```bash
ag setup
```

**APIキーを使用する場合:**

[Agno App](https://app.agno.com/settings)からAPIキーを取得し、ローカルで実行中のエージェントをAgnoプラットフォームにリンクするために使用します。

```bash
export AGNO_API_KEY=your_api_key_here
```

</Step>

<Step title="評価の追跡">
  評価を実行する際に、`monitoring=True`を設定すると、すべての実行がAgnoプラットフォームで追跡されます。
  ```python
from agno.agent import Agent
from agno.eval.accuracy import AccuracyEval
from agno.models.openai import OpenAIChat

evaluation = AccuracyEval(
    model=OpenAIChat(id="gpt-4o"),
    agent=Agent(model=OpenAIChat(id="gpt-4o")),
    input="What is 10*5 then to the power of 2? do it step by step",
    expected_output="2500",
    monitoring=True, # これにより監視がアクティブになります
)

# この実行はAgnoプラットフォームで追跡されます
result = evaluation.run(print_results=True)
  ```

  `AGNO_MONITOR`環境変数を`true`に設定することでも、すべての評価実行を追跡できます。
</Step>

<Step title="評価の表示">
    これで、[app.agno.com/evaluations](https://app.agno.com/evaluations)のAgnoプラットフォームで評価を表示できます。
</Step>
</Steps>
