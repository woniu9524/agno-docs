title: Reasoning Agents
---

Reasoning Agents は、Agno が開発した新しいタイプのマルチエージェントシステムで、思考連鎖（chain of thought）とツールの使用を組み合わせています。

`reasoning=True` を設定することで、任意の Agent で推論を有効にできます。

`reasoning=True` を持つ Agent にタスクが与えられると、まず個別の「Reasoning Agent」が思考連鎖を使用して問題を解決します。各ステップで、ツールを呼び出して情報を収集し、結果を検証し、最終的な回答に到達するまで反復します。Reasoning Agent が最終的な回答を得ると、元の Agent に結果を渡し、検証して応答を提供します。

### 例

```python reasoning_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning=True,
    markdown=True,
)
reasoning_agent.print_response(
    "トロッコ問題について解答してください。複数の倫理的フレームワークを評価し、解決策のASCII図を含めてください。",
    stream=True,
    show_full_reasoning=True,
)
```

## Agentic Reasoning の有効化

Agentic Reasoning を有効にするには、`reasoning=True` を設定するか、`reasoning_model` を構造化出力をサポートするモデルに設定します。`reasoning_model` を設定しない場合、プライマリ `Agent` モデルが推論に使用されます。

### Reasoning Model の要件

`reasoning_model` は構造化出力を処理できる必要があります。これには、構造化出力をネイティブにサポートする gpt-4o や claude-3-sonnet のようなモデル、または JSON モードを使用して構造化出力をサポートする gemini モデルが含まれます。

### ネイティブ推論をサポートする Reasoning Model の使用

`reasoning_model` を o3-mini や deepseek-r1 のようなネイティブ推論をサポートするモデルに設定した場合、推論にはその推論モデルが使用され、応答にはプライマリ `Agent` モデルが使用されます。詳細については、[Reasoning Models + Response Models](/reasoning/reasoning-models#reasoning-model-response-model) を参照してください。

## ツールの使用による推論

推論 Agent でツールを使用することもできます。推論できるファイナンス Agent を作成しましょう。

```python finance_reasoning.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True)],
    instructions=["Use tables to show data"],
    show_tool_calls=True,
    markdown=True,
    reasoning=True,
)
reasoning_agent.print_response("NVDAとTSLAを比較するレポートを作成してください", stream=True, show_full_reasoning=True)
```

## その他の例

### 論理パズル

```python logical_puzzle.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

task = (
    "3人の宣教師と3人のカニバルが川を渡る必要があります。 "
    "彼らは一度に最大2人を運べるボートを持っています。 "
    "川のどちらかの岸でカニバルの数が宣教師の数を上回った場合、カニバルは宣教師を食べます。 "
    "6人全員が安全に川を渡るにはどうすればよいですか？段階的な解決策を提供し、解決策をASCII図で表示してください。"
)
reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o-2024-08-06"), reasoning=True, markdown=True
)
reasoning_agent.print_response(task, stream=True, show_full_reasoning=True)
```
