---
title: 推論モデル
---

推論モデルは、回答する前に思考するよう強化学習でトレーニングされた新しいクラスの大規模言語モデルです。応答する前に、長い内部思考連鎖を生成します。推論モデルの例としては、以下のようなものがあります。
- OpenAI o1-pro および o3-mini
- Claude 3.7 sonnet (拡張思考モード)
- Gemini 2.0 flash thinking
- DeepSeek-R1

推論モデルは、行動を起こす前に計画を深く検討し、考え抜きます。それは、モデルが**応答の生成を開始する前に行うこと**すべてです。推論モデルは、シングルショットのユースケースに優れています。複数回のやり取りを必要としない難しい問題（コーディング、数学、物理学）の解決や、ツールの連続的な呼び出しに最適です。

## 例

### o3-mini

```python o3_mini.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(model=OpenAIChat(id="o3-mini"))
agent.print_response(
    "トロッコ問題について解決策を提示してください。複数の倫理的フレームワークを評価し、解決策のASCII図を含めてください。",
    stream=True,
)
```

### o3-mini とツール

```python o3_mini_with_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=OpenAIChat(id="o3-mini"),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True)],
    instructions="データを表形式で表示してください。",
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("NVDAとTSLAを比較するレポートを作成してください", stream=True)
```

### o3-mini と推論努力

```python o3_mini_with_reasoning_effort.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=OpenAIChat(id="o3-mini", reasoning_effort="high"),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True)],
    instructions="データを表形式で表示してください。",
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("NVDAとTSLAを比較するレポートを作成してください", stream=True)
```

### Groq を使用した DeepSeek-R1

```python deepseek_r1_using_groq.py
from agno.agent import Agent
from agno.models.groq import Groq

agent = Agent(
    model=Groq(
        id="deepseek-r1-distill-llama-70b", temperature=0.6, max_tokens=1024, top_p=0.95
    ),
    markdown=True,
)
agent.print_response("9.11 と 9.9 ではどちらが大きいですか？", stream=True)
```

## 推論モデル + レスポンスモデル

上記の DeepSeek-R1 エージェントを実行すると、応答があまり良くないことに気づくでしょう。これは、DeepSeek-R1 は問題解決には優れていますが、自然な方法で応答すること（claude sonnet や gpt-4.5 のような）はそれほど得意ではないためです。

推論のために推論モデルを使用し、応答を生成するために別のモデルを使用したい場合はどうすればよいでしょうか？

朗報です！Agno では、推論モデルと別のレスポンスモデルを組み合わせて使用できます。推論には個別のモデルを使用し、応答には別のモデルを使用することで、両方の長所を活かすことができます。

### DeepSeek-R1 + Claude Sonnet

```python deepseek_plus_claude.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.models.groq import Groq

deepseek_plus_claude = Agent(
    model=Claude(id="claude-3-7-sonnet-20250219"),
    reasoning_model=Groq(
        id="deepseek-r1-distill-llama-70b", temperature=0.6, max_tokens=1024, top_p=0.95
    ),
)
deepseek_plus_claude.print_response("9.11 と 9.9 ではどちらが大きいですか？", stream=True)
```
