---
title: Reasoning Tools
---

モデルにスクラッチパッドのような構造化された思考のためのツールを与えることで、推論能力を大幅に向上させる新しいクラスの研究が登場しています。

例えば、モデルに**「思考」ツール**を与えることで、問題解決のための専用スペースを提供し、推論能力を大幅に向上させることができます。これは、推論能力を持たないモデルに推論を追加するためのシンプルでありながら効果的なアプローチです。

このテクニックは、Anthropicによって[このブログ記事](https://www.anthropic.com/engineering/claude-think-tool)で初めて公開されましたが、公開されるずっと前から私たち自身のチームを含む多くのAIエンジニアによって実践されてきました。

## v0: The Think Tool

Think Toolの最初のバージョンは、Anthropicによって[このブログ記事](https://www.anthropic.com/engineering/claude-think-tool)で公開されました。

```python claude_thinking_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.thinking import ThinkingTools
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=Claude(id="claude-3-7-sonnet-latest"),
    tools=[
        ThinkingTools(add_instructions=True),
        YFinanceTools(
            stock_price=True,
            analyst_recommendations=True,
            company_info=True,
            company_news=True,
        ),
    ],
    instructions="Use tables where possible",
    markdown=True,
)

if __name__ == "__main__":
    reasoning_agent.print_response(
        "Write a report on NVDA. Only the report, no other text.",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
```

## v1: The Reasoning Tools

v0のThink Toolは素晴らしいスタートですが、思考スペースしか提供しないという点で制限があります。v1のReasoning Toolsは、エージェントがアクション（ツール呼び出し）の結果を**分析**できるようにすることで、これをさらに一歩進め、シーケンシャルなツール呼び出しを必要とする問題の解決能力を大幅に向上させます。

**ReasoningTools = `think` + `analyze`**

```python claude_reasoning_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.reasoning import ReasoningTools
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=Claude(id="claude-3-7-sonnet-20250219"),
    tools=[
        ReasoningTools(add_instructions=True),
        YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True),
    ],
    show_tool_calls=True,
)
reasoning_agent.print_response(
    "Write a report comparing NVDA to TSLA", stream=True, markdown=True
)
```

## v2: The Knowledge Tools

Knowledge Toolsは、v1のReasoning Toolsをさらに一歩進め、エージェントがナレッジベースを**検索**し、アクションの結果を**分析**できるようにします。

**KnowledgeTools = `think` + `search` + `analyze`**

```python knowledge_tools.py
import os
from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.openai import OpenAIChat
from agno.tools.knowledge import KnowledgeTools
from agno.vectordb.lancedb import LanceDb, SearchType


agno_docs = UrlKnowledge(
    urls=["https://docs.agno.com/llms-full.txt"],

    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        embedder=OpenAIEmbedder(id="text-embedding-3-small"),
    ),
)


knowledge_tools = KnowledgeTools(
    knowledge=agno_docs,
    think=True,
    search=True,
    analyze=True,
    add_few_shot=True,
)


agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[knowledge_tools],
    show_tool_calls=True,
    markdown=True,
)


agno_docs.load(recreate=True)


agent.print_response("How do I build multi-agent teams with Agno?", stream=True)
```
