Reasoning Agentは、応答する前に内部的な思考連鎖を経て、さまざまなアイデアを検討し、必要に応じて検証および修正を行います。Agnoは、Reasoningのアプローチとして以下の3つをサポートしています。
1. [Reasoning Models](#reasoning-models)
2. [Reasoning Tools](#reasoning-tools)
3. [Reasoning Agents](#reasoning-agents)

どのアプローチが最適かは、ユースケースによって異なります。すべて試してみて、このReasoning Agentの新時代を体験することをお勧めします。

## Reasoning Models

Reasoning Modelsは、応答する前に思考するように強化学習でトレーニングされた、独立したクラスの大規模言語モデルです。応答する前に内部的な思考連鎖を生成します。Reasoning Modelsの例としては、OpenAI o-series、拡張思考モードのClaude 3.7 sonnet、Gemini 2.0 flash thinking、DeepSeek-R1などがあります。

モデルレイヤーでのReasoningは、モデルが応答の生成を開始する**前**に行われることすべてを意味します。Reasoning Modelsは、シングルショットのユースケースに優れています。複数のターンを必要としない難しい問題（コーディング、数学、物理学）の解決や、ツールのシーケンシャルな呼び出しに最適です。

### 例

```python o3_mini.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(model=OpenAIChat(id="o3-mini"))
agent.print_response(
    "トロッコ問題の解決策を示してください。複数の倫理的フレームワークを評価し、解決策のASCII図を含めてください。",
    stream=True,
)
```

Reasoning Modelsの詳細については、[Reasoning Models Guide](/reasoning/reasoning-models)をお読みください。

## Reasoning Model + Response Model

Reasoning ModelでReasoningを行い、別のモデルで応答を生成したい場合はどうすればよいでしょうか。Reasoning Modelは問題解決に優れていますが、自然な方法で応答すること（claude sonnetやgpt-4oのような）はそれほど得意ではないことが広く知られています。

Reasoning用のモデルと応答用のモデルを別々に使用することで、両方の長所を活かすことができます。

### 例

Groqのdeepseek-r1をReasoningに使用し、claude sonnetを自然な応答に使用してみましょう。

```python deepseek_plus_claude.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.models.groq import Groq

deepseek_plus_claude = Agent(
    model=Claude(id="claude-3-7-sonnet-20250219"),
    reasoning_model=Groq(
        id="deepseek-r1-distill-llama-70b", temperature=0.6, max_tokens=1024, top_p=0.95
    ),
)
deepseek_plus_claude.print_response("9.11と9.9ではどちらが大きいですか？", stream=True)
```

## Reasoning Tools

モデルに**「思考」ツール**を与えることで、構造化された思考のための専用スペースを提供し、そのReasoning能力を大幅に向上させることができます。これは、Reasoning能力を持たないモデルにReasoningを追加するためのシンプルでありながら効果的なアプローチです。

この研究は、Anthropicが[このブログ記事](https://www.anthropic.com/engineering/claude-think-tool)で最初に発表しましたが、公開されるずっと前から、多くのAIエンジニア（当社のチームを含む）によって実践されてきました。

### 例

```python claude_thinking_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.thinking import ThinkingTools
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=Claude(id="claude-3-7-sonnet-latest"),
    tools=[
        ThinkingTools(add_instructions=True),
        YFinanceTools(
            stock_price=True,
            analyst_recommendations=True,
            company_info=True,
            company_news=True,
        ),
    ],
    instructions="可能な場合は表を使用してください",
    markdown=True,
)

if __name__ == "__main__":
    reasoning_agent.print_response(
        "NVDAに関するレポートを作成してください。レポートのみで、他のテキストは含めないでください。",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
```

Reasoning Toolsの詳細については、[Reasoning Tools Guide](/reasoning/reasoning-tools)をお読みください。

## Reasoning Agents

Reasoning Agentsは、Agnoによって開発された新しいタイプのマルチエージェントシステムであり、思考連鎖のReasoningとツールの使用を組み合わせています。

`reasoning=True`を設定することで、任意のAgentでReasoningを有効にできます。

`reasoning=True`を持つAgentにタスクが与えられると、別の「Reasoning Agent」がまず思考連鎖を使用して問題を解決します。各ステップで、ツールを呼び出して情報を収集し、結果を検証し、最終的な回答に到達するまで反復します。Reasoning Agentが最終的な回答を得ると、その結果を元のAgentに渡して検証し、応答を提供します。

### 例

```python reasoning_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning=True,
    markdown=True,
)
reasoning_agent.print_response(
    "トロッコ問題の解決策を示してください。複数の倫理的フレームワークを評価し、解決策のASCII図を含めてください。",
    stream=True,
    show_full_reasoning=True,
)
```

Reasoning Agentsの詳細については、[Reasoning Agents Guide](/reasoning/reasoning-agents)をお読みください。
