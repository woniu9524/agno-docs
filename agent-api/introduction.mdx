Simple Agent API へようこそ。これは、エージェントを API として提供するための、堅牢で本番稼働可能なアプリケーションです。以下が含まれます。

- API リクエストを処理するための FastAPI サーバー。
- エージェントのセッション、知識、メモリを保存するための PostgreSQL データベース。
- 開始点として使用できる、事前に構築されたエージェントのセット。
