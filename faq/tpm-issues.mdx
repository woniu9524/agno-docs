プロプライエタリモデル（OpenAIモデルなど）でレート制限に直面した場合、`exponential_backoff=True` を設定し、`delay_between_retries` を秒単位の値（デフォルトは1秒）に変更するオプションを提供しています。

例：
```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="You are an enthusiastic news reporter with a flair for storytelling!",
    markdown=True,
    exponential_backoff=True,
    delay_between_retries=2
)
agent.print_response("Tell me about a breaking news story from New York.", stream=True)
```

レート制限に関する特定の情報については、[models documentation](../models/) を参照してください。

OpenAIの場合、ティアベースのレート制限があります。[docs](https://platform.openai.com/docs/guides/rate-limits/usage-tiers) で詳細を確認してください。
