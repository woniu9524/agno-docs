Agno Playground で接続の問題が発生している場合、特に **ローカルエンドポイント** に接続しようとしている場合は、このガイドがお役に立ちます。

## ブラウザの互換性

一部のブラウザには、混合コンテンツセキュリティの問題により、localhost ドメインへの接続をブロックするセキュリティ制限があります。各ブラウザについて知っておくべきことは次のとおりです。

### 推奨ブラウザ
- **Chrome & Edge**: これらのブラウザは、デフォルトでローカル接続とうまく連携し、推奨される選択肢です。
- **Firefox**: 一般的にローカル接続とうまく連携します。

### 問題が確認されているブラウザ
- **Safari**: 厳格なセキュリティポリシーにより、ローカル接続をブロックする場合があります。
- **Brave**: シールド機能により、デフォルトでローカル接続をブロックします。

## ソリューション

### Brave ユーザー向け
Brave ブラウザを使用している場合は、次の手順を試してください。
1. アドレスバーの Brave シールドアイコンをクリックします。
2. 現在のサイトのシールドをオフにします。
3. エンドポイントを更新して、再度接続を試みます。

### その他のブラウザ向け
Safari を使用している場合や、その他のブラウザで問題が発生している場合は、次のいずれかのソリューションを使用できます。

#### 1. Chrome または Edge を使用する
最も簡単な解決策は、ローカル接続のサポートが向上している Chrome または Edge ブラウザを使用することです。

#### 2. トンネリングサービスを使用する
トンネリングサービスを使用して、ローカルエンドポイントをインターネットに公開できます。

##### ngrok を使用する
1. [ngrok.com](https://ngrok.com) から ngrok をインストールします。
2. ローカルサーバーを実行します。
3. ngrok でトンネルを作成します。
```bash
ngrok http <your-local-port>
```
4. 提供された ngrok URL をプレイグラウンドで使用します。

##### Cloudflare Tunnel を使用する
1. [Cloudflare のウェブサイト](https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/install-and-setup/installation/) から Cloudflare Tunnel (cloudflared) をインストールします。
2. Cloudflare で認証します。
3. トンネルを作成します。
```bash
cloudflared tunnel --url http://localhost:<your-local-port>
```
4. 提供された Cloudflare URL をプレイグラウンドで使用します。
