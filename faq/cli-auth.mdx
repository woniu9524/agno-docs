`ag auth` を実行した際に `CLI authentication failed` というエラーが表示されるか、CLI が以下の状態から進まない場合：

```
Waiting for a response from browser...
```

これは、CLI が [app.agno.com](https://app.agno.com) 上の Agno アカウントと認証できなかったことを意味します。

この問題を最も迅速に解決するには、`AGNO_API_KEY` 環境変数をエクスポートします。以下のコマンドを実行して行うことができます。

```bash
export AGNO_API_KEY=<your_api_key>
```

API キーは、[app.agno.com](https://app.agno.com/settings) のサイドバーにある `API Key` から見つけることができます。

<img src="/images/cli-faq.png" alt="agno-api-key" width={600}/>

CLI 認証が失敗する理由：

- Safari や Brave のような一部のブラウザは、localhost ドメインへの接続をブロックします。Chrome のようなブラウザは `ag setup` とうまく連携します。
