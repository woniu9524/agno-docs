从 Agno 1.4.0 版本开始，**Memory V2** 已成为 Agno Agent 的默认内存。它取代了之前已弃用但仍可用的 `AgentMemory` 和 `TeamMemory` 类。

Memory V2 是一个更强大、更灵活的内存系统，允许您管理消息历史记录、会话摘要和长期用户记忆。

## 如何继续使用 AgentMemory (Memory V1)

如果您想继续使用 `AgentMemory` 并避免重大更改，可以通过更新导入来实现。默认情况下，Agent 现在使用 `Memory` 类：

```python
from agno.memory.v2 import Memory
```

要改用旧版的 AgentMemory 类，请按以下方式导入：

```python
from agno.memory import AgentMemory

agent = Agent(
    memory=AgentMemory()
)
```

## Memory V2 的主要变更

- **访问消息：**

  - **之前：**
    ```python
    agent.memory.messages
    ```
  - **现在：**
    ```python
    [run.messages for run in agent.memory.runs]
    # 或者
    agent.get_messages_for_session()
    ```

- **用户记忆：**

  - **之前：**

    ```python
    from agno.memory import AgentMemory

    memory = AgentMemory(create_user_memories=True)
    agent = Agent(memory=memory)
    ```

  - **现在：**

    ```python
    from agno.memory.v2 import Memory

    memory = Memory()
    agent = Agent(create_user_memories=True, memory=memory) 或者 team = Team(create_user_memories=True, memory=memory)
    ```

- **会话摘要：**

  - **之前：**

    ```python
    from agno.memory import AgentMemory

    memory = AgentMemory(create_session_summary=True)
    agent = Agent(memory=memory)
    ```

  - **现在：**

    ```python
    from agno.memory.v2 import Memory

    memory = Memory()
    agent = Agent(enable_session_summaries=True, memory=memory) 或者 team = Team(enable_session_summaries=True, memory=memory)
