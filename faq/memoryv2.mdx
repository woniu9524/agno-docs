Agno バージョン 1.4.0 以降、**Memory V2** が Agno Agent のデフォルトメモリとなりました。これにより、以前の `AgentMemory` および `TeamMemory` クラスは非推奨となりましたが、引き続き使用可能です。

Memory V2 は、メッセージ履歴、セッション要約、長期的なユーザーメモリを管理できる、より強力で柔軟なメモリシステムです。

## AgentMemory (Memory V1) の使用を継続する方法

`AgentMemory` を使用し続け、破壊的な変更を回避したい場合は、インポートを更新することで可能です。デフォルトでは、Agent は `Memory` クラスを使用します。

```python
from agno.memory.v2 import Memory
```

代わりにレガシーな `AgentMemory` クラスを使用するには、次のようにインポートします。

```python
from agno.memory import AgentMemory

agent = Agent(
    memory=AgentMemory()
)
```

## Memory V2 の主な変更点

- **メッセージへのアクセス:**

  - **以前:**
    ```python
    agent.memory.messages
    ```
  - **現在:**
    ```python
    [run.messages for run in agent.memory.runs]
    # または
    agent.get_messages_for_session()
    ```

- **ユーザーメモリ:**

  - **以前:**

    ```python
    from agno.memory import AgentMemory

    memory = AgentMemory(create_user_memories=True)
    agent = Agent(memory=memory)
    ```

  - **現在:**

    ```python
    from agno.memory.v2 import Memory

    memory = Memory()
    agent = Agent(create_user_memories=True, memory=memory) または team = Team(create_user_memories=True, memory=memory)
    ```

- **セッション要約:**

  - **以前:**

    ```python
    from agno.memory import AgentMemory

    memory = AgentMemory(create_session_summary=True)
    agent = Agent(memory=memory)
    ```

  - **現在:**

    ```python
    from agno.memory.v2 import Memory

    memory = Memory()
    agent = Agent(enable_session_summaries=True, memory=memory) または team = Team(enable_session_summaries=True, memory=memory)
