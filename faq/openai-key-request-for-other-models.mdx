明示的にOpenAIを設定していないにもかかわらずOpenAI APIキーが要求される場合、それはAgnoがいくつかの場所でデフォルトでOpenAIモデルを使用しているためです。これには以下が含まれます：

- `Agent`で指定されていない場合のデフォルトモデル
- 指定されていない限り、VectorDBでのデフォルトの埋め込みモデルは`OpenAIEmbedder`です。

## クイックフィックス：別のモデルを設定する

エージェントのモデルを明示的に指定することをお勧めします。指定しない場合、デフォルトで`OpenAIChat`になります。

例えば、OpenAIの代わりにGoogleのGeminiを使用するには、次のようにします：

```python
from agno.agent import Agent, RunResponse
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(id="gemini-1.5-flash"),
    markdown=True,
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

異なるモデルプロバイダーの設定に関する詳細は、[モデルドキュメント](../models/)をご覧ください。

## クイックフィックス：別の埋め込みモデルを設定する

埋め込みモデルについても同様です。`OpenAIEmbedder`の代わりに別の埋め込みモデルを使用したい場合は、明示的に設定してください。

例えば、GoogleのGeminiを埋め込みモデルとして使用するには、`GeminiEmbedder`を使用します：

```python
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.google import GeminiEmbedder

# 文章をデータベースに埋め込みます
embeddings = GeminiEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力します
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 知識ベースで埋め込みモデルを使用します
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="gemini_embeddings",
        embedder=GeminiEmbedder(),
    ),
    num_documents=2,
)
```

異なるモデルプロバイダーの設定に関する詳細は、[埋め込みドキュメント](../embedder/)をご覧ください。
