如果你想检查你的 `pgvector` 容器以探索你的存储或知识库，你可以使用 TablePlus。请遵循以下步骤：

## 步骤 1：启动你的 `pgvector` 容器

运行以下命令在本地启动一个 `pgvector` 容器：

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

- `POSTGRES_DB=ai` 设置默认数据库名称。
- `POSTGRES_USER=ai` 和 `POSTGRES_PASSWORD=ai` 定义数据库凭据。
- 该容器暴露端口 `5432`（映射到你本地机器的 `5532`）。

## 步骤 2：配置 TablePlus

1. **打开 TablePlus**：启动 TablePlus 应用程序。
2. **创建新连接**：点击 `+` 图标添加一个新连接。
3. **选择 `PostgreSQL`**：选择 PostgreSQL 作为数据库类型。

填写以下连接详细信息：

- **Host**：`localhost`
- **Port**：`5532`
- **Database**：`ai`
- **User**：`ai`
- **Password**：`ai`
