如果您想检查 `pgvector` 容器以探索您的存储或知识库，您可以使用 TablePlus。请按照以下步骤操作：

## 步骤 1：启动您的 `pgvector` 容器

运行以下命令在本地启动一个 `pgvector` 容器：

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

- `POSTGRES_DB=ai` 设置默认数据库名称。
- `POSTGRES_USER=ai` 和 `POSTGRES_PASSWORD=ai` 定义了数据库的凭据。
- 该容器公开端口 `5432`（在您的本地计算机上映射到 `5532`）。

## 步骤 2：配置 TablePlus

1. **打开 TablePlus**: 启动 TablePlus 应用程序。
2. **创建新连接**: 点击 `+` 图标添加新连接。
3. **选择 `PostgreSQL`**: 为数据库类型选择 PostgreSQL。

填写以下连接详细信息：

- **Host**: `localhost`
- **Port**: `5532`
- **Database**: `ai`
- **User**: `ai`
- **Password**: `ai`
