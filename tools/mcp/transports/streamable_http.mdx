新しい[Streamable HTTPトランスポート](https://modelcontextprotocol.io/specification/draft/basic/transports#streamable-http)は、プロトコルバージョン2024-11-05のHTTP+SSEトランスポートを置き換えるものです。

このトランスポートにより、MCPサーバーは複数のクライアント接続を処理できるようになり、サーバーからクライアントへのストリーミングにSSEを使用することもできます。

使用するには、MCPサーバーのURLを渡して`MCPTools`を初期化し、トランスポートを`streamable-http`に設定します。


```python
from agno.agent import Agent
    from agno.models.openai import OpenAIChat
    from agno.tools.mcp import MCPTools

    server_url = "http://localhost:8000/mcp"

    async with MCPTools(url=server_url, transport="streamable-http") as mcp_tools:
        agent = Agent(model=OpenAIChat(id="gpt-4o"), tools=[mcp_tools])
        await agent.aprint_response("What is the license for this project?", stream=True)
```


`server_params`引数を使用してMCP接続を定義することもできます。これにより、各リクエストでMCPサーバーに送信するヘッダーやタイムアウト値を指定できます。

```python
from agno.tools.mcp import MCPTools, StreamableHTTPClientParams

server_params = StreamableHTTPClientParams(
    url=...,
    headers=...,
    timeout=...,
    sse_read_timeout=...,
    terminate_on_close=...,

)

async with MCPTools(server_params=server_params) as mcp_tools:
    ...
```


## 完全な例

簡単なローカルサーバーをセットアップし、Streamable HTTPトランスポートを使用して接続してみましょう。


<Steps>
    <Step title="サーバーのセットアップ">
        ```python streamable_http_server.py
        from mcp.server.fastmcp import FastMCP

        mcp = FastMCP("calendar_assistant")


        @mcp.tool()
        def get_events(day: str) -> str:
            return f"There are no events scheduled for {day}."


        @mcp.tool()
        def get_birthdays_this_week() -> str:
            return "It is your mom's birthday tomorrow"


        if __name__ == "__main__":
            mcp.run(transport="streamable-http")
        ```
    </Step>

    <Step title="クライアントのセットアップ">
        ```python streamable_http_client.py
        import asyncio

        from agno.agent import Agent
        from agno.models.openai import OpenAIChat
        from agno.tools.mcp import MCPTools, MultiMCPTools

        # これは使用したいMCPサーバーのURLです。
        server_url = "http://localhost:8000/mcp"


        async def run_agent(message: str) -> None:
            async with MCPTools(transport="streamable-http", url=server_url) as mcp_tools:
                agent = Agent(
                    model=OpenAIChat(id="gpt-4o"),
                    tools=[mcp_tools],
                    markdown=True,
                )
                await agent.aprint_response(message=message, stream=True, markdown=True)


        # MultiMCPToolsを使用すると、異なるトランスポートを使用している場合でも、複数のMCPサーバーに同時に接続できます。
        # この例では、例のサーバー（Streamable HTTPトランスポート）と別のサーバー（stdioトランスポート）の両方に接続します。
        async def run_agent_with_multimcp(message: str) -> None:
            async with MultiMCPTools(
                commands=["npx -y @openbnb/mcp-server-airbnb --ignore-robots-txt"],
                urls=[server_url],
                urls_transports=["streamable-http"],
            ) as mcp_tools:
                agent = Agent(
                    model=OpenAIChat(id="gpt-4o"),
                    tools=[mcp_tools],
                    markdown=True,
                )
                await agent.aprint_response(message=message, stream=True, markdown=True)


        if __name__ == "__main__":
            asyncio.run(run_agent("Do I have any birthdays this week?"))
            asyncio.run(
                run_agent_with_multimcp(
                    "Can you check when is my mom's birthday, and if there are any AirBnb listings in SF for two people for that day?"
                )
            )
        ```
    </Step>

    <Step title="サーバーの実行">
        ```bash
        python streamable_http_server.py
        ```
    </Step>

    <Step title="クライアントの実行">
        ```bash
        python sse_client.py
        ```
    </Step>
</Steps>
