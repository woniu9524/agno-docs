Agno の MCP 統合は、[SSE トランスポート](https://modelcontextprotocol.io/docs/concepts/transports#server-sent-events-sse)をサポートしています。このトランスポートはサーバーからクライアントへのストリーミングを可能にし、制限されたネットワークでの作業時には [stdio](https://modelcontextprotocol.io/docs/concepts/transports#standard-input%2Foutput-stdio) よりも有用であることが証明される場合があります。

使用するには、MCP サーバーの URL を渡して `MCPTools` を初期化し、トランスポートを `sse` に設定します。

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.mcp import MCPTools

server_url = "http://localhost:8000/sse"

async with MCPTools(url=server_url, transport="sse") as mcp_tools:
    agent = Agent(model=OpenAIChat(id="gpt-4o"), tools=[mcp_tools])
    await agent.aprint_response("What is the license for this project?", stream=True)
```

`server_params` 引数を使用して MCP 接続を定義することもできます。これにより、MCP サーバーに送信するヘッダーとタイムアウト値を指定できます。

```python
from agno.tools.mcp import MCPTools, SSEClientParams

server_params = SSEClientParams(
    url=...,
    headers=...,
    timeout=...,
    sse_read_timeout=...,
)

async with MCPTools(server_params=server_params) as mcp_tools:
    ...
```

## 完全な例

簡単なローカルサーバーをセットアップし、SSE トランスポートを使用してそれに接続しましょう。

<Steps>
    <Step title="サーバーのセットアップ">
        ```python sse_server.py
        from mcp.server.fastmcp import FastMCP

        mcp = FastMCP("calendar_assistant")


        @mcp.tool()
        def get_events(day: str) -> str:
            return f"There are no events scheduled for {day}."


        @mcp.tool()
        def get_birthdays_this_week() -> str:
            return "It is your mom's birthday tomorrow"


        if __name__ == "__main__":
            mcp.run(transport="sse")
        ```
    </Step>

    <Step title="クライアントのセットアップ">
        ```python sse_client.py
        import asyncio

        from agno.agent import Agent
        from agno.models.openai import OpenAIChat
        from agno.tools.mcp import MCPTools, MultiMCPTools

        # これは、使用したい MCP サーバーの URL です。
        server_url = "http://localhost:8000/sse"


        async def run_agent(message: str) -> None:
            async with MCPTools(transport="sse", url=server_url) as mcp_tools:
                agent = Agent(
                    model=OpenAIChat(id="gpt-4o"),
                    tools=[mcp_tools],
                    markdown=True,
                )
                await agent.aprint_response(message=message, stream=True, markdown=True)


        # MultiMCPTools を使用すると、異なるトランスポートを使用している場合でも、複数の MCP サーバーに同時に接続できます。
        # この例では、例のサーバー (SSE トランスポート) と別のサーバー (stdio トランスポート) の両方に接続しています。
        async def run_agent_with_multimcp(message: str) -> None:
            async with MultiMCPTools(
                commands=["npx -y @openbnb/mcp-server-airbnb --ignore-robots-txt"],
                urls=[server_url],
            ) as mcp_tools:
                agent = Agent(
                    model=OpenAIChat(id="gpt-4o"),
                    tools=[mcp_tools],
                    markdown=True,
                )
                await agent.aprint_response(message=message, stream=True, markdown=True)


        if __name__ == "__main__":
            asyncio.run(run_agent("Do I have any birthdays this week?"))
            asyncio.run(
                run_agent_with_multimcp(
                    "Can you check when is my mom's birthday, and if there are any AirBnb listings in SF for two people for that day?"
                )
            )
        ```
    </Step>

    <Step title="サーバーの実行">
        ```bash
        python sse_server.py
        ```
    </Step>

    <Step title="クライアントの実行">
        ```bash
        python sse_client.py
        ```
    </Step>
</Steps>
