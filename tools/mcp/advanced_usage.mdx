Agno の MCP 統合は、複数のサーバーへの接続の処理、サーバーパラメータの指定、および独自の MCP サーバーの使用もサポートしています。

## 複数の MCP サーバーへの接続

`MultiMCPTools` クラスを使用することで、単一のエージェントで複数の MCP サーバーを使用できます。

```python multiple_mcp_servers.py
import asyncio
import os

from agno.agent import Agent
from agno.tools.mcp import MultiMCPTools


async def run_agent(message: str) -> None:
    """指定されたメッセージで Airbnb および Google Maps エージェントを実行します。"""

    env = {
        **os.environ,
        "GOOGLE_MAPS_API_KEY": os.getenv("GOOGLE_MAPS_API_KEY"),
    }

    async with MultiMCPTools(
        [
            "npx -y @openbnb/mcp-server-airbnb --ignore-robots-txt",
            "npx -y @modelcontextprotocol/server-google-maps",
        ],
        env=env,
    ) as mcp_tools:
        agent = Agent(
            tools=[mcp_tools],
            markdown=True,
            show_tool_calls=True,
        )

        await agent.aprint_response(message, stream=True)


# 使用例
if __name__ == "__main__":
    # Pull request の例
    asyncio.run(
        run_agent(
            "2025年8月1日から4日まで、ケープタウンで2名で3泊利用可能なリスティングは何件ありますか？"
        )
    )
```

### サーバーパラメータの理解

`MCPTools` または `MultiMCPTools` を構成する推奨方法は、`command` または `url` パラメータを使用することです。

または、`MCPTools` で `server_params` パラメータを使用して、MCP サーバーへの接続をより詳細に構成することもできます。

**stdio** トランスポートを使用する場合、`server_params` パラメータは `StdioServerParameters` のインスタンスである必要があります。これには以下のキーが含まれます。
- `command`: MCP サーバーを実行するコマンド。
    - npm を介してインストールできる mcp サーバーには `npx` を使用します（Windows で実行する場合は `node` を使用します）。
    - uvx を介してインストールできる mcp サーバーには `uvx` を使用します。
- `args`: MCP サーバーに渡す引数。
- `env`: MCP サーバーに渡すオプションの環境変数。現在のすべての環境変数を `env` 辞書に含めることを忘れないでください。`env` が提供されない場合、現在の環境変数が使用されます。
例：
```python
{
    **os.environ,
    "GOOGLE_MAPS_API_KEY": os.getenv("GOOGLE_MAPS_API_KEY"),
}
```

**SSE** トランスポートを使用する場合、`server_params` パラメータは `SSEClientParams` のインスタンスである必要があります。これには以下のフィールドが含まれます。
- `url`: MCP サーバーの URL。
- `headers`: MCP サーバーに渡すヘッダー（オプション）。
- `timeout`: MCP サーバーへの接続タイムアウト（オプション）。
- `sse_read_timeout`: SSE 接続自体のタイムアウト（オプション）。

**Streamable HTTP** トランスポートを使用する場合、`server_params` パラメータは `StreamableHTTPClientParams` のインスタンスである必要があります。これには以下のフィールドが含まれます。
- `url`: MCP サーバーの URL。
- `headers`: MCP サーバーに渡すヘッダー（オプション）。
- `timeout`: MCP サーバーへの接続タイムアウト（オプション）。
- `sse_read_timeout`: クライアントが新しいイベントを待つ時間（秒単位）（切断されるまで）。他のすべての HTTP 操作は `timeout` によって制御されます（オプション）。
- `terminate_on_close`: クライアントが閉じられたときに接続を終了するかどうか（オプション）。

## より柔軟に

MCP サーバー自体を作成し、それを `MCPTools` コンストラクタに渡すこともできます。

```python filesystem_agent.py
import asyncio
from pathlib import Path
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.mcp import MCPTools
from mcp import ClientSession, StdioServerParameters
from mcp.client.stdio import stdio_client


async def create_filesystem_agent(session):
    """MCP ツールでファイルシステムエージェントを作成および構成します。"""
    # MCP ツールキットの初期化
    mcp_tools = MCPTools(session=session)
    await mcp_tools.initialize()

    # MCP ツールキットでエージェントを作成します
    return Agent(
        model=OpenAIChat(id="gpt-4o"),
        tools=[mcp_tools],
        instructions=dedent("""\
            あなたはファイルシステムアシスタントです。ユーザーがファイルやディレクトリを探索するのを手伝ってください。

            - ファイルシステムをナビゲートして質問に答える
            - list_allowed_directories ツールを使用してアクセスできるディレクトリを見つける
            - 検査するファイルに関する明確なコンテキストを提供する
            - 見出しを使用して応答を整理する
            - 簡潔にし、関連情報に焦点を当てる\
        """),
        markdown=True,
        show_tool_calls=True,
    )


async def run_agent(message: str) -> None:
    """指定されたメッセージでファイルシステムエージェントを実行します。"""

    # MCP サーバーの初期化
    server_params = StdioServerParameters(
        command="npx",
        args=[
            "-y",
            "@modelcontextprotocol/server-filesystem",
            str(Path(__file__).parent.parent.parent.parent),  # 探索したいプロジェクトのルートに設定します
        ],
    )

    # MCP サーバーに接続するためのクライアントセッションを作成します
    async with stdio_client(server_params) as (read, write):
        async with ClientSession(read, write) as session:
            agent = await create_filesystem_agent(session)

            # エージェントを実行します
            await agent.aprint_response(message, stream=True)


# 使用例
if __name__ == "__main__":
    # 基本的な例 - プロジェクトライセンスの探索
    asyncio.run(run_agent("このプロジェクトのライセンスは何ですか？"))
