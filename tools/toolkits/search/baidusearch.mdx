**BaiduSearch** は、Agent が百度検索エンジンを使用して Web 上の情報を検索できるようにします。

## 前提条件

次の例では、`baidusearch` ライブラリが必要です。BaiduSearch をインストールするには、次のコマンドを実行します。

```shell
pip install -U baidusearch
```

## 例

```python cookbook/tools/baidusearch_tools.py
from agno.agent import Agent
from agno.tools.baidusearch import BaiduSearchTools

agent = Agent(
    tools=[BaiduSearchTools()],
    description="百度を使用して最も関連性の高い情報を見つけるのに役立つ検索エージェントです。",
    instructions=[
        "ユーザーからトピックが与えられたら、そのトピックに関する最も関連性の高い検索結果を 3 つ返信してください。",
        "5 件の結果を検索し、上位 3 件の一意なアイテムを選択してください。",
        "英語と中国語の両方で検索してください。",
    ],
    show_tool_calls=True,
)

agent.print_response("AI の最新の進歩は何ですか？", markdown=True)
```

## Toolkit のパラメータ

| パラメータ           | タイプ  | デフォルト | 説明                                                                                             |
| ------------------- | ----- | ------- | ---------------------------------------------------------------------------------------------------- |
| `fixed_max_results` | `int` | -       | 返される最大結果数を固定します。デフォルトは提供されていないため、使用する場合は指定する必要があります。 |
| `fixed_language`    | `str` | -       | 結果の固定言語を設定します。                                                                         |
| `headers`           | `Any` | -       | 検索リクエストで使用されるヘッダー。                                                                 |
| `proxy`             | `str` | -       | HTTP リクエストで使用される単一のプロキシアドレスを文字列として指定します。                               |
| `timeout`           | `int` | `10`    | HTTP リクエストのタイムアウトを秒単位で設定します。                                                      |

## Toolkit の関数

| 関数       | 説明                                    |
| ---------- | --------------------------------------- |
| `baidu_search` | この関数を使用して百度でクエリを検索します。 |
