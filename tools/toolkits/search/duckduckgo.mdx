**DuckDuckGo** を使用すると、エージェントがウェブで情報を検索できます。

## 前提条件

次の例では、`duckduckgo-search` ライブラリが必要です。DuckDuckGo をインストールするには、次のコマンドを実行します。

```shell
pip install -U duckduckgo-search
```

## 例

```python cookbook/tools/duckduckgo.py
from agno.agent import Agent
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(tools=[DuckDuckGoTools()], show_tool_calls=True)
agent.print_response("Whats happening in France?", markdown=True)
```

## Toolkit パラメータ

| パラメータ           | タイプ   | デフォルト | 説明                                                                                             |
| ------------------- | ------ | ------- | ---------------------------------------------------------------------------------------------------- |
| `search`            | `bool` | `True`  | DuckDuckGo でクエリを検索するために `duckduckgo_search` 関数を使用できるようにします。                |
| `news`              | `bool` | `True`  | DuckDuckGo を介して最新ニュースを取得するために `duckduckgo_news` 関数を使用できるようにします。           |
| `fixed_max_results` | `int`  | -       | 返される結果の最大数を固定します。デフォルトはないため、使用する場合は指定する必要があります。 |
| `headers`           | `Any`  | -       | HTTP リクエストと共に送信される任意のヘッダー値を受け入れます。                                     |
| `proxy`             | `str`  | -       | HTTP リクエストに使用される単一のプロキシアドレスを文字列として指定します。                       |
| `proxies`           | `Any`  | -       | HTTP リクエストに使用されるプロキシの辞書を受け入れます。                                         |
| `timeout`           | `int`  | `10`    | HTTP リクエストのタイムアウトを秒単位で設定します。                                                |

## Toolkit 関数

| 関数            | 説明                                          |
| ------------------- | --------------------------------------------------------- |
| `duckduckgo_search` | この関数を使用して DuckDuckGo でクエリを検索します。       |
| `duckduckgo_news`   | この関数を使用して DuckDuckGo から最新ニュースを取得します。 |
