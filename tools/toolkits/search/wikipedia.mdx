**WikipediaTools** は、エージェントがウェブサイトであるWikipediaを検索し、そのコンテンツをナレッジベースに追加できるようにします。

## 前提条件

以下の例では、`wikipedia` ライブラリが必要です。

```shell
pip install -U wikipedia
```

## 例

以下のエージェントは、Wikipediaで「ai」を検索し、その応答を表示します。

```python cookbook/tools/wikipedia_tools.py
from agno.agent import Agent
from agno.tools.wikipedia import WikipediaTools

agent = Agent(tools=[WikipediaTools()], show_tool_calls=True)
agent.print_response("Search wikipedia for 'ai'")
```

## Toolkit パラメータ

| 名前             | タイプ                     | デフォルト | 説明                                                                                             |
| ---------------- | ------------------------ | ------- | ------------------------------------------------------------------------------------------------ |
| `knowledge_base` | `WikipediaKnowledgeBase` | -       | Wikipediaに関連付けられたナレッジベースで、Wikipediaのコンテンツにリンクされたさまざまなデータとリソースが含まれています。 |

## Toolkit 関数

| 関数名                                       | 説明                                                                                             |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| `search_wikipedia_and_update_knowledge_base` | この関数は、Wikipediaでトピックを検索し、結果をナレッジベースに追加して返します。 |
| `search_wikipedia`                           | クエリに対してWikipediaを検索します。                                                              |
