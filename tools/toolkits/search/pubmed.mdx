**PubmedTools** を使用すると、エージェントが Pubmed で記事を検索できるようになります。

## 例

以下のエージェントは、"ulcerative colitis" に関連する記事を Pubmed で検索します。

```python cookbook/tools/pubmed.py
from agno.agent import Agent
from agno.tools.pubmed import PubmedTools

agent = Agent(tools=[PubmedTools()], show_tool_calls=True)
agent.print_response("Tell me about ulcerative colitis.")
```

## Toolkit パラメータ

| パラメータ     | タイプ  | デフォルト                    | 説明                                                                 |
| ------------- | ----- | -------------------------- | -------------------------------------------------------------------- |
| `email`       | `str` | `"your_email@example.com"` | 使用するメールアドレスを指定します。                                         |
| `max_results` | `int` | `None`                     | 返される結果の最大数を指定するためのオプションのパラメータです。 |

## Toolkit 関数

| 関数        | 説明                                                                                                                                                                                                                                                                                                                      |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `search_pubmed` | 指定されたクエリに基づいて PubMed で記事を検索します。パラメータには、検索用語の `query` と、返される最大結果数の `max_results` (デフォルトは 10) が含まれます。発行日、タイトル、概要を含む検索結果を含む JSON 文字列を返します。 |
