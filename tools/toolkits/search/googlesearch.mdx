**GoogleSearch** は、エージェントがWebクロールおよびスクレイピングタスクを実行できるようにします。

## 前提条件

以下の例では、`googlesearch` および `pycountry` ライブラリが必要です。

```shell
pip install -U googlesearch-python pycountry
```

## 例

以下のエージェントは、Googleで「Mistral AI」に関する最新ニュースを検索します。

```python cookbook/tools/googlesearch_tools.py
from agno.agent import Agent
from agno.tools.googlesearch import GoogleSearchTools

agent = Agent(
    tools=[GoogleSearchTools()],
    description="あなたは、ユーザーが最新ニュースを見つけるのを手助けするニュースエージェントです。",
    instructions=[
        "ユーザーから提供されたトピックについて、そのトピックに関する最新ニュースを4件返信してください。",
        "ニュース記事を10件検索し、上位4件の一意な記事を選択してください。",
        "英語とフランス語で検索してください。",
    ],
    show_tool_calls=True,
    debug_mode=True,
)

agent.print_response("Mistral AI", markdown=True)
```

## Toolkit Params

| パラメータ           | タイプ  | デフォルト | 説明                                         |
| ------------------- | ----- | ------- | --------------------------------------------------- |
| `fixed_max_results` | `int` | `None`  | 返される結果の最大数を固定するオプション。 |
| `fixed_language`    | `str` | `None`  | リクエストの固定言語（オプション）。           |
| `headers`           | `Any` | `None`  | リクエストに含めるヘッダー（オプション）。        |
| `proxy`             | `str` | `None`  | リクエストに使用するプロキシ（オプション）。      |
| `timeout`           | `int` | `None`  | リクエストのタイムアウト（秒単位、オプション）。  |

## Toolkit Functions

| 関数        | 説明                                                                                                                                                                                                                                                                            |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `google_search` | 指定されたクエリでGoogleを検索します。パラメータには、検索用語の`query`、最大結果数の`max_results`（デフォルトは5）、検索結果の言語の`language`（デフォルトは「en」）が含まれます。検索結果をJSON形式の文字列で返します。 |
