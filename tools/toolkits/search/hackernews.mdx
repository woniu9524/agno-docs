**HackerNews** 使代理能够搜索 Hacker News 网站。

## 示例

以下代理将对 Hacker News 上排名前 2 位的故事及其用户进行引人入胜的总结。

```python cookbook/tools/hackernews.py
from agno.agent import Agent
from agno.tools.hackernews import HackerNewsTools

agent = Agent(
    name="Hackernews Team",
    tools=[HackerNewsTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response(
    "Write an engaging summary of the "
    "users with the top 2 stories on hackernews. "
    "Please mention the stories as well.",
)
```

## Toolkit 参数

| 参数          | 类型   | 默认值 | 描述                    |
| ------------- | ------ | ------ | ----------------------- |
| `get_top_stories` | `bool` | `True` | 启用获取热门故事。 |
| `get_user_details` | `bool` | `True` | 启用获取用户详情。 |

## Toolkit 函数

| 函数                     | 描述                                                                                                                                                               |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `get_top_hackernews_stories` | 从 Hacker News 获取热门故事。参数包括 `num_stories`，用于指定要返回的故事数量（默认为 10）。以 JSON 格式返回热门故事。 |
| `get_user_details`           | 通过用户名获取 Hacker News 用户详情。参数包括 `username`，用于指定用户。以 JSON 格式返回用户详情。                       |
