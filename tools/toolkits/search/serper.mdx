**SerperApiTools** は、エージェントがGoogleでクエリを検索できるようにします。

## 前提条件

以下の例では、[SerperApi](https://serper.dev/) からAPIキーが必要です。

```shell
export SERPER_API_KEY=***
```

## 例

以下のエージェントは、Googleで「Whats happening in the USA」というクエリを検索し、結果を共有します。

```python cookbook/tools/serper_tools.py
from agno.agent import Agent
from agno.tools.serper import SerperTools

agent = Agent(tools=[SerperTools(location="us")], show_tool_calls=True)
agent.print_response("Whats happening in the USA?", markdown=True)
```

## Toolkit パラメータ

| パラメータ  | タイプ  | デフォルト | 説明                                 |
| ---------- | ----- | ------- | ------------------------------------ |
| `api_key`  | `str` | -       | 認証目的のAPIキー。                  |
| `location` | `str` | `"us"`  | 検索元の場所。                       |

## Toolkit 関数

| 関数        | 説明                               |
| --------------- | ---------------------------------- |
| `search_google` | この関数はGoogleでクエリを検索します。 |
