**SerpApiTools** を使用すると、エージェントがクエリをGoogleおよびYouTubeで検索できるようになります。

## 前提条件

次の例では、`google-search-results` ライブラリと [SerpApi](https://serpapi.com/) からのAPIキーが必要です。

```shell
pip install -U google-search-results
```

```shell
export SERP_API_KEY=***
```

## 例

次のエージェントは、クエリ「Whats happening in the USA」をGoogleで検索し、結果を共有します。

```python cookbook/tools/serpapi_tools.py
from agno.agent import Agent
from agno.tools.serpapi import SerpApiTools

agent = Agent(tools=[SerpApiTools()])
agent.print_response("Whats happening in the USA?", markdown=True)
```

## Toolkit Params

| パラメータ      | タイプ   | デフォルト | 説明                                       |
| --------------- | ------ | ------- | ------------------------------------------ |
| `api_key`       | `str`  | -       | 認証用のAPIキー。                          |
| `search_youtube` | `bool` | `False` | YouTubeでコンテンツを検索する機能を有効にします。 |

## Toolkit Functions

| 関数           | 説明                               |
| -------------- | ---------------------------------- |
| `search_google` | この関数はGoogleでクエリを検索します。 |
| `search_youtube` | クエリをYouTubeで検索します。        |
