**SerpApiTools** 使 Agent 能够搜索 Google 和 YouTube 的查询。

## 先决条件

以下示例需要 `google-search-results` 库和来自 [SerpApi](https://serpapi.com/) 的 API 密钥。

```shell
pip install -U google-search-results
```

```shell
export SERP_API_KEY=***
```

## 示例

以下 Agent 将在 Google 上搜索查询：“Whats happening in the USA” 并分享结果。

```python cookbook/tools/serpapi_tools.py
from agno.agent import Agent
from agno.tools.serpapi import SerpApiTools

agent = Agent(tools=[SerpApiTools()])
agent.print_response("Whats happening in the USA?", markdown=True)
```

## Toolkit 参数

| 参数        | 类型   | 默认值 | 描述                                       |
| ----------- | ------ | ------- | ------------------------------------------ |
| `api_key`   | `str`  | -       | 用于身份验证的 API 密钥。                  |
| `search_youtube` | `bool` | `False` | 启用在 YouTube 上搜索内容的功能。 |

## Toolkit 函数

| 函数         | 描述                               |
| ----------- | ---------------------------------- |
| `search_google`  | 此函数在 Google 上搜索查询。     |
| `search_youtube` | 在 YouTube 上搜索查询。          |
