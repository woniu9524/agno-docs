**ArxivTools** は、Agent が Arxiv で出版物を検索できるようにします。

## 前提条件

次の例では、`arxiv` および `pypdf` ライブラリが必要です。

```shell
pip install -U arxiv pypdf
```

## 例

次の Agent は、ArXiv で「language models」を検索し、応答を表示します。

```python cookbook/tools/arxiv_tools.py
from agno.agent import Agent
from agno.tools.arxiv import ArxivTools

agent = Agent(tools=[ArxivTools()], show_tool_calls=True)
agent.print_response("Search arxiv for 'language models'", markdown=True)
```

## Toolkit パラメータ

| パラメータ           | タイプ   | デフォルト | 説明                                                               |
| ------------------- | ------ | ------- | ------------------------------------------------------------------ |
| `search_arxiv`      | `bool` | `True`  | ArXiv データベースを検索する機能を有効にします。                                     |
| `read_arxiv_papers` | `bool` | `True`  | ArXiv の論文を直接読み取ることを許可します。                                     |
| `download_dir`      | `Path` | -       | ダウンロードされたファイルが保存されるディレクトリパスを指定します。                               |

## Toolkit 関数

| 関数                                 | 説明                                                                                             |
| ---------------------------------------- | -------------------------------------------------------------------------------------------------- |
| `search_arxiv_and_update_knowledge_base` | ArXiv でトピックを検索し、結果をナレッジベースに追加して返します。                                                               |
| `search_arxiv`                           | ArXiv でクエリを検索します。                                                                         |
