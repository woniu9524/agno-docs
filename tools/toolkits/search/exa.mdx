**ExaTools** を使用すると、エージェントは Exa を使ってウェブを検索し、URL からコンテンツを取得し、類似したコンテンツを見つけ、AI による回答を得ることができます。

## 前提条件

以下の例では、`exa-py` ライブラリと、[Exa](https://exa.ai) から取得できる API キーが必要です。

```shell
pip install -U exa-py
```

```shell
export EXA_API_KEY=***
```

## 例

以下のエージェントは、Exa で AAPL ニュースを検索し、応答を表示します。

```python cookbook/tools/exa_tools.py
from agno.agent import Agent
from agno.tools.exa import ExaTools

agent = Agent(
    tools=[ExaTools(
        include_domains=["cnbc.com", "reuters.com", "bloomberg.com"],
        category="news",
        text_length_limit=1000,
    )],
    show_tool_calls=True,
)
agent.print_response("Search for AAPL news", markdown=True)
```

## ツールキット関数

| 関数 | 説明 |
| -------- | ----------- |
| `search_exa` | カテゴリフィルタリングをオプションで指定して Exa を検索します |
| `get_contents` | 特定の URL から詳細なコンテンツを取得します |
| `find_similar` | 指定された URL と類似したコンテンツを見つけます |
| `exa_answer` | Exa の検索結果を使用して、質問に対する AI による回答を取得します |

## ツールキットパラメータ

| パラメータ | タイプ | デフォルト | 説明 |
| --------- | ---- | ------- | ----------- |
| `search` | `bool` | `True` | 検索機能を有効にします |
| `get_contents` | `bool` | `True` | コンテンツ取得機能を有効にします |
| `find_similar` | `bool` | `True` | 類似コンテンツ検索機能を有効にします |
| `answer` | `bool` | `True` | AI による回答機能を有効にします |
| `text` | `bool` | `True` | 結果にテキストコンテンツを含めます |
| `text_length_limit` | `int` | `1000` | 結果ごとのテキストコンテンツの最大長 |
| `highlights` | `bool` | `True` | ハイライトされたスニペットを含めます |
| `summary` | `bool` | `False` | 結果の要約を含めます |
| `num_results` | `Optional[int]` | `None` | 結果のデフォルト数 |
| `livecrawl` | `str` | `"always"` | Livecrawl の動作 |
| `start_crawl_date` | `Optional[str]` | `None` | 指定された日付以降にクロールされた結果を含めます (YYYY-MM-DD) |
| `end_crawl_date` | `Optional[str]` | `None` | 指定された日付以前にクロールされた結果を含めます (YYYY-MM-DD) |
| `start_published_date` | `Optional[str]` | `None` | 指定された日付以降に公開された結果を含めます (YYYY-MM-DD) |
| `end_published_date` | `Optional[str]` | `None` | 指定された日付以前に公開された結果を含めます (YYYY-MM-DD) |
| `use_autoprompt` | `Optional[bool]` | `None` | autoprompt 機能を有効にします |
| `type` | `Optional[str]` | `None` | コンテンツタイプのフィルタリング (例: article, blog, video) |
| `category` | `Optional[str]` | `None` | カテゴリのフィルタリング (例: news, research paper) |
| `include_domains` | `Optional[List[str]]` | `None` | 結果をこれらのドメインに制限します |
| `exclude_domains` | `Optional[List[str]]` | `None` | これらのドメインからの結果を除外します |
| `show_results` | `bool` | `False` | デバッグのために検索結果をログに記録します |
| `model` | `Optional[str]` | `None` | 使用する検索モデル ('exa' または 'exa-pro') |

### カテゴリ

フィルタリングに使用できるカテゴリ:
- company
- research paper
- news
- pdf
- github
- tweet
- personal site
- linkedin profile
- financial report
