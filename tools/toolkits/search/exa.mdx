**ExaTools** 使 Agent 能够使用 Exa 搜索网络、从 URL 检索内容、查找相似内容以及获取由 AI 驱动的答案。

## 先决条件

以下示例需要 `exa-py` 库和一个 API 密钥，API 密钥可从 [Exa](https://exa.ai) 获取。

```shell
pip install -U exa-py
```

```shell
export EXA_API_KEY=***
```

## 示例

以下 Agent 将搜索 Exa 以获取 AAPL 新闻并打印响应。

```python cookbook/tools/exa_tools.py
from agno.agent import Agent
from agno.tools.exa import ExaTools

agent = Agent(
    tools=[ExaTools(
        include_domains=["cnbc.com", "reuters.com", "bloomberg.com"],
        category="news",
        text_length_limit=1000,
    )],
    show_tool_calls=True,
)
agent.print_response("Search for AAPL news", markdown=True)
```

## Toolkit 函数

| 函数 | 描述 |
| -------- | ----------- |
| `search_exa` | 使用可选的类别过滤搜索 Exa 查询 |
| `get_contents` | 从特定 URL 检索详细内容 |
| `find_similar` | 查找与给定 URL 相似的内容 |
| `exa_answer` | 使用 Exa 搜索结果获取问题的 AI 驱动答案 |

## Toolkit 参数

| 参数 | 类型 | 默认值 | 描述 |
| -------- | ---- | ------- | ----------- |
| `search` | `bool` | `True` | 启用搜索功能 |
| `get_contents` | `bool` | `True` | 启用内容检索 |
| `find_similar` | `bool` | `True` | 启用查找相似内容 |
| `answer` | `bool` | `True` | 启用 AI 驱动的答案 |
| `text` | `bool` | `True` | 在结果中包含文本内容 |
| `text_length_limit` | `int` | `1000` | 每个结果的最大文本内容长度 |
| `highlights` | `bool` | `True` | 包含高亮片段 |
| `summary` | `bool` | `False` | 包含结果摘要 |
| `num_results` | `Optional[int]` | `None` | 默认结果数量 |
| `livecrawl` | `str` | `"always"` | Livecrawl 行为 |
| `start_crawl_date` | `Optional[str]` | `None` | 包含在此日期之后爬取的结果 (YYYY-MM-DD) |
| `end_crawl_date` | `Optional[str]` | `None` | 包含在此日期之前爬取的结果 (YYYY-MM-DD) |
| `start_published_date` | `Optional[str]` | `None` | 包含在此日期之后发布的结果 (YYYY-MM-DD) |
| `end_published_date` | `Optional[str]` | `None` | 包含在此日期之前发布的結果 (YYYY-MM-DD) |
| `use_autoprompt` | `Optional[bool]` | `None` | 启用自动提示功能 |
| `type` | `Optional[str]` | `None` | 内容类型过滤器 (例如，article, blog, video) |
| `category` | `Optional[str]` | `None` | 类别过滤器 (例如，news, research paper) |
| `include_domains` | `Optional[List[str]]` | `None` | 将结果限制在这些域内 |
| `exclude_domains` | `Optional[List[str]]` | `None` | 从这些域中排除结果 |
| `show_results` | `bool` | `False` | 为调试记录搜索结果 |
| `model` | `Optional[str]` | `None` | 要使用的搜索模型 ('exa' 或 'exa-pro') |

### 类别

可用于过滤的类别：
- company
- research paper
- news
- pdf
- github
- tweet
- personal site
- linkedin profile
- financial report
