---
title: カスタムAPI
---

**CustomApiTools** を使用すると、エージェントはカスタマイズ可能な認証とパラメータを使用して、任意の外部APIへのHTTPリクエストを実行できます。

## 前提条件

以下の例では、`requests` ライブラリが必要です。

```shell
pip install requests
```

## 例

以下のエージェントは、CustomApiTools を使用して Dog CEO API へのAPI呼び出しを行います。

```python cookbook/tools/custom_api_tools.py
from agno.agent import Agent
from agno.tools.api import CustomApiTools

agent = Agent(
    tools=[CustomApiTools(base_url="https://dog.ceo/api")],
    markdown=True,
)

agent.print_response(
    '以下の2つの異なるエンドポイントにAPI呼び出しを行ってください: /breeds/image/random および /breeds/list/all。それぞれランダムな犬の画像と犬種のリストを取得します。両方の呼び出しでGETメソッドを使用してください。'
)
```

## Toolkitパラメータ

| パラメータ     | タイプ                | デフォルト | 説明                                         |
| ------------- | ------------------- | ------- | -------------------------------------------- |
| `base_url`    | `str`               | `None`  | API呼び出しのベースURL                               |
| `username`    | `str`               | `None`  | 基本認証のユーザー名                               |
| `password`    | `str`               | `None`  | 基本認証のパスワード                               |
| `api_key`     | `str`               | `None`  | ベアラートークン認証のAPIキー                           |
| `headers`     | `Dict[str, str]`    | `{}`    | リクエストに含めるデフォルトヘッダー                         |
| `verify_ssl`  | `bool`              | `True`  | SSL証明書を検証するかどうか                               |
| `timeout`     | `int`               | `30`    | リクエストタイムアウト（秒）                               |
| `make_request`| `bool`              | `True`  | make_request 関数を登録するかどうか                         |

## Toolkit関数

| 関数       | 説明                                                                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `make_request` | APIへのHTTPリクエストを実行します。メソッド（GET、POSTなど）、エンドポイント、およびオプションのparams、data、headers、json_dataパラメータを受け取ります。 |
