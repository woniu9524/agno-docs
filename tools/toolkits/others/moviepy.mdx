## 前提条件

`MoviePyVideoTools` を使用するには、`moviepy` とその依存関係である `ffmpeg` をインストールする必要があります。

```shell
pip install moviepy ffmpeg
```

**キャプションワークフローの重要事項:**
`create_srt` および `embed_captions` ツールには、ビデオのオーディオの文字起こしが必要です。`MoviePyVideoTools` 自体は音声認識を実行しません。通常、`OpenAITools` の `transcribe_audio` 関数のような別のツールを使用して文字起こし（通常は SRT 形式）を生成し、それをこれらのツールで使用します。

## 例

次の例は、エージェントが `MoviePyVideoTools` と `OpenAITools` を連携させて、次のことを行う完全なワークフローを示しています。
1. ビデオファイルからオーディオを抽出する
2. OpenAI の音声テキスト変換を使用してオーディオを文字起こしする
3. 文字起こしから SRT キャプションファイルを生成する
4. 単語レベルのハイライトでキャプションをビデオに埋め込む

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.moviepy_video import MoviePyVideoTools
from agno.tools.openai import OpenAITools

video_tools = MoviePyVideoTools(
    process_video=True, generate_captions=True, embed_captions=True
)

openai_tools = OpenAITools()

video_caption_agent = Agent(
    name="Video Caption Generator Agent",
    model=OpenAIChat(
        id="gpt-4o",
    ),
    tools=[video_tools, openai_tools],
    description="You are an AI agent that can generate and embed captions for videos.",
    instructions=[
        "When a user provides a video, process it to generate captions.",
        "Use the video processing tools in this sequence:",
        "1. Extract audio from the video using extract_audio",
        "2. Transcribe the audio using transcribe_audio",
        "3. Generate SRT captions using create_srt",
        "4. Embed captions into the video using embed_captions",
    ],
    markdown=True,
)


video_caption_agent.print_response(
    "Generate captions for {video with location} and embed them in the video"
)
```

## ツールキット関数

これらは `MoviePyVideoTools` によって公開される関数です。

| 関数          | 説明                                                                                                |
| ----------------- | ---------------------------------------------------------------------------------------------------------- |
| `extract_audio`   | ビデオファイルからオーディオトラックを抽出し、指定された出力パスに保存します。                        |
| `create_srt`      | 指定された文字起こし（SRT 形式が期待されます）を指定された出力パスの `.srt` ファイルに保存します。          |
| `embed_captions`  | SRT ファイルからキャプションをビデオに埋め込み、単語レベルのハイライトが含まれる新しいビデオファイルを作成します。     |

## ツールキットパラメータ

これらは `MoviePyVideoTools` コンストラクタに渡されるパラメータです。

| パラメータ           | タイプ   | デフォルト | 説明                                              |
| ------------------- | ------ | ------- | -------------------------------------------------------- |
| `process_video`     | `bool` | `True`  | `extract_audio` ツールを有効にします。                        |
| `generate_captions` | `bool` | `True`  | `create_srt` ツールを有効にします。                           |
| `embed_captions`    | `bool` | `True`  | `embed_captions` ツールを有効にします。                       |
