## 前提条件

`requests` ライブラリをインストールする必要があります。

```bash
pip install requests
```

`MODELS_LAB_API_KEY` 環境変数を設定します。

```bash
export MODELS_LAB_API_KEY=****
```

## 例

以下のエージェントは、ModelsLabs を使用してテキストプロンプトに基づいてビデオを生成します。

```python cookbook/tools/models_labs_tools.py
from agno.agent import Agent
from agno.tools.models_labs import ModelsLabsTools

# ModelsLabs ツールでエージェントを作成する
agent = Agent(tools=[ModelsLabsTools()], name="ModelsLabs Agent")

agent.print_response("Generate a video of a beautiful sunset over the ocean", markdown=True)
```

## Toolkit パラメータ

| パラメータ             | タイプ   | デフォルト                                           | 説明                                                                |
| --------------------- | ------ | ------------------------------------------------- | -------------------------------------------------------------------------- |
| `api_key`             | `str`  | `None`                                            | 認証用の ModelsLab API キー                                   |
| `wait_for_completion` | `bool` | `False`                                           | ビデオの準備ができるまで待機するかどうか                                  |
| `add_to_eta`          | `int`  | `15`                                              | ビデオの取得にかかる時間を考慮して ETA に追加する時間                               |
| `max_wait_time`       | `int`  | `60`                                              | ビデオの準備ができるまで待機する最大時間                             |
| `file_type`           | `str`  | `"mp4"`                                           | 生成するファイルのタイプ                                               |

## Toolkit 関数

| 関数         | 説明                                     |
| ---------------- | ----------------------------------------------- |
| `generate_media` | テキストプロンプトに基づいてビデオまたは GIF を生成する |
