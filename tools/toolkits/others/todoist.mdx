**TodoistTools** を使用すると、エージェントが [Todoist](https://www.todoist.com/) とやり取りできるようになります。

## 前提条件
以下の例では、`todoist-api-python` ライブラリが必要です。また、[Todoist Developer Portal](https://app.todoist.com/app/settings/integrations/developer) から取得できる Todoist API トークンが必要です。

```shell
pip install todoist-api-python
```

```shell
export TODOIST_API_TOKEN=***
```

## 例

以下のエージェントは、Todoist に新しいタスクを作成します。

```python cookbook/tools/todoist.py
"""
Todoist Tools を Agno で使用する方法を示す例

要件:
- Todoist にサインアップ/ログインし、Todoist API トークンを取得する (https://app.todoist.com/app/settings/integrations/developer から取得)
- pip install todoist-api-python

使い方:
- 以下の環境変数を設定します:
    export TODOIST_API_TOKEN="your_api_token"

- または、TodoistTools インスタンスを作成する際に指定します
"""

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.todoist import TodoistTools

todoist_agent = Agent(
    name="Todoist Agent",
    role="Manage your todoist tasks",
    instructions=[
        "When given a task, create a todoist task for it.",
        "When given a list of tasks, create a todoist task for each one.",
        "When given a task to update, update the todoist task.",
        "When given a task to delete, delete the todoist task.",
        "When given a task to get, get the todoist task.",
    ],
    agent_id="todoist-agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[TodoistTools()],
    markdown=True,
    debug_mode=True,
    show_tool_calls=True,
)

# 例 1: タスクを作成する
print("\n=== Create a task ===")
todoist_agent.print_response("Create a todoist task to buy groceries tomorrow at 10am")


# 例 2: タスクを削除する
print("\n=== Delete a task ===")
todoist_agent.print_response(
    "Delete the todoist task to buy groceries tomorrow at 10am"
)


# 例 3: すべてのタスクを取得する
print("\n=== Get all tasks ===")
todoist_agent.print_response("Get all the todoist tasks")
```

## Toolkit パラメータ

| パラメータ | タイプ  | デフォルト | 説明                                       |
| ---------- | ----- | ------- | ------------------------------------------------- |
| `api_token` | `str` | `None`  | TODOIST_API_TOKEN を手動で供給する場合。 |

## Toolkit 関数

| 関数 | 説明 |
|----------|-------------|
| `create_task` | プロジェクト、期日、優先度、ラベルをオプションで指定して、Todoist に新しいタスクを作成します。 |
| `get_task` | 特定のタスクを取得します。 |
| `update_task` | コンテンツ、期日、優先度などの新しいプロパティで既存のタスクを更新します。 |
| `close_task` | タスクを完了としてマークします。 |
| `delete_task` | Todoist から特定のタスクを削除します。 |
| `get_active_tasks` | すべてのアクティブな (完了していない) タスクを取得します。 |
| `get_projects` | Todoist のすべてのプロジェクトを取得します。 |
