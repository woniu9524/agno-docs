## 前提条件

`openai` ライブラリをインストールする必要があります。

```bash
pip install openai
```

`OPENAI_API_KEY` 環境変数を設定します。

```bash
export OPENAI_API_KEY=****
```

## 例

以下のエージェントは、テキストプロンプトに基づいて画像を生成するために DALL-E を使用します。

```python cookbook/tools/dalle_tools.py
from agno.agent import Agent
from agno.tools.dalle import DalleTools

# DALL-E ツールを使用してエージェントを作成します
agent = Agent(tools=[DalleTools()], name="DALL-E Image Generator")

# 例 1: デフォルト設定で基本的な画像を生成します
agent.print_response("Generate an image of a futuristic city with flying cars and tall skyscrapers", markdown=True)

# 例 2: カスタム設定で画像を生成します
custom_dalle = Dalle(model="dall-e-3", size="1792x1024", quality="hd", style="natural")

agent_custom = Agent(
    tools=[custom_dalle],
    name="Custom DALL-E Generator",
    show_tool_calls=True,
)

agent_custom.print_response("Create a panoramic nature scene showing a peaceful mountain lake at sunset", markdown=True)
```

## Toolkit パラメータ

| パラメータ | タイプ  | デフォルト       | 説明                                                       |
| --------- | ----- | ------------- | ----------------------------------------------------------------- |
| `model`   | `str` | `"dall-e-3"`  | 使用する DALL-E モデル                                           |
| `n`       | `int` | `1`           | 生成する画像の数                                                 |
| `size`    | `str` | `"1024x1024"` | 画像サイズ (256x256、512x512、1024x1024、1792x1024、または 1024x1792) |
| `quality` | `str` | `"standard"`  | 画像の品質 (standard または hd)                                    |
| `style`   | `str` | `"vivid"`     | 画像のスタイル (vivid または natural)                               |
| `api_key` | `str` | `None`        | 認証用の OpenAI API キー                                          |

## Toolkit 関数

| 関数         | 説明                                |
| ------------ | ----------------------------------------- |
| `generate_image` | テキストプロンプトに基づいて画像を生成します |
