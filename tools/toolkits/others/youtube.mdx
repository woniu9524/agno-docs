**YouTubeTools** は、動画 URL が提供された場合に、エージェントが YouTube 動画の字幕とメタデータにアクセスできるようにします。

## 前提条件

次の例では、`youtube_transcript_api` ライブラリが必要です。

```shell
pip install -U youtube_transcript_api
```

## 例

以下のエージェントは、YouTube 動画の要約を提供します。

```python cookbook/tools/youtube_tools.py
from agno.agent import Agent
from agno.tools.youtube import YouTubeTools

agent = Agent(
    tools=[YouTubeTools()],
    show_tool_calls=True,
    description="あなたは YouTube エージェントです。YouTube 動画の字幕を取得し、質問に答えてください。",
)

agent.print_response("この動画の要約を教えてください https://www.youtube.com/watch?v=Iv9dewmcFbs&t", markdown=True)
```

## Toolkit パラメータ

| パラメータ           | タイプ        | デフォルト | 説明                                                               |
| -------------------- | ----------- | ------- | ------------------------------------------------------------------ |
| `get_video_captions` | `bool`      | `True`  | 動画の字幕を取得する機能を有効にします。                             |
| `get_video_data`     | `bool`      | `True`  | 動画のメタデータおよびその他の関連データを取得する機能を有効にします。 |
| `languages`          | `List[str]` | -       | 該当する場合、データの取得対象となる言語のリストを指定します。         |

## Toolkit 関数

| 関数                     | 説明                                       |
| ------------------------ | ------------------------------------------ |
| `get_youtube_video_captions` | この関数は YouTube 動画の字幕を取得します。 |
| `get_youtube_video_data`     | この関数は YouTube 動画のメタデータを取得します。 |
