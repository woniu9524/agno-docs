**ResendTools** は、Agent が Resend を使用してメールを送信できるようにします。

## 前提条件

以下の例では、`resend` ライブラリと [Resend](https://resend.com/) からの API キーが必要です。

```shell
pip install -U resend
```

```shell
export RESEND_API_KEY=***
```

## 例

以下のエージェントは、Resend を使用してメールを送信します。

```python cookbook/tools/resend_tools.py
from agno.agent import Agent
from agno.tools.resend import ResendTools

from_email = "<enter_from_email>"
to_email = "<enter_to_email>"

agent = Agent(tools=[ResendTools(from_email=from_email)], show_tool_calls=True)
agent.print_response(f"Send an email to {to_email} greeting them with hello world")
```

## Toolkit パラメータ

| パラメータ    | タイプ  | デフォルト | 説明                                       |
| ------------ | ----- | ------- | ------------------------------------------ |
| `api_key`    | `str` | -       | 認証目的の API キー。                      |
| `from_email` | `str` | -       | メール通信で使用される送信者のメールアドレス。 |

## Toolkit 関数

| 関数       | 説明                           |
| ---------- | ------------------------------ |
| `send_email` | Resend API を使用してメールを送信します。 |
