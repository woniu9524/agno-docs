**Daytona** は、AI生成コードを実行するための安全で弾力性のあるインフラストラクチャを提供します。Agno では、Daytona と統合し、エージェントとチームが Daytona サンドボックスでコードを実行できるようにします。

## 前提条件

Daytona ツールには、`daytona_sdk` Python パッケージが必要です。

```shell
pip install daytona_sdk
```

Daytona API キーも必要です。これは、[Daytona アカウント](https://app.daytona.io/account)から取得できます。

```shell
export DAYTONA_API_KEY=your_api_key
```

## 例

次の例は、Daytona サンドボックスで Python コードを実行できるエージェントを作成する方法を示しています。

```python cookbook/tools/daytona_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.daytona import DaytonaTools

daytona_tools = DaytonaTools()

# Daytona ツールにアクセスできるコーディングタスクに特化したエージェントを設定します
agent = Agent(
    name="Daytona ツールを備えたコーディングエージェント",
    agent_id="coding-agent",
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[daytona_tools],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "あなたは Python コードの作成と検証のエキスパートです。リモートで安全な Daytona サンドボックスにアクセスできます。",
        "あなたの主な目的は次のとおりです。",
        "1. ユーザーのリクエストに基づいて、明確で効率的な Python コードを作成する",
        "2. Daytona サンドボックスでコードを実行および検証する",
        "3. 完全なコードをユーザーと共有する。これが主なユースケースです",
        "4. コードの仕組みについて詳細な説明を提供する",
        "run_python_code ツールを使用して、Daytona サンドボックスで Python コードを実行できます。",
        "ガイドライン:",
        "- コードブロックで適切にフォーマットされた完全なコードを常にユーザーと共有する",
        "- コードを共有する前にサンドボックスで実行して、コードの機能を検証する",
        "- コードが正しく機能するように、必要に応じてコードを反復処理およびデバッグする",
        "- データ分析には、必要に応じて pandas、matplotlib、その他の Python ライブラリを使用する",
        "- 要求されたときに適切な視覚化を作成し、インラインで表示するために画像アーティファクトとして追加する",
        "- ファイルのアップロードとダウンロードを適切に処理する",
        "- アプローチとコードの機能について詳細に説明する",
        "- 最大限の明瞭さのために、コードと説明の両方で応答をフォーマットする",
        "- エラーを適切に処理し、発生した問題について説明する",
    ],
)

# 例: フィボナッチ数列を生成する
agent.print_response(
    "最初の 10 個のフィボナッチ数を生成し、その合計と平均を計算する Python コードを作成してください"
)
```

## ツールキットのパラメータ

| パラメータ            | タイプ   | デフォルト | 説明                                        |
| -------------------- | ------ | ------- | -------------------------------------------------- |
| `api_key`            | `str`  | `None`  | Daytona API キー。提供されない場合は、DAYTONA_API_KEY 環境変数を使用します。 |
| `api_url`            | `str`  | `None`  | Daytona API URL。提供されない場合は、DAYTONA_API_URL 環境変数または Daytona のデフォルトを使用します。 |
| `sandbox_language`   | `CodeLanguage`  | `Python`  | サンドボックスで実行するプログラミング言語 |
| `sandbox_target_region` | `str` | `None`  | サンドボックスが作成されるリージョン |
| `sandbox_os`         | `str`  | `None`  | サンドボックスで実行するオペレーティングシステム (デフォルト: ubuntu) |
| `sandbox_os_user`    | `str`  | `None`  | サンドボックスを実行するユーザー (デフォルト: root) |
| `sandbox_env_vars`   | `dict` | `None`  | サンドボックスで設定する環境変数 |
| `sandbox_labels`     | `dict` | `None`  | サンドボックスで設定するラベル |
| `sandbox_public`     | `bool` | `None`  | サンドボックスを公開するかどうか |
| `sandbox_auto_stop_interval` | `int` | `None`  | サンドボックスが自動的に停止される間隔 (秒単位) |
| `organization_id`    | `str`  | `None`  | サンドボックスに使用する組織 ID |
| `timeout`            | `int`  | `300`   | サンドボックスとの通信のタイムアウト (秒単位) (デフォルト: 5 分) |

### コード実行ツール

| 関数           | 説明                                                  |
| ------------------ | ------------------------------------------------------------ |
| `run_python_code`  | コンテキストの Daytona サンドボックスで Python コードを実行します。               |
| `run_code`         | コンテキストの Daytona サンドボックスで Python 以外のコードを実行します。               |
