**AWSSESTool** 使 Agent 能够使用 Amazon Simple Email Service (SES) 发送电子邮件。

## 先决条件

以下示例需要 `boto3` 库和有效的 AWS 凭证。您可以通过 pip 安装 `boto3`：

```shell
pip install boto3
```

您还必须配置您的 AWS 凭证，以便 SDK 能够向 SES 进行身份验证。最简单的方法是通过 AWS CLI：

```shell
aws configure
# 或者手动设置环境变量
export AWS_ACCESS_KEY_ID=****
export AWS_SECRET_ACCESS_KEY=****
export AWS_DEFAULT_REGION=us-east-1
```

<Note>
  请确保将您要从中发送邮件的域名或电子邮件地址（如果仍处于沙盒模式，也包括收件人地址）添加到 [SES 控制台](https://console.aws.amazon.com/ses/home) 的已验证电子邮件列表中。
</Note>

## 示例

以下 Agent 研究最新的 AI 新闻，然后通过 AWS SES 发送摘要电子邮件：

```python aws_ses_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.aws_ses import AWSSESTool
from agno.tools.duckduckgo import DuckDuckGoTools

# 配置电子邮件设置
sender_email = "verified-sender@example.com"  # 您已验证的 SES 电子邮件
sender_name = "Sender Name"
region_name = "us-east-1"

agent = Agent(
    name="Research Newsletter Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        AWSSESTool(
            sender_email=sender_email,
            sender_name=sender_name,
            region_name=region_name
        ),
        DuckDuckGoTools(),
    ],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "当收到提示时：",
        "1. 提取收件人的完整电子邮件地址（例如 user@domain.com）",
        "2. 使用 DuckDuckGo 研究最新的 AI 发展",
        "3. 撰写一封简洁、引人入胜的电子邮件，总结 3-4 个关键发展",
        "4. 使用 AWS SES 通过 send_email 工具发送电子邮件",
    ],
)

agent.print_response(
    "研究医疗保健领域的最新人工智能发展，并将摘要发送至 johndoe@example.com"
)
```

## Toolkit 参数

| 参数      | 类型  | 默认值      | 描述                              |
| ----------- | ----- | ----------- | --------------------------------- |
| `sender_email` | `str` | `None`      | 已验证的 SES 发件人地址。         |
| `sender_name`  | `str` | `None`      | 显示给收件人的名称。              |
| `region_name`  | `str` | `"us-east-1"` | SES 预置的 AWS 区域。             |

## Toolkit 函数

| 函数     | 描述                                                                          |
| -------- | ----------------------------------------------------------------------------- |
| `send_email` | 发送纯文本电子邮件。接受参数：`subject`、`body`、`receiver_email`。 |
