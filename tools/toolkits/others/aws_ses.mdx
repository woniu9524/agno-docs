**AWSSESTool** は、Amazon Simple Email Service (SES) を使用してエージェントが電子メールを送信できるようにします。

## 前提条件

次の例では、`boto3` ライブラリと有効な AWS 認証情報が必要です。`boto3` は pip を使用してインストールできます。

```shell
pip install boto3
```

SDK が SES に認証できるように、AWS 認証情報も設定する必要があります。最も簡単な方法は、AWS CLI を使用することです。

```shell
aws configure
# または、環境変数を手動で設定します
export AWS_ACCESS_KEY_ID=****
export AWS_SECRET_ACCESS_KEY=****
export AWS_DEFAULT_REGION=us-east-1
```

<Note>
  送信元として使用したいドメインまたは電子メールアドレス（サンドボックスモードの場合は、送信先アドレスも）を、[SES コンソール](https://console.aws.amazon.com/ses/home) の検証済みメールに追加してください。
</Note>

## 例

以下のエージェントは、最新の AI ニュースを調査し、その概要を AWS SES 経由で電子メールで送信します。

```python aws_ses_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.aws_ses import AWSSESTool
from agno.tools.duckduckgo import DuckDuckGoTools

# 電子メール設定の構成
sender_email = "verified-sender@example.com"  # 検証済みの SES 電子メール
sender_name = "Sender Name"
region_name = "us-east-1"

agent = Agent(
    name="Research Newsletter Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        AWSSESTool(
            sender_email=sender_email,
            sender_name=sender_name,
            region_name=region_name
        ),
        DuckDuckGoTools(),
    ],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "プロンプトが与えられた場合:",
        "1. 受信者の完全な電子メールアドレス（例：user@domain.com）を抽出します。",
        "2. DuckDuckGo を使用して最新の AI 開発を調査します。",
        "3. 3 ～ 4 件の主要な開発を要約した、簡潔で魅力的な電子メールを作成します。",
        "4. send_email ツールを使用して AWS SES で電子メールを送信します。",
    ],
)

agent.print_response(
    "ヘルスケア分野の最近の AI 開発を調査し、johndoe@example.com に概要を電子メールで送信してください"
)
```

## Toolkit のパラメータ

| パラメータ      | タイプ  | デフォルト       | 説明                                       |
| -------------- | ----- | ------------- | ---------------------------------------- |
| `sender_email` | `str` | `None`        | 検証済みの SES 送信元アドレス。            |
| `sender_name`  | `str` | `None`        | 受信者に表示される表示名。                 |
| `region_name`  | `str` | `"us-east-1"` | SES がプロビジョニングされている AWS リージョン。 |

## Toolkit の関数

| 関数     | 説明                                                                          |
| -------- | ----------------------------------------------------------------------------- |
| `send_email` | プレーンテキストの電子メールを送信します。`subject`、`body`、`receiver_email` の引数を受け取ります。 |
