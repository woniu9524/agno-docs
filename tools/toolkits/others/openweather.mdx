**OpenWeatherTools** は、エージェントがOpenWeatherMap APIから天気データにアクセスできるようにします。

## 前提条件

次の例では、`requests` ライブラリと、[OpenWeatherMap](https://openweathermap.org/api) から取得できるAPIキーが必要です。サインアップ後、指定されたAPIキーは数時間後にアクティブになりますので、しばらくお待ちください。

```shell
export OPENWEATHER_API_KEY=***
```

## 例

以下のエージェントは、OpenWeatherMapを使用して東京の現在の天気情報を取得します。

```python cookbook/tools/openweather_tools.py
from agno.agent import Agent
from agno.tools.openweather import OpenWeatherTools

# OpenWeatherTools を使用してエージェントを作成します
agent = Agent(
    tools=[
        OpenWeatherTools(
            units="imperial",  # オプション: 'standard', 'metric', 'imperial'
        )
    ],
    markdown=True,
)

# 場所の現在の天気情報を取得します
agent.print_response("What's the current weather in Tokyo?", markdown=True)
```

## Toolkit Params

| パラメータ         | タイプ   | デフォルト   | 説明                                                                                             |
| ------------------ | ------ | --------- | ------------------------------------------------------------------------------------------------ |
| `api_key`          | `str`  | `None`    | OpenWeatherMap API キー。指定しない場合は、OPENWEATHER_API_KEY 環境変数を使用します。                               |
| `units`            | `str`  | `metric`  | 測定単位。オプション: 'standard', 'metric', 'imperial'。                                                              |
| `current_weather`  | `bool` | `True`    | 現在の天気機能を使用可能にします。                                                                                             |
| `forecast`         | `bool` | `True`    | 予報機能を使用可能にします。                                                                                             |
| `air_pollution`    | `bool` | `True`    | 大気汚染機能を使用可能にします。                                                                                             |
| `geocoding`        | `bool` | `True`    | ジオコーディング機能を使用可能にします。                                                                                             |

## Toolkit Functions

| 関数                 | 説明                                                                                                                             |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `get_current_weather`| 場所の現在の天気データを取得します。場所の名前（例：「ロンドン」）を取ります。                                                                                                |
| `get_forecast`       | 場所の天気予報を取得します。場所の名前とオプションで日数（デフォルトは5日）を取ります。                                                                                             |
| `get_air_pollution`  | 場所の現在の空気汚染データを取得します。場所の名前を取ります。                                                                                               |
| `geocode_location`   | 場所の名前を地理座標に変換します。場所の名前とオプションで結果の制限を取ります。                                                                                             |
