[**ComposioTools**](https://docs.composio.dev/framework/phidata) は、Agent が Gmail、Salesforce、Github などのツールを操作できるようにします。

## 前提条件

以下の例では、`composio-agno` ライブラリが必要です。

```shell
pip install composio-agno
composio add github # Githubにログイン
```

## 例

以下のエージェントは、Composio Toolkit の Github Tool を使用してリポジトリをスター付けします。

```python cookbook/tools/composio_tools.py
from agno.agent import Agent
from composio_agno import Action, ComposioToolSet


toolset = ComposioToolSet()
composio_tools = toolset.get_tools(
  actions=[Action.GITHUB_STAR_A_REPOSITORY_FOR_THE_AUTHENTICATED_USER]
)

agent = Agent(tools=composio_tools, show_tool_calls=True)
agent.print_response("Can you star agno-agi/agno repo?")
```

## Toolkit パラメータ

GitHub のリポジトリのスター付けアクションを呼び出す際に使用されるパラメータは次のとおりです。

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `owner` | `str` | - | スターを付けるリポジトリのオーナー。 |
| `repo` | `str` | - | スターを付けるリポジトリの名前。 |

## Toolkit 関数

Composio Toolkit は、さまざまなソフトウェアツールに接続するための 1000 を超える関数を提供しています。
完全な関数のリストを表示するには、この[リンク](https://composio.dev/tools)を開いてください。
