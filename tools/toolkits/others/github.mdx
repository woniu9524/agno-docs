**GithubTools** は、エージェントがGithubリポジトリにアクセスし、オープンなプルリクエストやイシューなどをリストするタスクを実行できるようにします。

## 前提条件

以下の例では、`PyGithub`ライブラリと、[こちら](https://github.com/settings/tokens)から取得できるGithubアクセストークンが必要です。

```shell
pip install -U PyGithub
```

```shell
export GITHUB_ACCESS_TOKEN=***
```

## 例

以下のエージェントは、"Mistral AI"に関する最新ニュースをGoogleで検索します。

```python cookbook/tools/github_tools.py
from agno.agent import Agent
from agno.tools.github import GithubTools

agent = Agent(
    instructions=[
        "Use your tools to answer questions about the repo: agno-agi/agno",
        "Do not create any issues or pull requests unless explicitly asked to do so",
    ],
    tools=[GithubTools()],
    show_tool_calls=True,
)

agent.print_response("List open pull requests", markdown=True)
```

## Toolkit Params

| パラメータ                  | タイプ   | デフォルト | 説明                                                                                                 |
| -------------------------- | ------ | ------- | ----------------------------------------------------------------------------------------------------------- |
| `access_token`             | `str`  | `None`  | 認証用のGithubアクセストークン。提供されない場合は、`GITHUB_ACCESS_TOKEN`環境変数を使用します。 |
| `base_url`                 | `str`  | `None`  | Github EnterpriseインストールのためのオプションのベースURL。                                                      |
| `search_repositories`      | `bool` | `True`  | Githubリポジトリの検索を有効にします。                                                                       |
| `list_repositories`        | `bool` | `True`  | ユーザー/組織のリポジトリのリスト表示を有効にします。                                                        |
| `get_repository`           | `bool` | `True`  | リポジトリの詳細取得を有効にします。                                                                          |
| `list_pull_requests`       | `bool` | `True`  | リポジトリのプルリクエストのリスト表示を有効にします。                                                        |
| `get_pull_request`         | `bool` | `True`  | プルリクエストの詳細取得を有効にします。                                                                        |
| `get_pull_request_changes` | `bool` | `True`  | プルリクエストのファイル変更取得を有効にします。                                                                   |
| `create_issue`             | `bool` | `True`  | リポジトリでのイシュー作成を有効にします。                                                                       |

## Toolkit Functions

| 関数                   | 説明                                          |
| -------------------------- | ---------------------------------------------------- |
| `search_repositories`      | クエリに基づいてGithubリポジトリを検索します。       |
| `list_repositories`        | 指定されたユーザーまたは組織のリポジトリをリストします。 |
| `get_repository`           | 特定のリポジトリの詳細を取得します。            |
| `list_pull_requests`       | リポジトリのプルリクエストをリストします。                |
| `get_pull_request`         | 特定のプルリクエストの詳細を取得します。          |
| `get_pull_request_changes` | プルリクエストのファイル変更を取得します。             |
| `create_issue`             | リポジトリに新しいイシューを作成します。                 |
