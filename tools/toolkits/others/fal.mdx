**FalTools** を使用すると、エージェントはメディア生成タスクを実行できます。

## 前提条件

次の例では、`fal_client` ライブラリと、[Fal](https://fal.ai/) から取得できる API キーが必要です。

```shell
pip install -U fal_client
```

```shell
export FAL_KEY=***
```

## 例

次のエージェントは、FAL を使用してユーザーが要求した任意のビデオを生成します。

```python cookbook/tools/fal_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.fal import FalTools

fal_agent = Agent(
    name="Fal Video Generator Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[FalTools("fal-ai/hunyuan-video")],
    description="You are an AI agent that can generate videos using the Fal API.",
    instructions=[
        "When the user asks you to create a video, use the `generate_media` tool to create the video.",
        "Return the URL as raw to the user.",
        "Don't convert video URL to markdown or anything else.",
    ],
    markdown=True,
    debug_mode=True,
    show_tool_calls=True,
)

fal_agent.print_response("Generate video of balloon in the ocean")
```

## Toolkit Params

| Parameter | Type  | Default | Description                                |
| --------- | ----- | ------- | ------------------------------------------ |
| `api_key` | `str` | `None`  | API キー（認証用）。       |
| `model`   | `str` | `None`  | メディア生成に使用するモデル。 |

## Toolkit Functions

| Function         | Description                                                    |
| ---------------- | -------------------------------------------------------------- |
| `generate_media` | ユーザープロンプトに応じて画像またはビデオを生成します。 |
| `image_to_image` | テキストプロンプトに基づいて入力画像を変換します。               |
