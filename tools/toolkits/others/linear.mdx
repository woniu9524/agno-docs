**LinearTool** は、エージェントが [Linear](https://linear.app/) タスクを実行できるようにします。

## 前提条件
以下の例では、Linear API キーが必要です。これは [こちら](https://linear.app/settings/account/security) から取得できます。

```shell
export LINEAR_API_KEY="LINEAR_API_KEY"
```

## 例

以下のエージェントは、Linear API を使用して、特定のユーザーのプロジェクト内の課題を検索します。

```python cookbook/tools/linear_tools.py
from agno.agent import Agent
from agno.tools.linear import LinearTools

agent = Agent(
    name="Linear Tool Agent",
    tools=[LinearTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Show all the issues assigned to user id: 12021")
```

## Toolkit パラメータ

| パラメータ                  | タイプ   | デフォルト | 説明                             |
| -------------------------- | ------ | ------- | -------------------------------- |
| `get_user_details`         | `bool` | `True`  | `get_user_details` ツールを有効にします。 |
| `get_issue_details`        | `bool` | `True`  | `get_issue_details` ツールを有効にします。 |
| `create_issue`             | `bool` | `True`  | `create_issue` ツールを有効にします。     |
| `update_issue`             | `bool` | `True`  | `update_issue` ツールを有効にします。     |
| `get_user_assigned_issues` | `bool` | `True`  | `get_user_assigned_issues` ツールを有効にします。 |
| `get_workflow_issues`      | `bool` | `True`  | `get_workflow_issues` ツールを有効にします。 |
| `get_high_priority_issues` | `bool` | `True`  | `get_high_priority_issues` ツールを有効にします。 |

## Toolkit 関数

| 関数                   | 説明                                                       |
| -------------------------- | ---------------------------------------------------------- |
| `get_user_details`         | 認証されたユーザーの詳細を取得します。                       |
| `get_issue_details`        | 課題 ID で特定の課題の詳細を取得します。                     |
| `create_issue`             | 特定のプロジェクトとチーム内に新しい課題を作成します。         |
| `update_issue`             | 課題 ID で特定の課題のタイトルまたは状態を更新します。         |
| `get_user_assigned_issues` | ユーザー ID で特定のユーザーに割り当てられた課題を取得します。 |
| `get_workflow_issues`      | ワークフロー ID で特定のワークフロー状態内の課題を取得します。 |
| `get_high_priority_issues` | 優先度の高い課題（優先度 `<=` 2）を取得します。             |
