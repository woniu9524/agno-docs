**LinearTool** 使 Agent 能够执行 [Linear](https://linear.app/) 任务。

## 前提条件
以下示例需要 Linear API 密钥，可在此处获取：[ Linear API 密钥](https://linear.app/settings/account/security)。


```shell
export LINEAR_API_KEY="LINEAR_API_KEY"
```

## 示例

以下 Agent 将使用 Linear API 为特定用户搜索项目中的问题。

```python cookbook/tools/linear_tools.py
from agno.agent import Agent
from agno.tools.linear import LinearTools

agent = Agent(
    name="Linear Tool Agent",
    tools=[LinearTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Show all the issues assigned to user id: 12021")
```

## Toolkit 参数

| 参数                  | 类型   | 默认值 | 描述                           |
| ---------------------- | ------ | ------ | ------------------------------- |
| `get_user_details`     | `bool` | `True` | 启用 `get_user_details` 工具。 |
| `get_issue_details`    | `bool` | `True` | 启用 `get_issue_details` 工具。 |
| `create_issue`         | `bool` | `True` | 启用 `create_issue` 工具。     |
| `update_issue`         | `bool` | `True` | 启用 `update_issue` 工具。     |
| `get_user_assigned_issues` | `bool` | `True` | 启用 `get_user_assigned_issues` 工具。 |
| `get_workflow_issues`  | `bool` | `True` | 启用 `get_workflow_issues` 工具。 |
| `get_high_priority_issues` | `bool` | `True` | 启用 `get_high_priority_issues` 工具。 |

## Toolkit 函数

| 函数                   | 描述                                                   |
| ---------------------- | ------------------------------------------------------ |
| `get_user_details`     | 获取已认证的用户详细信息。                           |
| `get_issue_details`    | 通过问题 ID 检索特定问题的详细信息。                   |
| `create_issue`         | 在特定项目和团队中创建新问题。                         |
| `update_issue`         | 通过问题 ID 更新特定问题的标题或状态。                 |
| `get_user_assigned_issues` | 通过用户 ID 检索分配给特定用户的问题。                 |
| `get_workflow_issues`  | 通过工作流 ID 检索特定工作流状态中的问题。             |
| `get_high_priority_issues` | 检索具有高优先级的（优先级 <= 2）问题。              |
