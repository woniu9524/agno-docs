## 前提条件

以下の例では、`trello` ライブラリと Trello API の認証情報が必要です。これらは、Trello の開発者向けドキュメントに従って取得できます。

```shell
pip install -U trello
```

以下の環境変数を設定してください。
```shell
export TRELLO_API_KEY="YOUR_API_KEY"
export TRELLO_API_SECRET="YOUR_API_SECRET"
export TRELLO_TOKEN="YOUR_TOKEN"
```

## 例

以下のエージェントは、`ai-agent` という名前のボードを作成し、その中に `todo` と `doing` という名前のリストを作成し、さらにそれぞれのリストの中に `create agent` という名前のカードを作成します。

```python
from agno.agent import Agent
from agno.tools.trello import TrelloTools

agent = Agent(
    instructions=[
        "あなたは Trello ボード、リスト、カードの整理と管理を支援する Trello 管理アシスタントです",
        "以下のようなタスクでユーザーを支援してください:",
        "- ボード、リスト、カードの作成と整理",
        "- カードをリスト間で移動する",
        "- ボードとリスト情報の取得",
        "- カードの詳細と説明の管理",
        "常に操作の成功を確認し、関連するボード/リスト/カードの ID と URL を提供してください",
        "エラーが発生した場合は、明確な説明を提供し、解決策を提案してください",
    ],
    tools=[TrelloTools()],
    show_tool_calls=True,
)

agent.print_response(
    "ai-agent という名前のボードを作成し、その中に 'todo' と 'doing' という名前のリストを作成し、さらにそれぞれのリストの中に 'create agent' という名前のカードを作成してください",
    stream=True,
)

```

## Toolkit 関数

| 関数          | 説明                                                                 |
| ------------- | -------------------------------------------------------------------- |
| `create_card`     | 指定されたボードとリストに新しいカードを作成します。                               |
| `get_board_lists` | 指定された Trello ボードのすべてのリストを取得します。                               |
| `move_card`       | カードを別のリストに移動します。                                                   |
| `get_cards`       | 指定されたリストからすべてのカードを取得します。                                     |
| `create_board`    | 新しい Trello ボードを作成します。                                                 |
| `create_list`     | 指定されたボードに新しいリストを作成します。                                         |
| `list_boards`     | 認証されたユーザーがアクセスできるすべての Trello ボードを一覧表示します。                             |

## Toolkit パラメータ

これらのパラメータは `TrelloTools` コンストラクタに渡されます。

| パラメータ         | タイプ          | デフォルト | 説明                                                                       |
| ----------------- | ------------- | ------- | -------------------------------------------------------------------------- |
| `api_key`         | `Optional[str]` | `None`  | Trello API キー。デフォルトでは `TRELLO_API_KEY` 環境変数を使用します。                  |
| `api_secret`      | `Optional[str]` | `None`  | Trello API シークレット。デフォルトでは `TRELLO_API_SECRET` 環境変数を使用します。           |
| `token`           | `Optional[str]` | `None`  | Trello トークン。デフォルトでは `TRELLO_TOKEN` 環境変数を使用します。                     |
| `create_card`     | `bool`        | `True`  | `create_card` ツールを有効にします。                                                 |
| `get_board_lists` | `bool`        | `True`  | `get_board_lists` ツールを有効にします。                                               |
| `move_card`       | `bool`        | `True`  | `move_card` ツールを有効にします。                                                   |
| `get_cards`       | `bool`        | `True`  | `get_cards` ツールを有効にします。                                                   |
| `create_board`    | `bool`        | `True`  | `create_board` ツールを有効にします。                                                |
| `create_list`     | `bool`        | `True`  | `create_list` ツールを有効にします。                                                 |
| `list_boards`     | `bool`        | `True`  | `list_boards` ツールを有効にします。                                                 |

### `list_boards` のボードフィルターオプション

`list_boards` 関数は、以下のオプションを持つ `board_filter` 引数を受け入れます。
- `all` (デフォルト)
- `open`
- `closed`
- `organization`
- `public`
- `starred`
