**ConfluenceTools** を使用すると、エージェントは Confluence のページを取得、作成、更新できます。また、スペースやページの詳細を探索することもできます。

## 前提条件

次の例では、`atlassian-python-api` ライブラリと Confluence の認証情報が必要です。API トークンは、[こちら](https://id.atlassian.com/manage-profile/security) から取得できます。

```shell
pip install atlassian-python-api
```

```shell
export CONFLUENCE_URL="https://your-confluence-instance"
export CONFLUENCE_USERNAME="your-username"
export CONFLUENCE_PASSWORD="your-password"
# または
export CONFLUENCE_API_KEY="your-api-key"
```

## 例
次のエージェントは、スペースの数とその名前を取得します。

```python
from agno.agent import Agent
from agno.tools.confluence import ConfluenceTools

agent = Agent(
    name="Confluence agent",
    tools=[ConfluenceTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("How many spaces are there and what are their names?")
```

## Toolkit Functions

| Parameter | Type     | Default | Description                                                                                                                    |
| --------- | -------- | ------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `username`| `str`    | -  | Confluence のユーザー名。環境変数 CONFLUENCE_USERNAME でも設定できます。                                           |
| `password`| `str`    | -  | Confluence のパスワードまたは API キー。環境変数 CONFLUENCE_API_KEY または CONFLUENCE_PASSWORD でも設定できます。       |
| `url`     | `str`    | -  | Confluence インスタンスの URL。環境変数 CONFLUENCE_URL でも設定できます。                                            |
| `api_key` | `str`    | -  | Confluence API キー (パスワードの代替)。                                                                                 |
| `ssl_verify` | `bool`    | `True`  | True の場合、SSL 証明書を検証します。                                                                                 |

## Toolkit Functions

| Function                                                         | Description                                                                                                                                         |
| ---------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `get_page_content` | 特定のページのコンテンツを取得します。                                |
| `get_all_space_detail`                                         | すべての Confluence スペースの詳細を取得します。                                                     |
| `get_space_key`                                 | 指定されたスペースの Confluence キーを取得します。                                                        |
| `get_all_page_from_space`                       | 指定されたスペースのすべてのページの詳細を取得します。                                                     |
| `create_page` | 提供されたタイトルと本文で新しい Confluence ページを作成します。       |
| `update_page`               | 既存の Confluence ページを更新します。                                    |
