このガイドでは、Agno フレームワーク内で Apify Actor を統合および使用して、Web スクレイピング、クロール、データ抽出、Web 自動化機能で AI エージェントを強化する方法を説明します。

## Apify とは？

[Apify](https://apify.com/) は、以下を提供するプラットフォームです。
- ソーシャルメディア、検索エンジン、オンラインマップ、E コマースサイト、旅行ポータル、または一般的な Web サイトからデータを抽出することに特化した、AI エージェント向けのデータ収集サービス
- さまざまなデータタスクに対応する、すぐに使用できる Actor (特殊なツール) のマーケットプレイス
- 独自の AI エージェントを実行および収益化するためのインフラストラクチャ

## 前提条件

1. [Apify アカウント](https://console.apify.com/sign-up)にサインアップします。
2. Apify API トークンを取得します ([Apify](https://docs.apify.com/platform/integrations/api) から取得できます)。
3. 必要なパッケージをインストールします。

```bash
pip install agno apify-client
```

## 基本的な使い方

Agno フレームワークを使用すると、Apify Actor をエージェントに簡単に統合できます。簡単な例を以下に示します。

```python
from agno.agent import Agent
from agno.tools.apify import ApifyTools

# ApifyTools を使用してエージェントを作成します
agent = Agent(
    tools=[
        ApifyTools(
            actors=["apify/rag-web-browser"],  # 使用する Apify Actor を指定します。必要に応じて複数指定できます。
            apify_api_token="your_apify_api_key"  # または APIFY_API_TOKEN 環境変数を設定します。
        )
    ],
    show_tool_calls=True,
    markdown=True
)

# エージェントを使用して Web サイトのコンテンツを取得します
agent.print_response("https://docs.agno.com/introduction からどのような情報が見つかりますか？", markdown=True)
```

## 利用可能な Apify ツール

任意の Apify Actor をツールとして簡単に統合できます。以下に例を示します。

### 1. RAG Web Browser

[RAG Web Browser](https://apify.com/apify/rag-web-browser) Actor は、AI および LLM アプリケーション向けに特別に設計されています。クエリを Web で検索するか、URL を処理してから、エージェント向けにコンテンツをクリーニングおよびフォーマットします。このツールはデフォルトで有効になっています。

```python
from agno.agent import Agent
from agno.tools.apify import ApifyTools

agent = Agent(
    tools=[
        ApifyTools(actors=["apify/rag-web-browser"])
    ],
    show_tool_calls=True,
    markdown=True
)

# 情報を検索して結果を処理します
agent.print_response("大規模言語モデルの最新の開発状況について教えてください。", markdown=True)
```

### 2. Website Content Crawler

このツールは、Apify の [Website Content Crawler](https://apify.com/apify/website-content-crawler) Actor を使用して Web サイトからテキストコンテンツを抽出するため、RAG アプリケーションに最適です。

```python
from agno.agent import Agent
from agno.tools.apify import ApifyTools

agent = Agent(
    tools=[
        ApifyTools(actors=["apify/website-content-crawler"])
    ],
    markdown=True
)

# エージェントに Web コンテンツを処理するように依頼します
agent.print_response("https://docs.agno.com/introduction のコンテンツを要約してください。", markdown=True)
```

### 3. Google Places Crawler

[Google Places Crawler](https://apify.com/compass/crawler-google-places) は、Google マップおよび Google プレイスからビジネスに関するデータを抽出します。

```python
from agno.agent import Agent
from agno.tools.apify import ApifyTools

agent = Agent(
    tools=[
        ApifyTools(actors=["compass/crawler-google-places"])
    ],
    show_tool_calls=True
)

# 特定の場所のビジネス情報を検索します
agent.print_response("サンフランシスコで評価の高いレストランは何ですか？", markdown=True)
agent.print_response("プラハのコーヒーショップを探してください。", markdown=True)
```

## ユースケース例

### RAG Web Browser + Google Places Crawler
この例では、Web 検索とローカルビジネスデータを組み合わせて、トピックに関する包括的な情報を提供します。

```python
from agno.agent import Agent
from agno.tools.apify import ApifyTools

agent = Agent(
    tools=[
        ApifyTools(actors=[
            "apify/rag-web-browser",
            "compass/crawler-google-places"
        ])
    ],
    show_tool_calls=True
)

# 一般情報とローカルビジネス情報を取得します
agent.print_response(
    """
    来月東京に旅行します。
    1. 旅行に最適な時期と主要なアトラクションを調査してください。
    2. 新宿近くで評価の高い寿司レストランを 1軒見つけてください。
    この情報を使用して、包括的な旅行ガイドを作成してください。
    """,
    markdown=True
)
```

## Toolkit パラメータ

| パラメータ                    | タイプ                | デフォルト | 説明                                                              |
| ---------------------------- | ------------------- | ------- | ------------------------------------------------------------------ |
| `apify_api_token`            | `str`               | `None`  | Apify API トークン (または APIFY_API_TOKEN 環境変数で設定)  |
| `actors`                     | `str` または `List[str]`| `None`  | 単一の Actor ID または登録する Actor ID のリスト                   |
