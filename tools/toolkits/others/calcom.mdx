## 前提条件

以下の例では、`pytz` および `requests` ライブラリが必要です。

```shell
pip install requests pytz
```

```shell
export CALCOM_API_KEY="your_api_key"
export CALCOM_EVENT_TYPE_ID="your_event_type_id"
```

## 例

以下のエージェントは、Cal.com を使用して、明日あなたの Cal.com アカウントのすべてのイベントを一覧表示します。

```python cookbook/tools/calcom_tools.py

agent = Agent(
    name="Calendar Assistant",
    instructions=[
        f"You're scheduing assistant. Today is {datetime.now()}.",
        "You can help users by:",
        "- Finding available time slots",
        "- Creating new bookings",
        "- Managing existing bookings (view, reschedule, cancel) ",
        "- Getting booking details",
        "- IMPORTANT: In case of rescheduling or cancelling booking, call the get_upcoming_bookings function to get the booking uid. check available slots before making a booking for given time",
        "Always confirm important details before making bookings or changes.",
    ],
    model=OpenAIChat(id="gpt-4"),
    tools=[CalComTools(user_timezone="America/New_York")],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("What are my bookings for tomorrow?")
```

## Toolkit パラメータ

| パラメータ               | タイプ   | デフォルト | 説明                                      |
| ----------------------- | ------ | ------- | ----------------------------------------- |
| `api_key`               | `str`  | `None`  | Cal.com API キー                          |
| `event_type_id`         | `int`  | `None`  | スケジュール設定用のイベントタイプ ID     |
| `user_timezone`         | `str`  | `None`  | ユーザーのタイムゾーン (例: "America/New_York") |
| `get_available_slots`   | `bool` | `True`  | 利用可能なタイムスロットの取得を有効にする |
| `create_booking`        | `bool` | `True`  | 新規予約の作成を有効にする                |
| `get_upcoming_bookings` | `bool` | `True`  | 今後の予約の取得を有効にする              |
| `reschedule_booking`    | `bool` | `True`  | 予約のリスケジュールを有効にする          |
| `cancel_booking`        | `bool` | `True`  | 予約のキャンセルを有効にする              |

## Toolkit 関数

| 関数                | 説明                                       |
| ------------------- | ------------------------------------------ |
| `get_available_slots`   | 指定された日付範囲の利用可能なタイムスロットを取得します |
| `create_booking`        | 提供された詳細で新しい予約を作成します       |
| `get_upcoming_bookings` | 今後の予約のリストを取得します               |
| `get_booking_details`   | 特定の予約の詳細を取得します               |
| `reschedule_booking`    | 既存の予約をリスケジュールします           |
| `cancel_booking`        | 既存の予約をキャンセルします               |
