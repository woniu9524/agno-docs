## 前提条件

以下の例では、`boto3` ライブラリが必要です。

```shell
pip install openai boto3
```

## 例

以下のエージェントは、AWS Lambda を使用して AWS アカウント内のすべての Lambda 関数を一覧表示し、特定の Lambda 関数を呼び出します。

```python cookbook/tools/aws_lambda_tools.py

from agno.agent import Agent
from agno.tools.aws_lambda import AWSLambdaTools


# AWSLambdaTool を使用した Agent の作成
agent = Agent(
    tools=[AWSLambdaTools(region_name="us-east-1")],
    name="AWS Lambda Agent",
    show_tool_calls=True,
)

# 例 1: すべての Lambda 関数を一覧表示
agent.print_response("List all Lambda functions in our AWS account", markdown=True)

# 例 2: 特定の Lambda 関数を呼び出す
agent.print_response("Invoke the 'hello-world' Lambda function with an empty payload", markdown=True)
```

## Toolkit パラメータ

| パラメータ     | タイプ  | デフォルト       | 説明                                         |
| ------------- | ----- | ------------- | --------------------------------------------------- |
| `region_name` | `str` | `"us-east-1"` | Lambda 関数が存在する AWS リージョン名。 |

## Toolkit 関数

| 関数          | 説明                                                                                                           |
| ----------------- | --------------------------------------------------------------------------------------------------------------------- |
| `list_functions`  | AWS アカウントで利用可能なすべての Lambda 関数を一覧表示します。                                                              |
| `invoke_function` | オプションのペイロードを指定して、特定の Lambda 関数を呼び出します。`function_name` およびオプションの `payload` パラメータを受け取ります。 |
