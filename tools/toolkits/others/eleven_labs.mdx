**ElevenLabsTools** は、[ElevenLabs](https://elevenlabs.io/docs/product/introduction) を使用して音声生成タスクを実行できるようにします。

## 前提条件

`elevenlabs` ライブラリと、[Eleven Labs](https://elevenlabs.io/) から取得できる API キーをインストールする必要があります。

```bash
pip install elevenlabs
```

`ELEVEN_LABS_API_KEY` 環境変数を設定します。

```bash
export ELEVEN_LABS_API_KEY=****
```

## 例

以下のエージェントは、Eleven Labs を使用してユーザーのプロンプトに基づいて音声を生成します。

```python cookbook/tools/eleven_labs_tools.py
from agno.agent import Agent
from agno.tools.eleven_labs import ElevenLabsTools

# ElevenLabs ツールを使用してエージェントを作成します
agent = Agent(tools=[
    ElevenLabsTools(
        voice_id="JBFqnCBsd6RMkjVDRZzb", model_id="eleven_multilingual_v2", target_directory="audio_generations"
    )
], name="ElevenLabs Agent")

agent.print_response("Generate a audio summary of the big bang theory", markdown=True)
```

## Toolkit パラメータ

| パラメータ          | タイプ            | デフォルト               | 説明                                                                                                                                                           |
| ------------------ | --------------- | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `api_key`          | `str`           | `None`                   | 認証用の Eleven Labs API キー                                                                                                                                  |
| `voice_id`         | `str`           | `JBFqnCBsd6RMkjVDRZzb`   | 音声生成に使用するボイス ID                                                                                                                                    |
| `target_directory` | `Optional[str]` | `None`                   | 音声ファイルを保存するディレクトリ                                                                                                                             |
| `model_id`         | `str`           | `eleven_multilingual_v2` | 音声生成に使用するモデル ID                                                                                                                                    |
| `output_format`    | `str`           | `mp3_44100_64`           | 音声生成に使用する出力フォーマット（詳細については[ドキュメント](https://elevenlabs.io/docs/api-reference/text-to-speech#parameter-output-format)を参照してください） |

## Toolkit 関数

| 関数                | 説明                                  |
| ------------------- | ------------------------------------- |
| `text_to_speech`    | テキストを音声に変換します。              |
| `generate_sound_effect` | テキストプロンプトから効果音を生成します。 |
| `get_voices`        | 利用可能なボイスのリストを取得します。      |
