**MLX Transcribe** は、MLX Whisper を使用して音声ファイルを文字起こしするためのツールです。

## 前提条件

1. **ffmpeg のインストール**

   - macOS: `brew install ffmpeg`
   - Ubuntu: `sudo apt-get install ffmpeg`
   - Windows: https://ffmpeg.org/download.html からダウンロードしてください。

2. **mlx-whisper ライブラリのインストール**

   ```shell
   pip install mlx-whisper
   ```

3. **音声ファイルの準備**

   - `storage/audio` ディレクトリを作成します。
   - 音声ファイルをこのディレクトリに配置します。
   - サポートされている形式: mp3, mp4, wav など。

4. **サンプル音声のダウンロード** (オプション)
   - [audio-samples](https://audio-samples.github.io/) (例) にアクセスし、音声ファイルを `storage/audio` ディレクトリに保存します。

## 例

以下のエージェントは、MLX Transcribe を使用して音声ファイルを文字起こしします。

```python cookbook/tools/mlx_transcribe_tools.py

from pathlib import Path
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.mlx_transcribe import MLXTranscribeTools

# storage/audio ディレクトリから音声ファイルを取得
agno_root_dir = Path(__file__).parent.parent.parent.resolve()
audio_storage_dir = agno_root_dir.joinpath("storage/audio")
if not audio_storage_dir.exists():
    audio_storage_dir.mkdir(exist_ok=True, parents=True)

agent = Agent(
    name="Transcription Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[MLXTranscribeTools(base_dir=audio_storage_dir)],
    instructions=[
        "音声ファイルを文字起こしするには、`transcribe` ツールを使用し、引数として音声ファイル名を指定してください。",
        "利用可能なすべての音声ファイルは `read_files` ツールを使用して見つけることができます。",
    ],
    markdown=True,
)

agent.print_response("reid hoffman の ted talk を要約し、セクションに分割してください", stream=True)
```

## Toolkit パラメータ

| パラメータ                         | タイプ                           | デフォルト                               | 説明                                     |
| --------------------------------- | ------------------------------ | ---------------------------------------- | ---------------------------------------- |
| `base_dir`                        | `Path`                         | `Path.cwd()`                             | 音声ファイルのベースディレクトリ         |
| `read_files_in_base_dir`          | `bool`                         | `True`                                   | `read_files` 関数を登録するかどうか       |
| `path_or_hf_repo`                 | `str`                          | `"mlx-community/whisper-large-v3-turbo"` | モデルのパスまたは HuggingFace リポジトリ |
| `verbose`                         | `bool`                         | `None`                                   | 詳細出力を有効にする                     |
| `temperature`                     | `float` or `Tuple[float, ...]` | `None`                                   | サンプリングのための温度                 |
| `compression_ratio_threshold`     | `float`                        | `None`                                   | 圧縮率のしきい値                         |
| `logprob_threshold`               | `float`                        | `None`                                   | ログ確率のしきい値                       |
| `no_speech_threshold`             | `float`                        | `None`                                   | 無音のしきい値                           |
| `condition_on_previous_text`      | `bool`                         | `None`                                   | 前のテキストに基づいて条件付けするかどうか |
| `initial_prompt`                  | `str`                          | `None`                                   | 文字起こしのための初期プロンプト         |
| `word_timestamps`                 | `bool`                         | `None`                                   | 単語レベルのタイムスタンプを有効にする   |
| `prepend_punctuations`            | `str`                          | `None`                                   | 先頭に追加する句読点                     |
| `append_punctuations`             | `str`                          | `None`                                   | 末尾に追加する句読点                     |
| `clip_timestamps`                 | `str` or `List[float]`         | `None`                                   | タイムスタンプをクリップする             |
| `hallucination_silence_threshold` | `float`                        | `None`                                   | 幻覚の無音しきい値                       |
| `decode_options`                  | `dict`                         | `None`                                   | 追加のデコードオプション                 |

## Toolkit 関数

| 関数       | 説明                               |
| ---------- | ---------------------------------- |
| `transcribe` | MLX Whisper を使用して音声ファイルを文字起こしします |
| `read_files` | ベースディレクトリ内のすべての音声ファイルをリストします |
