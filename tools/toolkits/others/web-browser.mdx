---
title: Web Browser Tools
description: WebBrowser Tools は、エージェントが Web ブラウザで URL を開くことを可能にします。
---

## 例

```python cookbook/tools/webbrowser_tools.py
from agno.agent import Agent
from agno.models.google import Gemini
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.webbrowser import WebBrowserTools

agent = Agent(
    model=Gemini("gemini-2.0-flash"),
    tools=[WebBrowserTools(), DuckDuckGoTools()],
    instructions=[
        "DuckDuckGo を使用して関連するウェブサイトやページを検索します"
        "ウェブブラウザを使用してサイトを開きます"
    ],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("MCP を説明する記事を見つけて、ウェブブラウザで開いてください。")
```

## ツールキット関数

| 関数 | 説明 |
| -------- | ----------- |
| `open_page` | Web ブラウザで URL を開きます |
