**YFinanceTools** は、エージェントが Yahoo Finance から株価データ、財務情報などにアクセスできるようにします。

## 前提条件

以下の例では、`yfinance` ライブラリが必要です。

```shell
pip install -U yfinance
```

## 例

以下のエージェントは、NVDA（Nvidia Corporation）の株価とアナリストの推奨事項に関する情報を提供します。

```python cookbook/tools/yfinance_tools.py
from agno.agent import Agent
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, stock_fundamentals=True)],
    show_tool_calls=True,
    description="あなたは、株価、アナリストの推奨事項、および企業のファンダメンタルズを調査する投資アナリストです。",
    instructions=["レスポンスはマークダウン形式でフォーマットし、可能な場合はテーブルを使用してデータを表示してください。"],
)
agent.print_response("NVDAの株価とアナリストの推奨事項を教えてください", markdown=True)
```

## Toolkit パラメータ

| パラメータ                 | タイプ   | デフォルト | 説明                                                                 |
| --------------------------- | ------- | -------- | -------------------------------------------------------------------- |
| `stock_price`               | `bool`  | `True`   | 現在の株価情報を取得する機能を有効にします。                               |
| `company_info`              | `bool`  | `False`  | 詳細な企業情報を取得する機能を有効にします。                               |
| `stock_fundamentals`        | `bool`  | `False`  | 株価のファンダメンタルデータを取得する機能を有効にします。                         |
| `income_statements`         | `bool`  | `False`  | 企業の損益計算書を取得する機能を有効にします。                               |
| `key_financial_ratios`      | `bool`  | `False`  | 企業の主要な財務比率を取得する機能を有効にします。                             |
| `analyst_recommendations`   | `bool`  | `False`  | 株価のアナリスト推奨事項を取得する機能を有効にします。                           |
| `company_news`              | `bool`  | `False`  | 企業に関連する最新ニュースを取得する機能を有効にします。                           |
| `technical_indicators`      | `bool`  | `False`  | 株価分析のテクニカル指標を取得する機能を有効にします。                           |
| `historical_prices`         | `bool`  | `False`  | 株価の履歴データを取得する機能を有効にします。                               |

## Toolkit 関数

| 関数                         | 説明                                         |
| ---------------------------- | -------------------------------------------- |
| `get_current_stock_price`    | この関数は、企業の現在の株価を取得します。                 |
| `get_company_info`           | この関数は、企業に関する詳細情報を取得します。               |
| `get_historical_stock_prices`| この関数は、企業の株価の履歴を取得します。                 |
| `get_stock_fundamentals`     | この関数は、株価のファンダメンタルデータを取得します。           |
| `get_income_statements`      | この関数は、企業の損益計算書を取得します。                 |
| `get_key_financial_ratios`   | この関数は、企業の主要な財務比率を取得します。               |
| `get_analyst_recommendations`| この関数は、株価のアナリスト推奨事項を取得します。           |
| `get_company_news`           | この関数は、企業に関連する最新ニュースを取得します。           |
| `get_technical_indicators`   | この関数は、株価分析のテクニカル指標を取得します。           |
