**LumaLabTools** は、エージェントが [Lumalabs プラットフォーム](https://lumalabs.ai/dream-machine) を使用してメディアを生成できるようにします。

## 前提条件

```shell
export LUMAAI_API_KEY=***
```

次の例では、`lumaai` ライブラリが必要です。Lumalabs クライアントをインストールするには、次のコマンドを実行します。

```shell
pip install -U lumaai
```

## 例

次のエージェントは、Lumalabs を使用してユーザーが要求したビデオを生成します。

```python cookbook/tools/lumalabs_tool.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.lumalab import LumaLabTools

luma_agent = Agent(
    name="Luma Video Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[LumaLabTools()],  # 作成した LumaLab ツールを使用
    markdown=True,
    debug_mode=True,
    show_tool_calls=True,
    instructions=[
        "あなたは Luma AI API を使用してビデオを生成するように設計されたエージェントです。",
        "ビデオは 2 つの方法で生成できます。",
        "1. テキストからビデオへの生成:",
        "2. 画像からビデオへの生成:",
        "ユーザーが画像 URL を提供するか、テキストプロンプトのみを提供するかによって、適切な関数を選択してください。",
        "ビデオは応答の下の UI に自動的に表示されるため、応答にビデオ URL を表示する必要はありません。",
    ],
    system_message=(
        "テキストからビデオへのリクエストには generate_video を使用し、画像ベースの生成には image_to_video を使用してください。特に要求されない限り、デフォルトのパラメータは変更しないでください。ビデオ生成ステータスについて常に明確なフィードバックを提供してください。"
    ),
)

luma_agent.run("Generate a video of a car in a sky")
```

## Toolkit Params

| パラメータ | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `api_key` | `str` | `None` | Lumalabs API キーを手動で指定する場合。 |

## Toolkit Functions

| 関数 | 説明 |
|---|---|
| `generate_video` | プロンプトからビデオを生成します。 |
| `image_to_video` | プロンプト、開始画像、終了画像からビデオを生成します。 |
