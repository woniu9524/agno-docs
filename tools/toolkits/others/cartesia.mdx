---

**CartesiaTools** 使 Agent 能够执行文本转语音、列出可用语音以及使用 [Cartesia](https://docs.cartesia.ai/) 进行语音本地化。

## 先决条件

以下示例需要 `cartesia` 库和 API 密钥。

```bash
pip install cartesia
```

```bash
export CARTESIA_API_KEY="your_api_key_here"
```

## 示例

```python
from agno.agent import Agent
from agno.tools.cartesia import CartesiaTools
from agno.utils.audio import write_audio_to_file

agent = Agent(
    name="Cartesia TTS Agent",
    description="一个使用 Cartesia 进行文本转语音的 Agent",
    tools=[CartesiaTools()],
    show_tool_calls=True,
)

response = agent.run(
    "使用文本转语音生成一个简单的问候语：说“欢迎来到 Cartesia，先进的语音合成平台。此语音由 Agent 生成。”"
)
if response.audio:
    write_audio_to_file(
        response.audio[0].base64_audio,
        filename="greeting.mp3",
    )
```

## 高级示例：翻译和语音本地化

此示例演示如何使用 CartesiaTools 翻译文本、分析情感、本地化新语音以及生成语音笔记。

```python
from textwrap import dedent
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.cartesia import CartesiaTools
from agno.utils.audio import write_audio_to_file

agent_instructions = dedent(
    """请按顺序执行以下步骤来翻译文本并生成本地化语音笔记：
    1. 从用户请求中识别要翻译的文本和目标语言。
    2. 将文本准确翻译成目标语言。
    3. 分析翻译后文本所传达的情感。
    4. 调用 `list_voices` 来检索可用语音。
    5. 选择一个与语言和情感匹配的基础语音。
    6. 调用 `localize_voice` 来创建新的本地化语音。
    7. 调用 `text_to_speech` 来生成最终音频。
    """
)

agent = Agent(
    name="情感感知翻译 Agent",
    description="使用 Cartesia TTS 工具翻译文本、分析情感、选择合适的语音、创建本地化语音并生成语音笔记（音频文件）。",
    instructions=agent_instructions,
    model=OpenAIChat(id="gpt-4o"),
    tools=[CartesiaTools(voice_localize_enabled=True)],
    show_tool_calls=True,
)

agent.print_response(
    "将“你好！你好吗？告诉我更多关于巴黎的天气？”翻译成法语并创建一个语音笔记。"
)
response = agent.run_response

if response.audio:
    write_audio_to_file(
        response.audio[0].base64_audio,
        filename="french_weather.mp3",
    )
```

## Toolkit 参数

| 参数                | 类型   | 默认值                               | 描述                                                                                             |
|---------------------|--------|--------------------------------------|--------------------------------------------------------------------------------------------------|
| `api_key`           | `str`  | `None`                               | 用于身份验证的 Cartesia API 密钥。如果未提供，则使用 `CARTESIA_API_KEY` 环境变量。                               |
| `model_id`          | `str`  | `sonic-2`                            | 用于文本转语音的模型 ID。                                                                                |
| `default_voice_id`  | `str`  | `78ab82d5-25be-4f7d-82b3-7ad64e5b85b2` | 用于文本转语音和语音本地化的默认语音 ID。                                                                      |
| `text_to_speech_enabled` | `bool` | `True`                               | 启用文本转语音功能。                                                                                     |
| `list_voices_enabled`    | `bool` | `True`                               | 启用列出可用语音功能。                                                                                   |
| `voice_localize_enabled` | `bool` | `False`                              | 启用语音本地化功能。                                                                                     |

## Toolkit 函数

| 函数         | 描述                                                                                                                                                            |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `list_voices`    | 列出 Cartesia 中可用的语音。                                                                                                                                    |
| `text_to_speech` | 将文本转换为语音。                                                                                                                                              |
| `localize_voice` | 创建一个新的本地化语音。                                                                                                                                          |
