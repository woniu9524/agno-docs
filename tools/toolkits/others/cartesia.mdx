**CartesiaTools** を使用すると、エージェントはテキスト読み上げ、利用可能な音声のリスト表示、[Cartesia](https://docs.cartesia.ai/) を使用した音声のローカライズを実行できます。

## 前提条件

以下の例では、`cartesia` ライブラリとAPIキーが必要です。

```bash
pip install cartesia
```

```bash
export CARTESIA_API_KEY="your_api_key_here"
```

## 例

```python
from agno.agent import Agent
from agno.tools.cartesia import CartesiaTools
from agno.utils.audio import write_audio_to_file

agent = Agent(
    name="Cartesia TTS Agent",
    description="Cartesiaを使用してテキスト読み上げを行うエージェント",
    tools=[CartesiaTools()],
    show_tool_calls=True,
)

response = agent.run(
    "テキスト読み上げを使用して簡単な挨拶を生成してください: \"Welcome to Cartesia, the advanced speech synthesis platform. This speech is generated by an agent.\""
)
if response.audio:
    write_audio_to_file(
        response.audio[0].base64_audio,
        filename="greeting.mp3",
    )
```

## 高度な例：翻訳と音声ローカライズ

この例では、テキストの翻訳、感情分析、新しい音声のローカライズ、およびCartesiaToolsを使用したボイスノートの生成方法を示します。

```python
from textwrap import dedent
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.cartesia import CartesiaTools
from agno.utils.audio import write_audio_to_file

agent_instructions = dedent(
    """テキストを翻訳し、ローカライズされたボイスノートを生成するために、以下の手順を順に実行してください：
    1. ユーザーのリクエストから、翻訳するテキストとターゲット言語を特定します。
    2. テキストをターゲット言語に正確に翻訳します。
    3. 翻訳されたテキストが伝える感情を分析します。
    4. `list_voices` を呼び出して、利用可能な音声を取得します。
    5. 言語と感情に一致するベースとなる音声を選択します。
    6. `localize_voice` を呼び出して、新しいローカライズされた音声を作成します。
    7. `text_to_speech` を呼び出して、最終的な音声を生成します。
    """
)

agent = Agent(
    name="Emotion-Aware Translator Agent",
    description="Cartesia TTSツールを使用して、テキストの翻訳、感情分析、適切な音声の選択、ローカライズされた音声の作成、およびボイスノート（音声ファイル）の生成を行います。",
    instructions=agent_instructions,
    model=OpenAIChat(id="gpt-4o"),
    tools=[CartesiaTools(voice_localize_enabled=True)],
    show_tool_calls=True,
)

agent.print_response(
    "「こんにちは！お元気ですか？パリの天気についてもっと教えてください？」をフランス語に翻訳し、ボイスノートを作成してください。"
)
response = agent.run_response

if response.audio:
    write_audio_to_file(
        response.audio[0].base64_audio,
        filename="french_weather.mp3",
    )
```

## Toolkitパラメータ

| パラメータ                | タイプ             | デフォルト                                   | 説明                                                                                             |
|--------------------------|------------------|--------------------------------------------|---------------------------------------------------------------------------------------------------|
| `api_key`                | `str`            | `None`                                     | 認証用のCartesia APIキー。提供されない場合は、`CARTESIA_API_KEY` 環境変数が使用されます。 |
| `model_id`               | `str`            | `sonic-2`                                  | テキスト読み上げに使用するモデルID。                                                         |
| `default_voice_id`       | `str`            | `78ab82d5-25be-4f7d-82b3-7ad64e5b85b2`      | テキスト読み上げおよびローカライズに使用するデフォルトの音声ID。                                 |
| `text_to_speech_enabled` | `bool`           | `True`                                     | テキスト読み上げ機能を有効にします。                                                             |
| `list_voices_enabled`    | `bool`           | `True`                                     | 利用可能な音声のリスト表示機能を有効にします。                                                   |
| `voice_localize_enabled` | `bool`           | `False`                                    | 音声ローカライズ機能を有効にします。                                                         |

## Toolkit関数

| 関数         | 説明                                                                                                                                                                 |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `list_voices`    | Cartesiaから利用可能な音声のリストを取得します。                 |
| `text_to_speech` | テキストを音声に変換します。  |
| `localize_voice` | 新しいローカライズされた音声を作成します。 |
