**ZendeskTools** を使用すると、エージェントは Zendesk API にアクセスして記事を検索できます。

## 前提条件

次の例では、`requests` ライブラリと認証情報が必要です。

```shell
pip install -U requests
```

```shell
export ZENDESK_USERNAME=***
export ZENDESK_PW=***
export ZENDESK_COMPANY_NAME=***
```

## 例

次のエージェントは、「ログイン方法」を検索するために Zendesk を実行し、応答を出力します。

```python cookbook/tools/zendesk_tools.py
from agno.agent import Agent
from agno.tools.zendesk import ZendeskTools

agent = Agent(tools=[ZendeskTools()], show_tool_calls=True)
agent.print_response("How do I login?", markdown=True)
```

## Toolkit パラメータ

| パラメータ      | タイプ  | デフォルト | 説明                                                                                              |
| -------------- | ----- | ------- | ------------------------------------------------------------------------------------------------- |
| `username`     | `str` | -       | 認証または識別の目的で使用されるユーザー名。                                                              |
| `password`     | `str` | -       | 認証目的でユーザー名に関連付けられたパスワード。                                                              |
| `company_name` | `str` | -       | ユーザーまたはアクセスされるデータに関連する会社名。                                                              |

## Toolkit 関数

| 関数         | 説明                                                                                              |
| -------------- | ------------------------------------------------------------------------------------------------- |
| `search_zendesk` | この関数は、指定された検索文字列に一致する Zendesk ヘルプセンターの記事を検索します。                                                              |
