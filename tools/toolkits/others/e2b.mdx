**E2BTools** は、Python、ファイル操作、Webサーバー機能をサポートする、安全なサンドボックス環境でのコード実行をエージェントに可能にします。

## 前提条件

E2Bツールには、`e2b_code_interpreter` PythonパッケージとE2B APIキーが必要です。

```shell
pip install e2b_code_interpreter
```

```shell
export E2B_API_KEY=your_api_key
```

## 例

以下の例は、安全なサンドボックスでPythonコードを実行できるエージェントを作成する方法を示しています。

```python cookbook/tools/e2b_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.e2b import E2BTools

e2b_tools = E2BTools(
    timeout=600,  # 10分間のタイムアウト（秒単位）
)

agent = Agent(
    name="Code Execution Sandbox",
    agent_id="e2b-sandbox",
    model=OpenAIChat(id="gpt-4o"),
    tools=[e2b_tools],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "あなたは、安全なE2Bサンドボックス環境を使用してPythonコードを作成および検証する専門家です。",
        "あなたの主な目的は次のとおりです:",
        "1. ユーザーのリクエストに基づいて、明確で効率的なPythonコードを作成する",
        "2. E2Bサンドボックスでコードを実行および検証する",
        "3. コード全体をユーザーと共有する（これが主なユースケースです）",
        "4. コードの仕組みについて詳細な説明を提供する",
        "",
        "これらのツールを使用できます:",
        "1. Pythonコードの実行 (run_python_code)",
        "2. サンドボックスへのファイルのアップロード (upload_file)",
        "3. サンドボックスからのファイルのダウンロード (download_file_from_sandbox)",
        "4. 可視化を画像アーティファクトとして生成および追加する (download_png_result)",
        "5. サンドボックス内のファイルを一覧表示する (list_files)",
        "6. ファイルの内容の読み書きを行う (read_file_content, write_file_content)",
        "7. Webサーバーを起動し、公開URLを取得する (run_server, get_public_url)",
        "8. サンドボックスのライフサイクルを管理する (set_sandbox_timeout, get_sandbox_status, shutdown_sandbox)",
    ],
)

# 例：フィボナッチ数を生成する
agent.print_response(
    "最初の10個のフィボナッチ数を生成し、その合計と平均を計算するPythonコードを記述してください"
)

```

## Toolkit パラメータ

| パラメータ            | タイプ   | デフォルト | 説明                                                              |
| -------------------- | ------- | ------- | ----------------------------------------------------------------- |
| `api_key`            | `str`  | `None`  | E2B APIキー。指定しない場合は、E2B_API_KEY環境変数を使用します。 |
| `run_code`           | `bool` | `True`  | `run_code`関数を登録するかどうか                                  |
| `upload_file`        | `bool` | `True`  | `upload_file`関数を登録するかどうか                               |
| `download_result`    | `bool` | `True`  | `download_result`関数を登録するかどうか                           |
| `filesystem`         | `bool` | `False` | ファイルシステム操作を登録するかどうか                            |
| `internet_access`    | `bool` | `False` | インターネットアクセス関数を登録するかどうか                      |
| `sandbox_management` | `bool` | `False` | サンドボックス管理関数を登録するかどうか                          |
| `timeout`            | `int`  | `300`   | サンドボックスのタイムアウト（秒単位）（デフォルト：5分）         |
| `sandbox_options`    | `dict` | `None`  | Sandboxコンストラクタに渡す追加オプション                       |
| `command_execution`  | `bool` | `False` | コマンド実行関数を登録するかどうか                                |

## Toolkit 関数

### コード実行

| 関数           | 説明                                           |
| -------------- | ---------------------------------------------- |
| `run_python_code`  | E2Bサンドボックス環境でPythonコードを実行する |

### ファイル操作

| 関数                     | 説明                                                              |
| ------------------------ | ----------------------------------------------------------------- |
| `upload_file`                | サンドボックスにファイルをアップロードする                                |
| `download_png_result`        | PNG画像をエージェントのImageArtifactとして追加する                     |
| `download_chart_data`        | 結果のインタラクティブチャートからチャートデータを抽出する                   |
| `download_file_from_sandbox` | サンドボックスからローカルシステムにファイルをダウンロードする                   |

### ファイルシステム操作

| 関数             | 説明                                                 |
| ---------------- | ---------------------------------------------------- |
| `list_files`         | サンドボックス内のパスにあるファイルとディレクトリを一覧表示する |
| `read_file_content`  | サンドボックスからファイルのコンテンツを読み取る                 |
| `write_file_content` | サンドボックス内のファイルにテキストコンテンツを書き込む           |
| `watch_directory`    | 指定された期間、ディレクトリの変更を監視する                   |

### コマンド実行

| 関数                | 説明                                           |
| ------------------- | ---------------------------------------------- |
| `run_command`           | サンドボックス環境でシェルコマンドを実行する             |
| `stream_command`        | シェルコマンドを実行し、その出力をストリーミングする     |
| `run_background_command`| シェルコマンドをバックグラウンドで実行する             |
| `kill_background_command`| バックグラウンドコマンドを終了する                   |

### インターネットアクセス

| 関数           | 説明                                                 |
| -------------- | ---------------------------------------------------- |
| `get_public_url`   | サンドボックスで実行中のサービスに公開URLを取得する       |
| `run_server`       | サンドボックスでサーバーを起動し、その公開URLを返す     |

### サンドボックス管理

| 関数                | 説明                                           |
| ------------------- | ---------------------------------------------- |
| `set_sandbox_timeout`   | サンドボックスのタイムアウトを更新する             |
| `get_sandbox_status`    | サンドボックスの現在のステータスを取得する           |
| `shutdown_sandbox`      | サンドボックスを直ちにシャットダウンする             |
| `list_running_sandboxes`| 実行中のすべてのサンドボックスを一覧表示する         |
