**GoogleMapTools** を使用すると、エージェントは場所ベースの操作のためにさまざまなGoogle Mapsサービスと連携できます。これには、場所の検索、経路案内、ジオコーディングなどが含まれます。

## 前提条件

以下の例では、`googlemaps` ライブラリとAPIキーが必要です。APIキーは [Google Cloud Console](https://console.cloud.google.com/projectselector2/google/maps-apis/credentials) から取得できます。

```shell
pip install googlemaps
```

```shell
export GOOGLE_MAPS_API_KEY=your_api_key_here
```

Google Cloud Console で以下のAPIを有効にする必要があります。
- Places API
- Directions API
- Geocoding API
- Address Validation API
- Distance Matrix API
- Elevation API
- Time Zone API

## 例

Google Mapsツールの基本的な使い方：

```python
from agno.agent import Agent
from agno.tools.google_maps import GoogleMapTools

agent = Agent(tools=[GoogleMapTools()], show_tool_calls=True)
agent.print_response("Find coffee shops in San Francisco")
```

その他の例については、[Google Maps Tools Examples](/examples/concepts/tools/others/google_maps) を参照してください。

## ツールキットのパラメータ

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `key` | `Optional[str]` | `None` | オプションのAPIキー。指定しない場合は、GOOGLE_MAPS_API_KEY 環境変数を使用します |
| `search_places` | `bool` | `True` | 場所検索機能を有効にします |
| `get_directions` | `bool` | `True` | 経路案内機能を有効にします |
| `validate_address` | `bool` | `True` | 住所検証機能を有効にします |
| `geocode_address` | `bool` | `True` | ジオコーディング機能を有効にします |
| `reverse_geocode` | `bool` | `True` | リバースジオコーディング機能を有効にします |
| `get_distance_matrix` | `bool` | `True` | 距離マトリックス機能を有効にします |
| `get_elevation` | `bool` | `True` | 標高取得機能を有効にします |
| `get_timezone` | `bool` | `True` | タイムゾーン取得機能を有効にします |

## ツールキットの関数

| 関数 | 説明 |
|----------|-------------|
| `search_places` | Google Maps Places API を使用して場所を検索します。パラメータ：検索クエリ用の `query` (str)。場所の詳細（名前、住所、電話番号、ウェブサイト、評価、営業時間など）を含む文字列化されたJSONを返します。 |
| `get_directions` | 場所間の経路案内を取得します。パラメータ：`origin` (str)、`destination` (str)、移動手段用のオプション `mode` (str)、回避する機能用のオプション `avoid` (List[str])。ルート情報が返されます。 |
| `validate_address` | 住所を検証します。パラメータ：`address` (str)、オプションの `region_code` (str)、オプションの `locality` (str)。住所検証の結果が返されます。 |
| `geocode_address` | 住所を座標に変換します。パラメータ：`address` (str)、オプションの `region` (str)。座標を含む場所情報が返されます。 |
| `reverse_geocode` | 座標を住所に変換します。パラメータ：`lat` (float)、`lng` (float)、オプションの `result_type` および `location_type` (List[str])。住所情報が返されます。 |
| `get_distance_matrix` | 場所間の距離を計算します。パラメータ：`origins` (List[str])、`destinations` (List[str])、オプションの `mode` (str) および `avoid` (List[str])。距離と所要時間のマトリックスが返されます。 |
| `get_elevation` | 場所の標高を取得します。パラメータ：`lat` (float)、`lng` (float)。標高データが返されます。 |
| `get_timezone` | 場所のタイムゾーンを取得します。パラメータ：`lat` (float)、`lng` (float)、オプションの `timestamp` (datetime)。タイムゾーン情報が返されます。 |

## レート制限

Google Maps API には、サービスおよび請求プランによって異なる使用制限と割り当てがあります。詳細については、[Google Maps Platform の料金](https://cloud.google.com/maps-platform/pricing) を参照してください。
