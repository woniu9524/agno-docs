**OpenBBTools** は、エージェントが株式や企業に関する情報を提供する機能を有効にします。

```python cookbook/tools/openbb_tools.py
from agno.agent import Agent
from agno.tools.openbb import OpenBBTools


agent = Agent(tools=[OpenBBTools()], debug_mode=True, show_tool_calls=True)

# 株価分析を示す例
agent.print_response(
    "Apple (AAPL) の現在の株価と主要情報を教えてください"
)

# 市場分析を示す例
agent.print_response(
    "今日の市場でトップの上げ銘柄は何ですか？"
)

# 経済指標を示す例
agent.print_response(
    "米国の最新のGDP成長率とインフレ率を示してください"
)
```

## Toolkit Params

| パラメータ         | タイプ   | デフォルト | 説明                                                                 |
| ------------------ | ------ | ------- | -------------------------------------------------------------------- |
| `read_article`     | `bool` | `True`  | 記事の全文を読み取る機能を有効にします。                               |
| `include_summary`  | `bool` | `False` | 全文コンテンツとともに記事の要約を含めるかどうかを指定します。           |
| `article_length`   | `int`  | -       | 処理または返される記事またはその要約の最大長。                         |

## Toolkit Functions

| 関数                | 説明                                                               |
| ------------------- | ------------------------------------------------------------------ |
| `get_stock_price`   | この関数は、株式シンボルまたはシンボルのリストの現在の株価を取得します。 |
| `search_company_symbol` | この関数は、企業の株式シンボルを検索します。                       |
| `get_price_targets` | この関数は、株式シンボルまたはシンボルのリストの価格ターゲットを取得します。 |
| `get_company_news`  | この関数は、株式シンボルまたはシンボルのリストの最新ニュースを取得します。 |
| `get_company_profile` | この関数は、株式シンボルまたはシンボルのリストの企業プロフィールを取得します。 |
