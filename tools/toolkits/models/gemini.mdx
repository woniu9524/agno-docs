`GeminiTools` は、エージェントが画像や動画を生成するために Google AI API サービスと対話できるツールのセットです。

## 前提条件

`GeminiTools` を使用する前に、`google-genai` ライブラリがインストールされ、認証情報が設定されていることを確認してください。

1. **ライブラリのインストール:**
   ```bash
   pip install google-genai agno
   ```

2. **認証情報の設定:**
   - Gemini API の場合:
     ```bash
     export GOOGLE_API_KEY="your-google-genai-api-key"
     ```
   - Vertex AI の場合:
     ```bash
     export GOOGLE_CLOUD_PROJECT="your-google-cloud-project-id"
     export GOOGLE_CLOUD_LOCATION="your-google-cloud-location"
     export GOOGLE_GENAI_USE_VERTEXAI=true
     ```

## 初期化

`GeminiTools` をインポートし、エージェントのツールリストに追加します。

```python
from agno.agent import Agent
from agno.tools.models.gemini import GeminiTools

agent = Agent(
    tools=[GeminiTools()],
    show_tool_calls=True,
)
```

## 使用例

GeminiTools はさまざまなタスクに使用できます。以下に例を示します。
### 画像生成

```python image_generation_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.models.gemini import GeminiTools
from agno.utils.media import save_base64_data

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[GeminiTools()],
    show_tool_calls=True,
)

agent.print_response(
    "サイバーパンクなサムライの芸術的なポートレートを雨の降る都市で作成してください",
)
response = agent.run_response
if response.images:
    save_base64_data(response.images[0].content, "tmp/cyberpunk_samurai.png")
```

### 動画生成

<Note>
動画生成には Vertex AI が必要です。
</Note>

```python video_generation_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.models.gemini import GeminiTools
from agno.utils.media import save_base64_data

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[GeminiTools(vertexai=True)],
    show_tool_calls=True,
    debug_mode=True,
)

agent.print_response(
    "ピアノを弾いている子猫の 5 秒間の動画を生成してください",
)
response = agent.run_response
if response.videos:
    for video in response.videos:
        save_base64_data(video.content, f"tmp/kitten_piano_{video.id}.mp4")
```



## ツールキット関数

| 関数             | 説明                                     |
| ---------------- | ---------------------------------------- |
| `generate_image` | テキストプロンプトに基づいて画像を生成します |
| `generate_video` | テキストプロンプトに基づいて動画を生成します |
