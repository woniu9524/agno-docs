`GroqTools` を使用すると、Agent は Groq API と対話して、高速な音声文字起こし、翻訳、テキスト読み上げ (TTS) を実行できます。

## 前提条件

`GroqTools` を使用する前に、`groq` ライブラリがインストールされており、Groq API キーが設定されていることを確認してください。

1.  **ライブラリのインストール:**
    ```bash
    pip install -U groq
    ```

2.  **API キーの設定:** [Groq Console](https://console.groq.com/keys) から API キーを取得し、環境変数として設定します。

    <CodeGroup>
    ```bash Mac
    export GROQ_API_KEY="your-groq-api-key"
    ```
    ```bash Windows
    setx GROQ_API_KEY "your-groq-api-key"
    ```
    </CodeGroup>

## 初期化

`GroqTools` をインポートし、Agent のツールリストに追加します。

```python
from agno.agent import Agent
from agno.tools.models.groq import GroqTools

agent = Agent(
    instructions=[
        "You are a helpful assistant that can transcribe audio, translate text and generate speech."
    ],
    tools=[GroqTools()],
    show_tool_calls=True,
)
```

## 使用例

### 1. 音声の文字起こし

この例では、URL でホストされている音声ファイルを文字起こしする方法を示します。

```python transcription_agent.py
import os
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.models.groq import GroqTools

audio_url = "https://agno-public.s3.amazonaws.com/demo_data/sample_conversation.wav"

agent = Agent(
    name="Groq Transcription Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[GroqTools()],
    show_tool_calls=True,
)

agent.print_response(f"Please transcribe the audio file located at '{audio_url}'")
```

### 2. 音声の翻訳と音声生成

この例では、音声ファイル (例: フランス語) を英語に翻訳し、その後、翻訳されたテキストから新しい音声ファイルを生成する方法を示します。

```python translation_agent.py
from pathlib import Path
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.models.groq import GroqTools
from agno.utils.media import save_base64_data

local_audio_path = "tmp/sample-fr.mp3"
output_path = Path("tmp/sample-en.mp3")
output_path.parent.mkdir(parents=True, exist_ok=True)

agent = Agent(
    name="Groq Translation Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[GroqTools()],
    show_tool_calls=True,
)

instruction = (
    f"Translate the audio file at '{local_audio_path}' to English. "
    f"Then, generate a new audio file using the translated English text."
)
agent.print_response(instruction)

response = agent.run_response
if response and response.audio:
    save_base64_data(response.audio[0].base64_audio, output_path)
```

初期化中に、文字起こし、翻訳、TTS に使用される基盤となる Groq モデルをカスタマイズできます。

```python
groq_tools = GroqTools(
    transcription_model="whisper-large-v3",
    translation_model="whisper-large-v3",
    tts_model="playai-tts",
    tts_voice="Chip-PlayAI"
)
```

## ツールキット関数

`GroqTools` ツールキットは、次の関数を提供します。

| 関数           | 説明                                                                   |
| -------------- | ---------------------------------------------------------------------- |
| `transcribe_audio` | Groq Whisper を使用して、ローカルファイルパスまたは公開 URL から音声を文字起こしします。 |
| `translate_audio`  | Groq を使用して、ローカルファイルパスまたは公開 URL から音声を英語に翻訳します。 |
| `generate_speech`  | Groq TTS を使用して、テキストから音声を生成します。                               |
