**AgentQLTools** は、エージェントが AgentQL API を使用してウェブサイトを閲覧およびスクレイピングできるようにします。

## 前提条件

以下の例では、`agentql` ライブラリと、[AgentQL](https://agentql.com/) から取得できる API トークンが必要です。

```shell
pip install -U agentql
```

```shell
export AGENTQL_API_KEY=***
```

## 例

以下 のエージェントは、ウェブブラウザを開き、ページからすべてのテキストをスクレイピングします。

```python cookbook/tools/agentql_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.agentql import AgentQLTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"), tools=[AgentQLTools()], show_tool_calls=True
)

agent.print_response("https://docs.agno.com/introduction", markdown=True)
```

<Note>
  AgentQL はブラウザインスタンスを開きます（閉じないでください）。サイトのスクレイピングを行います。
</Note>

## Toolkit Params

| パラメータ       | タイプ   | デフォルト | 説明                         |
| --------------- | ------- | -------- | ---------------------------- |
| `api_key`       | `str`  | `None`  | AgentQL の API キー          |
| `scrape`        | `bool` | `True`  | スクラップテキストツールを使用するかどうか |
| `agentql_query` | `str`  | `None`  | カスタム AgentQL クエリ      |

## Toolkit Functions

| 関数                | 説明                                       |
| ------------------- | ------------------------------------------ |
| `scrape_website`    | ウェブページからすべてのテキストをスクレイピングするために使用 |
| `custom_scrape_website` | カスタム `agentql_query` を使用してウェブページをスクレイピング |
