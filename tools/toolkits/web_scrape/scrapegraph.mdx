## 前提条件

以下の例では、`scrapegraph-py` ライブラリが必要です。

```shell
pip install -U scrapegraph-py
```

オプションで、ScrapeGraph の設定または特定のモデルが API キーを必要とする場合は、`SGAI_API_KEY` 環境変数を設定してください。

```shell
export SGAI_API_KEY="YOUR_SGAI_API_KEY"
```

## 例

以下のエージェントは、`ScrapeGraphTools` を使用して、`smartscraper` 機能を使用してウェブページから特定の情報を抽出します。

```python
from agno.agent import Agent
from agno.tools.scrapegraph import ScrapeGraphTools

agent = Agent(
    tools=[ScrapeGraphTools(smartscraper=True)],
    show_tool_calls=True,
)

agent.print_response("""
    "https://www.wired.com/category/science/ から以下の情報を smartscraper を使用して抽出してください:
- ニュース記事
- 見出し
- 画像
- リンク
- 著者
""",
)
```

## ツールキット関数

| 関数       | 説明                                                                                             |
| -------------- | ------------------------------------------------------------------------------------------------ |
| `smartscraper` | LLM と自然言語プロンプトを使用して、ウェブページから構造化されたデータを抽出します。結果の JSON 文字列またはエラーを返します。 |
| `markdownify`  | ウェブページをマークダウン形式に変換します。マークダウン文字列またはエラーを返します。                               |

## ツールキットパラメータ

これらのパラメータは、`ScrapeGraphTools` コンストラクタに渡されます。

| パラメータ      | タイプ            | デフォルト | 説明                                                                                                                             |
| -------------- | --------------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `api_key`      | `Optional[str]` | `None`  | ScrapeGraph サービス用の API キー。指定しない場合、`SGAI_API_KEY` 環境変数からデフォルト値が使用されます。                             |
| `smartscraper` | `bool`          | `True`  | `smartscraper` ツールを有効にします。                                                                                                         |
| `markdownify`  | `bool`          | `False` | `markdownify` ツールを有効にします。 |
