**Newspaper4k** は、Agent が Newspaper4k ライブラリを使用してニュース記事を読むことを可能にします。

## 前提条件

以下の例では、`newspaper4k` および `lxml_html_clean` ライブラリが必要です。

```shell
pip install -U newspaper4k lxml_html_clean
```

## 例

以下のエージェントは、記事を要約します: https://www.rockymountaineer.com/blog/experience-icefields-parkway-scenic-drive-lifetime。

```python cookbook/tools/newspaper4k_tools.py
from agno.agent import Agent
from agno.tools.newspaper4k import Newspaper4kTools

agent = Agent(tools=[Newspaper4kTools()], debug_mode=True, show_tool_calls=True)
agent.print_response("Please summarize https://www.rockymountaineer.com/blog/experience-icefields-parkway-scenic-drive-lifetime")
```

## Toolkit パラメータ

| パラメータ         | タイプ   | デフォルト | 説明                                                               |
| ------------------ | ------ | ------- | ------------------------------------------------------------------ |
| `read_article`     | `bool` | `True`  | 記事の全文を読む機能を使用可能にします。                               |
| `include_summary`  | `bool` | `False` | 全文と合わせて記事の要約を含めるかどうかを指定します。                   |
| `article_length`   | `int`  | -       | 処理または返される記事またはその要約の最大長。                         |

## Toolkit 関数

| 関数           | 説明                                     |
| -------------- | ---------------------------------------- |
| `get_article_data` | この関数は、記事の全文およびデータを読み取ります。 |
| `read_article`     | この関数は、記事の全文を読み取ります。       |
