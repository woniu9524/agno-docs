**FirecrawlTools** は、エージェントがウェブのクロールとスクレイピングタスクを実行できるようにします。

## 前提条件

以下の例では、`firecrawl-py` ライブラリと、[Firecrawl](https://firecrawl.dev) から取得できる API キーが必要です。

```shell
pip install -U firecrawl-py
```

```shell
export FIRECRAWL_API_KEY=***
```

## 例

以下のエージェントは、https://finance.yahoo.com/ からコンテンツをスクレイピングし、そのコンテンツの要約を返します。

```python cookbook/tools/firecrawl_tools.py
from agno.agent import Agent
from agno.tools.firecrawl import FirecrawlTools

agent = Agent(tools=[FirecrawlTools(scrape=False, crawl=True)], show_tool_calls=True, markdown=True)
agent.print_response("Summarize this https://finance.yahoo.com/")
```

## Toolkit Params

| パラメータ | タイプ | デフォルト | 説明 |
| --------- | ----------- | ------- | ------------------------------------------------------------- |
| `api_key` | `str` | `None` | 認証目的のオプションのAPIキー。`FIRECRAWL_API_KEY` 環境変数にフォールバックします。 |
| `formats` | `List[str]` | `None` | 操作に使用するフォーマットのオプションリスト。 |
| `limit` | `int` | `10` | 取得するアイテムの最大数。デフォルト値は10です。 |
| `poll_interval` | `int` | `30` | 結果をポーリングする間の秒単位の間隔。 |
| `scrape` | `bool` | `True` | スクレイピング機能を有効にします。デフォルトはTrueです。 |
| `crawl` | `bool` | `False` | クローリング機能を有効にします。デフォルトはFalseです。 |
| `mapping` | `bool` | `False` | ウェブサイトのマッピング機能を有効にします。 |
| `search` | `bool` | `False` | ウェブ検索機能を有効にします。 |
| `search_params` | `Dict[str, Any]` | `None` | 検索操作のオプションパラメータ。 |

## Toolkit Functions

| 関数 | 説明 |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `scrape_website` | Firecrawlを使用してウェブサイトをスクレイピングします。パラメータには、スクレイピングするURLを指定する `url` が含まれます。この関数は、指定された場合にオプションのフォーマットをサポートします。スクレイピングの結果をJSON形式で返します。 |
| `crawl_website` | Firecrawlを使用してウェブサイトをクロールします。パラメータには、クロールするURLを指定する `url` と、クロールする最大ページ数を定義するオプションの `limit` が含まれます。この関数はオプションのフォーマットをサポートし、クロール結果をJSON形式で返します。 |
| `map_website` | Firecrawlを使用してウェブサイトの構造をマッピングします。パラメータには、マッピングするURLを指定する `url` が含まれます。マッピング結果をJSON形式で返します。 |
| `search` | Firecrawlを使用してウェブ検索を実行します。パラメータには、検索用語の `query` と、最大結果数のオプションの `limit` が含まれます。検索結果をJSON形式で返します。 |
