title: Website Tools
---

**WebsiteTools** は、エージェントがウェブサイトを解析し、そのコンテンツをナレッジベースに追加できるようにします。

## 前提条件

以下の例では、`beautifulsoup4` ライブラリが必要です。

```shell
pip install -U beautifulsoup4
```

## 例

以下のエージェントは、ウェブサイトのコンテンツを読み取り、それをナレッジベースに追加します。

```python cookbook/tools/website_tools.py
from agno.agent import Agent
from agno.tools.website import WebsiteTools

agent = Agent(tools=[WebsiteTools()], show_tool_calls=True)
agent.print_response("Search web page: 'https://docs.agno.com/introduction'", markdown=True)
```

## Toolkit パラメータ

| パラメータ        | タイプ                   | デフォルト | 説明                                                                                                                               |
| ---------------- | ---------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| `knowledge_base` | `WebsiteKnowledgeBase` | -       | ウェブサイトに関連付けられたナレッジベースで、ウェブサイトのコンテンツにリンクされたさまざまなデータやリソースが含まれています。 |

## Toolkit 関数

| 関数                        | 説明                                                                                                                                                                                                                         |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `add_website_to_knowledge_base` | この関数は、ウェブサイトのコンテンツをナレッジベースに追加します。**注意:** ウェブサイトは `https://` で始まる必要があり、有効なウェブサイトである必要があります。この関数を使用して、インターネットから製品に関する情報を取得してください。 |
| `read_url`                      | この関数は、URL を読み取り、そのコンテンツを返します。                                                                                                                                                                          |
