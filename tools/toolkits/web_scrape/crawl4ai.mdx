---

**Crawl4aiTools** は、Crawl4ai ライブラリを使用してエージェントがウェブクロールおよびスクレイピングタスクを実行できるようにします。

## 前提条件

次の例では、`crawl4ai` ライブラリが必要です。

```shell
pip install -U crawl4ai
```

## 例

次のエージェントは、https://github.com/agno-agi/agno ウェブページからコンテンツをスクレイピングします。

```python cookbook/tools/crawl4ai_tools.py
from agno.agent import Agent
from agno.tools.crawl4ai import Crawl4aiTools

agent = Agent(tools=[Crawl4aiTools(max_length=None)], show_tool_calls=True)
agent.print_response("Tell me about https://github.com/agno-agi/agno.")
```

## Toolkit パラメータ

| パラメータ    | タイプ  | デフォルト | 説明                                                                                             |
| ------------ | ----- | ------- | ------------------------------------------------------------------------------------------------ |
| `max_length` | `int` | `1000`  | 返されるウェブページからのテキストの最大長を指定します。                                                               |

## Toolkit 関数

| 関数      | 説明                                                                                                                                                                                                    |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `web_crawler` | crawl4ai の WebCrawler を使用してウェブサイトをクロールします。パラメータには、クロールする URL の 'url' と、抽出コンテンツの長さを制限するためのオプションの 'max_length' が含まれます。'max_length' のデフォルト値は 1000 です。 |
