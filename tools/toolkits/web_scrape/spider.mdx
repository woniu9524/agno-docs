**SpiderTools** は、LLM 向けのデータを返すオープンソースの Web スクレイパー＆クローラーです。Spider を使用するには、[Spider ダッシュボード](https://spider.cloud)から API キーが必要です。

## 前提条件

以下の例では、`spider-client` ライブラリが必要です。

```shell
pip install -U spider-client
```

## 例

以下のエージェントは、USA の最新ニュースを取得するための検索クエリを実行し、最初の検索結果をスクレイピングします。エージェントは、スクレイピングされたデータをマークダウン形式で返します。

```python cookbook/tools/spider_tools.py
from agno.agent import Agent
from agno.tools.spider import SpiderTools

agent = Agent(tools=[SpiderTools()])
agent.print_response('Can you scrape the first search result from a search on "news in USA"?', markdown=True)
```

## Toolkit Params

| パラメータ     | タイプ  | デフォルト | 説明                                           |
| ------------- | ----- | ------- | ---------------------------------------------- |
| `max_results` | `int` | -       | 返される検索結果の最大数                         |
| `url`         | `str` | -       | スクレイピングまたはクロールされる URL           |

## Toolkit Functions

| 関数     | 説明                               |
| -------- | ---------------------------------- |
| `search` | 指定されたクエリで Web を検索します。 |
| `scrape` | 指定された URL をスクレイピングします。 |
| `crawl`  | 指定された URL をクロールします。     |
