---
title: Jina Reader
---

**JinaReaderTools** は、Agent が Jina を使用して Web 検索タスクを実行できるようにします。

## 前提条件

以下の例では、`jina` ライブラリが必要です。

```shell
pip install -U jina
```

## 例

以下のエージェントは、Jina API を使用して https://github.com/AgnoAgi のコンテンツを要約します。

```python cookbook/tools/jinareader.py
from agno.agent import Agent
from agno.tools.jina import JinaReaderTools

agent = Agent(tools=[JinaReaderTools()])
agent.print_response("Summarize: https://github.com/AgnoAgi")
```

## Toolkit Params

| パラメータ            | タイプ  | デフォルト | 説明                                                                 |
| -------------------- | ----- | ------- | -------------------------------------------------------------------- |
| `api_key`            | `str` | -       | 認証用の API キー。設定から取得されます。                                 |
| `base_url`           | `str` | -       | API のベース URL。設定から取得されます。                                   |
| `search_url`         | `str` | -       | 検索クエリに使用される URL。設定から取得されます。                             |
| `max_content_length` | `int` | -       | 許可されるコンテンツの最大長。設定から取得されます。                           |

## Toolkit Functions

| 関数       | 説明                                                                                                                                                                                          |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `read_url` | Jina Reader API を使用して指定された URL のコンテンツを読み取ります。パラメータには、読み取る URL を指定する `url` が含まれます。リクエストが失敗した場合、切り捨てられたコンテンツまたはエラーメッセージを返します。 |
| `search_query` | 指定されたクエリに基づいて Jina Reader API を使用して Web 検索を実行します。パラメータには、検索用語を指定する `query` が含まれます。リクエストが失敗した場合、切り捨てられた検索結果またはエラーメッセージを返します。 |
