**Gmail** を使用すると、Agent は Gmail と対話して、メールの読み取り、検索、送信、管理を行うことができます。

## 前提条件

Gmail ツールキットには、Google API クライアントライブラリと適切な認証設定が必要です。必要な依存関係をインストールします。

```shell
pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib
```

Google Cloud の認証情報も設定する必要があります。

1. [Google Cloud Console](https://console.cloud.google.com) にアクセスします。
2. プロジェクトを作成するか、既存のプロジェクトを選択します。
3. Gmail API を有効にします。
4. OAuth 2.0 認証情報を作成します。
5. 環境変数を設定します。

```shell
export GOOGLE_CLIENT_ID=your_client_id_here
export GOOGLE_CLIENT_SECRET=your_client_secret_here
export GOOGLE_PROJECT_ID=your_project_id_here
export GOOGLE_REDIRECT_URI=http://localhost  # デフォルト値
```

## 例

```python cookbook/tools/gmail_tools.py
from agno.agent import Agent
from agno.tools.gmail import GmailTools

agent = Agent(tools=[GmailTools()], show_tool_calls=True)
agent.print_response("Show me my latest 5 unread emails", markdown=True)
```

## ツールキットのパラメータ

| パラメータ              | タイプ    | デフォルト | 説明                                                 |
| --------------------- | ------- | ------- | ----------------------------------------------------------- |
| `get_latest_emails`   | `bool`  | `True`  | 受信トレイから最新のメールを取得できるようにします。                  |
| `get_emails_from_user`| `bool`  | `True`  | 特定の送信者からのメールを取得できるようにします。                   |
| `get_unread_emails`   | `bool`  | `True`  | 未読メールを取得できるようにします。                             |
| `get_starred_emails`  | `bool`  | `True`  | スター付きメールを取得できるようにします。                         |
| `get_emails_by_context`| `bool` | `True`  | コンテキストでメールを検索できるようにします。                     |
| `get_emails_by_date`  | `bool`  | `True`  | 日付範囲内のメールを取得できるようにします。                       |
| `create_draft_email`  | `bool`  | `True`  | メールドラフトを作成できるようにします。                           |
| `send_email`          | `bool`  | `True`  | メールをすぐに送信できるようにします。                             |
| `search_emails`       | `bool`  | `True`  | 自然言語クエリを使用してメールを検索できるようにします。                 |

## ツールキットの関数

| 関数               | 説明                                                 |
| --------------------- | ----------------------------------------------------------- |
| `get_latest_emails`   | ユーザーの受信トレイから最新の X 件のメールを取得します。              |
| `get_emails_from_user`| 特定の送信者から X 件のメールを取得します。                   |
| `get_unread_emails`   | 最新の X 件の未読メールを取得します。                            |
| `get_starred_emails`  | スター付きメールを X 件取得します。                            |
| `get_emails_by_context`| 特定のコンテキストに一致するメールを X 件取得します。        |
| `get_emails_by_date`  | 特定の日付範囲内のメールを取得します。                   |
| `create_draft_email`  | メールドラフトを作成して保存します。                            |
| `send_email`          | メールをすぐに送信します。                                 |
| `search_emails`       | 自然言語クエリを使用してメールを検索します。                 |
