## 前提条件

以下の例では、`slack-sdk` ライブラリが必要です。

```shell
pip install openai slack-sdk
```

Slack トークンを [こちら](https://api.slack.com/tutorials/tracks/getting-a-token) から取得してください。

```shell
export SLACK_TOKEN=***
```

## 例

以下のエージェントは、Slack を使用してチャンネルにメッセージを送信し、すべてのチャンネルをリストし、特定のチャンネルのメッセージ履歴を取得します。

```python cookbook/tools/slack_tools.py
import os

from agno.agent import Agent
from agno.tools.slack import SlackTools

slack_tools = SlackTools()

agent = Agent(tools=[slack_tools], show_tool_calls=True)

# 例 1: Slack チャンネルにメッセージを送信する
agent.print_response("Send a message 'Hello from Agno!' to the channel #general", markdown=True)

# 例 2: Slack ワークスペース内のすべてのチャンネルをリストする
agent.print_response("List all channels in our Slack workspace", markdown=True)

# 例 3: チャンネル ID を使用して特定のチャンネルのメッセージ履歴を取得する
agent.print_response("Get the last 10 messages from the channel 1231241", markdown=True)

```

## Toolkit パラメータ

| パラメータ             | タイプ   | デフォルト | 説明                                                               |
| --------------------- | ------ | ------- | ------------------------------------------------------------------ |
| `token`               | `str`  | -       | 認証用の Slack API トークン                                          |
| `send_message`        | `bool` | `True`  | Slack チャンネルへのメッセージ送信機能を有効にします                 |
| `list_channels`       | `bool` | `True`  | 利用可能な Slack チャンネルをリストする機能を有効にします            |
| `get_channel_history` | `bool` | `True`  | チャンネルからメッセージ履歴を取得する機能を有効にします             |

## Toolkit 関数

| 関数              | 説明                                      |
| ----------------- | ----------------------------------------- |
| `send_message`    | 指定された Slack チャンネルにメッセージを送信します |
| `list_channels`   | Slack ワークスペース内のすべてのチャンネルをリストします |
| `get_channel_history` | 指定されたチャンネルからメッセージ履歴を取得します |
