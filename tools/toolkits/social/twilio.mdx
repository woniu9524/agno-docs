title: Twilio
---

**TwilioTools** 使代理能够与 [Twilio](https://www.twilio.com/docs) 服务进行交互，例如发送短信、检索通话详情和列出消息。

## 先决条件

以下示例需要 `twilio` 库和相应的 Twilio 凭证，这些凭证可从[此处](https://www.twilio.com/console)获取。

```shell
pip install twilio
```

设置以下环境变量：

```shell
export TWILIO_ACCOUNT_SID=***
export TWILIO_AUTH_TOKEN=***
```

## 示例

以下代理将使用 Twilio 发送短信：

```python
from agno.agent import Agent
from agno.tools.twilio import TwilioTools

agent = Agent(
    instructions=[
        "Use your tools to send SMS using Twilio.",
    ],
    tools=[TwilioTools(debug=True)],
    show_tool_calls=True,
)

agent.print_response("Send an SMS to +1234567890", markdown=True)
```

## Toolkit 参数

| 名称          | 类型            | 默认值 | 描述                                       |
| ------------- | --------------- | ------- | ------------------------------------------------- |
| `account_sid` | `Optional[str]` | `None`  | 用于身份验证的 Twilio 账户 SID。            |
| `auth_token`  | `Optional[str]` | `None`  | 用于身份验证的 Twilio 认证令牌。             |
| `api_key`     | `Optional[str]` | `None`  | 用于备用身份验证的 Twilio API 密钥。    |
| `api_secret`  | `Optional[str]` | `None`  | 用于备用身份验证的 Twilio API 密钥。 |
| `region`      | `Optional[str]` | `None`  | 可选的 Twilio 区域（例如 `au1`）。             |
| `edge`        | `Optional[str]` | `None`  | 可选的 Twilio 边缘位置（例如 `sydney`）。   |
| `debug`       | `bool`          | `False` | 启用调试日志记录以进行故障排除。         |

## Toolkit 函数

| 函数           | 描述                                                                                                                                                                 |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `send_sms`         | 向收件人发送短信。需要收件人电话号码、发件人号码（Twilio）和消息正文。成功时返回消息 SID，失败时返回错误消息。          |
| `get_call_details` | 使用通话 SID 检索通话详情。需要通话 SID，并返回包含通话详情（例如状态、持续时间）的字典。                                          |
| `list_messages`    | 列出最近的短信消息。需要返回的消息数量限制（默认为 20）。返回消息详情列表（例如 SID、发件人、收件人、正文、状态）。 |
