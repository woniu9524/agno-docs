title: Twilio
---

**TwilioTools** は、エージェントがSMSの送信、通話詳細の取得、メッセージの一覧表示などの[Twilio](https://www.twilio.com/docs)サービスとやり取りできるようにします。

## 前提条件

以下の例では、`twilio`ライブラリと適切なTwilio認証情報が必要です。これらは[こちら](https://www.twilio.com/console)から取得できます。

```shell
pip install twilio
```

以下の環境変数を設定してください。

```shell
export TWILIO_ACCOUNT_SID=***
export TWILIO_AUTH_TOKEN=***
```

## 例

以下のエージェントは、Twilioを使用してSMSメッセージを送信します。

```python
from agno.agent import Agent
from agno.tools.twilio import TwilioTools

agent = Agent(
    instructions=[
        "Use your tools to send SMS using Twilio.",
    ],
    tools=[TwilioTools(debug=True)],
    show_tool_calls=True,
)

agent.print_response("Send an SMS to +1234567890", markdown=True)
```

## Toolkitパラメータ

| 名前          | タイプ            | デフォルト | 説明                                       |
| ------------- | --------------- | ------- | ------------------------------------------------- |
| `account_sid` | `Optional[str]` | `None`  | 認証用のTwilioアカウントSID。            |
| `auth_token`  | `Optional[str]` | `None`  | 認証用のTwilio認証トークン。             |
| `api_key`     | `Optional[str]` | `None`  | 代替認証用のTwilio APIキー。    |
| `api_secret`  | `Optional[str]` | `None`  | 代替認証用のTwilio APIシークレット。 |
| `region`      | `Optional[str]` | `None`  | オプションのTwilioリージョン（例：`au1`）。             |
| `edge`        | `Optional[str]` | `None`  | オプションのTwilioエッジロケーション（例：`sydney`）。   |
| `debug`       | `bool`          | `False` | トラブルシューティングのためのデバッグログを有効にします。         |

## Toolkit関数

| 関数           | 説明                                                                                                                                                                 |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `send_sms`         | 受信者にSMSを送信します。受信者の電話番号、送信者番号（Twilio）、メッセージ本文を受け取ります。成功した場合はメッセージSID、失敗した場合はエラーメッセージを返します。          |
| `get_call_details` | 通話SIDを使用して通話の詳細を取得します。通話SIDを受け取り、通話の詳細（例：ステータス、期間）を含む辞書を返します。                                          |
| `list_messages`    | 最近のSMSメッセージを一覧表示します。返されるメッセージの数（デフォルトは20）の制限を受け取ります。メッセージの詳細（例：SID、送信者、受信者、本文、ステータス）のリストを返します。 |
