**TelegramTools** は、Telegram Bot API を使用して Telegram チャットにメッセージを送信できる Agent を有効にします。

## 前提条件

```shell
pip install -U agno httpx
```

```shell
export TELEGRAM_TOKEN=***
```

## 例

以下の Agent は、Telegram チャットにメッセージを送信します。

```python cookbook/tools/tavily_tools.py
from agno.agent import Agent
from agno.tools.telegram import TelegramTools

# トークンと chat_id の取得方法:
# 1. Telegram の BotFather で新しいボットを作成します。 https://core.telegram.org/bots/features#creating-a-new-bot
# 2. BotFather からトークンを取得します。
# 3. ボットにメッセージを送信します。
# 4. 次の URL にアクセスして chat_id を取得します:
#    https://api.telegram.org/bot/<your-bot-token>/getUpdates

telegram_token = "<enter-your-bot-token>"
chat_id = "<enter-your-chat-id>"

agent = Agent(
    name="telegram",
    tools=[TelegramTools(token=telegram_token, chat_id=chat_id)],
)

agent.print_response("Send message to telegram chat a paragraph about the moon")
```

## Toolkit パラメータ

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `token` | `Optional[str]` | `None` | Telegram Bot API トークン。指定しない場合は、TELEGRAM_TOKEN 環境変数がチェックされます。 |
| `chat_id` | `Union[str, int]` | - | メッセージを送信するチャットの ID。 |

## Toolkit 関数

| 関数 | 説明 |
|----------|-------------|
| `send_message` | 指定された Telegram チャットにメッセージを送信します。メッセージ文字列を入力として受け取り、API レスポンスをテキストとして返します。エラーが発生した場合は、エラーメッセージを返します。 |
