**XTools** 允许代理与 X 进行交互，提供发布、消息和搜索推文的功能。

## 先决条件

安装所需的库：

```shell
pip install tweepy
```

<Info>Tweepy 是一个用于与 X API 交互的 Python 库。</Info>

## 设置

1. **创建 X 开发者账户**
   - 访问 [developer.x.com](https://developer.x.com) 并申请开发者访问权限
   - 在您的开发者门户中创建一个新的项目和应用

2. **生成 API 凭证**
   - 导航到您应用的“密钥和令牌”部分
   - 生成并复制这些凭证：
     - API 密钥和密钥
     - Bearer Token
     - 访问令牌和密钥

3. **配置环境**
   ```shell
   export X_CONSUMER_KEY=your_api_key
   export X_CONSUMER_SECRET=your_api_secret
   export X_ACCESS_TOKEN=your_access_token
   export X_ACCESS_TOKEN_SECRET=your_access_token_secret
   export X_BEARER_TOKEN=your_bearer_token
   ```

## 示例

```python cookbook/tools/x_tools.py
from agno.agent import Agent
from agno.tools.x import XTools

# 初始化 X 工具包
x_tools = XTools(
    wait_on_rate_limit=True # 达到速率限制时重试
)

# 创建一个配备 X 工具包的代理
agent = Agent(
    instructions=[
        "使用 X 工具以授权用户的身份进行交互",
        "在被要求创建帖子时生成适当的内容",
        "仅在明确指示时发布内容",
        "遵守 X 的使用政策和速率限制",
    ],
    tools=[x_tools],
    show_tool_calls=True,
)

# 搜索帖子
agent.print_response("搜索关于 AI 代理的最新帖子", markdown=True)

# 创建并发布一条推文
agent.print_response("创建一条关于 AI 伦理的帖子", markdown=True)

# 获取用户时间线
agent.print_response("获取我的时间线", markdown=True)

# 回复帖子
agent.print_response(
    "能否回复这条 [帖子 ID] 帖子，以通用消息的形式说明这个项目有多棒：https://x.com/AgnoAgi",
    markdown=True,
)

# 获取用户信息
agent.print_response("能否检索关于此用户的信息 https://x.com/AgnoAgi ", markdown=True)

# 发送私信
agent.print_response(
    "向用户 @AgnoAgi 发送私信，告诉他们我想更多地了解他们，并附上他们社区的链接。",
    markdown=True,
)

# 获取用户个人资料
agent.print_response("获取我的 X 个人资料", markdown=True)
```

<Note> 查看 [Tweet Analysis Agent](/examples/agents/tweet-analysis-agent) 获取更高级的示例。</Note>

## 工具包参数

| 参数               | 类型                | 默认值 | 描述                                                     |
| ------------------- | ------------------- | ------- | -------------------------------------------------------- |
| `bearer_token`      | `str`     | `None`  | 用于身份验证的 Bearer token                                |
| `consumer_key`      | `str`     | `None`  | 用于身份验证的 Consumer key                                |
| `consumer_secret`   | `str`     | `None`  | 用于身份验证的 Consumer secret                             |
| `access_token`      | `str`     | `None`  | 用于身份验证的 Access token                                |
| `access_token_secret` | `str`     | `None`  | 用于身份验证的 Access token secret                         |
| `include_post_metrics` | `bool`              | `False` | 在搜索结果中包含帖子指标（点赞、转推等）                   |
| `wait_on_rate_limit` | `bool`              | `False` | 达到速率限制时重试                                       |

## 工具包函数

| 函数            | 描述                                 |
| ------------------- | ------------------------------------ |
| `create_post`      | 创建并发布新帖子                   |
| `reply_to_post`    | 回复现有帖子                       |
| `send_dm`           | 向 X 用户发送私信                  |
| `get_user_info`     | 检索 X 用户的信息                  |
| `get_home_timeline` | 获取已认证用户的首页时间线         |
| `search_posts`      | 搜索推文                           |
