**XTools** は、エージェントがXとやり取りできるようにするもので、ツイートの投稿、メッセージ送信、検索などの機能を提供します。

## 前提条件

必要なライブラリをインストールします。

```shell
pip install tweepy
```

<Info>Tweepyは、X APIとやり取りするためのPythonライブラリです。</Info>

## セットアップ

1. **X開発者アカウントの作成**
   - [developer.x.com](https://developer.x.com) にアクセスし、開発者アクセスを申請します。
   - 開発者ポータルで新しいプロジェクトとアプリを作成します。

2. **API認証情報の発行**
   - アプリの「Keys and tokens」セクションに移動します。
   - 次の認証情報を生成してコピーします。
     - APIキーとシークレット
     - ベアラートークン
     - アクセストークンとシークレット

3. **環境設定**
   ```shell
   export X_CONSUMER_KEY=your_api_key
   export X_CONSUMER_SECRET=your_api_secret
   export X_ACCESS_TOKEN=your_access_token
   export X_ACCESS_TOKEN_SECRET=your_access_token_secret
   export X_BEARER_TOKEN=your_bearer_token
   ```

## 例

```python cookbook/tools/x_tools.py
from agno.agent import Agent
from agno.tools.x import XTools

# Xツールキットの初期化
x_tools = XTools(
    wait_on_rate_limit=True # レート制限に達した場合に再試行
) 

# Xツールキットを装備したエージェントの作成
agent = Agent(
    instructions=[
        "承認されたユーザーとしてやり取りするためにXツールを使用する",
        "投稿を作成するように求められた場合は、適切なコンテンツを生成する",
        "明示的に指示された場合にのみ投稿する",
        "Xの利用ポリシーとレート制限を尊重する",
    ],
    tools=[x_tools],
    show_tool_calls=True,
)

# 投稿の検索
agent.print_response("AIエージェントに関する最近の投稿を検索する", markdown=True)

# ツイートの作成と投稿
agent.print_response("AI倫理に関する投稿を作成する", markdown=True)

# ユーザータイムラインの取得
agent.print_response("私のタイムラインを取得する", markdown=True)

# 投稿への返信
agent.print_response(
    "この[投稿ID]の投稿に、このプロジェクトがいかに素晴らしいかという一般的なメッセージとして返信できますか？: https://x.com/AgnoAgi",
    markdown=True,
)

# ユーザー情報の取得
agent.print_response("このユーザーに関する情報を取得できますか？ https://x.com/AgnoAgi ", markdown=True)

# ダイレクトメッセージの送信
agent.print_response(
    "ユーザー@AgnoAgiに、彼らについてもっと知りたいことと、彼らのコミュニティへのリンクを伝えるダイレクトメッセージを送信してください。",
    markdown=True,
)

# ユーザープロフィールの取得
agent.print_response("私のXプロフィールを取得する", markdown=True)
```

<Note> より高度な例については、[ツイート分析エージェント](/examples/agents/tweet-analysis-agent) を参照してください。 </Note>

## ツールキットのパラメータ

| パラメータ               | タイプ                | デフォルト | 説明                                       |
| ----------------------- | ------------------- | ------- | ------------------------------------------ |
| `bearer_token`          | `str`     | `None`  | 認証用のベアラートークン                               |
| `consumer_key`          | `str`     | `None`  | 認証用のコンシューマーキー                               |
| `consumer_secret`       | `str`     | `None`  | 認証用のコンシューマーシークレット                           |
| `access_token`          | `str`     | `None`  | 認証用のアクセストークン                               |
| `access_token_secret`   | `str`     | `None`  | 認証用のアクセストークンシークレット                         |
| `include_post_metrics`  | `bool`              | `False` | 検索結果に投稿のメトリクス（いいね、リツイートなど）を含める |
| `wait_on_rate_limit`    | `bool`              | `False` | レート制限に達した場合に再試行する                         |

## ツールキットの関数

| 関数            | 説明                                 |
| ------------------- | ------------------------------------------- |
| `create_post`      | 新しい投稿を作成して投稿する               |
| `reply_to_post`    | 既存の投稿に返信する                |
| `send_dm`           | Xユーザーにダイレクトメッセージを送信する    |
| `get_user_info`     | Xユーザーの情報を取得する  |
| `get_home_timeline` | 認証済みユーザーのホームタイムラインを取得する |
| `search_posts`      | ツイートを検索する                          |
