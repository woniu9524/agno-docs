title: Discord

**DiscordTools** を使用すると、エージェントはメッセージの送信、メッセージ履歴の読み取り、チャンネルの管理、メッセージの削除を Discord で行うことができます。

## 前提条件

以下の例では、[こちら](https://discord.com/developers/applications) から取得できる Discord ボットトークンが必要です。

```shell
export DISCORD_BOT_TOKEN=***
```

## 例

```python cookbook/tools/discord.py
from agno.agent import Agent
from agno.tools.discord import DiscordTools

agent = Agent(
    tools=[DiscordTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Send 'Hello World!' to channel 1234567890", markdown=True)
```
## Toolkit のパラメータ

| パラメータ                   | タイプ    | デフォルト | 説明                                                                           |
| --------------------------- | ------- | ------- | ------------------------------------------------------------------------------------- |
| `bot_token`                 | `str`   | -  | 認証用の Discord ボットトークン。                                                 |
| `enable_messaging`          | `bool`  | `True`  | チャンネルへのメッセージ送信を有効にするかどうか。                                       |
| `enable_history`            | `bool`  | `True`  | チャンネルからのメッセージ履歴の取得を有効にするかどうか。                           |
| `enable_channel_management` | `bool`  | `True`  | チャンネル情報の取得やチャンネルの一覧表示を有効にするかどうか。                         |
| `enable_message_management` | `bool`  | `True`  | チャンネルからのメッセージ削除を有効にするかどうか。                                    |

## Toolkit の関数

| 関数                                          | 説明                                                                                                   |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `send_message`     | 指定されたチャンネルにメッセージを送信します。成功またはエラーメッセージを返します。                                    |
| `get_channel_info`               | 指定されたチャンネルの情報を取得します。チャンネル情報を JSON 文字列として返します。                    |
| `list_channels`                    | 指定されたサーバー（ギルド）のすべてのチャンネルを一覧表示します。チャンネルのリストを JSON として返します。                        |
| `get_channel_messages`| 指定されたチャンネルからメッセージ履歴を取得します。メッセージを JSON 文字列として返します。                         |
| `delete_message`| 指定されたチャンネルから特定のメッセージ ID を持つメッセージを削除します。成功またはエラーメッセージを返します。                 |
