**WebexTools** を使用すると、エージェントは Cisco Webex と対話し、メッセージの送信やルームのリスト表示が可能になります。

## 前提条件

次の例では、`webexpythonsdk` ライブラリと Webex アクセストークンが必要です。これは [Webex Developer Portal](https://developer.webex.com/docs/bots) から取得できます。

Webex を開始するには：

1.  **Webex ボットを作成する：**
    *   [Developer Portal](https://developer.webex.com/) にアクセスします。
    *   「My Webex Apps」→「Create a Bot」に移動します。
    *   ボットの詳細を入力し、「Add Bot」をクリックします。

2.  **アクセストークンを取得する：**
    *   ボット作成後に表示されるトークンをコピーします。
    *   または、「My Webex Apps」→「Edit Bot」から再生成します。
    *   `WEBEX_ACCESS_TOKEN` 環境変数として設定します。

3.  **ボットを Webex に追加する：**
    *   Webex を起動し、ボットをスペースに追加します。
    *   ボットのメールアドレス（例：test@webex.bot）を使用します。

```shell
pip install webexpythonsdk
```

```shell
export WEBEX_ACCESS_TOKEN=your_access_token_here
```

## 例

次のエージェントは、すべてのスペースをリスト表示し、Webex を使用してメッセージを送信します。

```python cookbook/tools/webex_tool.py
from agno.agent import Agent
from agno.tools.webex import WebexTools

agent = Agent(tools=[WebexTools()], show_tool_calls=True)

# Webex のすべてのスペースをリスト表示する
agent.print_response("List all space on our Webex", markdown=True)

# Webex のスペースにメッセージを送信する
agent.print_response(
    "Send a funny ice-breaking message to the webex Welcome space", markdown=True
)
```

## Toolkit パラメータ

| パラメータ      | タイプ   | デフォルト | 説明                                                                                             |
| -------------- | ------- | ------- | ------------------------------------------------------------------------------------------------ |
| `access_token` | `str`   | `None`  | 認証用の Webex アクセストークン。提供されない場合は、`WEBEX_ACCESS_TOKEN` 環境変数を使用します。 |
| `send_message` | `bool`  | `True`  | Webex スペースへのメッセージ送信を有効にします。                                                   |
| `list_rooms`   | `bool`  | `True`  | Webex スペース/ルームのリスト表示を有効にします。                                                  |

## Toolkit 関数

| 関数         | 説明                                                                                                                               |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| `send_message` | Webex ルームにメッセージを送信します。パラメータ：対象ルームの `room_id` (str)、メッセージの `text` (str)。                               |
| `list_rooms`   | ID、タイトル、タイプ、表示設定を含む詳細とともに、利用可能なすべての Webex ルーム/スペースをリスト表示します。                                                              |
