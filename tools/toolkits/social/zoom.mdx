**Zoom** を使用すると、Agent は Zoom と対話して、会議のスケジュール設定、録画の管理、および Zoom API を介したさまざまな会議関連の操作を処理できます。このツールキットは、Zoom のサーバー間 OAuth 認証を使用して安全な API アクセスを実現します。

## 前提条件

Zoom ツールキットには、次のセットアップが必要です。

1. 必要な依存関係をインストールします。
```shell
pip install requests
```

2. Zoom Marketplace でサーバー間 OAuth アプリをセットアップします。
   - [Zoom Marketplace](https://marketplace.zoom.us/) にアクセスします。
   - 「Develop」→「Build App」をクリックします。
   - 「Server-to-Server OAuth」アプリタイプを選択します。
   - 必要なスコープでアプリを構成します。
     - `/meeting:write:admin`
     - `/meeting:read:admin`
     - `/recording:read:admin`
   - アカウント ID、クライアント ID、およびクライアントシークレットをメモします。

3. 環境変数をセットアップします。
```shell
export ZOOM_ACCOUNT_ID=your_account_id
export ZOOM_CLIENT_ID=your_client_id
export ZOOM_CLIENT_SECRET=your_client_secret
```

## 使用例

```python
from agno.agent import Agent
from agno.tools.zoom import ZoomTools

# 認証情報を使用して Zoom ツールを初期化します
zoom_tools = ZoomTools(
    account_id="your_account_id",
    client_id="your_client_id",
    client_secret="your_client_secret"
)

# Zoom 機能を持つエージェントを作成します
agent = Agent(tools=[zoom_tools], show_tool_calls=True)

# 会議をスケジュールします
response = agent.print_response("""
次の詳細を使用してチーム会議をスケジュールしてください。
- トピック: 週次チーム同期
- 時間: 明日午後 2 時 UTC
- 時間: 45 分
""", markdown=True)
```

## ツールキットのパラメータ

| パラメータ           | タイプ     | デフォルト      | 説明                                                 |
| ------------------ | -------- | ------------ | ---------------------------------------------------------- |
| `account_id`       | `str`    | `None`       | Zoom アカウント ID (サーバー間 OAuth アプリから)          |
| `client_id`        | `str`    | `None`       | クライアント ID (サーバー間 OAuth アプリから)                |
| `client_secret`    | `str`    | `None`       | クライアントシークレット (サーバー間 OAuth アプリから)            |


## ツールキットの関数

| 関数                | 説明                                                 |
| ---------------------- | ---------------------------------------------------------- |
| `schedule_meeting`     | 新しい Zoom 会議をスケジュールします                                 |
| `get_upcoming_meetings`| 今後の会議のリストを取得します                                   |
| `list_meetings`        | タイプに基づいてすべての会議をリストします                                 |
| `get_meeting_recordings`| 特定の会議の録画を取得します                                     |
| `delete_meeting`       | スケジュールされた会議を削除します                                   |
| `get_meeting`          | 特定の会議の詳細情報を取得します                                   |


## レート制限

Zoom API には、エンドポイントとアカウントタイプによって異なるレート制限があります。
- サーバー間 OAuth アプリ: 100 リクエスト/秒
- 会議エンドポイント: アカウントタイプに基づいた特定の制限が適用されます
- 録画エンドポイント: レート制限が低いため、Zoom のドキュメントを確認してください。

詳細なレート制限については、[Zoom API レート制限](https://developers.zoom.us/docs/api/#rate-limits)を参照してください。
