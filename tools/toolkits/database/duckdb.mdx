**DuckDbTools** を使用すると、Agent が SQL を実行し、DuckDb を使用してデータを分析できます。

## 前提条件

以下の例では DuckDB ライブラリが必要です。DuckDB をインストールするには、次のコマンドを実行します。

```shell
pip install duckdb
```

その他のインストールオプションについては、[DuckDB ドキュメント](https://duckdb.org/docs/installation)を参照してください。

## 例

以下のエージェントは、SQL を使用して映画ファイルを分析し、結果を返します。

```python cookbook/tools/duckdb_tools.py
from agno.agent import Agent
from agno.tools.duckdb import DuckDbTools

agent = Agent(
    tools=[DuckDbTools()],
    show_tool_calls=True,
    system_message="Use this file for Movies data: https://agno-public.s3.amazonaws.com/demo_data/IMDB-Movie-Data.csv",
)

agent.print_response("What is the average rating of movies?", markdown=True, stream=False)
```

## Toolkit Params

| パラメータ          | タイプ                 | デフォルト | 説明                                                              |
| ------------------- | -------------------- | ------- | ----------------------------------------------------------------- |
| `db_path`           | `str`                | -       | データベースファイルへのパスを指定します。                          |
| `connection`        | `DuckDBPyConnection` | -       | 既存の DuckDB 接続オブジェクトを提供します。                      |
| `init_commands`     | `List`               | -       | データベース接続で実行する初期 SQL コマンドのリスト。               |
| `read_only`         | `bool`               | `False` | データベース接続を読み取り専用に設定します。                        |
| `config`            | `dict`               | -       | データベース接続の構成オプション。                                |
| `run_queries`       | `bool`               | `True`  | 操作中に SQL クエリを実行するかどうかを決定します。                   |
| `inspect_queries`   | `bool`               | `False` | クエリを実行せずに SQL クエリの検査を有効にします。                 |
| `create_tables`     | `bool`               | `True`  | 操作中にデータベースにテーブルを作成することを許可します。            |
| `summarize_tables`  | `bool`               | `True`  | テーブルデータの要約を有効にします。                                |
| `export_tables`     | `bool`               | `False` | 操作中にテーブルを外部形式にエクスポートすることを許可します。        |

## Toolkit Functions

| 関数                   | 説明                                                                                                                                                                                                                                   |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `show_tables`          | データベース内のテーブルを表示する関数                                                                                                                                                                                                   |
| `describe_table`       | テーブルを記述する関数                                                                                                                                                                                                                 |
| `inspect_query`        | クエリを検査し、クエリプランを返す関数。実行前に必ずクエリを検査してください。                                                                                                                                                           |
| `run_query`            | クエリを実行し、結果を返す関数。                                                                                                                                                                                                       |
| `summarize_table`      | テーブルに対して多数の集計を計算する関数。この関数は、min、max、avg、std、approx_unique を含む多数の集計をすべての列に対して計算するクエリを起動します。                                                                                             |
| `get_table_name_from_path` | パスからテーブル名を取得します                                                                                                                                                                                                           |
| `create_table_from_path`   | パスからテーブルを作成します                                                                                                                                                                                                           |
| `export_table_to_path`     | テーブルを指定された形式（デフォルト：parquet）で保存します。パスが指定されている場合、テーブルはそのパスの下に保存されます。例：パスが /tmp の場合、テーブルは /tmp/table.parquet として保存されます。指定しない場合は、現在のディレクトリに保存されます。 |
| `load_local_path_to_table` | ローカルファイルを duckdb にロードします                                                                                                                                                                                                 |
| `load_local_csv_to_table`  | ローカル CSV ファイルを duckdb にロードします                                                                                                                                                                                            |
| `load_s3_path_to_table`    | S3 からファイルを duckdb にロードします                                                                                                                                                                                                  |
| `load_s3_csv_to_table`     | S3 から CSV ファイルを duckdb にロードします                                                                                                                                                                                             |
| `create_fts_index`         | テーブルに全文検索インデックスを作成します                                                                                                                                                                                             |
| `full_text_search`         | テーブルの列で特定のテキスト/キーワードに対する全文検索を実行します                                                                                                                                                                      |
