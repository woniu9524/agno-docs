**SQLTools** を使用すると、エージェントは SQL クエリを実行し、データベースと対話できます。

## 前提条件

次の例では、`sqlalchemy` ライブラリとデータベース URL が必要です。

```shell
pip install -U sqlalchemy
```

また、使用する特定のデータベースに対応する Python アダプターもインストールする必要があります。

### PostgreSQL

PostgreSQL の場合、`psycopg2-binary` アダプターをインストールできます。

```shell
pip install -U psycopg2-binary
```

### MySQL

MySQL の場合、`mysqlclient` アダプターをインストールできます。

```shell
pip install -U mysqlclient
```
`mysqlclient` アダプターには、追加のシステムレベルの依存関係が必要になる場合があります。詳細については、[公式インストールガイド](https://github.com/PyMySQL/mysqlclient/blob/main/README.md#install)を参照してください。

データベースも必要になります。次の例では、Docker コンテナで実行されている Postgres データベースを使用します。

```shell
 docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

## 例

次のエージェントは、SQL クエリを実行してデータベース内のすべてのテーブルを一覧表示し、テーブルの 1 つの内容を説明します。

```python cookbook/tools/sql_tools.py
from agno.agent import Agent
from agno.tools.sql import SQLTools

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

agent = Agent(tools=[SQLTools(db_url=db_url)])
agent.print_response("List the tables in the database. Tell me about contents of one of the tables", markdown=True)
```

## Toolkit Params

| パラメータ        | タイプ             | デフォルト | 説明                                                                 |
| ---------------- | ---------------- | ------- | --------------------------------------------------------------------------- |
| `db_url`         | `str`            | -       | データベースへの接続 URL。                                     |
| `db_engine`      | `Engine`         | -       | 接続および操作に使用されるデータベースエンジン。                    |
| `user`           | `str`            | -       | データベース認証用のユーザー名。                                   |
| `password`       | `str`            | -       | データベース認証用のパスワード。                                   |
| `host`           | `str`            | -       | データベースサーバーのホスト名または IP アドレス。                          |
| `port`           | `int`            | -       | データベースサーバーがリッスンしているポート番号。                  |
| `schema`         | `str`            | -       | 使用するデータベース内の特定のスキーマ。                             |
| `dialect`        | `str`            | -       | データベースで使用される SQL 方言。                                       |
| `tables`         | `Dict[str, Any]` | -       | テーブル名をそれぞれのメタデータまたは構造にマッピングする辞書。 |
| `list_tables`    | `bool`           | `True`  | データベース内のすべてのテーブルを一覧表示する機能を有効にします。 |
| `describe_table` | `bool`           | `True`  | 特定のテーブルのスキーマを説明する機能を有効にします。 |
| `run_sql_query`  | `bool`           | `True`  | SQL クエリを直接実行する機能を有効にします。                  |

## Toolkit Functions

| 関数         | 説明                                |
| ---------------- | ----------------------------------------- |
| `list_tables`    | データベース内のすべてのテーブルを一覧表示します。         |
| `describe_table` | 特定のテーブルのスキーマを説明します。 |
| `run_sql_query`  | SQL クエリを直接実行します。            |
