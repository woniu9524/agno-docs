**CsvTools** を使用すると、エージェントは CSV ファイルの読み書きを行うことができます。

## 例

以下のエージェントは、IMDB の CSV ファイルをダウンロードし、CLI アプリを使用してユーザーがクエリできるようにします。

```python cookbook/tools/csv_tools.py
import httpx
from pathlib import Path
from agno.agent import Agent
from agno.tools.csv_toolkit import CsvTools

url = "https://agno-public.s3.amazonaws.com/demo_data/IMDB-Movie-Data.csv"
response = httpx.get(url)

imdb_csv = Path(__file__).parent.joinpath("wip").joinpath("imdb.csv")
imdb_csv.parent.mkdir(parents=True, exist_ok=True)
imdb_csv.write_bytes(response.content)

agent = Agent(
    tools=[CsvTools(csvs=[imdb_csv])],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "First always get the list of files",
        "Then check the columns in the file",
        "Then run the query to answer the question",
        "Always wrap column names with double quotes if they contain spaces or special characters",
        "Remember to escape the quotes in the JSON string (use \")",
        "Use single quotes for string values"
    ],
)

agent.cli_app(stream=False)
```

## Toolkit のパラメータ

| パラメータ           | タイプ                     | デフォルト | 説明                                                              |
| ------------------- | ------------------------ | ------- | ---------------------------------------------------------------------- |
| `csvs`              | `List[Union[str, Path]]` | -       | 処理または読み込み対象の CSV ファイルまたはパスのリスト。                  |
| `row_limit`         | `int`                    | -       | 各 CSV ファイルから処理する最大行数。                                    |
| `read_csvs`         | `bool`                   | `True`  | 指定された CSV ファイルからのデータの読み込み機能を有効にします。                 |
| `list_csvs`         | `bool`                   | `True`  | 利用可能なすべての CSV ファイルを一覧表示する機能を有効にします。              |
| `query_csvs`        | `bool`                   | `True`  | CSV ファイル内のデータに対するクエリを実行する機能を有効にします。              |
| `read_column_names` | `bool`                   | `True`  | CSV ファイルから列名を読み取る機能を有効にします。                           |
| `duckdb_connection` | `Any`                    | -       | DuckDB データベース操作のための接続インスタンスを指定します。                 |
| `duckdb_kwargs`     | `Dict[str, Any]`         | -       | DuckDB 操作を構成するためのキーワード引数の辞書。                         |

## Toolkit の関数

| 関数             | 説明                                     |
| ---------------- | ---------------------------------------- |
| `list_csv_files` | 利用可能なすべての CSV ファイルを一覧表示します。 |
| `read_csv_file`  | この関数は CSV ファイルの内容を読み取ります。   |
| `get_columns`    | この関数は CSV ファイルの列を返します。     |
| `query_csv_file` | この関数は CSV ファイルの内容をクエリします。  |
