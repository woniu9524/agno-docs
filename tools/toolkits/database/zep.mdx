**ZepTools** は、エージェントが Zep メモリシステムと対話できるようにし、ユーザーセッションに関連付けられたメモリデータを保存、取得、検索する機能を提供します。

## 前提条件

ZepTools には、`zep-cloud` Python パッケージと Zep API キーが必要です。

```shell
pip install zep-cloud
```

```shell
export ZEP_API_KEY=your_api_key
```

## 例

以下の例は、Zep メモリにアクセスできるエージェントを作成する方法を示しています。

```python cookbook/tools/zep_tools.py
import time

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.zep import ZepTools

# ZepTools を初期化します
zep_tools = ZepTools(user_id="agno", session_id="agno-session", add_instructions=True)

# Agent を初期化します
agent = Agent(
    model=OpenAIChat(),
    tools=[zep_tools],
    context={"memory": zep_tools.get_zep_memory(memory_type="context")},
    add_context=True,
)

# エージェントと対話して、ユーザーについて学習できるようにします
agent.print_response("私の名前は John Billings です")
agent.print_response("私は NYC に住んでいます")
agent.print_response("明日コンサートに行きます")

# メモリが Zep データベースと同期するのを待ちます
time.sleep(10)

# コンテキストを更新します
agent.context["memory"] = zep_tools.get_zep_memory(memory_type="context")

# エージェントにユーザーについて尋ねます
agent.print_response("私のことについて何を知っていますか？")
```

## Toolkit パラメータ

| パラメータ                | タイプ     | デフォルト | 説明                                           |
| ------------------------ | -------- | ------- | ----------------------------------------------------- |
| `session_id`                | `str`    | `None`  | オプションのセッション ID。提供されない場合は自動生成されます。  |
| `user_id`                | `str`    | `None`  | オプションのユーザー ID。提供されない場合は自動生成されます。     |
| `api_key`                | `str`    | `None`  | Zep API キー。提供されない場合は、ZEP_API_KEY 環境変数が使用されます。 |
| `ignore_assistant_messages` | `bool` | `False` | メモリに追加する際にアシスタントメッセージを無視するかどうか。 |
| `add_zep_message`        | `bool`   | `True`  | 現在の Zep セッションメモリにメッセージを追加します。        |
| `get_zep_memory`         | `bool`   | `True`  | 現在の Zep セッションのメモリを取得します。       |
| `search_zep_memory`      | `bool`   | `True`  | Zep メモリ ストアで関連情報を検索します。   |
| `instructions`           | `str`    | `None`  | Zep ツールを使用するためのカスタム指示。          |
| `add_instructions`       | `bool`   | `False` | デフォルトの指示を追加するかどうか。                  |

## Toolkit 関数

| 関数 | 説明 |
| -------- | ----------- |
| `add_zep_message` | 現在の Zep セッションメモリにメッセージを追加します。メッセージ送信者の `role` (str) とメッセージテキストの `content` (str) を受け取ります。確認またはエラーメッセージを返します。 |
| `get_zep_memory` | 現在の Zep セッションのメモリを取得します。オプションの `memory_type` (str) パラメータを受け取り、"context" (デフォルト)、"summary"、または "messages" のオプションがあります。要求されたメモリの内容またはエラーを返します。 |
| `search_zep_memory` | Zep メモリ ストアで関連情報を検索します。関連する事実を見つけるための `query` (str) と、オプションの `search_scope` (str) パラメータを受け取ります。"messages" (デフォルト) または "summary" のオプションがあります。検索結果またはエラーメッセージを返します。 |

## Async Toolkit

`ZepAsyncTools` クラスは `ZepTools` クラスを拡張し、ツール関数の非同期バージョンを提供します。
