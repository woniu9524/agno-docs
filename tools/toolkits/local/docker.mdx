**DockerTools** は、エージェントが Docker コンテナ、イメージ、ボリューム、ネットワークと対話できるようにします。

## 前提条件

Docker ツールには `docker` Python パッケージが必要です。また、システムに Docker がインストールされ、実行されている必要があります。

```shell
pip install docker
```

## 例

次の例では、Docker リソースを管理できるエージェントを作成します。

```python cookbook/tools/docker_tools.py
import sys
from agno.agent import Agent

try:
    from agno.tools.docker import DockerTools

    docker_tools = DockerTools(
        enable_container_management=True,
        enable_image_management=True,
        enable_volume_management=True,
        enable_network_management=True,
    )

    # Docker ツールを使用してエージェントを作成します
    docker_agent = Agent(
        name="Docker Agent",
        instructions=[
            "You are a Docker management assistant that can perform various Docker operations.",
            "You can manage containers, images, volumes, and networks.",
        ],
        tools=[docker_tools],
        show_tool_calls=True,
        markdown=True,
    )

    # 例: 実行中のすべての Docker コンテナを一覧表示します
    docker_agent.print_response("List all running Docker containers", stream=True)

    # 例: NGINX コンテナをプルして実行します
    docker_agent.print_response("Pull the latest nginx image", stream=True)
    docker_agent.print_response("Run an nginx container named 'web-server' on port 8080", stream=True)

except ValueError as e:
    print(f"\n❌ Docker Tool Error: {e}")
    print("\n🔍 Troubleshooting steps:")

    if sys.platform == "darwin":  # macOS
        print("1. Ensure Docker Desktop is running")
        print("2. Check Docker Desktop settings")
        print("3. Try running 'docker ps' in terminal to verify access")

    elif sys.platform == "linux":
        print("1. Check if Docker service is running:")
        print("   systemctl status docker")
        print("2. Make sure your user has permissions to access Docker:")
        print("   sudo usermod -aG docker $USER")

    elif sys.platform == "win32":
        print("1. Ensure Docker Desktop is running")
        print("2. Check Docker Desktop settings")
```

## Toolkit パラメータ

| パラメータ                    | タイプ   | デフォルト | 説明                                                     |
| ------------------------------ | ------ | ------- | -------------------------------------------------------- |
| `enable_container_management` | `bool` | `True`  | コンテナ管理機能（一覧表示、開始、停止など）を有効にします |
| `enable_image_management`    | `bool` | `True`  | イメージ管理機能（プル、ビルドなど）を有効にします         |
| `enable_volume_management`   | `bool` | `False` | ボリューム管理機能を有効にします                         |
| `enable_network_management`  | `bool` | `False` | ネットワーク管理機能を有効にします                       |

## Toolkit 関数

### コンテナ管理

| 関数           | 説明                                           |
| -------------- | ---------------------------------------------- |
| `list_containers`  | すべてのコンテナまたは実行中のコンテナのみを一覧表示します |
| `start_container`  | 停止中のコンテナを開始します                   |
| `stop_container`   | 実行中のコンテナを停止します                   |
| `remove_container` | コンテナを削除します                           |
| `get_container_logs` | コンテナからログを取得します                   |
| `inspect_container` | コンテナの詳細情報を取得します                 |
| `run_container`    | 新しいコンテナを作成して開始します             |
| `exec_in_container` | 実行中のコンテナ内でコマンドを実行します       |

### イメージ管理

| 関数           | 説明                                       |
| -------------- | ------------------------------------------ |
| `list_images`      | システム上のすべてのイメージを一覧表示します |
| `pull_image`       | レジストリからイメージをプルします         |
| `remove_image`     | イメージを削除します                       |
| `build_image`      | Dockerfile からイメージをビルドします      |
| `tag_image`        | イメージにタグを付けます                   |
| `inspect_image`    | イメージの詳細情報を取得します             |

### ボリューム管理

| 関数           | 説明                           |
| -------------- | ------------------------------ |
| `list_volumes`     | すべてのボリュームを一覧表示します |
| `create_volume`    | 新しいボリュームを作成します   |
| `remove_volume`    | ボリュームを削除します         |
| `inspect_volume`   | ボリュームの詳細情報を取得します |

### ネットワーク管理

| 関数                          | 説明                                       |
| --------------------------------- | ------------------------------------------ |
| `list_networks`                   | すべてのネットワークを一覧表示します       |
| `create_network`                  | 新しいネットワークを作成します             |
| `remove_network`                  | ネットワークを削除します                   |
| `inspect_network`                 | ネットワークの詳細情報を取得します         |
| `connect_container_to_network`    | コンテナをネットワークに接続します         |
| `disconnect_container_from_network` | コンテナをネットワークから切断します       |
