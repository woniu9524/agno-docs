**ShellTools** を使用すると、エージェントがシェルと対話し、コマンドを実行できます。

## 例

以下のエージェントは、シェルコマンドを実行し、現在のディレクトリの内容を表示します。

<Note>
  OS をエージェントに伝えることで、正しいコマンドが実行されることを確認してください。
</Note>

```python cookbook/tools/shell_tools.py
from agno.agent import Agent
from agno.tools.shell import ShellTools

agent = Agent(tools=[ShellTools()], show_tool_calls=True)
agent.print_response("Show me the contents of the current directory", markdown=True)
```

## Toolkit の機能

| 関数名            | 説明                                           |
| ------------------- | ----------------------------------------------------- |
| `run_shell_command` | シェルコマンドを実行し、出力またはエラーを返します。 |
