## 例

以下のエージェントは、`sleep` ツールを使用して指定された秒数だけ実行を一時停止します。

```python cookbook/tools/sleep_tools.py
from agno.agent import Agent
from agno.tools.sleep import SleepTools

# Sleep ツールでエージェントを作成
agent = Agent(tools=[SleepTools()], name="Sleep Agent")

# 例 1: 2秒間スリープ
agent.print_response("Sleep for 2 seconds")

# 例 2: より長い時間スリープ
agent.print_response("Sleep for 5 seconds")
```

## Toolkit パラメータ

| パラメータ | タイプ  | デフォルト   | 説明          |
| --------- | ----- | --------- | ------------- |
| `name`    | `str` | `"sleep"` | ツールの名前 |

## Toolkit 関数

| 関数 | 説明                                     |
| -------- | ---------------------------------------- |
| `sleep`  | 指定された秒数だけ実行を一時停止します |
