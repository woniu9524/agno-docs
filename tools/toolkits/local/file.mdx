**FileTools** は、エージェントがローカルファイルシステム上のファイルを読み書きできるようにします。

## 例

以下は、回答を生成してファイルに保存するエージェントの例です。

```python cookbook/tools/file_tools.py
from agno.agent import Agent
from agno.tools.file import FileTools

agent = Agent(tools=[FileTools()], show_tool_calls=True)
agent.print_response("現在最も先進的なLLMは何ですか？回答をファイルに保存してください。", markdown=True)
```

## Toolkit パラメータ

| 名前         | タイプ   | デフォルト | 説明                                                               |
| ------------ | ------- | -------- | ------------------------------------------------------------------ |
| `base_dir`   | `Path`  | -        | ファイル操作のベースディレクトリパスを指定します。                               |
| `save_files` | `bool`  | `True`   | 操作中にファイルを保存するかどうかを決定します。                                   |
| `read_files` | `bool`  | `True`   | 操作中にファイルから読み取ることを許可します。                                   |
| `list_files` | `bool`  | `True`   | 指定されたディレクトリ内のファイルのリスト表示を有効にします。                               |

## Toolkit 関数

| 名前         | 説明                                                                         |
| ------------ | ---------------------------------------------------------------------------- |
| `save_file`  | 指定された `file_name` に内容を保存し、成功した場合はファイル名を返します。                               |
| `read_file`  | 指定された `file_name` の内容を読み取り、成功した場合は内容を返します。                                   |
| `list_files` | ベースディレクトリ内のファイルのリストを返します。                                                       |
