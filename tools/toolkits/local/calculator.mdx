**Calculator** は、エージェントが数学的計算を実行できるようにします。

## 例

以下のエージェントは、`10*5` の結果を計算し、それを `2` 乗します。

```python cookbook/tools/calculator_tools.py
from agno.agent import Agent
from agno.tools.calculator import CalculatorTools

agent = Agent(
    tools=[
        CalculatorTools(
            add=True,
            subtract=True,
            multiply=True,
            divide=True,
            exponentiate=True,
            factorial=True,
            is_prime=True,
            square_root=True,
        )
    ],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("10*5 の結果を計算してから、それを2乗してください。ステップバイステップで実行してください。")
```

## Toolkit パラメータ

| パラメータ      | タイプ   | デフォルト | 説明                                                                 |
| -------------- | ------ | ------- | -------------------------------------------------------------------- |
| `add`          | `bool` | `True`  | 加算を実行する機能を有効にします。                                                   |
| `subtract`     | `bool` | `True`  | 減算を実行する機能を有効にします。                                                   |
| `multiply`     | `bool` | `True`  | 乗算を実行する機能を有効にします。                                                   |
| `divide`       | `bool` | `True`  | 除算を実行する機能を有効にします。                                                   |
| `exponentiate` | `bool` | `False` | 指数計算を実行する機能を有効にします。                                                 |
| `factorial`    | `bool` | `False` | 数値の階乗を計算する機能を有効にします。                                               |
| `is_prime`     | `bool` | `False` | 数値が素数かどうかをチェックする機能を有効にします。                                         |
| `square_root`  | `bool` | `False` | 数値の平方根を計算する機能を有効にします。                                               |

## Toolkit 関数

| 関数       | 説明                                                                                             |
| -------------- | ------------------------------------------------------------------------------------------------ |
| `add`          | 2つの数値を加算し、結果を返します。                                                                             |
| `subtract`     | 最初の数値から2番目の数値を減算し、結果を返します。                                                                         |
| `multiply`     | 2つの数値を乗算し、結果を返します。                                                                             |
| `divide`       | 最初の数値を2番目の数値で除算し、結果を返します。ゼロ除算を処理します。                                                               |
| `exponentiate` | 最初の数値を2番目の数値のべき乗にし、結果を返します。                                                                       |
| `factorial`    | 数値の階乗を計算し、結果を返します。負の数を処理します。                                                                     |
| `is_prime`     | 数値が素数かどうかをチェックし、結果を返します。                                                                           |
| `square_root`  | 数値の平方根を計算し、結果を返します。負の数を処理します。                                                                     |
