`KnowledgeTools` ツールキットは、エージェントがナレッジベースから情報を検索、取得、分析できるようにします。このツールキットは `AgentKnowledge` と統合されており、ユーザーに応答する前に、関連情報を検索および評価するための構造化されたワークフローを提供します。

このツールキットは「思考 → 検索 → 分析」サイクルを実装しており、エージェントは以下のことができます。
1. 問題について考え、検索クエリを計画する
2. ナレッジベースで関連情報を検索する
3. 結果を分析して、十分かどうか、または追加の検索が必要かどうかを判断する

このアプローチにより、エージェントは知識を見つけ、評価し、統合するためのツールを持つことで、正確な情報を提供する能力が大幅に向上します。

ツールキットには以下のツールが含まれています。

- `think`: 計画、キーワードのブレインストーミング、アプローチの洗練のためのスクラッチパッド。これらの思考はエージェントの内部に留まり、ユーザーには表示されません。
- `search`: ナレッジベースに対してクエリを実行し、関連ドキュメントを取得します。
- `analyze`: 返されたドキュメントが正しいか十分かを評価し、さらなる検索が必要かどうかを判断します。

## 例

`KnowledgeTools` ツールキットの使用例を以下に示します。

```python
from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.openai import OpenAIChat
from agno.tools.knowledge import KnowledgeTools
from agno.vectordb.lancedb import LanceDb, SearchType

# URLからの情報を含むナレッジベースを作成
agno_docs = UrlKnowledge(
    urls=["https://docs.agno.com/llms-full.txt"],
    # LanceDB をベクトルデータベースとして使用し、埋め込みを `agno_docs` テーブルに保存
    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        embedder=OpenAIEmbedder(id="text-embedding-3-small"),
    ),
)

knowledge_tools = KnowledgeTools(
    knowledge=agno_docs,
    think=True,
    search=True,
    analyze=True,
    add_few_shot=True,
)

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[knowledge_tools],
    show_tool_calls=True,
    markdown=True,
)

if __name__ == "__main__":
    # ナレッジベースをロード（初回実行後にコメントアウト）
    agno_docs.load(recreate=True)
    agent.print_response("How do I build multi-agent teams with Agno?", stream=True)
```

このツールキットには、エージェントがツールを効果的に使用するのに役立つデフォルトの指示と少数例が含まれています。それらを構成する方法は次のとおりです。

```python
from agno.tools.knowledge import KnowledgeTools

knowledge_tools = KnowledgeTools(
    knowledge=my_knowledge_base,
    think=True,                # think ツールを有効にする
    search=True,               # search ツールを有効にする
    analyze=True,              # analyze ツールを有効にする
    add_instructions=True,     # デフォルトの指示を追加する
    add_few_shot=True,         # 少数例を追加する
    few_shot_examples=None,    # オプションのカスタム少数例
)
