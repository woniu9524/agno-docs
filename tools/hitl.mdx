Human in the loop (HITL) を使用すると、ツール呼び出しを実行する前または後に、ユーザーからの入力を受け取ることができます。

以下の例では、ツール呼び出しを実行する前にユーザーの確認を得るためにツールフックを使用する方法を示します。

## 例: ツールフックを使用した Human in the loop

この例では、次の方法を示します。
- ユーザー確認のためにツールにフックを追加する
- ツール実行中にユーザー入力を処理する
- ユーザーの選択に基づいて操作を正常にキャンセルする

```python hitl.py
"""🤝 Human-in-the-Loop: ツール呼び出しへのユーザー確認の追加

この例では、Agno ツールに human-in-the-loop 機能を実装する方法を示します。
これは次の方法を示します。
- ユーザー確認のためにツールにツールフックを追加する
- ツール実行中にユーザー入力を処理する
- ユーザーの選択に基づいて操作を正常にキャンセルする

実用的なアプリケーションの例:
- 実行前の機密操作の確認
- API 呼び出し前のレビュー
- データ変換の検証
- クリティカルシステムにおける自動アクションの承認

依存関係をインストールするには `pip install openai httpx rich agno` を実行してください。
"""

import json
from typing import Any, Callable, Dict, Iterator

import httpx
from agno.agent import Agent
from agno.exceptions import StopAgentRun
from agno.models.openai import OpenAIChat
from agno.tools import FunctionCall, tool
from rich.console import Console
from rich.pretty import pprint
from rich.prompt import Prompt

# これは print_response メソッドによって使用されるコンソールインスタンスです
# これを使用してライブ表示を停止および再開し、ユーザー確認を求めることができます
console = Console()


def confirmation_hook(
    function_name: str, function_call: Callable, arguments: Dict[str, Any]
):
    # コンソールからライブ表示インスタンスを取得します
    live = console._live

    # ユーザー確認を求めるためにライブ表示を一時的に停止します
    live.stop()  # type: ignore

    # 確認を求めます
    console.print(f"\n実行しようとしています [bold blue]{fc.function.name}[/]")
    message = (
        Prompt.ask("続行しますか？", choices=["y", "n"], default="y")
        .strip()
        .lower()
    )

    # ライブ表示を再開します
    live.start()  # type: ignore

    # ユーザーが続行しない場合は、StopExecution 例外を発生させます
    if message != "y":
        raise StopAgentRun(
            "ユーザーによってツール呼び出しがキャンセルされました",
            agent_message="許可が得られなかったため、実行を停止しています。",
        )
    
    # 関数を呼び出します
    result = function_call(**arguments)

    # オプションで結果を変換します

    return result


@tool(tool_hooks=[confirmation_hook])
def get_top_hackernews_stories(num_stories: int) -> Iterator[str]:
    """Hacker News からトップストーリーを取得します。

    Args:
        num_stories (int): 取得するストーリーの数

    Returns:
        str: ストーリーの詳細を含む JSON 文字列
    """
    # トップストーリー ID を取得します
    response = httpx.get("https://hacker-news.firebaseio.com/v0/topstories.json")
    story_ids = response.json()

    # ストーリーの詳細を yield します
    final_stories = []
    for story_id in story_ids[:num_stories]:
        story_response = httpx.get(
            f"https://hacker-news.firebaseio.com/v0/item/{story_id}.json"
        )
        story = story_response.json()
        if "text" in story:
            story.pop("text", None)
        final_stories.append(story)

    return json.dumps(final_stories)


# テクノロジーに精通した個性と明確な指示でエージェントを初期化します
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[get_top_hackernews_stories],
    markdown=True,
)

agent.print_response(
    "トップ2件のハッカーニュースストーリーを取得しますか？", stream=True, console=console
)
