---
title: Tool Result Caching
---

Tool result caching は、関数呼び出しの結果をディスクに保存することで、不要な再計算を回避するように設計されています。
これは、開発プロセスを高速化し、レート制限を回避し、コストを削減するために、開発およびテスト中に役立ちます。

これは、すべての Agno Toolkits でサポートされています。

## 例

その Toolkit のキャッシュを有効にするには、`cache_results=True` を Toolkit コンストラクタに渡します。

```cache_tool_calls.py
import asyncio

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[DuckDuckGoTools(cache_results=True), YFinanceTools(cache_results=True)],
    show_tool_calls=True,
)

asyncio.run(
    agent.aprint_response(
        "What is the current stock price of AAPL and latest news on 'Apple'?",
        markdown=True,
    )
)
