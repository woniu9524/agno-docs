Agno は、`SingleStoreStorage` クラスを使用して、エージェントのストレージバックエンドとして Singlestore を利用することをサポートしています。

## 使用方法

Singlestore の認証情報は[こちら](https://portal.singlestore.com/)から取得してください。

```python singlestore_storage_for_agent.py
from os import getenv

from sqlalchemy.engine import create_engine

from agno.agent import Agent
from agno.storage.singlestore import SingleStoreStorage

# SingleStore の設定
USERNAME = getenv("SINGLESTORE_USERNAME")
PASSWORD = getenv("SINGLESTORE_PASSWORD")
HOST = getenv("SINGLESTORE_HOST")
PORT = getenv("SINGLESTORE_PORT")
DATABASE = getenv("SINGLESTORE_DATABASE")
SSL_CERT = getenv("SINGLESTORE_SSL_CERT", None)

# SingleStore DB URL
db_url = f"mysql+pymysql://{USERNAME}:{PASSWORD}@{HOST}:{PORT}/{DATABASE}?charset=utf8mb4"
if SSL_CERT:
    db_url += f"&ssl_ca={SSL_CERT}&ssl_verify_cert=true"

# データベースエンジンを作成
db_engine = create_engine(db_url)

# Singlestore データベースを使用してストレージバックエンドを作成
storage = SingleStoreStorage(
    # セッションを ai.sessions テーブルに保存
    table_name="agent_sessions",
    # db_engine: Singlestore データベースエンジン
    db_engine=db_engine,
    # schema: Singlestore スキーマ
    schema=DATABASE,
)

# Agent にストレージを追加
agent = Agent(storage=storage)
```
