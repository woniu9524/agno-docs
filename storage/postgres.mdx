Agno は、`PostgresStorage` クラスを使用して、エージェントのストレージバックエンドとして PostgreSQL をサポートしています。

## 使用方法

### PgVector の実行

[docker desktop](https://docs.docker.com/desktop/install/mac-install/) をインストールし、以下を使用してポート **5532** で **PgVector** を実行します。

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

```python postgres_storage_for_agent.py
from agno.storage.postgres import PostgresStorage

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

# Postgres データベースを使用してストレージバックエンドを作成します
storage = PostgresStorage(
    # セッションを ai.sessions テーブルに保存します
    table_name="agent_sessions",
    # db_url: Postgres データベース URL
    db_url=db_url,
)

# Agent にストレージを追加します
agent = Agent(storage=storage)
```
