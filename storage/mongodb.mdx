Agno は、`MongoDbStorage` クラスを使用して、エージェントのストレージバックエンドとして MongoDB をサポートしています。

## 使用方法

`db_url` または `client` のいずれかを提供する必要があります。次の例では `db_url` を使用しています。

```python mongodb_storage_for_agent.py
from agno.storage.mongodb import MongoDbStorage

db_url = "mongodb://ai:ai@localhost:27017/agno"

# Mongo データベースを使用してストレージバックエンドを作成します
storage = MongoDbStorage(
    # セッションを agent_sessions コレクションに保存します
    collection_name="agent_sessions",
    db_url=db_url,
)

# ストレージをエージェントに追加します
agent = Agent(storage=storage)
```
