Agno は、`DynamoDbStorage` クラスを使用して、エージェント、チーム、ワークフローのストレージバックエンドとして DynamoDB をサポートしています。

## 使用方法

`DynamoDbStorage` クラスに `aws_access_key_id` および `aws_secret_access_key` パラメータを提供する必要があります。

```python dynamodb_storage_for_agent.py
from agno.storage.dynamodb import DynamoDbStorage

# AWS Credentials
AWS_ACCESS_KEY_ID = getenv("AWS_ACCESS_KEY_ID")
AWS_SECRET_ACCESS_KEY = getenv("AWS_SECRET_ACCESS_KEY")

storage = DynamoDbStorage(
    # セッションを ai.sessions テーブルに保存
    table_name="agent_sessions",
    # region_name: DynamoDB のリージョン名
    region_name="us-east-1",
    # aws_access_key_id: AWS アクセスキーID
    aws_access_key_id=AWS_ACCESS_KEY_ID,
    # aws_secret_access_key: AWS シークレットアクセスキー
    aws_secret_access_key=AWS_SECRET_ACCESS_KEY,
)

# Agent にストレージを追加
agent = Agent(storage=storage)
```
