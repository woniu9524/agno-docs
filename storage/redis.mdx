Agno は、`RedisStorage` クラスを使用して、エージェントのストレージバックエンドとして Redis をサポートしています。

## 使用方法

### Redis の実行

[docker desktop](https://docs.docker.com/desktop/install/mac-install/) をインストールし、以下を使用してポート **6379** で **Redis** を実行します。

```bash
docker run --name my-redis -p 6379:6379 -d redis
```

```python redis_storage_for_agent.py
from agno.agent import Agent
from agno.storage.redis import RedisStorage
from agno.tools.duckduckgo import DuckDuckGoTools

# デフォルトのローカル接続で Redis ストレージを初期化
storage = RedisStorage(
    prefix="agno_test",    # セッションの名前空間を設定するための Redis キーのプレフィックス
    host="localhost",      # Redis ホストアドレス
    port=6379,             # Redis ポート番号
)

# Redis ストレージを使用してエージェントを作成
agent = Agent(
    storage=storage,
    tools=[DuckDuckGoTools()],
    add_history_to_messages=True,
)

agent.print_response("カナダの人口は何人ですか？")

agent.print_response("彼らの国歌は何と呼ばれていますか？")

# ストレージの内容を確認
print("\nストレージの内容を確認中...")
all_sessions = storage.get_all_sessions()
print(f"Redis 内のセッション合計: {len(all_sessions)}")

if all_sessions:
    print("\nセッション詳細:")
    session = all_sessions[0]
    print(f"セッション ID: {session.session_id}")
    print(f"メッセージ数: {len(session.memory['messages'])}")
```
