Agno は、`SqliteStorage` クラスを使用して、エージェントのストレージバックエンドとして Sqlite をサポートしています。

## 使用方法

`db_url`、`db_file`、または `db_engine` のいずれかを提供する必要があります。以下の例では `db_file` を使用しています。

```python sqlite_storage_for_agent.py
from agno.storage.sqlite import SqliteStorage

# Sqlite データベースを使用してストレージバックエンドを作成します
storage = SqliteStorage(
    # セッションを ai.sessions テーブルに保存します
    table_name="agent_sessions",
    # db_file: Sqlite データベースファイル
    db_file="tmp/data.db",
)

# Agent にストレージを追加します
agent = Agent(storage=storage)
```
