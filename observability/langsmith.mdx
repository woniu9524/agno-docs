## LangSmith と Agno の統合

LangSmith は、AI モデルの呼び出しをトレースおよび監視するための包括的なプラットフォームを提供します。Agno と LangSmith を統合することで、OpenInference を使用してトレースを送信し、エージェントのパフォーマンスに関する洞察を得ることができます。

## 前提条件

1. **LangSmith アカウントの作成**

   - [LangSmith](https://smith.langchain.com) でアカウントにサインアップします。
   - LangSmith ダッシュボードから API キーを取得します。

2. **環境変数の設定**

   LangSmith API キーおよびその他の必要な設定で環境を構成します。

   ```bash
   export LANGSMITH_API_KEY=<your-key>
   export LANGSMITH_TRACING=true
   export LANGSMITH_ENDPOINT=https://eu.api.smith.langchain.com  # 米国向けの場合は https://api.smith.langchain.com
   export LANGSMITH_PROJECT=<your-project-name>
   ```

3. **依存関係のインストール**

   必要なパッケージがインストールされていることを確認します。

   ```bash
   pip install openai openinference-instrumentation-agno opentelemetry-sdk opentelemetry-exporter-otlp
   ```

## LangSmith へのトレースの送信

この例では、Agno エージェントを OpenInference でインストルメント化し、LangSmith にトレースを送信する方法を示します。

```python
import os

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools
from openinference.instrumentation.agno import AgnoInstrumentor
from opentelemetry import trace as trace_api
from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter
from opentelemetry.sdk.trace import TracerProvider
from opentelemetry.sdk.trace.export import SimpleSpanProcessor

# LangSmith のエンドポイントとヘッダーを設定します
endpoint = "https://eu.api.smith.langchain.com/otel/v1/traces"
headers = {
    "x-api-key": os.getenv("LANGSMITH_API_KEY"),
    "Langsmith-Project": os.getenv("LANGSMITH_PROJECT"),
}

# トレーサープロバイダーを構成します
tracer_provider = TracerProvider()
tracer_provider.add_span_processor(
    SimpleSpanProcessor(OTLPSpanExporter(endpoint=endpoint, headers=headers))
)
trace_api.set_tracer_provider(tracer_provider=tracer_provider)

# agno のインストルメント化を開始します
AgnoInstrumentor().instrument()

# エージェントを作成および構成します
agent = Agent(
    name="Stock Market Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[DuckDuckGoTools()],
    markdown=True,
    debug_mode=True,
)

# エージェントを使用します
agent.print_response("What is news on the stock market?")
```

## 注意事項

- **環境変数**: API キー、エンドポイント、プロジェクト名について、環境変数が正しく設定されていることを確認してください。
- **データリージョン**: データリージョンに基づいて適切な `LANGSMITH_ENDPOINT` を選択してください。

これらの手順に従うことで、Agno と LangSmith を効果的に統合し、AI エージェントの包括的なオブザーバビリティと監視を有効にすることができます。
