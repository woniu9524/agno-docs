## Agno と Langfuse の統合

Langfuse は、AI モデルの呼び出しをトレースおよび監視するための堅牢なプラットフォームを提供します。Agno と Langfuse を統合することにより、OpenInference および OpenLIT を使用してトレースを送信し、エージェントのパフォーマンスに関する洞察を得ることができます。

## 前提条件

1. **依存関係のインストール**

   必要なパッケージがインストールされていることを確認してください。

   ```bash
   pip install agno openai langfuse opentelemetry-sdk opentelemetry-exporter-otlp openinference-instrumentation-agno
   ```

2. **Langfuse アカウントの設定**

   - Langfuse をセルフホストするか、[Langfuse](https://us.cloud.langfuse.com) でアカウントにサインアップしてください。
   - Langfuse ダッシュボードから公開 API キーとシークレット API キーを取得します。

3. **環境変数の設定**

   Langfuse API キーで環境を構成します。

   ```bash
   export LANGFUSE_PUBLIC_KEY=<your-public-key>
   export LANGFUSE_SECRET_KEY=<your-secret-key>
   ```

## Langfuse へのトレースの送信

- ### 例: OpenInference と Langfuse の使用

   この例では、Agno エージェントを OpenInference でインストルメント化し、Langfuse にトレースを送信する方法を示します。

   ```python
   import base64
   import os

   from agno.agent import Agent
   from agno.models.openai import OpenAIChat
   from agno.tools.yfinance import YFinanceTools
   from openinference.instrumentation.agno import AgnoInstrumentor
   from opentelemetry import trace as trace_api
   from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter
   from opentelemetry.sdk.trace import TracerProvider
   from opentelemetry.sdk.trace.export import SimpleSpanProcessor

   # Langfuse の環境変数を設定
   LANGFUSE_AUTH = base64.b64encode(
       f"{os.getenv('LANGFUSE_PUBLIC_KEY')}:{os.getenv('LANGFUSE_SECRET_KEY')}".encode()
   ).decode()
   os.environ["OTEL_EXPORTER_OTLP_ENDPOINT"] = "https://us.cloud.langfuse.com/api/public/otel"
   os.environ["OTEL_EXPORTER_OTLP_HEADERS"] = f"Authorization=Basic {LANGFUSE_AUTH}"

   # トレーサープロバイダーを構成
   tracer_provider = TracerProvider()
   tracer_provider.add_span_processor(SimpleSpanProcessor(OTLPSpanExporter()))
   trace_api.set_tracer_provider(tracer_provider=tracer_provider)

   # agno のインストルメンテーションを開始
   AgnoInstrumentor().instrument()

   # エージェントを作成および構成
   agent = Agent(
       name="Stock Price Agent",
       model=OpenAIChat(id="gpt-4o-mini"),
       tools=[YFinanceTools()],
       instructions="You are a stock price agent. Answer questions in the style of a stock analyst.",
       debug_mode=True,
   )

   # エージェントを使用
   agent.print_response("What is the current price of Tesla?")
   ```

- ### 例: OpenLIT と Langfuse の使用

   この例では、OpenLIT を介して Langfuse を使用してモデルの呼び出しをトレースする方法を示します。

   ```python
   import base64
   import os

   from agno.agent import Agent
   from agno.models.openai import OpenAIChat
   from agno.tools.duckduckgo import DuckDuckGoTools
   from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter
   from opentelemetry.sdk.trace import TracerProvider
   from opentelemetry.sdk.trace.export import SimpleSpanProcessor
   from opentelemetry import trace

   # Langfuse の環境変数を設定
   LANGFUSE_AUTH = base64.b64encode(
       f"{os.getenv('LANGFUSE_PUBLIC_KEY')}:{os.getenv('LANGFUSE_SECRET_KEY')}".encode()
   ).decode()
   os.environ["OTEL_EXPORTER_OTLP_ENDPOINT"] = "https://us.cloud.langfuse.com/api/public/otel"
   os.environ["OTEL_EXPORTER_OTLP_HEADERS"] = f"Authorization=Basic {LANGFUSE_AUTH}"

   # トレーサープロバイダーを構成
   trace_provider = TracerProvider()
   trace_provider.add_span_processor(SimpleSpanProcessor(OTLPSpanExporter()))
   trace.set_tracer_provider(trace_provider)

   # OpenLIT インストルメンテーションを初期化
   import openlit
   openlit.init(tracer=trace.get_tracer(__name__), disable_batch=True)

   # エージェントを作成および構成
   agent = Agent(
       model=OpenAIChat(id="gpt-4o-mini"),
       tools=[DuckDuckGoTools()],
       markdown=True,
       debug_mode=True,
   )

   # エージェントを使用
   agent.print_response("What is currently trending on Twitter?")
   ```

## 注意事項

- **環境変数**: API キーと OTLP エンドポイントの環境変数が正しく設定されていることを確認してください。
- **データリージョン**: データリージョンまたはローカルデプロイメントに合わせて `OTEL_EXPORTER_OTLP_ENDPOINT` を調整してください。利用可能なリージョンは次のとおりです。
  - 米国リージョンの場合: `https://us.cloud.langfuse.com/api/public/otel`
  - EU リージョンの場合: `https://eu.cloud.langfuse.com/api/public/otel`
  - ローカルデプロイメントの場合: `http://localhost:3000/api/public/otel`

これらの手順に従うことで、Agno と Langfuse を効果的に統合し、AI エージェントの包括的なオブザーバビリティと監視を有効にすることができます。
