## Agno と WandB の Weave との統合

[Weights & Biases (WandB) による Weave](https://weave-docs.wandb.ai/) は、モデル呼び出しのログ記録と視覚化のための強力なプラットフォームを提供します。Agno を Weave と統合することで、エージェントのパフォーマンスと動作を追跡および分析できます。

## 前提条件

1. **Weave のインストール**

   Weave パッケージがインストールされていることを確認してください。

   ```bash
   pip install weave
   ```

2. **認証**
   [WandB](https://wandb.ai) にアクセスして、API キーをコピーします。
   ```bash
   export WANDB_API_KEY=<your-api-key>
   ```

## Weave を使用したモデル呼び出しのログ記録

この例では、Weave を使用してモデル呼び出しをログに記録する方法を示します。

```python
import weave
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# プロジェクト名で Weave を初期化します
weave.init("agno")

# エージェントを作成して設定します
agent = Agent(model=OpenAIChat(id="gpt-4o"), markdown=True, debug_mode=True)

# weave.op() で装飾されたエージェントを実行する関数を定義します
@weave.op()
def run(content: str):
    return agent.run(content)

# 関数を使用してモデル呼び出しをログに記録します
run("Share a 2 sentence horror story")
```

## 注意事項

- **環境変数**: WandB API キーの環境変数が正しく設定されていることを確認してください。
- **初期化**: `weave.init("project-name")` を呼び出して、プロジェクト名で Weave を初期化します。
- **デコレータ**: Weave でログに記録したい関数を装飾するには `@weave.op()` を使用します。

これらの手順に従うことで、Agno を Weave と効果的に統合し、AI エージェントのモデル呼び出しの包括的なログ記録と視覚化を有効にすることができます。
