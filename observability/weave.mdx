## 将 Agno 与 WandB 的 Weave 集成

[Weights & Biases (WandB) 的 Weave](https://weave-docs.wandb.ai/) 提供了一个强大的平台来记录和可视化模型调用。通过将 Agno 与 Weave 集成，您可以跟踪和分析代理的性能和行为。

## 先决条件

1. **安装 Weave**

   确保已安装 Weave 包：

   ```bash
   pip install weave
   ```

2. **身份验证**
   前往 [WandB](https://wandb.ai) 并复制您的 API 密钥
   ```bash
   export WANDB_API_KEY=<your-api-key>
   ```

## 使用 Weave 记录模型调用

此示例演示了如何使用 Weave 记录模型调用。

```python
import weave
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# 使用您的项目名称初始化 Weave
weave.init("agno")

# 创建并配置代理
agent = Agent(model=OpenAIChat(id="gpt-4o"), markdown=True, debug_mode=True)

# 定义一个运行代理的函数，并使用 weave.op() 进行装饰
@weave.op()
def run(content: str):
    return agent.run(content)

# 使用该函数记录模型调用
run("Share a 2 sentence horror story")
```

## 注意事项

- **环境变量**: 确保您的环境变量已正确设置 WandB API 密钥。
- **初始化**: 调用 `weave.init("project-name")` 以您的项目名称初始化 Weave。
- **装饰器**: 使用 `@weave.op()` 装饰您希望使用 Weave 记录的函数。

通过遵循这些步骤，您可以有效地将 Agno 与 Weave 集成，从而实现对 AI 代理模型调用的全面记录和可视化。
