## AgnoとAgentOpsの統合

[AgentOps](https://app.agentops.ai/)は、Agnoエージェントの自動インストルメンテーションを提供し、エージェント間のやり取り、チームの連携、ツールの使用、ワークフローの実行など、すべての操作を追跡します。

## 前提条件

1. **AgentOpsのインストール**

   AgentOpsパッケージがインストールされていることを確認してください。

   ```bash
   pip install agentops
   ```

2. **認証**
   [AgentOps](https://app.agentops.ai/)にアクセスして、APIキーをコピーしてください。
   ```bash
   export AGENTOPS_API_KEY=<your-api-key>
   ```

## AgentOpsでのモデルコールのログ記録

この例では、AgentOpsを使用してモデルコールをログに記録する方法を示します。

```python
import agentops
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# AgentOpsの初期化
agentops.init()

# エージェントの作成と実行
agent = Agent(model=OpenAIChat(id="gpt-4o"))
response = agent.run("Share a 2 sentence horror story")

# レスポンスの表示
print(response.content)
```

## 注意事項

- **環境変数**: AgentOps APIキーの環境変数が正しく設定されていることを確認してください。
- **初期化**: AgentOpsを初期化するには、`agentops.init()`を呼び出してください。
- **AgentOpsドキュメント**: [AgentOps Docs](https://docs.agentops.ai/v2/integrations/agno)

これらの手順に従うことで、AgnoとAgentOpsが統合され、AIエージェントのモデルコールに関する包括的なログ記録と可視化が提供されます。
