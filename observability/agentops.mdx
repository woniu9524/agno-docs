## 将 Agno 与 AgentOps 集成

[AgentOps](https://app.agentops.ai/) 为您的 Agno 代理提供自动检测功能，以跟踪所有操作，包括代理交互、团队协调、工具使用和工作流执行。

## 先决条件

1. **安装 AgentOps**

   确保已安装 AgentOps 包：

   ```bash
   pip install agentops
   ```

2. **身份验证**
   前往 [AgentOps](https://app.agentops.ai/) 并复制您的 API 密钥
   ```bash
   export AGENTOPS_API_KEY=<your-api-key>
   ```

## 使用 AgentOps 记录模型调用

此示例演示如何使用 AgentOps 记录模型调用。

```python
import agentops
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# 初始化 AgentOps
agentops.init()

# 创建并运行代理
agent = Agent(model=OpenAIChat(id="gpt-4o"))
response = agent.run("Share a 2 sentence horror story")

# 打印响应
print(response.content)
```

## 注意事项

- **环境变量**: 确保您的环境变量已正确设置为 AgentOps API 密钥。
- **初始化**: 调用 `agentops.init()` 来初始化 AgentOps。
- **AgentOps 文档**: [AgentOps Docs](https://docs.agentops.ai/v2/integrations/agno)

遵循这些步骤将 Agno 与 AgentOps 集成，为您的 AI 代理的模型调用提供全面的日志记录和可视化。
