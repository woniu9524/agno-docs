## Agno と Arize Phoenix の統合

[Arize Phoenix](https://phoenix.arize.com/) は、AI モデルの監視と分析のための強力なプラットフォームです。Agno を Arize Phoenix と統合することで、OpenInference を活用してトレースを送信し、エージェントのパフォーマンスに関する洞察を得ることができます。

## 前提条件

1. **依存関係のインストール**

   必要なパッケージがインストールされていることを確認してください。

   ```bash
   pip install arize-phoenix openai openinference-instrumentation-agno opentelemetry-sdk opentelemetry-exporter-otlp
   ```

2. **Arize Phoenix アカウントの設定**

   - [Arize Phoenix](https://phoenix.arize.com/) でアカウントを作成します。
   - Arize Phoenix ダッシュボードから API キーを取得します。

3. **環境変数の設定**

   Arize Phoenix API キーで環境を構成します。

   ```bash
   export ARIZE_PHOENIX_API_KEY=<your-key>
   ```

## Arize Phoenix へのトレース送信

- ### 例: OpenInference と Arize Phoenix の使用

この例では、Agno エージェントを OpenInference でインストルメントし、Arize Phoenix にトレースを送信する方法を示します。

```python
import asyncio
import os

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from phoenix.otel import register

# Arize Phoenix の環境変数を設定
os.environ["PHOENIX_CLIENT_HEADERS"] = f"api_key={os.getenv('ARIZE_PHOENIX_API_KEY')}"
os.environ["PHOENIX_COLLECTOR_ENDPOINT"] = "https://app.phoenix.arize.com"

# Phoenix トレーサーの設定
tracer_provider = register(
    project_name="agno-stock-price-agent",  # デフォルトは 'default'
    auto_instrument=True,  # インストール済みの OpenInference インストルメンテーションを自動的に使用
)

# エージェントの作成と設定
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="あなたは株価エージェントです。株のアナリストのようなスタイルで回答してください。",
    debug_mode=True,
)

# エージェントの使用
agent.print_response("現在のテスラの株価はいくらですか？")
```

次に、[phoenix cloud](https://app.phoenix.arize.com) にアクセスして、エージェントによって作成されたトレースを表示します。Arize Phoenix ダッシュボードから直接、実行フローを視覚化し、パフォーマンスを監視し、問題をデバッグできます。

- ### 例: ローカルコレクターの設定

ローカル開発では、ローカルコレクターを実行できます。
```bash
phoenix serve
```

```python
import os

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from phoenix.otel import register

# ローカルコレクターのエンドポイントを設定
os.environ["PHOENIX_COLLECTOR_ENDPOINT"] = "http://localhost:6006"

# Phoenix トレーサーの設定
tracer_provider = register(
    project_name="agno-stock-price-agent",  # デフォルトは 'default'
    auto_instrument=True,  # インストール済みの OpenInference インストルメンテーションを自動的に使用
)

# エージェントの作成と設定
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="あなたは株価エージェントです。株のアナリストのようなスタイルで回答してください。",
    debug_mode=True,
)

# エージェントの使用
agent.print_response("現在のテスラの株価はいくらですか？")
```

## 注意事項

- **環境変数**: API キーとコレクター エンドポイントの環境変数が正しく設定されていることを確認してください。
- **ローカル開発**: 開発目的でローカル コレクターを起動するには、`phoenix serve` を使用します。

これらの手順に従うことで、Agno と Arize Phoenix を効果的に統合し、AI エージェントの包括的なオブザーバビリティと監視を有効にすることができます。
