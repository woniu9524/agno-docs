## Agent API コードベースを作成する

`agent-api` テンプレートを使用して、任意の名前でコードベースを作成します。

<CodeGroup>

```bash Mac
ag ws create --template agent-api --name agent-api
```

```bash Windows
ag ws create --template agent-api --name agent-api
```

</CodeGroup>

これにより、`agent-api` という名前のフォルダが作成され、以下の構造になります。

```bash
agent-api                     # ルートディレクトリ
├── agents                  # Agent をここに追加します
├── api                     # FastAPI ルートをここに追加します
├── db                      # データベーステーブルをここに追加します
├── Dockerfile              # アプリケーションの Dockerfile
├── pyproject.toml          # Python プロジェクト定義
├── requirements.txt        # pyproject.toml によって生成された Python 依存関係
├── scripts                 # ヘルパースクリプト
├── utils                   # 共有ユーティリティ
└── workspace               # agno ワークスペースディレクトリ
    ├── dev_resources.py    # ローカルで実行される開発リソース
    ├── prd_resources.py    # AWS で実行される本番リソース
    ├── secrets             # シークレット
    └── settings.py         # agno ワークスペース設定
