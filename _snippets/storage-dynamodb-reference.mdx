### 参数

| 参数 | 类型 | 说明 | 默认值 |
|-----------|------|-------------|---------|
| `table_name` | `str` | DynamoDB 表的名称 | 必需 |
| `region_name` | `Optional[str]` | AWS 区域名称 | `None` |
| `aws_access_key_id` | `Optional[str]` | AWS 访问密钥 ID | `None` |
| `aws_secret_access_key` | `Optional[str]` | AWS 密钥访问密钥 | `None` |
| `endpoint_url` | `Optional[str]` | 自定义端点 URL | `None` |
| `create_table_if_not_exists` | `bool` | 如果表不存在，是否自动创建 | `True` |
