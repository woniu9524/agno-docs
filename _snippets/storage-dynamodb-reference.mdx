### パラメータ

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `table_name` | `str` | DynamoDB テーブルの名前 | 必須 |
| `region_name` | `Optional[str]` | AWS リージョンの名前 | `None` |
| `aws_access_key_id` | `Optional[str]` | AWS アクセスキー ID | `None` |
| `aws_secret_access_key` | `Optional[str]` | AWS シークレットアクセスキー | `None` |
| `endpoint_url` | `Optional[str]` | カスタムエンドポイント URL | `None` |
| `create_table_if_not_exists` | `bool` | テーブルが存在しない場合に自動作成する | `True` |
