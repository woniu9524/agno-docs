## 构建你的 Web 应用

[Django 教程](https://docs.djangoproject.com/en/4.2/intro/tutorial01/) 是学习 Django、进行试验和根据你的用例更新 Web 应用的绝佳起点。以下是如何创建一个简单的聊天应用程序：

**1. 创建一个新的 Django 应用**

```bash
docker exec -it django-dev-app python manage.py startapp chat
```

<Note>

在此了解有关 [Django 应用](https://docs.djangoproject.com/en/4.2/intro/tutorial01/#creating-the-polls-app) 的更多信息

</Note>

**2. 注册 chat 应用**

更新 `app/settings.py` 文件并注册 `chat` 应用

```python app/settings.py
...
INSTALLED_APPS = [
    "django.contrib.admin",
    "django.contrib.auth",
    "django.contrib.contenttypes",
    "django.contrib.sessions",
    "django.contrib.messages",
    "django.contrib.staticfiles",
    # 注册 chat 应用
    'chat',
]
...
```

**3. 为 chat 应用创建视图**

将 `chat/views.py` 文件更新为：

```python chat/views.py
from django.shortcuts import render, redirect

from agno.agent import Agent

def index(request):
    try:
        # 创建一个 agent
        agent = Agent()
        if 'messages' not in request.session:
            request.session['messages'] = []
        if request.method == 'POST':
            prompt = request.POST.get('prompt')
            # 将 prompt 添加到 messages
            request.session['messages'].append({"role": "user", "content": prompt})
            # 将 session 设置为已修改
            request.session.modified = True
            # 创建一个响应
            response = agent.run(prompt, stream=False)
            # 将响应添加到 messages
            request.session['messages'].append({"role": "agent", "content": response})
            # 将 session 设置为已修改
            request.session.modified = True
            # 重定向到主页
            context = {
                'messages': request.session['messages'],
                'prompt': '',
            }
            return render(request, 'chat/index.html', context)
        else:
            context = {
                'messages': request.session['messages'],
                'prompt': '',
            }
            return render(request, 'chat/index.html', context)
    except Exception as e:
        print(e)
        return redirect('index')


def new_chat(request):
    # -*- 清除 session messages 并重定向到主页 -*-
    request.session.pop('agent', None)
    request.session.pop('messages', None)
    return redirect('index')
```

**4. 配置 URL**

在 chat 文件夹中创建一个名为 `urls.py` 的文件：

```python chat/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
    path('new_chat/', views.new_chat, name='new_chat'),
]
```

将 `app/urls.py` 文件更新为：

```python app/urls.py
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path("admin/", admin.site.urls),
    path('chat/', include('chat.urls')),
]
```

**5. 创建模板**

在 `chat/templates/chat` 文件夹中为聊天界面创建 HTML 模板。

```python chat/templates/chat/base.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AI Chat | {% block title %}  {% endblock %}</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    {% block content %}
    {% endblock %}
</body>
</html>
```

```python chat/templates/chat/index.html
{% extends 'chat/base.html' %}
{% block title %} Home {% endblock %}
{% block content %}
<div class="row justify-content-center my-4">
    <div class="col-md-7 mt-4">
        <div class="card">
            <h2 class="card-header text-center">LLM Chat</h2>
            <div class="card-body">
              <div class="d-flex justify-content-end">
                <button type="button" class="btn btn-primary mb-3" onclick="location.href='{% url 'new_chat' %}'">New Chat</button>
              </div>
              <div class="chat-history mb-3">
                {% for message in messages %}
                  <div class="card mb-2 {% if message.role == 'agent' %}bg-success text-white{% endif %}">
                    <div class="card-body p-2">
                      <strong>{{ message.role|title }}:</strong> {{ message.content|linebreaksbr }}
                    </div>
                  </div>
                {% endfor %}
              </div>
              <form action="." method="POST">
                <!-- 这可以防止在提交过程中出现恶意攻击 -->
                {% csrf_token %}
                <label for="prompt" class="form-label">发送消息</label>
                <input class="form-control mb-2" required type="text" autofocus="autofocus" name="prompt" value="{{ prompt }}" id="">
                <button class="btn btn-success fw-bold" type="submit">
                  发送
                </button>
              </form>
            </div>
        </div>
    </div>
</div>
{% endblock %}
```

**6. 打开 Django 应用**

打开 [localhost:8000/chat/](http://localhost:8000/chat/) 来查看 LLM 聊天应用。
