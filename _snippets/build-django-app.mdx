## Web アプリの構築

[Django チュートリアル](https://docs.djangoproject.com/en/4.2/intro/tutorial01/)は、Django について学び、実際に試して、Web アプリをユースケースに合わせて更新するための優れたリソースです。簡単なチャット アプリケーションを作成する方法を以下に示します。

**1. 新しい Django アプリを作成する**

```bash
docker exec -it django-dev-app python manage.py startapp chat
```

<Note>

[Django アプリの詳細はこちら](https://docs.djangoproject.com/en/4.2/intro/tutorial01/#creating-the-polls-app)をご覧ください。

</Note>

**2. チャット アプリを登録する**

`app/settings.py` ファイルを更新して、`chat` を登録します。

```python app/settings.py
...
INSTALLED_APPS = [
    "django.contrib.admin",
    "django.contrib.auth",
    "django.contrib.contenttypes",
    "django.contrib.sessions",
    "django.contrib.messages",
    "django.contrib.staticfiles",
    # chat アプリを登録
    'chat',
]
...
```

**3. チャット アプリのビューを作成する**

`chat/views.py` ファイルを以下のように更新します。

```python chat/views.py
from django.shortcuts import render, redirect

from agno.agent import Agent

def index(request):
    try:
        # エージェントを作成
        agent = Agent()
        if 'messages' not in request.session:
            request.session['messages'] = []
        if request.method == 'POST':
            prompt = request.POST.get('prompt')
            # プロンプトをメッセージに追加
            request.session['messages'].append({"role": "user", "content": prompt})
            # セッションを変更したことを設定
            request.session.modified = True
            # レスポンスを作成
            response = agent.run(prompt, stream=False)
            # レスポンスをメッセージに追加
            request.session['messages'].append({"role": "agent", "content": response})
            # セッションを変更したことを設定
            request.session.modified = True
            # ホームページにリダイレクト
            context = {
                'messages': request.session['messages'],
                'prompt': '',
            }
            return render(request, 'chat/index.html', context)
        else:
            context = {
                'messages': request.session['messages'],
                'prompt': '',
            }
            return render(request, 'chat/index.html', context)
    except Exception as e:
        print(e)
        return redirect('index')


def new_chat(request):
    # -*- セッションメッセージをクリアしてホームページにリダイレクト -*-
    request.session.pop('agent', None)
    request.session.pop('messages', None)
    return redirect('index')
```

**4. URL を設定する**

`chat` フォルダに `urls.py` という名前のファイルを作成します。

```python chat/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
    path('new_chat/', views.new_chat, name='new_chat'),
]
```

`app/urls.py` ファイルを以下のように更新します。

```python app/urls.py
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path("admin/", admin.site.urls),
    path('chat/', include('chat.urls')),
]
```

**5. テンプレートを作成する**

`chat/templates/chat` フォルダにチャット インターフェイス用の HTML テンプレートを作成します。

```python chat/templates/chat/base.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AI Chat | {% block title %}  {% endblock %}</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    {% block content %}
    {% endblock %}
</body>
</html>
```

```python chat/templates/chat/index.html
{% extends 'chat/base.html' %}
{% block title %} Home {% endblock %}
{% block content %}
<div class="row justify-content-center my-4">
    <div class="col-md-7 mt-4">
        <div class="card">
            <h2 class="card-header text-center">LLM Chat</h2>
            <div class="card-body">
              <div class="d-flex justify-content-end">
                <button type="button" class="btn btn-primary mb-3" onclick="location.href='{% url 'new_chat' %}'">New Chat</button>
              </div>
              <div class="chat-history mb-3">
                {% for message in messages %}
                  <div class="card mb-2 {% if message.role == 'agent' %}bg-success text-white{% endif %}">
                    <div class="card-body p-2">
                      <strong>{{ message.role|title }}:</strong> {{ message.content|linebreaksbr }}
                    </div>
                  </div>
                {% endfor %}
              </div>
              <form action="." method="POST">
                <!-- これは、送信中の悪意のある攻撃からフォームを保護します -->
                {% csrf_token %}
                <label for="prompt" class="form-label">Send a message</label>
                <input class="form-control mb-2" required type="text" autofocus="autofocus" name="prompt" value="{{ prompt }}" id="">
                <button class="btn btn-success fw-bold" type="submit">
                  Send
                </button>
              </form>
            </div>
        </div>
    </div>
</div>
{% endblock %}
```

**6. Django アプリを開く**

[localhost:8000/chat/](http://localhost:8000/chat/) を開いて、LLM チャット アプリを表示します。
