| パラメータ            | タイプ                                   | デフォルト    | 説明                                                                                             |
| -------------------- | -------------------------------------- | ---------- | ------------------------------------------------------------------------------------------------ |
| `name`               | `str`                                  | -          | Pinecone インデックスの名前                                                                      |
| `dimension`          | `int`                                  | -          | 埋め込みの次元数                                                                                 |
| `spec`               | `Union[Dict, ServerlessSpec, PodSpec]` | -          | インデックスの仕様                                                                               |
| `embedder`           | `Optional[Embedder]`                   | `None`     | 埋め込みを作成するためのEmbedderインスタンス（提供されない場合はOpenAIEmbedderがデフォルト） |
| `metric`             | `Optional[str]`                        | `"cosine"` | 類似性検索に使用されるメトリック                                                                 |
| `additional_headers` | `Optional[Dict[str, str]]`             | `None`     | Pinecone クライアントに渡す追加ヘッダー                                                          |
| `pool_threads`       | `Optional[int]`                        | `1`        | Pinecone クライアントに使用するスレッド数                                                        |
| `namespace`          | `Optional[str]`                        | `None`     | Pinecone インデックスの名前空間                                                                  |
| `timeout`            | `Optional[int]`                        | `None`     | Pinecone 操作のタイムアウト                                                                      |
| `index_api`          | `Optional[Any]`                        | `None`     | Index API オブジェクト                                                                           |
| `api_key`            | `Optional[str]`                        | `None`     | Pinecone API キー                                                                                |
| `host`               | `Optional[str]`                        | `None`     | Pinecone ホスト                                                                                  |
| `config`             | `Optional[Config]`                     | `None`     | Pinecone 設定                                                                                    |
| `use_hybrid_search`  | `bool`                                 | `False`    | ハイブリッド検索を使用するかどうか                                                               |
| `hybrid_alpha`       | `float`                                | `0.5`      | ハイブリッド検索のアルファ値                                                                     |
