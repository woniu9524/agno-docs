| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `session_id` | `str` | Required | セッションのUUID |
| `agent_id` | `Optional[str]` | `None` | このセッションが関連付けられているエージェントのID |
| `user_id` | `Optional[str]` | `None` | このエージェントと対話しているユーザーのID |
| `team_session_id` | `Optional[str]` | `None` | このセッションが関連付けられている可能性のあるチームセッションのID |
| `memory` | `Optional[Dict[str, Any]]` | `None` | エージェントのメモリ |
| `agent_data` | `Optional[Dict[str, Any]]` | `None` | エージェントのデータ: agent_id、名前、モデル |
| `session_data` | `Optional[Dict[str, Any]]` | `None` | セッションのデータ: session_name、session_state、画像、動画、音声 |
| `extra_data` | `Optional[Dict[str, Any]]` | `None` | このエージェントと共に保存されている追加データ |
| `created_at` | `Optional[int]` | `None` | このセッションが作成されたときのUnixタイムスタンプ |
| `updated_at` | `Optional[int]` | `None` | このセッションが最後に更新されたときのUnixタイムスタンプ |
