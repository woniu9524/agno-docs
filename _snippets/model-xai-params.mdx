| パラメータ  | タイプ            | デフォルト                    | 説明                                                                                                               |
| ---------- | --------------- | -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `id`       | `str`           | `"grok-beta"`              | レスポンス生成に使用される特定のモデルID。                                                                           |
| `name`     | `str`           | `"xAI"`                    | xAIエージェントの名前識別子。                                                                                        |
| `provider` | `str`           | `"xAI"`                    | モデルのプロバイダーで、「xAI」とモデルIDを組み合わせたもの。                                                              |
| `api_key`  | `Optional[str]` | -                          | xAIサービスへのリクエスト認証用のAPIキー。環境変数 `XAI_API_KEY` から取得されます。                                        |
| `base_url` | `str`           | `"https://api.xai.xyz/v1"` | xAIサービスへのAPIリクエストを行うためのベースURL。                                                                   |
