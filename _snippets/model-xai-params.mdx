| 参数                | 类型                              | 默认值                    | 描述                                                                                                                                                             |
| ------------------- | --------------------------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `id`                | `str`                             | `"grok-3"`                 | 用于生成响应的特定模型 ID。                                                                                                                                    |
| `name`              | `str`                             | `"xAI"`                    | xAI Agent 的名称标识符。                                                                                                                                       |
| `provider`          | `str`                             | `"xAI"`                    | 模型的提供者，结合了 "xAI" 和模型 ID。                                                                                                                         |
| `api_key`           | `Optional[str]`                   | `None`                     | 用于向 xAI 服务进行身份验证请求的 API 密钥。从环境变量 `XAI_API_KEY` 中检索。                                                                                   |
| `base_url`          | `str`                             | `"https://api.x.ai/v1"`    | 向 xAI 服务发出 API 请求的基础 URL。                                                                                                                             |
| `search_parameters` | `Optional[Dict[str, Any]]`        | `None`                     | 用于启用实时搜索功能的参数。支持 [`mode`](#search_parameters), [`max_search_results`](#search_parameters), and [`return_citations`](#search_parameters)。 |
