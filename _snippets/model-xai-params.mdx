| 参数  | 类型            | 默认值                    | 描述                                                                                                               |
| ----- | --------------- | -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `id`  | `str`           | `"grok-beta"`              | 用于生成响应的特定模型 ID。                                                                                        |
| `name`| `str`           | `"xAI"`                    | xAI 代理的名称标识符。                                                                                             |
| `provider`| `str`           | `"xAI"`                    | 模型的提供者，将“xAI”与模型 ID 结合。                                                                               |
| `api_key`| `Optional[str]` | -                          | 用于向 xAI 服务进行身份验证的 API 密钥。从环境变量 `XAI_API_KEY` 中检索。                                           |
| `base_url`| `str`           | `"https://api.xai.xyz/v1"` | 向 xAI 服务发出 API 请求的基础 URL。                                                                               |
