| 参数 | 类型 | 默认值 | 描述 |

| model | str | "rerank-multilingual-v3.0" | 用于重新排序的 Cohere 模型。 |
| api_key | Optional[str] | None | Cohere API 密钥。如果未提供，将尝试使用环境变量。 |
| cohere_client | Optional[CohereClient] | None | 一个可选的预先配置好的 Cohere 客户端实例。 |
