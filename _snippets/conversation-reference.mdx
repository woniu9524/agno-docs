```python

    message: str,
    references: Optional[str] = None,

    ...
```

将 `Conversation` 对象和输入的 `message` 作为参数提供给此函数。它应返回用户提示字符串。

如果 `add_references_to_prompt` 为 True，则还会提供 `references` 作为参数。

如果 `add_chat_history_to_prompt` 为 True，则还会提供 `chat_history` 作为参数。
