## オプション: Jupyterlab の実行

Jupyter Notebook は AI 開発に必須であり、`ai-app` には必要な依存関係がプリインストールされたノートブックが付属しています。ノートブックを開始するには、以下を実行してください。

<Steps>
  <Step title="Jupyter の有効化">
    `workspace/settings.py` ファイルを更新し、`dev_jupyter_enabled=True` を設定します。

    ```python workspace/settings.py
    ...
    ws_settings = WorkspaceSettings(
        ...
        # 次の行のコメントを解除します
        dev_jupyter_enabled=True,
    ...
    ```

  </Step>
  <Step title="Jupyter の起動">

    <CodeGroup>

    ```bash terminal
    ag ws up --group jupyter
    ```

    ```bash shorthand
    ag ws up dev:docker:jupyter
    ```

    </CodeGroup>

    **Enter キーを押して**確認し、イメージのダウンロードに数分かかります（初回のみ）。Docker ダッシュボードでコンテナのステータスを確認し、ログを表示します。

  </Step>
  <Step title="JupyterLab UI の表示">
    - [localhost:8888](http://localhost:8888) を開いて Jupyterlab UI を表示します。パスワード: **admin**
    - `notebooks` フォルダ内のクックブックを試してみてください。

    ![Jupyter Notebook](/images/ai-app-jupyter-local.png)

  </Step>
</Steps>
