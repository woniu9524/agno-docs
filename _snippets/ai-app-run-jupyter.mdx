## 可选：运行 Jupyterlab

Jupyter notebook 是 AI 开发的必备工具，你的 `ai-app` 预装了 notebook 和所需的依赖项。要启动你的 notebook：

<Steps>
  <Step title="启用 Jupyter">
    更新 `workspace/settings.py` 文件并将 `dev_jupyter_enabled` 设置为 `True`

    ```python workspace/settings.py
    ...
    ws_settings = WorkspaceSettings(
        ...
        # 取消注释以下行
        dev_jupyter_enabled=True,
    ...
    ```

  </Step>
  <Step title="启动 Jupyter">

    <CodeGroup>

    ```bash terminal
    ag ws up --group jupyter
    ```

    ```bash shorthand
    ag ws up dev:docker:jupyter
    ```

    </CodeGroup>

    按 **Enter** 键确认，并等待几分钟以下载镜像（仅首次运行）。在 Docker Dashboard 上验证容器状态并查看日志。

  </Step>
  <Step title="查看 JupyterLab UI">
    - 打开 [localhost:8888](http://localhost:8888) 查看 Jupyterlab UI。密码：**admin**
    - 在 `notebooks` 文件夹中尝试 cookbooks。

    ![Jupyter Notebook](/images/ai-app-jupyter-local.png)

  </Step>
</Steps>
