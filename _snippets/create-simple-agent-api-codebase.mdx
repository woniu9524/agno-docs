## 文件夹结构

`agent-api` 文件夹包含以下结构：

```bash
agent-api                     # 根目录
├── agents                  # 在此处添加您的 Agents
├── api                     # 在此处添加 FastAPI 路由
├── db                      # 在此处添加数据库表
├── Dockerfile              # 应用程序的 Dockerfile
├── pyproject.toml          # Python 项目定义
├── requirements.txt        # 由 pyproject.toml 生成的 Python 依赖项
├── scripts                 # 辅助脚本
```

#### 预构建 Agent

`/agents` 文件夹包含您可以作为起点使用的预构建 Agent。

- **Web Search Agent**: 一个可以搜索网络的简单 Agent。
- **Agno Assist**: 一个可以帮助回答有关 Agno 问题的 Agent。
    - **重要提示**: 在使用此 Agent 之前，请确保加载 `agno_assist` 知识库。
- **Finance Agent**: 一个使用 Yahoo Finance API 获取股票价格和财务数据的 Agent。
