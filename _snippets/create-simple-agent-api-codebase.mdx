## フォルダ構造

`agent-api` フォルダには以下の構造が含まれています。

```bash
agent-api                     # ルートディレクトリ
├── agents                  # エージェントをここに追加します
├── api                     # FastAPI ルートをここに追加します
├── db                      # データベーステーブルをここに追加します
├── Dockerfile              # アプリケーションのDockerfile
├── pyproject.toml          # Pythonプロジェクト定義
├── requirements.txt        # pyproject.toml によって生成されたPython依存関係
├── scripts                 # ヘルパースクリプト
```

#### 事前構築済みエージェント

`/agents` フォルダには、開始点として使用できる事前構築済みエージェントが含まれています。

- **Web Search Agent**: Web を検索できるシンプルなエージェントです。
- **Agno Assist**: Agno に関する質問に答えるのに役立つエージェントです。
    - **重要**: このエージェントを使用する前に、`agno_assist` ナレッジベースをロードしてください。
- **Finance Agent**: Yahoo Finance API を使用して株価や財務データを取得するエージェントです。
