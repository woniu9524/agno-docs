| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `session_id` | `str` | Required | セッションのUUID |
| `team_id` | `Optional[str]` | `None` | このセッションが関連付けられているチームのID |
| `user_id` | `Optional[str]` | `None` | このチームとやり取りしているユーザーのID |
| `memory` | `Optional[Dict[str, Any]]` | `None` | チームのメモリ |
| `team_data` | `Optional[Dict[str, Any]]` | `None` | チームデータ: team_id、名前、モデル |
| `session_data` | `Optional[Dict[str, Any]]` | `None` | セッションデータ: session_name、session_state、images、videos、audio |
| `extra_data` | `Optional[Dict[str, Any]]` | `None` | このチームと共に保存されている追加データ |
| `created_at` | `Optional[int]` | `None` | このセッションが作成されたときのUnixタイムスタンプ |
| `updated_at` | `Optional[int]` | `None` | このセッションが最後に更新されたときのUnixタイムスタンプ |
