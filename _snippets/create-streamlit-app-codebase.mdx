## コードベースの作成

`streamlit-app` テンプレートを使用してコードベースを作成します。

<CodeGroup>

```bash Mac
ag ws create -t streamlit-app -n streamlit-app
```

```bash Windows
ag ws create -t streamlit-app -n streamlit-app
```

</CodeGroup>

これにより、`streamlit-app` というフォルダが作成され、以下の構造になります。

```bash
streamlit-app                 # streamlit-app のルートディレクトリ
├── ai                      # AI コンポーネントのディレクトリ
    ├── agents          # AI エージェント
    ├── knowledge_base.py   # エージェントのナレッジベース
    └── storage.py          # エージェントのストレージ
├── app                     # Streamlit アプリのディレクトリ
├── db                      # データベースコンポーネントのディレクトリ
├── Dockerfile              # アプリケーションの Dockerfile
├── pyproject.toml          # python プロジェクトの定義
├── requirements.txt        # pyproject.toml によって生成された python 依存関係
├── scripts                 # ヘルパースクリプトのディレクトリ
├── tests                   # 単体テストのディレクトリ
├── utils                   # 共有ユーティリティのディレクトリ
└── workspace               # agno ワークスペースディレクトリ
    ├── dev_resources.py    # ローカルで実行される開発リソース
    ├── prd_resources.py    # AWS 上で実行される本番リソース
    ├── secrets             # シークレットを格納するディレクトリ
    └── settings.py         # agno ワークスペースの設定
