## 创建你的代码库

使用 `streamlit-app` 模板创建你的代码库

<CodeGroup>

```bash Mac
ag ws create -t streamlit-app -n streamlit-app
```

```bash Windows
ag ws create -t streamlit-app -n streamlit-app
```

</CodeGroup>

这将会创建一个名为 `streamlit-app` 的文件夹，其结构如下：

```bash
streamlit-app                 # streamlit-app 的根目录
├── ai                      # AI 组件目录
    ├── agents          # AI 代理
    ├── knowledge_base.py   # 代理知识库
    └── storage.py          # 代理存储
├── app                     # Streamlit 应用目录
├── db                      # 数据库组件目录
├── Dockerfile              # 应用的 Dockerfile
├── pyproject.toml          # Python 项目定义
├── requirements.txt        # 由 pyproject.toml 生成的 Python 依赖
├── scripts                 # 辅助脚本目录
├── tests                   # 单元测试目录
├── utils                   # 共享工具目录
└── workspace               # agno workspace 目录
    ├── dev_resources.py    # 在本地运行的开发资源
    ├── prd_resources.py    # 在 AWS 上运行的生产资源
    ├── secrets             # 用于存储密钥的目录
    └── settings.py         # agno workspace 设置
```
