### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于 embeddings 的模型 ID | `"jinaai/jina-embeddings-v2-base-code"` |
| `api_key` | `Optional[str]` | Huggingface API 密钥 | 环境变量 `HUGGINGFACE_API_KEY` |
| `client_params` | `Optional[Dict[str, Any]]` | 用于客户端初始化的附加参数 | `None` |
| `huggingface_client` | `Optional[InferenceClient]` | 预配置的 Huggingface 客户端 | `None` |
