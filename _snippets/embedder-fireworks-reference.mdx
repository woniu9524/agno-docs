### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于嵌入的模型 ID | `"nomic-ai/nomic-embed-text-v1.5"` |
| `dimensions` | `int` | 嵌入的输出维度 | `768` |
| `api_key` | `Optional[str]` | Fireworks API 密钥 | 环境变量 `FIREWORKS_API_KEY` |
| `base_url` | `str` | API 请求的基础 URL | `"https://api.fireworks.ai/inference/v1"` |
