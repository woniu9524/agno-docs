## セットアップと設定
<Steps>

<Step title="前提条件">
以下を確認してください：
- Meta 開発者アカウント
- Meta ビジネスアカウント
- 有効な Facebook アカウント
- ngrok (開発用)
- Python 3.7 以降
</Step>

<Step title="Meta アプリの作成">
1. [Meta for Developers](https://developers.facebook.com/) にアクセスしてアカウントを確認してください。
2. [Meta Apps ダッシュボード](https://developers.facebook.com/apps/) で新しいアプリを作成してください。
3. 「ユースケース」で「その他」を選択してください。
4. アプリの種類として「ビジネス」を選択してください。
5. 以下を入力してください：
    - アプリ名
    - 連絡先メールアドレス
6. 「アプリを作成」をクリックしてください。
</Step>

<Step title="Meta ビジネスアカウントの設定">
1. [Meta Business Manager](https://business.facebook.com/) に移動してください。
2. 新しいビジネスアカウントを作成するか、既存のものを利用してください。
3. メールリンクをクリックしてビジネスを認証してください。
4. アプリのページに移動し、「アプリ設定 / 基本」に移動して、「ビジネス認証」の下にある「認証を開始」をクリックしてください。本番環境用の認証プロセスを完了してください。
5. アプリをビジネスアカウントに関連付け、「アプリを作成」をクリックしてください。
</Step>

<Step title="WhatsApp Business API の設定">
1. アプリの WhatsApp 設定ページに移動してください。
2. 「API の使用を開始」をクリックしてください (API セットアップ)。
3. アクセストークンを生成してください。
4. 電話番号 ID をコピーしてください。
5. WhatsApp ビジネスアカウント ID をコピーしてください。
6. テストに使用する「宛先」番号を追加してください (これはおそらく個人の番号になります)。
</Step>

<Step title="環境変数の設定">
プロジェクトのルートに `.envrc` ファイルを作成し、プレースホルダー値を実際の認証情報に置き換えて、以下の内容を追加してください：
```bash
export WHATSAPP_ACCESS_TOKEN="your_whatsapp_access_token"
export WHATSAPP_PHONE_NUMBER_ID="your_phone_number_id"
export WHATSAPP_WEBHOOK_URL="your_ngrok_url_plus_webhook_path" # 例: https://xxxxx.ngrok-free.app/webhook
export WHATSAPP_VERIFY_TOKEN="your_chosen_verify_token" # 作成した文字列
```
このファイルがシェルによってソースされていることを確認してください (例: `direnv allow` を使用)。
</Step>

<Step title="ngrok を使用した Webhook の設定">
1. ローカル開発の場合、ngrok を使用してローカルサーバーをインターネットに公開してください。静的な ngrok URL がない場合は、ngrok が新しい URL を割り当てるたびに `WHATSAPP_WEBHOOK_URL` 環境変数と Meta アプリの Webhook 設定を更新する必要があります。
2. ngrok を実行し、ポートが Agno WhatsApp アプリが実行されるポート (例: 8000) と一致していることを確認してください：
   ```bash
   ngrok http 8000
   # または、静的ドメインを持つ有料 ngrok プランをお持ちの場合：
   # ngrok http --domain=your-custom-domain.ngrok-free.app 8000
   ```
3. ngrok によって提供される `https://` URL をコピーしてください。これがベースの ngrok URL です。
4. ngrok URL に `/webhook` (または選択したプレフィックス) を追加して、完全な Webhook URL を構築してください (例: `https://<random-string>.ngrok-free.app/webhook`)。必要に応じて `.envrc` の `WHATSAPP_WEBHOOK_URL` を更新してください。
5. Meta アプリの WhatsApp 設定ページで、「Webhook」セクションに移動し、「編集」をクリックしてください。
6. Webhook を設定してください：
    -   **コールバック URL**: 完全な ngrok Webhook URL を入力してください。
    -   **Verify Token**: `.envrc` ファイルで使用した値と同じ値を入力してください。
7. 「検証して保存」をクリックしてください。検証を成功させるには、Agno アプリケーションがローカルで実行されている必要があります。
8. 検証が成功したら、Webhook フィールドの横にある「管理」をクリックしてください。`whatsapp_business_account` の下にある `messages` フィールドを購読してください。
</Step>

<Step title="アプリケーション環境の設定">
`APP_ENV` 環境変数を設定してください：
- **開発モード**の場合：
  ```bash
  export APP_ENV="development"
  ```
  (Webhook シグネチャ検証は、より緩やかになるか、バイパスされる場合があります)。
- **本番モード**の場合：
  ```bash
  export APP_ENV="production"
  ```
  Webhook シグネチャ検証のために `WHATSAPP_APP_SECRET` も設定する必要があります：
  ```bash
  export WHATSAPP_APP_SECRET="your_meta_app_secret"
  ```
  これは、Meta アプリの「アプリ設定 > 基本」ページにある「アプリシークレット」である必要があります。
</Step>

</Steps>
