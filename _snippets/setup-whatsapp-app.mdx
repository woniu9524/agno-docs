## 设置和配置
<Steps>

<Step title="先决条件">
确保您拥有以下条件：
- Meta开发者账户
- Meta商业账户
- 有效的Facebook账户
- ngrok（用于开发）
- Python 3.7+
</Step>

<Step title="创建Meta应用">
1. 前往 [Meta for Developers](https://developers.facebook.com/) 并验证您的账户。
2. 在 [Meta Apps Dashboard](https://developers.facebook.com/apps/) 创建新应用。
3. 在“用例”下选择“其他”。
4. 选择“Business”作为应用类型。
5. 提供：
    - 应用名称
    - 联系邮箱
6. 点击“创建应用”。
</Step>

<Step title="设置Meta商业账户">
1. 导航至 [Meta Business Manager](https://business.facebook.com/)。
2. 创建新的商业账户或使用现有账户。
3. 通过点击邮件链接验证您的业务。
4. 前往您的应用页面，导航至“应用设置/基本信息”，然后在“业务验证”下点击“开始验证”。完成生产环境的验证流程。
5. 将应用与您的商业账户关联，然后点击“创建应用”。
</Step>

<Step title="设置WhatsApp Business API">
1. 前往您应用的WhatsApp设置页面。
2. 点击“开始使用API”（API设置）。
3. 生成访问令牌。
4. 复制您的电话号码 ID。
5. 复制您的WhatsApp Business账户 ID。
6. 添加一个您将用于测试的“收件人”号码（这很可能是您的个人号码）。
</Step>

<Step title="设置环境变量">
在项目根目录下创建一个 `.envrc` 文件，并包含以下内容，将占位符值替换为您实际的凭证：
```bash
export WHATSAPP_ACCESS_TOKEN="your_whatsapp_access_token"
export WHATSAPP_PHONE_NUMBER_ID="your_phone_number_id"
export WHATSAPP_WEBHOOK_URL="your_ngrok_url_plus_webhook_path" # 例如：https://xxxxx.ngrok-free.app/webhook
export WHATSAPP_VERIFY_TOKEN="your_chosen_verify_token" # 您创建的一个字符串
```
确保您的 shell 已加载此文件（例如，通过使用 `direnv allow`）。
</Step>

<Step title="使用ngrok设置Webhook">
1. 对于本地开发，使用ngrok将您的本地服务器暴露给互联网。如果您没有静态ngrok URL，您需要在每次ngrok分配新URL时更新 `WHATSAPP_WEBHOOK_URL` 环境变量和您的Meta应用Webhook配置。
2. 运行ngrok，确保端口与您的Agno WhatsApp应用将运行的端口匹配（例如，8000）：
   ```bash
   ngrok http 8000
   # 或者，如果您有付费ngrok计划并使用自定义域名：
   # ngrok http --domain=your-custom-domain.ngrok-free.app 8000
   ```
3. 复制ngrok提供的 `https://` URL。这是您的基础ngrok URL。
4. 通过在ngrok URL后附加 `/webhook`（或您选择的前缀）来构建完整的Webhook URL（例如，`https://<random-string>.ngrok-free.app/webhook`）。如有必要，请更新 `.envrc` 中的 `WHATSAPP_WEBHOOK_URL`。
5. 在您的Meta应用WhatsApp设置页面，导航到“Webhook”部分并点击“编辑”。
6. 配置Webhook：
    - **回调URL**：输入您的完整ngrok Webhook URL。
    - **验证令牌**：输入您在 `.envrc` 文件中为 `WHATSAPP_VERIFY_TOKEN` 使用的相同值。
7. 点击“验证并保存”。您的Agno应用程序必须在本地运行才能成功验证。
8. 成功验证后，点击Webhook字段旁边的“管理”。在 `whatsapp_business_account` 下订阅 `messages` 字段。
</Step>

<Step title="配置应用程序环境">
设置 `APP_ENV` 环境变量：
- 对于**开发模式**：
  ```bash
  export APP_ENV="development"
  ```
  （Webhook签名验证可能不那么严格或被绕过）。
- 对于**生产模式**：
  ```bash
  export APP_ENV="production"
  ```
  您还需要设置 `WHATSAPP_APP_SECRET` 以进行Webhook签名验证：
  ```bash
  export WHATSAPP_APP_SECRET="your_meta_app_secret"
  ```
  这应该是您在Meta应用“应用设置 > 基本信息”页面找到的“应用密钥”。
</Step>

</Steps>
