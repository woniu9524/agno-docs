| 参数                | 类型     | 默认值          | 描述                                                         |
| ----------------------- | -------- | --------------- | ---------------------------------------------------------- |
| `id`                    | `str`    | `"not-provided"`| 用于生成响应的模型名称。                                       |
| `name`                  | `str`    | `"OpenAILike"`  | 此聊天模型实例的名称。                                       |
| `api_key`               | `str`    | `"not-provided"`| 用于对服务请求进行身份验证的 API 密钥。                          |
| `override_system_role`  | `bool`   | `False`         | 是否覆盖默认的系统角色。                                       |
| `system_message_role`   | `str`    | `"system"`      | 用于系统消息的角色。                                          |
