| パラメータ     | タイプ           | デフォルト | 説明                                                                                                                   |
| -------------- | -------------- | ------- | ---------------------------------------------------------------------------------------------------------------------- |
| `uri`         | `str`          | -       | 接続先のURI。                                                                                                          |
| `table`       | `LanceTable`   | -       | 使用するLanceテーブル。                                                                                                |
| `table_name`  | `str`          | -       | 使用するテーブル名。                                                                                                   |
| `connection`  | `DBConnection` | -       | 使用するデータベース接続。                                                                                             |
| `api_key`     | `str`          | -       | 使用するAPIキー。                                                                                                      |
| `embedder`    | `Embedder`     | -       | 使用するエンベッダー。                                                                                                 |
| `search_type` | `SearchType`   | vector  | 使用する検索タイプ。                                                                                                   |
| `distance`    | `Distance`     | cosine  | 使用する距離。                                                                                                         |
| `nprobes`     | `int`          | -       | 使用するプローブ数。[詳細情報](https://lancedb.github.io/lancedb/ann_indexes/#use-gpu-to-build-vector-index) |
| `reranker`    | `Reranker`     | -       | 使用するリランカー。[詳細情報](https://lancedb.github.io/lancedb/hybrid_search/eval/)                                |
| `use_tantivy` | `bool`         | -       | tantivyを使用するかどうか。                                                                                            |
