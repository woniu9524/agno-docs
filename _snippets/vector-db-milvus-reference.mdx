| 参数 | 类型 | 默认值 | 描述 |
| --------- | ---- | ------- | ----------- |
| `collection` | `str` | `None` | 用于存储向量和元数据的 Milvus collection 的名称 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成 embedding 的 Embedder 实例 |
| `distance` | `Distance` | `Distance.cosine` | 用于相似性搜索的距离度量 |
| `uri` | `str` | `"http://localhost:19530"` | Milvus 服务器的 URI。可以是 Milvus Lite 的本地文件路径、自托管 Milvus 的服务器地址，或 Zilliz Cloud 的端点 |
| `token` | `Optional[str]` | `None` | 用于身份验证的 token。对于启用了身份验证的自托管 Milvus，格式为 "username:password"，或对于 Zilliz Cloud，为 API 密钥 |
