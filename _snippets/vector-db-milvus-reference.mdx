| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `collection` | `str` | `None` | ベクトルとメタデータを格納するMilvusコレクションの名前 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 埋め込みを生成するためのEmbedderインスタンス |
| `distance` | `Distance` | `Distance.cosine` | 類似性検索に使用する距離メトリック |
| `uri` | `str` | `"http://localhost:19530"` | MilvusサーバーのURI。Milvus Liteの場合はローカルファイルパス、セルフホスト型Milvusの場合はサーバーアドレス、Zilliz Cloudの場合はエンドポイントを指定できます |
| `token` | `Optional[str]` | `None` | 認証用のトークン。セルフホスト型Milvusで認証が有効な場合は「ユーザー名:パスワード」形式、Zilliz Cloudの場合はAPIキーを指定します |
