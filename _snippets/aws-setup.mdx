## AWS のセットアップ

<Steps>
  <Step title="認証情報の更新">
    AWS で実行するには、以下のいずれかが必要です。

    1. `~/.aws/credentials` ファイルに AWS 認証情報を設定する
    2. **または** `AWS_ACCESS_KEY_ID` および `AWS_SECRET_ACCESS_KEY` 環境変数を設定する

    <Note>

    認証情報ファイルを作成するには、[aws cli](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) をインストールし、`aws configure` を実行してください。

    </Note>

  </Step>
  <Step title="リージョンとサブネットの更新">
    `workspace/settings.py` ファイルで `aws_region` を更新し、2 つの [サブネット](https://us-east-1.console.aws.amazon.com/vpc/home?#subnets:) を追加します (ECS サービスに必要)。

    ```python workspace/settings.py
    ws_settings = WorkspaceSettings(
        ...
        # -*- AWS settings
        # Subnet ID をここに追加してください
        subnet_ids=["subnet-xyz", "subnet-xyz"],
        ...
    )
    ```

    <Note>

    サブネットが選択した `aws_region` に属していることを確認してください。

    </Note>

  </Step>
</Steps>
