### 示例 API 端点

- 打开 [localhost:8000/docs](http://localhost:8000/docs) 以查看示例 API 端点。
- 使用 `/v1/assitants/load-knowledge-base` 加载知识库
- 使用以下内容测试 `v1/assitants/chat` 端点

```json
{
  "message": "How do I make chicken curry?",
  "agent": "AUTO_PDF"
}
```
