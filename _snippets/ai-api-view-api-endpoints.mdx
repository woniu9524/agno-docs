### サンプル API エンドポイント

- サンプル API エンドポイントを表示するには、[localhost:8000/docs](http://localhost:8000/docs) を開きます。
- `/v1/assitants/load-knowledge-base` を使用してナレッジベースをロードします。
- `v1/assitants/chat` エンドポイントを以下のようにテストします。

```json
{
  "message": "How do I make chicken curry?",
  "agent": "AUTO_PDF"
}
```
