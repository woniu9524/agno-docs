| パラメータ    | タイプ            | デフォルト                          | 説明                                                                                                                             |
| ------------ | --------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `id`         | `str`           | `"gpt-4o"`                       | レスポンス生成に使用される特定のモデルID。                                                                                             |
| `name`       | `str`           | `"OpenRouter"`                   | OpenRouterエージェントの名前識別子。                                                                                                 |
| `provider`   | `str`           | `"OpenRouter:"+id`               | モデルのプロバイダー。「OpenRouter」とモデルIDを組み合わせたもの。                                                                                     |
| `api_key`    | `Optional[str]` | -                                | OpenRouterサービスへのリクエスト認証用のAPIキー。環境変数 `OPENROUTER_API_KEY` から取得されます。                                                               |
| `base_url`   | `str`           | `"https://openrouter.ai/api/v1"` | OpenRouterサービスへのAPIリクエストを行うためのベースURL。                                                                                             |
| `max_tokens` | `int`           | `1024`                           | レスポンスで生成される最大トークン数。                                                                                                   |
