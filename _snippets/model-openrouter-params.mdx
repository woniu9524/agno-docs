| 参数    | 类型            | 默认值                          | 描述                                                                                                                           |
| ----------- | --------------- | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `id`        | `str`           | `"gpt-4o"`                       | 用于生成响应的具体模型 ID。                                                                                                    |
| `name`      | `str`           | `"OpenRouter"`                   | OpenRouter 代理的名称标识符。                                                                                                 |
| `provider`  | `str`           | `"OpenRouter:"+id`               | 模型提供者，结合了 "OpenRouter" 和模型 ID。                                                                                     |
| `api_key`   | `Optional[str]` | -                                | 用于向 OpenRouter 服务进行请求身份验证的 API 密钥。从环境变量 `OPENROUTER_API_KEY` 中检索。                                       |
| `base_url`  | `str`           | `"https://openrouter.ai/api/v1"` | 向 OpenRouter 服务发出 API 请求的基础 URL。                                                                                        |
| `max_tokens`| `int`           | `1024`                           | 响应中要生成的最大 token 数。                                                                                                   |
