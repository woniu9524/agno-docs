## 更新密钥

    更新 `workspace/secrets/prd_app_secrets.yml` 中的 Streamlit 应用密码和 API 密钥

    ```python workspace/secrets/prd_app_secrets.yml
    APP_PASSWORD: "admin"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
  <Step title="RDS 数据库密码">
    更新 `workspace/secrets/prd_db_secrets.yml` 中的 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by prd RDS database
    MASTER_USERNAME: ai
    MASTER_USER_PASSWORD: "ai9999!!"
    ```

  </Step>
</Steps>
