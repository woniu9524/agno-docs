## シークレットの更新

<Steps>
  <Step title="Streamlit アプリのパスワードと API シークレット">
    `workspace/secrets/prd_app_secrets.yml` 内の Streamlit アプリのパスワードと API シークレットを更新します。

    ```python workspace/secrets/prd_app_secrets.yml
    APP_PASSWORD: "admin"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
  <Step title="RDS データベースのパスワード">
    `workspace/secrets/prd_db_secrets.yml` 内の RDS データベースのパスワードを更新します。

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by prd RDS database
    MASTER_USERNAME: ai
    MASTER_USER_PASSWORD: "ai9999!!"
    ```

  </Step>
</Steps>
