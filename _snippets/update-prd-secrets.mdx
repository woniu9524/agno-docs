## 更新 Secret

<Steps>
  <Step title="Streamlit 应用密码和 API Secret">
    在 `workspace/secrets/prd_app_secrets.yml` 中更新 streamlit 应用密码和 API Secret

    ```python workspace/secrets/prd_app_secrets.yml
    APP_PASSWORD: "admin"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
  <Step title="RDS 数据库密码">
    在 `workspace/secrets/prd_db_secrets.yml` 中更新 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by prd RDS database
    MASTER_USERNAME: ai
    MASTER_USER_PASSWORD: "ai9999!!"
    ```

  </Step>
</Steps>
