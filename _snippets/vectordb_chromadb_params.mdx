| パラメータ           | タイプ       | デフォルト          | 説明                                                              |
| ------------------- | ---------- | ---------------- | ----------------------------------------------------------------- |
| `collection`        | `str`      | -                | 使用するコレクションの名前。                                                      |
| `embedder`          | `Embedder` | OpenAIEmbedder() | ドキュメントの内容を埋め込むために使用する埋め込みモデル。                                         |
| `distance`          | `Distance` | cosine           | 使用する距離メトリック。                                                          |
| `path`              | `str`      | "tmp/chromadb"   | ChromaDB のデータが保存されるパス。                                                 |
| `persistent_client` | `bool`     | False            | 持続的な ChromaDB クライアントを使用するかどうか。                                                  |
