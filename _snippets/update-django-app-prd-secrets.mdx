## シークレットの更新

<Steps>
  <Step title="RDS データベースパスワード">
    `workspace/secrets/prd_db_secrets.yml` で RDS データベースパスワードを更新します。

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by RDS Database
    MASTER_USERNAME: app
    MASTER_USER_PASSWORD: "app9999!!"
    ```

  </Step>
  <Step title="アプリのシークレット">
    アプリで使用されるその他のシークレットを `workspace/secrets/prd_app_secrets.yml` に追加します。

    ```python workspace/secrets/prd_app_secrets.yml
    SECRET_KEY: "django-insecure-...."
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
