## 更新 Secret

<Steps>
  <Step title="RDS 数据库密码">
    更新 `workspace/secrets/prd_db_secrets.yml` 中的 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by RDS Database
    MASTER_USERNAME: app
    MASTER_USER_PASSWORD: "app9999!!"
    ```

  </Step>
  <Step title="应用 Secret">
    将您的应用使用的任何其他 secret 添加到 `workspace/secrets/prd_app_secrets.yml`

    ```python workspace/secrets/prd_app_secrets.yml
    SECRET_KEY: "django-insecure-...."
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
