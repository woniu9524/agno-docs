| フィールド | 説明 |
|-------------------------|-----------------------------------------------------------------------------|
| input_tokens | モデルへのプロンプト/入力のトークン数。 |
| output_tokens | モデルが出力として生成したトークン数。 |
| total_tokens | 使用された合計トークン数（入力 + 出力）。 |
| prompt_tokens | プロンプトのトークン数（OpenAIの場合はinput_tokensと同じ）。 |
| completion_tokens | 完了のトークン数（OpenAIの場合はoutput_tokensと同じ）。 |
| audio_tokens | 合計オーディオトークン数（オーディオ入出力を使用した場合）。 |
| input_audio_tokens | 入力のオーディオトークン数。 |
| output_audio_tokens | 出力のオーディオトークン数。 |
| cached_tokens | キャッシュから提供されたトークン数（キャッシュが使用されている場合）。 |
| cache_write_tokens | キャッシュに書き込まれたトークン数。 |
| reasoning_tokens | 推論ステップに使用されたトークン数（有効な場合）。 |
| prompt_tokens_details | プロンプトトークンの詳細な内訳（OpenAIで使用）。 |
| completion_tokens_details | 完了トークンの詳細な内訳（OpenAIで使用）。 |
| additional_metrics | モデル/ツールによって提供される追加のメトリクス（例：レイテンシ、コストなど）。 |
| time | メッセージ生成にかかった時間（秒）。 |
| time_to_first_token | 最初のトークンが生成されるまでの時間（秒）。 |
> 注意: すべてのフィールドが常に存在するとは限りません。モデル/ツールおよび実行によって異なります。
