### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于 embeddings 的模型 ID | `"togethercomputer/m2-bert-80M-32k-retrieval"` |
| `dimensions` | `int` | 嵌入的输出维度 | `768` |
| `api_key` | `Optional[str]` | Together API 密钥 | Environment variable `TOGETHER_API_KEY` |
| `base_url` | `str` | API 请求的基础 URL | `"https://api.together.xyz/v1"` |
