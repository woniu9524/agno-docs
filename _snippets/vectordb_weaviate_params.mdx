| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `wcd_url` | `Optional[str]` | Weaviate Cloud URL (または WCD_URL 環境変数を使用) | `None` |
| `wcd_api_key` | `Optional[str]` | Weaviate Cloud API キー (または WCD_API_KEY 環境変数を使用) | `None` |
| `client` | `Optional[weaviate.WeaviateClient]` | 事前に設定された Weaviate クライアント | `None` |
| `local` | `bool` | ローカルの Weaviate インスタンスを使用するかどうか | `False` |
| `collection` | `str` | Weaviate コレクションの名前 | `"default"` |
| `vector_index` | `VectorIndex` | ベクトルインデックスのタイプ (HNSW, FLAT, DYNAMIC) | `VectorIndex.HNSW` |
| `distance` | `Distance` | 距離メトリック (COSINE, DOT など) | `Distance.COSINE` |
| `embedder` | `Optional[Embedder]` | 埋め込み生成に使用する埋め込み器 | `OpenAIEmbedder()` |
| `search_type` | `SearchType` | 検索タイプ (vector, keyword, hybrid) | `SearchType.vector` |
| `reranker` | `Optional[Reranker]` | 検索結果を絞り込むためのランカー | `None` |
| `hybrid_search_alpha` | `float` | ハイブリッド検索の重み付け係数 | `0.5` |
