## AWS リソースの削除

AWS リソースを操作した後、以下のコマンドを使用して削除します。

<CodeGroup>

```bash terminal
ag ws down --env prd --infra aws
```

```bash shorthand
ag ws down prd:aws
```

</CodeGroup>

または、個々のリソースグループを削除するには、以下を使用します。

<CodeGroup>

```bash app
ag ws down --env prd --infra aws --group app
```

```bash api
ag ws down --env prd --infra aws --group api
```

```bash database
ag ws down --env prd --infra aws --group db
```

</CodeGroup>
