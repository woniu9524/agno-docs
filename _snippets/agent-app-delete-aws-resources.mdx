## 删除 AWS 资源

尽情探索，然后使用以下命令删除 AWS 资源：

<CodeGroup>

```bash terminal
ag ws down --env prd --infra aws
```

```bash shorthand
ag ws down prd:aws
```

</CodeGroup>

或者使用以下命令删除单个资源组：

<CodeGroup>

```bash app
ag ws down --env prd --infra aws --group app
```

```bash api
ag ws down --env prd --infra aws --group api
```

```bash database
ag ws down --env prd --infra aws --group db
```

</CodeGroup>
