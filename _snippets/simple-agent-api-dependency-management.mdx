## Python の依存関係の管理

<Steps>

<Step title="pyproject.toml の変更">

`pyproject.toml` ファイルの `[tool.poetry.dependencies]` セクションに、目的の Python パッケージの依存関係を追加または更新します。

</Step>

<Step title="requirements.txt の生成">

`requirements.txt` ファイルは、アプリケーションイメージのビルドに使用されます。`pyproject.toml` を変更した後、以下を使用して `requirements.txt` を再生成します。

```bash
./scripts/generate_requirements.sh
```

既存のすべての依存関係を最新の互換性のあるバージョンにアップグレードするには、以下を実行します。

```bash
./scripts/generate_requirements.sh upgrade
```

</Step>

<Step title="Docker イメージの再ビルド">

更新された依存関係を含めるために、Docker イメージを再ビルドします。

```bash
docker compose up -d --build
```
</Step>

</Steps>
