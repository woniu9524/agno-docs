## Agent アプリをローカルで実行する

`agent-app` フォルダに `cd` します。

```bash
cd agent-app
```

以下のコマンドを使用して Agent アプリを起動します。

<CodeGroup>

```bash terminal
ag ws up
```

```bash shorthand
ag ws up dev:docker
```

```bash full options
ag ws up --env dev --infra docker
```

</CodeGroup>

確認するために **Enter** を押し、イメージのダウンロードに数秒かかります（初回のみ）。Docker ダッシュボードでコンテナのステータスを確認し、ログを表示します。

- Streamlit UI を表示するには [localhost:8501](http://localhost:8501) を開きます。
- FastAPI ルートを表示するには [localhost:8000/docs](http://localhost:8000/docs) を開きます。

![agent-app-ui](/images/agent-app-ui.png)

注記:

- `Agents` は `agents` フォルダで定義されています。
- Streamlit アプリは `ui` フォルダで定義されています。
- API ルートは `api` フォルダで定義されています。
