### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `prefix` | `str` | 用于 Redis 键的前缀，为记忆添加命名空间 | `"agno_memory"` |
| `host` | `str` | Redis 主机地址 | `"localhost"` |
| `port` | `int` | Redis 端口号 | `6379` |
| `db` | `int` | Redis 数据库编号 | `0` |
| `password` | `Optional[str]` | 如果需要身份验证，则为 Redis 密码 | `None` |
