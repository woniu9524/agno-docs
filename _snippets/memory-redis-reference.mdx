### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `prefix` | `str` | 用于 Redis 键的命名空间前缀，以隔离记忆 | `"agno_memory"` |
| `host` | `str` | Redis 主机地址 | `"localhost"` |
| `port` | `int` | Redis 端口号 | `6379` |
| `db` | `int` | Redis 数据库编号 | `0` |
| `password` | `Optional[str]` | 需要身份验证时的 Redis 密码 | `None` |
