| 参数  | 类型            | 默认值                                    | 描述                                                                                                                     |
| ---------- | --------------- | ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| `id`       | `str`           | `"meta/llama-3.3-70b-instruct"`            | 用于生成响应的特定模型 ID。                                                                                             |
| `name`     | `str`           | `"Nvidia"`                                 | Nvidia 代理的名称标识符。                                                                                                |
| `provider` | `str`           | `"Nvidia" + id`                                           | 模型的提供者，结合了 "Nvidia" 和模型 ID。                                                                               |
| `api_key`  | `Optional[str]` | -                                          | 用于向 Nvidia 服务进行身份验证请求的 API 密钥。从环境变量 `NVIDIA_API_KEY` 中检索。                                        |
| `base_url` | `str`           | `"https://integrate.api.nvidia.com/v1"`    | 向 Nvidia 服务发出 API 请求的基础 URL。                                                                                   |
