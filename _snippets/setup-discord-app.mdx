## 设置与配置
<Steps>

<Step title="先决条件">
请确保您拥有以下条件：
- Python 3.7+
- 拥有服务器管理权限的 Discord 账户
- 所需的 Python 包（将在后续步骤中安装）
</Step>

<Step title="创建 Discord 应用">
1. 前往 [Discord Developer Portal](https://discord.com/developers/applications)
2. 点击“New Application”
3. 提供一个应用名称（例如：“My Agno Bot”）
4. 接受开发者服务条款
5. 点击“Create”
</Step>

<Step title="创建 Bot 用户">
1. 在您的应用设置中，导航至“Bot”部分
2. 点击“Add Bot”
3. 通过点击“Yes, do it!”进行确认
4. 在“Token”部分，点击“Copy”复制您的 Bot Token
5. 将此 Token 安全地保存起来（稍后会用到）
</Step>

<Step title="配置 Bot 权限和 Intents">
1. 在 Bot 设置中，向下滚动至“Privileged Gateway Intents”
2. 启用以下 Intents：
   - **Presence Intent**（可选，用于用户状态）
   - **Server Members Intent**（用于成员相关事件）
   - **Message Content Intent**（读取消息内容必需）
3. 在“Bot Permissions”下，确保您的 Bot 拥有：
   - Send Messages
   - Read Message History
   - Create Public Threads
   - Use Slash Commands（可选）
</Step>

<Step title="设置环境变量">
在项目根目录下创建一个 `.envrc` 文件，并包含以下内容，将占位符替换为您实际的 Bot Token：
```bash
export DISCORD_BOT_TOKEN="your_bot_token_here"
```

在 Discord Developer Portal 的“Bot” > “Token”下查找您的 Bot Token。

确保您的 shell 已加载此文件（例如，通过使用 `direnv allow`）。
</Step>

<Step title="安装所需包">
安装必要的 Python 包：
```bash
pip install discord.py agno
```


</Step>

<Step title="将 Bot 邀请到您的 Discord 服务器">
1. 在您的应用设置中，前往“OAuth2” > “URL Generator”
2. 在“Scopes”下，选择：
   - `bot`
   - `applications.commands`（如果使用斜杠命令）
3. 在“Bot Permissions”下，选择您的 Bot 所需的权限：
   - **Send Messages**
   - **Create Public Threads**
   - **Read Message History**
   - **Attach Files**
   - **Embed Links**
   - **Use External Emojis**（可选）
4. 复制生成的 URL，在浏览器中打开它，然后选择您想添加 Bot 的服务器
</Step>

<Step title="测试您的 Bot">
1. 启动您的 Bot 应用
2. 前往您的 Discord 服务器
3. 在 Bot 有权访问的任何频道中发送一条消息
4. 您的 Bot 应该会自动创建一个线程并进行回复
5. 如果您使用的是媒体 Bot，请尝试上传图片或文件来测试媒体处理功能
</Step>

</Steps>

<Callout type="warning">
**安全提示**: 切勿将您的 Discord Bot Token 提交到版本控制系统中。始终使用环境变量或安全的配置管理来处理敏感数据。
</Callout>
