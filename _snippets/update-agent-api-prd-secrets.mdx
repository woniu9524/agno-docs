## 更新 Secrets

<Steps>
  <Step title="RDS 数据库密码">
    更新 `workspace/secrets/prd_db_secrets.yml` 中的 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # 用于 prd RDS 数据库的 Secrets
    MASTER_USERNAME: api
    MASTER_USER_PASSWORD: "api9999!!"
    ```

  </Step>
  <Step title="API Secrets">
    将您的 API 使用的其他任何 secrets 添加到 `workspace/secrets/prd_api_secrets.yml`

    ```python workspace/secrets/prd_api_secrets.yml
    SECRET_KEY: "very_secret"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
