## シークレットの更新

<Steps>
  <Step title="RDS データベースパスワード">
    `workspace/secrets/prd_db_secrets.yml` 内の RDS データベースパスワードを更新します。

    ```python workspace/secrets/prd_db_secrets.yml
    # Secrets used by prd RDS database
    MASTER_USERNAME: api
    MASTER_USER_PASSWORD: "api9999!!"
    ```

  </Step>
  <Step title="API シークレット">
    API で使用されるその他のシークレットを `workspace/secrets/prd_api_secrets.yml` に追加します。

    ```python workspace/secrets/prd_api_secrets.yml
    SECRET_KEY: "very_secret"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
