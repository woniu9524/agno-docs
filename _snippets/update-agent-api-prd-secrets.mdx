## 更新 Secrets

<Steps>
  <Step title="RDS 数据库密码">
    在 `workspace/secrets/prd_db_secrets.yml` 中更新 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # prd RDS 数据库使用的 Secrets
    MASTER_USERNAME: api
    MASTER_USER_PASSWORD: "api9999!!"
    ```

  </Step>
  <Step title="API Secrets">
    将 API 使用的任何其他 secrets 添加到 `workspace/secrets/prd_api_secrets.yml`

    ```python workspace/secrets/prd_api_secrets.yml
    SECRET_KEY: "very_secret"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
