### 本番環境 Django 管理サイト

**LoadBalancer DNS** + `/admin` ページを開いて、Django 管理サイトを表示します。

本番環境コンテナに SSH 接続して、管理者ユーザーを作成します。

```bash
ECS_CLUSTER=django-prd
TASK_ARN=$(aws ecs list-tasks --cluster django-prd --query "taskArns[0]" --output text)
CONTAINER_NAME=django-prd

aws ecs execute-command --cluster $ECS_CLUSTER \
    --task $TASK_ARN \
    --container $CONTAINER_NAME \
    --interactive \
    --command "python manage.py createsuperuser"
```

管理パネルにログインします。
