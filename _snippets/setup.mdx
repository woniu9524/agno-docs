## セットアップ

<Steps>
  <Step title="仮想環境の作成とアクティベーション">

    <CodeGroup>
    ```bash Mac
    python3 -m venv .venv
    source .venv/bin/activate
    ```

    ```bash Windows
    python3 -m venv aienv
    aienv/scripts/activate
    ```
    </CodeGroup>

  </Step>
  <Step title="Agno のインストール">

    <CodeGroup>
    ```bash Mac
    pip install -U "agno[aws]"
    ```

    ```bash Windows
    pip install -U "agno[aws]"
    ```
    </CodeGroup>

  </Step>
  <Step title="uv と docker のインストール">
    - Python 環境を管理するために [uv](https://docs.astral.sh/uv/#getting-started) をインストールします。

    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

    - アプリケーションをローカルで実行するために [docker desktop](https://docs.docker.com/desktop/install/mac-install/) をインストールします。
  </Step>
  <Step title="OpenAI キーのエクスポート">

    <CodeGroup>
    ```bash Mac
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```
    </CodeGroup>

    <Tip>
    Agno はあらゆるモデルプロバイダーと互換性があります。ワークスペース内のエージェントを更新するだけです。
    </Tip>
  </Step>
</Steps>
