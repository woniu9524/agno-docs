| パラメータ     | タイプ                                | デフォルト                                | 説明                                                                                                                                     |
|---------------|---------------------------------------|----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| `name`        | `Optional[str]`                       | `None`                                 | **ワークフロー名**                                                                                                                       |
| `workflow_id` | `Optional[str]`                       | `None`                                 | **ワークフローUUID** (設定されていない場合は自動生成)                                                                                    |
| `description` | `Optional[str]`                       | `None`                                 | **ワークフローの説明** (UIにのみ表示されます)                                                                                            |
| `user_id`     | `Optional[str]`                       | `None`                                 | **このワークフローを操作しているユーザーのID**                                                                                           |
| `session_id`  | `Optional[str]`                       | `None`                                 | **セッションUUID** (設定されていない場合は自動生成)                                                                                      |
| `session_name`| `Optional[str]`                       | `None`                                 | **セッション名**                                                                                                                         |
| `session_state`| `Dict[str, Any]`                     | `{}` (空の辞書)                      | **データベースに保存されているセッションの状態**                                                                                         |
| `memory`      | `Optional[WorkflowMemory]`            | `None`                                 | **ワークフローメモリ**                                                                                                                   |
| `storage`     | `Optional[WorkflowStorage]`           | `None`                                 | **ワークフローストレージ**                                                                                                               |
| `extra_data`  | `Optional[Dict[str, Any]]`            | `None`                                 | **このワークフローと共に保存される追加データ**                                                                                           |
| `debug_mode`  | `bool`                                | `False`                                | デバッグログを有効にする                                                                                                                 |
| `monitoring`  | `bool`                                | `False` (環境変数: `AGNO_MONITOR`)     | `True` の場合、ワークフロー情報を agno.com にログ記録して監視します。環境変数 `AGNO_MONITOR="true"` が設定されている場合はデフォルトで `True` になります。 |
| `telemetry`   | `bool`                                | `True` (環境変数: `AGNO_TELEMETRY`)    | `True` の場合、分析のために最小限のテレメトリをログ記録します。環境変数 `AGNO_TELEMETRY="true"` が設定されている場合はデフォルトで `True` になります。 |
| `run_id`      | `Optional[str]`                       | `None`                                 | **(手動で設定しないでください)** 各ワークフロー実行の一意のID                                                                            |
| `run_input`   | `Optional[Dict[str, Any]]`            | `None`                                 | **(手動で設定しないでください)** 現在の実行に渡された入力                                                                                 |
| `run_response`| `Optional[RunResponse]`               | `None`                                 | **(手動で設定しないでください)** 現在の実行によって生成された応答                                                                         |
| `images`      | `Optional[List[ImageArtifact]]`       | `None`                                 | **このセッション中に生成された画像**                                                                                                     |
| `videos`      | `Optional[List[VideoArtifact]]`       | `None`                                 | **このセッション中に生成されたビデオ**                                                                                                   |
| `audio`       | `Optional[List[AudioArtifact]]`       | `None`                                 | **このセッション中に生成されたオーディオ**                                                                                               |
