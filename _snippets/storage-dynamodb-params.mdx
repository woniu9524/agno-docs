| パラメータ                     | タイプ            | デフォルト | 説明                                                     |
| ---------------------------- | --------------- | ------- | -------------------------------------------------------- |
| `table_name`                 | `str`           | -       | 使用するテーブルの名前。                                         |
| `region_name`                | `Optional[str]` | `None`  | DynamoDBテーブルのリージョン名。                               |
| `aws_access_key_id`          | `Optional[str]` | `None`  | 提供されている場合のAWSアクセスキーID。                           |
| `aws_secret_access_key`      | `Optional[str]` | `None`  | 提供されている場合のAWSシークレットアクセスキー。                     |
| `endpoint_url`               | `Optional[str]` | `None`  | 提供されている場合のEndpoint URL。                               |
| `create_table_if_not_exists` | `bool`          | `True`  | trueの場合、テーブルが存在しない場合はテーブルを作成します。 |
