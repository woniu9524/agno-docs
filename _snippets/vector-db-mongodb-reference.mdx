| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `collection_name` | `str` | Required | ベクトルとメタデータを格納するMongoDBコレクションの名前 |
| `db_url` | `Optional[str]` | `"mongodb://localhost:27017/"` | MongoDB接続文字列 |
| `database` | `str` | `"ai"` | MongoDBデータベースの名前 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 埋め込みを生成するためのEmbedderインスタンス |
| `distance_metric` | `str` | `Distance.cosine` | 類似性検索に使用する距離メトリック |
| `overwrite` | `bool` | `False` | 既存のコレクションとインデックスを上書きするかどうか |
| `cosmos_compatibility` | `bool` | `False` | Azure Cosmos DB MongoDB vCoreのサポートを有効にするかどうか |
| `wait_until_index_ready_in_seconds` | `Optional[float]` | `None` | インデックスが準備できるまで待機する秒数 |
| `wait_after_insert_in_seconds` | `Optional[float]` | `None` | ドキュメント挿入後に待機する秒数 |
