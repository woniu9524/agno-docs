| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `table_name` | `str` | Required | ベクトルとメタデータを格納するテーブルの名前 |
| `schema` | `str` | `"ai"` | データベーススキーマ名 |
| `db_url` | `Optional[str]` | `None` | データベース接続URL |
| `db_engine` | `Optional[Engine]` | `None` | SQLAlchemyデータベースエンジン |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | エンベディングを生成するためのEmbedderインスタンス |
| `search_type` | `SearchType` | `SearchType.vector` | 実行する検索の種類（vector、keyword、またはhybrid） |
| `vector_index` | `Union[Ivfflat, HNSW]` | `HNSW()` | ベクトルインデックスの設定 |
| `distance` | `Distance` | `Distance.cosine` | ベクトル比較のための距離メトリック |
| `prefix_match` | `bool` | `False` | 全文検索のためのプレフィックスマッチを有効にする |
| `vector_score_weight` | `float` | `0.5` | ハイブリッド検索におけるベクトル類似性の重み |
| `content_language` | `str` | `"english"` | 全文検索の言語 |
| `schema_version` | `int` | `1` | データベーススキーマのバージョン |
| `auto_upgrade_schema` | `bool` | `False` | Trueの場合、スキーマを自動的にアップグレードする |
| `reranker` | `Optional[Reranker]` | `None` | 検索結果の後処理のためのRerankerインスタンス |
