| 参数    | 类型    | 默认值                          | 描述                                                                                                                      |
| ------- | ------- | ------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `id`    | `str`   | `"gpt-4o-mini"`                  | 用于生成响应的特定模型 ID。                                                                                                             |
| `name`  | `str`   | `"AIMLApi"`                      | AI/ML API 代理的名称标识符。                                                                                                                |
| `provider`| `str`   | `"AIMLApi:" + id`                | 模型提供商，结合了 `"AIMLApi"` 和模型 ID。                                                                                              |
| `api_key` | `Optional[str]` | –                                | 用于向 AI/ML API 服务进行身份验证请求的 API 密钥。从环境变量 `AIMLAPI_API_KEY` 中检索。     |
| `base_url`| `str`   | `"https://api.aimlapi.com/v1"`   | 向 AI/ML API 服务发出 API 请求的基础 URL。                                                                                              |
| `max_tokens`| `int`   | `4096`                           | 响应中要生成的最大 Token 数。                                                                                                     |
