| パラメータ    | タイプ            | デフォルト                          | 説明                                                                                                                             |
| ------------ | --------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `id`         | `str`           | `"gpt-4o-mini"`                  | レスポンス生成に使用される特定のモデルID。                                                                                       |
| `name`       | `str`           | `"AIMLApi"`                      | AI/ML APIエージェントの名前識別子。                                                                                              |
| `provider`   | `str`           | `"AIMLApi:" + id`                | モデルのプロバイダーで、`"AIMLApi"` とモデルIDを組み合わせたもの。                                                                 |
| `api_key`    | `Optional[str]` | –                                | AI/ML APIサービスへのリクエスト認証に使用するAPIキー。環境変数 `AIMLAPI_API_KEY` から取得されます。                                    |
| `base_url`   | `str`           | `"https://api.aimlapi.com/v1"`   | AI/ML APIサービスへのAPIリクエストを行うためのベースURL。                                                                          |
| `max_tokens` | `int`           | `4096`                           | レスポンスで生成される最大トークン数。                                                                                             |
