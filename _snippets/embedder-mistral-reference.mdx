### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于 embeddings 的模型 ID | `"mistral-embed"` |
| `dimensions` | `int` | embedding 的输出维度 | `1024` |
| `request_params` | `Optional[Dict[str, Any]]` | embedding 请求的其他参数 | `None` |
| `api_key` | `Optional[str]` | Mistral API 密钥 | 环境变量 `MISTRAL_API_KEY` |
| `endpoint` | `Optional[str]` | 自定义 API 端点 URL | `None` |
| `max_retries` | `Optional[int]` | 最大重试次数 | `None` |
| `timeout` | `Optional[int]` | 请求超时时间（秒） | `None` |
| `client_params` | `Optional[Dict[str, Any]]` | 客户端初始化其他参数 | `None` |
| `mistral_client` | `Optional[Mistral]` | 预配置的 Mistral 客户端 | `None` |
