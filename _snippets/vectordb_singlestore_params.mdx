| パラメータ    | タイプ               | デフォルト            | 説明                                                              |
| ------------ | ------------------ | ------------------ | --------------------------------------------------- |
| `collection` | `str`              | -                  | 使用するコレクションの名前。                                        |
| `schema`     | `Optional[str]`    | `"ai"`             | 使用するデータベーススキーマ。                                      |
| `db_url`     | `Optional[str]`    | `None`             | データベース接続URL。                                               |
| `db_engine`  | `Optional[Engine]` | `None`             | SQLAlchemyのエンジンインスタンス。                                  |
| `embedder`   | `Embedder`         | `OpenAIEmbedder()` | ベクトル埋め込みを作成するために使用するエンベッダー。                        |
| `distance`   | `Distance`         | `Distance.cosine`  | 類似性検索に使用する距離メトリック。                                |
