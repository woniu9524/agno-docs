### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `collection_name` | `str` | MongoDB collection 的名称 | Required |
| `db_url` | `Optional[str]` | MongoDB connection URL | `None` |
| `db_name` | `str` | database 的名称 | `"agno"` |
| `client` | `Optional[MongoClient]` | 预配置的 MongoDB client | `None` |
