| パラメータ             | タイプ               | デフォルト | 説明                                                               |
| --------------------- | ------------------ | ------- | ---------------------------------------------------------- |
| `table_name`          | `str`              | -       | 使用するテーブルの名前。                                                               |
| `schema`              | `Optional[str]`    | `"ai"`  | スキーマ名。デフォルトは "ai" です。                                                               |
| `db_url`              | `Optional[str]`    | `None`  | データベースのURL。提供された場合。                                                               |
| `db_engine`           | `Optional[Engine]` | `None`  | 使用するデータベースエンジン。                                                               |
| `schema_version`      | `int`              | `1`     | スキーマのバージョン。デフォルトは 1 です。                                                               |
| `auto_upgrade_schema` | `bool`             | `False` | スキーマが必要に応じて自動的にアップグレードされるかどうか。                                                               |
