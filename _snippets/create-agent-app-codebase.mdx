## Agent App コードベースの作成

`agent-app` テンプレートを使用してコードベースを作成し、任意の名前を付けます。

<CodeGroup>

```bash Mac
ag ws create --template agent-app --name agent-app
```

```bash Windows
ag ws create --template agent-app --name agent-app
```

</CodeGroup>

これにより、`agent-app` という名前のフォルダが作成され、以下の構造になります。

```bash
agent-app                     # ルートディレクトリ
├── agents                  # Agent を配置する場所
├── api                     # API ルートを配置する場所
├── ui                      # Streamlit アプリを配置する場所
├── db                      # データベーステーブルを配置する場所
├── Dockerfile              # アプリケーションの Dockerfile
├── pyproject.toml          # Python プロジェクトの定義
├── requirements.txt        # pyproject.toml を使用して生成された Python 依存関係
├── scripts                 # ヘルパー スクリプト
├── utils                   # 共有ユーティリティ
└── workspace               # Agno ワークスペースディレクトリ
    ├── dev_resources.py    # ローカルで実行される開発リソース
    ├── prd_resources.py    # AWS で実行される本番リソース
    ├── secrets             # シークレット
    └── settings.py         # Agno ワークスペースの設定
