## 创建你的 Agent App 代码库

使用 `agent-app` 模板创建你的代码库，可以随意命名。

<CodeGroup>

```bash Mac
ag ws create --template agent-app --name agent-app
```

```bash Windows
ag ws create --template agent-app --name agent-app
```

</CodeGroup>

这会创建一个名为 `agent-app` 的文件夹，其结构如下：

```bash
agent-app                     # 根目录
├── agents                  # 你的 Agents 存放于此
├── api                     # 你的 API 路由存放于此
├── ui                      # 你的 Streamlit 应用存放于此
├── db                      # 你的数据库表存放于此
├── Dockerfile              # 应用程序的 Dockerfile
├── pyproject.toml          # Python 项目定义
├── requirements.txt        # 使用 pyproject.toml 生成的 Python 依赖项
├── scripts                 # 辅助脚本
├── utils                   # 共享工具
└── workspace               # Agno 工作区目录
    ├── dev_resources.py    # 本地运行的开发资源
    ├── prd_resources.py    # 在 AWS 上运行的生产资源
    ├── secrets             # 敏感信息
    └── settings.py         # Agno 工作区设置
