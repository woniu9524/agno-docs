## コードベースの作成

[Django](https://www.djangoproject.com/) と [PostgreSQL](https://www.postgresql.org/) で事前に設定された `django-app` テンプレートを使用してコードベースを作成します。

<CodeGroup>

```bash Mac
ag ws create -t django-app -n django-app
```

```bash Windows
ag ws create -t django-app -n django-app
```

</CodeGroup>

これにより、`django-app` という名前のフォルダが作成され、以下の構造になります。

```bash
django-app                  # django-app のルートディレクトリ
├── app                   # Django プロジェクトのディレクトリ
├── nginx                 # 本番環境で静的ファイルに使用される nginx のディレクトリ
├── manage.py             # django-admin ファイル
├── dev.Dockerfile        # 開発アプリケーション用の Dockerfile
├── prd.Dockerfile        # 本番アプリケーション用の Dockerfile
├── pyproject.toml        # Python プロジェクトの定義
├── requirements.txt      # pyproject.toml から生成された Python 依存関係
├── scripts               # ヘルパースクリプトのディレクトリ
├── tests                 # 単体テストのディレクトリ
└── workspace             # agno ワークスペースディレクトリ
    ├── dev_resources.py  # ローカルで実行される開発リソース
    ├── prd_resources.py  # AWS 上で実行される本番リソース
    ├── secrets           # シークレットを格納するディレクトリ
    └── settings.py       # agno ワークスペースの設定
