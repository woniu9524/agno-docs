## 创建您的代码库

使用预先配置了 [Django](https://www.djangoproject.com/) 和 [PostgreSQL](https://www.postgresql.org/) 的 `django-app` 模板来创建您的代码库。

<CodeGroup>

```bash Mac
ag ws create -t django-app -n django-app
```

```bash Windows
ag ws create -t django-app -n django-app
```

</CodeGroup>

这会创建一个名为 `django-app` 的文件夹，其结构如下：

```bash
django-app                  # django-app 的根目录
├── app                   # Django 项目的目录
├── nginx                 # 生产环境中用于静态文件的 nginx 目录
├── manage.py             # django-admin 文件
├── dev.Dockerfile        # 开发应用程序的 Dockerfile
├── prd.Dockerfile        # 生产应用程序的 Dockerfile
├── pyproject.toml        # Python 项目定义
├── requirements.txt      # 由 pyproject.toml 生成的 Python 依赖项
├── scripts               # 辅助脚本的目录
├── tests                 # 单元测试的目录
└── workspace             # agno 工作区目录
    ├── dev_resources.py  # 本地运行的开发资源
    ├── prd_resources.py  # 在 AWS 上运行的生产资源
    ├── secrets           # 用于存储密钥的目录
    └── settings.py       # agno 工作区设置
```
