| パラメータ  | タイプ            | デフォルト                      | 説明                                                                                                                              |
| ---------- | --------------- | ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `id`       | `str`           | `"deepseek-chat"`            | レスポンス生成に使用される特定のモデルID。                                                                                             |
| `name`     | `str`           | `"DeepSeek"`                 | DeepSeekモデルの名前識別子。                                                                                                        |
| `provider` | `str`           | `"DeepSeek"`                 | モデルのプロバイダー。                                                                                                            |
| `api_key`  | `Optional[str]` | -                            | DeepSeekサービスへのリクエスト認証に使用されるAPIキー。環境変数`DEEPSEEK_API_KEY`から取得されます。                                                |
| `base_url` | `str`           | `"https://api.deepseek.com"` | DeepSeekサービスへのAPIリクエストを行うためのベースURL。                                                                                   |
