| パラメータ                     | タイプ                                   | デフォルト    | 説明                                                           |
| ----------------------------- | -------------------------------------- | ---------- | -------------------------------------------------------------- |
| `id`                          | `str`                                  | -          | 使用するモデルのID。                                           |
| `name`                        | `Optional[str]`                        | `None`     | このモデルの名前。モデルAPIには送信されません。                  |
| `provider`                    | `Optional[str]`                        | `None`     | このモデルのプロバイダー。モデルAPIには送信されません。          |
| `supports_native_structured_outputs` | `bool`                          | `False`    | モデルがネイティブに構造化出力をサポートしているかどうか（例：OpenAI）。 |
| `supports_json_schema_outputs` | `bool`                               | `False`    | モデルが構造化出力にjson_schemaを必要とするかどうか（例：LMStudio）。 |
| `system_prompt`               | `Optional[str]`                        | `None`     | モデルからエージェントに追加されるシステムプロンプト。             |
| `instructions`                | `Optional[List[str]]`                  | `None`     | モデルからエージェントに追加される指示。                         |
| `tool_message_role`           | `str`                                  | `"tool"`   | ツールメッセージのロール。                                     |
| `assistant_message_role`      | `str`                                  | `"assistant"` | アシスタントメッセージのロール。                               |
| `session_id`                  | `Optional[str]`                        | `None`     | 呼び出し元のエージェントまたはワークフローのセッションID。       |
| `structured_outputs`          | `Optional[bool]`                       | `None`     | このモデルで構造化出力を使用するかどうか。                     |
| `override_system_role`        | `bool`
