| 参数                     | 类型                                   | 默认值    | 描述                                                           |
| ------------------------ | -------------------------------------- | ---------- | -------------------------------------------------------------- |
| `id`                     | `str`                                  | -          | 要使用的模型的 ID。                                            |
| `name`                   | `Optional[str]`                        | `None`     | 此模型的名称。不发送到模型 API。                               |
| `provider`               | `Optional[str]`                        | `None`     | 此模型的提供商。不发送到模型 API。                             |
| `supports_native_structured_outputs` | `bool`                          | `False`    | 模型是否支持原生结构化输出（例如，OpenAI）。                   |
| `supports_json_schema_outputs` | `bool`                               | `False`    | 模型是否需要 JSON schema 来进行结构化输出（例如，LMStudio）。 |
| `system_prompt`          | `Optional[str]`                        | `None`     | 从模型添加到 Agent 的系统提示。                              |
| `instructions`           | `Optional[List[str]]`                  | `None`     | 从模型添加到 Agent 的指令。                                  |
| `tool_message_role`      | `str`                                  | `"tool"`   | 工具消息的角色。                                               |
| `assistant_message_role` | `str`                                  | `"assistant"` | 助手消息的角色。                                               |
| `session_id`             | `Optional[str]`                        | `None`     | 调用 Agent 或 Workflow 的会话 ID。                           |
| `structured_outputs`     | `Optional[bool]`                       | `None`     | 是否将结构化输出与此模型一起使用。                             |
| `override_system_role`   | `bool`                                 |            |                                                                |
