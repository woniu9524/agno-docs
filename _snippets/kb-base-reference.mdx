| 参数 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `reader` | `Optional[Reader]` | `None` | 用于读取文档的 Reader |
| `vector_db` | `Optional[VectorDb]` | `None` | 用于存储知识库的 VectorDb |
| `num_documents` | `int` | `2` | 搜索时返回的相关文档数量 |
| `optimize_on` | `Optional[int]` | `1000` | 优化 VectorDb 的文档数量 |
| `driver` | `str` | `"knowledge"` | Assistant 知识的驱动程序 |
