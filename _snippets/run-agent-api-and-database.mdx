## 使用 FastAPI 部署你的 Agent

[FastAPI](https://fastapi.tiangolo.com/) 是一个出色的 REST API 构建框架。它速度快、设计精良，并且深受所有使用它的人的喜爱。大多数生产应用程序都使用像 [next.js](https://nextjs.org) 这样的前端框架构建，并由 REST API 提供支持，而 FastAPI 在这方面表现出色。

你的代码库预先配置了 [FastAPI](https://fastapi.tiangolo.com/) 和 [PostgreSQL](https://www.postgresql.org/)，以及一些示例路由。使用以下命令启动你的工作区：

<CodeGroup>

```bash terminal
ag ws up
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

**按 Enter 键** 进行确认，并等待几分钟让镜像下载（仅首次下载）。在 Docker 仪表板上验证容器状态并查看日志。

- 打开 [localhost:8000/docs](http://localhost:8000/docs) 查看 API 端点。
- 使用以下内容测试 `/v1/agents/{agent_id}/run` 端点：

```json
{
  "message": "howdy",
  "agent_id": "sage",
  "stream": true
}
```
