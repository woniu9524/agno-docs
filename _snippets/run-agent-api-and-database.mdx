## 使用 FastAPI 托管您的 Agent

[FastAPI](https://fastapi.tiangolo.com/) 是一个用于构建 REST API 的卓越框架。它速度快、设计精良，深受使用者喜爱。大多数生产应用程序都使用像 [next.js](https://nextjs.org) 这样的前端框架，并由 REST API 提供支持，而 FastAPI 在此方面表现出色。

您的代码库已预先配置了 [FastAPI](https://fastapi.tiangolo.com/) 和 [PostgreSQL](https://www.postgresql.org/)，以及一些示例路由。使用以下命令启动您的工作区：

<CodeGroup>

```bash terminal
ag ws up
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

**按 Enter** 键确认，并等待几分钟让镜像下载（仅首次）。在 Docker 仪表板上验证容器状态并查看日志。

- 打开 [localhost:8000/docs](http://localhost:8000/docs) 查看 API 端点。
- 使用以下内容测试 `/v1/agents/{agent_id}/run` 端点：

```json
{
  "message": "howdy",
  "agent_id": "sage",
  "stream": true
}
