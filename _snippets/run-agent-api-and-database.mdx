## FastAPI を使用してエージェントをサーブする

[FastAPI](https://fastapi.tiangolo.com/) は、REST API を構築するための優れたフレームワークです。高速で、設計が優れており、利用するすべての人に愛されています。ほとんどの本番アプリケーションは、[next.js](https://nextjs.org) のようなフロントエンドフレームワークと、FastAPI が活躍する REST API によってバックアップされています。

コードベースには、[FastAPI](https://fastapi.tiangolo.com/) と [PostgreSQL](https://www.postgresql.org/) がプリセットされており、いくつかのサンプルルートが含まれています。ワークスペースは次のように起動します。

<CodeGroup>

```bash terminal
ag ws up
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

確認するために**Enter キーを押して**ください。イメージのダウンロードには数分かかります（初回のみ）。Docker ダッシュボードでコンテナのステータスを確認し、ログを表示します。

- [localhost:8000/docs](http://localhost:8000/docs) を開いて API エンドポイントを表示します。
- `/v1/agents/{agent_id}/run` エンドポイントを次のようにテストします。

```json
{
  "message": "howdy",
  "agent_id": "sage",
  "stream": true
}
