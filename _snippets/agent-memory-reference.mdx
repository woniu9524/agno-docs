### パラメータ

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `model` | `Optional[Model]` | メモリと要約に使用されるモデル | `None` |
| `memory_manager` | `Optional[MemoryManager]` | メモリ操作用のマネージャー | `None` |
| `summarizer` | `Optional[SessionSummarizer]` | セッション要約生成用のサマライザー | `None` |
| `db` | `Optional[MemoryDb]` | メモリを保存するためのデータベース | `None` |
| `debug_mode` | `bool` | デバッグログを有効にするかどうか | `False` |

### 主要なメソッド

#### ユーザーメモリ管理

| メソッド | 説明 | パラメータ | 戻り値 |
|--------|-------------|------------|---------|
| `get_user_memories` | ユーザーのすべてのメモリを取得します | `user_id: str` | `List[UserMemory]` |
| `get_user_memory` | 特定のメモリを取得します | `user_id: str, memory_id: str` | `UserMemory` |
| `add_user_memory` | 新しいメモリを追加し、メモリIDを取得します | `memory: UserMemory, user_id: Optional[str] = None` | `str` |
| `replace_user_memory` | 既存のメモリを更新し、メモリIDを取得します | `memory_id: str, memory: UserMemory, user_id: Optional[str] = None` | `str` |
| `delete_user_memory` | メモリを削除します | `user_id: str, memory_id: str` | `None` |
| `create_user_memories` | 1つ以上のメッセージからメモリを作成します | `message: Optional[str] = None, messages: Optional[List[Message]] = None, user_id: Optional[str] = None` | `str` |
| `acreate_user_memories` | 1つ以上のメッセージからメモリを作成します (非同期) | `message: Optional[str] = None, messages: Optional[List[Message]] = None, user_id: Optional[str] = None` | `str` |
| `search_user_memories` | 指定された検索方法を使用してユーザーメモリを検索します | `query: Optional[str] = None, limit: Optional[int] = None, retrieval_method: Optional[Literal["last_n", "first_n", "semantic"]] = None, user_id: Optional[str] = None` | `List[UserMemory]` |

#### セッション要約管理

| メソッド | 説明 | パラメータ |
|--------|-------------|------------|
| `get_session_summaries` | ユーザーのすべてのセッション要約を取得します | `user_id: str` |
| `get_session_summary` | 特定のセッション要約を取得します | `user_id: str, session_id: str` |
| `create_session_summary` | 保存されたセッション実行からセッションの要約を作成します | `session_id: str, user_id: Optional[str] = None` |
| `acreate_session_summary` | 保存されたセッション実行からセッションの要約を作成します (非同期) | `session_id: str, user_id: Optional[str] = None` |
| `delete_session_summary` | セッション要約を削除します | `user_id: str, session_id: str` |

### UserMemory

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `memory` | `str` | メモリの実際のコンテンツ | 必須 |
| `topics` | `Optional[List[str]]` | メモリのトピックまたはカテゴリ | `None` |
| `input` | `Optional[str]` | メモリを生成した元の入力 | `None` |
| `last_updated` | `Optional[datetime]` | メモリが最後に更新された日時 | `None` |
| `memory_id` | `Optional[str]` | メモリの一意な識別子 | `None` |

### SessionSummary

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `summary` | `str` | セッションの簡潔な要約 | 必須 |
| `topics` | `Optional[List[str]]` | セッションで議論されたトピック | `None` |
| `last_updated` | `Optional[datetime]` | 要約が最後に更新された日時 | `None` |

### Memory Manager

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `model` | `Optional[Model]` | メモリ管理に使用されるモデル | `None` |
| `system_message` | `Optional[str]` | メモリマネージャー用のカスタムシステムプロンプト | `None` |
| `additional_instructions` | `Optional[str]` | システムメッセージの末尾に追加される追加の指示 | `None` |


### Session Summarizer

| `system_message` | `Optional[str]` | サマライザー用のカスタムシステムプロンプト | `None` |
| `additional_instructions` | `Optional[str]` | システムメッセージの末尾に追加される追加の指示 | `None` |
