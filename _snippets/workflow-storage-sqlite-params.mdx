| パラメータ            | タイプ               | デフォルト | 説明                                                              |
| ------------------- | ------------------ | ------- | ----------------------------------------------------------------- |
| `table_name`        | `str`              | -       | ワークフローセッションを格納するテーブル名。                                |
| `db_url`            | `Optional[str]`    | `None`  | 接続するデータベースURL。                                                |
| `db_file`           | `Optional[str]`    | `None`  | 接続するデータベースファイル。                                              |
| `db_engine`         | `Optional[Engine]` | `None`  | 使用するSQLAlchemyデータベースエンジン。                                      |
| `schema_version`    | `int`              | `1`     | スキーマのバージョン。                                                          |
| `auto_upgrade_schema` | `bool`           | `False` | スキーマを自動的にアップグレードするかどうか。                                        |
