## Docker で PgVector を実行する

以下の内容で `resources.py` ファイルを作成します。

```python resources.py
from agno.docker.app.postgres import PgVectorDb
from agno.docker.resources import DockerResources

# -*- PgVector running on port 5432:5432
vector_db = PgVectorDb(
    pg_user="ai",
    pg_password="ai",
    pg_database="ai",
    debug_mode=True,
)

# -*- DockerResources
dev_docker_resources = DockerResources(apps=[vector_db])
```

以下を使用してリソースを開始します。

<CodeGroup>

```bash Mac
ag start resources.py
```

```bash Windows
ag start resources.py
```

</CodeGroup>

確認して Docker ダッシュボードでコンテナの状態を検証するには、**Enter キーを押します**。
