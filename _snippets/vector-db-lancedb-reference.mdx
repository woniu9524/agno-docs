| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `uri` | `lancedb.URI` | `"/tmp/lancedb"` | LanceDB の URI パスまたは接続文字列 |
| `table` | `Optional[lancedb.db.LanceTable]` | `None` | 事前に設定された LanceDB テーブルインスタンス |
| `table_name` | `Optional[str]` | `None` | ベクトルとメタデータを格納するテーブルの名前 |
| `connection` | `Optional[lancedb.LanceDBConnection]` | `None` | 事前に設定された LanceDB 接続 |
| `api_key` | `Optional[str]` | `None` | クラウド LanceDB 認証用の API キー |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 埋め込みを生成するための埋め込みインスタンス |
| `search_type` | `SearchType` | `SearchType.vector` | 実行する検索タイプ（vector、keyword、または hybrid） |
| `distance` | `Distance` | `Distance.cosine` | 類似性検索に使用する距離メトリック |
| `nprobes` | `Optional[int]` | `None` | 近似最近傍検索のプローブ数 |
| `reranker` | `Optional[Reranker]` | `None` | 検索結果を再ランク付けするためのオプションの再ランク付けインスタンス |
| `use_tantivy` | `bool` | `True` | フルテキスト検索インデックスに Tantivy を使用するかどうか |
