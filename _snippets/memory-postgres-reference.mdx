### パラメータ

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `table_name` | `str` | PostgreSQLテーブルの名前 | Required |
| `schema` | `Optional[str]` | データベーススキーマ名 | `"ai"` |
| `db_url` | `Optional[str]` | PostgreSQL接続URL | `None` |
| `db_engine` | `Optional[Engine]` | 事前に設定されたSQLAlchemyエンジン | `None` |
