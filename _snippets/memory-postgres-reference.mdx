### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `table_name` | `str` | PostgreSQL 表的名称 | Required |
| `schema` | `Optional[str]` | 数据库 schema 名称 | `"ai"` |
| `db_url` | `Optional[str]` | PostgreSQL 连接 URL | `None` |
| `db_engine` | `Optional[Engine]` | 预配置的 SQLAlchemy 引擎 | `None` |
