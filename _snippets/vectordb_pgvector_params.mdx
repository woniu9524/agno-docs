| パラメータ             | タイプ                   | デフォルト | 説明                                                                    |
| --------------------- | ---------------------- | ------- | ----------------------------------------------------------------------- |
| `table_name`          | `str`                  | -       | 使用するテーブルの名前。                                                |
| `schema`              | `str`                  | -       | 使用するスキーマ。                                                      |
| `db_url`              | `str`                  | -       | 接続するデータベースのURL。                                             |
| `db_engine`           | `Engine`               | -       | 使用するデータベースエンジン。                                          |
| `embedder`            | `Embedder`             | -       | 使用するエンベッダー。                                                  |
| `search_type`         | `SearchType`           | vector  | 使用する検索タイプ。                                                    |
| `vector_index`        | `Union[Ivfflat, HNSW]` | -       | 使用するベクトルインデックス。                                          |
| `distance`            | `Distance`             | cosine  | 使用する距離。                                                          |
| `prefix_match`        | `bool`                 | -       | プレフィックスマッチを使用するかどうか。                                |
| `vector_score_weight` | `float`                | 0.5     | ハイブリッド検索におけるベクトル類似性の重み。0から1の間でなければなりません。 |
| `content_language`    | `str`                  | -       | 使用するコンテンツの言語。                                              |
| `schema_version`      | `int`                  | -       | 使用するスキーマバージョン。                                            |
| `auto_upgrade_schema` | `bool`                 | -       | スキーマを自動アップグレードするかどうか。                              |
