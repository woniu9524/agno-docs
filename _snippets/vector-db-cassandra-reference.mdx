| 参数    | 类型                | 默认值          | 描述                                                               |
| ----------- | ------------------- | ---------------- | ------------------------------------------------------------------ |
| `table_name` | `str`               | `None`                | 用于在Cassandra中存储向量和元数据的表名                                |
| `keyspace`   | `str`               | `None`                | 将要在其中创建表的Cassandra的Keyspace名称                               |
| `embedder`   | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入的Embedder实例                                            |
| `session`    | `CassandraSession`  | `None`                | 用于数据库操作的活动Cassandra会话对象                                   |
