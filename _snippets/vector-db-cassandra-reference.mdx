| 参数    | 类型                | 默认值          | 描述                                                               |
| ----------- | ------------------- | ---------------- | ------------------------------------------------------------------ |
| `table_name` | `str`               | `None`                | 用于在 Cassandra 中存储向量和元数据的表名                                |
| `keyspace`   | `str`               | `None`                | 将要创建表的 Cassandra 中的 keyspace 名称                               |
| `embedder`   | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入的 embedder 实例                                         |
| `session`    | `CassandraSession`  | `None`                | 用于数据库操作的活动 Cassandra session 对象                             |
