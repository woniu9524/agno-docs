| パラメータ    | タイプ                | デフォルト          | 説明                                                                 |
| ----------- | ------------------- | ---------------- | -------------------------------------------------------------------- |
| `table_name` | `str`               | `None`                | Cassandraにベクトルとメタデータを格納するテーブルの名前                               |
| `keyspace`   | `str`               | `None`                | テーブルが作成されるCassandraのキースペース名                               |
| `embedder`   | `Optional[Embedder]` | `OpenAIEmbedder()` | ベクトルを生成するためのEmbedderインスタンス                               |
| `session`    | `CassandraSession`  | `None`                | データベース操作のためのアクティブなCassandraセッションオブジェクト                               |
