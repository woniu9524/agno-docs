| パラメータ  | タイプ            | デフォルト                    | 説明                                                                                                               |
| ---------- | --------------- | -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `id`       | `str`           | `"v0-1.0-md"`              | レスポンス生成に使用される特定のモデルID。                                                                         |
| `name`     | `str`           | `"v0"`                    | v0エージェントの名前識別子。                                                                                       |
| `provider` | `str`           | `"Vercel"`                 | モデルのプロバイダー。「v0」とモデルIDを組み合わせたものです。                                                        |
| `api_key`  | `Optional[str]` | -                          | v0サービスへのリクエスト認証に使用するAPIキー。環境変数 `V0_API_KEY` から取得されます。                               |
| `base_url` | `str`           | `"https://api.v0.dev/v1/"` | v0サービスへのAPIリクエストを行うためのベースURL。                                                                 |
