| 参数 | 类型 | 默认值 | 描述 |
| --------- | ---- | ------- | ----------- |
| `collection` | `str` | `None` | 要在 ChromaDB 中存储向量和元数据的集合名称 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入的 embedder 实例 |
| `distance` | `Distance` | `Distance.cosine` | 用于相似性搜索的距离度量 |
| `path` | `str` | `"tmp/chromadb"` | 使用持久化客户端时存储 ChromaDB 数据的路径 |
| `persistent_client` | `bool` | `False` | 是否使用持久化 ChromaDB 客户端 |
| `reranker` | `Optional[Reranker]` | `None` | 用于对搜索结果进行重排的可选 reranker 实例 |
