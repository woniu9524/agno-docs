### パラメータ

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|-------------|---------|
| `runs` | `List[TeamRun]` | `[]` | チームの会話実行のリスト |
| `messages` | `List[Message]` | `[]` | モデルに送信されたメッセージのリスト |
| `update_system_message_on_change` | `bool` | `True` | システムメッセージが変更されたときに更新するかどうか |
| `team_context` | `Optional[TeamContext]` | `None` | チームメンバー間で共有されるコンテキスト |
| `create_user_memories` | `bool` | `False` | ユーザーのパーソナライズされたメモリを作成するかどうか |
| `update_user_memories_after_run` | `bool` | `True` | 各実行後にメモリを更新するかどうか |
| `db` | `Optional[MemoryDb]` | `None` | パーソナライズされたメモリを保存するためのデータベース |
| `user_id` | `Optional[str]` | `None` | パーソナライズされたメモリのユーザー識別子 |
| `retrieval` | `MemoryRetrieval` | `MemoryRetrieval.last_n` | メモリ検索戦略 |
| `memories` | `Optional[List[Memory]]` | `None` | 取得されたメモリのリスト |
| `classifier` | `Optional[MemoryClassifier]` | `None` | メモリの重要度を分類するための分類器 |
| `manager` | `Optional[MemoryManager]` | `None` | メモリ操作のためのマネージャー |
| `num_memories` | `Optional[int]` | `None` | 取得するメモリの数 |

### TeamRun

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|-------------|---------|
| `message` | `Optional[Message]` | `None` | チーム実行に関連付けられたメッセージ |
| `member_runs` | `Optional[List[AgentRun]]` | `None` | メンバーエージェント実行のリスト |
| `response` | `Optional[TeamRunResponse]` | `None` | チーム実行中に生成された応答 |

### TeamContext

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|-------------|---------|
| `member_interactions` | `List[TeamMemberInteraction]` | `[]` | チームメンバー間のインタラクションのリスト |
| `text` | `Optional[str]` | `None` | チームのための共有テキストコンテキスト |

### TeamMemberInteraction

| パラメータ | タイプ | 説明 |
|-----------|------|-------------|
| `member_name` | `str` | チームメンバーの名前 |
| `task` | `str` | チームメンバーに割り当てられたタスク |
| `response` | `RunResponse` | チームメンバーからの応答 |

### メモリ検索

| パラメータ | タイプ | 説明 |
|-----------|------|-------------|
| `last_n` | `str` | 履歴から最後のN個のメモリを取得 |
| `first_n` | `str` | 履歴から最初のN個のメモリを取得 |
| `semantic` | `str` | セマンティック類似性に基づいてメモリを取得 |

### Memory

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `memory` | `str` | 実際のメモリの内容 | 必須 |
| `id` | `Optional[str]` | メモリの一意の識別子 | `None` |
| `topic` | `Optional[str]` | メモリのトピックまたはカテゴリ | `None` |
| `input` | `Optional[str]` | メモリを生成した元の入力 | `None` |

### Memory Classifier

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `model` | `Optional[Model]` | メモリの分類に使用されるモデル | `None` |
| `system_message` | `Optional[str]` | 分類器用のカスタムシステムプロンプト | `None` |
| `existing_memories` | `Optional[List[Memory]]` | チェック対象の既存のメモリのリスト | `None` |

### Memory Manager

| パラメータ | タイプ | 説明 | デフォルト |
|-----------|------|-------------|---------|
| `model` | `Optional[Model]` | メモリの管理に使用されるモデル | `None` |
| `user_id` | `Optional[str]` | ユーザーの一意の識別子 | `None` |
| `system_message` | `Optional[str]` | メモリマネージャー用のカスタムシステムプロンプト | `None` |
| `db` | `Optional[MemoryDb]` | メモリを保存するためのデータベース | `None` |
| `input_message` | `Optional[str]` | 現在処理中の入力メッセージ | `None` |
