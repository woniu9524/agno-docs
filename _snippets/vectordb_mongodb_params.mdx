| パラメータ | タイプ | 説明 | デフォルト |
|---|---|---|---|
| `collection_name` | `str` | MongoDBコレクションの名前 | Required |
| `db_url` | `Optional[str]` | MongoDB接続文字列 | `"mongodb://localhost:27017/"` |
| `database` | `str` | データベース名 | `"agno"` |
| `embedder` | `Optional[Embedder]` | 埋め込みを生成するためのEmbedderインスタンス | `OpenAIEmbedder()` |
| `distance_metric` | `str` | 類似性計算のための距離指標 | `Distance.cosine` |
| `overwrite` | `bool` | Trueの場合、既存のコレクションとインデックスを上書きします | `False` |
| `wait_until_index_ready` | `Optional[float]` | インデックスが準備できるまで待機する秒数 | `None` |
| `wait_after_insert` | `Optional[float]` | ドキュメント挿入後に待機する秒数 | `None` |
| `max_pool_size` | `int` | 接続プールの最大接続数 | `100` |
| `retry_writes` | `bool` | 書き込み操作を再試行するかどうか | `True` |
| `client` | `Optional[MongoClient]` | 既存のMongoClientインスタンス | `None` |
