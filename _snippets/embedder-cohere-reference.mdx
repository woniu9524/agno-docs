### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于生成嵌入的模型 ID | `"embed-english-v3.0"` |
| `input_type` | `str` | 要生成嵌入的输入类型（例如："search_query", "classification", "clustering") | `"search_query"` |
| `embedding_types` | `Optional[List[str]]` | 要生成的嵌入类型列表 | `None` |
| `api_key` | `Optional[str]` | Cohere API 密钥 | 环境变量 `COHERE_API_KEY` |
| `request_params` | `Optional[Dict[str, Any]]` | 嵌入请求的附加参数 | `None` |
| `client_params` | `Optional[Dict[str, Any]]` | 客户端初始化的附加参数 | `None` |
| `cohere_client` | `Optional[CohereClient]` | 预先配置的 Cohere 客户端 | `None` |
