| パラメータ | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `model` | `str` | - | 使用するモデルのID。 |
| `name` | `str` | - | このLLMの名前。注意: これはLLM APIに送信されません。 |
| `metrics` | `Dict[str, Any]` | - | このLLMのために収集されたメトリクス。注意: これはLLM APIに送信されません。 |
| `response_format` | `Any` | - | レスポンスのフォーマット。 |
| `tools` | `List[Union[Tool, Dict]]` | - | LLMに提供されるツールのリスト。ツールは、モデルがJSON入力を生成できる関数です。辞書を提供する場合、モデルによって呼び出されません。常に `add_tool()` メソッドを使用してツールを追加してください。 |
| `tool_choice` | `Union[str, Dict[str, Any]]` | - | モデルによって呼び出される関数（または関数がない場合）を制御します。`"none"` は、モデルが関数を呼び出さず、代わりにメッセージを生成することを意味します。`"auto"` は、モデルがメッセージの生成または関数の呼び出しを選択できることを意味します。`{"type": "function", "function": {"name": "my_function"}}` を介して特定の関数を指定すると、モデルはその関数を呼び出すように強制されます。関数が存在しない場合は `"none"` がデフォルトです。関数が存在する場合は `"auto"` がデフォルトです。 |
| `run_tools` | `bool` | `True` | Trueの場合、ツールを実行します。 |
| `show_tool_calls` | `bool` | - | Trueの場合、レスポンスでツール呼び出しを表示します。 |
| `functions` | `Dict[str, Function]` | - | ツールから抽出された関数。注意: これらはLLM APIに送信されず、実行にのみ使用されます。 |
| `function_call_limit` | `int` | `20` | 許可される最大関数呼び出し数。 |
| `function_call_stack` | `List[FunctionCall]` | - | 関数呼び出しのスタック。 |
| `system_prompt` | `str` | - | LLMに提供されるシステムプロンプト。 |
| `instructions` | `str` | - | LLMに提供される指示。 |
