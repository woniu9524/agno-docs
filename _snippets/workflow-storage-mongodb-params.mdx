| パラメータ            | タイプ                    | デフォルト | 説明                                                              |
| ------------------- | ----------------------- | ------- | ----------------------------------------------------------------- |
| `collection_name`   | `str`                   | -       | ワークフローセッションを保存するコレクションの名前。                                |
| `db_url`            | `Optional[str]`         | `None`  | MongoDB 接続 URL。                                                  |
| `db_name`           | `str`                   | `"agno"` | データベースの名前。                                                  |
| `client`            | `Optional[MongoClient]` | `None`  | 使用する既存の MongoDB クライアント（オプション）。                               |
