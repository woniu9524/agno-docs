| 参数            | 类型                    | 默认值 | 描述                                                       |
| --------------- | ----------------------- | ------ | ---------------------------------------------------------- |
| `collection_name` | `str`                   | -      | 用于存储工作流会话的集合名称。                               |
| `db_url`        | `Optional[str]`         | `None` | MongoDB 连接 URL。                                         |
| `db_name`       | `str`                   | `"agno"` | 数据库名称。                                               |
| `client`        | `Optional[MongoClient]` | `None` | 可选的现有 MongoDB 客户端以供使用。                          |
