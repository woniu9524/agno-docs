| 参数            | 类型                    | 默认 | 描述                                                 |
| ------------------- | ----------------------- | ------- | ---------------------------------------------------------- |
| `collection_name`   | `str`                   | -       | 用于存储 Workflow 会话的集合名称。      |
| `db_url`            | `Optional[str]`         | `None`  | MongoDB 连接 URL。                                 |
| `db_name`           | `str`                   | `"agno"` | 数据库名称。                                 |
| `client`            | `Optional[MongoClient]` | `None`  | 可选的现有 MongoDB 客户端用于连接。                   |
