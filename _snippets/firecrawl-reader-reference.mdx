| 参数 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `api_key` | `Optional[str]` | `None` | 用于身份验证的 Firecrawl API 密钥 |
| `params` | `Optional[Dict]` | `None` | 要传递给 Firecrawl API 的附加参数 |
| `mode` | `Literal["scrape", "crawl"]` | `"scrape"` | 操作模式 - "scrape" 用于单页，"crawl" 用于多页 |
| `url` | `str` | 必需 | 要抓取或爬取的网站的 URL |
