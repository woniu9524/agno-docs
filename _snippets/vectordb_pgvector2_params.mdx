| 参数    | 类型                             | 默认值             | 描述                                                                              |
| ------- | -------------------------------- | --------------- | --------------------------------------------------------------------------------- |
| `collection` | `str`                            | -               | 用于存储向量数据的 collection 名称                                                    |
| `schema`     | `Optional[str]`                  | `"ai"`          | 数据库 schema 名称                                                                  |
| `db_url`     | `Optional[str]`                  | `None`          | 数据库连接 URL                                                                    |
| `db_engine`  | `Optional[Engine]`               | `None`          | SQLAlchemy 数据库引擎                                                               |
| `embedder`   | `Optional[Embedder]`             | `None`          | 用于创建 embedding 的 embedder 实例（如果未提供，则默认为 OpenAIEmbedder）            |
| `distance`   | `Distance`                       | `Distance.cosine` | 向量比较的距离度量                                                                  |
| `index`      | `Optional[Union[Ivfflat, HNSW]]` | `HNSW()`        | 向量索引配置                                                                      |
