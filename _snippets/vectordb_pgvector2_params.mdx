| パラメータ    | タイプ                             | デフォルト           | 説明                                                                                             |
| ------------ | -------------------------------- | ----------------- | ------------------------------------------------------------------------------------------------ |
| `collection` | `str`                            | -                 | ベクトルデータを格納するコレクションの名前                                                                       |
| `schema`     | `Optional[str]`                  | `"ai"`            | データベーススキーマ名                                                                                             |
| `db_url`     | `Optional[str]`                  | `None`            | データベース接続URL                                                                                              |
| `db_engine`  | `Optional[Engine]`               | `None`            | SQLAlchemyデータベースエンジン                                                                                       |
| `embedder`   | `Optional[Embedder]`             | `None`            | エンベディング作成用のEmbedderインスタンス（提供されない場合はOpenAIEmbedderがデフォルト）                                                     |
| `distance`   | `Distance`                       | `Distance.cosine` | ベクトル比較用の距離メトリック                                                                                     |
| `index`      | `Optional[Union[Ivfflat, HNSW]]` | `HNSW()`          | ベクトルインデックスの設定                                                                                         |
