### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `collection_name` | `str` | MongoDB 集合的名称 | `"memory"` |
| `db_url` | `Optional[str]` | MongoDB 连接 URL | `None` |
| `db_name` | `str` | 数据库的名称 | `"agno"` |
| `client` | `Optional[MongoClient]` | 预配置的 MongoDB 客户端 | `None` |
