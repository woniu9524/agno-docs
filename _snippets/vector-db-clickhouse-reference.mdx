| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `table_name` | `str` | `None` | Clickhouse にベクトルとメタデータを格納するテーブル名 |
| `host` | `str` | `None` | Clickhouse サーバーのホスト名 |
| `username` | `Optional[str]` | `None` | Clickhouse 認証用のユーザー名 |
| `password` | `str` | `""` | Clickhouse 認証用のパスワード |
| `port` | `int` | `0` | Clickhouse 接続用のポート番号 |
| `database_name` | `str` | `"ai"` | Clickhouse で使用するデータベース名 |
| `dsn` | `Optional[str]` | `None` | Clickhouse 接続用の DSN 文字列 |
| `compress` | `str` | `"lz4"` | 使用する圧縮アルゴリズム |
| `client` | `Optional[Client]` | `None` | 事前に設定された Clickhouse クライアント (オプション) |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | Embedding を生成するための Embedder インスタンス |
| `distance` | `Distance` | `Distance.cosine` | Similarity Search に使用する距離メトリック |
| `index` | `Optional[HNSW]` | `HNSW()` | ベクトル Similarity Search 用の HNSW インデックス設定 |
