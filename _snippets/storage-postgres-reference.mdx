### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `table_name` | `str` | PostgreSQL 表名 | 必需 |
| `schema` | `Optional[str]` | 数据库 schema 名 | `"ai"` |
| `db_url` | `Optional[str]` | PostgreSQL 连接 URL | `None` |
| `db_engine` | `Optional[Engine]` | 预配置的 SQLAlchemy 引擎 | `None` |
| `schema_version` | `int` | Schema 版本号 | `1` |
| `auto_upgrade_schema` | `bool` | 自动升级 schema | `False` |
