## クイックスタート

以下の手順に従って、Agent API を起動してください。

**前提条件**: Docker Desktop がインストールされ、実行されている必要があります。

<Steps>
  <Step title="リポジトリをクローンする">
    ```bash
    git clone https://github.com/agno-agi/agent-api.git
    cd agent-api
    ```
  </Step>

  <Step title="OpenAI キーをエクスポートする">

    <CodeGroup>
    ```bash Mac
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```
    </CodeGroup>

  </Step>

  <Step title="アプリケーションを起動する">
    ```bash
    docker compose up -d
    ```
  </Step>

  <Step title="アプリケーションをテストする">

  このコマンドは以下を起動します:

  - FastAPI サーバー (`localhost:8000` で実行されます)。
  - PostgreSQL データベース (`localhost:5432` からアクセス可能です)。

  起動後、以下のことができます:

  - API を `localhost:8000/docs` でテストする。
  - Agno Playground または Agent UI に接続する:
      - Agno Playground を開く: [app.agno.com/playground/agents](https://app.agno.com/playground/agents)。
      - 新しいエンドポイントとして `http://localhost:8000` を追加します。名前は `Agent API` (または任意の名前) にできます。
      - 新しく追加したエンドポイントを選択し、エージェントとのチャットを開始します。

  </Step>

  <Step title="アプリケーションを停止する">
    ```bash
    docker compose down
    ```
  </Step>

</Steps>
