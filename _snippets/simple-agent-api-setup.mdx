## 快速入门

请按照以下步骤操作，以启动并运行您的 Agent API：

**先决条件**：应已安装并运行 Docker Desktop。


<Steps>
  <Step title="克隆仓库">
    ```bash
    git clone https://github.com/agno-agi/agent-api.git
    cd agent-api
    ```
  </Step>

  <Step title="导出您的 OpenAI 密钥">

    <CodeGroup>
    ```bash Mac
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```
    </CodeGroup>

  </Step>

  <Step title="启动应用程序">
    ```bash
    docker compose up -d
    ```
  </Step>

  <Step title="测试应用程序">

  此命令将启动：
  
  - FastAPI 服务器，运行在 [`localhost:8000`](http://localhost:8000)。
  - PostgreSQL 数据库，可通过 `localhost:5432` 访问。
  
  启动后，您可以：
  
  - 在 [localhost:8000/docs](http://localhost:8000/docs) 测试 API。
  - 连接到 Agno Playground 或 Agent UI：
      - 打开 Agno Playground [app.agno.com/playground/agents](https://app.agno.com/playground/agents)。
      - 添加 `http://localhost:8000` 作为新的端点。您可以将其命名为 `Agent API`（或任何您喜欢的名称）。
      - 选择您新添加的端点，然后开始与您的 Agent 聊天。

  </Step>

  <Step title="停止应用程序">  
    ```bash
    docker compose down
    ```
  </Step>

</Steps>
