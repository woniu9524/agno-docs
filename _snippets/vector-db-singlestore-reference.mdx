| パラメータ | タイプ | デフォルト | 説明 |
| -------- | ---- | ------- | ----------- |
| `collection` | `str` | Required | SingleStore テーブルの名前 |
| `schema` | `Optional[str]` | `"ai"` | テーブルのスキーマ名 |
| `db_url` | `Optional[str]` | `None` | データベース接続 URL |
| `db_engine` | `Optional[Engine]` | `None` | SQLAlchemy エンジンインスタンス |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 埋め込みを生成するための埋め込みインスタンス |
| `distance` | `Distance` | `Distance.cosine` | 類似性検索の距離メトリック |
| `reranker` | `Optional[Reranker]` | `None` | 結果の後処理のためのランカー |
