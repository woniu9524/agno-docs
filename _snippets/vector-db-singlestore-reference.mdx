| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `collection` | `str` | 必需 | SingleStore 表的名称 |
| `schema` | `Optional[str]` | `"ai"` | 表的模式名称 |
| `db_url` | `Optional[str]` | `None` | 数据库连接 URL |
| `db_engine` | `Optional[Engine]` | `None` | SQLAlchemy 引擎实例 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入的嵌入器实例 |
| `distance` | `Distance` | `Distance.cosine` | 用于相似性搜索的距离度量 |
| `reranker` | `Optional[Reranker]` | `None` | 用于后处理结果的重排器 |
