### 参数

| 参数 | 类型 | 说明 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于嵌入的模型 ID | `"openhermes"` |
| `dimensions` | `int` | 嵌入的输出维度 | `4096` |
| `host` | `Optional[str]` | Ollama 服务器的主机 URL | `None` |
| `timeout` | `Optional[Any]` | 请求超时 | `None` |
| `options` | `Optional[Any]` | 用于生成嵌入的附加选项 | `None` |
| `client_kwargs` | `Optional[Dict[str, Any]]` | 用于客户端初始化的附加参数 | `None` |
| `ollama_client` | `Optional[OllamaClient]` | 已配置好的 Ollama 客户端 | `None` |
