### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于 embeddings 的模型 ID | `"openhermes"` |
| `dimensions` | `int` | embedding 的输出维度 | `4096` |
| `host` | `Optional[str]` | Ollama 服务器的主机 URL | `None` |
| `timeout` | `Optional[Any]` | 请求超时 | `None` |
| `options` | `Optional[Any]` | 用于生成 embedding 的附加选项 | `None` |
| `client_kwargs` | `Optional[Dict[str, Any]]` | 用于客户端初始化的附加参数 | `None` |
| `ollama_client` | `Optional[OllamaClient]` | 预配置的 Ollama 客户端 | `None` |
