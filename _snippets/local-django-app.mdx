## 本地 Web 应用

您的代码库自带了一个预先配置好的 [Django](https://www.djangoproject.com/) 应用，并连接到 [Postgres](https://www.postgresql.org/) 数据库。运行它，命令如下：

<CodeGroup>

```bash terminal
ag ws up
```

```bash full options
ag ws up --env dev --infra docker
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

**按 Enter** 键确认，并等待几分钟让镜像下载（仅首次运行）。您可以在 Docker Dashboard 上验证容器状态和查看日志。

### 查看您的 Django 应用

打开 [localhost:8000](http://localhost:8000) 以在本地查看运行的 Django 应用。

![django-app-django-local](/images/django-app-django-local.png)

### Django Admin

打开 [localhost:8000/admin](http://localhost:8000/admin) 以查看 Django 管理站点。请运行以下命令创建一个管理员用户：

```bash
docker exec -it django-dev-app python manage.py createsuperuser
```

登录到管理面板：
