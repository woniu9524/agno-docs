## ローカルWebアプリケーション

コードベースには、[Postgres](https://www.postgresql.org/)データベースに接続された、事前に設定済みの[Django](https://www.djangoproject.com/)アプリケーションが含まれています。以下のコマンドで実行できます。

<CodeGroup>

```bash terminal
ag ws up
```

```bash full options
ag ws up --env dev --infra docker
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

**Enterキーを押して**確認し、イメージのダウンロードに数分かかります（初回のみ）。コンテナのステータスを確認し、Dockerダッシュボードでログを表示します。

### Djangoアプリの表示

[localhost:8000](http://localhost:8000)を開いて、ローカルで実行中のDjangoアプリを表示します。

![django-app-django-local](/images/django-app-django-local.png)

### Django管理画面

[localhost:8000/admin](http://localhost:8000/admin)を開いて、Django管理サイトを表示します。管理者ユーザーを作成するには、以下を実行します。

```bash
docker exec -it django-dev-app python manage.py createsuperuser
```

管理パネルにログインします。

![django-app-django-admin-local](/images/django-app-django-admin-local.png)
