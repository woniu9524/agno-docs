## 本地 Web 应用

您的代码库包含一个预先配置好的 [Django](https://www.djangoproject.com/) 应用，该应用已连接到 [Postgres](https://www.postgresql.org/) 数据库。请运行以下命令来启动它：

<CodeGroup>

```bash terminal
ag ws up
```

```bash full options
ag ws up --env dev --infra docker
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

**按 Enter 键** 进行确认，首次运行时需要几分钟时间下载镜像。您可以在 Docker 仪表板上查看容器状态和日志。

### 查看您的 Django 应用

打开 [localhost:8000](http://localhost:8000) 以查看本地运行的 Django 应用。

![django-app-django-local](/images/django-app-django-local.png)

### Django 管理后台

打开 [localhost:8000/admin](http://localhost:8000/admin) 以查看 Django 管理后台。请运行以下命令创建一个管理员用户：

```bash
docker exec -it django-dev-app python manage.py createsuperuser
```
