### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `table_name` | `str` | SQLite 表的名称 | `"memory"` |
| `db_url` | `Optional[str]` | SQLite 数据库 URL | `None` |
| `db_file` | `Optional[str]` | SQLite 数据库文件的路径 | `None` |
| `db_engine` | `Optional[Engine]` | 预配置的 SQLAlchemy 引擎 | `None` |
