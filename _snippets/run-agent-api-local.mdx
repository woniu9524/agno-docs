## Agent API をローカルで実行する

`agent-api` フォルダに `cd` します。

```bash
cd agent-api
```

以下のコマンドを使用して Agent API を起動します。

<CodeGroup>

```bash terminal
ag ws up
```

```bash shorthand
ag ws up dev:docker
```

```bash full options
ag ws up --env dev --infra docker
```

</CodeGroup>

確認するために **Enter キーを押します**。イメージのダウンロードには数秒かかります（初回のみ）。Docker ダッシュボードでコンテナのステータスを確認し、ログを表示します。

- [localhost:8000/docs](http://localhost:8000/docs) を開いて、FastAPI ルートを表示します。

注記:

- `Agents` は `agents` フォルダで定義されています。
- api ルートは `api` フォルダで定義されています。
