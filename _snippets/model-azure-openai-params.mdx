| パラメータ                 | タイプ                          | デフォルト             | 説明                                                                                             |
| ------------------------- | ----------------------------- | ------------------- | ------------------------------------------------------------------------------------------------ |
| `id`                      | `str`                         | -                   | レスポンス生成に使用される特定のモデルID。このフィールドは必須です。                                                               |
| `name`                    | `str`                         | `"AzureOpenAI"`     | エージェントの名前識別子。                                                                                             |
| `provider`                | `str`                         | `"Azure"`           | モデルのプロバイダー。                                                                                             |
| `api_key`                 | `Optional[str]`               | `"None"`            | Azure OpenAIサービスへのリクエストを認証するためのAPIキー。                                                                    |
| `api_version`             | `str`                         | `"2024-10-21"`      | 使用するAzure OpenAI APIのバージョン。                                                                                           |
| `azure_endpoint`          | `Optional[str]`               | `"None"`            | Azure OpenAIサービスのAPIエンドポイントURL。                                                                                      |
| `azure_deployment`        | `Optional[str]`               | `"None"`            | Azure内のデプロイメント名またはID。                                                                                             |
| `azure_ad_token`          | `Optional[str]`               | `"None"`            | リクエストを認証するためのAzure Active Directoryトークン。                                                                           |
| `azure_ad_token_provider` | `Optional[Any]`               | `"None"`            | Azure Active Directoryトークンを取得するためのプロバイダー。                                                                         |
| `openai_client`           | `Optional[AzureOpenAIClient]` | `"None"`            | APIリクエストを行うために提供されるAzureOpenAIClientのインスタンス。                                                                    |
