| パラメータ            | タイプ               | デフォルト | 説明                                                       |
| ------------------- | ------------------ | ------- | ---------------------------------------------------------- |
| `table_name`        | `str`              | -       | Workflow セッションを格納するテーブル名。                  |
| `schema`            | `Optional[str]`    | `"ai"`  | テーブルに使用するスキーマ。                               |
| `db_url`            | `Optional[str]`    | `None`  | 接続するデータベースの URL。                               |
| `db_engine`         | `Optional[Engine]` | `None`  | 使用する SQLAlchemy データベースエンジン。                   |
| `schema_version`    | `int`              | `1`     | スキーマのバージョン。                                     |
| `auto_upgrade_schema` | `bool`           | `False` | スキーマを自動的にアップグレードするかどうか。             |
