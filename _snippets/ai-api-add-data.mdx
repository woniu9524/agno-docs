## 可选：添加您的数据

PDF Agent 使用 `ai/knowledge_base.py` 文件中定义的 `pdf_knowledge_base`。要添加您的数据：

<Steps>
<Step title="创建包含您数据的文件夹">
  在应用的根目录中创建一个文件夹 `data/pdfs`
</Step>
<Step title="添加您的数据">

将您的文件添加到 `data/pdfs` 文件夹

</Step>

<Step title="更新知识库">
  使用 `/v1/assitants/load-knowledge-base` 更新知识库
</Step>
</Steps>

查看 `ai/knowledge_base.py` 文件以获取更多信息。
