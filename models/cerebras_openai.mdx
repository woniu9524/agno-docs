## OpenAI互換の統合

Cerebras は OpenAI 互換のインターフェイスを介しても使用できるため、OpenAI API を期待するツールやライブラリとの統合が容易になります。

### OpenAI互換クラスの使用

`CerebrasOpenAI` クラスは、Cerebras モデルに OpenAI スタイルのインターフェイスを提供します。

まず、openai をインストールします。

```shell
pip install openai
```


```python
from agno.agent import Agent
from agno.models.cerebras import CerebrasOpenAI

agent = Agent(
    model=CerebrasOpenAI(
        id="llama-4-scout-17b-16e-instruct",  # 使用するモデル ID
        # base_url="https://api.cerebras.ai", # オプション: Cerebras のデフォルトエンドポイント
    ),
    markdown=True,
)

# 応答をターミナルに出力します
agent.print_response("write a two sentence horror story")
```

### 設定オプション

`CerebrasOpenAI` クラスは、次のパラメータを受け入れます。

| パラメータ   | タイプ         | 説明                                                              | デフォルト                        |
| ----------- | ------------ | ----------------------------------------------------------------- | ------------------------------ |
| `id`        | str          | モデル識別子 (例: "llama-4-scout-17b-16e-instruct")                 | **必須**                   |
| `name`      | str          | モデルの表示名                                                      | "Cerebras"                     |
| `provider`  | str          | プロバイダー名                                                    | "Cerebras"                     |
| `api_key`   | str          | API キー (CEREBRAS_API_KEY 環境変数にフォールバックします)          | None                           |
| `base_url`  | str          | Cerebras OpenAI 互換エンドポイントの URL                            | "https://api.cerebras.ai"      |
