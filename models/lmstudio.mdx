LM Studio で大規模言語モデルをローカルで実行する

[LM Studio](https://lmstudio.ai) は、モデルをローカルで実行するための優れたツールです。

LM Studio は複数のオープンソースモデルをサポートしています。ライブラリは[こちら](https://lmstudio.ai/models)で確認できます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項を以下に示します。

- `llama3.3` モデルは、ほとんどの基本的なユースケースに適しています。
- `qwen` モデルは、ツール使用で特に優れたパフォーマンスを発揮します。
- `deepseek-r1` モデルは、強力な推論能力を備えています。
- `phi4` モデルは、非常に小さいサイズでありながら強力です。

## モデルの設定

[LM Studio](https://lmstudio.ai) をインストールし、使用したいモデルをダウンロードして実行します。

## 例

モデルをローカルに配置したら、`LMStudio` モデルクラスを使用してアクセスします。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.lmstudio import LMStudio

agent = Agent(
    model=LMStudio(id="qwen2.5-7b-instruct-1m"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/lmstudio)でご覧ください。 </Note>
