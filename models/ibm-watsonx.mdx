IBM WatsonX 通过 IBM 的云平台提供强大的基础模型访问。

在此处查看所有 IBM WatsonX 支持的模型：[https://www.ibm.com/products/watsonx/ai/foundation-models](https://www.ibm.com/products/watsonx/ai/foundation-models)。

- 我们推荐使用 `meta-llama/llama-3-3-70b-instruct` 进行通用用途
- 我们推荐使用 `ibm/granite-20b-code-instruct` 进行代码相关任务
- 我们推荐使用 `meta-llama/llama-3-2-11b-vision-instruct` 进行图像理解

#### 多模态支持

通过 WatsonX，我们支持将 `Image` 作为输入

## 身份验证

设置您的 `IBM_WATSONX_API_KEY` 和 `IBM_WATSONX_PROJECT_ID` 环境变量。从 [IBM Cloud](https://cloud.ibm.com/) 获取您的凭证。
您还可以设置 `IBM_WATSONX_URL` 环境变量为您想使用的 WatsonX API 的 URL。它默认为 `https://eu-de.ml.cloud.ibm.com`。

<CodeGroup>

```bash Mac
export IBM_WATSONX_API_KEY=***
export IBM_WATSONX_PROJECT_ID=***
```

```bash Windows
setx IBM_WATSONX_API_KEY ***
setx IBM_WATSONX_PROJECT_ID ***
```

</CodeGroup>

## 示例

将 `WatsonX` 与您的 `Agent` 一起使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.ibm import WatsonX

agent = Agent(
    model=WatsonX(id="meta-llama/llama-3-3-70b-instruct"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 在此处查看更多示例：[../examples/models/ibm](../examples/models/ibm)。 </Note>
