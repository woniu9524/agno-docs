IBM WatsonX は、IBM のクラウドプラットフォームを通じて強力な基盤モデルへのアクセスを提供します。

サポートされている IBM WatsonX のモデルはすべて[こちら](https://www.ibm.com/products/watsonx-ai/foundation-models)で確認できます。

- 一般的な用途には `meta-llama/llama-3-3-70b-instruct` の使用をお勧めします。
- コード関連のタスクには `ibm/granite-20b-code-instruct` の使用をお勧めします。
- 画像理解には `meta-llama/llama-3-2-11b-vision-instruct` の使用をお勧めします。

#### マルチモーダルサポート

WatsonX では、`Image` を入力としてサポートしています。

## 認証

`IBM_WATSONX_API_KEY` および `IBM_WATSONX_PROJECT_ID` 環境変数を設定してください。認証情報は[IBM Cloud](https://cloud.ibm.com/)から取得できます。
使用したい WatsonX API の URL を `IBM_WATSONX_URL` 環境変数に設定することもできます。デフォルトは `https://eu-de.ml.cloud.ibm.com` です。

<CodeGroup>

```bash Mac
export IBM_WATSONX_API_KEY=***
export IBM_WATSONX_PROJECT_ID=***
```

```bash Windows
setx IBM_WATSONX_API_KEY ***
setx IBM_WATSONX_PROJECT_ID ***
```

</CodeGroup>

## 例

`Agent` で `WatsonX` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.ibm import WatsonX

agent = Agent(
    model=WatsonX(id="meta-llama/llama-3-3-70b-instruct"),
    markdown=True
)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/ibm)でご覧ください。 </Note>
