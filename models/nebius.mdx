Nebius AI Studio は、Nebius が提供するプラットフォームであり、AI モデルを使用したアプリケーション構築プロセスを簡素化します。開発者がテキスト生成や画像生成などのさまざまな AI モデルを簡単にテスト、統合、ファインチューニングするためのツールとサービスを提供します。
利用可能なモデルのリストは[こちら](https://studio.nebius.com/)で確認できます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。

## 認証

`NEBIUS_API_KEY` 環境変数を設定します。キーは[Nebius AI Studio のこちら](https://studio.nebius.com/?modals=create-api-key)から取得してください。

<CodeGroup>

```bash Mac
export NEBIUS_API_KEY=***
```

```bash Windows
setx NEBIUS_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Nebius` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.nebius import Nebius

agent = Agent(
     model=Nebius(
        id="meta-llama/Llama-3.3-70B-Instruct",
        api_key=os.getenv("NEBIUS_API_KEY")
    ),
    markdown=True
)

# ターミナルに応答を表示します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/nebius)でご覧ください。 </Note>
