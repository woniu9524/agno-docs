---
title: 类似 OpenAI
description: 了解如何在 Agno 中使用类似 OpenAI 的模型。
---

许多提供商，如 Together、Groq、Nebius、Sambanova 等，都支持 OpenAI API 格式。通过替换 `base_url` 来使用 `OpenAILike` 模型访问它们。

## 示例

<CodeGroup>

```python agent.py
from os import getenv
from agno.agent import Agent, RunResponse
from agno.models.openai.like import OpenAILike

agent = Agent(
    model=OpenAILike(
        id="mistralai/Mixtral-8x7B-Instruct-v0.1",
        api_key=getenv("TOGETHER_API_KEY"),
        base_url="https://api.together.xyz/v1",
    )
)

# 在终端打印响应
agent.print_response("写一个两句话的恐怖故事。")
```

</CodeGroup>
