NVIDIA は、高度な NLP タスクに最適化された高性能言語モデルのスイートを提供しています。
これらのモデルは NeMo フレームワークの一部であり、最先端のモデルを効率的にトレーニング、ファインチューニング、デプロイするためのツールを提供します。NVIDIA の言語モデルは、GPU アクセラレーションによる大規模なワークロードを処理できるように設計されており、推論とトレーニングを高速化します。
NVIDIA のモデルを試して、アプリケーションに最適なものを見つけることをお勧めします。

NVIDIA のモデルは[こちら](https://build.nvidia.com/models)でご覧ください。

## 認証

`NVIDIA_API_KEY` 環境変数を設定します。キーは[こちら](https://build.nvidia.com/explore/discover)の Nvidia から取得してください。

<CodeGroup>

```bash Mac
export NVIDIA_API_KEY=***
```

```bash Windows
setx NVIDIA_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Nvidia` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.nvidia import Nvidia

agent = Agent(model=Nvidia(), markdown=True)

# ターミナルに応答を表示します
agent.print_response("Share a 2 sentence horror story")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/nvidia)でご覧ください。 </Note>
