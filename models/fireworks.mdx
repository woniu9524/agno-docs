Fireworks は、大規模言語モデルのエンドポイントを提供するプラットフォームです。

## 認証

`FIREWORKS_API_KEY` 環境変数を設定します。キーは [こちら](https://fireworks.ai/account/api-keys) から取得できます。

<CodeGroup>

```bash Mac
export FIREWORKS_API_KEY=***
```

```bash Windows
setx FIREWORKS_API_KEY ***
```

</CodeGroup>

## プロンプト キャッシュ

プロンプト キャッシュは、当社の `Fireworks` モデルを使用して自動的に行われます。Fireworks がプロンプト キャッシュを処理する方法については、[ドキュメント](https://docs.fireworks.ai/guides/prompt-caching#using-prompt-caching) を参照してください。

## 例

`Agent` で `Fireworks` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.fireworks import Fireworks

agent = Agent(
    model=Fireworks(id="accounts/fireworks/models/firefunction-v2"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は [こちら](../examples/models/fireworks) でご覧ください。 </Note>
