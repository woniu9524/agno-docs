Fireworks 是一个为大型语言模型提供端点的平台。

## 身份验证

设置您的 `FIREWORKS_API_KEY` 环境变量。在此处获取您的密钥：[https://fireworks.ai/account/api-keys](https://fireworks.ai/account/api-keys)。

<CodeGroup>

```bash Mac
export FIREWORKS_API_KEY=***
```

```bash Windows
setx FIREWORKS_API_KEY ***
```

</CodeGroup>

## Prompt 缓存

Prompt 缓存将使用我们的 `Fireworks` 模型自动进行。您可以在[他们的文档](https://docs.fireworks.ai/guides/prompt-caching#using-prompt-caching)中详细了解 Fireworks 如何处理缓存。

## 示例

将 `Fireworks` 与您的 `Agent` 一起使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.fireworks import Fireworks

agent = Agent(
    model=Fireworks(id="accounts/fireworks/models/firefunction-v2"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("分享一个两句话的恐怖故事。")

```

</CodeGroup>

<Note> 在此处查看更多示例：[../examples/models/fireworks](../examples/models/fireworks)。 </Note>
