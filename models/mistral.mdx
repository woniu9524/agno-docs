Mistral 是一个提供大型语言模型端点的平台。
在此处查看其模型库：[https://docs.mistral.ai/getting-started/models/models_overview/](https://docs.mistral.ai/getting-started/models/models_overview/)。

我们建议通过实验来找到最适合您用例的模型。以下是一些通用建议：

- `codestral` 模型非常适合代码生成和编辑。
- `mistral-large-latest` 模型适用于大多数用例。
- `open-mistral-nemo` 是一个免费模型，适用于大多数用例。
- `pixtral-12b-2409` 是一个视觉模型，非常适合 OCR、文档转录和图像比较。它在工具调用方面并不总是那么强大。

Mistral 有基于层级的速率限制。有关更多信息，请参阅[文档](https://docs.mistral.ai/deployment/laplateforme/tier/)。

## 身份验证

设置您的 `MISTRAL_API_KEY` 环境变量。在此处获取您的密钥：[https://console.mistral.ai/api-keys/](https://console.mistral.ai/api-keys/)。

<CodeGroup>

```bash Mac
export MISTRAL_API_KEY=***
```

```bash Windows
setx MISTRAL_API_KEY ***
```

</CodeGroup>

## 示例

在您的 `Agent` 中使用 `Mistral`：

<CodeGroup>

```python agent.py
import os

from agno.agent import Agent, RunResponse
from agno.models.mistral import MistralChat

mistral_api_key = os.getenv("MISTRAL_API_KEY")

agent = Agent(
    model=MistralChat(
        id="mistral-large-latest",
        api_key=mistral_api_key,
    ),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 在此处查看更多示例：[../examples/models/mistral](../examples/models/mistral)。 </Note>
