Mistral は、大規模言語モデルのエンドポイントを提供するプラットフォームです。
モデルのライブラリは[こちら](https://docs.mistral.ai/getting-started/models/models_overview/)で確認できます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項は以下のとおりです。

- `codestral` モデルは、コード生成および編集に適しています。
- `mistral-large-latest` モデルは、ほとんどのユースケースに適しています。
- `open-mistral-nemo` は、ほとんどのユースケースに適した無料モデルです。
- `pixtral-12b-2409` は、OCR、ドキュメントの文字起こし、画像比較に適したビジョンモデルです。ツール呼び出しには常に適しているわけではありません。

Mistral にはティアベースのレート制限があります。詳細については、[ドキュメント](https://docs.mistral.ai/deployment/laplateforme/tier/)を参照してください。

## 認証

`MISTRAL_API_KEY` 環境変数を設定します。キーは[こちら](https://console.mistral.ai/api-keys/)から取得できます。

<CodeGroup>

```bash Mac
export MISTRAL_API_KEY=***
```

```bash Windows
setx MISTRAL_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Mistral` を使用します。

<CodeGroup>

```python agent.py
import os

from agno.agent import Agent, RunResponse
from agno.models.mistral import MistralChat

mistral_api_key = os.getenv("MISTRAL_API_KEY")

agent = Agent(
    model=MistralChat(
        id="mistral-large-latest",
        api_key=mistral_api_key,
    ),
    markdown=True
)

# レスポンスをターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/mistral)でご覧ください。 </Note>
