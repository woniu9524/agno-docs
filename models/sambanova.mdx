Sambanova は、大規模言語モデルのエンドポイントを提供するプラットフォームです。Sambanova は現在、関数呼び出しをサポートしていないことに注意してください。

## 認証

`SAMBANOVA_API_KEY` 環境変数を設定します。キーは [こちら](https://cloud.sambanova.ai/apis) から取得してください。

<CodeGroup>

```bash Mac
export SAMBANOVA_API_KEY=***
```

```bash Windows
setx SAMBANOVA_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Sambanova` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.sambanova import Sambanova

agent = Agent(model=Sambanova(), markdown=True)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
