Sambanova 是一个为大型语言模型提供端点的平台。请注意，Sambanova 目前不支持函数调用。

## 身份验证

设置您的 `SAMBANOVA_API_KEY` 环境变量。从[此处](https://cloud.sambanova.ai/apis)获取您的密钥。

<CodeGroup>

```bash Mac
export SAMBANOVA_API_KEY=***
```

```bash Windows
setx SAMBANOVA_API_KEY ***
```

</CodeGroup>

## 示例

将 `Sambanova` 与您的 `Agent` 一起使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.sambanova import Sambanova

agent = Agent(model=Sambanova(), markdown=True)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
