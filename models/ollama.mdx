Ollama で大規模言語モデルをローカルで実行する

[Ollama](https://ollama.com) は、モデルをローカルで実行するための素晴らしいツールです。

Ollama は複数のオープンソースモデルをサポートしています。ライブラリは[こちら](https://ollama.com/library)で確認できます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項は以下のとおりです。

- `llama3.3` モデルは、ほとんどの基本的なユースケースに適しています。
- `qwen` モデルは、ツール使用で特に優れたパフォーマンスを発揮します。
- `deepseek-r1` モデルは、強力な推論能力を備えています。
- `phi4` モデルは、非常に小さいサイズでありながら強力です。

## モデルのセットアップ

[ollama](https://ollama.com) をインストールし、以下を使用してモデルを実行します。

```bash run model
ollama run llama3.1
```

これにより、モデルとのインタラクティブなセッションが提供されます。

または、Agno エージェントで使用するためにモデルをダウンロードするには

```bash pull model
ollama pull llama3.1
```

## 例

モデルをローカルに用意したら、`Ollama` モデルクラスを使用してアクセスします。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.ollama import Ollama

agent = Agent(
    model=Ollama(id="llama3.1"),
    markdown=True
)

# ターミナルに応答を表示する
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/ollama)でご覧ください。 </Note>
