Together は、大規模言語モデルのエンドポイントを提供するプラットフォームです。
モデルのライブラリは[こちら](https://www.together.ai/models)でご覧ください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。

Together にはティアベースのレート制限があります。詳細については、[ドキュメント](https://docs.together.ai/docs/rate-limits)をご覧ください。

## 認証

`TOGETHER_API_KEY` 環境変数を設定します。キーは[こちらから Together で取得](https://api.together.xyz/settings/api-keys)できます。

<CodeGroup>

```bash Mac
export TOGETHER_API_KEY=***
```

```bash Windows
setx TOGETHER_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Together` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.together import Together

agent = Agent(
    model=Together(id="meta-llama/Meta-Llama-3.1-8B-Instruct-Turbo"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/together)でご覧ください。 </Note>
