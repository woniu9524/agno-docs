Cohere の強力なコマンドモデルなどを活用します。

[Cohere](https://cohere.com) は幅広いモデルを提供しており、ファインチューニングに非常に適しています。モデルのライブラリは[こちら](https://docs.cohere.com/v2/docs/models)で確認できます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項は次のとおりです。

- `command` モデルは、ほとんどの基本的なユースケースに適しています。
- `command-light` モデルは、小規模なタスクや高速な推論に適しています。
- `command-r7b-12-2024` モデルは、RAG タスク、複雑な推論、および複数ステップのタスクに適しています。

Cohere はモデルのファインチューニングもサポートしています。ファインチューニングの方法については、[ガイド](https://docs.cohere.com/v2/docs/fine-tuning)をご覧ください。

Cohere にはティアベースのレート制限があります。詳細については、[ドキュメント](https://docs.cohere.com/v2/docs/rate-limits)を参照してください。

## 認証

`CO_API_KEY` 環境変数を設定します。キーは[こちら](https://dashboard.cohere.com/api-keys)から取得できます。

<CodeGroup>

```bash Mac
export CO_API_KEY=***
```

```bash Windows
setx CO_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Cohere` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.cohere import Cohere

agent = Agent(
    model=Cohere(id="command-r-08-2024"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/cohere)でご覧ください。 </Note>
