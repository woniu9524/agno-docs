Perplexity は、組み込みのウェブ検索機能を備えた強力な言語モデルを提供し、高度なリサーチおよび Q&A 機能を実現します。

Perplexity のモデルについては、[こちら](https://docs.perplexity.ai/guides/model-cards)をご覧ください。

## 認証

`PERPLEXITY_API_KEY` 環境変数を設定します。キーは、[こちら](https://www.perplexity.ai/settings/api)の Perplexity から取得してください。

<CodeGroup>

```bash Mac
export PERPLEXITY_API_KEY=***
```

```bash Windows
setx PERPLEXITY_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Perplexity` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.perplexity import Perplexity

agent = Agent(model=Perplexity(id="sonar-pro"), markdown=True)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story")

```

</CodeGroup>

<Note> その他の例は、[こちら](../examples/models/perplexity)をご覧ください。 </Note>
