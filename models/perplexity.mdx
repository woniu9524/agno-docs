Perplexity 提供了强大的语言模型，内置网页搜索功能，能够实现高级研究和问答功能。

在此处 [探索 Perplexity 的模型](https://docs.perplexity.ai/guides/model-cards)。

## 身份验证

设置您的 `PERPLEXITY_API_KEY` 环境变量。在此处 [从 Perplexity 获取您的密钥](https://www.perplexity.ai/settings/api)。

<CodeGroup>

```bash Mac
export PERPLEXITY_API_KEY=***
```

```bash Windows
setx PERPLEXITY_API_KEY ***
```

</CodeGroup>

## 示例

在您的 `Agent` 中使用 `Perplexity`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.perplexity import Perplexity

agent = Agent(model=Perplexity(id="sonar-pro"), markdown=True)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story")

```

</CodeGroup>

<Note> 在此处 [../examples/models/perplexity] 查看更多示例。 </Note>
