DeepSeek 是一个提供大型语言模型端点的平台。
在此处查看其模型库：[here](https://api-docs.deepseek.com/quick_start/pricing)。

我们建议进行实验以找到最适合您用例的模型。以下是一些通用建议：

- `deepseek-chat` 模型适用于大多数基本用例。
- `deepseek-reasoner` 模型适用于复杂的推理和多步任务。

DeepSeek 没有速率限制。请参阅其[文档](https://api-docs.deepseek.com/quick_start/rate_limit)了解在高流量期间如何处理响应较慢的情况。

## 身份验证

设置您的 `DEEPSEEK_API_KEY` 环境变量。在此处获取您的密钥：[here](https://platform.deepseek.com/api_keys)。

<CodeGroup>

```bash Mac
export DEEPSEEK_API_KEY=***
```

```bash Windows
setx DEEPSEEK_API_KEY ***
```

</CodeGroup>

## 示例

在您的 `Agent` 中使用 `DeepSeek`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.deepseek import DeepSeek

agent = Agent(model=DeepSeek(), markdown=True)

# 在终端中打印响应
agent.print_response("分享一个两句话的恐怖故事。")

```

</CodeGroup>

<Note> 在此处查看更多示例：[here](../examples/models/deepseek)。 </Note>
