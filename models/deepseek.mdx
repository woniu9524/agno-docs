DeepSeek は、大規模言語モデルのエンドポイントを提供するプラットフォームです。
モデルのライブラリについては、[こちら](https://api-docs.deepseek.com/quick_start/pricing)をご覧ください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項は以下のとおりです。

- `deepseek-chat` モデルは、ほとんどの基本的なユースケースに適しています。
- `deepseek-reasoner` モデルは、複雑な推論や複数ステップのタスクに適しています。

DeepSeek にはレート制限はありません。高トラフィック時の応答速度低下の対処方法については、[ドキュメント](https://api-docs.deepseek.com/quick_start/rate_limit)をご覧ください。

## 認証

`DEEPSEEK_API_KEY` 環境変数を設定します。キーは [こちら](https://platform.deepseek.com/api_keys) から取得してください。

<CodeGroup>

```bash Mac
export DEEPSEEK_API_KEY=***
```

```bash Windows
setx DEEPSEEK_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `DeepSeek` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.deepseek import DeepSeek

agent = Agent(model=DeepSeek(), markdown=True)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例については、[こちら](../examples/models/deepseek) をご覧ください。 </Note>
