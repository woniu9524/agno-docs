Groq は、大規模言語モデル向けの超高速 API エンドポイントを提供します。

Groq がサポートするすべてのモデルは、[こちら](https://console.groq.com/docs/models)で確認できます。

- 一般的な用途には `llama-3.3-70b-versatile` の使用をお勧めします。
- より高速な結果を得るには `llama-3.1-8b-instant` の使用をお勧めします。
- 画像理解には `llama-3.2-90b-vision-preview` の使用をお勧めします。

#### マルチモーダル サポート

Groq では、入力として `Image` をサポートしています。

## 認証

`GROQ_API_KEY` 環境変数を設定してください。キーは[こちら](https://console.groq.com/keys)から取得できます。

<CodeGroup>

```bash Mac
export GROQ_API_KEY=***
```

```bash Windows
setx GROQ_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Groq` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.groq import Groq

agent = Agent(
    model=Groq(id="llama-3.3-70b-versatile"),
    markdown=True
)

# 応答をターミナルに出力する
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/groq)で確認できます。 </Note>
