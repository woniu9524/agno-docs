Meta 提供了一套强大的多模态语言模型，在广泛的任务中表现出色，包括卓越的文本理解和视觉智能。

我们建议您进行实验，以找到最适合您用例的模型。以下是一些通用建议：

- `Llama-4-Scout-17B`：在大多数通用任务（包括多模态场景）中表现出色。
- `Llama-3.3-70B`：强大的指令遵循模型，适用于复杂的推理任务。

在此处 [Explore all the models](https://llama.developer.meta.com/docs/models) 探索所有模型。

## 身份验证

设置您的 `LLAMA_API_KEY` 环境变量：

<CodeGroup>
```bash Mac
export LLAMA_API_KEY=YOUR_API_KEY
```
```bash Windows
setx LLAMA_API_KEY YOUR_API_KEY
```
</CodeGroup>

## 示例

在您的 `Agent` 中使用 `Llama`：

<CodeGroup>
```python agent.py
from agno.agent import Agent
from agno.models.meta import Llama

agent = Agent(
    model=Llama(
        id="Llama-4-Maverick-17B-128E-Instruct-FP8",
    ),
    markdown=True
)

agent.print_response("Share a 2 sentence horror story.")
```
</CodeGroup>

<Note> 在此处 [here](../examples/models/meta) 查看更多示例。 </Note>
