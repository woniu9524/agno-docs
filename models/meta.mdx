Meta は、優れたテキスト理解やビジュアルインテリジェンスを含む、幅広いタスクで強力なパフォーマンスを発揮することで知られる、強力なマルチモーダル言語モデルのスイートを提供しています。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。以下に一般的な推奨事項を示します。

- `Llama-4-Scout-17B`: マルチモーダルシナリオを含む、ほとんどの一般的なタスクで優れたパフォーマンスを発揮します。
- `Llama-3.3-70B`: 複雑な推論タスク向けの強力な指示追従モデルです。

すべてのモデルは[こちら](https://llama.developer.meta.com/docs/models)でご覧ください。

## 認証

`LLAMA_API_KEY` 環境変数を設定します。

<CodeGroup>
```bash Mac
export LLAMA_API_KEY=YOUR_API_KEY
```
```bash Windows
setx LLAMA_API_KEY YOUR_API_KEY
```
</CodeGroup>

## 例

`Agent` で `Llama` を使用します。

<CodeGroup>
```python agent.py
from agno.agent import Agent
from agno.models.meta import Llama

agent = Agent(
    model=Llama(
        id="Llama-4-Maverick-17B-128E-Instruct-FP8",
    ),
    markdown=True
)

agent.print_response("Share a 2 sentence horror story.")
```
</CodeGroup>

<Note> その他の例は[こちら](../examples/models/meta)でご覧ください。 </Note>
