[vLLM](https://docs.vllm.ai/en/latest/) は、スループットが高くメモリ効率の良いLLMサービングのために設計された、高速で使いやすいLLM推論およびサービングライブラリです。

## 前提条件

vLLM をインストールしてモデルのサービングを開始します。

```bash install vLLM
pip install vllm
```

```bash start vLLM server
vllm serve Qwen/Qwen2.5-7B-Instruct \
    --enable-auto-tool-choice \
    --tool-call-parser hermes \
    --dtype float16 \
    --max-model-len 8192 \
    --gpu-memory-utilization 0.9
```

これにより、OpenAI 互換 API を持つ vLLM サーバーが起動します。

<Note>
  デフォルトの vLLM サーバー URL は `http://localhost:8000/` です。
</Note>

## 例

基本的なエージェント
<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.vllm import vLLM

agent = Agent(
    model=vLLM(
        id="meta-llama/Llama-3.1-8B-Instruct",
        base_url="http://localhost:8000/",
    ),
    markdown=True
)

agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

## 高度な使用方法

### ツールとの連携

vLLM モデルは、Agno ツールとシームレスに連携します。

```python with_tools.py
from agno.agent import Agent
from agno.models.vllm import vLLM
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=vLLM(id="meta-llama/Llama-3.1-8B-Instruct"),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True
)

agent.print_response("What's the latest news about AI?")
```

<Note> その他の例は [こちら](../examples/models/vllm) を参照してください。 </Note>

サポートされているモデルの全リストについては、[vLLM ドキュメント](https://docs.vllm.ai/en/latest/models/supported_models.html) を参照してください。
