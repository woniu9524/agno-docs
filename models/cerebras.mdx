[Cerebras Inference](https://inference-docs.cerebras.ai/introduction) は、Cerebras Wafer-Scale Engine と CS-3 システムによって強化された、高速・低遅延の AI モデル推論を提供します。Agno は Cerebras Python SDK と直接統合されており、シンプルなインターフェースで最先端の Llama モデルを使用できます。

## 前提条件

Agno で Cerebras を使用するには、以下の手順が必要です。

1. **必要なパッケージのインストール:**
   ```shell
   pip install cerebras-cloud-sdk
   ```

2. **API キーの設定:**
   Cerebras SDK は、API キーが環境変数として利用可能であることを想定しています。
   ```shell
   export CEREBRAS_API_KEY=your_api_key_here
   ```

## 基本的な使い方

Agno で Cerebras モデルを使用する方法は以下のとおりです。

```python
from agno.agent import Agent
from agno.models.cerebras import Cerebras

agent = Agent(
    model=Cerebras(id="llama-4-scout-17b-16e-instruct"),
    markdown=True,
)

# 応答をターミナルに出力します
agent.print_response("write a two sentence horror story")
```

## サポートされているモデル

Cerebras は現在、以下のモデルをサポートしています（最新リストについては[ドキュメント](https://inference-docs.cerebras.ai/introduction)を参照してください）。

| モデル名                      | モデル ID                       | パラメータ  | 知識     |
| ------------------------------- | ------------------------------ | ----------- | ------------- |
| Llama 4 Scout                   | llama-4-scout-17b-16e-instruct | 1090億 | 2024年8月   |
| Llama 3.1 8B                    | llama3.1-8b                    | 80億   | 2023年3月    |
| Llama 3.3 70B                   | llama-3.3-70b                  | 700億   | 2023年12月 |
| DeepSeek R1 Distill Llama 70B*  | deepseek-r1-distill-llama-70b  | 700億   | 2023年12月 |

\* DeepSeek R1 Distill Llama 70B はプライベートプレビューで利用可能です。

## 設定オプション

`Cerebras` クラスは、以下のパラメータを受け入れます。

| パラメータ     | タイプ           | 説明                                         | デフォルト                        |
| ------------- | -------------- | --------------------------------------------------- | ------------------------------ |
| `id`          | str            | モデル識別子（例: "llama-4-scout-17b-16e-instruct"） | **必須**                   |
| `name`        | str            | モデルの表示名                          | "Cerebras"                     |
| `provider`    | str            | プロバイダー名                                       | "Cerebras"                     |
| `api_key`     | Optional[str]  | API キー（`CEREBRAS_API_KEY` 環境変数にフォールバック）  | None                           |
| `max_tokens`  | Optional[int]  | 応答の最大トークン数                      | None                           |
| `temperature` | float          | サンプリング温度                                | 0.7                            |
| `top_p`       | float          | Top-p サンプリング値                                | 1.0                            |
| `request_params` | Optional[Dict[str, Any]] | 追加のリクエストパラメータ           | None                           |
