Azure のインフラストラクチャでホストされているさまざまなオープンソース モデルを使用します。詳細については、[こちら](https://learn.microsoft.com/azure/ai-services/models)をご覧ください。

Azure AI Foundry は、`Phi`、`Llama`、`Mistral`、`Cohere` などのモデルへのアクセスを提供します。

## 認証

[Azure Portal](https://portal.azure.com/) の Azure AI Foundry に移動してサービスを作成します。次に、環境変数を設定します。

<CodeGroup>

```bash Mac
export AZURE_API_KEY=***
export AZURE_ENDPOINT=***  # https://<your-host-name>.<your-azure-region>.models.ai.azure.com/models の形式
# オプション:
# export AZURE_API_VERSION=***
```

```bash Windows
setx AZURE_API_KEY ***  # https://<your-host-name>.<your-azure-region>.models.ai.azure.com/models の形式
setx AZURE_ENDPOINT ***
# オプション:
# setx AZURE_API_VERSION ***
```

</CodeGroup>

## 例

`Agent` で `AzureAIFoundry` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.azure import AzureAIFoundry

agent = Agent(
    model=AzureAIFoundry(id="Phi-4"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>
