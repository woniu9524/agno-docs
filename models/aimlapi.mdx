AI/ML API は、**Deepseek**、**Gemini**、**ChatGPT** など 300 以上の AI モデルへの統一されたアクセスを提供するプラットフォームです。本番環境レベルの稼働時間と高いレート制限を備えています。

## 認証

`AIMLAPI_API_KEY` 環境変数を設定します。キーは [aimlapi.com](https://aimlapi.com/?utm_source=agno&utm_medium=github&utm_campaign=integration) で取得できます。

<CodeGroup>

```bash Mac
export AIMLAPI_API_KEY=***
````

```bash Windows
setx AIMLAPI_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `AI/ML API` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.aimlapi import AIMLApi

agent = Agent(
    model=AIMLApi(id="meta-llama/Llama-3.2-11B-Vision-Instruct-Turbo"),
    markdown=True,
)

agent.print_response("Explain how black holes are formed.")
```

</CodeGroup>
