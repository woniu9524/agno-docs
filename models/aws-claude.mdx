通过 AWS Bedrock 使用 Claude 模型。这将提供针对 AWS 基础架构优化的原生 Claude 集成。

我们建议进行实验，以找到最适合您用例的模型。以下是一些通用建议：

- `anthropic.claude-3-5-sonnet-20241022-v2:0` 模型适用于大多数用例，并支持图像输入。
- `anthropic.claude-3-5-haiku-20241022-v2:0` 模型是他们最快的模型。

## 认证

设置您的 `AWS_ACCESS_KEY_ID`、`AWS_SECRET_ACCESS_KEY` 和 `AWS_REGION` 环境变量。

从[此处](https://us-east-1.console.aws.amazon.com/iam/home?region=us-east-1#/home)获取您的密钥。

<CodeGroup>

```bash Mac
export AWS_ACCESS_KEY_ID=***
export AWS_SECRET_ACCESS_KEY=***
export AWS_REGION=***
```

```bash Windows
setx AWS_ACCESS_KEY_ID ***
setx AWS_SECRET_ACCESS_KEY ***
setx AWS_REGION ***
```

</CodeGroup>

## 示例

在您的 `Agent` 中使用 `Claude`：

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.aws import Claude

agent = Agent(
    model=Claude(id="anthropic.claude-3-5-sonnet-20240620-v1:0"),
    markdown=True
)

# 在终端上打印响应
agent.print_response("讲一个两句话的恐怖故事。")
```

</CodeGroup>

<Note> 在[此处](../examples/models/aws/claude)查看更多示例。</Note>
