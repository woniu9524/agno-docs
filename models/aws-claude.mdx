AWS Bedrock を介して Claude モデルを使用します。これにより、AWS インフラストラクチャに最適化されたネイティブな Claude 統合が提供されます。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項を以下に示します。

- `anthropic.claude-3-5-sonnet-20241022-v2:0` モデルはほとんどのユースケースに適しており、画像入力をサポートしています。
- `anthropic.claude-3-5-haiku-20241022-v2:0` モデルは最も高速なモデルです。

## 認証

`AWS_ACCESS_KEY_ID`、`AWS_SECRET_ACCESS_KEY`、および `AWS_REGION` 環境変数を設定します。

キーは[こちら](https://us-east-1.console.aws.amazon.com/iam/home?region=us-east-1#/home)から取得してください。

<CodeGroup>

```bash Mac
export AWS_ACCESS_KEY_ID=***
export AWS_SECRET_ACCESS_KEY=***
export AWS_REGION=***
```

```bash Windows
setx AWS_ACCESS_KEY_ID ***
setx AWS_SECRET_ACCESS_KEY ***
setx AWS_REGION ***
```

</CodeGroup>

## 例

`Agent` で `Claude` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.aws import Claude

agent = Agent(
    model=Claude(id="anthropic.claude-3-5-sonnet-20240620-v1:0"),
    markdown=True
)

# Print the response on the terminal
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/aws/claude)でご覧ください。 </Note>
