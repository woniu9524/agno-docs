Google の Gemini モデルを [Google AI Studio](https://ai.google.dev/gemini-api/docs) または [Google Cloud Vertex AI](https://cloud.google.com/vertex-ai/generative-ai/docs/overview) を通じて使用します。これらのプラットフォームは、大規模言語モデルやその他のサービスへのアクセスを提供します。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。Gemini `2.x` ファミリーのモデルに関する一般的な推奨事項を以下に示します。

- `gemini-2.0-flash` はほとんどのユースケースに適しています。
- `gemini-2.0-flash-lite` は最も費用対効果の高いモデルです。
- `gemini-2.5-pro-exp-03-25` は最も強力なマルチモーダルモデルです。

利用可能なモデルバージョンについては、[Google AI Studio ドキュメント](https://ai.google.dev/gemini-api/docs/models)および[Vertex AI ドキュメント](https://cloud.google.com/vertex-ai/generative-ai/docs/learn/models)を参照してください。

## 認証

Gemini モデルは、Google AI Studio または Google Cloud の Vertex AI のいずれかを通じて使用できます。

### Google AI Studio

`GOOGLE_API_KEY` 環境変数を設定します。[Google AI Studio](https://ai.google.dev/gemini-api/docs/api-key) から取得できます。

<CodeGroup>

```bash Mac
export GOOGLE_API_KEY=***
```

```bash Windows
setx GOOGLE_API_KEY ***
```

</CodeGroup>

### Vertex AI

Google Cloud で Vertex AI を使用するには、次の手順を実行します。

1. プロジェクトと開発環境をセットアップするには、[Vertex AI ドキュメント](https://cloud.google.com/vertex-ai/docs/start/cloud-environment)を参照してください。

2. `gcloud` CLI をインストールして認証します（詳細については、[クイックスタート](https://cloud.google.com/vertex-ai/generative-ai/docs/start/quickstarts/quickstart-multimodal)を参照してください）。

```bash
gcloud auth application-default login
```

3. Vertex AI API を有効にし、プロジェクト ID 環境変数を設定します（または、`Agent` 設定で `project_id` を設定することもできます）。

次の変数をエクスポートします。

```bash
export GOOGLE_GENAI_USE_VERTEXAI="true"
export GOOGLE_CLOUD_PROJECT="your-gcloud-project-id"
export GOOGLE_CLOUD_LOCATION="your-gcloud-location"
```

または、Agent の設定を更新します。

```python
agent = Agent(
    model=Gemini(
        id="gemini-1.5-flash",
        vertexai=True,
        project_id="your-gcloud-project-id",
        location="your-gcloud-location",
    ),
)
```

## 例

`Agent` で `Gemini` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.google import Gemini

# Google AI Studio を使用する場合
agent = Agent(
    model=Gemini(id="gemini-2.0-flash"),
    markdown=True,
)

# または Vertex AI を使用する場合
agent = Agent(
    model=Gemini(
        id="gemini-2.0-flash",
        vertexai=True,
        project_id="your-project-id",  # GOOGLE_CLOUD_PROJECT が設定されている場合はオプション
        location="us-central1",  # オプション
    ),
    markdown=True,
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/gemini)でご覧ください。 </Note>

## Grounding と検索

Gemini モデルは、オプションのパラメータを通じて Grounding および検索機能をサポートします。これにより、Grounding または検索用のツールが Gemini に自動的に送信されます。詳細については、[こちら](https://ai.google.dev/gemini-api/docs/grounding?lang=python)を参照してください。

これらの機能を使用するには、Gemini モデルを初期化する際に対応するパラメータを設定します。

Grounding を使用するには:

<CodeGroup>

```python
from agno.agent import Agent
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(id="gemini-2.0-flash", grounding=True),
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Any news from USA?")
```

</CodeGroup>

検索を使用するには:

<CodeGroup>

```python
from agno.agent import Agent
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(id="gemini-2.0-flash", search=True),
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("What's happening in France?")
```

</CodeGroup>

Grounding または検索結果を実際に出力で確認するには、Agent 設定で `show_tool_calls=True` を設定します。
