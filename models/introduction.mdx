モデルはエージェントの**頭脳**として機能し、推論、行動、ユーザーへの応答を支援します。モデルが優れているほど、エージェントは賢くなります。

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="Share 15 minute healthy recipes.",
    markdown=True,
)
agent.print_response("Share a breakfast recipe.", stream=True)
```

## エラー処理

`Agent` で `exponential_backoff` を `True` に設定すると、サードパーティのモデルプロバイダーのエラーが原因で失敗したリクエストを自動的に再試行できます。

```python
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    exponential_backoff=True,
    retries=2,
    retry_delay=1,
)
```
