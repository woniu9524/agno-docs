模型是 Agent 的**大脑**——帮助它进行推理、行动和响应用户。模型越好，Agent 就越智能。

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="Share 15 minute healthy recipes.",
    markdown=True,
)
agent.print_response("Share a breakfast recipe.", stream=True)
```

## 错误处理

您可以将 `Agent` 上的 `exponential_backoff` 设置为 `True`，以自动重试因第三方模型提供商错误而失败的请求。

```python
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    exponential_backoff=True,
    retries=2,
    retry_delay=1,
)
```
