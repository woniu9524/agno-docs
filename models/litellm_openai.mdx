## プロキシサーバーとの統合

LiteLLMは、OpenAI互換のプロキシサーバーとしても使用でき、統一されたAPIを通じてさまざまなモデルにリクエストをルーティングできます。

### プロキシサーバーの起動

まず、プロキシサポート付きでLiteLLMをインストールします。

```shell
pip install 'litellm[proxy]'
```

プロキシサーバーを起動します。

```shell
litellm --model gpt-4o --host 127.0.0.1 --port 4000
```

### プロキシの使用

`LiteLLMOpenAI` クラスは、OpenAI互換のインターフェースを使用してLiteLLMプロキシに接続します。

```python
from agno.agent import Agent
from agno.models.litellm import LiteLLMOpenAI

agent = Agent(
    model=LiteLLMOpenAI(
        id="gpt-4o",  # 使用するモデルID
    ),
    markdown=True,
)

agent.print_response("Share a 2 sentence horror story")
```

### 設定オプション

`LiteLLMOpenAI` クラスは、以下のパラメータを受け入れます。

| パラメータ | タイプ | 説明 | デフォルト |
| --- | --- | --- | --- |
| `id` | str | モデル識別子 | "gpt-4o" |
| `name` | str | モデルの表示名 | "LiteLLM" |
| `provider` | str | プロバイダー名 | "LiteLLM" |
| `api_key` | str | APIキー (LITELLM_API_KEY環境変数にフォールバックします) | None |
| `base_url` | str | LiteLLMプロキシサーバーのURL | "http://0.0.0.0:4000" |
