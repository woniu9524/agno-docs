Hugging Face は、テキスト生成、要約、翻訳、質問応答など、さまざまな NLP タスクに対応した、幅広い最先端の言語モデルを提供しています。これらのモデルは Hugging Face Transformers ライブラリを通じて利用可能であり、使いやすさ、柔軟性、包括的なドキュメントにより広く採用されています。

HuggingFace の言語モデルは[こちら](https://huggingface.co/docs/text-generation-inference/en/supported_models)からご覧ください。

## 認証

`HF_TOKEN` 環境変数を設定してください。トークンは[HuggingFace のこちら](https://huggingface.co/settings/tokens)から取得できます。

<CodeGroup>

```bash Mac
export HF_TOKEN=***
```

```bash Windows
setx HF_TOKEN ***
```

</CodeGroup>

## 例

`Agent` で `HuggingFace` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.huggingface import HuggingFace

agent = Agent(
    model=HuggingFace(
        id="meta-llama/Meta-Llama-3-8B-Instruct",
        max_tokens=4096,
    ),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/huggingface)でご覧ください。 </Note>
