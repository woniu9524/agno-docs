Azure のインフラストラクチャを通じて OpenAI モデルを使用します。詳細については、[こちら](https://learn.microsoft.com/azure/ai-services/openai/overview) を参照してください。

Azure OpenAI は、`GPT-4o` や `o3-mini` などの OpenAI モデルへのアクセスを提供します。

## 認証

[Azure Portal](https://portal.azure.com/) の Azure OpenAI に移動し、サービスを作成します。次に、Azure AI Studio ポータルを使用して、デプロイを作成し、環境変数を設定します。

<CodeGroup>

```bash Mac
export AZURE_OPENAI_API_KEY=***
export AZURE_OPENAI_ENDPOINT=***  # https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name> の形式
# オプション:
# export AZURE_OPENAI_DEPLOYMENT=***
```

```bash Windows
setx AZURE_OPENAI_API_KEY ***  # https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name> の形式
setx AZURE_OPENAI_ENDPOINT ***
# オプション:
# setx AZURE_OPENAI_DEPLOYMENT ***
```

</CodeGroup>

## 例

`Agent` で `AzureOpenAI` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.azure import AzureOpenAI
from os import getenv

agent = Agent(
    model=AzureOpenAI(id="gpt-4o"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

## プロンプト キャッシュ

プロンプト キャッシュは、`AzureOpenAI` モデルを使用して自動的に行われます。OpenAI がキャッシュを処理する方法については、[ドキュメント](https://platform.openai.com/docs/guides/prompt-caching) を参照してください。
