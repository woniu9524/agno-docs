通过 Azure 的基础设施使用 OpenAI 模型。在此处了解更多信息：[https://learn.microsoft.com/azure/ai-services/openai/overview](https://learn.microsoft.com/azure/ai-services/openai/overview)。

Azure OpenAI 提供对 OpenAI 模型（如 `GPT-4o`、`o3-mini` 等）的访问。

## 身份验证

导航到 [Azure Portal](https://portal.azure.com/) 中的 Azure OpenAI 并创建一个服务。然后，使用 Azure AI Studio 门户创建部署并设置您的环境变量：

<CodeGroup>

```bash Mac
export AZURE_OPENAI_API_KEY=***
export AZURE_OPENAI_ENDPOINT=***  # 格式为 https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name>
# 可选：
# export AZURE_OPENAI_DEPLOYMENT=***
```

```bash Windows
setx AZURE_OPENAI_API_KEY ***  # 格式为 https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name>
setx AZURE_OPENAI_ENDPOINT ***
# 可选：
# setx AZURE_OPENAI_DEPLOYMENT ***
```

</CodeGroup>

## 示例

在您的 `Agent` 中使用 `AzureOpenAI`：

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.azure import AzureOpenAI
from os import getenv

agent = Agent(
    model=AzureOpenAI(id="gpt-4o"),
    markdown=True
)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

## Prompt 缓存

Prompt 缓存将自动使用我们的 `AzureOpenAI` 模型进行。您可以在[他们的文档](https://platform.openai.com/docs/guides/prompt-caching)中了解更多关于 OpenAI 如何处理缓存的信息。
