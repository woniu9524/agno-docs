OpenRouter は、大規模言語モデルのエンドポイントを提供するプラットフォームです。

## 認証

`OPENROUTER_API_KEY` 環境変数を設定してください。キーは [こちら](https://openrouter.ai/settings/keys) から取得できます。

<CodeGroup>

```bash Mac
export OPENROUTER_API_KEY=***
```

```bash Windows
setx OPENROUTER_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `OpenRouter` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.openrouter import OpenRouter

agent = Agent(
    model=OpenRouter(id="gpt-4o"),
    markdown=True
)

# ターミナルに応答を表示します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

## プロンプト キャッシュ

プロンプト キャッシュは、使用されているプロバイダーがサポートしている場合、`OpenRouter` モデルを使用して自動的に行われます。それ以外の場合は、`cache_control` ヘッダーを介してアクティブ化できます。OpenRouter でのプロンプト キャッシュの詳細については、[ドキュメント](https://openrouter.ai/docs/features/prompt-caching) を参照してください。
