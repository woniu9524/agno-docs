Claude は Anthropic による基盤となる AI モデルのファミリーであり、さまざまなアプリケーションで使用できます。
モデルの比較については、[こちら](https://docs.anthropic.com/en/docs/about-claude/models#model-comparison-table) を参照してください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項は次のとおりです。

- `claude-3-5-sonnet-20241022` モデルは、ほとんどのユースケースに適しており、画像入力をサポートしています。
- `claude-3-5-haiku-20241022` モデルは、最も高速なモデルです。

Anthropic は API にレート制限を設けています。詳細については、[ドキュメント](https://docs.anthropic.com/en/api/rate-limits#response-headers) を参照してください。

## 認証

`ANTHROPIC_API_KEY` 環境変数を設定します。Anthropic の [こちら](https://console.anthropic.com/settings/keys) から取得できます。

<CodeGroup>

```bash Mac
export ANTHROPIC_API_KEY=***
```

```bash Windows
setx ANTHROPIC_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `Claude` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(id="claude-3-5-sonnet-20240620"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")
```

## プロンプトキャッシュ

`cache_system_prompt` を `True` に設定することで、`Claude` モデルでシステムプロンプトキャッシュを有効にできます。

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-3-5-sonnet-20241022",
        cache_system_prompt=True,
    ),
)
```

Agno の `Claude` モデルでのプロンプトキャッシュの詳細については、[こちら](https://docs.agno.com/examples/models/anthropic/prompt_caching) を参照してください。

</CodeGroup>
