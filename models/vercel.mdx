Vercel v0 API は、最新の Web アプリケーションの構築用に設計された大規模言語モデルを提供します。テキストと画像の入力をサポートし、高速なストリーミング応答を提供し、OpenAI Chat Completions API フォーマットと互換性があります。フロントエンドおよびフルスタックの Web 開発コード生成に最適化されています。

詳細については、[公式 Vercel v0 API ドキュメント](https://vercel.com/docs/v0/api)を参照してください。

## 認証

`V0_API_KEY` 環境変数を設定します。API キーは [v0.dev](https://v0.dev/chat/settings/keys) で作成できます。

<CodeGroup>

```bash Mac
export V0_API_KEY=your-v0-api-key
```

```bash Windows
setx V0_API_KEY your-v0-api-key
```

</CodeGroup>

## 例

`Agent` で `v0` を使用します。次の例では、`agno/models/vercel.py` に配置された `v0` Python クラス (提供されているもの) があることを前提としています。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.vercel import v0

agent = Agent(
    model=v0(id="v0-1.0-md"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Create a simple web app that displays a random number between 1 and 100.")

# agent.print_response("Create a webapp to fetch the weather of a city and display humidity, temperature, and wind speed in cards, use shadcn components and tailwind css")

```

</CodeGroup>

<Note> その他の例は [こちら](/examples/models/vercel) でご覧ください。 </Note>
