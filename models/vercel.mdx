Vercel v0 API 提供大型语言模型，专为构建现代 Web 应用程序而设计。它支持文本和图像输入，提供快速的流式响应，并且与 OpenAI Chat Completions API 格式兼容。它针对前端和全栈 Web 开发代码生成进行了优化。

有关更多详细信息，请参阅 [官方 Vercel v0 API 文档](https://vercel.com/docs/v0/api)。

## 身份验证

设置您的 `V0_API_KEY` 环境变量。您可以在 [v0.dev](https://v0.dev/chat/settings/keys) 上创建 API 密钥。

<CodeGroup>

```bash Mac
export V0_API_KEY=your-v0-api-key
```

```bash Windows
setx V0_API_KEY your-v0-api-key
```

</CodeGroup>

## 示例

将 `v0` 与您的 `Agent` 一起使用。以下示例假定您在 `agno/models/vercel.py` 中提供了 `v0` Python 类。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.vercel import v0

agent = Agent(
    model=v0(id="v0-1.0-md"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("创建一个显示 1 到 100 之间随机数的简单 Web 应用。")

# agent.print_response("创建一个获取城市天气并在卡片中显示湿度、温度和风速的 Web 应用，使用 shadcn 组件和 tailwind css")

```

</CodeGroup>
