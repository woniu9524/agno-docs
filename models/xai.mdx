xAI は、大規模言語モデルのエンドポイントを提供するプラットフォームです。
モデルのリストは[こちら](https://docs.x.ai/docs/models)で確認してください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。`grok-beta` モデルはほとんどのユースケースに適しています。

xAI は API にレート制限があります。[ドキュメント](https://docs.x.ai/docs/usage-tiers-and-rate-limits)で詳細を確認してください。

## 認証

`XAI_API_KEY` 環境変数を設定してください。xAI から[こちら](https://console.x.ai/)で取得できます。

<CodeGroup>

```bash Mac
export XAI_API_KEY=sk-***
```

```bash Windows
setx XAI_API_KEY sk-***
```

</CodeGroup>

## 例

`Agent` で `xAI` を使用します。

<CodeGroup>

```python agent.py

from agno.agent import Agent, RunResponse
from agno.models.xai import xAI

agent = Agent(
    model=xAI(id="grok-beta"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/xai)でご覧ください。 </Note>
