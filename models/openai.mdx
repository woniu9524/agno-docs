GPT モデルはクラス最高の LLM であり、**Agents** のデフォルト LLM として使用されます。OpenAI は、世界クラスのさまざまなモデルをサポートしています。モデルについては、[こちら](https://platform.openai.com/docs/models)をご覧ください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項を以下に示します。

- `gpt-4o` は、ほとんどの一般的なユースケースに適しています。
- `gpt-4o-mini` モデルは、小規模なタスクと高速な推論に適しています。
- `o1` モデルは、複雑な推論と複数ステップのタスクに適しています。
- `o3-mini` は、ツール呼び出しと構造化された出力をサポートする強力な推論モデルですが、コストははるかに低いです。

OpenAI には、ティアベースのレート制限があります。詳細については、[ドキュメント](https://platform.openai.com/docs/guides/rate-limits/usage-tiers)をご覧ください。

## 認証

`OPENAI_API_KEY` 環境変数を設定します。[OpenAI からこちら](https://platform.openai.com/account/api-keys)で取得できます。

<CodeGroup>

```bash Mac
export OPENAI_API_KEY=sk-***
```

```bash Windows
setx OPENAI_API_KEY sk-***
```

</CodeGroup>

## 例

`Agent` で `OpenAIChat` を使用します。

<CodeGroup>

```python agent.py

from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    markdown=True
)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
