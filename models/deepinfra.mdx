DeepInfra の強力なコマンドモデルなどを活用します。

[DeepInfra](https://deepinfra.com) は、幅広いモデルをサポートしています。モデルのライブラリは[こちら](https://deepinfra.com/models)をご覧ください。

ユースケースに最適なモデルを見つけるために、実験することをお勧めします。一般的な推奨事項を以下に示します。

- `deepseek-ai/DeepSeek-R1-Distill-Llama-70B` モデルは、推論に適しています。
- `meta-llama/Llama-2-70b-chat-hf` モデルは、基本的なユースケースに適しています。
- `meta-llama/Llama-3.3-70B-Instruct` モデルは、複数ステップのタスクに適しています。

DeepInfra にはレート制限があります。詳細については、[ドキュメント](https://deepinfra.com/docs/advanced/rate-limits)をご覧ください。

## 認証

`DEEPINFRA_API_KEY` 環境変数を設定します。キーは[こちら](https://deepinfra.com/dash/api_keys)から取得してください。

<CodeGroup>

```bash Mac
export DEEPINFRA_API_KEY=***
```

```bash Windows
setx DEEPINFRA_API_KEY ***
```

</CodeGroup>

## 例

`Agent` で `DeepInfra` を使用します。

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.deepinfra import DeepInfra

agent = Agent(
    model=DeepInfra(id="meta-llama/Llama-2-70b-chat-hf"),
    markdown=True
)

# 応答をターミナルに出力します
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> その他の例は[こちら](../examples/models/deepinfra)をご覧ください。 </Note>
