使用 AWS Bedrock 访问 AWS 上的各种基础模型。在[门户网站](https://us-east-1.console.aws.amazon.com/bedrock/home?region=us-east-1#/model-catalog)上管理您的模型访问权限。

查看所有 [AWS Bedrock 基础模型](https://docs.aws.amazon.com/bedrock/latest/userguide/models-supported.html)。并非所有 Bedrock 模型都支持所有功能。请参阅[每个模型支持的功能](https://docs.aws.amazon.com/bedrock/latest/userguide/conversation-inference-supported-models-features.html)。

我们建议进行实验以找到最适合您用例的模型。以下是一些通用建议：

- 对于性能普遍良好的 Mistral 模型，请查看 `mistral.mistral-large-2402-v1:0`。
- 您可以试用 Amazon Nova 模型。将 `amazon.nova-pro-v1:0` 用于通用任务。
- 对于 Claude 模型，请参阅我们的[Claude 集成](/models/aws-claude)。

<Warning>
目前尚不支持 AWS Bedrock 的异步使用。当将 `AwsBedrock` 与 `Agent` 结合使用时，您只能使用 `agent.run` 和 `agent.print_response`。
</Warning>

## 身份验证

设置您的 `AWS_ACCESS_KEY_ID`、`AWS_SECRET_ACCESS_KEY` 和 `AWS_REGION` 环境变量。

从[此处](https://us-east-1.console.aws.amazon.com/iam/home?region=us-east-1#/home)获取您的密钥。

<CodeGroup>

```bash Mac
export AWS_ACCESS_KEY_ID=***
export AWS_SECRET_ACCESS_KEY=***
export AWS_REGION=***
```

```bash Windows
setx AWS_ACCESS_KEY_ID ***
setx AWS_SECRET_ACCESS_KEY ***
setx AWS_REGION ***
```

</CodeGroup>

## 示例

将 `AwsBedrock` 与您的 `Agent` 结合使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.aws import AwsBedrock

agent = Agent(
    model=AwsBedrock(id="mistral.mistral-large-2402-v1:0"),
    markdown=True
)

# 在终端上打印响应
agent.print_response("分享一个两句话的恐怖故事。")
```

</CodeGroup>

<Note>在此处查看更多示例：[/examples/models/aws/bedrock](/examples/models/aws/bedrock)。</Note>
