**PDFBytesKnowledgeBase** 从 **bytes 或 IO 流中读取 PDF 内容**，将其转换为向量嵌入并加载到向量数据库中。这在处理动态生成的 PDF、API 响应或文件上传时非常有用，无需将文件保存到磁盘。

## 用法

<Note>

我们将在此示例中使用本地的 LanceDB 数据库。[请确保它正在运行](https://docs.agno.com/vectordb/lancedb)

</Note>

```shell
pip install pypdf
```

```python knowledge_base.py
from agno.agent import Agent
from agno.knowledge.pdf import PDFBytesKnowledgeBase
from agno.vectordb.lancedb import LanceDb

vector_db = LanceDb(
    table_name="recipes_async",
    uri="tmp/lancedb",
)

with open("data/pdfs/ThaiRecipes.pdf", "rb") as f:
    pdf_bytes = f.read()

knowledge_base = PDFBytesKnowledgeBase(
    pdfs=[pdf_bytes],
    vector_db=vector_db,
)
knowledge_base.load(recreate=False)  # 首次运行后注释掉此行

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

agent.print_response("How to make Tom Kha Gai?", markdown=True)
```

## 参数

| 参数 | 类型 | 默认值 | 描述 |
| --------- | -------------- | ------- | --- |
| pdfs | Union[List[bytes], List[IO]] | - | PDF 内容列表，以 bytes 或 IO 流的形式提供。 |
| exclude_files | List[str] | [] | 要排除的文件模式列表（继承自基类）。 |
| reader | Union[PDFReader, PDFImageReader] | PDFReader() | 一个 PDFReader 或 PDFImageReader，用于将 PDF 转换为 Documents 以便存入向量数据库。 |

`PDFBytesKnowledgeBase` 是 [AgentKnowledge](/reference/knowledge/base) 类的子类，可以访问相同的参数。
