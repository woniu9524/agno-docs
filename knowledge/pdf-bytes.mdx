**PDFBytesKnowledgeBase** は、**バイトまたはIOストリームからPDFコンテンツを読み取り**、それをベクトル埋め込みに変換してベクトルデータベースにロードします。これは、ファイルをディスクに保存する必要なく、動的に生成されたPDF、API応答、またはファイルアップロードを扱う場合に便利です。

## 使用方法

<Note>

この例ではローカルのLanceDBデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/lancedb)。

</Note>

```shell
pip install pypdf
```

```python knowledge_base.py
from agno.agent import Agent
from agno.knowledge.pdf import PDFBytesKnowledgeBase
from agno.vectordb.lancedb import LanceDb

vector_db = LanceDb(
    table_name="recipes_async",
    uri="tmp/lancedb",
)

with open("data/pdfs/ThaiRecipes.pdf", "rb") as f:
    pdf_bytes = f.read()

knowledge_base = PDFBytesKnowledgeBase(
    pdfs=[pdf_bytes],
    vector_db=vector_db,
)
knowledge_base.load(recreate=False)  # 初回実行後にコメントアウトしてください

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

agent.print_response("How to make Tom Kha Gai?", markdown=True)
```

## パラメータ

| パラメータ | タイプ | デフォルト | 説明 |
| --------- | -------------- | ------- | --- |
| pdfs | Union[List[bytes], List[IO]] | - | バイトまたはIOストリームとしてのPDFコンテンツのリスト。 |
| exclude_files | List[str] | [] | 除外するファイルパターンのリスト（基底クラスから継承）。 |
| reader | Union[PDFReader, PDFImageReader] | PDFReader() | PDFをベクトルデータベース用のドキュメントに変換するPDFリーダーまたはPDF画像リーダー。 |

`PDFBytesKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
