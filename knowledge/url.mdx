**URLKnowledgeBase** は **URL** のリンクを読み込み、ベクトル埋め込みに変換してベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)。

</Note>

```shell
pip install qdrant-client
```

```python
from agno.agent import Agent
from agno.knowledge.url import UrlKnowledge
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

knowledge_base = UrlKnowledge(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)
```

次に、`Agent` で `knowledge_base` を使用します。

```python
from agno.agent import Agent
from agno.knowledge.url import UrlKnowledge
from agno.models.openai import OpenAIChat

# ナレッジベースの初期化
COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

knowledge_base = UrlKnowledge(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)
agent_with_knowledge = Agent(
    name="Agent with Knowledge",
    model=OpenAIChat(id="gpt-4o"),
    knowledge=knowledge_base,
    show_tool_calls=True,
    markdown=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    knowledge_base.load()

    agent_with_knowledge.print_response(
        "Tell me about teams with context to agno", stream=True
    )
```

#### URLKnowledgeBase は非同期ロードもサポートしています。

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.url import UrlKnowledge
from agno.models.openai import OpenAIChat
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

# ナレッジベースの初期化
agent_knowledge = UrlKnowledge(
    urls=["https://docs.agno.com/introduction"],
    vector_db=Qdrant(
        collection=COLLECTION_NAME,
        url="http://localhost:6333",
    ),
)

agent_with_knowledge = Agent(
    name="Agent with Knowledge",
    model=OpenAIChat(id="gpt-4o"),
    knowledge=agent_knowledge,
    show_tool_calls=True,
    markdown=True,
)

if __name__ == "__main__":
    asyncio.run(agent_knowledge.aload(recreate=False))

    asyncio.run(
        agent_with_knowledge.aprint_response(
            "Tell me about teams with context to agno", stream=True
        )
    )
```

## パラメータ

| パラメータ | タイプ               | デフォルト             | 説明                                                                           |
| --------- | ------------------ | ------------------- | ------------------------------------------------------------------------------------- |
| `urls`    | `List[str]`        | -                   | 読み込むウェブサイトのURL。                                                                                          |
| `reader`  | `URLReader`        | `URLReader()`       | ウェブサイトの `URL` を読み込み、ベクトルデータベース用の `Documents` に変換する `URLReader`。 |

`URLKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
