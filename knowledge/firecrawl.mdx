**FirecrawlKnowledgeBase** 读取 **URL** 链接，将其转换为向量嵌入并加载到向量数据库中。

## 用法

<Note>

我们在此示例中使用本地Qdrant数据库。[确保它正在运行](https://docs.agno.com/vectordb/qdrant)

</Note>

```shell
pip install firecrawl-py qdrant-client
```

```python
from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# 使用种子URL创建知识库
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)
```

然后将`knowledge_base`与`Agent`一起使用：

```python
from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# 使用种子URL创建知识库
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)

# 使用知识库创建代理
agent = Agent(knowledge=knowledge_base, search_knowledge=True, debug_mode=True)

if __name__ == "__main__":
    # 首次运行时注释掉
    knowledge_base.load(recreate=False)

    agent.print_response("How does agno work?", markdown=True)
```

#### FirecrawlKnowledgeBase也支持异步加载。

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# 使用种子URL创建知识库
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)

# 使用知识库创建代理
agent = Agent(knowledge=knowledge_base, search_knowledge=True, debug_mode=True)

if __name__ == "__main__":
    # 首次运行时注释掉
    asyncio.run(knowledge_base.aload(recreate=False))

    asyncio.run(agent.aprint_response("How does agno work?", markdown=True))
```

## 参数

| 参数 | 类型 | 默认值 | 描述 |
| --------- | ------ | ------ | ------ |
| `urls` | `List[str]` | - | 要读取的网站的URL。 |
| `reader` | `FirecrawlReader` | `FirecrawlReader()` | 一个`FirecrawlReader`，用于读取网站`URLs`并将其转换为向量数据库的`Documents`。 |

`FirecrawlKnowledgeBase` 是 `AgentKnowledge` 类的一个子类，可以访问相同的参数。
