**FirecrawlKnowledgeBase**は**URL**リンクを読み込み、ベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)。

</Note>

```shell
pip install firecrawl-py qdrant-client
```

```python
from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# シードURLでナレッジベースを作成します
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)
```

次に、`knowledge_base`を`Agent`で使用します。

```python
from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# シードURLでナレッジベースを作成します
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)

# ナレッジベースでエージェントを作成します
agent = Agent(knowledge=knowledge_base, search_knowledge=True, debug_mode=True)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    knowledge_base.load(recreate=False)

    agent.print_response("How does agno work?", markdown=True)
```

#### FirecrawlKnowledgeBaseは非同期ロードもサポートしています。

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.firecrawl import FireCrawlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# シードURLでナレッジベースを作成します
knowledge_base = FireCrawlKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    vector_db=vector_db,
)

# ナレッジベースでエージェントを作成します
agent = Agent(knowledge=knowledge_base, search_knowledge=True, debug_mode=True)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    asyncio.run(agent.aprint_response("How does agno work?", markdown=True))
```

## パラメータ

| パラメータ | タイプ               | デフォルト             | 説明                                                                                              |
| --------- | ------------------ | ------------------- | ------------------------------------------------------------------------------------------------- |
| `urls`    | `List[str]`        | -                   | 読み込むウェブサイトのURL。                                                                                             |
| `reader`  | `FirecrawlReader` | `FirecrawlReader()` | ウェブサイトの`URL`を読み込み、ベクトルデータベース用の`Documents`に変換する`FirecrawlReader`。 |

`FirecrawlKnowledgeBase`は[AgentKnowledge](/reference/knowledge/base)クラスのサブクラスであり、同じパラメータにアクセスできます。
