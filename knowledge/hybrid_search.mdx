ハイブリッド検索を使用すると、厳密な一致の精度とセマンティックな理解のインテリジェンスの両方を得ることができます。両方のアプローチを組み合わせることで、多くの場合、より包括的で関連性の高い結果が得られます。

## ハイブリッド検索とは具体的に何ですか？

**ハイブリッド検索**は、**ベクトル検索**（セマンティック検索）と**キーワード検索**（レキシカル検索）の両方の長所を組み合わせて、クエリに最も関連性の高い結果を見つける検索手法です。

- ベクトル検索は、埋め込み（密なベクトル）を使用してテキストのセマンティックな意味を捉え、正確な単語が一致しなくても意味的に類似した結果を見つけることができます。
- キーワード検索（BM25、TF-IDFなど）は、クエリ内の正確な単語やフレーズの存在と頻度に基づいてドキュメントを照合します。

ハイブリッド検索は、これらのアプローチをブレンドし、通常は両方の手法からの結果をスコアリングまたはランキング付けすることで、精度と再現率の両方を最大化します。

## キーワード検索 vs ベクトル検索 vs ハイブリッド検索

| 特徴                  | キーワード検索                     | ベクトル検索                       | ハイブリッド検索                              |
|-----------------------|------------------------------------|--------------------------------------|--------------------------------------------|
| ベースとなるもの       | レキシカルマッチング (BM25, TF-IDF)     | 埋め込みの類似性 (コサイン類似度, ドット積)   | 両方                                        |
| 強み                  | 正確な一致、関連性            | 文脈の意味                   | バランスの取れた関連性 + 意味                |
| 弱み                  | セマンティックな理解がない           | 正確なキーワードを見逃す                | 計算量がわずかに多い                 |
| マッチ例           | "チキンスープ" = *チキンスープ*     | "チキンスープ" = *鶏肉入りホットブロス* | 文字通りの概念と関連概念の両方       |
| 最適なユースケース       | 法務文書、構造化データ         | チャットボット、Q&A、セマンティック検索       | マルチモーダル、現実世界の曖昧なユーザー クエリ   |

<Note>
アプリケーションでハイブリッド検索が優れている理由

- **再現率の向上**: 純粋なキーワード検索またはベクトル検索で見逃された、より関連性の高い結果を捉えます。
- **バランスの取れた精度**: 正確な一致を優先しつつ、セマンティックに関連する結果も含まれます。
- **曖昧さに対する堅牢性**: スペルバリエーション、同義語、あいまいなユーザーの意図に対応します。
- **両方の長所**: キーワードは必要なときに重要であり、意味は必要なときに重要です。

**レシピ検索、カスタマーサポート、法務開示などの**実世界のアプリケーション**に最適です。**
</Note>

## 例: `pgvector` を使用したハイブリッド検索

```python
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector, SearchType

# データベースURL
db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

# ハイブリッドベクトルDBの初期化
hybrid_db = PgVector(
    table_name="recipes",
    db_url=db_url,
    search_type=SearchType.hybrid  # ハイブリッド検索
)

# ハイブリッド検索を使用してPDFナレッジベースをロード
knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=hybrid_db,
)

# データをDBにロード（初回セットアップ）
knowledge_base.load(recreate=True, upsert=True)

# ハイブリッド検索クエリの実行
results = hybrid_db.search("chicken coconut soup", limit=5)
print("Hybrid Search Results:", results)
```
