**CombinedKnowledgeBase** は、複数のナレッジベースを1つに統合し、アプリが複数のソースを使用して情報を必要とする場合に使用されます。

## 使用方法

<Note>

この例では、ローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```shell
pip install pypdf bs4
```

```python knowledge_base.py
from agno.knowledge.combined import CombinedKnowledgeBase
from agno.vectordb.pgvector import PgVector
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.knowledge.website import WebsiteKnowledgeBase
from agno.knowledge.pdf import PDFKnowledgeBase
from agno.reader.pdf import PDFReader # PDFReader をインポート


url_pdf_knowledge_base = PDFUrlKnowledgeBase(
    urls=["pdf_url"],
    # テーブル名: ai.pdf_documents
    vector_db=PgVector(
        table_name="pdf_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)

website_knowledge_base = WebsiteKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    # シードURLからたどるリンクの数
    max_links=10,
    # テーブル名: ai.website_documents
    vector_db=PgVector(
        table_name="website_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)

local_pdf_knowledge_base = PDFKnowledgeBase(
    path="data/pdfs",
    # テーブル名: ai.pdf_documents
    vector_db=PgVector(
        table_name="pdf_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
    reader=PDFReader(chunk=True),
)

knowledge_base = CombinedKnowledgeBase(
    sources=[
        url_pdf_knowledge_base,
        website_knowledge_base,
        local_pdf_knowledge_base,
    ],
    vector_db=PgVector(
        # テーブル名: ai.combined_documents
        table_name="combined_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`knowledge_base` をエージェントで使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("ナレッジベースについて何か質問してください")
```

## パラメータ

| パラメータ | タイプ                   | デフォルト | 説明                 |
| --------- | ---------------------- | ------- | -------------------- |
| `sources` | `List[AgentKnowledge]` | `[]`    | ナレッジベースのリスト。 |

`CombinedKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
