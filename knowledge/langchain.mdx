**LangchainKnowledgeBase** を使用すると、LangChain リトリーバーまたはベクトルストアをナレッジベースとして利用できます。

## 使用方法

```shell
pip install langchain
```

```python langchain_kb.py
from agno.agent import Agent
from agno.knowledge.langchain import LangChainKnowledgeBase

from langchain.embeddings import OpenAIEmbeddings
from langchain.document_loaders import TextLoader
from langchain.text_splitter import CharacterTextSplitter
from langchain.vectorstores import Chroma

chroma_db_dir = "./chroma_db"


def load_vector_store():
    state_of_the_union = ws_settings.ws_root.joinpath("data/demo/state_of_the_union.txt")
    # -*- ドキュメントをロード
    raw_documents = TextLoader(str(state_of_the_union)).load()
    # -*- チャンクに分割
    text_splitter = CharacterTextSplitter(chunk_size=1000, chunk_overlap=0)
    documents = text_splitter.split_documents(raw_documents)
    # -*- 各チャンクを埋め込み、ベクトルストアにロード
    Chroma.from_documents(documents, OpenAIEmbeddings(), persist_directory=str(chroma_db_dir))


# -*- ベクトルストアを取得
db = Chroma(embedding_function=OpenAIEmbeddings(), persist_directory=str(chroma_db_dir))
# -*- ベクトルストアからリトリーバーを作成
retriever = db.as_retriever()

# -*- ベクトルストアからナレッジベースを作成
knowledge_base = LangChainKnowledgeBase(retriever=retriever)

agent = Agent(knowledge_base=knowledge_base, add_references_to_prompt=True)
conv.print_response("What did the president say about technology?")
```

## パラメータ

| パラメータ       | タイプ                 | デフォルト | 説明                                                               |
| --------------- | -------------------- | ------- | ------------------------------------------------------------------------- |
| `loader`        | `Optional[Callable]` | `None`  | LangChain ローダー。                                                         |
| `vectorstore`   | `Optional[Any]`      | `None`  | リトリーバーの作成に使用される LangChain ベクトルストア。                        |
| `search_kwargs` | `Optional[dict]`     | `None`  | Langchain ベクトルストアを使用してリトリーバーを作成する際の検索キーワード引数。 |
| `retriever`     | `Optional[Any]`      | `None`  | LangChain リトリーバー。                                                      |

`LangChainKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
