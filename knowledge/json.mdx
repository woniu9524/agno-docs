**JSONKnowledgeBase**は、**ローカルJSON**ファイルを読み込み、それらをベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例ではローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```python knowledge_base.py
from agno.knowledge.json import JSONKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = JSONKnowledgeBase(
    path="data/json",
    # テーブル名: ai.json_documents
    vector_db=PgVector(
        table_name="json_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`Agent`で`knowledge_base`を使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("Ask me about something from the knowledge base")
```

#### JSONKnowledgeBaseは非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例ではローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio
from pathlib import Path

from agno.agent import Agent
from agno.knowledge.json import JSONKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "json-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

knowledge_base = JSONKnowledgeBase(
    path=Path("tmp/docs"),
    vector_db=vector_db,
    num_documents=5,  # 検索時に返されるドキュメント数
)

# Agentをknowledge_baseで初期化します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # Agentを作成して使用します
    asyncio.run(
        agent.aprint_response(
            "Ask anything from the json knowledge base", markdown=True
        )
    )
```

## パラメータ

| パラメータ | タイプ               | デフォルト   | 説明                                                                                             |
| --------- | ------------------ | -------------- | ------------------------------------------------------------------------------------------------ |
| `path`    | `Union[str, Path]` | -              | `JSON`ファイルへのパス。<br/>単一のJSONファイルまたはJSONファイルのディレクトリを指すことができます。 |
| `reader`  | `JSONReader`       | `JSONReader()` | ベクトルデータベース用の`JSON`ファイルを`Documents`に変換する`JSONReader`。                               |

`JSONKnowledgeBase`は[AgentKnowledge](/reference/knowledge/base)クラスのサブクラスであり、同じパラメータにアクセスできます。
