**PDFKnowledgeBase** は、**ローカルPDF**ファイルを読み込み、それらをベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例ではローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```shell
pip install pypdf
```

```python knowledge_base.py
from agno.knowledge.pdf import PDFKnowledgeBase, PDFReader
from agno.vectordb.pgvector import PgVector

pdf_knowledge_base = PDFKnowledgeBase(
    path="data/pdfs",
    # テーブル名: ai.pdf_documents
    vector_db=PgVector(
        table_name="pdf_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
    reader=PDFReader(chunk=True),
)
```

次に、`pdf_knowledge_base` をエージェントで使用します。

```python agent.py
from agno.agent import Agent

agent = Agent(
    knowledge=pdf_knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("ナレッジベースについて何か質問してください")
```

#### PDFKnowledgeBase は非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例ではローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.pdf import PDFKnowledgeBase, PDFReader
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "pdf-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# data/pdfs ディレクトリからPDFを使用してナレッジベースを作成します
knowledge_base = PDFKnowledgeBase(
    path="data/pdf",
    vector_db=vector_db,
    reader=PDFReader(chunk=True),
)

# ナレッジベースを使用してエージェントを作成します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # エージェントを作成して使用します
    asyncio.run(agent.aprint_response("タイカレーの作り方を教えてください", markdown=True))
```

## パラメータ

| パラメータ | タイプ                               | デフォルト       | 説明                                                                                             |
| --------- | ---------------------------------- | ------------- | ---------------------------------------------------------------------------------------------------- |
| `path`    | `Union[str, Path]`                 | -             | `PDF`ファイルへのパス。単一のPDFファイルまたはPDFファイルのディレクトリを指定できます。                     |
| `reader`  | `Union[PDFReader, PDFImageReader]` | `PDFReader()` | `PDF`をベクトルデータベース用の`Document`に変換する`PDFReader`または`PDFImageReader`。 |

`PDFKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
