**LightRAGKnowledgeBase** は、[LightRAG Server](https://github.com/HKUDS/LightRAG) と統合されます。これは、グラフ構造を使用してドキュメント検索と知識クエリ機能を強化する、シンプルで高速な検索拡張生成システムです。

## 使用方法


```python knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.light_rag import LightRagKnowledgeBase, lightrag_retriever
from agno.models.anthropic import Claude

# ナレッジベースを作成し、URL からドキュメントをロードします
knowledge_base = LightRagKnowledgeBase(
    lightrag_server_url="http://localhost:9621",
    path="tmp/",  # ローカルディレクトリからドキュメントをロードします
    urls=["https://docs.agno.com/introduction/agents.md"],  # URL からドキュメントをロードします
)

# ローカルディレクトリと URL からのドキュメントでナレッジベースをロードします
asyncio.run(knowledge_base.load())

# テキストでナレッジベースをロードします
asyncio.run(
    knowledge_base.load_text(text="Agno はエージェントを構築するための最高のフレームワークです")
)

```

次に、`lightrag_knowledge_base` を Agent と共に使用します。

```python agent.py
# ナレッジベースとリトリーバーでエージェントを作成します
agent = Agent(
    model=Claude(id="claude-3-7-sonnet-latest"),
    # Agent に `knowledge` が提供されている場合、エージェント RAG はデフォルトで有効になります。
    knowledge=knowledge_base,
    retriever=lightrag_retriever,
    # `search_knowledge=True` は、エージェントにオンデマンドで検索する機能を与えます
    # `search_knowledge` はデフォルトで True です
    search_knowledge=True,
    instructions=[
        "応答にソースを含めてください。",
        "質問に答える前に、必ず知識を検索してください。",
        "ナレッジベースを検索するには、async_search メソッドを使用してください。",
    ],
    markdown=True,
)

asyncio.run(agent.aprint_response("Agno エージェントとは何ですか?"))
```

## パラメータ

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|------|---------|-------------|
| `lightrag_server_url` | `str` | - | LightRAG サーバーへの URL。 |
| `path` | `Union[str, Path]` | - | ドキュメントへのパス。単一ファイルまたはファイルのディレクトリを指すことができます。 |
| `url` | `str` | - | 読み取るウェブサイトの URL。 |
