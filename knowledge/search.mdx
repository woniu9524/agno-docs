title: Agentic Search
---

エージェントを使用して情報を反復的に検索することは**Agentic Search**と呼ばれ、**検索、推論、応答**のプロセスは**Agentic RAG**として知られています。

モデルはクエリを解釈し、関連するキーワードを生成して、その知識を検索します。

<Tip>
エージェントの応答は、その検索の質に依存します。**より良い検索 = より良い応答**
</Tip>

セマンティック検索、キーワード検索、またはハイブリッド検索を使用できます。クラス最高のAgentic Searchには、**再ランキング付きハイブリッド検索**の使用をお勧めします。

エージェントが必要な情報を検索するため、このパターンは**Agentic Search**と呼ばれ、エージェントビルダーの間で非常に人気が高まっています。

<Check>
Agentic Searchが実際にどのように機能するかを確認するために、いくつかの例を構築しましょう。
</Check>

## Agentic RAG

エージェントにナレッジベースを追加すると、バックグラウンドでモデルにそのナレッジベースを検索して必要な情報を取得するツールが提供されます。

モデルはキーワードのセットを生成し、`search_knowledge_base()` ツールを呼び出して関連情報または少数ショットの例を取得します。

これは、ハイブリッド検索と再ランキングを使用した動作例です。

<Tip>
必要ない場合は、再ランキングステップを削除してもかまいません。
</Tip>

```python agentic_rag.py
"""このクックブックは、ハイブリッド検索と再ランキングを使用したAgentic RAGの実装方法を示しています。
1. 依存関係をインストールするには、`pip install agno anthropic cohere lancedb tantivy sqlalchemy` を実行します。
2. ANTHROPIC_API_KEY と CO_API_KEY をエクスポートします。
3. エージェントを実行するには、`python cookbook/agent_concepts/agentic_search/agentic_rag.py` を実行します。
"""

from agno.agent import Agent
from agno.embedder.cohere import CohereEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.anthropic import Claude
from agno.reranker.cohere import CohereReranker
from agno.vectordb.lancedb import LanceDb, SearchType

# URLからドキュメントをロードしたナレッジベースを作成します
knowledge_base = UrlKnowledge(
    urls=["https://docs.agno.com/introduction/agents.md"],
    # LanceDBをベクトルデータベースとして使用し、埋め込みを `agno_docs` テーブルに保存します
    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        embedder=CohereEmbedder(id="embed-v4.0"),
        reranker=CohereReranker(model="rerank-v3.5"),
    ),
)

agent = Agent(
    model=Claude(id="claude-3-7-sonnet-latest"),
    # `knowledge` がエージェントに提供されると、Agentic RAGはデフォルトで有効になります。
    knowledge=knowledge_base,
    # search_knowledge=True は、エージェントにオンデマンドで検索する機能を与えます
    # search_knowledge はデフォルトでTrueです
    search_knowledge=True,
    instructions=[
        "Include sources in your response.",
        "Always search your knowledge before answering the question.",
        "Only include the output in your response. No other text.",
    ],
    markdown=True,
)

if __name__ == "__main__":
    # ナレッジベースをロードします。初回実行後にコメントアウトしてください。
    # knowledge_base.load(recreate=True)
    agent.print_response("What are Agents?", stream=True)
```

## Agentic RAG with Reasoning

推論能力をエージェントに与えることで、エージェントの検索機能をさらに向上させることができます。

推論を追加することで、エージェントはまず検索する内容について「考え」、次に検索結果を「分析」します。

これは、検索結果の品質を向上させるために推論を使用するAgentic RAGエージェントの例です。

```python agentic_rag_reasoning.py
"""このクックブックは、推論を使用したAgentic RAGの実装方法を示しています。
1. 依存関係をインストールするには、`pip install agno anthropic cohere lancedb tantivy sqlalchemy` を実行します。
2. ANTHROPIC_API_KEY と CO_API_KEY をエクスポートします。
3. エージェントを実行するには、`python cookbook/agent_concepts/agentic_search/agentic_rag_with_reasoning.py` を実行します。
"""

from agno.agent import Agent
from agno.embedder.cohere import CohereEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.anthropic import Claude
from agno.reranker.cohere import CohereReranker
from agno.tools.reasoning import ReasoningTools
from agno.vectordb.lancedb import LanceDb, SearchType

# URLからドキュメントをロードしたナレッジベースを作成します
knowledge_base = UrlKnowledge(
    urls=["https://docs.agno.com/introduction/agents.md"],
    # LanceDBをベクトルデータベースとして使用し、埋め込みを `agno_docs` テーブルに保存します
    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        embedder=CohereEmbedder(id="embed-v4.0"),
        reranker=CohereReranker(model="rerank-v3.5"),
    ),
)

agent = Agent(
    model=Claude(id="claude-3-7-sonnet-latest"),
    # `knowledge` がエージェントに提供されると、Agentic RAGはデフォルトで有効になります。
    knowledge=knowledge_base,
    # search_knowledge=True は、エージェントにオンデマンドで検索する機能を与えます
    # search_knowledge はデフォルトでTrueです
    search_knowledge=True,
    tools=[ReasoningTools(add_instructions=True)],
    instructions=[
        "Include sources in your response.",
        "Always search your knowledge before answering the question.",
        "Only include the output in your response. No other text.",
    ],
    markdown=True,
)

if __name__ == "__main__":
    # ナレッジベースをロードします。初回実行後にコメントアウトしてください。
    # knowledge_base.load(recreate=True)
    agent.print_response(
        "What are Agents?",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
