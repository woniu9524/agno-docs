**CSVKnowledgeBase** は、**ローカルCSV**ファイルを読み込み、ベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```python
from agno.knowledge.csv import CSVKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = CSVKnowledgeBase(
    path="data/csv",
    # テーブル名: ai.csv_documents
    vector_db=PgVector(
        table_name="csv_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`Agent` で `knowledge_base` を使用します。

```python
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("Ask me about something from the knowledge base")
```

#### CSVKnowledgeBase は非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例では、ローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio
from pathlib import Path

from agno.agent import Agent
from agno.knowledge.csv import CSVKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "csv-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")


knowledge_base = CSVKnowledgeBase(
    path=Path("data/csv"),
    vector_db=vector_db,
    num_documents=5,  # 検索時に返されるドキュメント数
)

# Agent を knowledge_base で初期化します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # Agent を作成して使用します
    asyncio.run(agent.aprint_response("What is the csv file about", markdown=True))
```

## パラメータ

| パラメータ | タイプ               | デフォルト    | 説明                                                                                             |
| --------- | ------------------ | ------------- | ---------------------------------------------------------------------------------------------- |
| `path`    | `Union[str, Path]` | -             | CSVファイルへのパス                                                                            |
| `reader`  | `CSVReader`        | `CSVReader()` | CSVファイルを読み込み、ベクトルデータベース用の `Documents` に変換する `CSVReader` |

`CSVKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
