**WebsiteKnowledgeBase** はウェブサイトを読み込み、ベクトル埋め込みに変換して `vector_db` にロードします。

## Usage

<Note>

この例では、ローカルの PgVector データベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```shell
pip install bs4
```

```python knowledge_base.py
from agno.knowledge.website import WebsiteKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = WebsiteKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    # シードURLからたどるリンクの数
    max_links=10,
    # テーブル名: ai.website_documents
    vector_db=PgVector(
        table_name="website_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`Agent` で `knowledge_base` を使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("Ask me about something from the knowledge base")
```

#### WebsiteKnowledgeBase は非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例では、ローカルの Qdrant データベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio

import asyncio

from agno.agent import Agent
from agno.knowledge.website import WebsiteKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "website-content"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")


# シードURLでナレッジベースを作成
knowledge_base = WebsiteKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    # シードURLからたどるリンクの数
    max_links=5,
    # テーブル名: ai.website_documents
    vector_db=vector_db,
)

# ナレッジベースでエージェントを作成
agent = Agent(knowledge=knowledge_base, search_knowledge=True, debug_mode=True)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # エージェントを作成して使用
    asyncio.run(agent.aprint_response("How does agno work?", markdown=True))
```


## Params

| Parameter   | Type                      | Default | Description                                                                                       |
| ----------- | ------------------------- | ------- | ------------------------------------------------------------------------------------------------- |
| `urls`      | `List[str]`               | `[]`    | 読み込むURL                                                                                       |
| `reader`    | `Optional[WebsiteReader]` | `None`  | URLを読み込み、`Documents` に変換してベクトルデータベースにロードする `WebsiteReader`。                               |
| `max_depth` | `int`                     | `3`     | クロールする最大深度。                                                                            |
| `max_links` | `int`                     | `10`    | クロールするリンクの数。                                                                          |

`WebsiteKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
