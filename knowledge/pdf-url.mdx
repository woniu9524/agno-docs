**PDFUrlKnowledgeBase** は、**URL から PDF を読み込み**、ベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルの PgVector データベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```shell
pip install pypdf
```

```python knowledge_base.py
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = PDFUrlKnowledgeBase(
    urls=["pdf_url"],
    # テーブル名: ai.pdf_documents
    vector_db=PgVector(
        table_name="pdf_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`knowledge_base` を Agent と共に使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("Ask me about something from the knowledge base")
```

#### PDFUrlKnowledgeBase は非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例では、ローカルの Qdrant データベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase, PDFUrlReader
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "pdf-url-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# data/pdfs ディレクトリから PDF を使用してナレッジベースを作成します
knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=vector_db,
    reader=PDFUrlReader(chunk=True),
)

# ナレッジベースを使用してエージェントを作成します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # エージェントを作成して使用します
    asyncio.run(agent.aprint_response("How to make Thai curry?", markdown=True))
```
## パラメータ

| パラメータ | タイプ           | デフォルト | 説明                                                                         |
| --------- | -------------- | ------- | ----------------------------------------------------------------------------------- |
| `urls`    | `List[str]`    | -       | `PDF` ファイルの URL。                                                               |
| `reader`  | `PDFUrlReader` | -       | ベクトルデータベース用の `PDF` を `Documents` に変換する `PDFUrlReader`。 |

`PDFUrlKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
