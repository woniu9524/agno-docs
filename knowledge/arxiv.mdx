**ArxivKnowledgeBase**は、Arxivの記事を読み込み、ベクトル埋め込みに変換し、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```shell
pip install arxiv
```

```python knowledge_base.py
from agno.knowledge.arxiv import ArxivKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = ArxivKnowledgeBase(
    queries=["Generative AI", "Machine Learning"],
    # テーブル名: ai.arxiv_documents
    vector_db=PgVector(
        table_name="arxiv_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`Agent`で`knowledge_base`を使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("Ask me about something from the knowledge base")
```

#### ArxivKnowledgeBaseは非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例では、ローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.arxiv import ArxivKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "arxiv-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")

# ArXivドキュメントでナレッジベースを作成します
knowledge_base = ArxivKnowledgeBase(
    queries=["Generative AI", "Machine Learning"], vector_db=vector_db
)

# ナレッジベースでエージェントを作成します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # エージェントを作成して使用します
    asyncio.run(
        agent.aprint_response(
            "Ask me about generative ai from the knowledge base", markdown=True
        )
    )
```

## パラメータ

| パラメータ | タイプ          | デフォルト         | 説明                                                                                        |
| --------- | ------------- | --------------- | -------------------------------------------------------------------------------------------------- |
| `queries` | `List[str]`   | `[]`            | 検索クエリ                                                                                  |
| `reader`  | `ArxivReader` | `ArxivReader()` | ベクトルデータベース用の記事を読み込み、`Documents`に変換する`ArxivReader` |

`ArxivKnowledgeBase`は、[AgentKnowledge](/reference/knowledge/base)クラスのサブクラスであり、同じパラメータにアクセスできます。
