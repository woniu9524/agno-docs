**CSVUrlKnowledgeBase** は、**URLからCSVを読み込み**、ベクトル埋め込みに変換して、ベクトルデータベースにロードします。

## 使用方法

<Note>

この例では、ローカルのPgVectorデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/pgvector)

</Note>

```python knowledge_base.py
from agno.knowledge.csv_url import CSVUrlKnowledgeBase
from agno.vectordb.pgvector import PgVector

knowledge_base = CSVUrlKnowledgeBase(
    urls=["csv_url"],
    # テーブル名: ai.csv_documents
    vector_db=PgVector(
        table_name="csv_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
```

次に、`knowledge_base` をエージェントで使用します。

```python agent.py
from agno.agent import Agent
from knowledge_base import knowledge_base

agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.knowledge.load(recreate=False)

agent.print_response("ナレッジベースについて何か質問してください")
```

#### CSVUrlKnowledgeBase は非同期ロードもサポートしています。

```shell
pip install qdrant-client
```
この例では、ローカルのQdrantデータベースを使用しています。[実行されていることを確認してください](https://docs.agno.com/vectordb/qdrant)

```python async_knowledge_base.py
import asyncio

from agno.agent import Agent
from agno.knowledge.csv_url import CSVUrlKnowledgeBase
from agno.vectordb.qdrant import Qdrant

COLLECTION_NAME = "csv-reader"

vector_db = Qdrant(collection=COLLECTION_NAME, url="http://localhost:6333")


knowledge_base = CSVUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/demo_data/IMDB-Movie-Data.csv"],
    vector_db=vector_db,
    num_documents=5,  # 検索時に返されるドキュメント数
)

# ナレッジベースでエージェントを初期化します
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)

if __name__ == "__main__":
    # 初回実行後にコメントアウトしてください
    asyncio.run(knowledge_base.aload(recreate=False))

    # エージェントを作成して使用します
    asyncio.run(
        agent.aprint_response("ここにはどのようなジャンルの映画がありますか？", markdown=True)
    )
```

## パラメータ

| パラメータ | タイプ           | デフォルト          | 説明                                                                                             |
| --------- | -------------- | ---------------- | ------------------------------------------------------------------------------------------------ |
| `urls`    | `List[str]`    | -                | `PDF` ファイルのURL。                                                                            |
| `reader`  | `CSVUrlReader` | `CSVUrlReader()` | URLからCSVファイルを読み込み、ベクトルデータベース用の`Documents`に変換する `CSVUrlReader`。 |

`CSVUrlKnowledgeBase` は [AgentKnowledge](/reference/knowledge/base) クラスのサブクラスであり、同じパラメータにアクセスできます。
