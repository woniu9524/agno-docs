Discord Bot インテグレーションを使用すると、discord.py ライブラリを使用して Discord イベントを処理し、メッセージを送信することで、Discord 経由でエージェントまたはチームを提供できます。

## セットアップ手順

<Snippet file="setup-discord-app.mdx" />

### 使用例

エージェントを作成し、`DiscordClient` でラップして実行します。

```python
from agno.agent import Agent
from agno.app.discord import DiscordClient
from agno.models.openai import OpenAIChat

basic_agent = Agent(
    name="Basic Agent",
    model=OpenAIChat(id="gpt-4o"),
    add_history_to_messages=True,
    num_history_responses=3,
    add_datetime_to_instructions=True,
)

discord_agent = DiscordClient(basic_agent)
if __name__ == "__main__":
    discord_agent.serve()
```

## コアコンポーネント

- `DiscordClient`: discord.py を使用して Agno エージェント/チームを Discord インテグレーション用にラップします。
- `DiscordClient.serve`: 指定されたトークンで Discord ボットクライアントを開始します。

## `DiscordClient` クラス

Agno Discord ボットアプリケーションのエントリポイントです。

### 初期化パラメータ

| パラメータ | タイプ              | デフォルト | 説明                                   |
| --------- | ----------------- | ------- | -------------------------------------- |
| `agent`   | `Optional[Agent]` | `None`  | Agno `Agent` インスタンス。              |
| `team`    | `Optional[Team]`  | `None`  | Agno `Team` インスタンス。               |

*`agent` または `team` のいずれか一方を指定してください。両方を指定しないでください。*

## イベント処理

Discord ボットは、さまざまな Discord イベントを自動的に処理します。

### メッセージイベント
- **説明**: ユーザーからのすべての受信メッセージを処理します。
- **メディアサポート**: 画像、ビデオ、音声ファイル、ドキュメントを処理します。
- **スレッディング**: 会話のために自動的にスレッドを作成します。
- **機能**:
  - 各会話のための自動スレッド作成
  - メディアの処理とエージェントへの転送
  - 1500 文字を超える応答のためのメッセージ分割
  - 推論コンテンツの表示のサポート
  - ユーザー名とメッセージ URL を使用したコンテキストの強化

### サポートされているメディアタイプ
- **画像**: 画像分析のために直接 URL を処理します。
- **ビデオ**: ビデオコンテンツをダウンロードして処理します。
- **音声**: URL ベースの音声処理を行います。
- **ファイル**: ドキュメント添付ファイルをダウンロードして処理します。

## 環境変数

以下の環境変数を設定してください。

```bash
export DISCORD_BOT_TOKEN="your-discord-bot-token"
```

## メッセージ処理

ボットは以下のワークフローでメッセージを処理します。

1. **メッセージ受信**: Discord チャネルからメッセージを受信します。
2. **メディア処理**: 添付されたメディアをダウンロードして処理します。
3. **スレッド管理**: 会話のためにスレッドを作成または既存のスレッドを使用します。
4. **エージェント/チーム実行**: 設定されたエージェントまたはチームにメッセージとメディアを転送します。
5. **応答処理**: 必要に応じて長いメッセージを分割して、応答を Discord に送信します。
6. **推論表示**: 利用可能な場合は、推論コンテンツをイタリック体で表示します。

## 機能

### 自動スレッド作成
- ユーザーの最初のメッセージごとに新しいスレッドを作成します。
- スレッド内で会話コンテキストを維持します。
- フォーマット: `{username}'s thread` を使用します。

### メディアサポート
- **画像**: URL を持つ `Image` オブジェクトとして渡されます。
- **ビデオ**: コンテンツを持つ `Video` オブジェクトとしてダウンロードされ、渡されます。
- **音声**: URL を持つ `Audio` オブジェクトとして渡されます。
- **ファイル**: コンテンツを持つ `File` オブジェクトとしてダウンロードされ、渡されます。

### メッセージフォーマット
- 長いメッセージ (>1500 文字) は自動的に分割されます。
- 推論コンテンツはイタリック体で表示されます。
- 分割メッセージのバッチ番号: `[1/3] message content`

## インテグレーションのテスト

1. Discord ボットトークンを設定します: `export DISCORD_BOT_TOKEN="your-token"`
2. アプリケーションを実行します: `python your_discord_bot.py`
3. ボットを Discord サーバーに招待します。
4. ボットがアクセスできる任意のチャネルにメッセージを送信します。
5. ボットは自動的にスレッドを作成して応答します。
