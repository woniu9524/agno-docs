Discord Bot 集成允许您通过 Discord 提供 Agent 或 Team，使用 discord.py 库来处理 Discord 事件和发送消息。

## 设置步骤

<Snippet file="setup-discord-app.mdx" />

### 示例用法

创建一个 Agent，用 `DiscordClient` 包装它，然后运行它：

```python
from agno.agent import Agent
from agno.app.discord import DiscordClient
from agno.models.openai import OpenAIChat

basic_agent = Agent(
    name="Basic Agent",
    model=OpenAIChat(id="gpt-4o"),
    add_history_to_messages=True,
    num_history_responses=3,
    add_datetime_to_instructions=True,
)

discord_agent = DiscordClient(basic_agent)
if __name__ == "__main__":
    discord_agent.serve()
```

## 核心组件

- `DiscordClient`: 使用 discord.py 包装 Agno Agent/Team 以实现 Discord 集成。
- `DiscordClient.serve`: 使用提供的 token 启动 Discord Bot 客户端。

## `DiscordClient` 类

Agno Discord Bot 应用程序的主入口点。

### 初始化参数

| 参数 | 类型 | 默认值 | 描述 |
| --------- | -------- | ------- | -------- |
| `agent` | `Optional[Agent]` | `None` | Agno `Agent` 实例。 |
| `team` | `Optional[Team]` | `None` | Agno `Team` 实例。 |

*请提供 `agent` 或 `team`，但不要同时提供。*

## 事件处理

Discord Bot 会自动处理各种 Discord 事件：

### 消息事件
- **描述**: 处理来自用户的全部入站消息
- **媒体支持**: 处理图片、视频、音频文件和文档
- **线程**: 自动为对话创建线程
- **功能**:
  - 为每次对话自动创建线程
  - 媒体处理和转发给 Agent
  - 对于长度超过 1500 个字符的响应进行消息拆分
  - 支持推理内容显示
  - 使用用户名和消息 URL 进行上下文丰富

### 支持的媒体类型
- **图片**: 直接 URL 处理以进行图像分析
- **视频**: 下载并处理视频内容
- **音频**: 基于 URL 的音频处理
- **文件**: 下载并处理文档附件

## 环境变量

请确保设置了以下环境变量：

```bash
export DISCORD_BOT_TOKEN="your-discord-bot-token"
```

## 消息处理

Bot 按照以下工作流程处理消息：

1. **消息接收**: 从 Discord 频道接收消息
2. **媒体处理**: 下载并处理任何附加的媒体
3. **线程管理**: 创建或使用现有线程进行对话
4. **Agent/Team 执行**: 将消息和媒体转发给配置的 Agent 或 Team
5. **响应处理**: 将响应发送回 Discord，并在必要时拆分长消息
6. **推理显示**: 如果可用，以斜体显示推理内容

## 功能

### 自动线程创建
- 为用户的每条第一条消息创建一个新线程
- 在线程中维护对话上下文
- 使用格式：`{username}'s thread`

### 媒体支持
- **图片**: 作为带有 URL 的 `Image` 对象传递
- **视频**: 下载并作为带有内容的 `Video` 对象传递
- **音频**: 作为带有 URL 的 `Audio` 对象传递
- **文件**: 下载并作为带有内容的 `File` 对象传递

### 消息格式化
- 长消息（>1500 个字符）会自动拆分
- 推理内容以斜体显示
- 拆分消息的批次编号：`[1/3] message content`

## 测试集成

1. 设置您的 Discord Bot token：`export DISCORD_BOT_TOKEN="your-token"`
2. 运行您的应用程序：`python your_discord_bot.py`
3. 将 Bot 邀请到您的 Discord 服务器
4. 在 Bot 具有访问权限的任何频道中发送消息
5. Bot 将自动创建一个线程并进行响应
