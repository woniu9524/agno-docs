Whatsapp App は、WhatsApp を介してやり取りするエージェントまたはチームにサービスを提供するために使用されます。FastAPI サーバーを使用して Webhook イベントを処理し、メッセージを送信します。

<Snippet file="setup-whatsapp-app.mdx" />

### 使用例

エージェントを作成し、`WhatsappAPI` でラップして提供します。

```python
from agno.agent import Agent
from agno.app.whatsapp.app import WhatsappAPI
from agno.models.openai import OpenAIChat
from agno.tools.openai import OpenAITools

image_agent = Agent(
    model=OpenAIChat(id="gpt-4o"), # OPENAI_API_KEY が設定されていることを確認してください
    tools=[OpenAITools(image_model="gpt-image-1")],
    markdown=True,
    show_tool_calls=True,
    debug_mode=True,
    add_history_to_messages=True,
)

# デフォルトで非同期ルーター (use_async=True)
whatsapp_app = WhatsappAPI(
    agent=image_agent,
    name="Image Generation Tools",
    app_id="image_generation_tools",
    description="A tool that generates images using the OpenAI API.",
)

app = whatsapp_app.get_app()

if __name__ == "__main__":
    whatsapp_app.serve(app="image_generation_tools:app", port=8000, reload=True)
```

**実行するには:**
1.  OpenAI モデルを使用する場合は、`OPENAI_API_KEY` 環境変数が設定されていることを確認してください。
2.  API は実行されます (例: `http://localhost:8000`) が、対話は主に設定された Webhook を介して WhatsApp で行われます。
3.  API ドキュメント (設定で有効になっている場合) は `http://localhost:8000/docs` にある可能性があります。



## コアコンポーネント

- `WhatsappAPI`: Agno エージェント/チームを FastAPI を介した WhatsApp 統合のためにラップします。
- `WhatsappAPI.serve`: WhatsApp 用に構成された Uvicorn を使用して FastAPI アプリを提供します。

## `WhatsappAPI` クラス

Agno WhatsApp アプリケーションのエントリポイントです。

### 初期化パラメータ

| パラメータ  | タイプ                      | デフォルト | 説明                                           |
| ---------- | ------------------------- | ------- | ----------------------------------------------------- |
| `agent`    | `Optional[Agent]`         | `None`  | Agno `Agent` インスタンス。                                |
| `team`     | `Optional[Team]`          | `None`  | Agno `Team` インスタンス。                                 |
| `settings` | `Optional[APIAppSettings]`| `None`  | API 設定。`None` の場合はデフォルト値が使用されます。                |
| `api_app`  | `Optional[FastAPI]`       | `None`  | 既存の FastAPI アプリ。`None` の場合は新規作成されます。 |
| `router`   | `Optional[APIRouter]`     | `None`  | 既存の APIRouter。`None` の場合は新規作成されます。        |
| `app_id`    | `Optional[str]`           | `None`  | アプリの識別子 (設定されていない場合は自動生成されます)。                         |
| `name`    | `Optional[str]`           | `None`  | アプリの名前。                         |
| `description`    | `Optional[str]`           | `None`  | アプリの説明。                         |

*`agent` または `team` のいずれかを提供してください。両方は不要です。*

### 主要なメソッド

| メソッド | パラメータ | 戻り値の型 | 説明 |
|--------|------------|-------------|-------------|
| `get_app` | `use_async: bool = True`<br/>`prefix: str = ""` | `FastAPI` | 設定された FastAPI アプリを返します。プレフィックス、エラーハンドラを設定し、WhatsApp ルーターを含めます。デフォルトでは非同期ルーターが使用されます。 |

## エンドポイント

エンドポイントは `prefix` (デフォルトはルートレベル: `""`) でアクセスできます。

### 1. `GET /webhook`
    *   **説明**: WhatsApp Webhook (チャレンジ) を検証します。
    *   **応答**:
        *   `200 OK`: トークンが一致する場合、`hub.challenge` を返します。
        *   `403 Forbidden`: トークン不一致または無効なモード。
        *   `500 Internal Server Error`: `WHATSAPP_VERIFY_TOKEN` が設定されていません。

### 2. `POST /webhook`
    *   **説明**: 着信 WhatsApp メッセージおよびイベントを受信します。
    *   **処理**:
        *   署名を検証します (`APP_ENV="production"` かつ `WHATSAPP_APP_SECRET` が設定されている場合)。
        *   `agent.arun()` または `team.arun()` を介してメッセージ (テキスト、画像、ビデオ、オーディオ、ドキュメント) を処理します。
        *   WhatsApp を介して返信を送信します。
    *   **応答**:
        *   `200 OK`: `{"status": "processing"}` または `{"status": "ignored"}`。
        *   `403 Forbidden`: 無効な署名。
        *   `500 Internal Server Error`: その他の処理エラー。

### パラメータ

| パラメータ | タイプ                | デフォルト     | 説明                                        |
| --------- | ------------------- | ----------- | -------------------------------------------------- |
| `app`     | `Union[str, FastAPI]` | `N/A`       | FastAPI アプリインスタンスまたはインポート文字列 (必須)。 |
| `host`    | `str`               | `"localhost"` | バインドするホスト。                                      |
| `port`    | `int`               | `7777`      | バインドするポート。                                      |
| `reload`  | `bool`              | `False`     | 開発のために自動リロードを有効にします。                |
