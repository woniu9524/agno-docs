FastAPI アプリは、REST API インターフェイスを持つ FastAPI サーバーを使用して、エージェントまたはチームを提供するために使用されます。

### 使用例

エージェントを作成し、`FastAPIApp` でラップして提供します。

```python
from agno.agent import Agent
from agno.app.fastapi.app import FastAPIApp
from agno.models.openai import OpenAIChat

basic_agent = Agent(
    name="Basic Agent",
    model=OpenAIChat(id="gpt-4o"), # OPENAI_API_KEY を設定してください
    add_history_to_messages=True,
    num_history_responses=3,
    add_datetime_to_instructions=True,
    markdown=True,
)

# デフォルトで非同期ルーター (use_async=True)
fastapi_app = FastAPIApp(
    agent=basic_agent,
    name="Basic Agent",
    app_id="basic_agent",
    description="A basic agent that can answer questions and help with tasks.",
)

app = fastapi_app.get_app()

# 同期ルーターの場合:
# app = fastapi_app.get_app(use_async=False)

if __name__ == "__main__":
    fastapi_app.serve(app="basic:app", port=8001, reload=True)

```

**実行するには:**

1.  `OPENAI_API_KEY` 環境変数を設定します。
2.  API は `http://localhost:8001`、ドキュメントは `http://localhost:8001/docs` で利用可能です。

`http://localhost:8001/v1/run` に `POST` リクエストを送信します。
```json
{
  "message": "Hello Basic Agent, tell me a fun fact!",
  "stream": false
}
```

## コアコンポーネント

- `FastAPIApp`: Agno エージェント/チームを FastAPI 用にラップします。
- `FastAPIApp.serve`: Uvicorn を使用して FastAPI アプリを提供します。

`FastAPIApp` はルーティングのためのヘルパー関数を使用します。

## `FastAPIApp` クラス

Agno FastAPI アプリのメインエントリポイントです。

### 初期化パラメータ

| パラメータ  | タイプ                      | デフォルト | 説明                                           |
| ---------- | ------------------------- | ------- | ----------------------------------------------------- |
| `agents`    | `Optional[List[Agent]]`         | `None`  | Agno `Agent` インスタンスのリスト。                |
| `teams`     | `Optional[List[Team]]`          | `None`  | Agno `Team` インスタンスのリスト。                 |
| `workflows`     | `Optional[List[Team]]`          | `None`  | Agno `Workflow` インスタンスのリスト。                 |
| `settings` | `Optional[APIAppSettings]`| `None`  | API 設定。`None` の場合はデフォルト値が使用されます。                |
| `api_app`  | `Optional[FastAPI]`       | `None`  | 既存の FastAPI アプリ。`None` の場合は新しいものが作成されます。|
| `router`   | `Optional[APIRouter]`     | `None`  | 既存の APIRouter。`None` の場合は新しいものが作成されます。        |
| `app_id`    | `Optional[str]`           | `None`  | アプリの識別子 (`None` の場合は自動生成されます)。                         |
| `name`    | `Optional[str]`           | `None`  | アプリの名前。                         |
| `description`    | `Optional[str]`           | `None`  | アプリの説明。                         |

*`agent` または `team` のいずれかを提供してください。両方は提供しないでください。*

### 主要なメソッド

| メソッド | パラメータ | 戻り値の型 | 説明 |
|---|---|---|---|
| `get_app` | `use_async: bool = True`<br/>`prefix: str = "/v1"` | `FastAPI` | 設定済みの FastAPI アプリを返します (デフォルトは非同期)。プレフィックス、エラーハンドラ、CORS、ドキュメントを設定します。 |

## エンドポイント

エンドポイントは指定された `prefix` (デフォルト `/v1`) で利用可能です。

### 1. `POST /run`
    *   **説明**: エージェント/チームと対話します (`agent.run()`/`arun()` または `team.run()`/`arun()` を使用)。
    *   **リクエストフォームパラメータ**:
        | パラメータ    | タイプ                        | デフォルト | 説明                               |
        | ------------ | --------------------------- | ------- | ----------------------------------------- |
        | `message`    | `str`                       | `...`   | 入力メッセージ (必須)。                 |
        | `stream`     | `bool`                      | `True` (同期)、`False` (非同期デフォルト) | レスポンスをストリームします。 |
        | `monitor`    | `bool`                      | `False` | モニタリングを有効にします。                        |
        | `session_id` | `Optional[str]`             | `None`  | 会話の継続性のためのセッション ID。   |
        | `user_id`    | `Optional[str]`             | `None`  | ユーザー ID。                                  |
        | `files`      | `Optional[List[UploadFile]]`| `None`  | アップロードするファイル。                          |
    *   **レスポンス**:
        *   `stream=True`: JSON `RunResponse`/`TeamRunResponse` イベントを含む `StreamingResponse` (`text/event-stream`)。
        *   `stream=False`: JSON `RunResponse`/`TeamRunResponse` 辞書。

### パラメータ

| パラメータ | タイプ                | デフォルト     | 説明                                        |
| --------- | ------------------- | ----------- | -------------------------------------------------- |
| `app`     | `Union[str, FastAPI]` | `N/A`       | FastAPI アプリインスタンスまたはインポート文字列 (必須)。 |
| `host`    | `str`               | `"localhost"` | バインドするホスト。                                      |
| `port`    | `int`               | `7777`      | バインドするポート。                                      |
| `reload`  | `bool`              | `False`     | 開発のために自動リロードを有効にします。                |
