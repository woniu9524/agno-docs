Playground アプリは、FastAPI サーバーといくつかのエンドポイントを使用して、エージェント、チーム、ワークフローを提供するために使用されます。これにより、[Agno Playground](/introduction/playground) 上の `Agents`、`Workflows`、`Teams` を管理および操作できます。

### 使用例

エージェントを作成し、`Playground` で提供します。

```python
from agno.agent import Agent
from agno.memory.agent import AgentMemory
from agno.memory.db.postgres import PgMemoryDb
from agno.models.openai import OpenAIChat
from agno.playground import Playground
from agno.storage.postgres import PostgresStorage

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

basic_agent = Agent(
    name="Basic Agent",
    model=OpenAIChat(id="gpt-4o"), # OPENAI_API_KEY を設定してください
    memory=AgentMemory(
        db=PgMemoryDb(
            table_name="agent_memory",
            db_url=db_url,
        ),
        create_user_memories=True,
        update_user_memories_after_run=True,
        create_session_summary=True,
        update_session_summary_after_run=True,
    ),
    storage=PostgresStorage(
        table_name="agent_sessions", db_url=db_url, auto_upgrade_schema=True
    ),
    add_history_to_messages=True,
    num_history_responses=3,
    add_datetime_to_instructions=True,
    markdown=True,
)

playground = Playground(
    agents=[
        basic_agent,
    ],
    name="Basic Agent",
    description="A playground for basic agent",
    app_id="basic-agent",
)
app = playground.get_app()

if __name__ == "__main__":
    playground.serve(app="basic:app", reload=True)
```

**実行するには:**

1.  PostgreSQL サーバーが実行されており、`db_url` 経由でアクセス可能であることを確認してください。
2.  `OPENAI_API_KEY` 環境変数を設定してください。
3.  Playground UI は `http://localhost:7777` で利用可能になります。API ドキュメント（設定で有効になっている場合）は通常 `http://localhost:7777/docs` にあります。
4.  [Agent Playground](/introduction/playground) で Playground を使用してください。

## コアコンポーネント

- `Playground`: Agno エージェント、チーム、またはワークフローを API でラップします。
- `Playground.serve`: Uvicorn を使用して Playground FastAPI アプリを提供します。

`Playground` クラスは、Agno Playground アプリケーションを作成するためのメインのエントリーポイントです。これにより、[Agent Playground](/introduction/playground) または [Agent UI](/agent-ui/introduction) を使用して、エージェント、チーム、ワークフローを Web インターフェイスを通じて簡単に公開できます。

## `Playground` クラス

### 初期化パラメータ

| パラメータ   | タイプ                            | デフォルト | 説明                                                                 |
| :----------- | :-------------------------------- | :-------- | :------------------------------------------------------------------- |
| `agents`    | `Optional[List[Agent]]`         | `None`  | Agno `Agent` インスタンスのリスト。                                             |
| `teams`     | `Optional[List[Team]]`          | `None`  | Agno `Team` インスタンスのリスト。                                              |
| `workflows` | `Optional[List[Workflow]]`      | `None`  | Agno `Workflow` インスタンスのリスト。                                          |
| `settings`  | `Optional[PlaygroundSettings]`  | `None`  | Playground の設定。`None` の場合はデフォルト値が使用されます。                               |
| `api_app`   | `Optional[FastAPI]`             | `None`  | 既存の FastAPI アプリ。`None` の場合は新しいものが作成されます。                               |
| `router`    | `Optional[APIRouter]`           | `None`  | 既存の APIRouter。`None` の場合は新しいものが作成されます。                               |
| `app_id`    | `Optional[str]`           | `None`  | アプリケーション識別子（設定されていない場合は自動生成されます）。                         |
| `name`    | `Optional[str]`           | `None`  | アプリケーションの名前。                         |
| `description`    | `Optional[str]`           | `None`  | アプリケーションの説明。                         |

*少なくとも `agents`、`teams`、または `workflows` のいずれか 1 つを指定してください。*

### 主要なメソッド

| メソッド                 | パラメータ                                                    | 返り値の型          | 説明                                                                                                |
| :----------------------- | :------------------------------------------------------------ | :------------------ | :-------------------------------------------------------------------------------------------------- |
| `get_app`              | `use_async: bool = True`<br/>`prefix: str = "/v1"`            | `FastAPI`            | 設定済みの FastAPI アプリを返します（デフォルトは非同期）。プレフィックス、エラーハンドラ、CORS、ドキュメントを設定します。                 |
| `get_router`           |                                                               | `APIRouter`          | Playground エンドポイント用の同期 APIRouter を返します。                                                  |
| `get_async_router`     |                                                               | `APIRouter`          | Playground エンドポイント用の非同期 APIRouter を返します。                                               |

### エンドポイント

エンドポイントは、指定された `prefix`（デフォルトは `/v1`）と Playground ルーターのプレフィックス (`/playground`) を組み合わせて利用できます。たとえば、ステータスエンドポイントは通常 `/v1/playground/status` です。

### パラメータ

| パラメータ | タイプ                | デフォルト     | 説明                                         |
| :--------- | :-------------------- | :----------- | :------------------------------------------- |
| `app`     | `Union[str, FastAPI]` | `N/A`       | FastAPI アプリケーションインスタンスまたはインポート文字列（必須）。 |
| `host`    | `str`               | `"localhost"` | バインドするホスト。                                      |
| `port`    | `int`               | `7777`      | バインドするポート。                                      |
| `reload`  | `bool`              | `False`     | 開発用に自動リロードを有効にします。                |
