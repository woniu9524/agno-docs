エンジニアや研究者はAgnoを使用して以下を構築します。

- **レベル1:** ツールと指示を持つエージェント（[例](/introduction/agents#level-1%3A-agents-with-tools-and-instructions)）。
- **レベル2:** 知識とストレージを持つエージェント（[例](/introduction/agents#level-2%3A-agents-with-knowledge-and-storage)）。
- **レベル3:** メモリと推論を持つエージェント（[例](/introduction/agents#level-3%3A-agents-with-memory-and-reasoning)）。
- **レベル4:** 推論と協調が可能なエージェントチーム（[例](/introduction/multi-agent-systems#level-4%3A-agent-teams-that-can-reason-and-collaborate)）。
- **レベル5:** 状態と決定論を持つエージェンティックワークフロー（[例](/introduction/multi-agent-systems#level-5%3A-agentic-workflows-with-state-and-determinism)）。

**例:** YFinance APIを使用して質問に答えるレベル1の推論エージェント：

```python Reasoning Finance Agent
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.reasoning import ReasoningTools
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[
        ReasoningTools(add_instructions=True),
        YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True),
    ],
    instructions="Use tables to display data.",
    markdown=True,
)
```

# なぜAgnoなのか？

Agnoは、クラス最高の高パフォーマンスなエージェンティックシステムを構築するのに役立ち、リサーチや定型コードの時間を節約します。Agnoを際立たせる主な機能は次のとおりです。

- **モデルに依存しない**: Agnoは23以上のモデルプロバイダーへの統一インターフェースを提供し、ロックインしません。
- **高パフォーマンス**: エージェントは**約3μs**でインスタンス化され、平均**約6.5KiB**のメモリを使用します。
- **推論は第一級市民**: 推論は信頼性を向上させ、複雑な自律エージェントには必須です。Agnoは3つの推論アプローチをサポートしています：推論モデル、`ReasoningTools`、またはカスタムの`chain-of-thought`アプローチ。
- **ネイティブマルチモーダル**: Agnoエージェントはネイティブにマルチモーダルであり、テキスト、画像、音声、動画を入力として受け入れ、テキスト、画像、音声、動画を出力として生成します。
- **高度なマルチエージェントアーキテクチャ**: Agnoは、推論、メモリ、共有コンテキストを備えた業界をリードするマルチエージェントアーキテクチャ（**エージェントチーム**）を提供します。
- **組み込みエージェンティック検索**: エージェントは、20以上のベクトルデータベースを使用して実行時に情報を検索できます。Agnoは最先端のエージェンティックRAGを提供し、**完全に非同期で高パフォーマンス**です。
- **組み込みメモリ＆セッションストレージ**: エージェントには組み込みの`Storage`＆`Memory`ドライバーが付属しており、エージェントに長期メモリとセッションストレージを提供します。
- **構造化出力**: Agnoエージェントは、モデル提供の構造化出力または`json_mode`を使用して、完全に型付けされた応答を返すことができます。
- **組み込みFastAPIルート**: エージェントを構築した後、組み込みのFastAPIルートを使用して提供します。数分で本番稼働。
- **監視**: [agno.com](https://app.agno.com)でエージェントセッションとパフォーマンスをリアルタイムで監視します。
