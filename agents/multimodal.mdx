---
title: マルチモーダルエージェント
---

Agnoエージェントは、テキスト、画像、音声、動画の入力をサポートし、テキスト、画像、音声、動画の出力を生成できます。完全な概要については、[互換性マトリックス](/models/compatibility)をご覧ください。

## エージェントへのマルチモーダル入力

画像ファイルを理解し、必要に応じてツール呼び出しを行うエージェントを作成しましょう。

### 画像エージェント

```python image_agent.py
from agno.agent import Agent
from agno.media import Image
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    markdown=True,
)

agent.print_response(
    "この画像について教えてください。また、それに関する最新ニュースも教えてください。",
    images=[
        Image(
            url="https://upload.wikimedia.org/wikipedia/commons/0/0c/GoldenGateBridge-001.jpg"
        )
    ],
    stream=True,
)
```

エージェントを実行します。

```shell
python image_agent.py
```

画像と同様に、音声や動画も入力として使用できます。

### 音声エージェント

```python audio_agent.py
import base64

import requests
from agno.agent import Agent, RunResponse  # noqa
from agno.media import Audio
from agno.models.openai import OpenAIChat

# 音声ファイルを取得し、base64エンコードされた文字列に変換します
url = "https://openaiassets.blob.core.windows.net/$web/API/docs/audio/alloy.wav"
response = requests.get(url)
response.raise_for_status()
wav_data = response.content

agent = Agent(
    model=OpenAIChat(id="gpt-4o-audio-preview", modalities=["text"]),
    markdown=True,
)
agent.print_response(
    "この音声には何が含まれていますか？", audio=[Audio(content=wav_data, format="wav")]
)
```

### 動画エージェント

<Note>現在、AgnoはGeminiモデルへの入力としてのみ動画をサポートしています。</Note>

```python video_agent.py
from pathlib import Path

from agno.agent import Agent
from agno.media import Video
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(id="gemini-2.0-flash-exp"),
    markdown=True,
)

# wget https://storage.googleapis.com/generativeai-downloads/images/GreatRedSpot.mp4
# を使用して「GreatRedSpot.mp4」をダウンロードしてください
video_path = Path(__file__).parent.joinpath("GreatRedSpot.mp4")

agent.print_response("この動画について教えてください", videos=[Video(filepath=video_path)])
```

## エージェントからのマルチモーダル出力

マルチモーダル入力の提供と同様に、エージェントからマルチモーダル出力も取得できます。

### 画像生成

次の例は、DALL-Eを使用してエージェントで画像を生成する方法を示しています。

```python image_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.dalle import DalleTools

image_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[DalleTools()],
    description="あなたはDALL-Eを使用して画像を生成できるAIエージェントです。",
    instructions="画像を作成するようにユーザーに求められた場合は、`create_image`ツールを使用して画像を作成してください。",
    markdown=True,
    show_tool_calls=True,
)

image_agent.print_response("白いシャム猫の画像を生成してください")

images = image_agent.get_images()
if images and isinstance(images, list):
    for image_response in images:
        image_url = image_response.url
        print(image_url)
```

### 音声応答

次の例は、テキストと音声の両方の応答をエージェントから取得する方法を示しています。エージェントは、ファイルに保存できるテキストと音声バイトで応答します。

```python audio_agent.py
from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat
from agno.utils.audio import write_audio_to_file

agent = Agent(
    model=OpenAIChat(
        id="gpt-4o-audio-preview",
        modalities=["text", "audio"],
        audio={"voice": "alloy", "format": "wav"},
    ),
    markdown=True,
)
response: RunResponse = agent.run("5秒間の怖い話を教えてください")

# 応答音声をファイルに保存します
if response.response_audio is not None:
    write_audio_to_file(
        audio=agent.run_response.response_audio.content, filename="tmp/scary_story.wav"
    )
```

## マルチモーダル入力と出力の組み合わせ

マルチモーダル入力を受け取り、マルチモーダル出力を返すエージェントを作成できます。次の例は、エージェントに音声とテキストの入力の組み合わせを提供し、テキストと音声の両方の出力を取得する方法を示しています。

### 音声入力と音声出力

```python audio_agent.py
import base64

import requests
from agno.agent import Agent
from agno.media import Audio
from agno.models.openai import OpenAIChat
from agno.utils.audio import write_audio_to_file

# 音声ファイルを取得し、base64エンコードされた文字列に変換します
url = "https://openaiassets.blob.core.windows.net/$web/API/docs/audio/alloy.wav"
response = requests.get(url)
response.raise_for_status()
wav_data = response.content

agent = Agent(
    model=OpenAIChat(
        id="gpt-4o-audio-preview",
        modalities=["text", "audio"],
        audio={"voice": "alloy", "format": "wav"},
    ),
    markdown=True,
)

agent.run("これらの録音には何が入っていますか？", audio=[Audio(content=wav_data, format="wav")])

if agent.run_response.response_audio is not None:
    write_audio_to_file(
        audio=agent.run_response.response_audio.content, filename="tmp/result.wav"
    )
