---
title: 结构化输出
---

我们最喜欢的功能之一是使用 Agent 来生成结构化数据（即 pydantic 模型）。使用此功能可以提取特征、对数据进行分类、生成假数据等。最棒的是，它们可以与函数调用、知识库以及所有其他功能协同工作。

## 示例

让我们创建一个电影 Agent 来为我们编写 `MovieScript`。

```python movie_agent.py
from typing import List
from rich.pretty import pprint
from pydantic import BaseModel, Field
from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat

class MovieScript(BaseModel):
    setting: str = Field(..., description="为一部大片提供一个不错的场景。")
    ending: str = Field(..., description="电影的结局。如果不可用，请提供一个快乐的结局。")
    genre: str = Field(
        ..., description="电影的类型。如果不可用，请选择动作、惊悚或浪漫喜剧。"
    )
    name: str = Field(..., description="给这部电影起个名字")
    characters: List[str] = Field(..., description="这部电影的角色名字。")
    storyline: str = Field(..., description="电影的3句话剧情简介。要令人兴奋！")

# 使用 JSON 模式的 Agent
json_mode_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="你编写电影剧本。",
    response_model=MovieScript,
    use_json_mode=True,
)
json_mode_agent.print_response("纽约")

# 使用结构化输出的 Agent
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="你编写电影剧本。",
    response_model=MovieScript,
)

structured_output_agent.print_response("纽约")
```

## 使用解析器模型

您可以使用额外的模型来解析和结构化主模型生成的输出。当主模型针对推理任务进行了优化，但可能无法始终生成详细的结构化响应时，这种方法尤其有效。

```python
agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    description="你编写电影剧本。",
    response_model=MovieScript,
    parser_model=OpenAIChat(id="gpt-4o"),
)
```

您还可以为解析器模型提供自定义的 `parser_model_prompt`。

## 流式传输结构化输出

流式传输可以与 `response_model` 结合使用。这将在事件流中将结构化输出作为单个事件返回。

```python streaming_agent.py
import asyncio
from typing import Dict, List

from agno.agent import Agent
from agno.models.openai.chat import OpenAIChat
from pydantic import BaseModel, Field


class MovieScript(BaseModel):
    setting: str = Field(
        ..., description="为一部大片提供一个不错的场景。"
    )
    ending: str = Field(
        ...,
        description="电影的结局。如果不可用，请提供一个快乐的结局。",
    )
    genre: str = Field(
        ...,
        description="电影的类型。如果不可用，请选择动作、惊悚或浪漫喜剧。",
    )
    name: str = Field(..., description="给这部电影起个名字")
    characters: List[str] = Field(..., description="这部电影的角色名字。")
    storyline: str = Field(
        ..., description="电影的3句话剧情简介。要令人兴奋！"
    )
    rating: Dict[str, int] = Field(
        ...,
        description="你对电影的评分。1-10。返回一个包含 'story' 和 'acting' 键的字典。",
    )


# 使用结构化输出和流式传输的 Agent
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="你编写电影剧本。",
    response_model=MovieScript,
)

structured_output_agent.print_response(
    "纽约", stream=True, stream_intermediate_steps=True
)
```
