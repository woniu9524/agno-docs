---
title: 構造化出力
---

私たちのお気に入りの機能の1つは、エージェントを使用して構造化データ（つまりpydanticモデル）を生成することです。この機能を使用して、機能の抽出、データの分類、偽データの生成などを行います。最も良い点は、関数呼び出し、ナレッジベース、およびその他のすべての機能と連携することです。

## 例

映画スクリプトを記述する映画エージェントを作成しましょう。

```python movie_agent.py
from typing import List
from rich.pretty import pprint
from pydantic import BaseModel, Field
from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat

class MovieScript(BaseModel):
    setting: str = Field(..., description="ブロックバスター映画の素敵な設定を提供してください。")
    ending: str = Field(..., description="映画のエンディング。利用できない場合は、ハッピーエンディングを提供してください。")
    genre: str = Field(
        ..., description="映画のジャンル。利用できない場合は、アクション、スリラー、またはロマンティックコメディを選択してください。"
    )
    name: str = Field(..., description="この映画に名前を付けてください")
    characters: List[str] = Field(..., description="この映画のキャラクターの名前。")
    storyline: str = Field(..., description="映画の3文のストーリーライン。エキサイティングにしてください！")

# JSONモードを使用するエージェント
json_mode_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="映画のスクリプトを書きます。",
    response_model=MovieScript,
    use_json_mode=True,
)
json_mode_agent.print_response("New York")

# 構造化出力を利用するエージェント
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="映画のスクリプトを書きます。",
    response_model=MovieScript,
)

structured_output_agent.print_response("New York")
```

スクリプトを実行して出力を確認してください。


## パーサーモデルの使用

プライマリモデルからの出力を解析および構造化するために、追加のモデルを使用できます。このアプローチは、プライマリモデルが推論タスクに最適化されている場合に特に効果的です。そのようなモデルは、詳細な構造化応答を一貫して生成しない可能性があるためです。

```python
agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    description="映画のスクリプトを書きます。",
    response_model=MovieScript,
    parser_model=OpenAIChat(id="gpt-4o"),
)
```

カスタムの`parser_model_prompt`をパーサーモデルに提供することもできます。

## ストリーミング構造化出力

ストリーミングは、`response_model`と組み合わせて使用できます。これにより、構造化出力がイベントストリーム内の単一のイベントとして返されます。

```python streaming_agent.py
import asyncio
from typing import Dict, List

from agno.agent import Agent
from agno.models.openai.chat import OpenAIChat
from pydantic import BaseModel, Field


class MovieScript(BaseModel):
    setting: str = Field(
        ..., description="ブロックバスター映画の素敵な設定を提供してください。"
    )
    ending: str = Field(
        ...,
        description="映画のエンディング。利用できない場合は、ハッピーエンディングを提供してください。",
    )
    genre: str = Field(
        ...,
        description="映画のジャンル。利用できない場合は、アクション、スリラー、またはロマンティックコメディを選択してください。",
    )
    name: str = Field(..., description="この映画に名前を付けてください")
    characters: List[str] = Field(..., description="この映画のキャラクターの名前。")
    storyline: str = Field(
        ..., description="映画の3文のストーリーライン。エキサイティングにしてください！"
    )
    rating: Dict[str, int] = Field(
        ...,
        description="映画自身の評価。1〜10。キー「story」と「acting」を持つ辞書を返してください。",
    )


# 構造化出力を利用するエージェント（ストリーミング使用）
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="映画のスクリプトを書きます。",
    response_model=MovieScript,
)

structured_output_agent.print_response(
    "New York", stream=True, stream_intermediate_steps=True
)
```
