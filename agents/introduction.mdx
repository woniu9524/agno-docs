**エージェント**は自律的に動作するAIプログラムです。従来のソフトウェアは、事前にプログラムされた一連のステップに従います。エージェントは、機械学習**モデル**を使用して、動的に行動方針を決定します。

エージェントのコアは、**モデル**、**ツール**、**指示**です。

- **モデル:** 実行フローを制御します。推論、行動、応答のいずれを行うかを決定します。
- **ツール:** エージェントがアクションを実行し、外部システムと対話できるようにします。
- **指示:** エージェントをプログラムし、ツールの使用方法や応答方法を教える方法です。

エージェントは、**メモリ**、**知識**、**ストレージ**、および**推論**能力も備えています。

- **推論:** エージェントが応答する前に「考え」、アクション（ツール呼び出し）の結果を「分析」できるようにし、応答の信頼性と品質を向上させます。
- **知識:** エージェントが実行時に**検索**して、より良い意思決定を行い、正確な応答を提供できるドメイン固有の情報です（RAG）。知識はベクトルデータベースに保存され、この**実行時の検索**パターンはAgentic RAG/Agentic Searchとして知られています。
- **ストレージ:** エージェントがセッション履歴と状態をデータベースに保存するために使用されます。モデルAPIはステートレスであり、ストレージにより、会話を中断したところから再開できます。これにより、エージェントはステートフルになり、マルチターン、長期的な会話が可能になります。
- **メモリ:** エージェントが以前の対話からの情報を保存および呼び出す能力を与え、ユーザーの好みを学習し、応答をパーソナライズできるようにします。

## 例：リサーチエージェント

Exaを使用してリサーチエージェントを構築し、エージェントにレポートを特定の形式で生成するようにガイドする方法を示します。高度なケースでは、代わりに[構造化出力](/agents/structured-output)を使用する必要があります。

<Note>
  説明と指示はシステムメッセージに変換され、
  入力はユーザーメッセージとして渡されます。
  `debug_mode=True` を設定すると、舞台裏のログを表示できます。
</Note>

<Steps>
  <Step title="リサーチエージェントの作成">
    `research_agent.py` ファイルを作成します。

    ```python research_agent.py
    from datetime import datetime
    from pathlib import Path
    from textwrap import dedent

    from agno.agent import Agent
    from agno.models.openai import OpenAIChat
    from agno.tools.exa import ExaTools

    today = datetime.now().strftime("%Y-%m-%d")

    agent = Agent(
        model=OpenAIChat(id="gpt-4o"),
        tools=[ExaTools(start_published_date=today, type="keyword")],
        description=dedent("""\
            あなたは、複雑な情報を分析および合成する専門知識を持つ、著名なAIリサーチサイエンティストであるProfessor X-1000です。
            あなたの専門は、学術的な厳密さと魅力的な物語を組み合わせた、説得力のある事実に基づいたレポートを作成することです。

            あなたの文章スタイルは次のとおりです。
            - 明確で権威がある
            - 魅力的でありながらプロフェッショナル
            - 正確な引用を含む事実に基づいた
            - 教育を受けた非専門家にも分かりやすい\
        """),
        instructions=dedent("""\
            包括的な情報を収集するために、まず3つの異なる検索を実行してください。
            ソースの正確性と関連性を分析および相互参照してください。
            学術的な基準に従ってレポートを構成しますが、読みやすさを維持してください。
            検証可能な事実のみを適切な引用とともに含めてください。
            読者を複雑なトピックに導く、魅力的な物語を作成してください。
            実行可能な要点と将来の含意で締めくくってください。\
        """),
        expected_output=dedent("""\
        Markdown形式のプロフェッショナルなリサーチレポート：

        # {トピックの本質を捉える魅力的なタイトル}

        ## エグゼクティブサマリー
        {主要な発見と重要性の簡単な概要}

        ## イントロダクション
        {トピックの背景と重要性}
        {リサーチ/議論の現状}

        ## 主要な発見
        {主要な発見または進展}
        {裏付けとなる証拠と分析}

        ## インプリケーション
        {分野/社会への影響}
        {将来の方向性}

        ## 主要な要点
        - {箇条書き1}
        - {箇条書き2}
        - {箇条書き3}

        ## 参考文献
        - [ソース1](link) - 主要な発見/引用
        - [ソース2](link) - 主要な発見/引用
        - [ソース3](link) - 主要な発見/引用

        ---
        Professor X-1000によって生成されたレポート
        先進リサーチシステム部門
        日付： {current_date}\
        """),
        markdown=True,
        show_tool_calls=True,
        add_datetime_to_instructions=True,
    )

    # 使用例
    if __name__ == "__main__":
        # 最先端のトピックに関するリサーチレポートを生成する
        agent.print_response(
            "脳コンピューターインターフェースの最新動向を調査する", stream=True
        )

    # 試すその他の例プロンプト：
    """
    これらのリサーチトピックを試してください：
    1. "全固体電池の現状を分析する"
    2. "CRISPR遺伝子編集の最近の進歩を調査する"
    3. "自動運転車の開発を調査する"
    4. "量子機械学習の進歩を探る"
    5. "人工知能が医療に与える影響を研究する"
    """
    ```

  </Step>
  <Step title="エージェントの実行">
    ライブラリをインストールします。

    ```shell
    pip install openai exa-py agno
    ```

    エージェントを実行します。

    ```shell
    python research_agent.py
    ```

  </Step>
</Steps>
