Agent Context は Agno のもう一つの素晴らしい機能です。`context` は、エージェントが実行される前に解決される関数（または依存関係）のセットを含む辞書です。

<Note>
Context は、エージェントの説明と指示に依存関係を注入する方法です。

Context を使用して、メモリ、動的な few-shot 例、"取得した" ドキュメントなどを注入できます。
</Note>

```python agent_context.py
import json
from textwrap import dedent

import httpx
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def get_top_hackernews_stories(num_stories: int = 5) -> str:
    """HackerNews のトップストーリーを取得して返します。

    Args:
        num_stories: 取得するトップストーリーの数 (デフォルト: 5)
    Returns:
        ストーリーの詳細 (タイトル、URL、スコアなど) を含む JSON 文字列
    """
    # トップストーリーを取得
    stories = [
        {
            k: v
            for k, v in httpx.get(
                f"https://hacker-news.firebaseio.com/v0/item/{id}.json"
            )
            .json()
            .items()
            if k != "kids"  # ディスカッションスレッドを除外
        }
        for id in httpx.get(
            "https://hacker-news.firebaseio.com/v0/topstories.json"
        ).json()[:num_stories]
    ]
    return json.dumps(stories, indent=4)


# リアルタイムの HackerNews データにアクセスできる Context を認識するエージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    # Context 内の各関数は、エージェントが実行されるときに評価されます。
    # エージェントの依存関係注入と考えてください
    context={"top_hackernews_stories": get_top_hackernews_stories},
    # または、Context を手動で指示に追加することもできます
    instructions=dedent("""\
        あなたは洞察力のあるテックトレンドオブザーバーです！📰

        HackerNews のトップストーリーはこちらです:
        {top_hackernews_stories}\
    """),
    # add_state_in_messages は、`top_hackernews_stories` 変数を
    # 指示で利用可能にします
    add_state_in_messages=True,
    markdown=True,
)

# 使用例
agent.print_response(
    "HackerNews のトップストーリーを要約し、興味深いトレンドを特定してください。",
    stream=True,
)
```

## Context 全体をユーザーメッセージに追加する

`add_context=True` を設定すると、Context 全体がユーザーメッセージに追加されます。これにより、Context を手動で指示に追加する必要がなくなります。

```python agent_context_instructions.py
import json
from textwrap import dedent

import httpx
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def get_top_hackernews_stories(num_stories: int = 5) -> str:
    """HackerNews のトップストーリーを取得して返します。

    Args:
        num_stories: 取得するトップストーリーの数 (デフォルト: 5)
    Returns:
        ストーリーの詳細 (タイトル、URL、スコアなど) を含む JSON 文字列
    """
    # トップストーリーを取得
    stories = [
        {
            k: v
            for k, v in httpx.get(
                f"https://hacker-news.firebaseio.com/v0/item/{id}.json"
            )
            .json()
            .items()
            if k != "kids"  # ディスカッションスレッドを除外
        }
        for id in httpx.get(
            "https://hacker-news.firebaseio.com/v0/topstories.json"
        ).json()[:num_stories]
    ]
    return json.dumps(stories, indent=4)


# リアルタイムの HackerNews データにアクセスできる Context を認識するエージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    # 各 Context 内の関数は、エージェントが実行されるときに解決されます。
    # エージェントの依存関係注入と考えてください
    context={"top_hackernews_stories": get_top_hackernews_stories},
    # Context の辞書全体を指示に追加できます
    add_context=True,
    markdown=True,
)

# 使用例
agent.print_response(
    "HackerNews のトップストーリーを要約し、興味深いトレンドを特定してください。",
    stream=True,
)
