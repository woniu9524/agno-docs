title: Agent State

**State** 是 Agent 在运行过程中需要维护的任何类型的数据。

<Check>
Agents 一个简单但常见的用例是为用户管理列表、项目和其他“信息”。例如，购物清单、待办事项列表、愿望清单等。

这可以使用 `session_state` 轻松管理。Agent 在工具调用中更新 `session_state`，并在 `description` 和 `instructions` 中将其暴露给模型。
</Check>

Agno 提供了一个强大而优雅的状态管理系统，其工作原理如下：

- `Agent` 具有 `session_state` 参数。
- 我们将状态变量添加到此 `session_state` 字典中。
- 我们在工具调用或其他函数中更新 `session_state` 字典。
- 我们在 `description` 和 `instructions` 中与模型共享当前的 `session_state`。
- `session_state` 与 Agent 会话一起存储，并持久化在数据库中。这意味着它在执行周期之间可用。

以下是一个管理购物清单的 Agent 示例：

```python session_state.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# 定义一个工具，用于递增计数器并返回新值
def add_item(agent: Agent, item: str) -> str:
    """将商品添加到购物清单。"""
    agent.session_state["shopping_list"].append(item)
    return f"购物清单现在是 {agent.session_state['shopping_list']}"


# 创建一个维护状态的 Agent
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 初始化 session state，计数器从 0 开始
    session_state={"shopping_list": []},
    tools=[add_item],
    # 您可以在 instructions 中使用 session state 中的变量
    instructions="当前状态（购物清单）是：{shopping_list}",
    # 重要提示：将 state 添加到 messages 中
    add_state_in_messages=True,
    markdown=True,
)

# 示例用法
agent.print_response("将牛奶、鸡蛋和面包添加到购物清单", stream=True)
print(f"最终 session state: {agent.session_state}")
```

<Tip>
这就是状态管理所能达到的最佳和最优雅的程度。
</Tip>

## 在多个运行中维护状态

**会话**的一个巨大优势是能够在多个运行中维护状态。例如，假设 Agent 正在帮助用户跟踪他们的购物清单。

<Note>
通过设置 `add_state_in_messages=True`，`session_state` 字典的键可以在 `description` 和 `instructions` 中作为变量使用。

使用此模式将购物清单直接添加到 instructions 中。
</Note>

```python shopping_list.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat


# 定义管理购物清单的工具
def add_item(agent: Agent, item: str) -> str:
    """将商品添加到购物清单并返回确认。"""
    # 如果商品不在列表中，则添加它
    if item.lower() not in [i.lower() for i in agent.session_state["shopping_list"]]:
        agent.session_state["shopping_list"].append(item)
        return f"已将 '{item}' 添加到购物清单"
    else:
        return f"'{item}' 已在购物清单中"


def remove_item(agent: Agent, item: str) -> str:
    """按名称从购物清单中移除商品。"""
    # 不区分大小写的搜索
    for i, list_item in enumerate(agent.session_state["shopping_list"]):
        if list_item.lower() == item.lower():
            agent.session_state["shopping_list"].pop(i)
            return f"已从购物清单中移除 '{list_item}'"

    return f"在购物清单中未找到 '{item}'"


def list_items(agent: Agent) -> str:
    """列出购物清单中的所有商品。"""
    shopping_list = agent.session_state["shopping_list"]

    if not shopping_list:
        return "购物清单为空。"

    items_text = "\n".join([f"- {item}" for item in shopping_list])
    return f"当前购物清单：\n{items_text}"


# 创建一个维护状态的购物清单管理器 Agent
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 初始化 session state，包含一个空的购物清单
    session_state={"shopping_list": []},
    tools=[add_item, remove_item, list_items],
    # 您可以在 instructions 中使用 session state 中的变量
    instructions=dedent("""\
        你的工作是管理一个购物清单。

        购物清单最初是空的。您可以添加商品、按名称移除商品以及列出所有商品。

        当前购物清单：{shopping_list}
    """),
    show_tool_calls=True,
    add_state_in_messages=True,
    markdown=True,
)

# 示例用法
agent.print_response("将牛奶、鸡蛋和面包添加到购物清单", stream=True)
print(f"Session state: {agent.session_state}")

agent.print_response("我拿到了面包", stream=True)
print(f"Session state: {agent.session_state}")

agent.print_response("我需要苹果和橙子", stream=True)
print(f"Session state: {agent.session_state}")

agent.print_response("我的清单上有什么？", stream=True)
print(f"Session state: {agent.session_state}")

agent.print_response("将我清单上的所有东西都清空，然后重新开始，只保留香蕉和酸奶", stream=True)
print(f"Session state: {agent.session_state}")
```

<Tip>
我们非常喜欢在多个运行中维护和传递状态的优雅方式。
</Tip>

## 在 instructions 中使用状态

通过设置 `add_state_in_messages=True`，您可以在 instructions 中使用 `session_state` 中的变量。

<Tip>
不要在 instructions 中使用 f-string 语法。直接使用 `{key}` 语法，Agno 会为您替换值。
</Tip>

```python state_in_instructions.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat


agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 初始化 session state，包含一个变量
    session_state={"user_name": "John"},
    # 您可以在 instructions 中使用 session state 中的变量
    instructions="用户的名字是 {user_name}",
    show_tool_calls=True,
    add_state_in_messages=True,
    markdown=True,
)

agent.print_response("我的名字是什么？", stream=True)
```

## 将状态持久化到数据库

`session_state` 是 Agent 会话的一部分，如果在运行后提供了 `storage` 驱动程序，它将被保存在数据库中。

这是一个维护购物清单并在数据库中持久化状态的 Agent 示例。多次运行此脚本以查看状态的持久化。

```python session_state_storage.py
"""运行 `pip install agno openai sqlalchemy` 来安装依赖项。"""

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.storage.sqlite import SqliteStorage


# 定义一个将商品添加到购物清单的工具
def add_item(agent: Agent, item: str) -> str:
    """将商品添加到购物清单。"""
    if item not in agent.session_state["shopping_list"]:
        agent.session_state["shopping_list"].append(item)
    return f"购物清单现在是 {agent.session_state['shopping_list']}"


agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 固定 session id 以在执行周期之间继续同一个会话
    session_id="fixed_id_for_demo",
    # 初始化 session state，包含一个空的购物清单
    session_state={"shopping_list": []},
    # 添加一个将商品添加到购物清单的工具
    tools=[add_item],
    # 将 session state 存储在 SQLite 数据库中
    storage=SqliteStorage(table_name="agent_sessions", db_file="tmp/data.db"),
    # 在 instructions 中添加当前的购物清单
    instructions="当前购物清单是：{shopping_list}",
    # 重要提示：设置 `add_state_in_messages=True`
    # 以便在 instructions 中可以使用 `{shopping_list}`
    add_state_in_messages=True,
    markdown=True,
)

# 示例用法
agent.print_response("我的购物清单上有什么？", stream=True)
print(f"Session state: {agent.session_state}")
agent.print_response("添加牛奶、鸡蛋和面包", stream=True)
print(f"Session state: {agent.session_state}")
