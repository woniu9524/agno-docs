**State** は、Agent が実行全体を通じて維持する必要があるあらゆる種類のデータです。

<Check>
Agent のシンプルかつ一般的なユースケースは、ユーザーのリスト、アイテム、その他の「情報」を管理することです。たとえば、ショッピングリスト、ToDoリスト、ウィッシュリストなどです。

これは `session_state` を使用して簡単に管理できます。Agent はツール呼び出しで `session_state` を更新し、`description` および `instructions` でモデルに公開します。
</Check>

Agno's は強力でエレガントな状態管理システムを提供します。仕組みは次のとおりです。

- `Agent` には `session_state` パラメータがあります。
- この `session_state` ディクショナリに状態変数を追加します。
- ツール呼び出しまたはその他の関数で `session_state` ディクショナリを更新します。
- 現在の `session_state` をモデルと `description` および `instructions` で共有します。
- `session_state` は Agent セッションとともに保存され、データベースに永続化されます。つまり、実行サイクル間で利用可能です。

ショッピングリストを管理する Agent の例を次に示します。

```python session_state.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# カウンターをインクリメントして新しい値を返すツールを定義します
def add_item(agent: Agent, item: str) -> str:
    """ショッピングリストにアイテムを追加します。"""
    agent.session_state["shopping_list"].append(item)
    return f"ショッピングリストは現在 {agent.session_state['shopping_list']} です"


# 状態を維持する Agent を作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # セッション状態をカウンターが 0 から始まるように初期化します
    session_state={"shopping_list": []},
    tools=[add_item],
    # instructions でセッション状態の変数を使用できます
    instructions="現在の状態 (ショッピングリスト) は: {shopping_list}",
    # 重要: メッセージに状態を追加します
    add_state_in_messages=True,
    markdown=True,
)

# 使用例
agent.print_response("ミルク、卵、パンをショッピングリストに追加してください", stream=True)
print(f"最終セッション状態: {agent.session_state}")
```

<Tip>
これは状態管理としては、これ以上ないほどエレガントです。
</Tip>

## 複数回の実行で状態を維持する

**セッション**の大きな利点は、複数回の実行で状態を維持できることです。たとえば、Agent がユーザーのショッピングリストの追跡を支援しているとします。

<Note>
`add_state_in_messages=True` を設定すると、`session_state` ディクショナリのキーが `description` および `instructions` で変数として利用可能になります。

このパターンを使用して、ショッピングリストを instructions に直接追加します。
</Note>

```python shopping_list.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat


# ショッピングリストを管理するためのツールを定義します
def add_item(agent: Agent, item: str) -> str:
    """ショッピングリストにアイテムを追加し、確認を返します。"""
    # リストに既に存在しない場合はアイテムを追加します
    if item.lower() not in [i.lower() for i in agent.session_state["shopping_list"]]:
        agent.session_state["shopping_list"].append(item)
        return f"'{item}' をショッピングリストに追加しました"
    else:
        return f"'{item}' は既にショッピングリストにあります"


def remove_item(agent: Agent, item: str) -> str:
    """名前でショッピングリストからアイテムを削除します。"""
    # 大文字小文字を区別しない検索
    for i, list_item in enumerate(agent.session_state["shopping_list"]):
        if list_item.lower() == item.lower():
            agent.session_state["shopping_list"].pop(i)
            return f"'{list_item}' をショッピングリストから削除しました"

    return f"'{item}' はショッピングリストに見つかりませんでした"


def list_items(agent: Agent) -> str:
    """ショッピングリストのすべてのアイテムを一覧表示します。"""
    shopping_list = agent.session_state["shopping_list"]

    if not shopping_list:
        return "ショッピングリストは空です。"

    items_text = "\n".join([f"- {item}" for item in shopping_list])
    return f"現在のショッピングリスト:\n{items_text}"


# 状態を維持するショッピングリストマネージャー Agent を作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 空のショッピングリストでセッション状態を初期化します
    session_state={"shopping_list": []},
    tools=[add_item, remove_item, list_items],
    # instructions でセッション状態の変数を使用できます
    instructions=dedent("""\
        あなたの仕事はショッピングリストを管理することです。

        ショッピングリストは空の状態から始まります。アイテムの追加、名前によるアイテムの削除、すべてのアイテムの一覧表示ができます。

        現在のショッピングリスト: {shopping_list}
    """),
    show_tool_calls=True,
    add_state_in_messages=True,
    markdown=True,
)

# 使用例
agent.print_response("ミルク、卵、パンをショッピングリストに追加してください", stream=True)
print(f"セッション状態: {agent.session_state}")

agent.print_response("パンを手に入れました", stream=True)
print(f"セッション状態: {agent.session_state}")

agent.print_response("りんごとオレンジが必要です", stream=True)
print(f"セッション状態: {agent.session_state}")

agent.print_response("私のリストには何がありますか？", stream=True)
print(f"セッション状態: {agent.session_state}")

agent.print_response("リストからすべてクリアして、バナナとヨーグルトだけでやり直してください", stream=True)
print(f"セッション状態: {agent.session_state}")
```

<Tip>
複数回の実行で状態を維持し、渡すことができるエレガントさに感動します。
</Tip>

## instructions での状態の使用

`add_state_in_messages=True` を設定することで、セッション状態の変数を instructions で使用できます。

<Tip>
instructions で f-string 構文を使用しないでください。直接 `{key}` 構文を使用してください。Agno が値を代入します。
</Tip>

```python state_in_instructions.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat


agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 変数でセッション状態を初期化します
    session_state={"user_name": "John"},
    # instructions でセッション状態の変数を使用できます
    instructions="ユーザーの名前は {user_name} です",
    show_tool_calls=True,
    add_state_in_messages=True,
    markdown=True,
)

agent.print_response("私の名前は何ですか？", stream=True)
```

## データベースへの状態の永続化

`session_state` は Agent セッションの一部であり、`storage` ドライバーが提供されている場合、各実行後にデータベースに保存されます。

ここでは、ショッピングリストを維持し、状態をデータベースに永続化する Agent の例を示します。このスクリプトを複数回実行して、状態が永続化される様子を確認してください。

```python session_state_storage.py
"""依存関係をインストールするには `pip install agno openai sqlalchemy` を実行してください。"""

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.storage.sqlite import SqliteStorage


# ショッピングリストにアイテムを追加するツールを定義します
def add_item(agent: Agent, item: str) -> str:
    """ショッピングリストにアイテムを追加します。"""
    if item not in agent.session_state["shopping_list"]:
        agent.session_state["shopping_list"].append(item)
    return f"ショッピングリストは現在 {agent.session_state['shopping_list']} です"


agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # 同じセッションを複数の実行サイクルで継続するためにセッションIDを固定します
    session_id="fixed_id_for_demo",
    # 空のショッピングリストでセッション状態を初期化します
    session_state={"shopping_list": []},
    # ショッピングリストにアイテムを追加するツールを追加します
    tools=[add_item],
    # セッション状態を SQLite データベースに保存します
    storage=SqliteStorage(table_name="agent_sessions", db_file="tmp/data.db"),
    # 現在のショッピングリストを状態から instructions に追加します
    instructions="現在のショッピングリストは: {shopping_list}",
    # 重要: `add_state_in_messages=True` を設定します
    # これにより、instructions で `{shopping_list}` が利用可能になります
    add_state_in_messages=True,
    markdown=True,
)

# 使用例
agent.print_response("私のショッピングリストは何ですか？", stream=True)
print(f"セッション状態: {agent.session_state}")
agent.print_response("ミルク、卵、パンを追加してください", stream=True)
print(f"セッション状態: {agent.session_state}")
