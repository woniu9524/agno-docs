# エージェントによる知識フィルタリング

エージェントによるフィルタリングにより、エージェントはクエリテキストからフィルタ基準を自動的に抽出し、より自然でインタラクティブなエクスペリエンスを提供します。

## ステップ 1: メタデータの添付

ドキュメントにメタデータを添付するには、2つの方法があります。

1. **知識ベースの初期化時にメタデータを添付する**

    ```python
    knowledge_base = PDFKnowledgeBase(
        path=[
            {
                "path": "path/to/cv1.pdf",
                "metadata": {
                    "user_id": "jordan_mitchell",
                    "document_type": "cv",
                    "year": 2025,
                },
            },
            # ... その他のドキュメント ...
        ],
        vector_db=vector_db,
    )
    knowledge_base.load(recreate=True)
    ```

2. **ドキュメントを一つずつロードする際にメタデータを添付する**

    ```python
    # PDFKnowledgeBase を初期化する
    knowledge_base = PDFKnowledgeBase(
        vector_db=vector_db,
        num_documents=5,
    )

    # user_1 メタデータで最初のドキュメントをロードする
    knowledge_base.load_document(
        path=path/to/cv1.pdf,
        metadata={"user_id": "jordan_mitchell", "document_type": "cv", "year": 2025},
        recreate=True,  # 最初の実行時のみ True に設定し、その後は False に設定してください
    )

    # user_2 メタデータで 2 番目のドキュメントをロードする
    knowledge_base.load_document(
        path=path/to/cv2.pdf,
        metadata={"user_id": "taylor_brooks", "document_type": "cv", "year": 2025},
    )
    ```

---

## 仕組み

エージェントフィルタリングを有効にすると (`enable_agentic_knowledge_filters=True`)、エージェントはクエリを分析し、検出したメタデータに基づいてフィルタを適用します。

**例:**
```python
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
    enable_agentic_knowledge_filters=True,
)
agent.print_response(
    "Tell me about Jordan Mitchell's experience and skills with jordan_mitchell as user id and document type cv",
    markdown=True,
)
```

この例では、エージェントは自動的に以下を使用します。
- `user_id = "jordan_mitchell"`
- `document_type = "cv"`

- より会話的でユーザーフレンドリーなエクスペリエンスを求めている場合。
- ユーザーが正確なフィルタ構文を知らない可能性がある場合。
