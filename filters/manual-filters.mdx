# 手動ナレッジフィルタ

手動フィルタリングでは、コード内で直接フィルタを指定することで、検索対象となるドキュメントを完全に制御できます。

## ステップ 1: メタデータの添付

ドキュメントにメタデータを添付するには、2つの方法があります。

1. **ナレッジベースの初期化時にメタデータを添付する**

    ```python
    knowledge_base = PDFKnowledgeBase(
        path=[
            {
                "path": "path/to/cv1.pdf",
                "metadata": {
                    "user_id": "jordan_mitchell",
                    "document_type": "cv",
                    "year": 2025,
                },
            },
            # ... その他のドキュメント ...
        ],
        vector_db=vector_db,
    )
    knowledge_base.load(recreate=True)
    ```

2. **ドキュメントを1つずつロードする際にメタデータを添付する**

    ```python
    # PDFKnowledgeBase の初期化
    knowledge_base = PDFKnowledgeBase(
        vector_db=vector_db,
        num_documents=5,
    )

    # user_1 メタデータを持つ最初のドキュメントをロード
    knowledge_base.load_document(
        path=path/to/cv1.pdf,
        metadata={"user_id": "jordan_mitchell", "document_type": "cv", "year": 2025},
        recreate=True,  # 最初の実行時のみ True に設定し、その後は False に設定します
    )

    # user_2 メタデータを持つ 2 番目のドキュメントをロード
    knowledge_base.load_document(
        path=path/to/cv2.pdf,
        metadata={"user_id": "taylor_brooks", "document_type": "cv", "year": 2025},
    )
    ```

---

> 💡 **ヒント:**
> • すべてのドキュメントとメタデータが一度に準備できている場合は、**オプション 1** を使用します。
> • ドキュメントを段階的に、または利用可能になった時点で追加したい場合は、**オプション 2** を使用します。

## ステップ 2: フィルタを使用したクエリ

フィルタは2つの方法で渡すことができます。

### 1. エージェントで（すべてのクエリに適用）

```python
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
    knowledge_filters={"user_id": "jordan_mitchell"},
)
agent.print_response(
    "Jordan Mitchell の経験とスキルについて教えてください",
    markdown=True,
)
```

### 2. 各クエリで（その実行のエージェントフィルタをオーバーライドします）

```python
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
)
agent.print_response(
    "Jordan Mitchell の経験とスキルについて教えてください",
    knowledge_filters={"user_id": "jordan_mitchell"},
    markdown=True,
)
```

<Note>エージェントとクエリの両方でフィルタを渡した場合、クエリレベルのフィルタが優先されます。</Note>

## 複数フィルタの組み合わせ

複数のフィールドでフィルタリングできます。

```python
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
    knowledge_filters={
        "user_id": "jordan_mitchell",
        "document_type": "cv",
        "year": 2025,
    }
)
agent.print_response(
    "Jordan Mitchell の経験とスキルについて教えてください",
    markdown=True,
)
```

- さまざまなフィルタの組み合わせでクエリを実行します。
