## agno のインストール

強く推奨される方法：

- Python の仮想環境で `pip` を使用して `agno` をインストールします。

<Steps>
  <Step title="仮想環境を作成する">
    <CodeGroup>

    ```bash Mac
    python3 -m venv ~/.venvs/agno
    source ~/.venvs/agno/bin/activate
    ```

    ```bash Windows
    python3 -m venv agnoenv
    agnoenv/scripts/activate
    ```

    </CodeGroup>

  </Step>
  <Step title="agno をインストールする">
    pip を使用して `agno` をインストールします。

    <CodeGroup>

    ```bash Mac
    pip install -U agno
    ```

    ```bash Windows
    pip install -U agno
    ```

    </CodeGroup>

  </Step>
</Steps>

<br />

<Note>

エラーが発生した場合は、`python -m pip install --upgrade pip` を使用して pip を更新してみてください。

</Note>

---

## agno のアップグレード

`agno` をアップグレードするには、仮想環境で以下を実行します。

```bash
pip install -U agno --no-cache-dir
```
