title: Agno Platform での認証
description: "認証を設定して、監視、パフォーマンスメトリクスの追跡、Agno ワークスペースの管理を開始します。"
---

Agno で認証するには、2 つの方法があります。

1. CLI セットアップコマンド (`ag setup`) を使用する
2. 環境で API キーを手動で設定する

### 方法 1: CLI 認証

Agno CLI を使用するのが最も速い認証方法です。

```bash
ag setup
```

このコマンドを実行すると、ブラウザが開き、Agno Platform で認証され、ワークスペースが自動的に構成されます。

### 方法 2: 手動 API キー設定

または、API キーを手動で設定することもできます。

1. [app.agno.com/settings](https://app.agno.com/settings) から API キーを取得します。
2. 環境で API キーを設定します。

<CodeGroup>
```bash Mac
export AGNO_API_KEY=ag-***
```

```bash Windows
setx AGNO_API_KEY ag-***
```
</CodeGroup>
