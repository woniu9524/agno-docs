---
title: 使用 Agno 平台进行身份验证
description: "设置身份验证以开始监控、跟踪性能指标和管理您的 Agno 工作区。"
---

有两种方法可以与 Agno 进行身份验证：

1. 使用 CLI 设置命令 (`ag setup`)
2. 在您的环境中手动设置 API 密钥

### 方法 1：CLI 身份验证

与 Agno 进行身份验证的最快方法是使用 Agno CLI：

```bash
ag setup
```

此命令将在您的浏览器中打开 Agno 平台进行身份验证，并自动配置您的工作区。

### 方法 2：手动设置 API 密钥

或者，您可以手动设置您的 API 密钥：

1. 从 [app.agno.com/settings](https://app.agno.com/settings) 获取您的 API 密钥
2. 在您的环境中设置 API 密钥

<CodeGroup>
```bash Mac
export AGNO_API_KEY=ag-***
```

```bash Windows
setx AGNO_API_KEY ag-***
```
</CodeGroup>
