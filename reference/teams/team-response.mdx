`TeamRunResponse` クラスは、チーム全体の応答と個々のメンバーの応答の両方を含む、チーム実行の応答を表します。ストリーミングをサポートし、チーム実行中にリアルタイムイベントを提供します。

## TeamRunResponse の属性

| 属性 | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `content` | `Any` | `None` | 応答のコンテンツ |
| `content_type` | `str` | `"str"` | コンテンツのデータ型を指定します |
| `thinking` | `str` | `None` | モデルが生成した思考コンテンツ（Anthropic モデルで使用） |
| `messages` | `List[Message]` | `None` | 応答に含まれるメッセージのリスト |
| `metrics` | `Dict[str, Any]` | `None` | 実行の使用状況メトリクス |
| `model` | `str` | `None` | 実行で使用されたモデル |
| `model_provider` | `str` | `None` | 実行で使用されたモデルプロバイダー |
| `member_responses` | `List[Union[TeamRunResponse, RunResponse]]` | `[]` | 個々のチームメンバーからの応答 |
| `run_id` | `str` | `None` | 実行 ID |
| `team_id` | `str` | `None` | 実行のチーム ID |
| `session_id` | `str` | `None` | 実行のセッション ID |
| `tools` | `List[Dict[str, Any]]` | `None` | モデルに提供されたツールのリスト |
| `images` | `List[Image]` | `None` | メンバー実行からの画像のリスト |
| `videos` | `List[Video]` | `None` | ビデオのリスト |
| `audio` | `List[Audio]` | `None` | オーディオスニペットのリスト |
| `response_audio` | `ModelResponseAudio` | `None` | モデルの生のオーディオ応答 |
| `reasoning_content` | `str` | `None` | モデルが生成した推論コンテンツ |
| `citations` | `Citations` | `None` | 応答で使用された引用 |
| `created_at` | `int` | 現在のタイムスタンプ | 応答作成の Unix タイムスタンプ |
| `extra_data` | `RunResponseExtraData` | `None` | `references`、`add_messages`、`history`、`reasoning_steps`、`reasoning_messages` のようなオプションフィールドを含む追加データ |

## TeamRunResponse イベントの種類

`Team.run()` 関数は、チームの設定に応じて次のイベントを送信します。

### コアイベント

| イベントの種類 | 説明 |
|---|---|
| `TeamRunStarted` | チーム実行の開始を示します |
| `TeamRunResponseContent` | モデルの応答テキストを個別のチャンクとして含みます |
| `TeamRunCompleted` | チーム実行の正常な完了を示します |
| `TeamRunError` | チーム実行中にエラーが発生したことを示します |
| `TeamRunCancelled` | チーム実行がキャンセルされたことを示します |

### ツールイベント

| イベントの種類 | 説明 |
|---|---|
| `TeamToolCallStarted` | ツール呼び出しの開始を示します |
| `TeamToolCallCompleted` | ツール呼び出しの完了（ツール呼び出しの結果を含む）を示します |

### 推論イベント

| イベントの種類 | 説明 |
|---|---|
| `TeamReasoningStarted` | チームの推論プロセスの開始を示します |
| `TeamReasoningStep` | 推論プロセスの単一ステップを含みます |
| `TeamReasoningCompleted` | 推論プロセスの完了を示します |

### メモリイベント

| イベントの種類 | 説明 |
|---|---|
| `TeamMemoryUpdateStarted` | チームがメモリを更新していることを示します |
| `TeamMemoryUpdateCompleted` | メモリ更新の完了を示します |

## イベント属性

### BaseTeamRunResponseEvent

すべてのイベントは `BaseTeamRunResponseEvent` を継承しており、以下の共通属性を提供します。

| 属性 | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `created_at` | `int` | 現在のタイムスタンプ | イベント作成の Unix タイムスタンプ |
| `event` | `str` | イベントタイプの値 | イベントの種類 |
| `team_id` | `str` | `""` | イベントを生成したチームの ID |
| `run_id` | `Optional[str]` | `None` | 現在の実行の ID |
| `session_id` | `Optional[str]` | `None` | 現在のセッションの ID |
| `content` | `Optional[Any]` | `None` | 後方互換性のため |
