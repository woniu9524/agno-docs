## RunResponse の属性

| 属性 | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `content` | `Any` | `None` | レスポンスの内容。 |
| `content_type` | `str` | `"str"` | コンテンツのデータ型を指定します。 |
| `thinking` | `str` | `None` | モデルが生成した思考内容（Anthropic モデルで使用されます）。 |
| `reasoning_content` | `str` | `None` | モデルが生成した推論内容。 |
| `messages` | `List[Message]` | `None` | レスポンスに含まれるメッセージのリスト。 |
| `metrics` | `Dict[str, Any]` | `None` | 実行の使用状況メトリクス。 |
| `model` | `str` | `None` | 実行で使用されたモデル。 |
| `model_provider` | `str` | `None` | 実行で使用されたモデルプロバイダー。 |
| `run_id` | `str` | `None` | 実行 ID。 |
| `agent_id` | `str` | `None` | 実行のエージェント ID。 |
| `session_id` | `str` | `None` | 実行のセッション ID。 |
| `tools` | `List[Dict[str, Any]]` | `None` | モデルに提供されたツールのリスト。 |
| `images` | `List[Image]` | `None` | モデルが生成した画像のリスト。 |
| `videos` | `List[Video]` | `None` | モデルが生成したビデオのリスト。 |
| `audio` | `List[Audio]` | `None` | モデルが生成したオーディオスニペットのリスト。 |
| `response_audio` | `ModelResponseAudio` | `None` | モデルの生のオーディオレスポンス。 |
| `citations` | `Citations` | `None` | レスポンスで使用された引用。 |
| `created_at` | `int` | - | レスポンス作成のUnixタイムスタンプ。 |
| `extra_data` | `RunResponseExtraData` | `None` | `references`、`add_messages`、`history`、`reasoning_steps`、`reasoning_messages` のようなオプションフィールドを含む追加データ。 |

## RunResponseEvent のタイプと属性

### 基本的な RunResponseEvent の属性

すべてのイベントは `BaseAgentRunResponseEvent` を継承しており、以下の共通属性を提供します。

| 属性 | タイプ | デフォルト | 説明 |
|---|---|---|---|
| `created_at` | `int` | 現在のタイムスタンプ | イベント作成のUnixタイムスタンプ |
| `event` | `str` | イベントタイプの値 | イベントのタイプ |
| `agent_id` | `str` | `""` | イベントを生成したエージェントのID |
| `run_id` | `Optional[str]` | `None` | 現在の実行のID |
| `session_id` | `Optional[str]` | `None` | 現在のセッションのID |
| `content` | `Optional[Any]` | `None` | 後方互換性のためのもの |
