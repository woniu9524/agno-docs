Agno は、エージェントと対話するための美しく、完全にオープンソースで、無料で利用および拡張可能な UI を提供します。これは、エージェントとチャットしたり、そのメモリ、知識などを表示したりできるシンプルなインターフェースです。

<Note>
[agno.com](https://app.agno.com) にはデータは送信されません。すべてのエージェントデータは、ローカルの sqlite データベースに保存されます。
</Note>

オープンソースのエージェント UI は、Next.js と TypeScript で構築されています。[Agent Playground](/introduction/playground) の成功の後、コミュニティからセルフホスト可能な代替手段を求める声があり、それに応えました！

# Agent UI の始め方

Agent UI をクローンするには、ターミナルで次のコマンドを実行します。

```bash
npx create-agent-ui@latest
```

新しいプロジェクトを作成するには `y` を入力し、依存関係をインストールしてから、次のコマンドで agent-ui を実行します。

```bash
cd agent-ui && npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いて Agent UI を表示しますが、ローカルのエージェントに接続することを忘れないでください。

## ローカルエージェントへの接続

Agent UI は、ローカルまたは任意のクラウドプロバイダーで実行できるプレイグラウンドサーバーに接続する必要があります。

まずローカルのプレイグラウンドサーバーから始めましょう。`playground.py` というファイルを作成します。

```python playground.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.playground import Playground
from agno.storage.sqlite import SqliteStorage
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.yfinance import YFinanceTools

agent_storage: str = "tmp/agents.db"

web_agent = Agent(
    name="Web Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    instructions=["Always include sources"],
    # Store the agent sessions in a sqlite database
    storage=SqliteStorage(table_name="web_agent", db_file=agent_storage),
    # Adds the current date and time to the instructions
    add_datetime_to_instructions=True,
    # Adds the history of the conversation to the messages
    add_history_to_messages=True,
    # Number of history responses to add to the messages
    num_history_responses=5,
    # Adds markdown formatting to the messages
    markdown=True,
)

finance_agent = Agent(
    name="Finance Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True)],
    instructions=["Always use tables to display data"],
    storage=SqliteStorage(table_name="finance_agent", db_file=agent_storage),
    add_datetime_to_instructions=True,
    add_history_to_messages=True,
    num_history_responses=5,
    markdown=True,
)

playground = Playground(agents=[web_agent, finance_agent])
app = playground.get_app()

if __name__ == "__main__":
    playground.serve("playground:app", reload=True)
```

別のターミナルで、プレイグラウンドサーバーを実行します。

<Steps>
  <Step title="仮想環境をセットアップする">

    <CodeGroup>
    ```bash Mac
    python3 -m venv .venv
    source .venv/bin/activate
    ```

    ```bash Windows
    python3 -m venv aienv
    aienv/scripts/activate
    ```
    </CodeGroup>

  </Step>
  <Step title="依存関係をインストールする">

    <CodeGroup>
    ```bash Mac
    pip install -U openai duckduckgo-search yfinance sqlalchemy 'fastapi[standard]' agno
    ```

    ```bash Windows
    pip install -U openai duckduckgo-search yfinance sqlalchemy 'fastapi[standard]' agno
    ```
    </CodeGroup>

  </Step>
  <Step title="OpenAI キーをエクスポートする">

    <CodeGroup>
    ```bash Mac
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```
    </CodeGroup>

  </Step>
  <Step title="プレイグラウンドを実行する">

    ```shell
    python playground.py
    ```

  </Step>
</Steps>

<Tip>`serve_playground_app()` が `Playground` アプリを含むファイルを参照していることを確認してください。</Tip>
