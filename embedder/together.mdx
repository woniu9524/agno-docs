`TogetherEmbedder` は、Together API を使用してテキストデータをベクトルに埋め込むために使用できます。Together は OpenAI API 仕様を使用しているため、`TogetherEmbedder` クラスは `OpenAIEmbedder` クラスに似ており、Together プラットフォームとの互換性を確保するための調整が組み込まれています。キーは[こちら](https://api.together.xyz/settings/api-keys)から取得してください。

## 使用方法

```python cookbook/embedders/together_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.together import TogetherEmbedder

# データベースに文章を埋め込む
embeddings = TogetherEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 埋め込みをナレッジベースで使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="together_embeddings",
        embedder=TogetherEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ    | タイプ  | デフォルト                               | 説明                                                              |
| ------------ | ----- | ---------------------------------------- | ----------------------------------------------------------------- |
| `model`      | `str` | `"nomic-ai/nomic-embed-text-v1.5"`       | 埋め込みの生成に使用されるモデルの名前。                                |
| `dimensions` | `int` | `768`                                    | モデルによって生成される埋め込みの次元数。                                |
| `api_key`    | `str` |                                          | リクエストの認証に使用される API キー。                                 |
| `base_url`   | `str` | `"https://api.Together.ai/inference/v1"` | API エンドポイントのベース URL。                                      |
