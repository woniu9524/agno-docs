`HuggingfaceCustomEmbedder` クラスは、Hugging Face API を使用してテキストデータをベクトルに埋め込むために使用されます。API キーは[こちら](https://huggingface.co/settings/tokens)から取得できます。

## 使用方法

```python cookbook/embedders/huggingface_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.huggingface import HuggingfaceCustomEmbedder

# データベースに文を埋め込む
embeddings = HuggingfaceCustomEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 知識ベースで埋め込みを使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="huggingface_embeddings",
        embedder=HuggingfaceCustomEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ            | タイプ                       | デフォルト            | 説明                                       |
| -------------------- | -------------------------- | ------------------ | ------------------------------------------ |
| `dimensions`         | `int`                      | -                  | 生成される埋め込みの次元数                 |
| `model`              | `str`                      | `all-MiniLM-L6-v2` | 使用する HuggingFace モデルの名前          |
| `api_key`            | `str`                      | -                  | リクエストの認証に使用される API キー      |
| `client_params`      | `Optional[Dict[str, Any]]` | -                  | HuggingFace クライアントのオプションのパラメータ辞書 |
| `huggingface_client` | `Any`                      | -                  | 事前に設定された HuggingFace クライアントインスタンス (オプション) |
