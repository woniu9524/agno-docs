Agno は、ベクトルデータベースのデフォルトのエンベッダーとして `OpenAIEmbedder` を使用します。`OpenAIEmbedder` クラスは、OpenAI API を使用してテキストデータをベクトルに埋め込むために使用されます。キーは[こちら](https://platform.openai.com/api-keys)から取得してください。

## Usage

```python cookbook/embedders/openai_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.openai import OpenAIEmbedder

# Embed sentence in database
embeddings = OpenAIEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# Print the embeddings and their dimensions
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# Use an embedder in a knowledge base
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="openai_embeddings",
        embedder=OpenAIEmbedder(),
    ),
    num_documents=2,
)
```

## Params

| Parameter         | Type                         | Default                    | Description                                                                      |
| ----------------- | ---------------------------- | -------------------------- | -------------------------------------------------------------------------------- |
| `model`           | `str`                        | `"text-embedding-ada-002"` | エンベディング生成に使用されるモデルの名前。                            |
| `dimensions`      | `int`                        | `1536`                     | モデルによって生成されるエンベディングの次元数。                     |
| `encoding_format` | `Literal['float', 'base64']` | `"float"`                  | エンベディングがエンコードされる形式。「float」または「base64」のいずれかを選択できます。 |
| `user`            | `str`                        | -                          | API リクエストに関連付けられたユーザー。                                        |
| `api_key`         | `str`                        | -                          | リクエストの認証に使用される API キー。                                    |
| `organization`    | `str`                        | -                          | API リクエストに関連付けられた組織。                                |
| `base_url`        | `str`                        | -                          | API エンドポイントのベース URL。                                               |
| `request_params`  | `Optional[Dict[str, Any]]`   | -                          | API リクエストに含める追加のパラメータ。                             |
| `client_params`   | `Optional[Dict[str, Any]]`   | -                          | API クライアントの設定に使用する追加のパラメータ。                            |
| `openai_client`   | `Optional[OpenAIClient]`     | -                          | API リクエストを行うために使用する OpenAIClient のインスタンス。                  |
