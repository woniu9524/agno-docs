`OllamaEmbedder` は、Ollama を使用してテキストデータをローカルでベクトルに埋め込むために使用できます。

<Note>埋め込みを生成するために使用されるモデルはローカルで実行する必要があります。この場合は `openhermes` なので、[ `ollama` をインストール](https://ollama.com/download) し、ターミナルで `ollama pull openhermes` を実行する必要があります。</Note>

## Usage

```python cookbook/embedders/ollama_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.ollama import OllamaEmbedder

# Embed sentence in database
embeddings = OllamaEmbedder(id="openhermes").get_embedding("The quick brown fox jumps over the lazy dog.")

# Print the embeddings and their dimensions
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# Use an embedder in a knowledge base
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="ollama_embeddings",
        embedder=OllamaEmbedder(),
    ),
    num_documents=2,
)
```

## Params

| Parameter       | Type                       | Default        | Description                                                               |
| --------------- | -------------------------- | -------------- | ------------------------------------------------------------------------- |
| `model`         | `str`                      | `"openhermes"` | 埋め込み生成に使用されるモデルの名前。                                      |
| `dimensions`    | `int`                      | `4096`         | モデルによって生成された埋め込みの次元数。                                  |
| `host`          | `str`                      | -              | API エンドポイントのホストアドレス。                                      |
| `timeout`       | `Any`                      | -              | API リクエストのタイムアウト期間。                                        |
| `options`       | `Any`                      | -              | API リクエストの構成に追加のオプション。                                  |
| `client_kwargs` | `Optional[Dict[str, Any]]` | -              | API クライアントの構成に追加のキーワード引数。オプション。                  |
| `ollama_client` | `Optional[OllamaClient]`   | -              | API リクエストを行うために使用される OllamaClient のインスタンス。オプション。 |
