`VoyageAIEmbedder` クラスは、Voyage AI API を使用してテキストデータをベクトルに埋め込むために使用されます。キーは[こちら](https://dash.voyageai.com/api-keys)から取得してください。

## 使用方法

```python cookbook/embedders/voyageai_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.voyageai import VoyageAIEmbedder

# データベースに文を埋め込む
embeddings = VoyageAIEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 知識ベースで埋め込みを使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="voyageai_embeddings",
        embedder=VoyageAIEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ        | タイプ                       | デフォルト                                 | 説明                                                         |
| ---------------- | -------------------------- | ------------------------------------------ | ------------------------------------------------------------ |
| `model`          | `str`                      | `"voyage-2"`                               | 埋め込みの生成に使用されるモデルの名前。                         |
| `dimensions`     | `int`                      | `1024`                                     | モデルによって生成される埋め込みの次元数。                       |
| `request_params` | `Optional[Dict[str, Any]]` | -                                          | API リクエストに含める追加のパラメータ。オプション。             |
| `api_key`        | `str`                      | -                                          | リクエストの認証に使用される API キー。                        |
| `base_url`       | `str`                      | `"https://api.voyageai.com/v1/embeddings"` | API エンドポイントのベース URL。                               |
| `max_retries`    | `Optional[int]`            | -                                          | API リクエストのリトライ最大回数。オプション。                   |
| `timeout`        | `Optional[float]`          | -                                          | API リクエストのタイムアウト期間。オプション。                   |
| `client_params`  | `Optional[Dict[str, Any]]` | -                                          | API クライアントの設定に使用する追加のパラメータ。オプション。   |
| `voyage_client`  | `Optional[Client]`         | -                                          | API リクエストを行うために使用する Client のインスタンス。オプション。 |
