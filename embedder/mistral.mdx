`MistralEmbedder` クラスは、Mistral API を使用してテキストデータをベクトルに埋め込むために使用されます。キーは[こちら](https://console.mistral.ai/api-keys/)から取得してください。

## 使用方法

```python cookbook/embedders/mistral_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.mistral import MistralEmbedder

# 文章をデータベースに埋め込む
embeddings = MistralEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元数を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 知識ベースで埋め込みを使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="mistral_embeddings",
        embedder=MistralEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ        | タイプ                       | デフォルト           | 説明                                                                |
| ----------------- | -------------------------- | ----------------- | -------------------------------------------------------------------------- |
| `model`           | `str`                      | `"mistral-embed"` | 埋め込み生成に使用されるモデルの名前。                      |
| `dimensions`      | `int`                      | `1024`            | モデルによって生成される埋め込みの次元数。               |
| `request_params`  | `Optional[Dict[str, Any]]` | -                 | API リクエストに含める追加パラメータ。オプション。             |
| `api_key`         | `str`                      | -                 | リクエストの認証に使用される API キー。                              |
| `endpoint`        | `str`                      | -                 | API リクエストのエンドポイント URL。                                     |
| `max_retries`     | `Optional[int]`            | -                 | API リクエストの最大リトライ回数。オプション。                  |
| `timeout`         | `Optional[int]`            | -                 | API リクエストのタイムアウト期間。オプション。                           |
| `client_params`   | `Optional[Dict[str, Any]]` | -                 | API クライアントの設定に使用する追加パラメータ。オプション。 |
| `mistral_client`  | `Optional[MistralClient]`  | -                 | API リクエストを行うために使用する MistralClient インスタンス。オプション。 |
