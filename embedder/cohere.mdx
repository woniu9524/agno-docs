`CohereEmbedder` クラスは、Cohere API を使用してテキストデータをベクトルに埋め込むために使用されます。Cohere は[こちら](https://docs.cohere.com/reference/about)から利用を開始できます。

キーは[こちら](https://dashboard.cohere.com/api-keys)から取得してください。

## 使用方法

```python cookbook/embedders/cohere_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.cohere import CohereEmbedder

# データベースに埋め込みを追加
embeddings = CohereEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")
# 埋め込みとその次元を出力
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 埋め込みを知識ベースで使用
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="cohere_embeddings",
        embedder=CohereEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ        | タイプ                       | デフォルト           | 説明                                                                |
| ---------------- | -------------------------- | ----------------- | -------------------------------------------------------------------------- |
| `model`          | `str`                      | `"embed-english-v3.0"` | 埋め込みの生成に使用されるモデルの名前。                 |
| `input_type`     | `str`                      | `search_query`    | 埋め込む入力のタイプ。[こちら](https://docs.cohere.com/docs/embeddings#the-input_type-parameter)で詳細を確認できます。 |
| `embedding_types`| `Optional[List[str]]`      | -                 | 生成する埋め込みのタイプ。オプションです。                              |
| `api_key`        | `str`                      | -                 | リクエストの認証に使用される Cohere API キー。                              |
| `request_params` | `Optional[Dict[str, Any]]` | -                 | API リクエストに含める追加のパラメータ。オプションです。             |
| `client_params`  | `Optional[Dict[str, Any]]` | -                 | API クライアントの設定に使用する追加のパラメータ。オプションです。             |
| `cohere_client`  | `Optional[CohereClient]`   | -                 | API リクエストの作成に使用する CohereClient のインスタンス。オプションです。   |
