---
title: Embedderとは何ですか？
sidebarTitle: 概要
description: AgnoでEmbedderを使用して、複雑な情報をベクトル表現に変換する方法を学びます。
---

Embedderは、複雑な情報をベクトル表現に変換し、それをベクトルデータベースに保存できるようにします。データを埋め込みに変換することで、Embedderは文脈的に関連性の高い情報を効率的に検索および取得できるようにします。このプロセスは、言語モデルに必要なビジネスコンテキストを提供することで、その応答を強化し、文脈を認識できるようにします。AgnoはデフォルトのEmbedderとして`OpenAIEmbedder`を使用しますが、他のEmbedderもサポートされています。以下に例を示します。

```python
from agno.agent import Agent, AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.openai import OpenAIEmbedder

# ナレッジベースを作成
knowledge_base=AgentKnowledge(
    vector_db=PgVector(
        db_url=db_url,
        table_name=embeddings_table,
        embedder=OpenAIEmbedder(),
    ),
    # 2つの参照がプロンプトに追加されます
    num_documents=2,
),

# ナレッジベースに情報を追加
knowledge_base.load_text("The sky is blue")

# ナレッジベースをAgentに追加
agent = Agent(knowledge_base=knowledge_base)
```
