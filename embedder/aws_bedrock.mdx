`AwsBedrockEmbedder` クラスは、AWS Bedrock API を使用してテキストデータをベクトルに埋め込むために使用されます。デフォルトでは、Cohere Embed Multilingual V3 モデルを使用して埋め込みを生成します。

# セットアップ

## AWS の認証情報を設定する

```bash
export AWS_ACCESS_KEY_ID = xxx
export AWS_SECRET_ACCESS_KEY = xxx
export AWS_REGION = xxx
```

<Note>
デフォルトでは、この埋め込み機能は `cohere.embed-multilingual-v3` モデルを使用します。この埋め込み機能を使用する前に、AWS Bedrock モデルカタログからこのモデルへのアクセスを有効にする必要があります。
</Note>

## PgVector を実行する
```bash
docker run - d \
    - e POSTGRES_DB = ai \
    - e POSTGRES_USER = ai \
    - e POSTGRES_PASSWORD = ai \
    - e PGDATA = /var/lib/postgresql/data/pgdata \
    - v pgvolume: / var/lib/postgresql/data \
    - p 5532: 5432 \
    - -name pgvector \
    agnohq/pgvector: 16
```

# 使用方法

```python cookbook/embedders/aws_bedrock_embedder.py

# データベースに文章を埋め込む
embeddings = AwsBedrockEmbedder().get_embedding(
    "The quick brown fox jumps over the lazy dog."
)
# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# PDF ナレッジベースを使用した使用例
knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    reader=PDFUrlReader(
        chunk_size=2048
    ),  # Cohere モデルの固定サイズが 2048 であるため必須
    vector_db=PgVector(
        table_name="recipes",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        embedder=AwsBedrockEmbedder(),
    ),
)
knowledge_base.load(recreate=False)
```

# パラメータ

| パラメータ | タイプ | デフォルト | 説明 |
| --------------------------------------- | --------------------------------------------- | --------------------------------------------- | ------------------------------------------------------------------------------------------ |
| `id` | `str` | `"cohere.embed-multilingual-v3"` | 使用するモデル ID。AWS Bedrock モデルカタログでこのモデルを有効にする必要があります。 |
| `dimensions` | `int` | `1024` | モデルによって生成される埋め込みの次元数 (Cohere モデルの場合は 1024)。 |
| `input_type` | `str` | `"search_query"` | タイプを区別するために特殊トークンを前に付けます。オプション: 'search_document', 'search_query', 'classification', 'clustering'。 |
| `truncate` | `Optional[str]` | `None` | 最大トークン長を超える入力の処理方法。オプション: 'NONE', 'START', 'END'。 |
| `embedding_types` | `Optional[List[str]]` | `None` | 返される埋め込みのタイプ。オプション: 'float', 'int8', 'uint8', 'binary', 'ubinary'。 |
| `aws_region` | `Optional[str]` | `None` | 使用する AWS リージョン。指定しない場合は、AWS_REGION 環境変数にフォールバックします。 |
| `aws_access_key_id` | `Optional[str]` | `None` | AWS アクセスキー ID。指定しない場合は、AWS_ACCESS_KEY_ID 環境変数にフォールバックします。 |
| `aws_secret_access_key` | `Optional[str]` | `None` | AWS シークレットアクセスキー。指定しない場合は、AWS_SECRET_ACCESS_KEY 環境変数にフォールバックします。 |
| `session` | `Optional[Session]` | `None` | 認証に使用する boto3 Session オブジェクト。 |
| `request_params` | `Optional[Dict[str, Any]]` | `None` | API リクエストに渡す追加のパラメータ。 |
| `client_params` | `Optional[Dict[str, Any]]` | `None` | boto3 クライアントに渡す追加のパラメータ。 |
| `client` | `Optional[AwsClient]` | `None` | API リクエストを行うために使用する AWS Bedrock クライアントのインスタンス。 |
