`FastEmbedEmbedder` クラスは、[FastEmbed](https://qdrant.github.io/fastembed/) を使用してテキストデータをベクトルに埋め込むために使用されます。

## 使用方法

```python cookbook/embedders/qdrant_fastembed.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.fastembed import FastEmbedEmbedder

# データベース内の文を埋め込む
embeddings = FastEmbedEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 埋め込みモジュールをナレッジベースで使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="qdrant_embeddings",
        embedder=FastEmbedEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ    | タイプ  | デフォルト                  | 説明                                       |
| ------------ | ----- | ------------------------ | ------------------------------------------ |
| `dimensions` | `int` | -                        | 生成される埋め込みの次元数                     |
| `model`      | `str` | `BAAI/bge-small-en-v1.5` | 使用する qdrant_fastembed モデルの名前 |
