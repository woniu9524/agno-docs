`GeminiEmbedder` クラスは、Gemini API を使用してテキストデータをベクトルに埋め込むために使用されます。このクラスは[こちら](https://ai.google.dev/aistudio)から取得できます。

## 使用方法

```python cookbook/embedders/gemini_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.google import GeminiEmbedder

# データベース内の文を埋め込む
embeddings = GeminiEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 知識ベースで埋め込みを使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="gemini_embeddings",
        embedder=GeminiEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ        | タイプ                       | デフォルト                     | 説明                                                 |
| ---------------- | -------------------------- | --------------------------- | ----------------------------------------------------------- |
| `dimensions`     | `int`                      | `768`                       | 生成される埋め込みの次元数                                             |
| `model`          | `str`                      | `models/text-embedding-004` | 使用する Gemini モデルの名前                                         |
| `task_type`      | `str`                      | -                           | 埋め込みが生成されるタスクの種類                                     |
| `title`          | `str`                      | -                           | 埋め込みタスクのオプションのタイトル                                     |
| `api_key`        | `str`                      | -                           | リクエストの認証に使用される API キー                                     |
| `request_params` | `Optional[Dict[str, Any]]` | -                           | 埋め込みリクエストのパラメータのオプションの辞書                               |
| `client_params`  | `Optional[Dict[str, Any]]` | -                           | Gemini クライアントのパラメータのオプションの辞書                               |
| `gemini_client`  | `Optional[Client]`         | -                           | 事前に設定された Gemini クライアントインスタンスのオプション                               |
