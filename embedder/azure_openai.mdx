`AzureOpenAIEmbedder` クラスは、Azure OpenAI API を使用してテキストデータをベクトルに埋め込むために使用されます。キーは[こちら](https://ai.azure.com/)から取得してください。

## セットアップ

### API キーの設定

```bash
export AZURE_EMBEDDER_OPENAI_API_KEY=xxx
export AZURE_EMBEDDER_OPENAI_ENDPOINT=xxx
export AZURE_EMBEDDER_DEPLOYMENT=xxx
```

### PgVector の実行
```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agnohq/pgvector:16
```

## 使用方法

```python cookbook/embedders/azure_embedder.py
from agno.agent import AgentKnowledge
from agno.vectordb.pgvector import PgVector
from agno.embedder.azure_openai import AzureOpenAIEmbedder

# データベースに文を埋め込む
embeddings = AzureOpenAIEmbedder().get_embedding("The quick brown fox jumps over the lazy dog.")

# 埋め込みとその次元を出力する
print(f"Embeddings: {embeddings[:5]}")
print(f"Dimensions: {len(embeddings)}")

# 埋め込みを知識ベースで使用する
knowledge_base = AgentKnowledge(
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="azure_openai_embeddings",
        embedder=AzureOpenAIEmbedder(),
    ),
    num_documents=2,
)
```

## パラメータ

| パラメータ                 | タイプ                          | デフォルト                    | 説明                                                               |
| ------------------------- | ----------------------------- | -------------------------- | ------------------------------------------------------------------ |
| `model`                   | `str`                         | `"text-embedding-ada-002"` | 埋め込みの生成に使用されるモデルの名前。                                 |
| `dimensions`              | `int`                         | `1536`                     | モデルによって生成される埋め込みの次元数。                               |
| `encoding_format`         | `Literal['float', 'base64']`  | `"float"`                  | 埋め込みがエンコードされる形式。「float」または「base64」のオプションがあります。 |
| `user`                    | `str`                         | -                          | API リクエストに関連付けられたユーザー。                                 |
| `api_key`                 | `str`                         | -                          | リクエストの認証に使用される API キー。                                  |
| `api_version`             | `str`                         | `"2024-02-01"`             | リクエストに使用する API のバージョン。                                  |
| `azure_endpoint`          | `str`                         | -                          | API リクエストの Azure エンドポイント。                                |
| `azure_deployment`        | `str`                         | -                          | API リクエストの Azure デプロイメント名。                                |
| `base_url`                | `str`                         | -                          | API エンドポイントのベース URL。                                       |
| `azure_ad_token`          | `str`                         | -                          | 認証用の Azure Active Directory トークン。                               |
| `azure_ad_token_provider` | `Any`                         | -                          | Azure AD トークンを取得するためのプロバイダー。                            |
| `organization`            | `str`                         | -                          | API リクエストに関連付けられた組織。                                   |
| `request_params`          | `Optional[Dict[str, Any]]`    | -                          | API リクエストに含める追加のパラメータ。オプション。                         |
| `client_params`           | `Optional[Dict[str, Any]]`    | -                          | API クライアントの設定に使用する追加のパラメータ。オプション。                   |
| `openai_client`           | `Optional[AzureOpenAIClient]` | -                          | API リクエストを行うために使用する AzureOpenAIClient のインスタンス。オプション。 |
