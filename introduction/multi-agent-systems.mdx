## レベル 4: 推論と協調ができるエージェントチーム

エージェントは作業の原子単位であり、スコープが狭く、少数のツールを持つ場合に最も効果的に機能します。ツールの数がモデルが処理できる量を超えた場合や、複数の概念を処理する必要がある場合は、エージェントチームを使用して負荷を分散します。

Agno は、推論、協調、調整ができるエージェントチームを構築できる、業界をリードするマルチエージェントアーキテクチャを提供します。この例では、半導体市場のパフォーマンスを分析するために 2 つのエージェントのチームを構築し、ステップバイステップで推論を行います。

```python level_4_team.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.reasoning import ReasoningTools
from agno.tools.yfinance import YFinanceTools

web_agent = Agent(
    name="Web Search Agent",
    role="Handle web search requests and general research",
    model=OpenAIChat(id="gpt-4.1"),
    tools=[DuckDuckGoTools()],
    instructions="Always include sources",
    add_datetime_to_instructions=True,
)

finance_agent = Agent(
    name="Finance Agent",
    role="Handle financial data requests and market analysis",
    model=OpenAIChat(id="gpt-4.1"),
    tools=[YFinanceTools(stock_price=True, stock_fundamentals=True,analyst_recommendations=True, company_info=True)],
    instructions=[
        "Use tables to display stock prices, fundamentals (P/E, Market Cap), and recommendations.",
        "Clearly state the company name and ticker symbol.",
        "Focus on delivering actionable financial insights.",
    ],
    add_datetime_to_instructions=True,
)

reasoning_finance_team = Team(
    name="Reasoning Finance Team",
    mode="coordinate",
    model=Claude(id="claude-sonnet-4-20250514"),
    members=[web_agent, finance_agent],
    tools=[ReasoningTools(add_instructions=True)],
    instructions=[
        "Collaborate to provide comprehensive financial and investment insights",
        "Consider both fundamental analysis and market sentiment",
        "Use tables and charts to display data clearly and professionally",
        "Present findings in a structured, easy-to-follow format",
        "Only output the final consolidated analysis, not individual agent responses",
    ],
    markdown=True,
    show_members_responses=True,
    enable_agentic_context=True,
    add_datetime_to_instructions=True,
    success_criteria="The team has provided a complete financial analysis with data, visualizations, risk assessment, and actionable investment recommendations supported by quantitative analysis and market research.",
)

if __name__ == "__main__":
    reasoning_finance_team.print_response("""Compare the tech sector giants (AAPL, GOOGL, MSFT) performance:
        1. Get financial data for all three companies
        2. Analyze recent news affecting the tech sector
        3. Calculate comparative metrics and correlations
        4. Recommend portfolio allocation weights""",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
```

依存関係をインストールしてエージェントチームを実行する

## レベル 5: 状態と決定論を備えたエージェントワークフロー

ワークフローは、本番アプリケーション用に構築された、決定論的でステートフルなマルチエージェントプログラムです。ワークフローは純粋な Python で記述されるため、実行フローに対して極めて高い制御が可能です。

数百ものエージェントシステムを構築してきた経験から、**どのようなフレームワークやステップベースのアプローチも、純粋な Python の柔軟性と信頼性には及びません**。ループが必要な場合は while/for を使用し、条件分岐が必要な場合は if/else を使用し、例外処理が必要な場合は try/except を使用します。


<Check>

ワークフローロジックは Python 関数であるため、AI コードエディタはワークフローのコードを生成できます。

`https://docs.agno.com` をドキュメントソースとして追加し、自由にコーディングしてください。

</Check>

ここでは、以前の出力をキャッシュする簡単なワークフローを示します。キャッシュする内容、ストリームする内容、ログに記録する内容、返す内容をすべて制御できます。

```python level_5_workflow.py
from typing import Iterator
from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat
from agno.utils.log import logger
from agno.utils.pprint import pprint_run_response
from agno.workflow import Workflow


class CacheWorkflow(Workflow):
    # エージェントまたはチームをワークフローの属性として追加します
    agent = Agent(model=OpenAIChat(id="gpt-4o-mini"))

    # `run()` メソッドでロジックを記述します
    def run(self, message: str) -> Iterator[RunResponse]:
        logger.info(f"Checking cache for '{message}'")
        # 出力が既にキャッシュされているか確認します
        if self.session_state.get(message):
            logger.info(f"Cache hit for '{message}'")
            yield RunResponse(
                run_id=self.run_id, content=self.session_state.get(message)
            )
            return

        logger.info(f"Cache miss for '{message}'")
        # エージェントを実行し、レスポンスを yield します
        yield from self.agent.run(message, stream=True)

        # レスポンスが yield された後に、出力をキャッシュします
        self.session_state[message] = self.agent.run_response.content


if __name__ == "__main__":
    workflow = CacheWorkflow()
    # ワークフローを実行します (これには約 1 秒かかります)
    response: Iterator[RunResponse] = workflow.run(message="Tell me a joke.")
    # レスポンスを出力します
    pprint_run_response(response, markdown=True, show_time=True)
    # ワークフローを再度実行します (キャッシュにより即座に実行されます)
    response: Iterator[RunResponse] = workflow.run(message="Tell me a joke.")
    # レスポンスを出力します
    pprint_run_response(response, markdown=True, show_time=True)
```

ワークフローを実行する

```shell
python level_5_workflow.py
```
