Playgroundは、豊富な機能を持つエージェントシステムをテストするための堅牢なインターフェースを提供します。

- **ストリーミングサポート**: リアルタイムの応答ストリーミングと中間状態をユーザーに返します。

- **セッション履歴**: Playground内で会話履歴を視覚化します。

- **ユーザーメモリ**: 会話全体にわたるユーザーの詳細と好みを視覚化します。

- **設定**: エージェントのパラメータ、モデル設定、ツール設定を表示できる包括的な設定インターフェース。

- **推論サポート**: Playgroundインターフェースに表示される詳細な推論トレースを組み込みサポート。

- **ヒューマン・イン・ザ・ループ・サポート**: 特別な人間の監視と承認によるエージェントワークフローでの手動介入を有効にします。

- **マルチモーダルサポート**: テキスト、画像、音声、その他のメディアタイプの処理と生成をサポートします。

- **マルチエージェントシステム**: マルチエージェントチームとワークフローをサポートします。

## ローカルでエージェントと対話する

<Steps>

<Step title="サンプルコードを含むファイルを作成する">



```python playground.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.playground import Playground
from agno.storage.sqlite import SqliteStorage
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.yfinance import YFinanceTools

agent_storage: str = "tmp/agents.db"

web_agent = Agent(
    name="Web Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    instructions=["Always include sources"],
    # Store the agent sessions in a sqlite database
    storage=SqliteStorage(table_name="web_agent", db_file=agent_storage),
    # Adds the current date and time to the instructions
    add_datetime_to_instructions=True,
    # Adds the history of the conversation to the messages
    add_history_to_messages=True,
    # Number of history responses to add to the messages
    num_history_responses=5,
    # Adds markdown formatting to the messages
    markdown=True,
)

finance_agent = Agent(
    name="Finance Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True)],
    instructions=["Always use tables to display data"],
    storage=SqliteStorage(table_name="finance_agent", db_file=agent_storage),
    add_datetime_to_instructions=True,
    add_history_to_messages=True,
    num_history_responses=5,
    markdown=True,
)

playground_app = Playground(agents=[web_agent, finance_agent])
app = playground_app.get_app()

if __name__ == "__main__":
    playground_app.serve("playground:app", reload=True)
```

Playgroundアプリケーションを実行する前に、`OPENAI_API_KEY`をエクスポートしてください。

<Tip>`serve()`が`Playground`アプリを含むファイルを参照していることを確認してください。</Tip>

</Step>

<Step title="Agnoで認証する">

ローカルアプリケーションがagnoにどのポートを実行しているかを知らせるために、[agno.com](https://app.agno.com)で認証してください。

認証方法については、[認証ガイド](how-to/authentication)をご覧ください。

<Note>
agno.comにデータは送信されません。すべてのエージェントデータは、ローカルのsqliteデータベースに保存されます。
</Note>



</Step>

<Step title="Playgroundサーバーを実行する">

依存関係をインストールし、Playgroundサーバーを実行します。

```shell
pip install openai duckduckgo-search yfinance sqlalchemy 'fastapi[standard]' agno

python playground.py
```

</Step>

<Step title="Playgroundを表示する">

- 提供されたリンクを開くか、`http://app.agno.com/playground`に移動します（ログインが必要です）。
- `localhost:7777`エンドポイントを追加/選択して、エージェントとのチャットを開始してください！

### Agent UIの開始方法

```bash
# 新しいAgent UIプロジェクトを作成する
npx create-agent-ui@latest

# またはクローンして手動で実行する
git clone https://github.com/agno-agi/agent-ui.git
cd agent-ui && pnpm install && pnpm dev
```

UIはデフォルトで`localhost:7777`に接続し、上記のPlaygroundセットアップと一致します。詳細については[GitHub](https://github.com/agno-agi/agent-ui)をご覧ください。
