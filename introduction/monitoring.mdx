# モニタリング

[app.agno.com](https://app.agno.com) でエージェントをリアルタイムで追跡できます。

## 認証
セッションの監視を開始するには、[agno.com](https://app.agno.com) で認証してください。
認証方法については、[認証ガイド](how-to/authentication) を参照してください。

## モニタリングの有効化

`AGNO_MONITOR=true` を設定して、単一のエージェントまたはすべてのエージェントに対してグローバルに監視を有効にします。

### 特定のエージェントの場合

```python
agent = Agent(markdown=True, monitoring=True)
```

### すべてのエージェントに対してグローバルに

```bash
export AGNO_MONITOR=true
```

## エージェントの監視

エージェントを実行し、[セッションページ](https://app.agno.com/sessions) でセッションを表示します。

## デバッグログ

システムプロンプト、ユーザーメッセージ、ツール呼び出しを確認したいですか？

Agno には、ターミナルにデバッグログを出力する組み込みデバッガーが含まれています。いずれかのアージェントで `debug_mode=True` を設定するか、環境で `AGNO_DEBUG=true` を設定します。

```python debug_logs.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[YFinanceTools(stock_price=True)],
    instructions="Use tables to display data. Don't include any other text.",
    markdown=True,
    debug_mode=True,
)
agent.print_response("What is the stock price of Apple?", stream=True)
```

エージェントを実行して、ターミナルでデバッグログを表示します。

```shell
python debug_logs.py
```
