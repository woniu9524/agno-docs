传统软件遵循预先编程的步骤顺序。Agent 使用机器学习**模型**动态确定其行动方案，其核心组件包括：

- **模型 (Model):** 控制执行流程。它决定是进行推理、采取行动还是响应。
- **工具 (Tools):** 使 Agent 能够采取行动并与外部系统进行交互。
- **指令 (Instructions):** 这是我们编程 Agent 的方式，教它如何使用工具和响应。

Agent 还拥有**记忆 (memory)**、**知识 (knowledge)**、**存储 (storage)** 和**推理 (reasoning)** 能力：

- **推理 (Reasoning):** 使 Agent 能够在响应前“思考”并“分析”其行动（即工具调用）的结果，这提高了响应的可靠性和质量。
- **知识 (Knowledge):** 是 Agent 在**运行时搜索**以做出更好决策和提供准确响应的领域特定信息（RAG）。知识存储在向量数据库中，这种**运行时搜索**模式被称为 Agentic RAG/Agentic Search。
- **存储 (Storage):** Agent 用于将会话历史和状态保存在数据库中。模型 API 是无状态的，存储使我们能够从中断的地方继续对话。这使得 Agent 具有状态，从而能够进行多轮、长期的对话。
- **记忆 (Memory):** 使 Agent 能够存储和回忆先前交互中的信息，从而允许它们学习用户偏好并个性化其响应。

<Check>让我们构建几个 Agent 来看看它们是如何工作的。</Check>

## Level 1: 带有工具和指令的 Agent

最简单的 Agent 包含一个模型、一个工具和指令。让我们构建一个可以使用 `yfinance` 库获取数据的 Agent，并附带显示结果的表格指令。

```python level_1_agent.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[YFinanceTools(stock_price=True)],
    instructions="使用表格显示数据。不要包含任何其他文本。",
    markdown=True,
)
agent.print_response("苹果的股价是多少？", stream=True)
```

## Level 2: 带有知识和存储的 Agent

**知识 (Knowledge):** 虽然模型拥有大量的训练数据，但我们几乎总是需要提供领域特定的信息来做出更好的决策并提供准确的响应（RAG）。我们将此信息存储在向量数据库中，并允许 Agent 在运行时进行**搜索**。

**存储 (Storage):** 模型 API 是无状态的，`Storage` 驱动程序将聊天历史和状态保存到数据库中。当 Agent 运行时，它会从数据库中读取聊天历史和状态，并将其添加到消息列表中，从而恢复对话并使 Agent 具有状态。

在此示例中，我们将使用：
- `UrlKnowledge` 将 Agno 文档加载到 LanceDB 中，使用 OpenAI 进行嵌入。
- `SqliteStorage` 将 Agent 的会话历史和状态保存在数据库中。

```python level_2_agent.py
from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.url import UrlKnowledge
from agno.models.anthropic import Claude
from agno.storage.sqlite import SqliteStorage
from agno.vectordb.lancedb import LanceDb, SearchType

# 将 Agno 文档加载到知识库中
# 你也可以使用 `https://docs.agno.com/llms-full.txt` 获取完整文档
knowledge = UrlKnowledge(
    urls=["https://docs.agno.com/introduction.md"],
    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="agno_docs",
        search_type=SearchType.hybrid,
        # 使用 OpenAI 进行嵌入
        embedder=OpenAIEmbedder(id="text-embedding-3-small", dimensions=1536),
    ),
)

# 将 Agent 会话存储在 SQLite 数据库中
storage = SqliteStorage(table_name="agent_sessions", db_file="tmp/agent.db")

agent = Agent(
    name="Agno Assist",
    model=Claude(id="claude-sonnet-4-20250514"),
    instructions=[
        "回答问题前先搜索你的知识。",
        "只在你的响应中包含输出。不要包含其他文本。",
    ],
    knowledge=knowledge,
    storage=storage,
    add_datetime_to_instructions=True,
    # 将聊天历史添加到消息中
    add_history_to_messages=True,
    # 历史运行次数
    num_history_runs=3,
    markdown=True,
)

if __name__ == "__main__":
    # 加载知识库，首次运行后注释掉此行
    # 如果需要重新创建知识库，请将 recreate 设置为 True
    agent.knowledge.load(recreate=False)
    agent.print_response("什么是 Agno？", stream=True)
```

## Level 3: 带有记忆和推理的 Agent

- **推理 (Reasoning):** 使 Agent 能够**“思考”和“分析”**，提高可靠性和质量。`ReasoningTools` 是提高 Agent 响应质量的最佳方法之一。
- **记忆 (Memory):** 使 Agent 能够分类、存储和回忆用户偏好，从而个性化其响应。记忆帮助 Agent 构建个性并从先前的交互中学习。

```python level_3_agent.py
from agno.agent import Agent
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory
from agno.models.anthropic import Claude
from agno.tools.reasoning import ReasoningTools
from agno.tools.yfinance import YFinanceTools

memory = Memory(
    # 使用任何模型来创建和管理记忆
    model=Claude(id="claude-sonnet-4-20250514"),
    # 将记忆存储在 SQLite 数据库中
    db=SqliteMemoryDb(table_name="user_memories", db_file="tmp/agent.db"),
    # 我们默认禁用删除，如果需要请启用
    delete_memories=True,
    clear_memories=True,
)

agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[
        ReasoningTools(add_instructions=True),
        YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True),
    ],
    # 用于存储记忆的用户 ID，如果未提供则为 `default`
    user_id="ava",
    instructions=[
        "使用表格显示数据。",
        "在你的响应中包含来源。",
        "只在你的响应中包含报告。不要包含其他文本。",
    ],
    memory=memory,
    # 让 Agent 管理其记忆
    enable_agentic_memory=True,
    markdown=True,
)

if __name__ == "__main__":
    # 这将创建一个记忆，即 "ava" 最喜欢的股票是 NVIDIA 和 TSLA
    agent.print_response(
        "我最喜欢的股票是 NVIDIA 和 TSLA",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
    # 这将使用记忆来回答问题
    agent.print_response(
        "你能比较一下我最喜欢的股票吗？",
        stream=True,
        show_full_reasoning=True,
        stream_intermediate_steps=True,
    )
```

运行 Agent

```shell
python level_3_agent.py
```

<Tip>你可以单独使用 `Memory` 和 `Reasoning`，不必同时使用它们。</Tip>
