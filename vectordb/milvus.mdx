## セットアップ

```shell
pip install pymilvus
```

## Milvus の初期化

URI とトークンを Milvus サーバー用に設定します。

- 小規模なデータやプロトタイピングのためにローカルのベクトルデータベースが必要なだけであれば、URI をローカルファイル（例: `./milvus.db`）として設定するのが最も便利な方法です。これにより、すべてのデータがこのファイルに自動的に格納されるように [Milvus Lite](https://milvus.io/docs/milvus_lite.md) が利用されます。
- 大規模なデータ（例えば、100 万ベクトル以上）がある場合は、[Docker または Kubernetes](https://milvus.io/docs/quickstart.md) 上によりパフォーマンスの高い Milvus サーバーをセットアップできます。
  このセットアップでは、URI としてサーバーのアドレスとポートを使用してください（例: `http://localhost:19530`）。Milvus で認証機能を有効にしている場合は、トークンとして `your_username:your_password` を使用してください。それ以外の場合は、トークンを設定しないでください。
- [Zilliz Cloud](https://zilliz.com/cloud)（Milvus のフルマネージドクラウドサービス）を使用する場合は、Zilliz Cloud の [Public Endpoint および API キー](https://docs.zilliz.com/docs/on-zilliz-cloud-console#cluster-details) に対応する `uri` と `token` を調整してください。

## 例

```python agent_with_knowledge.py
from agno.agent import Agent
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.milvus import Milvus

vector_db = Milvus(
    collection="recipes",
    uri="./milvus.db",
)
# ナレッジベースを作成
knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=vector_db,
)

knowledge_base.load(recreate=False)  # 初回実行後にコメントアウトしてください

# エージェントを作成して使用
agent = Agent(knowledge=knowledge_base, use_tools=True, show_tool_calls=True)
agent.print_response("How to make Tom Kha Gai", markdown=True)
agent.print_response("What was my last question?", stream=True)
```

    ```python async_milvus_db.py
    # pymilvus をインストール - `pip install pymilvus`
    import asyncio

    from agno.agent import Agent
    from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
    from agno.vectordb.milvus import Milvus

    # ローカルファイルで Milvus を初期化
    vector_db = Milvus(
        collection="recipes",
        uri="tmp/milvus.db",  # ファイルベースのローカルストレージ用
    )

    # ナレッジベースを作成
    knowledge_base = PDFUrlKnowledgeBase(
        urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
        vector_db=vector_db,
    )

    # ナレッジベースでエージェントを作成
    agent = Agent(knowledge=knowledge_base)

    if __name__ == "__main__":
        # ナレッジベースを非同期でロード
        asyncio.run(knowledge_base.aload(recreate=False))  # 初回実行後にコメントアウトしてください

        # エージェントに非同期でクエリを実行
        asyncio.run(agent.aprint_response("How to make Tom Kha Gai", markdown=True))
    ```
    
    <Tip className="mt-4">
      高スループットアプリケーションでノンブロッキング操作を行うには、<code>asyncio.run()</code> と共に <code>aload()</code> および <code>aprint_response()</code> メソッドを使用してください。
    </Tip>
