## セットアップ

接続文字列を取得するには、[Azure Cosmos DB セットアップ ガイド](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/vcore)の手順に従ってください。

MongoDB パッケージをインストールします。

```shell
pip install "pymongo[srv]"
```

## 例

```python agent_with_knowledge.py
import urllib.parse
from agno.agent import Agent
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.mongodb import MongoDb

# Azure Cosmos DB MongoDB 接続文字列
"""
接続文字列の例:
"mongodb+srv://<username>:<encoded_password>@cluster0.mongocluster.cosmos.azure.com/?tls=true&authMechanism=SCRAM-SHA-256&retrywrites=false&maxIdleTimeMS=120000"
"""
mdb_connection_string = f"mongodb+srv://<username>:<encoded_password>@cluster0.mongocluster.cosmos.azure.com/?tls=true&authMechanism=SCRAM-SHA-256&retrywrites=false&maxIdleTimeMS=120000"

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=MongoDb(
        collection_name="recipes",
        db_url=mdb_connection_string,
        search_index_name="recipes",
        cosmos_compatibility=True,
    ),
)

# 初回実行後にコメントアウト
knowledge_base.load(recreate=True)

# エージェントを作成して使用する
agent = Agent(knowledge=knowledge_base, show_tool_calls=True)
agent.print_response("タイカレーの作り方を教えてください。", markdown=True)
```

## MongoDB パラメータ

- `collection_name`: データベース内のコレクションの名前。
- `db_url`: MongoDB データベースへの接続文字列。
- `search_index_name`: 使用する検索インデックスの名前。
- `cosmos_compatibility`: Azure Cosmos DB との互換性のために `True` に設定します。
