## セットアップ

### ローカルセットアップ (Docker)

Dockerを使用してCouchbaseをローカルで実行します。

```shell
docker run -d --name couchbase-server \
  -p 8091-8096:8091-8096 \
  -p 11210:11210 \
  -e COUCHBASE_ADMINISTRATOR_USERNAME=Administrator \
  -e COUCHBASE_ADMINISTRATOR_PASSWORD=password \
  couchbase:latest
```

1. http://localhost:8091 でCouchbase UIにアクセスします。
2. ユーザー名: `Administrator`、パスワード: `password` でログインします。
3. `recipe_bucket` という名前のバケット、`recipe_scope` という名前のスコープ、`recipes` という名前のコレクションを作成します。

### マネージドセットアップ (Capella)

マネージドクラスターの場合は、[Couchbase Capella](https://cloud.couchbase.com/) を使用します。
- CapellaのUIに従って、データベース、バケット、スコープ、コレクションを作成します。

### 環境変数

環境変数を設定します。

```shell
export COUCHBASE_USER="Administrator"
export COUCHBASE_PASSWORD="password"
export COUCHBASE_CONNECTION_STRING="couchbase://localhost"
export OPENAI_API_KEY="<your-openai-api-key>"
```

Capellaの場合は、`COUCHBASE_CONNECTION_STRING` をCapellaの接続文字列に設定します。

### 依存関係のインストール

```shell
pip install couchbase
```

## 例

```python agent_with_knowledge.py
import os
import time
from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.couchbase import CouchbaseSearch
from couchbase.options import ClusterOptions, KnownConfigProfiles
from couchbase.auth import PasswordAuthenticator
from couchbase.management.search import SearchIndex

# Couchbase接続設定
username = os.getenv("COUCHBASE_USER")
password = os.getenv("COUCHBASE_PASSWORD")
connection_string = os.getenv("COUCHBASE_CONNECTION_STRING")

# 認証付きクラスターオプションを作成
auth = PasswordAuthenticator(username, password)
cluster_options = ClusterOptions(auth)
cluster_options.apply_profile(KnownConfigProfiles.WanDevelopment)


knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=CouchbaseSearch(
        bucket_name="recipe_bucket",
        scope_name="recipe_scope",
        collection_name="recipes",
        couchbase_connection_string=connection_string,
        cluster_options=cluster_options,
        search_index="vector_search_fts_index",
        embedder=OpenAIEmbedder(
            id="text-embedding-3-large",
            dimensions=3072,
            api_key=os.getenv("OPENAI_API_KEY")
        ),
        wait_until_index_ready=60,
        overwrite=True
    ),
)

# ナレッジベースをロード
knowledge_base.load(recreate=True)

# KVへのベクトルインデックスの同期を待機
time.sleep(20)

# エージェントを作成して使用
agent = Agent(knowledge=knowledge_base, show_tool_calls=True)
agent.print_response("How to make Thai curry?", markdown=True)
```

    ```python async_couchbase.py
    import asyncio
    import os
    import time
    from agno.agent import Agent
    from agno.embedder.openai import OpenAIEmbedder
    from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
    from agno.vectordb.couchbase import CouchbaseSearch
    from couchbase.options import ClusterOptions, KnownConfigProfiles
    from couchbase.auth import PasswordAuthenticator
    from couchbase.management.search import SearchIndex

    # Couchbase接続設定
    username = os.getenv("COUCHBASE_USER")
    password = os.getenv("COUCHBASE_PASSWORD")
    connection_string = os.getenv("COUCHBASE_CONNECTION_STRING")

    # 認証付きクラスターオプションを作成
    auth = PasswordAuthenticator(username, password)
    cluster_options = ClusterOptions(auth)
    cluster_options.apply_profile(KnownConfigProfiles.WanDevelopment)

    knowledge_base = PDFUrlKnowledgeBase(
        urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
        vector_db=CouchbaseSearch(
            bucket_name="recipe_bucket",
            scope_name="recipe_scope",
            collection_name="recipes",
            couchbase_connection_string=connection_string,
            cluster_options=cluster_options,
            search_index="vector_search_fts_index",
            embedder=OpenAIEmbedder(
                id="text-embedding-3-large",
                dimensions=3072,
                api_key=os.getenv("OPENAI_API_KEY")
            ),
            wait_until_index_ready=60,
            overwrite=True
        ),
    )

    # エージェントを作成して使用
    agent = Agent(knowledge=knowledge_base, show_tool_calls=True)

    async def run_agent():
        await knowledge_base.aload(recreate=True)
        time.sleep(5)  # KVへのベクトルインデックスの同期を待機
        await agent.aprint_response("How to make Thai curry?", markdown=True)

    if __name__ == "__main__":
        # 初回実行後にコメントアウトしてください
        asyncio.run(run_agent())
    ```

    <Tip className="mt-4">
      高スループットアプリケーションでノンブロッキング操作を行うには、<code>asyncio.run()</code> と共に <code>aload()</code> および <code>aprint_response()</code> メソッドを使用します。

## 主要な設定に関する注意点

### 接続プロファイル

ローカルおよびクラウドデプロイメントの両方で `KnownConfigProfiles.WanDevelopment` を使用して、ネットワーク遅延とタイムアウトを適切に処理します。
