title: シナリオテスト
---

この例では、エージェントのシミュレーションベースのテストのために[Scenario](https://github.com/langwatch/scenario)フレームワークを使用する方法を説明します。Scenarioを使用すると、エージェント、ユーザーシミュレーター、およびジャッジ間の会話をシミュレートでき、制御された環境でエージェントの動作を簡単にテストおよび評価できます。

> **ヒント:** より高度なシナリオをご覧になりたいですか？[カスタマーサポートシナリオの例](https://github.com/langwatch/create-agent-app/tree/main/agno_example)を確認してください。この例では、ツール呼び出しや高度なシナリオ機能を含む、より複雑なエージェントを紹介しています。

## コード

```python cookbook/agent_concepts/other/scenario_testing.py
import pytest
import scenario
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# Scenarioのデフォルトを設定します（ユーザーシミュレーターとジャッジのモデル）
scenario.configure(default_model="openai/gpt-4.1-mini")

@pytest.mark.agent_test
@pytest.mark.asyncio
async def test_vegetarian_recipe_agent() -> None:
    # 1. エージェントをラップするためのAgentAdapterを定義します
    class VegetarianRecipeAgentAdapter(scenario.AgentAdapter):
        agent: Agent

        def __init__(self) -> None:
            self.agent = Agent(
                model=OpenAIChat(id="gpt-4.1-mini"),
                markdown=True,
                debug_mode=True,
                instructions="あなたはベジタリアンレシピのエージェントです。",
            )

        async def call(self, input: scenario.AgentInput) -> scenario.AgentReturnTypes:
            response = self.agent.run(
                message=input.last_new_user_message_str(), # 最後のユーザーメッセージのみを渡します
                session_id=input.thread_id, # スレッドIDを渡します。これにより、エージェントは履歴を追跡できます
            )
            return response.content

    # 2. シナリオシミュレーションを実行します
    result = await scenario.run(
        name="夕食のレシピリクエスト",
        description="ユーザーはベジタリアンの夕食のアイデアを探しています。",
        agents=[
            VegetarianRecipeAgentAdapter(),
            scenario.UserSimulatorAgent(),
            scenario.JudgeAgent(
                criteria=[
                    "エージェントは2つ以上のフォローアップ質問をしないこと",
                    "エージェントはレシピを生成すること",
                    "レシピには材料リストが含まれていること",
                    "レシピには段階的な調理手順が含まれていること",
                    "レシピはベジタリアンであり、いかなる種類の肉も含まないこと",
                ]
            ),
        ],
    )

    # 3. 結果をアサートして検査します
    assert result.success
```

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="APIキーを設定する">
    ```bash
    export OPENAI_API_KEY=xxx
    export LANGWATCH_API_KEY=xxx # オプション、シミュレーション監視に必要
    ```
  </Step>

  <Step title="ライブラリをインストールする">
    ```bash
    pip install -U openai agno langwatch-scenario pytest pytest-asyncio
    # または
    uv add agno langwatch-scenario openai pytest
    ```
  </Step>

  <Step title="エージェントを実行する">
    ```bash
    pytest cookbook/agent_concepts/other/scenario_testing.py
    ```
  </Step>
