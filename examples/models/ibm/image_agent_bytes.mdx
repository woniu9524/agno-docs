## コード

```python cookbook/models/ibm/watsonx/image_agent_bytes.py
from pathlib import Path

from agno.agent import Agent
from agno.media import Image
from agno.models.ibm import WatsonX
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=WatsonX(id="meta-llama/llama-3-2-11b-vision-instruct"),
    tools=[DuckDuckGoTools()],
    markdown=True,
)

image_path = Path(__file__).parent.joinpath("sample.jpg")

# 画像ファイルをバイトとして読み込む
with open(image_path, "rb") as img_file:
    image_bytes = img_file.read()

agent.print_response(
    "Tell me about this image and give me the latest news about it.",
    images=[
        Image(content=image_bytes),
    ],
    stream=True,
)
```

    <CodeGroup>

    ```bash Mac
    python cookbook/models/ibm/watsonx/image_agent_bytes.py
    ```

    </CodeGroup>
  </Step>

この例では、ビジョン機能を持つIBM WatsonXの使用方法を示しています。ファイルから画像を読み込み、プロンプトと共にモデルに渡します。その後、モデルは画像を分析し、関連情報を提供できます。

注意: この例では、ビジョン対応モデル（`meta-llama/llama-3-2-11b-vision-instruct`）を使用しており、サンプル画像ファイルが必要です。
