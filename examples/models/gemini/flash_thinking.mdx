## コード

```python cookbook/models/google/gemini/flash_thinking_agent.py
from agno.agent import Agent
from agno.models.google import Gemini

task = (
    "3人の宣教師と3人のカニバルが川を渡る必要があります。 "
    "一度に最大2人まで乗れるボートがあります。 "
    "川のどちらかの岸で、カニバルの数が宣教師の数を上回った場合、カニバルは宣教師を食べます。 "
    "6人全員が安全に川を渡るにはどうすればよいですか？段階的な解決策を提供し、解決策をASCII図で示してください"
)

agent = Agent(model=Gemini(id="gemini-2.0-flash-thinking-exp-1219"), markdown=True)
agent.print_response(task, stream=True)
```
