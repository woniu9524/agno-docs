## コード

```python cookbook/models/nebius/basic.py
from agno.agent import Agent
from agno.models.nebius import Nebius

agent = Agent(
    model=Nebius(),
    show_tool_calls=True,
    markdown=True,
    debug_mode=True,
)

# ターミナルに応答を出力します
agent.print_response("write a two sentence horror story")
```
