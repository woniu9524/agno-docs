## 代码

```python cookbook/models/nebius/basic_stream.py
from agno.agent import Agent
from agno.models.nebius import Nebius

agent = Agent(
    model=Nebius(),
    show_tool_calls=True,
    markdown=True,
    debug_mode=True,
)

# 在终端打印响应
agent.print_response("写一个两句话的恐怖故事", stream=True)
```
