## コード

```python cookbook/models/lmstudio/storage.py
from agno.agent import Agent
from agno.models.lmstudio import LMStudio
from agno.storage.postgres import PostgresStorage
from agno.tools.duckduckgo import DuckDuckGoTools

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

agent = Agent(
    model=LMStudio(id="qwen2.5-7b-instruct-1m"),
    storage=PostgresStorage(table_name="agent_sessions", db_url=db_url),
    tools=[DuckDuckGoTools()],
    add_history_to_messages=True,
)
agent.print_response("カナダには何人住んでいますか？")
agent.print_response("国歌の名前は何ですか？")
```

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="LM Studioのインストール">

    LM Studioを[こちら](https://lmstudio.ai/download)からインストールし、使用したいモデルをダウンロードしてください。

  </Step>

  <Step title="ライブラリのインストール">

    ```bash
    pip install -U sqlalchemy psycopg duckduckgo-search agno
    ```

  </Step>

  <Step title="PgVectorの実行">

    ```bash
    docker run -d \
      -e POSTGRES_DB=ai \
      -e POSTGRES_USER=ai \
      -e POSTGRES_PASSWORD=ai \
      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
      agnohq/pgvector:16
    ```

  </Step>

  <Step title="エージェントの実行">

    <CodeGroup>
    ```bash Mac
    python cookbook/models/lmstudio/storage.py
    ```

    ```bash Windows
    python cookbook/models/lmstudio/storage.py
    ```
    </CodeGroup>

  </Step>
</Steps>
