---
title: ファイルアップロード
description: Agno で Anthropic の Files API を使用する方法を学びます。
---

Anthropic の [Files API](https://docs.anthropic.com/en/docs/build-with-claude/files) を使用すると、ファイルをアップロードし、後で他の API 呼び出しで参照できます。
これは、同じフローでファイルが複数回参照される場合に便利です。

## 使用方法

<Steps>
<Step title="ファイルのアップロード">
Anthropic クライアントを初期化し、`client.beta.files.upload` を使用します。

```python
from anthropic import Anthropic

file_path = Path("path/to/your/file.pdf")

client = Anthropic()
uploaded_file = client.beta.files.upload(file=file_path)
```
</Step>
<Step title="Claude モデルの初期化">
`Claude` モデルを初期化する際に、必要なベータヘッダーを渡します。

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-opus-4-20250514",
        default_headers={"anthropic-beta": "files-api-2025-04-14"},
    )
)
```
</Step>
<Step title="ファイルの参照">
これで、Agno エージェントとの対話時にアップロードされたファイルを参照できます。

```python
agent.print_response(
    "Summarize the contents of the attached file.",
    files=[File(external=uploaded_file)],
)
```
</Step>
</Steps>


この機能には、いくつかのストレージ制限があることに注意してください。詳細については、Anthropic の [ドキュメント](https://docs.anthropic.com/en/docs/build-with-claude/files#file-storage-and-limits) を参照してください。


## 動作例

```python cookbook/models/anthropic/pdf_input_file_upload.py
from pathlib import Path

from agno.agent import Agent
from agno.media import File
from agno.models.anthropic import Claude
from agno.utils.media import download_file
from anthropic import Anthropic

pdf_path = Path(__file__).parent.joinpath("ThaiRecipes.pdf")

# download_file 関数を使用してファイルをダウンロードします
download_file(
    "https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf", str(pdf_path)
)

# Anthropic クライアントを初期化します
client = Anthropic()

# ファイルを Anthropic にアップロードします
uploaded_file = client.beta.files.upload(
    file=Path(pdf_path),
)

if uploaded_file is not None:
    agent = Agent(
        model=Claude(
            id="claude-opus-4-20250514",
            default_headers={"anthropic-beta": "files-api-2025-04-14"},
        ),
        markdown=True,
    )

    agent.print_response(
        "Summarize the contents of the attached file.",
        files=[File(external=uploaded_file)],
    )
