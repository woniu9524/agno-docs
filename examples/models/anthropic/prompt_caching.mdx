プロンプトキャッシングは、処理時間とコストを削減するのに役立ちます。同じプロンプトをフロー内で複数回使用する場合は、検討してください。

Anthropicモデルでのプロンプトキャッシングの詳細については、[こちら](https://docs.anthropic.com/en/docs/build-with-claude/prompt-caching)をご覧ください。

## 使用方法

Agnoセットアップでプロンプトキャッシングを使用するには、`Claude`モデルを初期化する際に`cache_system_prompt`引数を渡します。

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-3-5-sonnet-20241022",
        cache_system_prompt=True,
    ),
)
```

プロンプトキャッシングを機能させるには、プロンプトがある程度の長さである必要があることに注意してください。これについては、Anthropicの[ドキュメント](https://docs.anthropic.com/en/docs/build-with-claude/prompt-caching#cache-limitations)で詳しく読むことができます。

## 拡張キャッシュ

Anthropicの拡張キャッシュベータ機能を使用することもできます。これにより、キャッシュ期間が5分から1時間に更新されます。有効にするには、`extended_cache_time`引数と次のベータヘッダーを渡します。

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-3-5-sonnet-20241022",
        default_headers={"anthropic-beta": "extended-cache-ttl-2025-04-11"},
        cache_system_prompt=True,
        extended_cache_time=True,
    ),
)
```

## 動作例

```python cookbook/models/anthropic/prompt_caching_extended.py
from pathlib import Path

from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.utils.media import download_file

# S3から例の大きなシステムメッセージをロードします。このような大きなプロンプトはキャッシングから恩恵を受けるでしょう。
txt_path = Path(__file__).parent.joinpath("system_promt.txt")
download_file(
    "https://agno-public.s3.amazonaws.com/prompts/system_promt.txt",
    str(txt_path),
)
system_message = txt_path.read_text()

agent = Agent(
    model=Claude(
        id="claude-sonnet-4-20250514",
        default_headers={"anthropic-beta": "extended-cache-ttl-2025-04-11"}, # 拡張キャッシュ時間を使用するためにベータヘッダーを設定します
        system_prompt=system_message,
        cache_system_prompt=True,  # Anthropicがシステムプロンプトをキャッシュできるようにプロンプトキャッシングをアクティブ化します
        extended_cache_time=True,  # デフォルトからキャッシュ時間を1時間に延長します
    ),
    system_message=system_message,
    markdown=True,
)


# 初回実行 - キャッシュが作成されます
response = agent.run(
    "RESTとGraphQL APIの違いを例を挙げて説明してください"
)
print(f"初回実行キャッシュ書き込みトークン = {response.metrics['cache_write_tokens']}")  # type: ignore

# 2回目実行 - キャッシュされたシステムプロンプトが使用されます
response = agent.run(
    "クリーンコードの主要な原則は何ですか？また、Pythonでどのように適用しますか？"
)
print(f"2回目実行キャッシュ読み取りトークン = {response.metrics['cached_tokens']}")  # type: ignore
