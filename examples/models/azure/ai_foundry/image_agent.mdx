---
title: 画像入力を持つエージェント
---

Azure AI Foundry は、`Llama-3.2-11B-Vision-Instruct` のようなモデルで画像入力をサポートしています。これを使用して画像を分析し、それに関する情報を取得できます。

## コード

```python cookbook/models/azure/ai_foundry/image_agent.py
from pathlib import Path
from agno.agent import Agent
from agno.media import Image
from agno.models.azure import AzureAIFoundry

agent = Agent(
    model=AzureAIFoundry(id="Llama-3.2-11B-Vision-Instruct"),
    markdown=True,
)

image_path = Path(__file__).parent.joinpath("sample.jpg")

# 画像ファイルをバイトとして読み込む
with open(image_path, "rb") as img_file:
    image_bytes = img_file.read()

agent.print_response(
    "この画像について教えてください。",
    images=[
        Image(content=image_bytes),
    ],
    stream=True,
)
```
