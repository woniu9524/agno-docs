## コード

```python cookbook/models/ollama/storage.py
from agno.agent import Agent
from agno.models.ollama import Ollama
from agno.storage.postgres import PostgresStorage
from agno.tools.duckduckgo import DuckDuckGoTools

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

agent = Agent(
    model=Ollama(id="llama3.1:8b"),
    storage=PostgresStorage(table_name="agent_sessions", db_url=db_url),
    tools=[DuckDuckGoTools()],
    add_history_to_messages=True,
)
agent.print_response("How many people live in Canada?")
agent.print_response("What is their national anthem called?")
```

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="Ollamaのインストール">
    [インストールガイド](https://github.com/ollama/ollama?tab=readme-ov-file#macos)に従って、以下を実行してください:
    ```bash
    ollama pull llama3.1:8b
    ```
  </Step>

  <Step title="ライブラリのインストール">
    ```bash
    pip install -U ollama sqlalchemy psycopg duckduckgo-search agno
    ```
  </Step>

  <Step title="PgVectorの実行">
    ```bash
    docker run -d \
      -e POSTGRES_DB=ai \
      -e POSTGRES_USER=ai \
      -e POSTGRES_PASSWORD=ai \
      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
      agnohq/pgvector:16
    ```
  </Step>

  <Step title="エージェントの実行">
    <CodeGroup>
    ```bash Mac
    python cookbook/models/ollama/storage.py
    ```

    ```bash Windows
    python cookbook/models/ollama/storage.py
    ```
    </CodeGroup>
  </Step>
</Steps>
