## 代码

```python cookbook/models/ollama/knowledge.py
from agno.agent import Agent
from agno.embedder.ollama import OllamaEmbedder
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.models.ollama import Ollama
from agno.vectordb.pgvector import PgVector

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=PgVector(
        table_name="recipes",
        db_url=db_url,
        embedder=OllamaEmbedder(id="llama3.2", dimensions=3072),
    ),
)
knowledge_base.load(recreate=True)  # 首次运行时注释此行

agent = Agent(
    model=Ollama(id="llama3.2"),
    knowledge=knowledge_base,
    show_tool_calls=True,
)
agent.print_response("How to make Thai curry?", markdown=True)
```

  <Step title="安装 Ollama">
    请按照[安装指南](https://github.com/ollama/ollama?tab=readme-ov-file#macos)进行操作并运行：
    ```bash
    ollama pull llama3.2
    ```
  </Step>

  <Step title="安装库">
    ```bash
    pip install -U ollama sqlalchemy pgvector pypdf agno
    ```
  </Step>

  <Step title="运行 PgVector">
    ```bash
    docker run -d \
      -e POSTGRES_DB=ai \
      -e POSTGRES_USER=ai \
      -e POSTGRES_PASSWORD=ai \
      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
      agnohq/pgvector:16
    ```
  </Step>

  <Step title="运行 Agent">
    <CodeGroup>
    ```bash Mac
    python cookbook/models/ollama/knowledge.py
    ```

    ```bash Windows
    python cookbook/models/ollama/knowledge.py
    ```
    </CodeGroup>
  </Step>
</Steps>
