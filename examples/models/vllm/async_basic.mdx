## コード

```python cookbook/models/vllm/async_basic.py
import asyncio

from agno.agent import Agent
from agno.models.vllm import vLLM

agent = Agent(model=vLLM(id="Qwen/Qwen2.5-7B-Instruct"), markdown=True)
asyncio.run(agent.aprint_response("Share a 2 sentence horror story"))
```

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="ライブラリのインストール">
    ```bash
    pip install -U agno openai vllm
    ```
  </Step>

  <Step title="vLLMサーバーの起動">
    ```bash
    vllm serve Qwen/Qwen2.5-7B-Instruct \
        --enable-auto-tool-choice \
        --tool-call-parser hermes \
        --dtype float16 \
        --max-model-len 8192 \
        --gpu-memory-utilization 0.9
    ```
  </Step>

  <Step title="エージェントの実行">
    ```bash
    python cookbook/models/vllm/async_basic.py
    ```
  </Step>
</Steps>
