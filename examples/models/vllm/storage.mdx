## コード

```python cookbook/models/vllm/storage.py
from agno.agent import Agent
from agno.models.vllm import vLLM
from agno.storage.postgres import PostgresStorage
from agno.tools.duckduckgo import DuckDuckGoTools

DB_URL = "postgresql+psycopg://ai:ai@localhost:5532/ai"

agent = Agent(
    model=vLLM(id="Qwen/Qwen2.5-7B-Instruct"),
    storage=PostgresStorage(table_name="agent_sessions", db_url=DB_URL),
    tools=[DuckDuckGoTools()],
    add_history_to_messages=True,
)

agent.print_response("カナダには何人住んでいますか？")
agent.print_response("国歌の名前は何ですか？")
```

<Note>
 Postgresデータベースが実行されていることを確認してください。
</Note>

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="ライブラリのインストール">
    ```bash
    pip install -U agno openai vllm sqlalchemy psycopg[binary] duckduckgo-search
    ```
  </Step>

  <Step title="Postgresデータベースの起動">
    ```bash
    ./cookbook/scripts/run_pgvector.sh
    ```
  </Step>
  
  <Step title="vLLMサーバーの起動">
    ```bash
    vllm serve Qwen/Qwen2.5-7B-Instruct \
        --enable-auto-tool-choice \
        --tool-call-parser hermes \
        --dtype float16 \
        --max-model-len 8192 \
        --gpu-memory-utilization 0.9
    ```
  </Step>

  <Step title="エージェントの実行">
    ```bash
    python cookbook/models/vllm/storage.py
    ```
  </Step>
</Steps>
