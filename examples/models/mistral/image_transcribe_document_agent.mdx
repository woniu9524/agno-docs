## 概要
この例では、Mistral のビジョン機能を使用してドキュメント画像からテキストを書き起こす方法を示します。これは、手書きまたは古いドキュメントの処理に特に役立ちます。

## コード

```python cookbook/models/mistral/image_transcribe_document_agent
"""
このエージェントは、画像から古い手書きのドキュメントを書き起こします。
"""

from agno.agent import Agent
from agno.media import Image
from agno.models.mistral.mistral import MistralChat

agent = Agent(
    model=MistralChat(id="pixtral-12b-2409"),
    markdown=True,
)

# ドキュメント画像を処理して書き起こす
agent.print_response(
    "このドキュメントを書き起こしてください。",
    images=[
        Image(url="https://ciir.cs.umass.edu/irdemo/hw-demo/page_example.jpg"),
    ],
)
```
