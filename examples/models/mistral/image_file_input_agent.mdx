## 概要
この例では、Mistral のビジョンモデルを使用してローカル画像ファイルを処理し、DuckDuckGo 検索と組み合わせて画像コンテンツに関する追加情報を提供する方法を示します。

## コード

```python cookbook/models/mistral/image_file_input_agent.py
from pathlib import Path

from agno.agent import Agent
from agno.media import Image
from agno.models.mistral.mistral import MistralChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=MistralChat(id="pixtral-12b-2409"),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)

# ローカルファイルシステムから画像を読み込む
image_path = Path(__file__).parent.joinpath("sample.jpeg")

# 画像を分析し、関連ニュースを検索する
agent.print_response(
    "この画像について教えてください。また、duckduckgo から最新のニュースを教えてください。",
    images=[
        Image(filepath=image_path),
    ],
    stream=True,
)
```
