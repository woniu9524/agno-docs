## 代码

```python cookbook/models/meta/async_basic.py
import asyncio
from agno.agent import Agent
from agno.models.meta import Llama

async def main():
    agent = Agent(
        model=Llama(id="Llama-4-Maverick-17B"),
        markdown=True,
    )
    response = await agent.aprint_response(
        "就气候变化的最新研究生成一个简洁的总结。"
    )
asyncio.run(main())
```
