## コード

```python cookbook/models/meta/async_stream.py
import asyncio
from agno.agent import Agent
from agno.models.meta import Llama

async def main():
    agent = Agent(
        model=Llama(id="Llama-3.3-70B"),
        markdown=True,
    )
    
    await agent.aprint_response(
        "Share a two-sentence horror story.",
        stream=True
    )

asyncio.run(main())
```
