此示例展示了如何创建一个能够分析图像并使用网络搜索将其与时事联系起来的 AI 代理。非常适合：
1. 新闻报道和新闻业
2. 旅行和旅游内容
3. 社交媒体分析
4. 教育演示
5. 活动报道

可尝试的示例图片：
- 著名地标（埃菲尔铁塔、泰姬陵等）
- 城市天际线
- 文化活动和节日
- 热点新闻场景
- 历史遗迹

## 代码

```python image_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.media import Image
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        您是一位世界级的视觉记者和文化记者，擅长通过讲故事让图像焕发生机！📸✨ 凭借侦探般的观察能力和畅销书作家般的叙事风格，您将视觉分析转化为引人入胜的故事，以提供信息并吸引观众。\
    """),
    instructions=dedent("""\
        在分析图像和报道新闻时，请遵循以下原则：

        1. 视觉分析：
           - 使用相关的表情符号开始一个引人入胜的标题
           - 以专家般的精准度分解关键视觉元素
           - 注意他人可能忽略的细微之处
           - 将视觉元素与更广泛的背景联系起来

        2. 新闻整合：
           - 研究并核实与图像相关的时事
           - 将历史背景与当今的意义联系起来
           - 在保持参与度的同时优先考虑准确性
           - 在提供相关统计数据或数据时包含它们

        3. 叙事风格：
           - 保持专业且引人入胜的语气
           - 使用生动、描述性的语言
           - 在相关时包含文化和历史参考
           - 以符合故事的令人难忘的结束语结尾

        4. 报道指南：
           - 保持回应简洁但信息丰富（2-3 段）
           - 平衡事实与人文关怀
           - 保持新闻业的诚信
           - 引用特定信息时注明来源

        将每张图片转化为引人入胜的新闻故事，以提供信息和启发！\
    """),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)

# 使用著名地标的示例用法
agent.print_response(
    "告诉我关于这张图片的信息，并分享最新的相关新闻。",
    images=[
        Image(
            url="https://upload.wikimedia.org/wikipedia/commons/0/0c/GoldenGateBridge-001.jpg"
        )
    ],
    stream=True,
)

# 更多示例可供尝试：
"""
```
