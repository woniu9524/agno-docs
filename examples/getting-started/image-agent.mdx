この例では、画像を分析し、ウェブ検索を通じて現在の出来事と結びつけることができるAIエージェントの作成方法を示します。以下に最適です：
1. ニュース報道とジャーナリズム
2. 旅行と観光コンテンツ
3. ソーシャルメディア分析
4. 教育プレゼンテーション
5. イベントカバレッジ

試してみる画像例：
- 有名なランドマーク（エッフェル塔、タージマハルなど）
- 都市のスカイライン
- 文化イベントとフェスティバル
- 速報シーン
- 歴史的な場所

## コード

```python image_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.media import Image
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたは、画像を分析し、ストーリーテリングを通じて命を吹き込む才能を持つ、世界クラスのビジュアルジャーナリストであり文化特派員です！📸✨
        探偵のような観察力と、ベストセラー作家のような物語の才能を持ち、ビジュアル分析を、情報を伝え魅了する説得力のあるストーリーに変えます。
    """),
    instructions=dedent("""\
        画像を分析しニュースを報道する際は、以下の原則に従ってください：

        1. ビジュアル分析：
           - 関連する絵文字を使った、注意を引く見出しから始める
           - 主要なビジュアル要素を専門的な精度で分解する
           - 他の人が見逃すような微妙なディテールに気づく
           - ビジュアル要素をより広い文脈に結びつける

        2. ニュース統合：
           - 画像に関連する現在の出来事を調査し、検証する
           - 歴史的文脈と現代的な重要性を結びつける
           - エンゲージメントを維持しながら正確性を優先する
           - 利用可能な場合は、関連する統計またはデータを含める

        3. ストーリーテリングスタイル：
           - プロフェッショナルでありながら魅力的なトーンを維持する
           - 生き生きとした描写的な言葉を使用する
           - 関連する場合は、文化的および歴史的な参照を含める
           - ストーリーに合った記憶に残る締めくくりで終わる

        4. 報道ガイドライン：
           - 回答は簡潔かつ有益に保つ（2〜3段落）
           - 事実と人間的興味のバランスを取る
           - ジャーナリスティックな誠実さを維持する
           - 特定の情報を引用する際は、情報源を明記する

        すべての画像を、情報を提供しインスピレーションを与える説得力のあるニュースストーリーに変えましょう！
    """),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)

# 有名なランドマークを使用した使用例
agent.print_response(
    "この画像について教えてください。関連する最新ニュースも共有してください。",
    images=[
        Image(
            url="https://upload.wikimedia.org/wikipedia/commons/0/0c/GoldenGateBridge-001.jpg"
        )
    ],
    stream=True,
)

# 試してみる追加の例：
"""
探索するサンプルプロンプト：
1. 「この場所の歴史的重要性は何ですか？」
2. 「この場所は時間とともにどのように変化しましたか？」
3. 「ここでどのような文化イベントが開催されますか？」
4. 「建築様式とその影響は何ですか？」
5. 「この地域に影響を与える最近の開発は何ですか？」

分析するサンプル画像URL：
1. エッフェル塔：「https://upload.wikimedia.org/wikipedia/commons/8/85/Tour_Eiffel_Wikimedia_Commons_%28cropped%29.jpg」
2. タージマハル：「https://upload.wikimedia.org/wikipedia/commons/b/bd/Taj_Mahal%2C_Agra%2C_India_edit3.jpg」
3. ゴールデンゲートブリッジ：「https://upload.wikimedia.org/wikipedia/commons/0/0c/GoldenGateBridge-001.jpg」
"""
```
