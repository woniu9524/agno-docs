---
title: ストレージ付きエージェント
---

この例では、キュレーションされたレシピデータベースからの知識と、Web検索機能および永続ストレージを組み合わせたAI料理アシスタントを作成する方法を示します。このエージェントは、本格的なタイ料理レシピのPDFナレッジベースを使用し、必要に応じてWeb検索で情報を補完できます。

試すためのプロンプト例：
- 「本格的なパッタイの作り方を教えてください？」
- 「レッドカレーとグリーンカレーの違いは何ですか？」
- 「ガランガルについて説明し、代替品を教えてもらえますか？」
- 「トムヤムクンスープの歴史について教えてください」
- 「タイの食料庫に不可欠な材料は何ですか？」
- 「タイバジルチキン（パット・グラパオ）の作り方を教えてください？」

## コード

```python agent_with_storage.py
from textwrap import dedent
from typing import List, Optional

import typer
from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.models.openai import OpenAIChat
from agno.storage.sqlite import SqliteStorage
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.vectordb.lancedb import LanceDb, SearchType
from rich import print

agent_knowledge = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=LanceDb(
        uri="tmp/lancedb",
        table_name="recipe_knowledge",
        search_type=SearchType.hybrid,
        embedder=OpenAIEmbedder(id="text-embedding-3-small"),
    ),
)

agent_storage = SqliteStorage(table_name="recipe_agent", db_file="tmp/agents.db")


def recipe_agent(user: str = "user"):
    session_id: Optional[str] = None

    # ユーザーに新しいセッションを開始するか、既存のセッションを続行するかを尋ねます
    new = typer.confirm("新しいセッションを開始しますか？")

    if not new:
        existing_sessions: List[str] = agent_storage.get_all_session_ids(user)
        if len(existing_sessions) > 0:
            session_id = existing_sessions[0]

    agent = Agent(
        user_id=user,
        session_id=session_id,
        model=OpenAIChat(id="gpt-4o"),
        instructions=dedent("""\
            あなたは情熱的で知識豊富なタイ料理のエキスパートです！ 🧑‍🍳
            温かく励みになる料理インストラクター、タイ料理の歴史家、文化大使の組み合わせと考えてください。

            質問に答える際は、次の手順に従ってください：
            1. まず、ナレッジベースで本格的なタイ料理レシピと料理情報 を検索してください。
            2. ナレッジベースの情報が不完全な場合、またはユーザーがWebに適した質問をした場合は、Webを検索してギャップを埋めてください。
            3. ナレッジベースで情報が見つかった場合は、Webを検索する必要はありません。
            4. 本格性を保つために、常にナレッジベースの情報をWeb検索の結果よりも優先してください。
            5. 必要に応じて、Web検索で以下の情報を補完してください：
               - 最新の適応または材料の代替品
               - 文化的な文脈と歴史的背景
               - 追加の料理のヒントとトラブルシューティング

            コミュニケーションスタイル：
            1. 各応答は関連する料理の絵文字で始めてください。
            2. 応答を明確に構造化してください：
               - 簡単な紹介または文脈
               - メインコンテンツ（レシピ、説明、または歴史）
               - プロのヒントまたは文化的な洞察
               - 励みになる結論
            3. レシピには以下を含めてください：
               - 代替品の可能性のある材料のリスト
               - 明確で番号付けされた調理手順
               - 成功のためのヒントと一般的な落とし穴
            4. フレンドリーで励みになる言葉遣いを使用してください。

            特別な機能：
            - タイの馴染みのない食材を説明し、代替案を提案してください。
            - 関連する文化的文脈と伝統を共有してください。
            - さまざまな食事のニーズに適応させるためのヒントを提供してください。
            - 盛り付けの提案と付け合わせを含めてください。

            各応答は、次のような前向きな結びの言葉で締めくくってください：
            - 「Happy cooking! ขอให้อร่อย (お食事をお楽しみください)！」
            - 「May your Thai cooking adventure bring joy!」
            - 「Enjoy your homemade Thai feast!」

            覚えておいてください：
            - ナレッジベースでレシピの本格性を常に確認してください。
            - 情報がWebソースから来ている場合は、明確に示してください。
            - すべてのスキルレベルの家庭料理人を励まし、サポートしてください。
        """),
        storage=agent_storage,
        knowledge=agent_knowledge,
        tools=[DuckDuckGoTools()],
        # ツール呼び出しを応答に表示します
        show_tool_calls=True,
        # エージェントにチャット履歴を提供するために
        # 次のいずれかを行うことができます：
        # 1. エージェントにチャット履歴を読み取るツールを提供します
        # 2. チャット履歴をモデルに送信されるメッセージに自動的に追加します
        #
        # 1. エージェントにチャット履歴を読み取るツールを提供します
        read_chat_history=True,
        # 2. チャット履歴をモデルに送信されるメッセージに自動的に追加します
        # add_history_to_messages=True,
        # メッセージに追加する履歴応答の数。
        # num_history_responses=3,
        markdown=True,
    )

    print("エージェントとチャットを開始します！")
    if session_id is None:
        session_id = agent.session_id
        if session_id is not None:
            print(f"セッション開始: {session_id}\n")
        else:
            print("セッション開始\n")
    else:
        print(f"セッション続行: {session_id}\n")

    # エージェントをコマンドラインアプリケーションとして実行します
    agent.cli_app(markdown=True)


if __name__ == "__main__":
    # ナレッジベースがロードされた後にコメントアウトしてください
    if agent_knowledge is not None:
        agent_knowledge.load()

    typer.run(recipe_agent)
```
