title: Audio Agent
---

この例では、音声入力を処理し、音声応答を生成できるAIエージェントの作成方法を示します。このエージェントは、音声コンテンツの分析から自然な応答の生成まで、さまざまな音声ベースの対話に使用できます。

試してみるべきオーディオインタラクションの例：
- 分析のために会話の録音をアップロードする
- エージェントに音声出力で質問に答えてもらう
- さまざまな言語やアクセントを処理する
- 音声内のトーンや感情を分析する

## コード

```python audio_agent.py
from textwrap import dedent

import requests
from agno.agent import Agent
from agno.media import Audio
from agno.models.openai import OpenAIChat
from agno.utils.audio import write_audio_to_file

# AI音声インタラクションエージェントを作成する
agent = Agent(
    model=OpenAIChat(
        id="gpt-4o-audio-preview",
        modalities=["text", "audio"],
        audio={"voice": "alloy", "format": "wav"},
    ),
    description=dedent("""\
        あなたは音声処理と音声インタラクションの専門家であり、話し言葉の内容を理解・分析し、
        自然で魅力的な音声応答を提供することができます。文脈、感情、話し言葉のニュアンスを理解することに長けています。\
    """),
    instructions=dedent("""\
        音声インタラクションのスペシャリストとして、以下のガイドラインに従ってください：
        1. 音声入力を注意深く聞き、内容と文脈の両方を理解する
        2. 主要なポイントに対処する、明確で簡潔な応答を提供する
        3. 音声応答を生成する際は、自然で会話的なトーンを維持する
        4. 話し手のトーンと感情を分析に考慮する
        5. 音声が不明瞭な場合は、明確にするよう求める

        魅力的で役立つ音声インタラクションを作成することに集中してください！\
    """),
)

# オーディオファイルを取得し、base64エンコードされた文字列に変換する
url = "https://openaiassets.blob.core.windows.net/$web/API/docs/audio/alloy.wav"
response = requests.get(url)
response.raise_for_status()

# 音声を処理し、応答を取得する
agent.run(
    "この録音には何が含まれていますか？内容とトーンを分析してください。",
    audio=[Audio(content=response.content, format="wav")],
)

# 音声応答が利用可能な場合は保存する
if agent.run_response.response_audio is not None:
    write_audio_to_file(
        audio=agent.run_response.response_audio.content, filename="tmp/response.wav"
    )

# 試してみるべきその他のインタラクション例：
"""
これらの音声インタラクションシナリオを試してください：
1. 「この録音で話し合われた主なポイントを要約してもらえますか？」
2. 「話し手の声からどのような感情やトーンを検出しましたか？」
3. 「話し方のパターンと明瞭さの詳細な分析を提供してもらえますか？」
4. 「背景ノイズやオーディオ品質の問題を特定できますか？」
5. 「この録音の全体的な文脈と目的は何ですか？」

注意：独自のオーディオファイルは、base64形式に変換して使用できます。
独自のオーディオファイルを使用する場合の例：

with open('your_audio.wav', 'rb') as audio_file:
    audio_data = audio_file.read()
    agent.run("このオーディオを分析してください", audio=[Audio(content=audio_data, format="wav")])
"""
```
