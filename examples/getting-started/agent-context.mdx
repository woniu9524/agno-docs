title: Agent Context
---

この例では、外部依存関係をエージェントに注入する方法を示します。コンテキストはエージェントの実行時に評価され、エージェントの依存関係注入のように機能します。

試してみるプロンプトの例：
- "HackerNewsのトップ記事を要約して"
- "現在、どのようなテクノロジーに関する議論がトレンドになっていますか？"
- "現在のトップ記事を分析し、トレンドを特定してください"
- "今日の最も多くのアップボートを獲得した記事は何ですか？"

## コード

```python agent_context.py
import json
from textwrap import dedent

import httpx
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def get_top_hackernews_stories(num_stories: int = 5) -> str:
    """HackerNewsからトップ記事を取得して返します。

    Args:
        num_stories: 取得するトップ記事の数（デフォルト：5）
    Returns:
        記事の詳細（タイトル、URL、スコアなど）を含むJSON文字列
    """
    # トップ記事を取得
    stories = [
        {
            k: v
            for k, v in httpx.get(
                f"https://hacker-news.firebaseio.com/v0/item/{id}.json"
            )
            .json()
            .items()
            if k != "kids"  # ディスカッショントレッドを除外
        }
        for id in httpx.get(
            "https://hacker-news.firebaseio.com/v0/topstories.json"
        ).json()[:num_stories]
    ]
    return json.dumps(stories, indent=4)


# リアルタイムのHackerNewsデータにアクセスできるコンテキスト認識エージェントを作成
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    # コンテキスト内の各関数はエージェントの実行時に評価されます。
    # エージェントの依存関係注入と考えてください
    context={"top_hackernews_stories": get_top_hackernews_stories},
    # add_contextは自動的にコンテキストをユーザーメッセージに追加します
    # add_context=True,
    # または、指示に手動でコンテキストを追加することもできます
    instructions=dedent("""\
        あなたは洞察力のあるテクノトレンドオブザーバーです！📰

        HackerNewsのトップ記事はこちらです：
        {top_hackernews_stories}\
    """),
    markdown=True,
)

# 使用例
agent.print_response(
    "HackerNewsのトップ記事を要約し、興味深いトレンドを特定してください。",
    stream=True,
)
```
