title: Video Generation
---

この例では、ModelsLabs を使用してビデオを生成する AI エージェントの作成方法を示します。
このエージェントを使用して、アニメーションシーンから創造的なビジュアルストーリーまで、さまざまな種類の短いビデオを作成できます。

試してみるプロンプトの例：
- 「夕暮れのビーチに打ち寄せる波の穏やかなビデオを作成してください」
- 「魔法の森を飛ぶ蝶の魔法のようなビデオを生成してください」
- 「庭に咲く花のタイムラプスを作成してください」
- 「夜空で踊るオーロラのビデオを生成してください」

## コード

```python video_generation.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.models_labs import ModelsLabTools

# クリエイティブな AI ビデオディレクターエージェントを作成
video_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[ModelsLabTools()],
    description=dedent("""\
        あなたは、自然のシーンから芸術的なアニメーションまで、さまざまなビデオスタイルに精通した経験豊富な AI ビデオディレクターです。
        モーション、タイミング、ビデオコンテンツを通じたビジュアルストーリーテリングに関する深い理解を持っています。\
    """),
    instructions=dedent("""\
        AI ビデオディレクターとして、以下のガイドラインに従ってください：
        1. ユーザーのリクエストを注意深く分析し、希望するスタイルとムードを理解する
        2. 生成する前に、モーション、タイミング、雰囲気に関する詳細を追加してプロンプトを強化する
        3. `generate_media` ツールを、詳細でよく作成されたプロンプトで使用する
        4. 作成したクリエイティブな選択について簡単な説明を提供する
        5. リクエストが不明確な場合は、スタイルの好みについて明確化を求める

        ビデオは、応答の下の UI に自動的に表示されます。
        常にユーザーのビジョンを実現する、魅力的で意味のあるビデオを作成することを目指してください！\
    """),
    markdown=True,
    show_tool_calls=True,
)

# 使用例
video_agent.print_response(
    "カラフルな星雲を巡る宇宙の旅を生成してください", stream=True
)

# 生成されたビデオを取得して表示
videos = video_agent.get_videos()
if videos:
    for video in videos:
        print(f"生成されたビデオの URL: {video.url}")

# 試してみる追加のプロンプトの例：

"""
```
