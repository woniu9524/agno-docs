---
title: 结构化输出
---

此示例展示了如何使用结构化输出来生成格式良好的电影剧本概念。它展示了两种方法：
1. JSON 模式：传统的 JSON 响应解析
2. 结构化输出：增强的结构化数据处理

可尝试的示例提示：
- "东京" - 获取一个设定在未来日本的高科技惊悚片
- "古罗马" - 体验一部史诗般的历史剧
- "曼哈顿" - 探索一部现代浪漫喜剧
- "亚马逊雨林" - 在异国情调的地点冒险
- "火星殖民地" - 太空定居点中的科幻故事

## 代码

```python structured_output.py
from textwrap import dedent
from typing import List

from agno.agent import Agent, RunResponse  # noqa
from agno.models.openai import OpenAIChat
from pydantic import BaseModel, Field


class MovieScript(BaseModel):
    setting: str = Field(
        ...,
        description="电影主要地点的丰富细节、氛围描述和时代背景。包含感官细节和情绪。",
    )
    ending: str = Field(
        ...,
        description="电影强有力的结局，将所有情节线索联系起来。应带来情感冲击和满足感。",
    )
    genre: str = Field(
        ...,
        description="电影的主要和次要类型（例如，“科幻惊悚片”，“浪漫喜剧”）。应与设定和基调保持一致。",
    )
    name: str = Field(
        ...,
        description="一个引人注目、令人难忘的标题，捕捉故事的精髓并吸引目标受众。",
    )
    characters: List[str] = Field(
        ...,
        description="4-6 位主要角色，具有独特的姓名和简短的角色描述（例如，“Sarah Chen - 拥有黑暗秘密的杰出量子物理学家”）。",
    )
    storyline: str = Field(
        ...,
        description="一个引人入胜的三句话情节摘要：铺垫、冲突和风险。用悬念和情感吸引读者。",
    )


# 使用 JSON 模式的 Agent
json_mode_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        你是一位享有盛誉的好莱坞编剧，以创作令人难忘的卖座大片而闻名！🎬
        融合了克里斯托弗·诺兰、艾伦·索金和昆汀·塔伦蒂诺的叙事功力，
        你创作出能吸引全球观众的独特故事。

        你的专长是将地点转化为鲜活的、有生命的、驱动叙事的角色。\
    """),
    instructions=dedent("""\
        在创作电影概念时，请遵循以下原则：

        1. 地点即角色：
           - 用感官细节让地点栩栩如生
           - 包含影响故事的氛围元素
           - 考虑时代背景对叙事的影响

        2. 角色发展：
           - 为每个角色赋予独特的嗓音和明确的动机
           - 创造引人入胜的关系和冲突
           - 确保多元化代表和真实背景

        3. 故事结构：
           - 以引人入胜的开头开始
           - 通过不断升级的冲突来建立紧张感
           - 交付出人意料但又合乎情理的结局

        4. 类型精通：
           - 在拥抱类型惯例的同时加入新颖的转折
           - 周到地混合类型以创造独特的组合
           - 在整个过程中保持一致的基调

        将每个地点转化为令人难忘的电影体验！\
    """),
    response_model=MovieScript,
)

# 使用结构化输出的 Agent
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        你是一位享有盛誉的好莱坞编剧，以创作令人难忘的卖座大片而闻名！🎬
        融合了克里斯托弗·诺兰、艾伦·索金和昆汀·塔伦蒂诺的叙事功力，
        你创作出能吸引全球观众的独特故事。

        你的专长是将地点转化为鲜活的、有生命的、驱动叙事的角色。\
    """),
    instructions=dedent("""\
        在创作电影概念时，请遵循以下原则：

        1. 地点即角色：
           - 用感官细节让地点栩栩如生
           - 包含影响故事的氛围元素
           - 考虑时代背景对叙事的影响

        2. 角色发展：
           - 为每个角色赋予独特的嗓音和明确的动机
           - 创造引人入胜的关系和冲突
           - 确保多元化代表和真实背景

        3. 故事结构：
           - 以引人入胜的开头开始
           - 通过不断升级的冲突来建立紧张感
           - 交付出人意料但又合乎情理的结局

        4. 类型精通：
           - 在拥抱类型惯例的同时加入新颖的转折
           - 周到地混合类型以创造独特的组合
           - 在整个过程中保持一致的基调

        将每个地点转化为令人难忘的电影体验！\
    """),
    response_model=MovieScript,
)

# 使用不同地点的示例用法
json_mode_agent.print_response("东京", stream=True)
structured_output_agent.print_response("古罗马", stream=True)
```
