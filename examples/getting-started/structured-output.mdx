title: Structured Output
---

この例では、AIエージェントで構造化された出力を利用して、フォーマットの整った映画の脚本コンセプトを生成する方法を示します。ここでは2つのアプローチを紹介します。
1. JSONモード：従来のJSONレスポンス解析
2. 構造化出力：強化された構造化データハンドリング

試してみる例：
- "東京" - 未来の日本を舞台にしたハイテクスリラーを取得
- "古代ローマ" - 壮大な歴史ドラマを体験
- "マンハッタン" - 現代のロマンティックコメディを探求
- "アマゾン熱帯雨林" - エキゾチックな場所での冒険
- "火星コロニー" - 宇宙開拓地でのサイエンスフィクション

## コード

```python structured_output.py
from textwrap import dedent
from typing import List

from agno.agent import Agent, RunResponse  # noqa
from agno.models.openai import OpenAIChat
from pydantic import BaseModel, Field


class MovieScript(BaseModel):
    setting: str = Field(
        ...,
        description="映画の主要な場所と時代設定を豊かで詳細な雰囲気で描写する。感覚的なディテールとムードを含める。",
    )
    ending: str = Field(
        ...,
        description="すべてのプロットの糸をまとめる、映画の力強い結末。感情的なインパクトと満足感を与えるべきである。",
    )
    genre: str = Field(
        ...,
        description="映画の主要なジャンルと二次的なジャンル（例：「SFスリラー」、「ロマンティックコメディ」）。設定とトーンに合わせるべきである。",
    )
    name: str = Field(
        ...,
        description="ストーリーの本質を捉え、ターゲットオーディエンスにアピールする、注意を引き、記憶に残るタイトル。",
    )
    characters: List[str] = Field(
        ...,
        description="特徴的な名前と簡単な役割の説明を持つ4〜6人の主要キャラクター（例：「サラ・チェン - 秘密を抱えた優秀な量子物理学者」）。",
    )
    storyline: str = Field(
        ...,
        description="説得力のある3文のプロット概要：セットアップ、葛藤、および賭け。読者を引き込むような興味と感情で。",
    )


# JSONモードを使用するエージェント
json_mode_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたは忘れられない大ヒット作を生み出すことで知られる、高く評価されているハリウッドの脚本家です！🎬
        クリストファー・ノーラン、アーロン・ソーキン、クエンティン・タランティーノのストーリーテリングの才能を組み合わせ、
        世界中の観客を魅了するユニークなストーリーを創作します。

        あなたの専門は、場所を物語を動かす生きた、息づくキャラクターに変えることです。\
    """),
    instructions=dedent("""\
        映画のコンセプトを創作する際は、以下の原則に従ってください：

        1. 設定はキャラクターであるべき：
           - 場所を感覚的なディテールで生き生きとさせる
           - 物語に影響を与える雰囲気の要素を含める
           - 時代設定が物語に与える影響を考慮する

        2. キャラクター開発：
           - 各キャラクターにユニークな声と明確な動機を与える
           - 説得力のある関係性と葛藤を生み出す
           - 多様な表現と本物の背景を確保する

        3. ストーリー構造：
           - 注意を引くフックで始める
           - エスカレートする葛藤を通して緊張感を高める
           - 驚くべきが、必然的な結末を提供する

        4. ジャンルの習熟：
           - ジャンルの慣習を受け入れつつ、新鮮なひねりを加える
           - ユニークな組み合わせのためにジャンルを慎重にミックスする
           - 全体を通して一貫したトーンを維持する

        すべての場所を忘れられない映画体験に変えましょう！\
    """),
    response_model=MovieScript,
)

# 構造化出力を使用するエージェント
structured_output_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたは忘れられない大ヒット作を生み出すことで知られる、高く評価されているハリウッドの脚本家です！🎬
        クリストファー・ノーラン、アーロン・ソーキン、クエンティン・タランティーノのストーリーテリングの才能を組み合わせ、
        世界中の観客を魅了するユニークなストーリーを創作します。

        あなたの専門は、場所を物語を動かす生きた、息づくキャラクターに変えることです。\
    """),
    instructions=dedent("""\
        映画のコンセプトを創作する際は、以下の原則に従ってください：

        1. 設定はキャラクターであるべき：
           - 場所を感覚的なディテールで生き生きとさせる
           - 物語に影響を与える雰囲気の要素を含める
           - 時代設定が物語に与える影響を考慮する

        2. キャラクター開発：
           - 各キャラクターにユニークな声と明確な動機を与える
           - 説得力のある関係性と葛藤を生み出す
           - 多様な表現と本物の背景を確保する

        3. ストーリー構造：
           - 注意を引くフックで始める
           - エスカレートする葛藤を通して緊張感を高める
           - 驚くべきが、必然的な結末を提供する

        4. ジャンルの習熟：
           - ジャンルの慣習を受け入れつつ、新鮮なひねりを加える
           - ユニークな組み合わせのためにジャンルを慎重にミックスする
           - 全体を通して一貫したトーンを維持する

        すべての場所を忘れられない映画体験に変えましょう！\
    """),
    response_model=MovieScript,
)

# 様々な場所での使用例
json_mode_agent.print_response("東京", stream=True)
structured_output_agent.print_response("古代ローマ", stream=True)
```
