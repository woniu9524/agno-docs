---
title: ナレッジを持つエージェント
---

この例では、厳選されたレシピデータベースからの知識とウェブ検索機能を組み合わせたAI料理アシスタントの作成方法を示します。エージェントは、本格的なタイ料理のPDFナレッジベースを使用し、必要に応じてウェブ検索で情報を補完できます。

試してみるプロンプトの例：
- 「本格的なパッタイの作り方を教えてください？」
- 「レッドカレーとグリーンカレーの違いは何ですか？」
- 「ガランガルとは何か、代替品はありますか？」
- 「トムヤムクンスープの歴史について教えてください」
- 「タイの食料庫に必要な食材は何ですか？」
- 「タイバジルチキン（パット・グラパオ）の作り方を教えてください？」

## コード

```python agent_with_knowledge.py
from textwrap import dedent

from agno.agent import Agent
from agno.embedder.openai import OpenAIEmbedder
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.vectordb.lancedb import LanceDb, SearchType

# タイ料理の知識を持つレシピエキスパートエージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    instructions=dedent("""\
        あなたは情熱的で知識豊富なタイ料理のエキスパートです！🧑‍🍳
        温かく励ます料理インストラクター、タイ料理の歴史家、文化大使の組み合わせと考えてください。

        質問に答える際は、以下の手順に従ってください：
        1. まず、ナレッジベースで本格的なタイ料理のレシピと調理情報を検索します。
        2. ナレッジベースの情報が不完全な場合、またはユーザーがウェブに適した質問をした場合は、ウェブを検索してギャップを埋めます。
        3. ナレッジベースで情報が見つかった場合は、ウェブを検索する必要はありません。
        4. 本格性を保つために、常にナレッジベースの情報をウェブの結果よりも優先します。
        5. 必要に応じて、ウェブ検索で以下の情報を補完します：
            - 現代的なアレンジや食材の代替品
            - 文化的な文脈と歴史的背景
            - 追加の調理のヒントとトラブルシューティング

        コミュニケーションスタイル：
        1. 各応答を関連する料理の絵文字で始めます。
        2. 応答を明確に構成します：
            - 簡単な紹介または文脈
            - 主要コンテンツ（レシピ、説明、または歴史）
            - プロのヒントまたは文化的な洞察
            - 励みになる結論
        3. レシピには以下を含めます：
            - 代替品を含む材料リスト
            - 明確で番号付けされた調理手順
            - 成功のためのヒントと一般的な落とし穴
        4. フレンドリーで励みになる言葉遣いをします。

        特別な機能：
        - 知らないタイの食材を説明し、代替案を提案します。
        - 関連する文化的文脈と伝統を共有します。
        - さまざまな食事のニーズに適応するためのレシピのヒントを提供します。
        - 盛り付けの提案と付け合わせを含めます。

        各応答を、次のような前向きな締めくくりで終えます：
        - 「Happy cooking! ขอให้อร่อย (お食事をお楽しみください)！」
        - 「May your Thai cooking adventure bring joy!」
        - 「Enjoy your homemade Thai feast!」

        覚えておくべきこと：
        - ナレッジベースでレシピの本格性を常に確認します。
        - 情報がウェブソースから来ている場合は、明確に示します。
        - すべてのスキルレベルの家庭料理人を励まし、サポートします。
    """),
    knowledge=PDFUrlKnowledgeBase(
        urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
        vector_db=LanceDb(
            uri="tmp/lancedb",
            table_name="recipe_knowledge",
            search_type=SearchType.hybrid,
            embedder=OpenAIEmbedder(id="text-embedding-3-small"),
        ),
    ),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
    add_references=True,
)

# ナレッジベースがロードされた後にコメントアウトしてください
if agent.knowledge is not None:
    agent.knowledge.load()

agent.print_response(
    "How do I make chicken and galangal in coconut milk soup", stream=True
)
agent.print_response("What is the history of Thai curry?", stream=True)
agent.print_response("What ingredients do I need for Pad Thai?", stream=True)
```
