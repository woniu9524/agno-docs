title: カスタムツール
---

この例では、Agno で独自のカスタムツールを作成して使用する方法を示します。Hacker News の機能を、好きな API やサービスに置き換えることができます！

独自のツールのアイデア:
- 天気データ取得ツール
- 株価分析ツール
- 個人カレンダー連携
- カスタムデータベースクエリ
- ローカルファイル操作

## コード

```python custom_tools.py
import json
from textwrap import dedent

import httpx
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def get_top_hackernews_stories(num_stories: int = 10) -> str:
    """Hacker News のトップストーリーを取得するためにこの関数を使用します。

    Args:
        num_stories (int): 返されるストーリーの数。デフォルトは 10 です。

    Returns:
        str: トップストーリーの JSON 文字列。
    """

    # トップストーリー ID を取得
    response = httpx.get("https://hacker-news.firebaseio.com/v0/topstories.json")
    story_ids = response.json()

    # ストーリーの詳細を取得
    stories = []
    for story_id in story_ids[:num_stories]:
        story_response = httpx.get(
            f"https://hacker-news.firebaseio.com/v0/item/{story_id}.json"
        )
        story = story_response.json()
        if "text" in story:
            story.pop("text", None)
        stories.append(story)
    return json.dumps(stories)


# シリコンバレーのペルソナを持つテックニュースレポーターエージェントを作成
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    instructions=dedent("""\
        あなたはテクノロジーのすべてに情熱を注ぐ、テクノロジーに精通した Hacker News レポーターです！🤖
        シリコンバレーのインサイダーとテックジャーナリストのミックスと考えてください。

        あなたのスタイルガイド:
        - 絵文字を使った目を引くテックの見出しで始める
        - 熱意とテクノロジー重視の姿勢で Hacker News のストーリーを紹介する
        - 回答は簡潔かつ有益に保つ
        - 必要に応じて、テック業界の参照やスタートアップの専門用語を使用する
        - 「ターミナルに戻ります！」や「本番にプッシュします！」のようなキャッチーなテックテーマの締めくくりで終わる

        HN ストーリーを徹底的に分析し、テックへの熱意を高く保つことを忘れないでください！\
    """),
    tools=[get_top_hackernews_stories],
    show_tool_calls=True,
    markdown=True,
)

# 試すための質問例:
# - "現在 HN でトレンドになっているテクノロジーに関する議論は何ですか？"
# - "Hacker News のトップ 5 件のストーリーを要約してください"
# - "今日の最もアップボートされたストーリーは何ですか？"
agent.print_response("Summarize the top 5 stories on hackernews?", stream=True)
```
