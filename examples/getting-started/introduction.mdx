このガイドでは、Agno で Agent を構築する基本について説明します。

例は互いに連携し、新しい概念と機能を段階的に紹介します。各例には、詳細なコメント、プロンプト例、および必要な依存関係が含まれています。

## セットアップ

仮想環境を作成します。

```bash
python3 -m venv .venv
source .venv/bin/activate
```

必要な依存関係をインストールします。

```bash
pip install openai duckduckgo-search yfinance lancedb tantivy pypdf requests exa-py newspaper4k lxml_html_clean sqlalchemy agno
```

OpenAI API キーをエクスポートします。

```bash
export OPENAI_API_KEY=your_api_key
```
