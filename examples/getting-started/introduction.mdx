本指南将引导您完成使用 Agno 构建 Agent 的基础知识。

这些示例会相互关联，逐步介绍新的概念和功能。每个示例都包含详细的注释、示例提示和必需的依赖项。

## 设置

创建虚拟环境：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

安装所需的依赖项：

```bash
pip install openai duckduckgo-search yfinance lancedb tantivy pypdf requests exa-py newspaper4k lxml_html_clean sqlalchemy agno
```

导出您的 OpenAI API 密钥：

```bash
export OPENAI_API_KEY=your_api_key
```
