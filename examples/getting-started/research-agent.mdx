---
title: Research Agent
---

この例では、exa の検索機能と学術的なライティングスキルを組み合わせて、構造化された事実に基づいたレポートを提供する、高度なリサーチエージェントの作成方法を示します。

実証される主な機能：
- 学術およびニュース検索のための Exa.ai の使用
- 参考文献付きの構造化されたレポート生成
- カスタムフォーマットとファイル保存機能

試してみるプロンプトの例：
- "量子コンピューティングにおける最新の開発は何ですか？"
- "人工意識の現状を調査する"
- "核融合エネルギーにおける最近のブレークスルーを分析する"
- "宇宙観光の環境への影響を調査する"
- "長寿研究における最新の発見を探る"

## Code

```python research_agent.py
from datetime import datetime
from pathlib import Path
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

cwd = Path(__file__).parent.resolve()
tmp = cwd.joinpath("tmp")
if not tmp.exists():
    tmp.mkdir(exist_ok=True, parents=True)

today = datetime.now().strftime("%Y-%m-%d")

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[ExaTools(start_published_date=today, type="keyword")],
    description=dedent("""\
        あなたは、複雑な情報を分析および統合する専門知識を持つ、著名なAIリサーチサイエンティストであるProfessor X-1000です。
        あなたの専門は、学術的な厳密さと魅力的なナラティブを組み合わせた、説得力のある事実に基づいたレポートを作成することです。

        あなたの文章スタイルは以下の通りです：
        - 明確で権威がある
        - 魅力的でありながらプロフェッショナル
        - 正確な引用を伴う事実中心
        - 教育を受けた非専門家にも分かりやすい\
    """),
    instructions=dedent("""\
        包括的な情報を収集するために、まず3つの異なる検索を実行してください。
        ソースの正確性と関連性を分析し、相互参照してください。
        レポートは学術的な基準に従って構造化してくださいが、読みやすさを維持してください。
        検証可能な事実のみを適切な引用とともに含めてください。
        複雑なトピックを読者にガイドする魅力的なナラティブを作成してください。
        実行可能な要点と将来の含意で締めくくってください。\
    """),
    expected_output=dedent("""\
    Markdown形式のプロフェッショナルなリサーチレポート：

    # {トピックの本質を捉える魅力的なタイトル}

    ## エグゼクティブサマリー
    {主要な発見と重要性の概要}

    ## イントロダクション
    {トピックの背景と重要性}
    {研究/議論の現状}

    ## 主要な発見
    {主要な発見または開発}
    {裏付けとなる証拠と分析}

    ## 含意
    {分野/社会への影響}
    {将来の方向性}

    ## 主要な要点
    - {箇条書き1}
    - {箇条書き2}
    - {箇条書き3}

    ## 参考文献
    - [ソース1](link) - 主要な発見/引用
    - [ソース2](link) - 主要な発見/引用
    - [ソース3](link) - 主要な発見/引用

    ---
    レポート生成者：Professor X-1000
    アドバンストリサーチシステム部門
    日付：{current_date}\
    """),
    markdown=True,
    show_tool_calls=True,
    add_datetime_to_instructions=True,
    save_response_to_file=str(tmp.joinpath("{message}.md")),
)

# 使用例
if __name__ == "__main__":
    # 最先端のトピックに関するリサーチレポートを生成する
    agent.print_response(
        "脳コンピューターインターフェースにおける最新の開発を調査する", stream=True
    )

# 試してみる追加のプロンプト：
"""
これらのリサーチトピックを試してください：
1. "全固体電池の現状を分析する"
2. "CRISPR遺伝子編集における最近のブレークスルーを調査する"
3. "自動運転車の開発を調査する"
4. "量子機械学習の進歩を探る"
5. "人工知能がヘルスケアに与える影響を研究する"
"""
```
