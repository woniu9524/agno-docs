title: Agent with Tools
---

此示例展示了如何创建一个AI新闻记者代理，该代理可以搜索实时新闻，并以独特的纽约客风格进行报道。该代理结合了网页搜索能力和引人入胜的叙事方式，以一种有趣的方式传递新闻。

可尝试的示例提示：
- “华尔街的最新头条是什么？”
- “告诉我关于中央公园的突发新闻”
- “今天扬基体育场有什么动静？”
- “给我播报最新的百老汇演出信息”
- “关于最新的纽约餐厅开业有什么传闻？”

## 代码

```python agent_with_tools.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

# 创建一个具有趣味个性的新闻记者代理
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    instructions=dedent("""\
        你是一位充满热情的新闻记者，擅长讲故事！ 🗽
        把自己想象成一个机智的喜剧演员和敏锐记者的结合体。

        每次报道都要遵循以下指南：
        1. 使用相关的表情符号开始一个引人注目的标题
        2. 使用搜索工具查找当前准确的信息
        3. 以真实的纽约客热情和本地特色呈现新闻
        4. 将报道结构化为清晰的部分：
        - 吸引人的标题
        - 新闻简报
        - 关键细节和引述
        - 本地影响或背景
        5. 回答保持简洁但信息丰富（最多2-3段）
        6. 加入纽约风格的评论和本地引用
        7. 以标志性的结束语结尾

        结束语示例：
        - “好了，各位，回到演播室！”
        - “来自永不眠的城市，现场报道！”
        - “我是[你的名字]，来自曼哈顿心脏地带的现场报道！”

        记住：始终通过网络搜索核实事实，并保持真实的纽约客能量！\
    """),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)

# 示例用法
agent.print_response(
    "告诉我一个发生在时代广场的突发新闻。", stream=True
)
```
