title: Agent with Tools
---

この例では、AIニュースレポーターエージェントを作成する方法を示します。このエージェントは、リアルタイムのニュースを検索し、特徴的なNYCのペルソナでそれらを提示できます。エージェントは、ウェブ検索機能と魅力的なストーリーテリングを組み合わせて、ニュースを楽しい方法で配信します。

試してみるプロンプトの例：
- 「ウォール街からの最新の見出しは何ですか？」
- 「セントラルパークでの速報について教えてください」
- 「今日のヤンキースタジアムで何が起こっていますか？」
- 「最新のブロードウェイショーに関するアップデートを教えてください」
- 「NYCの最新レストランのオープニングに関する話題は何ですか？」

## コード

```python agent_with_tools.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools

# Funなペルソナを持つニュースレポーターエージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    instructions=dedent("""\
        あなたはストーリーテリングの才能を持つ、熱狂的なニュースレポーターです！ 🗽
        機知に富んだコメディアンと鋭いジャーナリストのミックスと考えてください。

        すべてのレポートで次のガイドラインに従ってください：
        1. 関連する絵文字を使用して、注意を引く見出しで開始します
        2. 検索ツールを使用して、現在の正確な情報を検索します
        3. 本物のNYCの熱意と地元の風味でニュースを提示します
        4. レポートを明確なセクションに構成します：
        - キャッチーな見出し
        - ニュースの簡単な概要
        - 主要な詳細と引用
        - 地域への影響または文脈
        5. 回答は簡潔かつ有益に保ちます（最大2〜3段落）
        6. NYCスタイルの解説と地元の参照を含めます
        7. シグネチャーの締めくくりのフレーズで終了します

        締めくくりの例：
        - 「スタジオのみんなに返すよ！」
        - 「眠らない街からライブでお伝えします！」
        - 「マンハッタンの中心からライブでお伝えしているのは、[あなたの名前]です！」

        覚えておいてください：常にウェブ検索で事実を確認し、本物のNYCのエネルギーを維持してください！\
    """),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)

# 使用例
agent.print_response(
    "タイムズスクエアで起こっている速報について教えてください。", stream=True
)

# 試してみるその他のプロンプト：
"""
これらの魅力的なニュースクエリを試してください：
1. 「NYCのテクノロジーシーンの最新の開発は何ですか？」
2. 「マディソン・スクエア・ガーデンでの今後のイベントについて教えてください」
3. 「今日のNYCへの天気の影響は何ですか？」
4. 「NYCの地下鉄システムに関するアップデートはありますか？」
5. 「現在、マンハッタンで最も人気のあるフードトレンドは何ですか？」
"""
```
