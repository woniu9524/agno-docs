title: Basic Agent
---

この例では、明確な個性を持つ基本的なAIエージェントを作成する方法を示します。ここでは、NYCの態度と創造的なストーリーテリングを組み合わせた、楽しいニュースレポーターを作成します。これにより、個性とスタイルの指示がエージェントの応答をどのように形成できるかがわかります。

試すためのプロンプト例：
- 「セントラルパークからの最新スクープは何ですか？」
- 「ウォール街からの速報について教えてください」
- 「ヤンキースの試合で今何が起こっていますか？」
- 「新しいブロードウェイショーについての話題を教えてください」

## コード

```python basic_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat

# 楽しい個性を持つニュースレポーターを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    instructions=dedent("""\
        あなたはストーリーテリングの才能を持つ、熱狂的なニュースレポーターです！ 🗽
        機知に富んだコメディアンと鋭いジャーナリストのミックスと考えてください。

        あなたのスタイルガイド：
        - 絵文字を使った注意を引く見出しで始める
        - 熱意とNYCの態度でニュースを共有する
        - 回答は簡潔かつエンターテイニングに保つ
        - 適切な場合は、地元の参照とNYCのスラングを挟む
        - 「スタジオにお返しします！」や「ビッグアップルから生中継！」のようなキャッチーな締めくくりで終わる

        NYCのエネルギーを高く保ちながら、すべての事実を確認することを忘れないでください！\
    """),
    markdown=True,
)

# 使用例
agent.print_response(
    "タイムズスクエアで発生している速報について教えてください。", stream=True
)

# 試すための追加のプロンプト例：
"""
これらの楽しいシナリオを試してください：
1. 「ブルックリンを席巻している最新のフードトレンドは何ですか？」
2. 「今日の地下鉄での奇妙な出来事について教えてください」
3. 「マンハッタンの新しい屋上庭園についてのスクープは何ですか？」
4. 「逃げ出した動物園の動物によって引き起こされた異常な交通渋滞を報道してください」
5. 「グランドセントラルでのフラッシュモブ結婚式のプロポーズをカバーしてください」
"""
```
