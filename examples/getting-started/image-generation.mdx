---
title: 画像生成
---

この例では、DALL-E を使用して画像を生成する AI エージェントの作成方法を示します。
このエージェントを使用して、リアルな写真から芸術的なイラスト、創造的なコンセプトまで、さまざまな種類の画像を生成できます。

試してみるプロンプトの例:
- 「夕暮れ時の雲に浮かぶ都市のシュールな絵を作成してください」
- 「居心地の良いコーヒーショップのインテリアの写実的な画像を生成してください」
- 「テクノロジー企業のスタートアップのためのキュートな漫画のマスコットをデザインしてください」
- 「サイバーパンク侍の芸術的なポートレートを作成してください」

## コード

```python image_generation.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.dalle import DalleTools

# クリエイティブなAIアーティストエージェントを作成
image_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[DalleTools()],
    description=dedent("""\
        あなたは、写実主義から抽象芸術まで、さまざまな芸術スタイルに精通した経験豊富なAIアーティストです。
        構図、色彩理論、ビジュアルストーリーテリングについて深い理解を持っています。\
    """),
    instructions=dedent("""\
        AIアーティストとして、以下のガイドラインに従ってください:
        1. ユーザーのリクエストを注意深く分析し、望ましいスタイルとムードを理解してください
        2. 生成する前に、照明、視点、雰囲気などの芸術的な詳細でプロンプトを強化してください
        3. 詳細でよく練られたプロンプトを使用して `create_image` ツールを使用してください
        4. 選択した芸術的な選択について簡単な説明を提供してください
        5. リクエストが不明確な場合は、スタイルの好みについて明確化を求めてください

        常にユーザーのビジョンを捉える、視覚的に印象的で意味のある画像を作成することを目指してください！\
    """),
    markdown=True,
    show_tool_calls=True,
)

# 使用例
image_agent.print_response(
    "浮遊する本と光るクリスタルがある魔法の図書館を作成してください", stream=True
)

# 生成された画像を取得して表示
images = image_agent.get_images()
if images and isinstance(images, list):
    for image_response in images:
        image_url = image_response.url
        print(f"生成された画像のURL: {image_url}")

# 試してみるさらに多くのプロンプトの例:
"""

2. 「桜の季節の平和な禅庭園をデザインしてください」
3. 「生物発光する建物がある水中都市を作成してください」
4. 「夜の雪景色の中に居心地の良いキャビンを生成してください」
5. 「飛行車と高層ビルがある未来的な都市景観を作成してください」
"""
```
