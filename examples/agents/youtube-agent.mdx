この例では、詳細な動画の分析、タイムスタンプ、要約を提供するインテリジェントなYouTubeコンテンツアナライザーを作成する方法を示します。コンテンツクリエイター、研究者、動画コンテンツを効率的にナビゲートしたい視聴者に最適です。

試すプロンプトの例：
- 「このテックレビューを分析してください：[video_url]」
- 「このコーディングチュートリアルのタイムスタンプを取得してください：[video_url]」
- 「この講義の要点を分解してください：[video_url]」
- 「このドキュメンタリーの主要トピックを要約してください：[video_url]」
- 「この教育ビデオから学習ガイドを作成してください：[video_url]」

## コード

```python youtube_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.youtube import YouTubeTools

youtube_agent = Agent(
    name="YouTube Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[YouTubeTools()],
    show_tool_calls=True,
    instructions=dedent("""\
        あなたは、細部にまでこだわるYouTubeコンテンツの専門アナリストです！🎓
        包括的な動画分析のために、以下の手順に従ってください：
        1. 動画概要
           - 動画の長さと基本メタデータをチェックする
           - 動画の種類（チュートリアル、レビュー、講義など）を特定する
           - コンテンツの構造をメモする
        2. タイムスタンプ作成
           - 正確で意味のあるタイムスタンプを作成する
           - 主要なトピックの遷移に焦点を当てる
           - キーとなる瞬間とデモンストレーションをハイライトする
           - フォーマット：[開始時間, 終了時間, 詳細な要約]
        3. コンテンツの整理
           - 関連するセグメントをグループ化する
           - 主要なテーマを特定する
           - トピックの進行を追跡する

        あなたの分析スタイル：
        - 動画の概要から始める
        - 明確で説明的なセグメントタイトルを使用する
        - コンテンツタイプに応じて関連する絵文字を含める：
          📚 教育的
          💻 技術的
          🎮 ゲーム
          📱 テックレビュー
          🎨 クリエイティブ
        - 主要な学習ポイントをハイライトする
        - 実践的なデモンストレーションをメモする
        - 重要な参照をマークする

        品質ガイドライン：
        - タイムスタンプの正確性を検証する
        - タイムスタンプの幻覚（誤った情報の生成）を避ける
        - 包括的なカバレッジを保証する
        - 一貫した詳細レベルを維持する
        - 価値のあるコンテンツマーカーに焦点を当てる
    """),
    add_datetime_to_instructions=True,
    markdown=True,
)

# 様々な種類の動画での使用例
youtube_agent.print_response(
    "Analyze this video: https://www.youtube.com/watch?v=zjkBMFhNj_g",
    stream=True,
)

# 探索するための追加のプロンプト例：
"""
チュートリアル分析：
1. 「コード例に焦点を当てて、このPythonチュートリアルを分解してください」
2. 「このWeb開発コースから学習パスを作成してください」
3. 「このプログラミングガイドからすべての実践演習を抽出してください」
4. 「主要な概念と実装例を特定してください」

教育コンテンツ：
1. 「この数学の講義のタイムスタンプ付き学習ガイドを作成してください」
2. 「この科学ビデオから主要な理論と例を抽出してください」
3. 「この歴史ドキュメンタリーを主要なイベントに分解してください」
4. 「この学術プレゼンテーションの主要な議論を要約してください」

テックレビュー：
1. 「タイムスタンプ付きで、このレビューで言及されたすべての製品機能をリストアップしてください」
2. 「このレビューで議論された長所と短所を比較してください」
3. 「技術仕様とベンチマークを抽出してください」
4. 「主要な比較ポイントと結論を特定してください」

クリエイティブコンテンツ：
1. 「このアートチュートリアルで示されたテクニックを分解してください」
2. 「このDIYビデオのプロジェクトステップのタイムラインを作成してください」
3. 「タイムスタンプ付きで、言及されたすべてのツールと材料をリストアップしてください」
4. 「デモンストレーション付きのヒントとコツを抽出してください」
"""
```
