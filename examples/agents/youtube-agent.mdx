此示例展示了如何创建一个智能的 YouTube 内容分析器，该分析器可提供详细的视频解析、时间戳和摘要。非常适合内容创作者、研究人员以及希望高效浏览视频内容的用户。

可尝试的示例提示：
- "分析这个科技评测：[video_url]"
- "获取这个编程教程的时间戳：[video_url]"
- "解析这个讲座的关键点：[video_url]"
- "总结这个纪录片的主要话题：[video_url]"
- "从这个教育视频创建一个学习指南：[video_url]"

## 代码

```python youtube_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.youtube import YouTubeTools

youtube_agent = Agent(
    name="YouTube Agent",
    model=OpenAIChat(id="gpt-4o"),
    tools=[YouTubeTools()],
    show_tool_calls=True,
    instructions=dedent("""\
        你是一位经验丰富的 YouTube 内容分析专家，拥有敏锐的细节洞察力！🎓
        请遵循以下步骤进行全面的视频分析：
        1. 视频概述
           - 查看视频时长和基本元数据
           - 识别视频类型（教程、评测、讲座等）
           - 记录内容结构
        2. 时间戳创建
           - 创建精确、有意义的时间戳
           - 关注主要话题的转换点
           - 突出关键时刻和演示
           - 格式：[start_time, end_time, detailed_summary]
        3. 内容组织
           - 对相关片段进行分组
           - 识别主要主题
           - 追踪话题的进展

        你的分析风格：
        - 以视频概述开始
        - 使用清晰、描述性的片段标题
        - 为内容类型包含相关的表情符号：
          📚 教育
          💻 技术
          🎮 游戏
          📱 科技评测
          🎨 创意
        - 突出关键学习点
        - 记录实际演示
        - 标记重要参考信息

        质量指南：
        - 验证时间戳的准确性
        - 避免时间戳幻觉
        - 确保内容覆盖全面
        - 保持一致的细节级别
        - 专注于有价值的内容标记
    """),
    add_datetime_to_instructions=True,
    markdown=True,
)

# 使用不同类型的视频进行示例用法
youtube_agent.print_response(
    "Analyze this video: https://www.youtube.com/watch?v=zjkBMFhNj_g",
    stream=True,
)
```
