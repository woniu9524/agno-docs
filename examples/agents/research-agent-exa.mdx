---
title: Exa を使用したリサーチエージェント
---

この例では、学術検索機能と学術執筆の専門知識を組み合わせた、洗練されたリサーチエージェントの作成方法を示します。このエージェントは、Exa の学術検索を使用して徹底的なリサーチを行い、最近の出版物を分析し、あらゆるトピックに関する構造化された学術スタイルのレポートを提供します。

主な機能:
- 高度な学術文献検索
- 最近の出版物の分析
- 学際的な統合
- 学術執筆の専門知識
- 引用管理

試すためのプロンプト例:
- 「量子機械学習の最近の進歩を探る」
- 「核融合エネルギー研究の現状を分析する」
- 「CRISPR遺伝子編集の最新動向を調査する」
- 「ブロックチェーンと持続可能なエネルギーの交差点をリサーチする」
- 「脳コンピュータインターフェースの最近のブレークスルーを検証する」

## コード

```python research_agent_exa.py
from datetime import datetime
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

# 学術的な機能を備えた学術リサーチエージェントを初期化します
research_scholar = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        ExaTools(
            start_published_date=datetime.now().strftime("%Y-%m-%d"), type="keyword"
        )
    ],
    description=dedent("""\
        あなたは、複数の分野に専門知識を持つ著名な研究学者です。
        あなたの学術的資格には以下が含まれます: 📚

        - 高度なリサーチ方法論
        - 学際的な統合
        - 学術文献分析
        - 科学的執筆の卓越性
        - 査読経験
        - 引用管理
        - データ解釈
        - 技術コミュニケーション
        - 新興トレンド分析\
    """),
    instructions=dedent("""\
        1. リサーチ方法論 🔍
           - 3つの異なる学術検索を実施する
           - 査読付き出版物を優先する
           - 最近の画期的な発見を優先する
           - 主要な研究者と機関を特定する

        2. 分析フレームワーク 📊
           - ソース全体の結果を統合する
           - リサーチ方法論を評価する
           - コンセンサスと論争を特定する
           - 実用的な影響を評価する

        3. レポート構造 📝
           - 魅力的な学術タイトルを作成する
           - 説得力のある要約を作成する
           - 方法論を明確に提示する
           - 結果を体系的に提示する
           - 証拠に基づいた結論を導き出す

        4. 品質基準 ✓
           - 正確な引用を保証する
           - 学術的な厳密性を維持する
           - バランスの取れた視点を提示する
           - 将来の研究の方向性を強調する\
    """),
    expected_output=dedent("""\
        # {魅力的なタイトル} 📚

        ## 要約
        {リサーチと主要な発見の簡潔な概要}

        ## 導入
        {背景と重要性}
        {リサーチ目標}

        ## 方法論
        {検索戦略}
        {選択基準}

        ## 文献レビュー
        {現在のリサーチ状況}
        {主要な発見とブレークスルー}
        {新興トレンド}

        ## 分析
        {批判的評価}
        {研究間の比較}
        {リサーチギャップ}

        ## 将来の方向性
        {新たなリサーチ機会}
        {潜在的な応用}
        {未解決の質問}

        ## 結論
        {主要な発見の要約}
        {分野への影響}

        ## 参考文献
        {適切にフォーマットされた学術的な引用}

        ---
        AI Academic Scholar によるリサーチ
        公開日: {current_date}
        最終更新日: {current_time}\
    """),
    markdown=True,
    show_tool_calls=True,
    add_datetime_to_instructions=True,
    save_response_to_file="tmp/{message}.md",
)

# 学術リクエストを使用したサンプル使用法
if __name__ == "__main__":
    research_scholar.print_response(
        "量子コンピューティングアーキテクチャの最近の開発を分析する",
        stream=True,
    )

# 探求すべき高度なリサーチトピック:
"""
量子科学 & コンピューティング:
1. 「量子誤り訂正の最近のブレークスルーを調査する」
2. 「トポロジカル量子コンピューティングの開発を分析する」
3. 「量子機械学習アルゴリズムとアプリケーションをリサーチする」
4. 「量子センシング技術の進歩を探る」

バイオテクノロジー & 医療:
1. 「mRNAワクチン技術の最近の開発を検証する」
2. 「オルガノイドリサーチのブレークスルーを分析する」
3. 「精密医療の進歩を調査する」
4. 「ニューロテクノロジーの開発をリサーチする」

材料科学:
1. 「メタマテリアルの最近の進歩を探る」
2. 「グラフェン以外の2D材料の開発を分析する」
3. 「自己修復材料の進歩をリサーチする」
4. 「新しいバッテリー技術を調査する」

人工知能:
1. 「基盤モデルの最近の進歩を検証する」
2. 「AI安全性リサーチの開発を分析する」
3. 「ニューロモルフィックコンピューティングの進歩をリサーチする」
4. 「説明可能なAIの進歩を調査する」
"""
```
