---
title: 使用 Exa 的研究代理
---

本示例展示了如何创建一个结合了学术搜索能力和学术写作专业知识的复杂研究代理。该代理使用 Exa 的学术搜索进行深入研究，分析最新出版物，并就任何主题交付结构良好、风格学术的报告。

主要功能：
- 高级学术文献搜索
- 最新出版物分析
- 跨学科综合
- 学术写作专业知识
- 引用管理

可尝试的示例提示：
- "探索量子机器学习的最新进展"
- "分析聚变能源研究的现状"
- "研究 CRISPR 基因编辑的最新发展"
- "研究区块链与可持续能源的交叉点"
- "考察脑机接口的最新突破"

## 代码

```python research_agent_exa.py
from datetime import datetime
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

# 初始化具有学术能力的学术研究代理
research_scholar = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        ExaTools(
            start_published_date=datetime.now().strftime("%Y-%m-%d"), type="keyword"
        )
    ],
    description=dedent("""\
        你是一位杰出的研究学者，精通多个学科。
        你的学术资历包括：📚

        - 高级研究方法
        - 跨学科综合
        - 学术文献分析
        - 卓越的科学写作
        - 同行评审经验
        - 引用管理
        - 数据解读
        - 技术沟通
        - 新兴趋势分析\
    """),
    instructions=dedent("""\
        1. 研究方法 🔍
           - 进行 3 次独立学术搜索
           - 专注于同行评审的出版物
           - 优先考虑最新的突破性发现
           - 识别关键研究人员和机构

        2. 分析框架 📊
           - 综合各来源的发现
           - 评估研究方法
           - 识别共识和争议
           - 评估实际意义

        3. 报告结构 📝
           - 创建引人入胜的学术标题
           - 撰写引人注目的摘要
           - 清晰地呈现研究方法
           - 系统地讨论研究结果
           - 得出基于证据的结论

        4. 质量标准 ✓
           - 确保引用准确
           - 保持学术严谨性
           - 呈现平衡的观点
           - 强调未来的研究方向\
    """),
    expected_output=dedent("""\
        # {引人入胜的标题} 📚

        ## 摘要
        {研究和主要发现的简洁概述}

        ## 引言
        {背景和意义}
        {研究目标}

        ## 研究方法
        {搜索策略}
        {选择标准}

        ## 文献综述
        {当前研究状况}
        {关键发现和突破}
        {新兴趋势}

        ## 分析
        {批判性评估}
        {跨研究比较}
        {研究空白}

        ## 未来方向
        {新兴研究机会}
        {潜在应用}
        {开放性问题}

        ## 结论
        {主要发现总结}
        {对该领域的意义}

        ## 参考文献
        {格式正确的学术引用}

        ---
        由人工智能学术学者进行的研究
        发布日期：{current_date}
        最后更新日期：{current_time}\
    """),
    markdown=True,
    show_tool_calls=True,
    add_datetime_to_instructions=True,
    save_response_to_file="tmp/{message}.md",
)

# 使用学术研究请求的示例用法
if __name__ == "__main__":
    research_scholar.print_response(
        "分析量子计算架构的最新发展",
        stream=True,
    )

# 可探索的高级研究主题：
"""

2. "分析拓扑量子计算的发展"
3. "研究量子机器学习算法和应用"
4. "探索量子传感技术的进展"

生物技术与医学：
1. "考察 mRNA 疫苗技术的最新发展"
2. "分析类器官研究的突破"
3. "研究精准医疗的进展"
4. "研究神经技术的发展"

材料科学：
1. "探索超材料的最新进展"
2. "分析石墨烯以外二维材料的发展"
3. "研究自修复材料的进展"
4. "调查新型电池技术"

人工智能：
1. "考察基础模型的最新进展"
```
