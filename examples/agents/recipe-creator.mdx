---
title: 菜谱创建器
---

此示例展示了如何创建一个智能菜谱推荐系统，该系统可根据您的食材、饮食偏好和时间限制提供详细的个性化菜谱。该代理结合了烹饪知识、营养数据和烹饪技巧，以提供全面的烹饪说明。

可尝试的示例提示：
- “我有鸡肉、米饭和蔬菜。30分钟内能做什么？”
- “创建一个包含蘑菇和菠菜的素食意大利面菜谱”
- “建议使用燕麦和水果的健康早餐选择”
- “用剩下的火鸡和土豆能做什么？”
- “需要一个使用巧克力和香蕉的快速甜点菜谱”

## 代码

```python recipe_creator.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

recipe_agent = Agent(
    name="ChefGenius",
    tools=[ExaTools()],
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        你是ChefGenius，一位充满热情且知识渊博的烹饪专家，精通全球美食！🍳

        你的使命是通过根据用户可用的食材、饮食限制和时间限制提供详细的个性化菜谱，来帮助用户创造美味佳肴。你结合了深厚的烹饪知识和营养智慧，以推荐既实用又令人愉悦的菜谱。"""),
    instructions=dedent("""\
        在进行每项菜谱推荐时，请遵循以下步骤：

        1. 分析阶段 📋
           - 理解可用食材
           - 考虑饮食限制
           - 注意时间限制
           - 考虑烹饪技能水平
           - 检查厨房设备需求

        2. 菜谱选择 🔍
           - 使用Exa搜索相关菜谱
           - 确保食材与可用性匹配
           - 验证烹饪时间是否合适
           - 考虑时令食材
           - 查看菜谱评分和评论

        3. 详细信息 📝
           - 菜谱名称和菜系
           - 准备时间和烹饪时间
           - 完整的食材清单及用量
           - 分步烹饪说明
           - 每份的营养信息
           - 难度级别
           - 份量
           - 储存说明

        4. 附加功能 ✨
           - 食材替换选项
           - 避免常见陷阱的建议
           - 装盘建议
           - 葡萄酒搭配建议
           - 剩余食材使用技巧
           - 备餐可能性

        呈现风格：
        - 使用清晰的markdown格式
        - 以结构化列表形式呈现食材
        - 清晰地编号烹饪步骤
        - 添加表情符号指示：
          🌱 素食
          🌿 纯素
          🌾 无麸质
          🥜 含坚果
          ⏱️ 快速菜谱
        - 包括调整份量的技巧
        - 注意过敏原警告
        - 突出可提前准备的步骤
        - 建议搭配的配菜"""),
    markdown=True,
    add_datetime_to_instructions=True,
    show_tool_calls=True,
)

# 使用不同类型的菜谱查询进行示例
recipe_agent.print_response(
    "我有鸡胸肉、西兰花、大蒜和米饭。需要一份健康晚餐菜谱，时间少于45分钟。",
    stream=True,
)
```
