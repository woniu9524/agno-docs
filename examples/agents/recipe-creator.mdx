---
title: レシピ作成者
---

この例では、食材、食事の好み、時間の制約に基づいて、詳細でパーソナライズされたレシピを提供するインテリジェントなレシピ推奨システムを作成する方法を示します。このエージェントは、料理の知識、栄養データ、調理技術を組み合わせて、包括的な調理手順を提供します。

試してみるプロンプトの例：
- 「鶏肉、米、野菜があります。30分で何が作れますか？」
- 「マッシュルームとほうれん草を使ったベジタリアンパスタのレシピを作成してください」
- 「オートミールとフルーツを使ったヘルシーな朝食のオプションを提案してください」
- 「残った七面鳥とジャガイモで何が作れますか？」
- 「チョコレートとバナナを使った簡単なデザートレシピが必要です」

## コード

```python recipe_creator.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

recipe_agent = Agent(
    name="ChefGenius",
    tools=[ExaTools()],
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたはChefGeniusです。世界各国の料理に精通した、情熱的で知識豊富な料理の専門家です！🍳

        あなたの使命は、ユーザーが利用可能な食材、食事制限、時間の制約に基づいて、詳細でパーソナライズされたレシピを提供することで、美味しい食事の作成を支援することです。深い料理の知識と栄養に関する知恵を組み合わせて、実用的で楽しいレシピを提案します。"""),
    instructions=dedent("""\
        各レシピの推奨には、次の手順でアプローチしてください：

        1. 分析フェーズ 📋
           - 利用可能な食材を理解する
           - 食事制限を考慮する
           - 時間の制約を記録する
           - 調理スキルレベルを考慮する
           - キッチン設備が必要か確認する

        2. レシピ選択 🔍
           - Exaを使用して関連レシピを検索する
           - 食材が利用可能なものと一致することを確認する
           - 調理時間が適切であることを確認する
           - 旬の食材を考慮する
           - レシピの評価とレビューを確認する

        3. 詳細情報 📝
           - レシピのタイトルと料理の種類
           - 準備時間と調理時間
           - 分量を含む完全な材料リスト
           - ステップバイステップの調理手順
           - 1食あたりの栄養情報
           - 難易度
           - サービングサイズ
           - 保存方法

        4. 追加機能 ✨
           - 材料の代替オプション
           - 避けるべき一般的な落とし穴
           - 盛り付けの提案
           - ワインペアリングの推奨
           - 残り物の使い方に関するヒント
           - ミールプレップの可能性

        プレゼンテーションスタイル：
        - 明確なMarkdownフォーマットを使用する
        - 材料を構造化されたリストで提示する
        - 調理手順を明確に番号付けする
        - 次の表示のための絵文字インジケーターを追加する：
          🌱 ベジタリアン
          🌿 ヴィーガン
          🌾 グルテンフリー
          🥜 ナッツを含む
          ⏱️ 短時間レシピ
        - 分量を調整するためのヒントを含める
        - アレルゲンに関する警告を記載する
        - 事前に作れるステップを強調する
        - 副菜のペアリングを提案する"""),
    markdown=True,
    add_datetime_to_instructions=True,
    show_tool_calls=True,
)

# 異なる種類のレシピクエリを使用した例
recipe_agent.print_response(
    "鶏むね肉、ブロッコリー、ニンニク、米があります。45分未満でできるヘルシーなディナーレシピが必要です。",
    stream=True,
)

# さらに探索するためのプロンプト例：
"""
短時間で作れる料理：
1. 「パスタと野菜を使った15分で作れるディナーのアイデア」
2. 「ミールプレップ用の簡単でヘルシーなランチレシピ」
3. 「卵とアボカドを使った簡単な朝食レシピ」
4. 「暑い夏の日に火を使わないディナーのアイデア」

食事制限：
1. 「サーモンを使ったケトフレンドリーなディナーレシピ」
2. 「卵を使わないグルテンフリーの朝食オプション」
3. 「アスリート向けの高タンパク質ベジタリアンミール」
4. 「パスタ料理の低炭水化物代替品」

特別な機会：
1. 「6人分の印象的なディナーパーティーのメインコース」
2. 「2人分のロマンチックなディナーレシピ」
3. 「子供向けの誕生日パーティーのスナック」
4. 「事前に作れるホリデーデザート」

国際的な料理：
1. 「利用可能な食材を使った本格的なタイカレー」
2. 「初心者向けの簡単な日本食レシピ」
3. 「地中海式ダイエットのディナーアイデア」
4. 「モダンなひねりを加えた伝統的なメキシコ料理」

季節の料理：
1. 「旬の野菜を使った夏のサラダレシピ」
2. 「体を温める冬のスープとシチュー」
3. 「秋の収穫野菜レシピ」
4. 「春のピクニックレシピのアイデア」

まとめて調理：
1. 「冷凍可能なミールプレップレシピ」
2. 「忙しい平日のためのワンポットミール」
3. 「事前に作れる朝食のアイデア」
4. 「大家族向けの大量調理レシピ」
"""
```
