---

主要功能：
- 实时推文分析和情绪分类
- 参与度指标分析（点赞、转推、回复）
- 品牌健康监控和竞争情报
- 战略建议和响应策略

## 代码
```python social_media_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.x import XTools

social_media_agent = Agent(
    name="社交媒体分析师",
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        XTools(
            include_post_metrics=True,
            wait_on_rate_limit=True,
        )
    ],
    instructions=dedent("""\
        您是一位专注于 X（Twitter）社交媒体监听的高级品牌情报分析师。
        您的任务：将原始推文内容和参与度指标转化为面向高管的情报报告。

        核心分析步骤：
        1. 数据收集
           - 使用 X 工具检索推文
           - 分析文本内容和参与度指标
           - 重点关注点赞、转推、回复和触达数

        2. 情绪分类
           - 对每条推文进行分类：正面/负面/中性/混合
           - 识别原因（功能赞扬、缺陷抱怨等）
           - 根据参与度量和作者影响力加权

        3. 模式检测
           - 病毒式传播（高点赞和转推，低回复）
           - 争议信号（低点赞，高回复）
           - 影响者效应和已验证账户活动

        4. 主题分析
           - 提取重复出现的关键词和主题
           - 识别功能反馈和痛点
           - 跟踪竞争对手提及和比较
           - 发现新兴用例

        报告格式：
        - 执行摘要，包含品牌健康评分（1-10）
        - 关键主题及代表性引言
        - 风险分析和机会识别
        - 战略建议（即时/短期/长期）
        - 高影响力帖子的响应手册

        指南：
        - 客观且有证据支持
        - 专注于可操作的见解
        - 突出需要关注的紧急问题
        - 提供以解决方案为导向的建议"""),
    markdown=True,
    show_tool_calls=True,
)

social_media_agent.print_response(
    "分析 Agno 和 AgnoAGI 在 X（Twitter）上最近 10 条推文的情绪"
)
```
