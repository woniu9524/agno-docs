---
title: Tweet Analysis Agent
description: 総合的なブランド監視とセンチメント分析を提供するツイート分析エージェント。
---

主な機能：
- リアルタイムツイート分析とセンチメント分類
- エンゲージメント指標分析（いいね、リツイート、返信）
- ブランド健全性監視と競合インテリジェンス
- 戦略的推奨事項と対応戦略

## コード
```python social_media_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.x import XTools

social_media_agent = Agent(
    name="ソーシャルメディアアナリスト",
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        XTools(
            include_post_metrics=True,
            wait_on_rate_limit=True,
        )
    ],
    instructions=dedent("""\
        あなたはX（Twitter）のソーシャルメディアリスニングを専門とする、シニアブランドインテリジェンスアナリストです。
        あなたのミッション：生のツイートコンテンツとエンゲージメント指標を、経営層向けのインテリジェンスレポートに変革すること。

        コア分析ステップ：
        1. データ収集
           - Xツールを使用してツイートを取得
           - テキストコンテンツとエンゲージメント指標を分析
           - いいね、リツイート、返信、リーチに焦点を当てる

        2. センチメント分類
           - 各ツイートを分類：ポジティブ/ネガティブ/ニュートラル/ミックス
           - 理由を特定（機能称賛、バグ苦情など）
           - エンゲージメント量と投稿者の影響力で重み付け

        3. パターン検出
           - バイラルな支持（いいね＆リツイートが多く、返信が少ない）
           - 論争のシグナル（いいねが少なく、返信が多い）
           - インフルエンサーの影響力と認証済みアカウントの活動

        4. テーマ分析
           - 繰り返し現れるキーワードとテーマを抽出
           - 機能フィードバックとペインポイントを特定
           - 競合他社の言及と比較を追跡
           - 新興のユースケースを発見

        レポート形式：
        - ブランド健全性スコア（1〜10）を含むエグゼクティブサマリー
        - 代表的な引用を含む主要テーマ
        - リスク分析と機会の特定
        - 戦略的推奨事項（即時/短期/長期）
        - インパクトの高い投稿に対する対応プレイブック

        ガイドライン：
        - 客観的かつ証拠に基づいたものにする
        - 実行可能な洞察に焦点を当てる
        - 緊急の注意が必要な問題を強調する
        - 解決策指向の推奨事項を提供する"""),
    markdown=True,
    show_tool_calls=True,
)

social_media_agent.print_response(
    "過去10件のツイートにおけるAgnoとAgnoAGIのX（Twitter）でのセンチメントを分析してください"
)
```

<Note> 詳細な[ソーシャルメディアエージェント](https://github.com/agno-agi/agno/blob/main/cookbook/examples/agents/social_media_agent.py)をご覧ください。 </Note>


試すための追加プロンプト：
- 「過去10件のツイートでXにおける当社のブランドに関するセンチメントを分析してください」
- 「競合他社の言及を監視し、当社のブランドと比較してセンチメントを比較してください」
- 「最近のソーシャルメディア活動からブランド健全性レポートを生成してください」
- 「製品に関するトレンドトピックとユーザーセンチメントを特定してください」
- 「経営陣レビュー用のソーシャルメディアインテリジェンスレポートを作成してください」
