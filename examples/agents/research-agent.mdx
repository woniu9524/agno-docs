title: Research Agent
---

この例では、Web検索機能とプロのジャーナリズムライティングスキルを組み合わせた、洗練されたリサーチエージェントの作成方法を示します。このエージェントは、複数のソースを使用して包括的なリサーチを行い、情報をファクトチェックし、あらゆるトピックについて、よく構成されたNYTスタイルの記事を提供します。

主な機能：
- 複数のソースにわたる高度なWeb検索
- コンテンツの抽出と分析
- クロスリファレンスによる検証
- プロフェッショナルなジャーナリズムライティング
- バランスの取れた客観的な報道

試してみるプロンプトの例：
- 「AIが医療提供と患者の転帰に与える影響を分析してください」
- 「量子コンピューティングにおける最新のブレークスルーについて報告してください」
- 「再生可能エネルギー源への世界的な移行を調査してください」
- 「サイバーセキュリティの脅威と防御の進化を探ってください」
- 「自動運転技術の開発をリサーチしてください」

## Code

```python research_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.newspaper4k import Newspaper4kTools

# 高度なジャーナリズム機能を備えたリサーチエージェントを初期化します
research_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools(), Newspaper4kTools()],
    description=dedent("""\
        あなたはニューヨーク・タイムズで数十年の経験を持つエリート調査報道記者です。
        あなたの専門知識は以下を網羅しています：📰

        - 深い調査リサーチと分析
        - 緻密なファクトチェックと情報源の検証
        - 説得力のある物語構成
        - データに基づいた報道と可視化
        - 専門家インタビューの合成
        - トレンド分析と将来予測
        - 複雑なトピックの単純化
        - 倫理的なジャーナリズムの実践
        - バランスの取れた視点の提示
        - グローバルな文脈の統合\
    """),
    instructions=dedent("""\
        1. リサーチフェーズ 🔍
           - トピックに関する10以上の権威ある情報源を検索してください
           - 最近の出版物と専門家の意見を優先してください
           - 主要な関係者と視点を特定してください

        2. 分析フェーズ 📊
           - 重要な情報を抽出し、検証してください
           - 複数の情報源にわたる事実をクロスリファレンスしてください
           - 新興のパターンとトレンドを特定してください
           - 対立する見解を評価してください

        3. ライティングフェーズ ✍️
           - 注目を集める見出しを作成してください
           - NYTスタイルでコンテンツを構成してください
           - 関連する引用と統計を含めてください
           - 客観性とバランスを維持してください
           - 複雑な概念を明確に説明してください

        4. 品質管理 ✓
           - すべての事実と帰属を検証してください
           - 物語の流れと読みやすさを確保してください
           - 必要に応じてコンテキストを追加してください
           - 将来的な影響を含めてください
    """),
    expected_output=dedent("""\
        # {説得力のある見出し} 📰

        ## エグゼクティブサマリー
        {主要な発見事項と重要性の簡潔な概要}

        ## 背景とコンテキスト
        {歴史的背景と重要性}
        {現在の状況概要}

        ## 主要な発見事項
        {主な発見と分析}
        {専門家の洞察と引用}
        {統計的証拠}

        ## 影響分析
        {現在の影響}
        {関係者の視点}
        {業界/社会への影響}

        ## 将来の展望
        {新興トレンド}
        {専門家の予測}
        {潜在的な課題と機会}

        ## 専門家の洞察
        {著名な業界リーダーからの引用と分析}
        {対立する見解}

        ## 情報源と方法論
        {主要な貢献を含む主要情報源のリスト}
        {リサーチ方法論の概要}

        ---
        AI調査報道記者によるリサーチ
        ニューヨーク・タイムズスタイルレポート
        公開日：{current_date}
        最終更新日：{current_time}\
    """),
    markdown=True,
    show_tool_calls=True,
    add_datetime_to_instructions=True,
)

# 詳細なリサーチリクエストを使用した使用例
if __name__ == "__main__":
    research_agent.print_response(
        "世界中の人工知能規制の現状と将来的な影響を分析してください",
        stream=True,
    )

# 探求すべき高度なリサーチトピック：
"""
テクノロジーとイノベーション：
1. 「2024年における大規模言語モデルの開発と影響を調査してください」
2. 「量子コンピューティングの現状と実用的な応用についてリサーチしてください」

環境と持続可能性：
1. 「革新的な炭素回収技術とその有効性について報告してください」

3. 「サーキュラーエコノミーの実践が世界的な持続可能性に与える影響を分析してください」
4. 「持続可能な航空技術の開発をリサーチしてください」

ヘルスケアとバイオテクノロジー：

2. 「AIが創薬と開発に与える影響を分析してください」
3. 「個別化医療アプローチの進化を調査してください」
4. 「長寿科学とアンチエイジング研究の現状をリサーチしてください」

社会的影響：
1. 「ソーシャルメディアが民主的プロセスに与える影響を検証してください」
2. 「リモートワークが都市開発に与える影響を分析してください」
3. 「ブロックチェーンが金融システムを変革する役割を調査してください」
4. 「デジタルプライバシーとデータ保護措置の進化をリサーチしてください」
"""
```
