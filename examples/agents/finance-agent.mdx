title: Finance Agent
---

この例では、リアルタイムデータを使用して包括的な市場インサイトを提供する高度な金融アナリストを作成する方法を示します。このエージェントは、株式市場データ、アナリストの推奨事項、企業情報、最新ニュースを組み合わせて、プロフェッショナルグレードの金融分析を提供します。

試すためのプロンプト例：
- 「Apple (AAPL) の最新ニュースと財務実績は何ですか？」
- 「Tesla (TSLA) の現在の市場ポジションについて詳細な分析を提供してください」
- 「Microsoft (MSFT) の財務状況はどうですか？アナリストの推奨事項を含めてください」
- 「NVIDIA (NVDA) の株価パフォーマンスと将来の見通しを分析してください」
- 「Amazon (AMZN) の最新四半期について市場は何と言っていますか？」

## コード

```python finance_agent.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools

finance_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        YFinanceTools(
            stock_price=True,
            analyst_recommendations=True,
            stock_fundamentals=True,
            historical_prices=True,
            company_info=True,
            company_news=True,
        )
    ],
    instructions=dedent("""\
        あなたは市場分析に深い専門知識を持つ、経験豊富なウォール街のアナリストです！📊

        包括的な財務分析のために、次の手順に従ってください：
        1. 市場概要
           - 最新の株価
           - 52週間の高値と安値
        2. 詳細な財務分析
           - 主要指標（PER、時価総額、EPS）
        3. プロフェッショナルなインサイト
           - アナリストの推奨事項の内訳
           - 最近のレーティング変更

        4. 市場コンテキスト
           - 業界のトレンドとポジショニング
           - 競合分析
           - 市場センチメント指標

        あなたのレポート作成スタイル：
        - エグゼクティブサマリーから始める
        - データ提示にはテーブルを使用する
        - 明確なセクションヘッダーを含める
        - トレンドを示す絵文字インジケーターを追加する（📈 📉）
        - 主要なインサイトを箇条書きで強調する
        - 業界平均と比較する
        - 技術用語の説明を含める
        - 将来の見通し分析で締めくくる

        リスク開示：
        - 常に潜在的なリスク要因を強調する
        - 市場の不確実性に言及する
        - 関連する規制上の懸念事項を含める
    """),
    add_datetime_to_instructions=True,
    show_tool_calls=True,
    markdown=True,
)

# 詳細な市場分析リクエストを使用した使用例
finance_agent.print_response(
    "What's the latest news and financial performance of Apple (AAPL)?", stream=True
)

# 半導体市場分析の例
finance_agent.print_response(
    dedent("""\
    半導体市場のパフォーマンスを次の点に焦点を当てて分析してください：
    - NVIDIA (NVDA)
    - AMD (AMD)
    - Intel (INTC)
    - Taiwan Semiconductor (TSM)
    それらの市場ポジション、成長指標、将来の見通しを比較してください。"""),
    stream=True,
)

# 自動車市場分析の例
finance_agent.print_response(
    dedent("""\
    自動車業界の現状を評価してください：
    - Tesla (TSLA)
    - Ford (F)
    - General Motors (GM)
    - Toyota (TM)
    EVへの移行の進捗状況と従来の自動車の指標を含めてください。"""),
    stream=True,
)

# 探索するためのその他のプロンプト例：
"""
高度な分析クエリ：
1. 「Tesla の評価指標を従来の自動車メーカーと比較してください」
2. 「最近の製品発売が AMD の株価パフォーマンスに与える影響を分析してください」
3. 「Meta の財務指標をソーシャルメディアの競合他社と比較してください」
4. 「Netflix の加入者数の増加が財務指標に与える影響を評価してください」
5. 「Amazon の収益源とセグメントパフォーマンスの内訳」

業界固有の分析：
半導体市場：
1. 「チップ不足が TSMC の市場ポジションにどのように影響していますか？」
2. 「NVIDIA の AI チップ収益成長を競合他社と比較してください」
3. 「Intel のファウンドリ戦略が株価パフォーマンスに与える影響を分析してください」
4. 「ASML や Applied Materials のような半導体製造装置メーカーを評価してください」

自動車業界：
1. 「EV メーカーの生産指標と利益率を比較してください」
2. 「従来の自動車メーカーの EV への移行の進捗状況を分析してください」
3. 「金利の上昇が自動車販売と株価パフォーマンスにどのように影響していますか？」
4. 「Tesla の収益性指標を従来の自動車メーカーと比較してください」
"""
```
