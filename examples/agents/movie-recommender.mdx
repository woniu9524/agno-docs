---
title: Movie Recommender
---

この例では、ユーザーの好みに基づいて包括的な映画の提案を提供するインテリジェントな映画推薦システムを作成する方法を示します。エージェントは、映画データベース、評価、レビュー、および今後のリリースを組み合わせて、パーソナライズされた映画の推薦を提供します。

試すためのプロンプト例：
- 「インセプションとシャッターアイランドに似たスリラー映画を提案してください」
- 「過去2年間で評価の高いコメディ映画は何ですか？」
- 「パラサイトとオールドボーイに似た韓国映画を見つけてください」
- 「評価の高い家族向けの冒険映画をおすすめしてください」
- 「今後6ヶ月以内に公開されるスーパーヒーロー映画は何ですか？」

## コード

```python movie_recommender.py
"""🎬 ムービーレコメンダー - あなたのパーソナルシネマキュレーター！

この例では、ユーザーの好みに基づいて包括的な映画の提案を提供するインテリジェントな映画推薦システムを作成する方法を示します。エージェントは、映画データベース、評価、レビュー、および今後のリリースを組み合わせて、パーソナライズされた映画の推薦を提供します。

試すためのプロンプト例：
- 「インセプションとシャッターアイランドに似たスリラー映画を提案してください」
- 「過去2年間で評価の高いコメディ映画は何ですか？」
- 「パラサイトとオールドボーイに似た韓国映画を見つけてください」
- 「評価の高い家族向けの冒険映画をおすすめしてください」
- 「今後6ヶ月以内に公開されるスーパーヒーロー映画は何ですか？」

実行：依存関係をインストールするには `pip install openai exa_py agno` を実行してください
"""

from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

movie_recommendation_agent = Agent(
    name="PopcornPal",
    tools=[ExaTools()],
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたはPopcornPal、世界中の映画に精通した情熱的で知識豊富な映画キュレーターです！🎥

        あなたの使命は、ユーザーの好み、視聴履歴、そして最新の映画情報を基に、
        詳細でパーソナライズされた推薦を提供することで、ユーザーが次のお気に入りの映画を見つけるのを支援することです。
        深い映画知識と現在の評価およびレビューを組み合わせて、各視聴者の心に響く映画を提案します。"""),
    instructions=dedent("""\
        各推薦は以下のステップでアプローチしてください：
        1. 分析フェーズ
           - ユーザーの入力を理解し、好みを把握する
           - 言及されたお気に入りの映画のテーマやスタイルを考慮する
           - 特定の要件（ジャンル、評価、言語）を考慮に入れる

        2. 検索とキュレーション
           - Exaを使用して関連映画を検索する
           - 推薦の多様性を確保する
           - すべての映画データが最新かつ正確であることを確認する

        3. 詳細情報
           - 映画のタイトルと公開年
           - ジャンルとサブジャンル
           - IMDB評価（7.5以上の評価に焦点を当てる）
           - 上映時間と主要言語
           - 短く魅力的なあらすじ
           - コンテンツアドバイザリー/年齢評価
           - 著名なキャストと監督

        4. 追加機能
           - 関連する予告編があれば含める
           - 同様のジャンルの今後のリリースを提案する
           - 分かればストリーミングの利用可能性に言及する

        プレゼンテーションスタイル：
        - 明確なマークダウンフォーマットを使用する
        - 主要な推薦は構造化されたテーブルで提示する
        - 同様の映画をグループ化する
        - ジャンルごとに絵文字インジケーターを追加する（🎭 🎬 🎪）
        - クエリごとに最低5つの推薦を含める
        - 各推薦について簡単な説明を含める
    """),
    markdown=True,
    add_datetime_to_instructions=True,
    show_tool_calls=True,
)

# さまざまな種類の映画クエリを使用した使用例
movie_recommendation_agent.print_response(
    "IMDBで8以上の評価を持つスリラー映画をいくつか提案してください。 "
    "以前のお気に入りのスリラー映画は、ダークナイト、ヴェノム、パラサイト、シャッターアイランドです。",
    stream=True,
)

# 探求するための追加のプロンプト例：
"""
ジャンル別クエリ：
1. 「ブラック・スワンやゴーン・ガールに似たサイコスリラーを見つけてください」
2. 「スタジオジブリの最高の長編アニメーション映画は何ですか？」
3. 「インセプションやインターステラーのような心を揺さぶるSF映画をおすすめしてください」
4. 「過去5年間で評価の高い犯罪ドキュメンタリーは何ですか？」

国際的な映画：
1. 「パラサイトや新感染半島エクスプレスに似た韓国映画を提案してください」
2. 「過去10年間の必見のフランス映画は何ですか？」
3. 「大人向けの日本のアニメ映画をおすすめしてください」
4. 「受賞歴のあるヨーロッパのドラマ映画を見つけてください」

家族やグループでの鑑賞：
1. 「8歳から12歳の子ども向けの家族映画は何ですか？」
2. 「グループ映画の夜に最適なコメディ映画を提案してください」
3. 「10代に適した教育ドキュメンタリーを見つけてください」
4. 「大人も子供も楽しめる冒険映画をおすすめしてください」

今後のリリース：
1. 「来月公開される最も期待されている映画は何ですか？」
2. 「今後のスーパーヒーロー映画のリリースを見せてください」
3. 「このハロウィンの季節に公開されるホラー映画は何ですか？」
4. 「今後の書籍から映画化された作品をリストアップしてください」
"""
```
