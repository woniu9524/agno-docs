---
title: 电影推荐器
---

本示例展示了如何创建一个智能电影推荐系统，该系统可根据您的偏好提供全面的电影建议。该代理结合了电影数据库、评分、评论和即将上映的影片，以提供个性化的电影推荐。

可尝试的示例提示：
- "推荐类似《盗梦空间》和《禁闭岛》的惊悚片"
- "过去两年评分最高的喜剧片是什么？"
- "找一些类似《寄生虫》和《老男孩》的韩国电影"
- "推荐评分高的适合家庭观看的冒险电影"
- "未来 6 个月有哪些即将上映的超级英雄电影？"

## 代码

```python movie_recommender.py
"""🎬 电影推荐器 - 您的专属电影策展人！

本示例展示了如何创建一个智能电影推荐系统，该系统可根据您的偏好提供全面的电影建议。该代理结合了电影数据库、评分、评论和即将上映的影片，以提供个性化的电影推荐。

可尝试的示例提示：
- "推荐类似《盗梦空间》和《禁闭岛》的惊悚片"
- "过去两年评分最高的喜剧片是什么？"
- "找一些类似《寄生虫》和《老男孩》的韩国电影"
- "推荐评分高的适合家庭观看的冒险电影"
- "未来 6 个月有哪些即将上映的超级英雄电影？"

运行：`pip install openai exa_py agno` 安装依赖项
"""

from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

movie_recommendation_agent = Agent(
    name="PopcornPal",
    tools=[ExaTools()],
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        您是 PopcornPal，一位充满热情且知识渊博的电影策展人，精通全球电影！🎥

        您的使命是通过根据用户的偏好、观看历史和最新电影动态，提供详细的个性化推荐，帮助用户发现他们下一部最喜欢的电影。您将深厚的电影知识与当前的评分和评论相结合，为每位观众推荐真正能引起共鸣的电影。"""),
    instructions=dedent("""\
        在每次推荐时都遵循以下步骤：
        1. 分析阶段
           - 理解用户输入中的偏好
           - 考虑提到的喜爱电影的主题和风格
           - 考虑任何特定要求（类型、评分、语言）

        2. 搜索与策展
           - 使用 Exa 搜索相关电影
           - 确保推荐的多样性
           - 验证所有电影数据是否最新且准确

        3. 详细信息
           - 电影名称和发行年份
           - 类型和子类型
           - IMDB 评分（重点关注评分 7.5+ 的影片）
           - 片长和主要语言
           - 简短、引人入胜的情节摘要
           - 内容咨询/年龄分级
           - 主要演员和导演

        4. 附加功能
           - 在可用时包含相关预告片
           - 推荐相似类型的即将上映的影片
           - 在已知时提及流媒体可用性

        呈现风格：
        - 使用清晰的 Markdown 格式
        - 以结构化的表格呈现主要推荐
        - 将相似的电影分组
        - 使用表情符号指示类型（🎭 🎬 🎪）
        - 每条查询至少推荐 5 部电影
        - 为每次推荐包含简要说明
    """),
    markdown=True,
    add_datetime_to_instructions=True,
    show_tool_calls=True,
)

# 使用不同类型的电影查询进行示例用法
movie_recommendation_agent.print_response(
    "推荐一些 IMDB 评分在 8 分及以上的惊悚片。我之前最喜欢的惊悚片是《蝙蝠侠：黑暗骑士》、《毒液》、《寄生虫》、《禁闭岛》。",
    stream=True,
)

# 更多示例提示供探索：
"""
特定类型查询：
1. "找一些类似《黑天鹅》和《消失的爱人》的心理惊悚片"
2. "吉卜力工作室有哪些最佳动画电影？"
3. "推荐一些类似《盗梦空间》和《星际穿越》的烧脑科幻片"
4. "过去 5 年评分最高的犯罪纪录片是什么？"

国际电影：
1. "推荐一些类似《寄生虫》和《釜山行》的韩国电影"
2. "过去十年必看的法国电影有哪些？"
3. "推荐适合成人的日本动画电影"
4. "找一些获奖的欧洲剧情片"

家庭与团体观看：
1. "有哪些适合 8-12 岁儿童的家庭电影？"
2. "推荐适合团体电影之夜的喜剧片"
3. "找一些适合青少年的教育纪录片"
4. "推荐适合成人和儿童一起观看的冒险电影"

即将上映的影片：
1. "下个月即将上映的最受期待的电影有哪些？"
2. "展示即将上映的超级英雄电影"
3. "这个万圣节季有哪些恐怖电影上映？"
4. "列出即将改编自书籍的电影"
"""
```
