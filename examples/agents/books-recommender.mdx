---
title: 書籍レコメンデーション
---

この例では、あなたの好みに基づいて包括的な文学的な提案を提供する、インテリジェントな書籍推薦システムを作成する方法を示します。エージェントは、書籍データベース、評価、レビュー、および今後のリリースを組み合わせて、パーソナライズされた読書のおすすめを提供します。

試してみるプロンプトの例：
- 「Evelyn Hugoの七人の夫」と「Daisy Jones & The Six」が大好きでした。次に何を読めばいいですか？
- 「ゴーン・ガール」や「サイレント・ペイシェント」のような心理スリラーを推薦してください。
- 過去2年間にリリースされた最高のファンタジー書籍は何ですか？
- 強い女性主人公が登場する歴史小説が好きですが、何か提案はありますか？
- 「ヘンリエッタ・ラックスの生きた証」のような小説のように読める科学の本を探しています。

## コード

```python books_recommender.py
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.exa import ExaTools

book_recommendation_agent = Agent(
    name="Shelfie",
    tools=[ExaTools()],
    model=OpenAIChat(id="gpt-4o"),
    description=dedent("""\
        あなたはShelfie、世界中の書籍に関する専門知識を持つ、情熱的で知識豊富な文学キュレーターです！📚

        あなたの使命は、読者が最新の文学や読者の好み、読書履歴に基づいて、詳細でパーソナライズされた推薦を提供することで、次のお気に入りの本を見つけるのを助けることです。深い文学的知識と現在の評価やレビューを組み合わせて、読者に本当に響く書籍を提案します。"""),
    instructions=dedent("""\
        各推薦には、これらのステップでアプローチしてください：

        1. 分析フェーズ 📖
           - 入力から読者の好みを理解する
           - 言及されたお気に入りの本のテーマとスタイルを考慮する
           - 特定の要件（ジャンル、長さ、コンテンツに関する注意）を考慮に入れる

        2. 検索とキュレーション 🔍
           - Exaを使用して関連書籍を検索する
           - 推薦の多様性を確保する
           - すべての書籍データが最新かつ正確であることを確認する

        3. 詳細情報 📝
           - 書籍のタイトルと著者
           - 出版年
           - ジャンルとサブジャンル
           - Goodreads/StoryGraphの評価
           - ページ数
           - 簡潔で魅力的なあらすじ
           - コンテンツに関する注意喚起
           - 受賞歴と評価

        4. 追加機能 ✨
           - シリーズ情報があれば含める
           - 似たような著者をおすすめする
           - オーディオブックの利用可能性に言及する
           - 今後の適応（映像化など）に注意を払う

        プレゼンテーションスタイル：
        - 明確なMarkdownフォーマットを使用する
        - 主要な推薦を構造化されたテーブルで提示する
        - 似たような本をグループ化する
        - ジャンルごとに絵文字インジケーターを追加する（📚 🔮 💕 🔪）
        - クエリごとに最低5つの推薦を含める
        - 各推薦について簡単な説明を含める
        - 著者と視点の多様性を強調する
        - 関連する場合はトリガー警告を記載する"""),
    markdown=True,
    add_datetime_to_instructions=True,
    show_tool_calls=True,
)

# さまざまな種類の書籍クエリを使用した例の使用方法
book_recommendation_agent.print_response(
    "「Anxious People」と「Lessons in Chemistry」がとても気に入りました。似たような本をいくつか提案してもらえますか？",
    stream=True,
)

# 探求するための追加の例プロンプト：
"""
ジャンル別のクエリ：
1. 「Beautiful World, Where Are You」のような現代文学フィクションを推薦してください。
2. 過去5年間で完了した最高のファンタジーシリーズは何ですか？
3. 「Mexican Gothic」や「Ninth House」のような雰囲気のあるゴシック小説を見つけてください。
4. 今年の最も評価の高いデビュー小説は何ですか？

現代的な問題：
1. あまり落ち込まない気候変動に関する本を提案してください。
2. テクニカルでない読者向けの人工知能に関する最高の書籍は何ですか？
3. 移民体験に関する回顧録を推薦してください。
4. 希望に満ちた結末を持つメンタルヘルスに関する本を見つけてください。

ブッククラブの選択：
1. ディスカッションを促す良いブッククラブの選択肢は何ですか？
2. 350ページ未満の文学フィクションを提案してください。
3. 現在の社会問題に取り組む、考えさせられる小説を見つけてください。
4. 複数の視点/ナレーションを持つ本を推薦してください。

今後のリリース：
1. 来月最も期待されている文学リリースは何ですか？
2. 私のお気に入りの著者からの今後のリリースを表示してください。
3. 今シーズン話題になっているデビュー小説は何ですか？
4. スクリーン適応される今後の書籍をリストアップしてください。
"""
```
