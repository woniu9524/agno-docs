title: "讨论团队"
---

此示例展示了如何创建一个允许多个代理就某个主题进行协作的讨论团队。

## 代码

```python discussion_team.py
import asyncio
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.arxiv import ArxivTools
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.googlesearch import GoogleSearchTools
from agno.tools.hackernews import HackerNewsTools

reddit_researcher = Agent(
    name="Reddit 研究员",
    role="在 Reddit 上研究某个主题",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools()],
    add_name_to_instructions=True,
    instructions=dedent("""
    你是一名 Reddit 研究员。
    你将获得一个需要研究的主题。
    你需要找到 Reddit 上最相关的帖子。
    """),
)

hackernews_researcher = Agent(
    name="HackerNews 研究员",
    model=OpenAIChat("gpt-4o"),
    role="在 HackerNews 上研究某个主题。",
    tools=[HackerNewsTools()],
    add_name_to_instructions=True,
    instructions=dedent("""
    你是一名 HackerNews 研究员。
    你将获得一个需要研究的主题。
    你需要找到 HackerNews 上最相关的帖子。
    """),
)

academic_paper_researcher = Agent(
    name="学术论文研究员",
    model=OpenAIChat("gpt-4o"),
    role="研究学术论文和学术内容",
    tools=[GoogleSearchTools(), ArxivTools()],
    add_name_to_instructions=True,
    instructions=dedent("""
    你是一名学术论文研究员。
    你将获得一个需要研究的主题。
    你需要找到相关的学术文章、论文和学术讨论。
    专注于同行评审的内容和信誉良好来源的引用。
    简要总结关键发现和研究方法。
    """),
)

twitter_researcher = Agent(
    name="Twitter 研究员",
    model=OpenAIChat("gpt-4o"),
    role="研究热门讨论和实时更新",
    tools=[DuckDuckGoTools()],
    add_name_to_instructions=True,
    instructions=dedent("""
    你是一名 Twitter/X 研究员。
    你将获得一个需要研究的主题。
    你需要找到热门讨论、有影响力的人物和实时更新。
    尽可能关注经过验证的账户和可信来源。
    跟踪相关的标签和正在进行的对话。
    """),
)


agent_team = Team(
    name="讨论团队",
    mode="collaborate",
    model=OpenAIChat("gpt-4o"),
    members=[
        reddit_researcher,
        hackernews_researcher,
        academic_paper_researcher,
        twitter_researcher,
    ],
    instructions=[
        "你是一名讨论大师。",
        "当你认为团队已达成共识时，你需要停止讨论。",
    ],
    success_criteria="团队已达成共识。",
    enable_agentic_context=True,
    update_team_context=True,
    show_tool_calls=True,
    markdown=True,
    debug_mode=True,
    show_members_responses=True,
)

if __name__ == "__main__":
    asyncio.run(
        agent_team.print_response(
            message="开始讨论主题：“学习编程的最佳方式是什么？”",
            stream=True,
            stream_intermediate_steps=True,
        )
    )
```
