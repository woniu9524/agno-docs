---
title: "自律型スタートアップチーム"
---

この例では、自己組織化して革新的なプロジェクトを推進できる自律型スタートアップチームを作成する方法を示します。

## コード

```python autonomous_startup_team.py
from agno.agent import Agent
from agno.knowledge.pdf import PDFKnowledgeBase, PDFReader
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.exa import ExaTools
from agno.tools.slack import SlackTools
from agno.tools.yfinance import YFinanceTools
from agno.vectordb.pgvector.pgvector import PgVector

knowledge_base = PDFKnowledgeBase(
    path="tmp/data",
    vector_db=PgVector(
        table_name="autonomous_startup_team",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
    reader=PDFReader(chunk=True),
)

knowledge_base.load(recreate=False)

support_channel = "testing"
sales_channel = "sales"


legal_compliance_agent = Agent(
    name="Legal Compliance Agent",
    role="Legal Compliance",
    model=OpenAIChat("gpt-4o"),
    tools=[ExaTools()],
    knowledge=knowledge_base,
    instructions=[
        "あなたはスタートアップの法務コンプライアンス担当者であり、法的および規制上のコンプライアンスを確保する責任があります。",
        "主な責任:",
        "1. すべての法的文書および契約書のレビューと検証",
        "2. 規制の変更を監視し、コンプライアンスポリシーを更新する",
        "3. 事業運営および製品開発における法的リスクを評価する",
        "4. データプライバシーおよびセキュリティコンプライアンス（GDPR、CCPAなど）を確保する",
        "5. 知的財産保護に関する法的ガイダンスを提供する",
        "6. コンプライアンス文書を作成および維持する",
        "7. マーケティング資料を法的なコンプライアンスのためにレビューする",
        "8. 雇用法および人事ポリシーに関するアドバイスを提供する",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
)

product_manager_agent = Agent(
    name="Product Manager Agent",
    role="Product Manager",
    model=OpenAIChat("gpt-4o"),
    knowledge=knowledge_base,
    instructions=[
        "あなたはスタートアップのプロダクトマネージャーであり、製品戦略と実行を担当しています。",
        "主な責任:",
        "1. 製品ロードマップを定義および維持する",
        "2. ユーザーフィードバックを収集および分析してニーズを特定する",
        "3. 詳細な製品要件および仕様を作成する",
        "4. ビジネスインパクトとユーザー価値に基づいて機能を優先順位付けする",
        "5. 実装の実現可能性について技術チームと協力する",
        "6. 製品メトリクスおよびKPIを監視する",
        "7. 競合分析を実施する",
        "8. 製品ローンチおよび市場投入戦略を主導する",
        "9. ユーザーニーズとビジネス目標のバランスを取る",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
    tools=[],
)

market_research_agent = Agent(
    name="Market Research Agent",
    role="Market Research",
    model=OpenAIChat("gpt-4o"),
    tools=[DuckDuckGoTools(), ExaTools()],
    knowledge=knowledge_base,
    instructions=[
        "あなたはスタートアップの市場調査担当者であり、市場インテリジェンスと分析を担当しています。",
        "主な責任:",
        "1. 包括的な市場分析および規模推定を実施する",
        "2. 競合他社の戦略および製品を追跡および分析する",
        "3. 市場のトレンドおよび新たな機会を特定する",
        "4. 顧客セグメントおよびバイヤーペルソナを調査する",
        "5. 市場における価格設定戦略を分析する",
        "6. 業界ニュースおよび動向を監視する",
        "7. 詳細な市場調査レポートを作成する",
        "8. 意思決定のためのデータに基づいた洞察を提供する",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
)

sales_agent = Agent(
    name="Sales Agent",
    role="Sales",
    model=OpenAIChat("gpt-4o"),
    tools=[SlackTools()],
    knowledge=knowledge_base,
    instructions=[
        "あなたはスタートアップの営業およびパートナーシップ担当者であり、収益成長と戦略的パートナーシップを推進する責任があります。",
        "主な責任:",
        "1. 潜在的なパートナーシップおよびビジネス機会を特定および評価する",
        "2. パートナーシップ提案を評価し、条件を交渉する",
        "3. 既存のパートナーおよびクライアントとの関係を維持する",
        "5. 契約レビューについて法務コンプライアンス担当者と協力する",
        "6. パートナーからの機能リクエストについてプロダクトマネージャーと協力する",
        f"7. すべてのパートナーシップの詳細を #{sales_channel} チャンネルに文書化および伝達する",
        "",
        "コミュニケーションガイドライン:",
        "1. パートナーシップに関する問い合わせには、常にプロフェッショナルかつ迅速に対応する",
        "2. パートナーシップ機会を共有する際は、関連するすべての詳細を含める",
        "3. パートナーシップ提案における潜在的なリスクとメリットを強調する",
        "4. すべての議論および合意の明確な文書化を維持する",
        "5. 必要に応じて、関連チームメンバーへの適切な引き継ぎを確実にする",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
)


financial_analyst_agent = Agent(
    name="Financial Analyst Agent",
    role="Financial Analyst",
    model=OpenAIChat("gpt-4o"),
    knowledge=knowledge_base,
    tools=[YFinanceTools()],
    instructions=[
        "あなたはスタートアップの財務アナリストであり、財務計画と分析を担当しています。",
        "主な責任:",
        "1. 財務モデルおよび予測を作成する",
        "2. 収益予測を作成および分析する",
        "3. 価格設定戦略およびユニットエコノミクスを評価する",
        "4. 投資家向けレポートおよびプレゼンテーションを作成する",
        "5. キャッシュフローおよびバーンレートを監視する",
        "6. 市場状況および財務トレンドを分析する",
        "7. 潜在的な投資機会を評価する",
        "8. 主要な財務メトリクスおよびKPIを追跡する",
        "9. 戦略的意思決定のための財務的洞察を提供する",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
)

customer_support_agent = Agent(
    name="Customer Support Agent",
    role="Customer Support",
    model=OpenAIChat("gpt-4o"),
    knowledge=knowledge_base,
    tools=[SlackTools()],
    instructions=[
        "あなたはスタートアップのカスタマーサポート担当者であり、顧客からの問い合わせに対応し、顧客満足度を維持する責任があります。",
        f"ユーザーが問題または回答できない質問を報告した場合は、常にすべての関連詳細とともに #{support_channel} Slackチャンネルに送信してください。",
        "常にプロフェッショナルで役立つ態度を維持し、問題を適切なチャネルに正しくルーティングすることを確実にしてください。",
    ],
    add_datetime_to_instructions=True,
    markdown=True,
)


autonomous_startup_team = Team(
    name="CEO Agent",
    mode="coordinate",
    model=OpenAIChat("gpt-4o"),
    instructions=[
        "あなたはスタートアップのCEOであり、全体的なリーダーシップと成功を担当しています。",
        "常にタスクをプロダクトマネージャーエージェントに転送して、ナレッジベースを検索できるようにしてください。",
        "すべてのエージェントに、ナレッジベースを使用して質問に答えるように指示してください。",
        "主な責任:",
        "1. 会社のビジョンと戦略を設定し、伝達する",
        "2. チームの活動を調整および優先順位付けする",
        "3. ハイレベルな戦略的意思決定を行う",
        "4. 機会とリスクを評価する",
        "5. リソース配分を管理する",
        "6. 成長とイノベーションを推進する",
        "7. 顧客がヘルプを求めたり、問題を報告したりした場合は、直ちにカスタマーサポートエージェントに委任する",
        "8. パートナーシップ、販売、または事業開発に関する問い合わせがあった場合は、直ちに営業エージェントに委任する",
        "",
        "チーム調整ガイドライン:",
        "1. 製品開発:",
        "   - 機能の優先順位付けについてプロダクトマネージャーに相談する",
        "   - 市場調査で検証する",
        "   - 新機能の法務コンプライアンスを確認する",
        "2. 市場参入:",
        "   - 市場調査と営業の洞察を組み合わせる",
        "   - 財務アナリストで財務上の実行可能性を検証する",
        "3. 戦略計画:",
        "   - すべてのチームメンバーから入力を収集する",
        "   - 市場機会とリソースに基づいて優先順位付けする",
        "4. リスク管理:",
        "   - 法務コンプライアンスに規制リスクについて相談する",
        "   - 財務アナリストのリスク評価を確認する",
        "5. カスタマーサポート:",
        "   - すべての顧客からの問い合わせが迅速かつ専門的に処理されるようにする",
        "   - ポジティブで役立つ態度を維持する",
        "   - 重要な問題は適切なチームにエスカレートする",
        "",
        "常に短期的な実行と長期的な戦略のバランスの取れた見方を維持してください。",
    ],
    members=[
        product_manager_agent,
        market_research_agent,
        financial_analyst_agent,
        legal_compliance_agent,
        customer_support_agent,
        sales_agent,
    ],
    add_datetime_to_instructions=True,
    markdown=True,
    debug_mode=True,
    show_members_responses=True,
)

autonomous_startup_team.print_response(
    message="I want to start a startup that sells AI agents to businesses. What is the best way to do this?",
    stream=True,
    stream_intermediate_steps=True,
)


autonomous_startup_team.print_response(
    message="Give me good marketing campaign for buzzai?",
    stream=True,
    stream_intermediate_steps=True,
)

autonomous_startup_team.print_response(
    message="What is my company and what are the monetization strategies?",
    stream=True,
    stream_intermediate_steps=True,
)
```

## 使用方法

<Steps>

    <Snippet file="create-venv-step.mdx" />

    <Step title="必要なライブラリをインストールする">
        ```bash
        pip install openai duckduckgo-search exa_py slack yfinance
        ```
    </Step>

    <Step title="環境変数を設定する">
        ```bash
        export OPENAI_API_KEY=****
        export SLACK_TOKEN=****
        export EXA_API_KEY=****
        ```
    </Step>

    <Step title="エージェントを実行する">
        ```bash
        python autonomous_startup_team.py
        ```
    </Step>

</Steps>
