---
title: "AI 支持团队"
---

此示例演示了如何创建一个 AI 支持团队，该团队可以根据客户咨询的性质将其路由给合适的座席。

## 代码

```python ai_support_team.py

from agno.agent import Agent
from agno.knowledge.website import WebsiteKnowledgeBase
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.exa import ExaTools
from agno.tools.slack import SlackTools
from agno.vectordb.pgvector.pgvector import PgVector

knowledge_base = WebsiteKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    # Follow links from the seed URLs
    max_links=10,
    # Table name: ai.website_documents
    vector_db=PgVector(
        table_name="website_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
knowledge_base.load(recreate=False)
support_channel = "testing"
feedback_channel = "testing"

doc_researcher_agent = Agent(
    name="文档研究员代理",
    role="在知识库中搜索信息",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools(), ExaTools()],
    knowledge=knowledge_base,
    search_knowledge=True,
    instructions=[
        "你是给定产品的文档专家。请彻底搜索知识库以回答用户问题。",
        "始终根据文档提供准确信息。",
        "如果问题匹配常见问题解答，请提供文档中的具体常见问题解答。",
        "在相关时，包含指向解决用户问题的特定文档页面的直接链接。",
        "如果你不确定答案，请承认并建议用户可以在哪里找到更多信息。",
        "使用标题、项目符号和代码示例（如果适用）清晰地格式化你的回复。",
        "始终验证你的答案是否直接解决了用户的具体问题。",
        "如果你在文档知识库中找不到答案，请使用 DuckDuckGoTools 或 ExaTools 搜索网络以获取相关信息来回答用户的问题。",
    ],
)


escalation_manager_agent = Agent(
    name="升级经理代理",
    role="将问题升级到 Slack 频道",
    model=OpenAIChat(id="gpt-4o"),
    tools=[SlackTools()],
    instructions=[
        "你是负责将关键问题路由给支持团队的升级经理。",
        f"当用户报告问题时，请始终使用 send_message 工具包函数将其以及所有相关详细信息发送到 #{support_channel} Slack 频道。",
        "包含用户的姓名、联系信息（如果可用）以及问题的清晰描述。",
        "升级问题后，回复用户确认他们的问题已升级。",
        "你的回复应专业且令人放心，让他们知道支持团队将很快处理。",
        "始终包含工单或参考号（如果可用），以帮助用户跟踪他们的问题。",
        "切勿尝试自行解决技术问题 - 你的角色严格是升级和沟通。",
    ],
)

feedback_collector_agent = Agent(
    name="反馈收集员代理",
    role="从用户那里收集反馈",
    model=OpenAIChat(id="gpt-4o"),
    tools=[SlackTools()],
    description="你是一个可以从用户那里收集反馈的 AI 代理。",
    instructions=[
        "你负责收集用户对产品或功能请求的反馈。",
        f"当用户提供反馈或建议功能时，请使用 Slack 工具包使用 send_message 工具包函数将其发送到 #{feedback_channel} 频道。",
        "在你的 Slack 消息中包含用户反馈的所有相关详细信息。",
        "将反馈发送到 Slack 后，请专业地回复用户，感谢他们的输入。",
        "你的回复应承认他们的反馈，并向他们保证他们的反馈将被考虑。",
        "在语气上要热情和感激，因为用户反馈对于改进我们的产品非常宝贵。",
        "不要承诺具体的时间表或保证他们的建议将被实施。",
    ],
)


customer_support_team = Team(
    name="客户支持团队",
    mode="route",
    model=OpenAIChat("gpt-4.5-preview"),
    enable_team_history=True,
    members=[doc_researcher_agent, escalation_manager_agent, feedback_collector_agent],
    show_tool_calls=True,
    markdown=True,
    debug_mode=True,
    show_members_responses=True,
    instructions=[
        "你是负责分类和路由客户咨询的首席客户支持座席。",
        "仔细分析每条用户消息，并确定它是：需要文档研究的问题、需要升级的错误报告，还是产品反馈。",
        "对于有关产品的常规问题，请路由给文档研究员代理，由其搜索文档以获取答案。",
        "如果文档研究员代理找不到问题的答案，请将其升级给升级经理代理。",
        "对于错误报告或技术问题，请立即路由给升级经理代理。",
        "对于功能请求或产品反馈，请路由给反馈收集员代理。",
        "始终提供关于你为何将咨询路由给特定代理的清晰解释。",
        "在收到来自相应代理的回复后，以专业且有帮助的方式将该信息传达给用户。",
        "通过在整个对话中保持上下文，确保用户获得无缝的体验。",
    ],
)

# 添加查询，代理将其重定向到相应的代理
customer_support_team.print_response(
    "嗨，团队，我想构建一个允许模型访问大量学习材料的教育平台，Agno 平台如何帮助我构建这个平台？",
    stream=True,
)

```

        python ai_support_team.py
