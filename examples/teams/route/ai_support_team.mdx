---
title: "AI サポートチーム"
---

この例では、問い合わせの内容に基づいて適切な担当者に顧客からの問い合わせをルーティングできる AI サポートチームを作成する方法を示します。

## コード

```python ai_support_team.py

from agno.agent import Agent
from agno.knowledge.website import WebsiteKnowledgeBase
from agno.models.openai import OpenAIChat
from agno.team.team import Team
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.exa import ExaTools
from agno.tools.slack import SlackTools
from agno.vectordb.pgvector.pgvector import PgVector

knowledge_base = WebsiteKnowledgeBase(
    urls=["https://docs.agno.com/introduction"],
    # Followするリンクの数
    max_links=10,
    # テーブル名: ai.website_documents
    vector_db=PgVector(
        table_name="website_documents",
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
    ),
)
knowledge_base.load(recreate=False)
support_channel = "testing"
feedback_channel = "testing"

doc_researcher_agent = Agent(
    name="ドキュメント調査エージェント",
    role="ナレッジベースを検索して情報を探す",
    model=OpenAIChat(id="gpt-4o"),
    tools=[DuckDuckGoTools(), ExaTools()],
    knowledge=knowledge_base,
    search_knowledge=True,
    instructions=[
        "あなたは、指定された製品のドキュメント専門家です。ユーザーの質問に答えるために、ナレッジベースを徹底的に検索してください。",
        "常にドキュメントに基づいた正確な情報を提供してください。",
        "質問がFAQに一致する場合は、ドキュメントから特定のFAQの回答を提供してください。",
        "関連する場合は、ユーザーの質問に対応する特定のドキュメントページへの直接リンクを含めてください。",
        "回答が不確かな場合は、それを認め、ユーザーがさらに情報を見つけられる可能性のある場所を提案してください。",
        "見出し、箇条書き、および適切な場合はコード例を使用して、応答を明確にフォーマットしてください。",
        "常に、あなたの回答がユーザーの特定の質問に直接対応していることを確認してください。",
        "ドキュメントナレッジベースで回答が見つからない場合は、DuckDuckGoToolsまたはExaToolsを使用して、ユーザーの質問に答えるためにウェブを検索してください。",
    ],
)


escalation_manager_agent = Agent(
    name="エスカレーションマネージャーエージェント",
    role="問題をSlackチャンネルにエスカレートする",
    model=OpenAIChat(id="gpt-4o"),
    tools=[SlackTools()],
    instructions=[
        "あなたは、重要な問題をサポートチームにエスカレートする責任を負うエスカレーションマネージャーです。",
        f"ユーザーが問題を報告するときは、常に `send_message` ツール関数を使用して、関連するすべての詳細とともに #{support_channel} Slackチャンネルに送信してください。",
        "ユーザーの名前、連絡先情報（利用可能な場合）、および問題の明確な説明を含めてください。",
        "問題をエスカレートした後、問題がエスカレートされたことを確認する応答をユーザーに送信してください。",
        "あなたの応答はプロフェッショナルで安心できるものであり、サポートチームがすぐにそれに対処することをユーザーに知らせるべきです。",
        "ユーザーが問題を追跡するのに役立つように、常にチケットまたは参照番号を含めてください。",
        "技術的な問題を自分で解決しようとしないでください。あなたの役割は厳密にエスカレートとコミュニケーションです。",
    ],
)

feedback_collector_agent = Agent(
    name="フィードバックコレクターエージェント",
    role="ユーザーからフィードバックを収集する",
    model=OpenAIChat(id="gpt-4o"),
    tools=[SlackTools()],
    description="あなたは、ユーザーからフィードバックを収集できるAIエージェントです。",
    instructions=[
        "あなたは、製品や機能リクエストに関するユーザーフィードバックを収集する責任があります。",
        f"ユーザーがフィードバックを提供したり、機能を提案したりするときは、Slackツールを使用して `send_message` ツール関数で #{feedback_channel} チャンネルに送信してください。",
        "ユーザーのフィードバックから関連するすべての詳細をSlackメッセージに含めてください。",
        "フィードバックをSlackに送信した後、入力に感謝するプロフェッショナルな応答をユーザーに送信してください。",
        "あなたの応答は、フィードバックを認め、それが考慮されることを保証するべきです。",
        "ユーザーフィードバックは製品改善に貴重であるため、温かく感謝の気持ちを込めてください。",
        "特定のタイムラインを約束したり、提案が実装されることを保証したりしないでください。",
    ],
)


customer_support_team = Team(
    name="カスタマーサポートチーム",
    mode="route",
    model=OpenAIChat("gpt-4.5-preview"),
    enable_team_history=True,
    members=[doc_researcher_agent, escalation_manager_agent, feedback_collector_agent],
    show_tool_calls=True,
    markdown=True,
    debug_mode=True,
    show_members_responses=True,
    instructions=[
        "あなたは、顧客からの問い合わせを分類およびルーティングする責任を負うリードカスタマーサポートエージェントです。",
        "各ユーザーメッセージを注意深く分析し、それがドキュメント調査が必要な質問、エスカレーションが必要なバグレポート、または製品フィードバックのいずれであるかを判断してください。",
        "製品に関する一般的な質問については、ドキュメントを検索して回答を見つける `doc_researcher_agent` にルーティングしてください。",
        "ドキュメント調査エージェントが質問への回答を見つけられない場合は、`escalation_manager_agent` にエスカレートしてください。",
        "バグレポートまたは技術的な問題については、直ちに `escalation_manager_agent` にルーティングしてください。",
        "機能リクエストまたは製品フィードバックについては、`feedback_collector_agent` にルーティングしてください。",
        "問い合わせを特定の担当者にルーティングする理由を常に明確に説明してください。",
        "適切な担当者から応答を受け取った後、その情報をプロフェッショナルで役立つ方法でユーザーに伝えてください。",
        "会話全体でコンテキストを維持することにより、ユーザーにとってシームレスなエクスペリエンスを確保してください。",
    ],
)

# クエリを追加し、エージェントが適切なエージェントにリダイレクトします
customer_support_team.print_response(
    "こんにちはチーム、モデルが大量の教材にアクセスできる教育プラットフォームを構築したいのですが、Agnoプラットフォームはどのように役立ちますか？",
    stream=True,
)

```

## 使用方法

<Steps>

    <Snippet file="create-venv-step.mdx" />

    <Step title="必要なライブラリをインストールする">
        ```bash
        pip install openai duckduckgo-search slack_sdk exa_py
        ```
    </Step>

    <Step title="環境変数を設定する">
        ```bash
        export OPENAI_API_KEY=****
        export SLACK_TOKEN=****
        export EXA_API_KEY=****
        ```
    </Step>

    <Step title="エージェントを実行する">
        ```bash
        python ai_support_team.py
        ```
    </Step>
