## コード

```python
from agno.agent import Agent
from agno.document.base import Document
from agno.knowledge.document import DocumentKnowledgeBase
from agno.vectordb.pgvector import PgVector

fun_facts = """
- 地球は太陽から3番目の惑星であり、生命を支える唯一知られている天体です。
- 地球の表面の約71%は水で覆われており、太平洋が最大です。
- 地球の大気は主に窒素（78%）と酸素（21%）で構成されており、微量の他のガスが含まれています。
- 地球は24時間ごとに軸を中心に1回転し、昼夜のサイクルをもたらします。
- 地球には1つの自然衛星である月があり、潮汐に影響を与え、地球の自転軸の傾きを安定させます。
- 地球の地殻変動は絶えず移動しており、地震や火山噴火などの地質活動を引き起こします。
- 地球で最も高い地点はエベレスト山で、海抜8,848メートル（29,029フィート）です。
- 海で最も深い場所はマリアナ海溝で、深さは11,000メートル（36,000フィート）を超えます。
- 地球には、熱帯雨林や砂漠からサンゴ礁やツンドラまで、多様な生態系があります。
- 地球の磁場は、有害な太陽放射線や宇宙線から身を守ることで生命を保護します。
"""

# data/docs ディレクトリからドキュメントをロードします
documents = [Document(content=fun_facts)]

# データベース接続URL
db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

# ロードされたドキュメントで知識ベースを作成します
knowledge_base = DocumentKnowledgeBase(
    documents=documents,
    vector_db=PgVector(
        table_name="documents",
        db_url=db_url,
    ),
)

# 知識ベースをロードします
knowledge_base.load(recreate=False)

# 知識ベースでエージェントを作成します
agent = Agent(
    knowledge=knowledge_base,
)

# エージェントに知識ベースについて質問します
agent.print_response(
    "Ask me about something from the knowledge base about earth", markdown=True
)
```

      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
