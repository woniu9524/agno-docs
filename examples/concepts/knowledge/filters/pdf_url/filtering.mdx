## コード

```python
"""
ユーザーレベルのナレッジフィルタリング PDF URL を使用した例

このクックブックは、URL を介してアクセスされる PDF ドキュメントでナレッジフィルターを使用する方法を示しています。
特定の料理、ソース、またはその他のメタデータ属性にナレッジベース検索を制限する方法を示しています。

実証されている主な概念:
1. 特定のメタデータを持つ URL から PDF ドキュメントをロードする
2. 料理タイプでナレッジベース検索をフィルタリングする
3. 複数のフィルター基準を組み合わせる
4. 異なるフィルターの組み合わせの結果を比較する

フィルターは次の方法で渡すことができます。
1. Agent のみに渡す場合、すべての実行に使用します
2. run/print_response のみに渡す場合、その実行に使用します
3. 両方に渡す場合、その実行で run/print_response に渡されたフィルターでオーバーライドします
"""

from agno.agent import Agent
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.vectordb.lancedb import LanceDb

# LanceDB の初期化
# デフォルトでは、データは /tmp/lancedb に保存されます
vector_db = LanceDb(
    table_name="recipes",
    uri="tmp/lancedb",  # データを別の場所に保存するには、このパスを変更できます
)

# ステップ 1: URL とメタデータでナレッジベースを初期化する
# ------------------------------------------------------------------------------
# ナレッジベースを初期化する際に、フィルタリングに使用されるメタデータをアタッチできます
# このメタデータには、料理タイプ、ソース、地域、またはその他の属性を含めることができます

knowledge_base = PDFUrlKnowledgeBase(
    urls=[
        {
            "url": "https://agno-public.s3.amazonaws.com/recipes/thai_recipes_short.pdf",
            "metadata": {
                "cuisine": "Thai",
                "source": "Thai Cookbook",
                "region": "Southeast Asia",
            },
        },
        {
            "url": "https://agno-public.s3.amazonaws.com/recipes/cape_recipes_short_2.pdf",
            "metadata": {
                "cuisine": "Cape",
                "source": "Cape Cookbook",
                "region": "South Africa",
            },
        },
    ],
    vector_db=vector_db,
)

# すべてのドキュメントをベクトルデータベースにロードする
knowledge_base.load(recreate=True)

# ステップ 2: さまざまなフィルターの組み合わせでナレッジベースをクエリする
# ------------------------------------------------------------------------------

# オプション 1: Agent でのフィルター
# ナレッジベースとフィルターで Agent を初期化する
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=True,
    # これにより、タイ料理のドキュメントからの情報のみが返されます
    knowledge_filters={"cuisine": "Thai"},
)

# タイ料理のレシピをクエリする
agent.print_response(
    "Tell me how to make Pad Thai",
    markdown=True,
)

# # オプション 2: run/print_response でのフィルター
# agent = Agent(

# )
```
