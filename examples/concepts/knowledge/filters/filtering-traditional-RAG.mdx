title: 従来のRAGを使用した知識フィルタリング
description: ユーザーID、ドキュメントタイプ、年などのメタデータを使用して、従来のRAGで知識をフィルタリングする方法を学びます。この例では、フィルタを使用して知識ベースをセットアップし、効果的にクエリする方法を示します。
---

## コード

```python filtering-traditional-RAG.py
from agno.agent import Agent
from agno.knowledge.text import TextKnowledgeBase
from agno.utils.media import (
    SampleDataFileExtension,
    download_knowledge_filters_sample_data,
)
from agno.vectordb.lancedb import LanceDb

# すべてのサンプルCVをダウンロードし、そのパスを取得します
downloaded_cv_paths = download_knowledge_filters_sample_data(
    num_files=5, file_extension=SampleDataFileExtension.TXT
)

# LanceDBを初期化します
# デフォルトでは、データは /tmp/lancedb に保存されます
vector_db = LanceDb(
    table_name="recipes",
    uri="tmp/lancedb",  # データを別の場所に保存するには、このパスを変更できます
)

# ステップ1: ドキュメントとメタデータで知識ベースを初期化します
knowledge_base = TextKnowledgeBase(
    path=[
        {
            "path": downloaded_cv_paths[0],
            "metadata": {
                "user_id": "jordan_mitchell",
                "document_type": "cv",
                "year": 2025,
            },
        },
        {
            "path": downloaded_cv_paths[1],
            "metadata": {
                "user_id": "taylor_brooks",
                "document_type": "cv",
                "year": 2025,
            },
        },
        {
            "path": downloaded_cv_paths[2],
            "metadata": {
                "user_id": "morgan_lee",
                "document_type": "cv",
                "year": 2025,
            },
        },
        {
            "path": downloaded_cv_paths[3],
            "metadata": {
                "user_id": "casey_jordan",
                "document_type": "cv",
                "year": 2025,
            },
        },
        {
            "path": downloaded_cv_paths[4],
            "metadata": {
                "user_id": "alex_rivera",
                "document_type": "cv",
                "year": 2025,
            },
        },
    ],
    vector_db=vector_db,
)

# すべてのドキュメントをベクトルデータベースにロードします
knowledge_base.load(recreate=True)

# ステップ2: さまざまなフィルタの組み合わせで知識ベースをクエリします

# オプション1: エージェントでのフィルタリング
agent = Agent(
    knowledge=knowledge_base,
    search_knowledge=False,
    add_references=True,
    knowledge_filters={"user_id": "jordan_mitchell"},
)

# Jordan Mitchellの経験とスキルについて尋ねます
agent.print_response(
    "Tell me about Jordan Mitchell's experience and skills",
    markdown=True,
)

# オプション2: run/print_response でのフィルタリング
# agent = Agent(
#     knowledge=knowledge_base,
#     add_references=True,
#     search_knowledge=False,
# )

# Taylor Brooks を候補者としてクエリします
# agent.print_response(
#     "Tell me about Taylor Brooks as a candidate",
#     knowledge_filters={"user_id": "taylor_brooks"},
#     markdown=True,
# )
```
