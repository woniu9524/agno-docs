## 代码

```python cookbook/tools/website_tools.py
from agno.agent import Agent
from agno.tools.website import WebsiteTools

agent = Agent(
    tools=[WebsiteTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Extract the main content from https://example.com")
```
