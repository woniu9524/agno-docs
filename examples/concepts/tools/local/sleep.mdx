## Code

```python cookbook/tools/sleep_tools.py
from agno.agent import Agent
from agno.tools.sleep import SleepTools

agent = Agent(
    tools=[SleepTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Wait for 5 seconds before continuing")
```
