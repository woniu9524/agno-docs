## 代码

```python cookbook/tools/shell_tools.py
from agno.agent import Agent
from agno.tools.shell import ShellTools

agent = Agent(
    tools=[ShellTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("List all files in the current directory")
```
