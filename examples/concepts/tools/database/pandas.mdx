Description:

AgnoライブラリとPandasToolsを使用して、自動化されたデータ分析のためのAIエージェントを実装しました。

このエージェントはCSVファイル（data.csv）をロードし、自然言語の指示に基づいて分析を実行します。

手動でのPandasコーディングなしでデータと対話でき、データ探索と洞察の抽出を簡素化します。

環境変数と依存関係の設定手順が含まれています。

---
title: Pandas Tools
---

## Code

```python cookbook/tools/pandas_tools.py
from agno.agent import Agent
from agno.tools.pandas import PandasTools

agent = Agent(
    tools=[PandasTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Load and analyze the dataset from data.csv")
```
