描述：

使用 agno 库和 PandasTools 实现了一个用于自动化数据分析的 AI 代理。

该代理加载 CSV 文件 (data.csv) 并根据自然语言指令执行分析。

支持在无需手动编写 Pandas 代码的情况下与数据进行交互，从而简化了数据探索和洞察提取。

包含环境变量和依赖项的设置说明。

---
title: Pandas Tools
---

## 代码

```python cookbook/tools/pandas_tools.py
from agno.agent import Agent
from agno.tools.pandas import PandasTools

agent = Agent(
    tools=[PandasTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Load and analyze the dataset from data.csv")
```
