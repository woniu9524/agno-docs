## 代码

```python cookbook/tools/yfinance_tools.py
from agno.agent import Agent
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    tools=[YFinanceTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("获取 AAPL 的当前股价和近期历史数据")
```
