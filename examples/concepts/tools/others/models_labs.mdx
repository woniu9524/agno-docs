## 代码

```python cookbook/tools/models_labs_tools.py
from agno.agent import Agent
from agno.tools.models_labs import ModelsLabsTools

agent = Agent(
    tools=[ModelsLabsTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("生成一张日落山峦的图片")
```
