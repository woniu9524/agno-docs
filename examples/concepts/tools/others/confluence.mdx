## Code

```python cookbook/tools/confluence_tools.py
from agno.agent import Agent
from agno.tools.confluence import ConfluenceTools

agent = Agent(
    name="Confluence agent",
    tools=[ConfluenceTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("スペースはいくつあり、それらの名前は何ですか？")
agent.print_response(
    "'Large language model in LLM space' という名前のページのコンテンツは何ですか？"
)
agent.print_response("'LLM' スペースからすべてのページ名を抽出できますか？")
agent.print_response("'LLM' スペースに 'TESTING' という名前の新しいページを作成できますか？")
```
