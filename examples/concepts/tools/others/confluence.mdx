## Code

```python cookbook/tools/confluence_tools.py
from agno.agent import Agent
from agno.tools.confluence import ConfluenceTools

agent = Agent(
    name="Confluence agent",
    tools=[ConfluenceTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("有多少个空间？它们分别叫什么名字？")
agent.print_response(
    "‘Large language model in LLM space’这个页面的内容是什么？"
)
agent.print_response("能否从‘LLM’空间提取所有页面名称？")
agent.print_response("能否在‘LLM’空间创建一个名为‘TESTING’的新页面？")
```
