## 代码

```python cookbook/tools/calcom_tools.py
from datetime import datetime

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.calcom import CalComTools

agent = Agent(
    name="日历助手",
    instructions=[
        f"你是一个日程安排助手。今天是 {datetime.now()}。",
        "你可以通过以下方式帮助用户：",
        "    - 查找可用时间段",
        "    - 创建新预订",
        "    - 管理现有预订（查看、重新安排、取消）",
        "    - 获取预订详情",
        "    - 重要提示：在重新安排或取消预订时，请调用 `get_upcoming_bookings` 函数获取预订 uid。在为给定时间进行预订之前，请检查可用时段。",
        "在进行预订或更改之前，请务必确认重要细节。",
    ],
    model=OpenAIChat(id="gpt-4"),
    tools=[CalComTools(user_timezone="America/New_York")],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("我明天的预订有哪些？")
```
