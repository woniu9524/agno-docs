## 代码

```python cookbook/tools/replicate_tools.py
from agno.agent import Agent
from agno.tools.replicate import ReplicateTools

agent = Agent(
    tools=[ReplicateTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("生成一张赛博朋克城市的图片")
```
