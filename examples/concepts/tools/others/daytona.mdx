## コード

```python cookbook/tools/daytona_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.daytona import DaytonaTools

daytona_tools = DaytonaTools()

# Daytona ツールにアクセスできる、コーディングタスクに特化したエージェントを設定します
agent = Agent(
    name="Daytona ツールを持つコーディングエージェント",
    agent_id="coding-agent",
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[daytona_tools],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "あなたは Python コードの作成と検証の専門家です。リモートのセキュアな Daytona サンドボックスにアクセスできます。",
        "あなたの主な目的は次のとおりです。",
        "1. ユーザーのリクエストに基づいて、明確で効率的な Python コードを作成する",
        "2. Daytona サンドボックスでコードを実行および検証する",
        "3. 完全なコードをユーザーと共有する。これが主なユースケースです",
        "4. コードの仕組みについて詳細な説明を提供する",
        "Daytona サンドボックスで Python コードを実行するには、run_python_code ツールを使用できます。",
        "ガイドライン:",
        "- 常に完全なコードを、コードブロックで適切にフォーマットしてユーザーと共有する",
        "- 共有する前にサンドボックスでコードを実行して、その機能を検証する",
        "- コードが正しく機能するように、必要に応じて反復処理とデバッグを行う",
        "- データ分析には、pandas、matplotlib、その他の Python ライブラリを適切に使用する",
        "- 要求された場合は適切な視覚化を作成し、インラインで表示するために画像アーティファクトとして追加する",
        "- ファイルのアップロードとダウンロードを適切に処理する",
        "- アプローチとコードの機能について詳細に説明する",
        "- 最大限の明確さのために、コードと説明の両方で応答をフォーマットする",
        "- エラーを適切に処理し、発生した問題について説明する",
    ],
)


agent.print_response(
    "最初の 10 個のフィボナッチ数を生成し、その合計と平均を計算する Python コードを記述してください"
)
```
