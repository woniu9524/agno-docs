## コード

```python cookbook/tools/e2b_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.e2b import E2BTools

e2b_tools = E2BTools(
    timeout=600,  # 10 分のタイムアウト (秒単位)
    filesystem=True,
    internet_access=True,
    sandbox_management=True,
    command_execution=True,
)

agent = Agent(
    name="コード実行サンドボックス",
    agent_id="e2b-sandbox",
    model=OpenAIChat(id="gpt-4o"),
    tools=[e2b_tools],
    markdown=True,
    show_tool_calls=True,
    instructions=[
        "あなたは、安全な E2B サンドボックス環境を使用して Python コードを記述および検証する専門家です。",
        "あなたの主な目的は次のとおりです。",
        "1. ユーザーのリクエストに基づいて、明確で効率的な Python コードを記述する",
        "2. E2B サンドボックスでコードを実行および検証する",
        "3. このユースケースが主なものであるため、完全なコードをユーザーと共有する",
        "4. コードの仕組みについて徹底的な説明を提供する",
    ],
)

# 例: フィボナッチ数の生成
agent.print_response(
    "最初の 10 個のフィボナッチ数を生成し、その合計と平均を計算する Python コードを記述してください"
)

# 例: データ視覚化
agent.print_response(
    "地域別の売上のサンプルデータセットを作成し、matplotlib で視覚化する Python スクリプトを記述してください"
)

# 例: Web サーバーの実行
agent.print_response(
    "'Hello from E2B Sandbox!' と表示する簡単な FastAPI Web サーバーを作成し、実行して公開 URL を取得してください"
)

# 例: サンドボックス管理
agent.print_response("サンドボックスの現在の状態と、タイムアウトまでの残り時間はどのくらいですか？")

# 例: ファイル操作
agent.print_response("現在の日時を含むテキストファイルを作成し、それを読み込んでください")
```

## 使用方法

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="E2B アカウントの作成">
    [E2B](https://e2b.dev/) でアカウントを作成し、ダッシュボードから API キーを取得してください。
  </Step>

  <Step title="ライブラリのインストール">
    ```bash
    pip install e2b_code_interpreter
    ```
  </Step>

  <Step title="API キーの設定">
    <CodeGroup>
    ```bash Mac/Linux
    export E2B_API_KEY=your_api_key_here
    ```

    ```bash Windows (コマンドプロンプト)
    set E2B_API_KEY=your_api_key_here
    ```

    ```bash Windows (PowerShell)
    $env:E2B_API_KEY="your_api_key_here"
    ```
    </CodeGroup>
  </Step>

  <Step title="エージェントの実行">
    <CodeGroup>
    ```bash Mac/Linux
    python cookbook/tools/e2b_tools.py
    ```

    ```bash Windows
    python cookbook\tools\e2b_tools.py
    ```
    </CodeGroup>
  </Step>
</Steps>
