[Supabase MCP server](https://github.com/supabase-community/supabase-mcp) を使用して、プロジェクト、データベーススキーマ、Edge Function などを作成できるエージェントを作成します。

```python
"""🔑 Supabase MCP Agent - Supabase MCP の機能を紹介します

この例では、Supabase MCP サーバーを使用してプロジェクト、データベーススキーマ、Edge Function などを作成する方法を示します。

セットアップ:
1. Python の依存関係をインストールします: `pip install agno mcp-sdk`
2. Supabase アクセストークンを作成します: https://supabase.com/dashboard/account/tokens で取得し、SUPABASE_ACCESS_TOKEN 環境変数として設定します。
"""

import asyncio
import os
from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.mcp import MCPTools
from agno.tools.reasoning import ReasoningTools
from agno.utils.log import log_error, log_exception, log_info


async def run_agent(task: str) -> None:
    token = os.getenv("SUPABASE_ACCESS_TOKEN")
    if not token:
        log_error("SUPABASE_ACCESS_TOKEN 環境変数が設定されていません。")
        return

    npx_cmd = "npx.cmd" if os.name == "nt" else "npx"

    try:
        async with MCPTools(
            f"{npx_cmd} -y @supabase/mcp-server-supabase@latest --access-token={token}"
        ) as mcp:
            instructions = dedent(f"""
                あなたは Supabase MCP のエキスパートアーキテクトです。プロジェクトの説明に基づいて、以下の手順を自動的に実行してください:
                {task}

                実行する手順:
                1. プロジェクトの説明に基づいて、データベーススキーマ全体を計画します。
                2. `list_organizations` を呼び出し、応答内の最初の組織を選択します。
                3. `get_cost(type='project')` を使用してプロジェクト作成のコストを見積もり、応答にコストを記載します。
                4. 確認されたコスト ID を渡して、`create_project` で新しい Supabase プロジェクトを作成します。
                5. ステータスが `ACTIVE_HEALTHY` になるまで `get_project` でプロジェクトのステータスをポーリングします。
                6. プロジェクトの要件を分析し、DDL ステートメントとして完全で正規化された SQL スキーマ（テーブル、カラム、データ型、インデックス、制約、トリガー、関数）を提案します。
                7. マイグレーション名を `initial_schema` として `apply_migration` を使用してスキーマを適用します。
                8. `list_tables` および `list_extensions` を介して、デプロイされたスキーマを検証します。
                8. `deploy_edge_function` を使用して、簡単なヘルスチェック Edge Function をデプロイします。
                9. プロジェクト URL (`get_project_url`) および匿名キー (`get_anon_key`) を取得して表示します。
            """)
            agent = Agent(
                model=OpenAIChat(id="o4-mini"),
                instructions=instructions,
                tools=[mcp, ReasoningTools(add_instructions=True)],
                markdown=True,
            )

            log_info(f"Supabase プロジェクトエージェントを実行します: {task}")
            await agent.aprint_response(
                message=task,
                stream=True,
                stream_intermediate_steps=True,
                show_full_reasoning=True,
            )
    except Exception as e:
        log_exception(f"予期しないエラー: {e}")


if __name__ == "__main__":
    demo_description = (
        "AI によるタスク提案、カレンダー同期、予測優先順位付け、チームコラボレーション、プロジェクト分析を備えたクラウドベースの SaaS プラットフォームを開発します。"
    )
    asyncio.run(run_agent(demo_description))
