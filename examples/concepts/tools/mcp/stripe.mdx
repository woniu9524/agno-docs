[Stripe MCP サーバー](https://github.com/stripe/agent-toolkit/tree/main/modelcontextprotocol)を使用して、Stripe API と対話できるエージェントを作成します。

```python
"""💵 Stripe MCP Agent - Stripe 操作の管理

この例では、Model Context Protocol (MCP) を介して Stripe API と対話する Agno エージェントの作成方法をデモンストレーションします。このエージェントは、自然言語コマンドを使用して、顧客、製品、価格、支払いリンクなどの Stripe オブジェクトを作成および管理できます。


セットアップ:
2. Python の依存関係をインストールします: `pip install agno mcp-sdk`
3. 環境変数を設定します: export STRIPE_SECRET_KEY=***。

Stripe MCP ドキュメント: https://github.com/stripe/agent-toolkit
"""

import asyncio
import os
from textwrap import dedent

from agno.agent import Agent
from agno.tools.mcp import MCPTools
from agno.utils.log import log_error, log_exception, log_info


async def run_agent(message: str) -> None:
    """
    Stripe MCP サーバーをセットアップし、Agno エージェントを初期化します
    """
    # Stripe API キーが利用可能か検証します
    stripe_api_key = os.getenv("STRIPE_SECRET_KEY")
    if not stripe_api_key:
        log_error("STRIPE_SECRET_KEY 環境変数が設定されていません。")
        return

    enabled_tools = "paymentLinks.create,products.create,prices.create,customers.create,customers.read"

    # 異なるオペレーティングシステムに対応します
    npx_command = "npx.cmd" if os.name == "nt" else "npx"

    try:
        # Stripe サーバーで MCP ツールキットを初期化します
        async with MCPTools(
            command=f"{npx_command} -y @stripe/mcp --tools={enabled_tools} --api-key={stripe_api_key}"
        ) as mcp_toolkit:
            agent = Agent(
                name="StripeAgent",
                instructions=dedent("""\
                    あなたは Stripe 操作の管理を専門とする AI アシスタントです。
                    利用可能なツールを使用して Stripe API と対話します。

                    - ユーザーのリクエストを理解し、Stripe オブジェクト（顧客、製品、価格、支払いリンク）を作成または一覧表示します。
                    - アクションの結果を明確に述べます。作成されたオブジェクトの ID や取得したリストなどを含めます。
                    - リクエストが曖昧な場合は、明確化を求めます。
                    - Markdown フォーマットを使用します。特にリンクやコードスニペットには効果的です。
                    - リクエストが複数のアクションを伴う場合（例：製品を作成し、次に価格を作成し、次にリンクを作成する）、必要な手順を順次実行します。
                """),
                tools=[mcp_toolkit],
                markdown=True,
                show_tool_calls=True,
            )

            # 提供されたタスクでエージェントを実行します
            log_info(f"実行中のエージェントの割り当て: '{message}'")
            await agent.aprint_response(message, stream=True)

    except FileNotFoundError:
        error_msg = f"エラー: '{npx_command}' コマンドが見つかりません。Node.js と npm/npx がインストールされており、システムの PATH に含まれていることを確認してください。"
        log_error(error_msg)
    except Exception as e:
        log_exception(f"エージェントの実行中に予期しないエラーが発生しました: {e}")


if __name__ == "__main__":
    task = "新しい Stripe 製品 'iPhone' を作成してください。次に、それに対して 999.99 ドルの価格を作成してください。最後に、その価格の支払いリンクを作成してください。"
    asyncio.run(run_agent(task))


# プロンプト例:
"""
顧客管理:
- "顧客を作成します。名前: ACME Corp、メール: billing@acme.example.com"
- "顧客を一覧表示します。"
- "メールで顧客を検索 'jane.doe@example.com'" # 注意: 'customers.retrieve' または検索機能が必要です

製品と価格の管理:
- "新しい製品 'Basic Plan' を作成します。"
- "製品 'Basic Plan' に対して、月額 10 ドルの繰り返し価格を作成します。"
- "製品 'Ebook Download' と 19.95 ドルの使い切り価格を作成します。"
- "すべての製品を一覧表示します。" # 注意: 'products.list' 機能が必要です
- "すべての価格を一覧表示します。" # 注意: 'prices.list' 機能が必要です

支払いリンク:
- "月額 10 ドルの 'Basic Plan' 価格の支払いリンクを作成します。"
- "'19.95 ドルの Ebook Download' の支払いリンクを生成します。"

複合タスク:
- "製品 'Pro Service' を作成し、価格 150 ドル（使い切り）を追加し、支払いリンクを提供してください。"
- "新しい顧客 'support@example.com'（名前: 'Support Team'）を登録してください。"
"""
