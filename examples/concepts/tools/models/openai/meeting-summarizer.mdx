この例では、OpenAIToolsとReasoningToolsを使用して会議録音を文字起こしし、主要な洞察を抽出し、視覚的な要約を生成し、音声要約を合成するマルチモーダル会議サマライザーおよびビジュアライザーエージェントについて説明します。

## コード

```python ref/meeting_summarizer_agent.py
"""例: 会議サマライザー & ビジュアライザーエージェント

このスクリプトは、OpenAITools (transcribe_audio, generate_image, generate_speech) を使用して、
会議録音を処理し、要約し、視覚化し、音声要約を作成します。

要件: pip install openai agno
"""

from pathlib import Path
from textwrap import dedent

from agno.agent import Agent
from agno.models.google import Gemini
from agno.tools.openai import OpenAITools
from agno.tools.reasoning import ReasoningTools
from agno.utils.media import download_file, save_base64_data

input_audio_url: str = (
    "https://agno-public.s3.us-east-1.amazonaws.com/demo_data/sample_audio.mp3"
)

local_audio_path = Path("tmp/meeting_recording.mp3")
print(f"ファイルをローカルパスにダウンロード中: {local_audio_path}")
download_file(input_audio_url, local_audio_path)

meeting_agent: Agent = Agent(
    model=Gemini(id="gemini-2.0-flash"),
    tools=[OpenAITools(), ReasoningTools()],
    description=dedent("""\
        あなたは効率的な会議アシスタントAIです。
        会議の音声録音を処理し、主要な情報を抽出し、
        視覚的な表現を作成し、音声要約を提供するのが目的です。
    """),
    instructions=dedent(f"""\
        これらの手順を正確に実行してください:
        1. 音声ファイルへのパスを受け取ります。
        2. `transcribe_audio` ツールを使用してテキストの文字起こしを取得します。
        3. 文字起こしを分析し、主要な議論点、決定事項、およびアクションアイテムを強調する簡潔な要約を作成します。
        4. ステップ3で作成した要約に基づいてのみ、フォーマットされた議事録を作成します。
        5. `generate_speech` ツールを使用して議事録を音声要約に変換します。
    """),
    markdown=True,
    show_tool_calls=True,
)

response = meeting_agent.run(
    f"'{local_audio_path}' にある会議録音を処理してください",
)
if response.audio:
    save_base64_data(response.audio[0].base64_audio, Path("tmp/meeting_summary.mp3"))
    print(f"会議要約を保存しました: {Path('tmp/meeting_summary.mp3')}")
```

    python ref/meeting_summarizer_agent.py

  </Step>
</Steps>
