---
title: レシピ RAG 画像エージェント
---

この例では、Groq と OpenAITools を使用して PDF レシピナレッジベースを検索し、ステップバイステップのビジュアルガイドを生成するマルチモーダル RAG エージェントを紹介します。

## コード

```python ref/recipe_rag_image.py
"""例: マルチモーダル RAG & 画像エージェント

Llama 4 をマルチモーダル RAG に使用し、OpenAITools を使用してレシピのビジュアルでステップバイステップの画像マニュアルを作成するエージェント。

実行: `pip install openai agno groq cohere` で依存関係をインストールしてください
"""

from pathlib import Path

from agno.agent import Agent
from agno.embedder.cohere import CohereEmbedder
from agno.knowledge.pdf_url import PDFUrlKnowledgeBase
from agno.models.groq import Groq
from agno.tools.openai import OpenAITools
from agno.utils.media import download_image
from agno.vectordb.pgvector import PgVector

knowledge_base = PDFUrlKnowledgeBase(
    urls=["https://agno-public.s3.amazonaws.com/recipes/ThaiRecipes.pdf"],
    vector_db=PgVector(
        db_url="postgresql+psycopg://ai:ai@localhost:5532/ai",
        table_name="embed_vision_documents",
        embedder=CohereEmbedder(
            id="embed-v4.0",
        ),
    ),
)

knowledge_base.load()

agent = Agent(
    name="EmbedVisionRAGAgent",
    model=Groq(id="meta-llama/llama-4-scout-17b-16e-instruct"),
    tools=[OpenAITools()],
    knowledge=knowledge_base,
    instructions=[
        "あなたは専門のレシピアシスタントです。",
        "レシピを求められたら:",
        "1. ナレッジベースを検索して、関連するレシピの詳細を取得します。",
        "2. 取得したレシピのステップを注意深く分析します。",
        "3. `generate_image` ツールを使用して、レシピのビジュアルでステップバイステップの画像マニュアルを作成します。",
        "4. レシピのテキストを明確に提示し、付属の画像マニュアルを生成したことを言及します。画像生成中に指示を追加します。",
    ],
    markdown=True,
    debug_mode=True,
)

agent.print_response(
    "タイカレーのレシピは何ですか？",
)

response = agent.run_response
if response.images:
    download_image(response.images[0].url, Path("tmp/recipe_image.png"))
```

## 使用方法

<Steps>
  <Step title="依存関係のインストール">
    ```bash
    pip install openai agno groq cohere
    ```
  </Step>

  <Step title="例の実行">
    ```bash
    python ref/recipe_rag_image.py
    ```
  </Step>
</Steps>

デフォルトでは、生成された画像は `tmp/recipe_image.png` に保存されます。
