## 代码

```python cookbook/tools/twilio_tools.py
from agno.agent import Agent
from agno.tools.twilio import TwilioTools

agent = Agent(
    tools=[TwilioTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("向 +1234567890 发送一条消息，内容为“来自 Agno 的问候！”")
```
