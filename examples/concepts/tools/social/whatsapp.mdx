## Code

```python cookbook/tools/whatsapp_tools.py
from agno.agent import Agent
from agno.models.google import Gemini
from agno.tools.whatsapp import WhatsAppTools

agent = Agent(
    name="whatsapp",
    model=Gemini(id="gemini-2.0-flash"),
    tools=[WhatsAppTools()],
)

# Example: Send a template message
# Note: Replace '''hello_world''' with your actual template name
agent.print_response(
    "Send a template message using the '''hello_world''' template in English to +91 1234567890"
)
```

## Usage

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="WhatsApp Business API のセットアップ">
    1. [Meta for Developers](https://developers.facebook.com/docs/whatsapp/cloud-api/get-started) にアクセスします。
    2. Meta アプリを作成し、WhatsApp Business API をセットアップします。
    3. 電話番号 ID と永続的なシステムユーザーアクセストークンを取得します。
  </Step>

  <Step title="API キーと識別子の設定">
    ```bash
    export WHATSAPP_ACCESS_TOKEN=xxx
    export WHATSAPP_PHONE_NUMBER_ID=xxx
    export OPENAI_API_KEY=xxx # またはお好みの LLM API キー
    ```
  </Step>

  <Step title="ライブラリのインストール">
    ```bash
    pip install -U agno openai google-generativeai # その他必要な WhatsApp SDK を追加してください
    ```
  </Step>

  <Step title="Agent の実行">
    <CodeGroup>
    ```bash Mac
    python cookbook/tools/whatsapp_tools.py
    ```

    ```bash Windows
    python cookbook/tools/whatsapp_tools.py
    ```
    </CodeGroup>
  </Step>
</Steps>
