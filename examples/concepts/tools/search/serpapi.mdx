## 代码

```python cookbook/tools/serpapi_tools.py
from agno.agent import Agent
from agno.tools.serpapi import SerpAPITools

agent = Agent(
    tools=[SerpAPITools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("机器学习”的顶级搜索结果是什么？")
```
