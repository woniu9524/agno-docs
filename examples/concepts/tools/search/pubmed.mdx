## 代码

```python cookbook/tools/pubmed_tools.py
from agno.agent import Agent
from agno.tools.pubmed import PubMedTools

agent = Agent(
    tools=[PubMedTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("查找关于 COVID-19 疫苗的最新研究论文")
```
