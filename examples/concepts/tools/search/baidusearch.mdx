## Code

```python cookbook/tools/baidusearch_tools.py
from agno.agent import Agent
from agno.tools.baidusearch import BaiduSearchTools

agent = Agent(
    tools=[BaiduSearchTools()],
    description="あなたはBaiduを使用して最も関連性の高い情報を見つけるのを支援する検索エージェントです。",
    instructions=[
        "ユーザーからトピックが与えられたら、そのトピックに関する最も関連性の高い検索結果を3つ返してください。",
        "5つの結果を検索し、上位3つのユニークなアイテムを選択してください。",
        "英語と中国語の両方で検索してください。",
    ],
    show_tool_calls=True,
)
agent.print_response("AIにおける最新の進歩は何ですか？", markdown=True)
```
