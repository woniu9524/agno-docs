## コード

```python cookbook/tools/tavily_tools.py
from agno.agent import Agent
from agno.tools.tavily import TavilyTools

agent = Agent(
    tools=[TavilyTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Search for recent breakthroughs in quantum computing")
```
