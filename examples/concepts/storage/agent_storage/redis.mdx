Agno は、`RedisStorage` クラスを使用して、エージェントのストレージバックエンドとして Redis をサポートしています。

## 使用方法

### Redis の実行

[docker desktop](https://docs.docker.com/desktop/install/mac-install/) をインストールし、以下のコマンドでポート **6379** で **Redis** を実行します。

```bash
docker run --name my-redis -p 6379:6379 -d redis
```

```python redis_storage_for_agent.py
from agno.agent import Agent
from agno.storage.redis import RedisStorage
from agno.tools.duckduckgo import DuckDuckGoTools

# デフォルトのローカル接続で Redis ストレージを初期化します
storage = RedisStorage(
    # セッションの名前空間を設定するための Redis キーのプレフィックス
    prefix="agno_test",
    # Redis ホストアドレス
    host="localhost",
    # Redis ポート番号
    port=6379,
)

# Redis ストレージを使用してエージェントを作成します
agent = Agent(
    storage=storage,
)
```
