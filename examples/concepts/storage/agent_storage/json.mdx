Agno は、`JsonStorage` クラスを使用して、エージェントのストレージバックエンドとしてローカル JSON ファイルを使用することをサポートしています。

## 使用方法

```python json_storage_for_agent.py
"""依存関係をインストールするには `pip install duckduckgo-search openai` を実行してください。"""

from agno.agent import Agent
from agno.storage.json import JsonStorage
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    storage=JsonStorage(dir_path="tmp/agent_sessions_json"),
    tools=[DuckDuckGoTools()],
    add_history_to_messages=True,
)
agent.print_response("カナダには何人住んでいますか？")
agent.print_response("国歌の名前は何と呼ばれていますか？")
```
