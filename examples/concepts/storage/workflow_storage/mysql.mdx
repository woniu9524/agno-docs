Agno 支持使用 `MySQLStorage` 类将 MySQL 作为工作流的存储后端。

## 用法

### 运行 MySQL

安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 并使用以下命令在 **3306** 端口运行 **MySQL**：

```bash
docker run -d \
  -e MYSQL_ROOT_PASSWORD=root \
  -e MYSQL_DATABASE=agno \
  -e MYSQL_USER=agno \
  -e MYSQL_PASSWORD=agno \
  -p 3306:3306 \
  --name mysql \
  mysql:8.0
```

```python mysql_storage_for_workflow.py
from agno.storage.mysql import MySQLStorage

db_url = "mysql+pymysql://agno:agno@localhost:3306/agno"

# 使用 MySQL 数据库创建存储后端
storage = MySQLStorage(
    # 将会话存储在 agno.workflows 表中
    table_name="workflow_sessions",
    # db_url：MySQL 数据库 URL
    db_url=db_url,
)

  # 将存储添加到工作流
workflow = Workflow(storage=storage)
```
