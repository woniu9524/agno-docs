title: 使用 Replicate 生成视频
---

## 代码

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.replicate import ReplicateTools

video_agent = Agent(
    name="视频生成代理",
    model=OpenAIChat(id="gpt-4o"),
    tools=[
        ReplicateTools(
            model="tencent/hunyuan-video:847dfa8b01e739637fc76f480ede0c1d76408e1d694b830b5dfb8e547bf98405"
        )
    ],
    description="你是一个可以使用 Replicate API 生成视频的 AI 代理。",
    instructions=[
        "当用户要求你创建视频时，使用 `generate_media` 工具来创建视频。",
        "将 URL 以原始格式返回给用户。",
        "不要将视频 URL 转换为 Markdown 或其他格式。",
    ],
    markdown=True,
    debug_mode=True,
    show_tool_calls=True,
)

video_agent.print_response("生成一个关于沙漠中马匹的视频。")
```
