この例では、エージェントの成功基準を設定する方法を示します。

## コード

```python cookbook/agent_concepts/other/success_criteria.py
from agno.agent import Agent
from agno.models.google import Gemini
from agno.tools.thinking import ThinkingTools

puzzle_master = Agent(
    model=Gemini(id="gemini-2.0-flash"),
    tools=[ThinkingTools(add_instructions=True)],
    instructions="あなたは小さな論理パズルのパズルマスターです。",
    show_tool_calls=False,
    markdown=False,
    stream_intermediate_steps=False,
    success_criteria="すべての飲み物が一意に割り当てられ、パズルが正しく解決されたこと。",
)


prompt = """
小さな論理パズルを作成してください：
3人の友人—アリス、ボブ、キャロル—はそれぞれ、紅茶、コーヒー、ミルクから異なる飲み物を選びます。
手がかり：
1. アリスは紅茶を飲みません。
2. コーヒーを飲む人はキャロルではありません。
質問：誰がどの飲み物を飲んでいますか？
"""

puzzle_master.print_response(prompt, stream=True, show_reasoning=True)
```
