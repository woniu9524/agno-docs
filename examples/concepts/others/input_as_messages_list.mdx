---
title: 入力をメッセージリストとして渡す
---

この例では、メッセージのリストをエージェントへの入力として渡す方法を示します。

## コード

```python cookbook/agent_concepts/other/input_as_messages_list.py
from agno.agent import Agent, Message

Agent().print_response(
    messages=[
        Message(
            role="user",
            content=[
                {"type": "text", "text": "こんにちは！私の名前はジョンです。"},
            ],
        ),
        Message(
            role="user",
            content=[
                {"type": "text", "text": "あなたは何ができますか？"},
            ],
        ),
    ],
    stream=True,
    markdown=True,
)
```
