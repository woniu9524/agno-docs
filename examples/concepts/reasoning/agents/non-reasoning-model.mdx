---
title: 非推論モデルエージェント
---
この例では、非推論モデルを推論モデルとして渡した場合の動作を示します。
デフォルトでは、デフォルトのOpenAI推論モデルを使用します。
適切な推論モデルを使用するか、デフォルトの連鎖思考推論を使用するために `reasoning=True` を渡すことをお勧めします。
## コード

```python cookbook/reasoning/agents/default_chain_of_thought.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning_model=OpenAIChat(
        id="gpt-4o", max_tokens=1200
    ),  # ネイティブの推論モデルではないため、デフォルトで手動COTが使用されるはずです
    markdown=True,
)
reasoning_agent.print_response(
    "フィボナッチ数列のPythonスクリプトを作成する手順を教えてください",
    stream=True,
    show_full_reasoning=True,
)


# エージェントのデフォルトモデルを使用します
reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o", max_tokens=1200),
    reasoning=True,
    markdown=True,
)
reasoning_agent.print_response(
    "フィボナッチ数列のPythonスクリプトを作成する手順を教えてください",
    stream=True,
    show_full_reasoning=True,
)


```
