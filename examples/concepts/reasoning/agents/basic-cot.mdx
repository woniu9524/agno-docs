title: 基本的な推論エージェント

これは、思考連鎖推論を備えた基本的な推論エージェントです。
## コード

```python cookbook/reasoning/agents/analyse_treaty_of_versailles.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

task = (
    "1919年のヴェルサイユ条約調印につながった主な要因を分析してください。 "
    "ドイツに対する条約の政治的、経済的、社会的な影響について議論し、それが第二次世界大戦の勃発にどのように寄与したかを説明してください。 "
    "複数の歴史的視点を含む、ニュアンスのある評価を提供してください。"
)

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning=True,
    markdown=True,
)
reasoning_agent.print_response(task, stream=True, show_full_reasoning=True)
```
