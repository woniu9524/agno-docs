---
title: 推論ツールを備えた推論エージェント
---

この例では、推論ツールを使用して、段階的な推論を通じて複雑な問題を解決するエージェントを作成する方法を示します。エージェントは、質問を分解し、中間結果を分析し、構造化された推論パスを構築して、十分に正当化された結論に到達します。


## コード

```python cookbook/reasoning/tools/reasoning_tools.py


from textwrap import dedent

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.reasoning import ReasoningTools

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    tools=[ReasoningTools(add_instructions=True)],
    instructions=dedent("""\
        あなたは、強力な分析スキルを持つ専門的な問題解決アシスタントです！🧠

        問題へのアプローチ：
        1. まず、複雑な質問を構成要素に分解します。
        2. 仮定を明確に述べます。
        3. 構造化された推論パスを開発します。
        4. 複数の視点を考慮します。
        5. 証拠と反論を評価します。
        6. 十分に正当化された結論を導き出します。

        問題を解決する際：
        - 明示的な段階的推論を使用します。
        - 主要な変数と制約を特定します。
        - 代替シナリオを探ります。
        - 不確実な領域を強調します。
        - 思考プロセスを明確に説明します。
        - 短期および長期的な影響の両方を考慮します。
        - トレードオフを明示的に評価します。

        定量的問題の場合：
        - 計算を示します。
        - 数値の重要性を説明します。
        - 適切な場合は信頼区間を考慮します。
        - ソースデータの信頼性を特定します。

        定性的な推論の場合：
        - さまざまな要因がどのように相互作用するかを評価します。
        - 心理的および社会的力学を考慮します。
        - 実用的な制約を評価します。
        - 価値に関する考慮事項に対処します。
        \
    """),
    add_datetime_to_instructions=True,
    stream_intermediate_steps=True,
    show_tool_calls=True,
    markdown=True,
)

# 例：複雑な推論問題での使用
reasoning_agent.print_response(
    "この論理パズルを解いてください：ある男がキツネ、ニワトリ、そして一袋の穀物を川を渡らなければなりません。 "
    "ボートは男と1つのアイテムしか乗れません。一緒に放置されると、キツネはニワトリを、ニワトリは穀物を食べます。 "
    "男はどうすればすべてを安全に渡すことができますか？",
    stream=True,
)


```
