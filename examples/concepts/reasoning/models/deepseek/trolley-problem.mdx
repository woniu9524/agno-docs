## Code

```python cookbook/reasoning/models/deepseek/trolley_problem.py
from agno.agent import Agent
from agno.models.deepseek import DeepSeek
from agno.models.openai import OpenAIChat

task = (
    "あなたは古典的な「トロッコ問題」を分析する任務を負った哲学者です。このシナリオでは、暴走したトロッコが、動けず縛られている5人の人々に線路上を猛スピードで向かっています。あなたは線路の上にある歩道橋で、見知らぬ大きな男の隣に立っています。線路上にいる5人を救う唯一の方法は、この見知らぬ男を橋から突き落として線路上に転落させることです。これにより見知らぬ男は死亡しますが、線路上にいる5人は救われます。5人を救うために見知らぬ男を突き落とすべきでしょうか？功利主義、義務論、徳倫理の枠組みを考慮した、十分な理由のある回答を提供してください。シナリオを説明する簡単なASCIIアート図も含めてください。"
)

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning_model=DeepSeek(id="deepseek-reasoner"),
    markdown=True,
)
reasoning_agent.print_response(task, stream=True)

```
