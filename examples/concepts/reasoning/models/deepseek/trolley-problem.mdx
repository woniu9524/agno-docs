## 代码

```python cookbook/reasoning/models/deepseek/trolley_problem.py
from agno.agent import Agent
from agno.models.deepseek import DeepSeek
from agno.models.openai import OpenAIChat

task = (
    "你是一名哲学家，负责分析经典的“电车难题”。在这种情况下，一辆失控的电车正沿着轨道冲向被绑在轨道上无法动弹的五个人。你站在轨道上方的天桥上，旁边有一个高大的陌生人。拯救这五个人的唯一方法是把这个陌生人推下桥，让他掉到下面的轨道上。这将杀死陌生人，但能拯救轨道上的五个人。你应该推陌生人以拯救这五个人吗？请提供一个考虑了功利主义、义务论和美德伦理学框架的深思熟虑的答案。包含一个简单的ASCII艺术图来阐述这个场景。"
)

reasoning_agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    reasoning_model=DeepSeek(id="deepseek-reasoner"),
    markdown=True,
)
reasoning_agent.print_response(task, stream=True)

```
