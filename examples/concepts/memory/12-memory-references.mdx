---
title: メモリ参照
---

この例では、エージェントの構成で `add_memory_references` パラメータを使用して、エージェントにユーザーメモリへの参照を追加する方法を示します。

## コード

```12_memory_references.py
from agno.agent.agent import Agent
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory, UserMemory
from agno.models.google.gemini import Gemini

memory_db = SqliteMemoryDb(table_name="memory", db_file="tmp/memory.db")

memory = Memory(db=memory_db)

memory.add_user_memory(
    memory=UserMemory(memory="I like to play soccer", topics=["soccer"]),
    user_id="john_doe@example.com",
)

agent = Agent(
    model=Gemini(id="gemini-2.0-flash-exp"),
    memory=memory,
    add_memory_references=True,  # 既存のメモリをエージェントに追加しますが、新しいメモリは作成しません
)

# または、ユーザーメモリを作成/更新できますが、エージェントには追加しません
# agent = Agent(
#     model=Gemini(id="gemini-2.0-flash-exp"),
#     memory=memory,
#     enable_user_memories=True,
#     add_memory_references=False,
# )

agent.print_response("What are my hobbies?", user_id="john_doe@example.com")
```
