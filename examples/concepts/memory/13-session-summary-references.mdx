---
title: セッション要約参照
---

この例では、エージェント設定の `add_session_summary_references` パラメータを使用して、エージェントにセッション要約への参照を追加する方法を示します。

## コード

```13_session_summary_references.py
from agno.agent.agent import Agent
from agno.memory.v2.db.postgres import PostgresMemoryDb
from agno.memory.v2.memory import Memory
from agno.models.google.gemini import Gemini
from agno.storage.postgres import PostgresStorage

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

memory_db = PostgresMemoryDb(table_name="memory", db_url=db_url)

# この例のためにリセットします
memory_db.clear()

memory = Memory(db=memory_db)

user_id = "john_doe@example.com"
session_id = "session_summaries"

agent = Agent(
    model=Gemini(id="gemini-2.0-flash-exp"),
    memory=memory,
    storage=PostgresStorage(table_name="agent_sessions", db_url=db_url),
    enable_session_summaries=True,
    session_id=session_id,
)

# これにより、新しいセッション要約が作成されます
agent.print_response(
    "私の名前はジョン・ドウです。週末は山でハイキングをするのが好きです。",
    user_id=user_id,
)

# 新しいセッションを作成または更新することなく、セッションストレージから既存のセッション要約を使用できます。
agent = Agent(
    model=Gemini(id="gemini-2.0-flash-exp"),
    memory=memory,
    storage=PostgresStorage(table_name="agent_sessions", db_url=db_url),
    add_session_summary_references=True,
    session_id=session_id,
)

agent.print_response("私の趣味は何ですか？", user_id=user_id)
```
