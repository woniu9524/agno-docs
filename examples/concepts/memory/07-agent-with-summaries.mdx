---
title: セッション要約付きエージェント
---

この例では、セッション要約を作成する方法を示します。

これを有効にするには、エージェントの構成で `enable_session_summaries=True` を設定します。

## コード

```python cookbook/agent_concepts/memory/08_agent_with_summaries.py

from agno.agent.agent import Agent
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory
from agno.memory.v2.summarizer import SessionSummarizer
from agno.models.anthropic.claude import Claude
from rich.pretty import pprint

memory_db = SqliteMemoryDb(table_name="memory", db_file="tmp/memory.db")

memory = Memory(
    db=memory_db,
    summarizer=SessionSummarizer(model=Claude(id="claude-3-5-sonnet-20241022")),
)

# この例のためにメモリをリセットします
memory.clear()

# セッションとユーザーIDが指定されていないため、自動的に生成されます
agent = Agent(
    model=Claude(id="claude-3-5-sonnet-20241022"),
    memory=memory,
    enable_user_memories=True,
    enable_session_summaries=True,
)

agent.print_response(
    "私の名前はジョン・ドウです。週末は山でハイキングをするのが好きです。",
    stream=True,
)

agent.print_response(
    "私の趣味は何ですか？",
    stream=True,
)


memories = memory.get_user_memories()
print("ジョン・ドウのメモリ:")
pprint(memories)
session_summary = agent.get_session_summary()
pprint(session_summary)


# 次に、別のユーザーで新しいセッションを行います
session_id_2 = "1002"
mark_gonzales_id = "mark@example.com"

agent.print_response(
    "私の名前はマーク・ゴンザレスです。アニメとビデオゲームが好きです。",
    stream=True,
    user_id=mark_gonzales_id,
    session_id=session_id_2,
)

agent.print_response(
    "私の趣味は何ですか？",
    stream=True,
    user_id=mark_gonzales_id,
    session_id=session_id_2,
)


memories = memory.get_user_memories(user_id=mark_gonzales_id)
print("マーク・ゴンザレスのメモリ:")
pprint(memories)

# セッション要約をメモリから取得することもできます
session_summary = memory.get_session_summary(
    session_id=session_id_2, user_id=mark_gonzales_id
)
pprint(session_summary)
```
