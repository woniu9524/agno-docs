---
title: Agentic Memory
---

この例では、永続的なメモリをエージェントで使用する方法を示します。

各実行中に、エージェントはユーザーのメモリを作成/更新/削除できます。

これを有効にするには、エージェントの設定で `enable_agentic_memory=True` を設定します。

## コード

```python cookbook/agent_concepts/memory/07_agentic_memory.py
from agno.agent.agent import Agent
from agno.memory.v2.db.sqlite import SqliteMemoryDb
from agno.memory.v2.memory import Memory
from agno.models.openai import OpenAIChat
from rich.pretty import pprint

memory_db = SqliteMemoryDb(table_name="memory", db_file="tmp/memory.db")

# モデルを設定する必要はありません。エージェントによってエージェントのモデルに設定されます。
memory = Memory(db=memory_db)

# この例のためにメモリをリセットします
memory.clear()

john_doe_id = "john_doe@example.com"

agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    memory=memory,
    enable_agentic_memory=True,
)

agent.print_response(
    "私の名前はジョン・ドウで、週末には山でハイキングをするのが好きです。",
    stream=True,
    user_id=john_doe_id,
)

agent.print_response("私の趣味は何ですか？", stream=True, user_id=john_doe_id)

memories = memory.get_user_memories(user_id=john_doe_id)
print("ジョン・ドウに関するメモリ:")
pprint(memories)


agent.print_response(
    "私に関する既存のすべてのメモリを削除してください。",
    stream=True,
    user_id=john_doe_id,
)

memories = memory.get_user_memories(user_id=john_doe_id)
print("ジョン・ドウに関するメモリ:")
pprint(memories)

agent.print_response(
    "私の名前はジョン・ドウで、絵を描くのが好きです。", stream=True, user_id=john_doe_id
)

memories = memory.get_user_memories(user_id=john_doe_id)
print("ジョン・ドウに関するメモリ:")
pprint(memories)


agent.print_response(
    "もう絵は描きません。代わりに描画します。", stream=True, user_id=john_doe_id
)

memories = memory.get_user_memories(user_id=john_doe_id)

print("ジョン・ドウに関するメモリ:")
pprint(memories)
```
