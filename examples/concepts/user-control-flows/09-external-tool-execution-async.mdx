## コード

```python cookbook/agent_concepts/user_control_flows/external_tool_execution_async.py
import asyncio
import subprocess

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools import tool
from agno.utils import pprint

@tool(external_execution=True)
def execute_shell_command(command: str) -> str:
    """シェルコマンドを実行します。

    Args:
        command (str): 実行するシェルコマンド

    Returns:
        str: シェルコマンドの出力
    """
    if command.startswith("ls"):
        return subprocess.check_output(command, shell=True).decode("utf-8")
    else:
        raise Exception(f"サポートされていないコマンド: {command}")

agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[execute_shell_command],
    markdown=True,
)

run_response = asyncio.run(agent.arun("現在のディレクトリにあるファイルは何ですか？"))
if run_response.is_paused:
    for tool in run_response.tools_awaiting_external_execution:
        if tool.tool_name == execute_shell_command.name:
            print(f"外部で {tool.tool_name} を引数 {tool.tool_args} で実行中")
            # ツールを自分で実行します。完全に外部のものを実行することも可能です。
            result = execute_shell_command.entrypoint(**tool.tool_args)
            # エージェントが続行できるように、ツール実行オブジェクトに結果を設定する必要があります
            tool.result = result

    run_response = asyncio.run(agent.acontinue_run(run_response=run_response))
    pprint.pprint_run_response(run_response)
```

## 主な機能

- 非同期エージェント実行に `agent.arun()` を使用
- 非同期継続に `agent.acontinue_run()` を実装
- 同期バージョンと同じ外部ツール実行フローを維持
- 外部ツールでの非同期実行の処理方法を実証

## ユースケース

- 非ブロッキング外部ツール実行
- 非同期実行を必要とする高性能アプリケーション
- 外部サービス呼び出しを伴うWebアプリケーション
- 外部ツールを使用した長時間実行操作
