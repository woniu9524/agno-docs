title: 基本的な状態管理
---
これは、動的なショッピングリストを維持することによってエージェントの状態を管理および更新する方法を示す、基本的なエージェントの状態管理の例です。
## コード

```python cookbook/agent_concepts/state/session_state.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def add_item(agent: Agent, item: str) -> str:
    """ショッピングリストにアイテムを追加します。"""
    agent.session_state["shopping_list"].append(item)
    return f"ショッピングリストは現在 {agent.session_state['shopping_list']} です"


# 状態を維持するエージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # セッション状態をカウンター0から初期化します
    session_state={"shopping_list": []},
    tools=[add_item],
    # セッション状態の変数を指示に使用できます
    instructions="現在の状態（ショッピングリスト）は: {shopping_list}",
    # 重要：状態をメッセージに追加します
    add_state_in_messages=True,
    markdown=True,
)

# 使用例
agent.print_response("ミルク、卵、パンをショッピングリストに追加してください", stream=True)
print(f"最終的なセッション状態: {agent.session_state}")

```
