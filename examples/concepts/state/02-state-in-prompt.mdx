title: InstructionsにおけるState
---
この例では、`add_state_in_messages`を使用して`session_state`変数をエージェントの指示に直接注入する方法を示します。
## コード

```python cookbook/agent_concepts/state/state_in_prompt.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o-mini"),
    # セッション状態を変数で初期化します
    session_state={"user_name": "John"},
    # セッション状態の変数を指示で使用できます
    instructions="Users name is {user_name}",
    show_tool_calls=True,
    add_state_in_messages=True,
    markdown=True,
)

agent.print_response("What is my name?", stream=True)
```
