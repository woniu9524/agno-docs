title: コンテキストの追加
---

この例では、HackerNews のリアルタイムデータにアクセスできるコンテキスト認識エージェントを作成する方法を示します。

## コード

```python cookbook/agent_concepts/context/01-add_context.py
import json
from textwrap import dedent

import httpx
from agno.agent import Agent
from agno.models.openai import OpenAIChat


def get_top_hackernews_stories(num_stories: int = 5) -> str:
    """HackerNews のトップストーリーを取得して返します。

    Args:
        num_stories: 取得するトップストーリーの数 (デフォルト: 5)
    Returns:
        ストーリーの詳細 (タイトル、URL、スコアなど) を含む JSON 文字列
    """
    # トップストーリーを取得
    stories = [
        {
            k: v
            for k, v in httpx.get(
                f"https://hacker-news.firebaseio.com/v0/item/{id}.json"
            )
            .json()
            .items()
            if k != "kids"  # ディスカッショントレッドを除外
        }
        for id in httpx.get(
            "https://hacker-news.firebaseio.com/v0/topstories.json"
        ).json()[:num_stories]
    ]
    return json.dumps(stories, indent=4)


# リアルタイムの HackerNews データにアクセスできるコンテキスト認識エージェントを作成します
agent = Agent(
    model=OpenAIChat(id="gpt-4"),
    # コンテキスト内の各関数は、エージェントの実行時に解決されます。
    # エージェントの依存性注入と考えてください。
    context={"top_hackernews_stories": get_top_hackernews_stories},
    # コンテキスト辞書全体をユーザーメッセージに追加できます
    add_context=True,
    markdown=True,
)

# 使用例
agent.print_response(
    "HackerNews のトップストーリーを要約し、興味深いトレンドを特定してください。",
    stream=True,
)
```
