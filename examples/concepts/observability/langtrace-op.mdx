## 概要

この例では、トレースと監視のために Agno エージェントを Langtrace でインストルメント化する方法を示します。

## コード

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from langtrace_python_sdk import langtrace
from langtrace_python_sdk.utils.with_root_span import with_langtrace_root_span

# Langtrace の初期化
langtrace.init()

# エージェントの作成と設定
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="You are a stock price agent. Answer questions in the style of a stock analyst.",
    debug_mode=True,
)

# エージェントの使用
agent.print_response("What is the current price of Tesla?")
```

## 使用方法

<Steps>
  <Step title="依存関係のインストール">
    ```bash
    pip install agno openai langtrace-python-sdk
    ```
  </Step>

  <Step title="環境変数の設定">
    ```bash
    export LANGTRACE_API_KEY=<your-key>
    ```
  </Step>

  <Step title="エージェントの実行">
    <CodeGroup>
    ```bash Mac
    python cookbook/observability/langtrace_op.py
    ```

    ```bash Windows

    ```
    </CodeGroup>
  </Step>
</Steps>

## 注意事項

- **初期化**: エージェントを使用する前に `langtrace.init()` を呼び出して Langtrace を初期化してください。
