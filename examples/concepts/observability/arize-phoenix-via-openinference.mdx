## 概要

この例では、Agno エージェントを OpenInference でインストルメント化し、Arize Phoenix にトレースを送信する方法を示します。

## コード

```python
import asyncio
import os

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from phoenix.otel import register

# Arize Phoenix の環境変数を設定
os.environ["PHOENIX_CLIENT_HEADERS"] = f"api_key={os.getenv('ARIZE_PHOENIX_API_KEY')}"
os.environ["PHOENIX_COLLECTOR_ENDPOINT"] = "https://app.phoenix.arize.com"

# Phoenix トレーサーを設定
tracer_provider = register(
    project_name="agno-stock-price-agent",  # デフォルトは 'default'
    auto_instrument=True,  # インストールされている OpenInference のインストルメンテーションを自動的に使用
)

# エージェントを作成および設定
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="あなたは株価エージェントです。株アナリストのようなスタイルで質問に答えてください。",
    debug_mode=True,
)

# エージェントを使用
agent.print_response("現在のテスラの株価はいくらですか？")
```
