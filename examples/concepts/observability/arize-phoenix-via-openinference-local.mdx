## 概要

この例では、Agno エージェントに OpenInference を使用して計装し、ローカルの Arize Phoenix コレクターにトレースを送信する方法を示します。

## コード

```python
import os

from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from phoenix.otel import register

# Arize Phoenix のローカルコレクターのエンドポイントを設定します
os.environ["PHOENIX_COLLECTOR_ENDPOINT"] = "http://localhost:6006"

# Phoenix トレーサーを設定します
tracer_provider = register(
    project_name="agno-stock-price-agent",  # デフォルトは 'default' です
    auto_instrument=True,  # インストールされている OpenInference 計装を自動的に使用します
)

# エージェントを作成および設定します
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="あなたは株価エージェントです。株価アナリストのようなスタイルで回答してください。",
    debug_mode=True,
)

# エージェントを使用します
agent.print_response("現在のテスラの株価はいくらですか？")
```

## 使用方法

<Steps>
  <Step title="依存関係のインストール">
    ```bash

    ```
  </Step>

  <Step title="ローカルコレクターの起動">
    ローカルコレクターを起動するには、次のコマンドを実行します。
    ```bash
    phoenix serve
    ```
  </Step>

    </CodeGroup>
  </Step>
</Steps>
