## 概要

この例では、Weights & Biases (WandB) の Weave を使用して、Agno エージェントからのモデル呼び出しをログに記録する方法を示します。

## コード

```python
import weave
from agno.agent import Agent
from agno.models.openai import OpenAIChat

# エージェントを作成および設定します
agent = Agent(model=OpenAIChat(id="gpt-4o"), markdown=True, debug_mode=True)

# プロジェクト名で Weave を初期化します
weave.init("agno")

# weave.op() で装飾されたエージェントを実行する関数を定義します
@weave.op()
def run(content: str):
    return agent.run(content)

# 関数を使用してモデル呼び出しをログに記録します
run("Share a 2 sentence horror story")
```

## 使用方法

<Steps>
  <Step title="Weave のインストール">
    ```bash
    pip install agno openai weave
    ```
  </Step>

  <Step title="WandB での認証">
    - [WandB](https://wandb.ai) にアクセスし、[こちら](https://wandb.ai/authorize) から API キーをコピーします。
    - プロンプトが表示されたらターミナルに API キーを入力するか、環境変数としてエクスポートします。
    ```bash
    export WANDB_API_KEY=<your-api-key>
    ```
  </Step>

  <Step title="エージェントの実行">
    <CodeGroup>
    ```bash Mac
    python cookbook/observability/weave_op.py
    ```

    ```bash Windows
    python cookbook/observability/weave_op.py
    ```
    </CodeGroup>
  </Step>
</Steps>

## 注意事項

- **初期化**: `weave.init("project-name")` を呼び出して、プロジェクト名で Weave を初期化します。
- **デコレータ**: Weave でログに記録したい関数を装飾するには `@weave.op()` を使用します。
