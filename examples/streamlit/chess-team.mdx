title: Chess Battle
---

Chess Battle 是一个国际象棋应用程序，多个 AI 代理在此协同玩国际象棋，展示了多代理系统在复杂游戏环境中的强大能力。

### 主要功能
- 多代理系统：包含用于选择棋步的白棋代理和黑棋代理
- 棋步验证：专门的合法棋步代理确保遵守游戏规则
- 游戏协调：主代理负责监督游戏流程和结束条件
- 交互式用户界面：使用 Streamlit 构建，可进行实时游戏可视化

### 系统组件
- 白棋代理：为白棋进行策略规划和棋步选择
- 黑棋代理：控制并决定黑棋的棋步
- 合法棋步代理：根据国际象棋规则验证所有提议的棋步
- 主代理：协调游戏流程并监控游戏状态

### 高级功能
该系统展示了复杂的代理交互，每个 AI 组件都有特定的角色。代理们进行通信和协调，以创造完整的国际象棋体验，展示了多个专业 AI 如何有效地协同工作。

### 代码
完整代码可在 [Agno 仓库](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/chess_team) 中找到。

### 使用方法
<Steps>
  <Step title="克隆仓库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # 在 Windows 上使用: .venv\Scripts\activate
    ```
  </Step>

  <Step title="安装依赖">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/chess_team/requirements.txt
    ```
  </Step>

  <Step title="设置 API 密钥">
    Chess Team Agent 使用 Anthropic API 进行代理推理：

    ```bash
    export ANTHROPIC_API_KEY=your_api_key_here
    ```
  </Step>

  <Step title="启动应用">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/chess_team/app.py
    ```
  </Step>

  <Step title="打开应用">
    然后在浏览器中打开 http://localhost:8501 开始观看 AI 代理下国际象棋。
  </Step>
</Steps>
