Chess Battle は、複数のAIエージェントが互いにチェスをプレイするために協力するチェスアプリケーションであり、複雑なゲーム環境におけるマルチエージェントシステムの力を実証します。

### 主な機能
- マルチエージェントシステム: 白と黒の駒エージェントが指し手を選択
- 指し手検証: 専用の合法手エージェントがゲームルールの遵守を保証
- ゲーム調整: マスターエージェントがゲームフローと終了条件を監督
- インタラクティブUI: Streamlitで構築され、リアルタイムのゲーム可視化を提供

### システムコンポーネント
- 白駒エージェント: 白の駒の戦略を立て、指し手を選択
- 黒駒エージェント: 黒の駒の指し手を制御し、決定
- 合法手エージェント: すべての提案された指し手をチェスルールに対して検証
- マスターエージェント: ゲームフローを調整し、ゲーム状況を監視

### 高度な機能
このシステムは、各AIコンポーネントが特定の役割を持つ複雑なエージェント間インタラクションを実証します。エージェントは通信および調整を行い、完全なチェスプレイ体験を創造し、複数の専門化されたAIが効果的に協力する方法を示します。

### コード
完全なコードは[Agnoリポジトリ](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/chess_team)で利用可能です。

### 使用方法
<Steps>
  <Step title="リポジトリをクローンする">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="仮想環境を作成する">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # Windowsの場合は: .venv\Scripts\activate を使用
    ```
  </Step>

  <Step title="依存関係をインストールする">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/chess_team/requirements.txt
    ```
  </Step>

  <Step title="APIキーを設定する">
    Chess Team Agent は、エージェントの推論に Anthropic API を使用します。

    ```bash
    export ANTHROPIC_API_KEY=your_api_key_here
    ```
  </Step>

  <Step title="アプリを起動する">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/chess_team/app.py
    ```
  </Step>

  <Step title="アプリを開く">
    次に、ブラウザで http://localhost:8501 を開いて、AIエージェントがチェスをプレイするのを見始めます。
  </Step>
</Steps>

### プロのヒント
- 完全なゲームを視聴する: エージェントの意思決定を理解するために、完全な対戦を観察する
- エージェント間のインタラクションを監視する: エージェントがどのように通信し、調整するかを注意深く見る
