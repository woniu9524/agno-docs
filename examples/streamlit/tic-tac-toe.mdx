title: 井字棋大战
---

井字棋大战是一款井字棋游戏，多个AI代理在此相互竞争，展示了多代理系统在复杂游戏环境中的强大能力。

### 主要功能
- 多代理系统：包含用于选择落子的X棋子代理和O棋子代理
- 落子校验：专门的棋盘代理确保游戏规则的遵守
- 游戏协调：主代理负责监督游戏流程和结束条件
- 交互式用户界面：使用Streamlit构建，实现实时游戏可视化

### 系统组件
- X棋子代理：为X棋子进行策略规划和落子选择
- O棋子代理：控制并决定O棋子的落子
- 棋盘代理：根据游戏规则校验所有提议的落子
- 主代理：协调游戏流程并监控游戏状态

### 高级特性
该系统展示了复杂的代理交互，其中每个AI组件都有特定的角色。代理们通过沟通和协调来创造一个完整的井字棋体验，展示了多个专业AI如何有效地协同工作。

### 代码
完整的代码可在[Agno仓库](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/tic_tac_toe)中找到。

### 使用方法
<Steps>
  <Step title="克隆仓库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # 在Windows上使用: .venv\Scripts\activate
    ```
  </Step>

  <Step title="安装依赖">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/tic_tac_toe/requirements.txt
    ```
  </Step>

  <Step title="设置API密钥">
    井字棋大战支持多种AI模型：

    ```bash
    # OpenAI模型 (GPT-4, GPT-3.5) 所需
    export OPENAI_API_KEY=your_api_key_here
    
    # 可选 - 用于其他模型
    export ANTHROPIC_API_KEY=your_api_key_here  # 用于Claude模型
    export GOOGLE_API_KEY=your_api_key_here     # 用于Gemini模型
    export GROQ_API_KEY=your_api_key_here       # 用于Groq模型
    ```
  </Step>

  <Step title="启动应用">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/tic_tac_toe/app.py
    ```
  </Step>

  <Step title="打开应用">
    然后在浏览器中打开 http://localhost:8501 开始观看AI代理玩井字棋。
  </Step>
</Steps>
