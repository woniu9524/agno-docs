title: 井字棋大战
---

井字棋大战是一个井字棋游戏，多个 AI 代理在此相互竞争，展示了多代理系统在复杂游戏环境中的强大能力。

### 主要功能
- 多代理系统：包含用于落子选择的 X 和 O 棋子代理
- 落子校验：专门的棋盘代理确保游戏规则的遵守
- 游戏协调：主代理负责监督游戏流程和结束条件
- 交互式用户界面：使用 Streamlit 构建，实现实时游戏可视化

### 系统组件
- X 棋子代理：为 X 棋子进行策略制定和落子选择
- O 棋子代理：控制和决定 O 棋子的落子
- 棋盘代理：根据游戏规则校验所有拟议落子
- 主代理：协调游戏流程并监控游戏状态

### 高级特性
该系统展示了复杂的代理交互，其中每个 AI 组件都有特定的角色。代理们通过沟通与协调，创造了完整的井字棋体验，展示了多个专业化 AI 如何有效地协同工作。

### 代码
完整代码可在 [Agno 仓库](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/tic_tac_toe) 获取。

### 使用方法
<Steps>
  <Step title="克隆仓库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # Windows 系统请使用: .venv\Scripts\activate
    ```
  </Step>

  <Step title="安装依赖">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/tic_tac_toe/requirements.txt
    ```
  </Step>

  <Step title="设置 API 密钥">
    井字棋大战支持多种 AI 模型：

    ```bash
    # OpenAI 模型（GPT-4、GPT-3.5）必需
    export OPENAI_API_KEY=your_api_key_here
    
    # 可选 - 用于更多模型
    export ANTHROPIC_API_KEY=your_api_key_here  # 用于 Claude 模型
    export GOOGLE_API_KEY=your_api_key_here     # 用于 Gemini 模型
    export GROQ_API_KEY=your_api_key_here       # 用于 Groq 模型
    ```
  </Step>

  <Step title="启动应用">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/tic_tac_toe/app.py
    ```
  </Step>

  <Step title="打开应用">
    然后在浏览器中打开 http://localhost:8501，开始观看 AI 代理们玩井字棋。
  </Step>
</Steps>
