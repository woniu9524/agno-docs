title: Tic Tac Toe Battle
---

Tic Tac Toe Battle は、複数のAIエージェントが互いに競い合う三目並べゲームであり、複雑なゲーム環境におけるマルチエージェントシステムの力を実証します。

### 主要な機能
- マルチエージェントシステム：駒の選択のためのXピースエージェントとOピースエージェントを特徴とします。
- 移動の検証：専用のボードエージェントがゲームルールの遵守を保証します。
- ゲームの調整：マスターエージェントがゲームフローと終了条件を監督します。
- インタラクティブUI：Streamlitで構築され、リアルタイムのゲーム視覚化を提供します。

### システムコンポーネント
- Xピースエージェント：Xピースの戦略を立て、移動を選択します。
- Oピースエージェント：Oピースの移動を制御し、決定します。
- ボードエージェント：提案されたすべての移動をゲームルールに対して検証します。
- マスターエージェント：ゲームフローを調整し、ゲームの状態を監視します。

### 高度な機能
このシステムは、各AIコンポーネントが特定の役割を持つ複雑なエージェント間インタラクションを実証します。エージェントは通信と調整を行い、完全な三目並べ体験を創り出し、複数の専門化されたAIが効果的に協力できることを示しています。

### コード
完全なコードは[Agnoリポジトリ](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/tic_tac_toe)で利用可能です。

### 使用方法
<Steps>
  <Step title="リポジトリをクローンする">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="仮想環境を作成する">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # Windows の場合は .venv\Scripts\activate を使用します
    ```
  </Step>

  <Step title="依存関係をインストールする">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/tic_tac_toe/requirements.txt
    ```
  </Step>

  <Step title="APIキーを設定する">
    Tic Tac Toe Battle は複数のAIモデルをサポートしています：

    ```bash
    # OpenAI モデル (GPT-4, GPT-3.5) に必要
    export OPENAI_API_KEY=your_api_key_here
    
    # オプション - 追加モデル用
    export ANTHROPIC_API_KEY=your_api_key_here  # Claude モデル用
    export GOOGLE_API_KEY=your_api_key_here     # Gemini モデル用
    export GROQ_API_KEY=your_api_key_here       # Groq モデル用
    ```
  </Step>

  <Step title="アプリを起動する">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/tic_tac_toe/app.py
    ```
  </Step>

  <Step title="アプリを開く">
    次に、ブラウザで http://localhost:8501 を開いて、AIエージェントが三目並べをプレイするのをご覧ください。
  </Step>
</Steps>

### プロのヒント
- 完全なゲームを観戦する：エージェントの意思決定を理解するために、完全な試合を観察します。
- エージェント間のやり取りを監視する：エージェントがどのように通信し、調整するかを注意深く見てください。
- 異なるモデルを試す：さまざまなAIモデルを試して、異なるプレイスタイルを確認します。
