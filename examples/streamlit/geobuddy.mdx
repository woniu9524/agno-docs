GeoBuddy 是一个地理代理，它分析图像以根据可见线索（如地标、建筑和文化符号）预测位置。

### 主要功能
- 位置识别：从上传的图像中预测位置详细信息
- 详细推理：基于视觉线索解释预测结果
- 用户友好的用户界面：使用 Streamlit 构建，提供直观的体验

### 简单的示例供您尝试
- 风景：城市天际线、山脉全景或著名地标
- 建筑：独特的建筑、桥梁或独特的城市景观
- 文化线索：招牌上的文字、语言提示、国旗或独特的服装

### 高级用法
尝试提供包含细微差别的图像，例如不同语言的商店招牌或标志性但不太知名的全球地标。GeoBuddy 将尝试更深入地分析建筑风格、环境（例如沙漠与热带）和文化参考。

### 用法
<Steps>
  <Step title="克隆存储库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
  </Step>

  <Step title="安装依赖项">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/geobuddy/requirements.txt
    ```
  </Step>

  <Step title="设置 API 密钥">
    GeoBuddy 使用 Google PaLM API 进行高级图像推理：

    ```bash
    export GOOGLE_API_KEY=***
    ```
  </Step>

  <Step title="启动应用程序">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/geobuddy/app.py
    ```
  </Step>

  <Step title="打开应用程序">
    然后，在浏览器中打开 http://localhost:8501 开始使用 GeoBuddy。
  </Step>
</Steps>


### 专业技巧
- 高分辨率图像：包含清晰可见招牌或地标的图像可以提高准确性。
- 多种角度：不同的角度（例如街道视角与鸟瞰图）可以展示独特的线索。
- 情境线索：有时，车牌、当地建筑元素甚至植被等细微差别可以显著影响位置猜测。
