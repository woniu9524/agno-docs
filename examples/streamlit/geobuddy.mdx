GeoBuddy は、ランドマーク、建築、文化シンボルなどの目に見える手がかりに基づいて場所を予測する、画像を分析する地理エージェントです。

### 主な機能
- 場所の特定：アップロードされた画像から場所の詳細を予測します
- 詳細な推論：視覚的な手がかりに基づいて予測を説明します
- ユーザーフレンドリーなUI：Streamlitで構築された直感的な体験

### 簡単な試用例
- 風景：都市のスカイライン、山のパノラマ、または有名なランドマーク
- 建築：特徴的な建物、橋、またはユニークな都市景観
- 文化的な手がかり：看板の文字、言語のヒント、旗、またはユニークな服装

### 上級者向けの使い方
店舗の看板が異なる言語で表示されていたり、世界的に有名ではないが象徴的なランドマークであったりするような、微妙な詳細を含む画像を提供してみてください。GeoBuddy は、建築様式、環境（例：砂漠か熱帯か）、文化的な参照について、より深く推論しようとします。


### コード
完全なコードは [Agno リポジトリ](https://github.com/agno-agi/agno) で入手できます。


### 使い方
<Steps>
  <Step title="リポジトリをクローンする">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="仮想環境を作成する">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
  </Step>

  <Step title="依存関係をインストールする">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/geobuddy/requirements.txt
    ```
  </Step>

  <Step title="APIキーを設定する">
    GeoBuddy は、高度な画像推論のために Google PaLM API を使用します。

    ```bash
    export GOOGLE_API_KEY=***
    ```
  </Step>

  <Step title="アプリを起動する">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/geobuddy/app.py
    ```
  </Step>

  <Step title="アプリを開く">
    次に、ブラウザで http://localhost:8501 を開いて GeoBuddy の使用を開始します。
  </Step>
</Steps>


### プロのヒント
- 高解像度の画像：看板やランドマークがはっきりと見える鮮明な画像は、精度を向上させます。
- さまざまな角度：さまざまな角度（例：路上からの視点 vs. 空撮）は、ユニークな手がかりを示すことができます。
- 文脈の手がかり：時には、ナンバープレート、地元の建築要素、さらには植生のような些細な詳細が、場所の推測に大きく影響する可能性があります。
