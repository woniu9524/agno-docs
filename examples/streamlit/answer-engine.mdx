この例では、ユーザーのクエリに基づいて、Web検索を実行するか、ExaToolsを使用して詳細な分析を行うかをインテリジェントに判断する、PerplexityライクなAnswer EngineであるSageの構築方法を示します。

Sageは以下の機能を提供します：
1. リアルタイムのWeb検索（DuckDuckGo）と詳細なコンテキスト分析（ExaTools）を使用して包括的な回答を提供します。
2. クエリの複雑さに基づいてインテリジェントにツールを選択します。
3. セッション管理とチャット履歴のエクスポートを備えたインタラクティブなStreamlit UIを提供します。
4. 複数のLLMプロバイダー（OpenAI、Anthropic、Google、Groq）をサポートします。

## 主要な機能
- 自然言語クエリの理解と処理
- DuckDuckGoとのリアルタイムWeb検索統合
- ExaToolsを使用した詳細なコンテキスト分析
- 複数のLLMプロバイダーサポート
- SQLiteを使用したセッション管理
- チャット履歴のエクスポート
- インタラクティブなStreamlit UI

## 試してみる簡単なクエリ
- "2025年に米国が課す関税について教えてください"
- "推論モデルとして、o3-miniとDeepSeek R1のどちらが良いですか？"
- "Agnoについて教えてください"
- "再生可能エネルギーの最新トレンドは何ですか？"

## 高度な分析クエリ
- "新興のAI規制がイノベーションにどのように影響するかを評価してください"
- "電気自動車と水素自動車の環境への影響を比較してください"
- "世界の半導体サプライチェーンの課題を分析してください"
- "量子コンピューティングが暗号化に与える影響を説明してください"

## コード

完全なコードは[Agnoリポジトリ](https://github.com/agno-agi/agno)で入手できます。

## 使用方法

<Steps>
  <Step title="リポジトリをクローンする">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="仮想環境を作成する">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # Windowsの場合: .venv\Scripts\activate
    ```
  </Step>

  <Step title="依存関係をインストールする">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/answer_engine/requirements.txt
    ```
  </Step>

  <Step title="APIキーを設定する">
    ```bash
    # 必須
    export OPENAI_API_KEY=***
    export EXA_API_KEY=***

    # オプション（追加モデル用）
    export ANTHROPIC_API_KEY=***
    export GOOGLE_API_KEY=***
    export GROQ_API_KEY=***
    ```
    最適なパフォーマンスのためには、gpt-4oの使用をお勧めします。
  </Step>

  <Step title="アプリを起動する">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/answer_engine/app.py
    ```
    [localhost:8501](http://localhost:8501)を開いてSageの使用を開始します。
  </Step>
</Steps>

## モデル選択
このアプリケーションは複数のモデルプロバイダーをサポートしています：
- OpenAI (o3-mini, gpt-4o)
- Anthropic (claude-3-5-sonnet)
- Google (gemini-2.0-flash-exp)
- Groq (llama-3.3-70b-versatile)

## エージェント設定
エージェントの設定は`agents.py`に、プロンプトは`prompts.py`にあります：
- プロンプトを変更するには、`prompts.py`ファイルを更新してください。
- 新しいツールやモデルを追加するには、`agents.py`ファイルを更新してください。
