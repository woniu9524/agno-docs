---
title: シークレットの追加
---

シークレット管理は、アプリケーションセキュリティの重要な部分であり、真剣に取り組む必要があります。

ローカルシークレットは `worspace/secrets` ディレクトリで定義され、バージョン管理から除外されています（`.gitignore` を参照）。その内容はパスワードと同様のセキュリティで取り扱う必要があります。

本番環境のシークレットは [AWS Secrets Manager](https://docs.aws.amazon.com/secretsmanager/latest/userguide/intro.html) によって管理されます。

<Note>

secrets ディレクトリが見つからない場合は、`workspace/example_secrets` をコピーしてください。

</Note>

## 開発用シークレット

ローカルで実行されるアプリケーションは、`yaml` ファイルを使用してシークレットを読み取ることができます。例：

```python dev_resources.py
dev_fastapi = FastApi(
    ...
    # secrets/dev_app_secrets.yml からシークレットを読み取る
    secrets_file=ws_settings.ws_root.joinpath("workspace/secrets/dev_app_secrets.yml"),
)
```

## 本番環境用シークレット

`AWS Secrets` は本番環境のシークレットを管理するために使用され、本番アプリケーションによって読み取られます。

```python prd_resources.py
# -*- 本番アプリケーション用のシークレット
prd_secret = SecretsManager(
    ...
    # workspace/secrets/prd_app_secrets.yml からシークレットを作成
    secret_files=[
        ws_settings.ws_root.joinpath("workspace/secrets/prd_app_secrets.yml")
    ],
)

# -*- 本番データベース用のシークレット
prd_db_secret = SecretsManager(
    ...
    # workspace/secrets/prd_db_secrets.yml からシークレットを作成
    secret_files=[ws_settings.ws_root.joinpath("workspace/secrets/prd_db_secrets.yml")],
)
```

本番アプリケーションでシークレットを読み取るには、以下を使用します：

<CodeGroup>

```python FastApi
prd_fastapi = FastApi(
    ...
    aws_secrets=[prd_secret],
    ...
)
```

```python RDS
prd_db = DbInstance(
    ...
    aws_secret=prd_db_secret,
    ...
)
```

</CodeGroup>

本番環境のリソースは、yaml ファイルを使用してシークレットを読み取ることもできますが、[AWS Secrets](https://docs.aws.amazon.com/secretsmanager/latest/userguide/intro.html) の使用を強く推奨します。
