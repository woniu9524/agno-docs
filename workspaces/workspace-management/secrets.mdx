---
title: 添加 Secrets
---

Secret 管理是应用程序安全的关键部分，应认真对待。

本地 secrets 定义在 `worspace/secrets` 目录中，该目录已从版本控制中排除（请参阅 `.gitignore`）。其内容应与密码一样受到同等安全级别的处理。

生产环境 secrets 由 [AWS Secrets Manager](https://docs.aws.amazon.com/secretsmanager/latest/userguide/intro.html) 管理。

<Note>

如果缺少 secrets 目录，请复制 `workspace/example_secrets`

</Note>

## 开发环境 Secrets

本地运行的应用程序可以通过 `yaml` 文件读取 secrets，例如：

```python dev_resources.py
dev_fastapi = FastApi(
    ...
    # 从 secrets/dev_app_secrets.yml 读取 secrets
    secrets_file=ws_settings.ws_root.joinpath("workspace/secrets/dev_app_secrets.yml"),
)
```

## 生产环境 Secrets

`AWS Secrets` 用于管理生产环境 secrets，这些 secrets 由生产应用程序读取。

```python prd_resources.py
# -*- 生产应用程序的 Secrets
prd_secret = SecretsManager(
    ...
    # 从 workspace/secrets/prd_app_secrets.yml 创建 secret
    secret_files=[
        ws_settings.ws_root.joinpath("workspace/secrets/prd_app_secrets.yml")
    ],
)

# -*- 生产数据库的 Secrets
prd_db_secret = SecretsManager(
    ...
    # 从 workspace/secrets/prd_db_secrets.yml 创建 secret
    secret_files=[ws_settings.ws_root.joinpath("workspace/secrets/prd_db_secrets.yml")],
)
```

使用以下方式在生产应用程序中读取 secret：

<CodeGroup>

```python FastApi
prd_fastapi = FastApi(
    ...
    aws_secrets=[prd_secret],
    ...
)
```

```python RDS
prd_db = DbInstance(
    ...
    aws_secret=prd_db_secret,
    ...
)
```

</CodeGroup>

生产环境资源也可以通过 yaml 文件读取 secrets，但我们强烈建议使用 [AWS Secrets](https://docs.aws.amazon.com/secretsmanager/latest/userguide/intro.html)。
