---
title: SSH アクセス
---

SSH アクセスは、開発者のワークフローにおいて重要な部分です。

## 開発環境 SSH アクセス

`docker exec` コマンドを使用して、開発コンテナに SSH 接続します。

```bash
docker exec -it ai-api zsh
```

## 本番環境 SSH アクセス

ECS タスクには、すでに SSH アクセスが有効になっています。本番環境コンテナに SSH 接続するには、以下を使用します。

```bash
ECS_CLUSTER=ai-app-prd-cluster
TASK_ARN=$(aws ecs list-tasks --cluster ai-app-prd-cluster --query "taskArns[0]" --output text)
CONTAINER_NAME=ai-api-prd

aws ecs execute-command --cluster $ECS_CLUSTER \
    --task $TASK_ARN \
    --container $CONTAINER_NAME \
    --interactive \
    --command "zsh"
