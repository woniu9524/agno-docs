---
title: 为新用户设置工作空间
sidebarTitle: "添加新用户"
---

请按照以下步骤设置现有工作空间：

<Steps>

<Step title="克隆 git 仓库">
克隆 git 仓库并 `cd` 进入工作空间目录

<CodeGroup>

```bash Mac
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

```bash Windows
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

</CodeGroup>

</Step>
<Step title="创建并激活虚拟环境">

<CodeGroup>

```bash Mac
python3 -m venv aienv
source aienv/bin/activate
```

```bash Windows
python3 -m venv aienv
aienv/scripts/activate
```

</CodeGroup>

</Step>
<Step title="安装 agno">

<CodeGroup>

```bash Mac
pip install -U agno
```

```bash Windows
pip install -U agno
```

</CodeGroup>

</Step>
<Step title="设置工作空间">

<CodeGroup>

```bash Mac
ag ws setup
```

```bash Windows
ag ws setup
```

</CodeGroup>

</Step>
<Step title="复制敏感信息">

将 `workspace/example_secrets` 复制到 `workspace/secrets`

<CodeGroup>

```bash Mac
cp -r workspace/example_secrets workspace/secrets
```

```bash Windows
cp -r workspace/example_secrets workspace/secrets
```

</CodeGroup>

</Step>
<Step title="启动工作空间">

<Note>

如果需要，请安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/)。

</Note>

<CodeGroup>

```bash terminal
ag ws up
```

```bash full options
ag ws up --env dev --infra docker
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

</Step>
<Step title="停止工作空间">

<CodeGroup>

```bash terminal
ag ws down
```

```bash full options
ag ws down --env dev --infra docker
```

```bash shorthand
ag ws down dev:docker
```

</CodeGroup>

</Step>

</Steps>
