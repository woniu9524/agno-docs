---
title: 新規ユーザー向けワークスペースのセットアップ
sidebarTitle: "新規ユーザーの追加"
---

既存のワークスペースをセットアップするには、以下の手順に従ってください。

<Steps>

<Step title="gitリポジトリのクローン">
gitリポジトリをクローンして、ワークスペースディレクトリに`cd`します。

<CodeGroup>

```bash Mac
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

```bash Windows
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

</CodeGroup>

</Step>
<Step title="仮想環境の作成とアクティベート">

<CodeGroup>

```bash Mac
python3 -m venv aienv
source aienv/bin/activate
```

```bash Windows
python3 -m venv aienv
aienv/scripts/activate
```

</CodeGroup>

</Step>
<Step title="agnoのインストール">

<CodeGroup>

```bash Mac
pip install -U agno
```

```bash Windows
pip install -U agno
```

</CodeGroup>

</Step>
<Step title="ワークスペースのセットアップ">

<CodeGroup>

```bash Mac
ag ws setup
```

```bash Windows
ag ws setup
```

</CodeGroup>

</Step>
<Step title="シークレットのコピー">

`workspace/example_secrets` を `workspace/secrets` にコピーします。

<CodeGroup>

```bash Mac
cp -r workspace/example_secrets workspace/secrets
```

```bash Windows
cp -r workspace/example_secrets workspace/secrets
```

</CodeGroup>

</Step>
<Step title="ワークスペースの起動">

<Note>

必要に応じて[docker desktop](https://docs.docker.com/desktop/install/mac-install/)をインストールしてください。

</Note>

<CodeGroup>

```bash terminal
ag ws up
```

```bash full options
ag ws up --env dev --infra docker
```

```bash shorthand
ag ws up dev:docker
```

</CodeGroup>

</Step>
<Step title="ワークスペースの停止">

<CodeGroup>

```bash terminal
ag ws down
```

```bash full options
ag ws down --env dev --infra docker
```

```bash shorthand
ag ws down dev:docker
```

</CodeGroup>

</Step>

</Steps>
