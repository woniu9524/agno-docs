---
title: Git リポジトリの作成
---

アプリケーションをチームと共有するために Git リポジトリを作成します。

<Steps>
  <Step title="Git リポジトリの作成">
    新しい [Git リポジトリ](https://github.com/new) を作成します。
  </Step>
  <Step title="コードのプッシュ">
    コードを Git リポジトリにプッシュします。

    ```bash terminal
    git init
    git add .
    git commit -m "Init LLM App"
    git branch -M main
    git remote add origin https://github.com/[YOUR_GIT_REPO].git
    git push -u origin main
    ```

  </Step>
  <Step title="チームメンバーへの参加依頼">
    チームメンバーに、このワークスペースを使用するために [新規ユーザーのセットアップ手順](/workspaces/workspace-management/new-users) に従うように依頼します。
  </Step>

</Steps>
