## Agno のインストール

以下を強く推奨します。

- Python の仮想環境で `pip` を使用して `agno` をインストールすること。
- AI ワークスペース用に `ai` ディレクトリを作成すること。

<Steps>
  <Step title="仮想環境の作成">
    `Terminal` を開き、Python の仮想環境を持つ `ai` ディレクトリを作成します。

    <CodeGroup>

    ```bash Mac
    mkdir ai && cd ai

    python3 -m venv aienv
    source aienv/bin/activate
    ```

    ```bash Windows
    mkdir ai; cd ai

    python3 -m venv aienv
    aienv/scripts/activate
    ```

    </CodeGroup>

  </Step>
  <Step title="Agno のインストール">
    pip を使用して `agno` をインストールします。

    <CodeGroup>

    ```bash Mac
    pip install -U agno
    ```

    ```bash Windows
    pip install -U agno
    ```

    </CodeGroup>

  </Step>
  <Step title="Docker のインストール">
    アプリをローカルで実行するために [Docker Desktop](https://docs.docker.com/desktop/install/mac-install/) をインストールします。
  </Step>
</Steps>

<br />

<Note>

エラーが発生した場合は、`python -m pip install --upgrade pip` を使用して pip を更新してみてください。

</Note>

---
## Agno のアップグレード

`agno` をアップグレードするには、仮想環境で以下を実行します。

```bash
pip install -U agno --no-cache-dir
```

---
## ワークスペースのセットアップ

既存の `agno` ワークスペースがある場合は、以下を使用してセットアップします。

```bash
ag ws setup
```

---
## Agno のリセット

`agno` の設定をリセットするには、以下を実行します。

```bash
ag init -r
```

<Note>

これにより、物理的なデータは削除されません。

</Note>
