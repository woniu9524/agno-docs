## 安装 Agno

我们强烈推荐：

- 在 Python 虚拟环境中，使用 `pip` 安装 `agno`。
- 创建一个 `ai` 目录用于存放您的 AI 工作区。

<Steps>
  <Step title="创建虚拟环境">
    打开 `Terminal`，创建一个 `ai` 目录并设置 Python 虚拟环境。

    <CodeGroup>

    ```bash Mac
    mkdir ai && cd ai

    python3 -m venv aienv
    source aienv/bin/activate
    ```

    ```bash Windows
    mkdir ai; cd ai

    python3 -m venv aienv
    aienv/scripts/activate
    ```

    </CodeGroup>

  </Step>
  <Step title="安装 Agno">
    使用 pip 安装 `agno`。

    <CodeGroup>

    ```bash Mac
    pip install -U agno
    ```

    ```bash Windows
    pip install -U agno
    ```

    </CodeGroup>

  </Step>
  <Step title="安装 Docker">
    安装 [Docker Desktop](https://docs.docker.com/desktop/install/mac-install/) 以在本地运行应用程序。
  </Step>
</Steps>

<br />

<Note>

如果遇到错误，请尝试使用 `python -m pip install --upgrade pip` 更新 pip。

</Note>

---
## 升级 Agno

要升级 `agno`，请在您的虚拟环境中运行以下命令：

```bash
pip install -U agno --no-cache-dir
```

---
## 设置工作区

如果您已有 `agno` 工作区，请使用以下命令进行设置：

```bash
ag ws setup
```

---
## 重置 Agno

要重置 `agno` 配置，请运行：

```bash
ag init -r
```

<Note>

这不会删除任何物理数据。

</Note>
