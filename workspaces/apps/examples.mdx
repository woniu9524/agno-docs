## 在 Docker 中运行 Jupyter

Jupyter Notebook 是 AI 开发的必备工具。请将 `resources.py` 文件更新为：

```python resources.py
from os import getenv

from agno.docker.app.jupyter import Jupyter
from agno.docker.app.postgres import PgVectorDb
from agno.docker.resources import DockerResources

# -*- PgVector 运行在端口 5432:5432
vector_db = PgVectorDb(
    pg_user="ai",
    pg_password="ai",
    pg_database="ai",
    debug_mode=True,
)

# -*- Jupyter 运行在端口 8888:8888
jupyter = Jupyter(
    mount_workspace=True,
    env_vars={"OPENAI_API_KEY": getenv("OPENAI_API_KEY")},
)

# -*- DockerResources
dev_docker_resources = DockerResources(
    apps=[vector_db, jupyter],
)
```

使用以下命令启动资源：

<CodeGroup>

```bash Mac
ag start resources.py
```

```bash Windows
ag start resources.py
```

</CodeGroup>

### 查看 Jupyterlab UI

- 打开 [localhost:8888](http://localhost:8888) 查看 Jupyterlab UI。密码：**admin**
- 该目录已自动挂载到 Notebook 中。

## 停止资源

<CodeGroup>

```bash Mac
ag stop resources.py
```

```bash Windows
ag stop resources.py
```

</CodeGroup>
