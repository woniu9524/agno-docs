Agent App 是我们构建 Agentic 系统的首选工作空间。它包含：

- 一个用于服务 Agents、Teams 和 Workflows 的 FastAPI 服务器。
- 一个用于调试和测试的 streamlit 应用程序。这个 streamlit 应用功能非常强大，可以用作 Agentic 系统的管理界面，并显示各种数据。
- 一个用于会话和向量存储的 postgres 数据库。

它被设计为使用 docker 在本地运行，并在 AWS 上进行生产部署。
