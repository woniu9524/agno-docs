アクティブなワークスペースのリソースを削除する

## パラメータ

<ResponseField name="resources_filter" type="str">
 リソースフィルター。フォーマット - ENV:INFRA:GROUP:NAME:TYPE
</ResponseField>
<ResponseField name="env_filter" type="str">
 デプロイする環境をフィルターします `--env` `-e`
</ResponseField>
<ResponseField name="infra_filter" type="str">
 デプロイするインフラストラクチャをフィルターします。 `--infra` `-i`
</ResponseField>
<ResponseField name="config_filter" type="str">
 デプロイする設定をフィルターします。 `--config` `-c`
</ResponseField>
<ResponseField name="group_filter" type="str">
 グループ名を使用してリソースをフィルターします。 `--group` `-g`
</ResponseField>
<ResponseField name="name_filter" type="str">
 名前を使用してリソースをフィルターします。 `--name` `-n`
</ResponseField>
<ResponseField name="type_filter" type="str">
 タイプを使用してリソースをフィルターします。 `--type` `-t`
</ResponseField>
<ResponseField name="dry_run" type="bool">
 リソースを出力して終了します。 `--dry-run` `-dr`
</ResponseField>
<ResponseField name="auto_confirm" type="bool">
 リソースのデプロイ前に確認をスキップします。 `--yes` `-y`
</ResponseField>
<ResponseField name="print_debug_log" type="bool">
 デバッグログを出力します。 `--debug` `-d`
</ResponseField>
<ResponseField name="force" type="bool">
 強制します。 `--force` `-f`
</ResponseField>
