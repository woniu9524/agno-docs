`WorkspaceSettings` オブジェクトは、通常 `workspace/settings.py` ファイルで定義され、ワークスペースのアプリやリソースで使用される共通の設定を定義するために使用されます。

これは必須ではなく、ワークスペースのアプリやリソースによって構成される値を保持する以外の目的はありません。`WorkspaceSettings` オブジェクトの値は、環境変数または `.env` ファイルを使用して設定することもできます。

## 例

`llm-app` テンプレートで使用される `WorkspaceSettings` の例です。このファイルは[github](https://github.com/agno-agi/llm-app/blob/main/workspace/settings.py)で表示できます。

```python workspace/settings.py
from pathlib import Path

from agno.workspace.settings import WorkspaceSettings

#
# -*- WorkspaceSettings オブジェクトを使用してワークスペース設定を定義します
# これらの値は、環境変数または .env ファイルを使用して設定することもできます
#
ws_settings = WorkspaceSettings(
    # Workspace name: リソースの命名に使用されます
    ws_name="ai",
    # Workspace のルートへのパス
    ws_root=Path(__file__).parent.parent.resolve(),
    # -*- Dev settings
    dev_env="dev",
    # -*- Dev Apps
    dev_app_enabled=True,
    dev_api_enabled=True,
    dev_db_enabled=True,
    # dev_jupyter_enabled=True,
    # -*- Production settings
    prd_env="prd",
    # -*- Production Apps
    prd_app_enabled=True,
    prd_api_enabled=True,
    prd_db_enabled=True,
    # -*- AWS settings
    # AWS リソースのリージョン
    aws_region="us-east-1",
    # AWS リソースのアベイラビリティゾーン
    aws_az1="us-east-1a",
    aws_az2="us-east-1b",
    # aws_region のサブネット ID
    # subnet_ids=["subnet-xyz", "subnet-xyz"],
    #
    # -*- Image Settings
    #
    # イメージのデフォルトリポジトリ
    image_repo="agno"
    # イメージをローカルでビルドします
    build_images=False
    # ビルド後にイメージをプッシュします
    push_images=False
    # イメージをビルドする際にキャッシュをスキップします
    skip_image_cache=False
    # FROM でイメージを強制的にプルします
    force_pull_images=False
)
```

## 使用方法

ワークスペース設定を使用して以下を行います。

- リソースに名前を付ける
- `ws_settings.ws_root` を使用してワークスペースのルートパスを取得する

```python dev_resources.py
...
# -*- Streamlit をポート 8501:8501 で実行します
dev_streamlit = Streamlit(
    name=f"{ws_settings.dev_key}-app",
    enabled=ws_settings.dev_app_enabled,
    ...
    # secrets/dev_app_secrets.yml からシークレットを読み込みます
    secrets_file=ws_settings.ws_root.joinpath("workspace/secrets/dev_app_secrets.yml")
)
```

- `availability zone` や `subnets` などの AWS 定数を保持する

```python prd_resources.py
# -*- FastAPI を ECS で実行します
prd_fastapi = FastApi(
    name=f"{ws_settings.prd_key}-api",
    enabled=ws_settings.prd_api_enabled,
    ...
    subnets=ws_settings.subnet_ids,
    ...
)

# -*- RDS Database Instance
prd_db = DbInstance(
    name=f"{ws_settings.prd_key}-db",
    enabled=ws_settings.prd_db_enabled,
    ...
    availability_zone=ws_settings.aws_az1,
    ...
)
