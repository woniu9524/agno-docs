AWS リソースを使用すると、AWS サービスを pydantic オブジェクトとして作成でき、ソフトウェア、アプリケーション、インフラストラクチャコードをすべて Python で記述するというビジョンを完成させることができます。

## 例

### S3 バケット

次のコードを `resources.py` というファイルにコピーし、`ag start resources.py` を実行して `my-bucket-885` という名前のバケットを作成します。

<CodeGroup>

```python resources.py
from agno.aws.resource.s3 import S3Bucket

# -*- my-bucket-885 という名前の S3 バケット
prd_bucket = S3Bucket(name="my-bucket-885")
```

</CodeGroup>

`ag stop resources.py` を使用してバケットを削除してください。

### Secret Manager

次のコードを `resources.py` というファイルにコピーし、`ag start resources.py` を実行して `my-secret` という名前のシークレットを作成します。

<CodeGroup>

```python resources.py
import json
from agno.aws.resource.secret import SecretsManager

# -*- my-secret という名前のシークレット
prd_secret = SecretsManager(
    name="my-secret",
    secret_string=json.dumps({"mysecretkey": "mysecretvalue"}),
    # my_secrets.yml からシークレット変数を読み込む
    # secret_files=[Path('my_secrets.yml')],
)
```

</CodeGroup>

`read_my_secret.py` という別のファイルでシークレットを読み込みます。

<CodeGroup>

```python read_my_secret.py
from resources import my_secret

print(my_secret.get_secret_value("mysecretkey"))
```

</CodeGroup>

このファイルを `python read_my_secret.py` を使用して実行します。

`ag stop resources.py` を使用してシークレットを削除してください。
